# 汉文帝六重合法性建构：外藩入继一日稳政与新君即位完整方法论

汉文帝刘恒从代国边地入继，一日一夜内完成六重合法性编织：天命（龟卜大横）+宗法（高帝见子最长）+功臣（八人再拜，覆盖谋士/军方/文官/宗室/诛吕功臣五系）+礼仪（西乡让三南乡让再）+法统（当夜诛伪皇子）+德政（大赦赐爵牛酒五日）。同时军权归心腹（宋昌、张武），名位给功臣（周勃、陈平）。文帝在位23年无严重政变，死后获"庙曰太宗"第二庙号。

**来源**：《史记·孝文本纪》（卷010）

> "〖;代王〗西乡让者三，南乡让者再。代王曰：'宗室将相王列侯以为莫宜寡人，寡人不敢辞。'遂即天子位。"
> "夜，有司分部诛灭梁、淮阳、常山王及少帝於邸。"

## 六重合法性矩阵

| 合法性类型 | 操作 | 瞄准的质疑者 |
|-----------|------|------------|
| 天命合法性 | 龟卜大横（"余为天王，夏启以光"）| 天人感应信徒 / 阴阳家势力 |
| 宗法合法性 | "高帝见子，最长，仁孝宽厚，立长故顺" | 宗室 / 儒者 / 普通民众 |
| 功臣合法性 | 八位元老联名（陈平、周勃、张苍、刘章等）| 军功集团 / 文官集团 / 宗室集团 |
| 礼仪合法性 | 西乡让三，南乡让再（共五次辞让）| 儒家礼教信徒 / 担忧篡位者 |
| 法统合法性 | 当夜诛伪皇子（定性少帝为非惠帝子）| 前少帝旧部 / 反对派拥立点 |
| 德政合法性 | 大赦天下 + 赐民爵一级 + 牛酒酺五日 | 民众 / 吏员 / 整个社会 |

## 八位元老联名的政治意义

> 丞相陈平、太尉周勃、大将军陈武、御史大夫张苍、宗正刘郢、朱虚侯刘章、东牟侯刘兴居、典客刘揭皆再拜言曰……

**覆盖所有关键系别**：

| 元老 | 系别 | 象征 |
|------|------|------|
| 陈平 | 谋士元老 | 智计派 |
| 周勃 | 军功元老 | 军方派 |
| 陈武 | 大将军 | 现任军方 |
| 张苍 | 御史大夫 | 文官派 |
| 刘郢 | 宗正 | 宗室派 |
| 刘章、刘兴居 | 诛吕功臣 | 正统守护者 |
| 刘揭 | 夺吕禄印者 | 政变执行者 |

**"八人再拜"是完美的政治表态秀**：没有任何一股势力被排除

## 一日一夜的时序

```
第一日（渭桥）：
  → 天命（龟卜大横）已在代国完成
  → 功臣合法性（八人再拜）在长安代邸完成
  → 礼仪合法性（西乡让三，南乡让再）在代邸完成
  → 即天子位

即日夕：
  → 入未央宫
  → 宋昌为卫将军，张武为郎中令（心腹接管禁军）

当夜：
  → 法统合法性（有司分部诛伪皇子）
  → 德政合法性（下诏大赦、赐民爵、牛酒酺五日）
```

**为何必须在一天一夜内完成**：时间越长质疑越多→功臣集团可能反悔→外藩可能发难→民心需要稳定

## 四个关键决策

### 决策1：让还是不让？

不让→被视为贪位，失礼仪合法性；让太多→被视为不愿负责，失功臣合法性

**最佳解**：让五次（西三南二），恰到好处

### 决策2：即位夜诛伪皇子，还是第二天？

等到第二天→给少帝旧部串联时间

**最佳解**：当夜完成（"代王即日夕入未央宫"→"夜，有司分部诛灭……"）

### 决策3：任命何人执掌禁军？

继续用功臣→失去对军权的直接掌控；全盘换人→让功臣集团离心

**最佳解**：心腹（宋昌、张武）接管禁军指挥权 + 保留周勃、陈平的相位 = **军权归心腹，名位给功臣**

### 决策4：给功臣什么级别的封赏？

**三等分级**（区分贡献，避免任何人独大）：
- 周勃：益封万户，赐金五千斤（最高）
- 陈平、灌婴：各三千户，金二千斤
- 刘章、纪通、刘兴居：各二千户，金千斤
- 刘揭：封阳信侯，赐金千斤

## 文帝后续的反向操作（让功臣大失所望）

**第一天就立规矩**：

> 文帝曰："所言公言之，所言私王者无私。"

**含义**：从第一天就告诉周勃，皇帝不能被私下安排——"左右屏人"已经不灵了

**周勃的错误预判**：以为选了个"可控"的外藩皇帝，结果文帝比所有人都难操控

**教训**：政变者在选定"最平衡"的人选后，往往低估了对方的政治能力

---

## 失败案例对比

**唐昭宗（被朱温控制）**：无功臣合法性（朝中无可靠元老）+ 无礼仪合法性（被胁迫）→被废杀

**明建文帝（失位）**：缺军功合法性（削藩时无军事支持）+ 缺法统合法性（朱棣可打"清君侧"旗号）→失败

**结论**：任何一重合法性缺失都可能导致危机；文帝成功在于六重全覆盖

## 检查清单

**预备阶段**：
- [ ] 识别六类潜在质疑者（天命/宗法/功臣/礼仪/法统/德政）

- [ ] 有可信赖的核心团队能迅速接管关键部门
