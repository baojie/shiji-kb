# 孙武式军事纪律建立：三令五申七步法与令出必行完整操作手册

孙武以宫中美姬练兵，一日建立完整军事纪律体系。七步操作法：明确规则（前视心/左视左手/右视右手）→设置可见后果（乃设鈇钺）→三令五申排除借口→首次违规责己（约束不明将之罪）→二次违规严惩（斩队长二人以徇）→顶住上级压力（将在军君命有所不受）→验证效果（妇人皆中规矩绳墨无敢出声）。三核心原则：令出必行（无例外）、责任分层（首责己次责人）、将在军君命有所不受。结果：西破强楚入郢，北威齐晋。

**来源**：《史记·孙子吴起列传》（卷065）

> "约束不明，申令不熟，将之罪也；既已明而不如法者，吏士之罪也。"
> "臣既已受命为将，将在军，君命有所不受。"遂斩队长二人以徇。
> "妇人左右前后跪起皆中规矩绳墨，无敢出声。"

## 七步操作流程

```
步骤1：明确规则（约束既布）
  → 提问确认理解："汝知而心与左右手背乎？"
  → 具体说明动作：前则视心，左视左手，右视右手，后即视背
  → 获得确认："诺"
↓
步骤2：设置后果（乃设鈇钺）
  → 摆出刑具，让所有人看到
  → 让后果可见、可信
↓
步骤3：三令五申
  → 口头+演示+反复确认
  → 排除一切"不知道"的借口
↓
步骤4：首次违规——责己（"将之罪也"）
  → "约束不明，申令不熟，将之罪也"
  → 复三令五申（再次强调，最后机会）
↓
步骤5：二次违规——严惩（"吏士之罪也"）
  → "既已明而不如法者，吏士之罪也"
  → 斩队长二人以徇（杀鸡儆猴）
↓
步骤6：顶住上级压力
  → 吴王命"勿斩"→孙武："将在军，君命有所不受"
  → 遂斩队长二人以徇
↓
步骤7：验证效果
  → 用其次为队长
  → 复鼓之
  → 妇人皆中规矩绳墨，无敢出声
```

**核心逻辑**：规则不是威胁，是承诺；说了就必须做到；一次妥协，规则全废

## 三大核心原则

### 原则1：令出必行——无例外

**不是**：
- 反复强调却不执行
- 说一套做一套
- 有例外、看情况

**而是**：
- 说到必须做到
- 违令必罚
- 无论对象是谁（即使是吴王宠爱的姬妾）

**孙武的果决**：说"设鈇钺"就真的设了 / 说"斩队长"就真的斩了 / 不是吓唬人

### 原则2：责任分层——公正执法

**第一次失败**：将之罪也
- 约束不明→责任在领导
- 申令不熟→责任在领导
- 处理：复三令五申（给第二次机会）

**第二次失败**：吏士之罪也
- 既已明而不如法→责任在执行者
- 处理：斩队长以徇（无可推诿）

**精妙之处**：第一次责己，展现公正、给机会；第二次严惩，排除一切借口

### 原则3：将在军，君命有所不受

**核心**：既已受命为将，军中事务由将决定，必要时可拒绝君主命令

**适用条件**：
- 涉及军纪军法
- 君主命令会破坏规则
- 维护威权高于一切

**孙武的判断**：已受命为将（有权力）/ 军法重于王命（值得）/ 必须建立威权（必要）

## 七步详解

### 步骤1：明确规则（约束既布）

**要点**：
- 规则要具体明确（"前则视心"而不是"向前看"）
- 不能有歧义
- 必须确认对方理解（提问+等待回答）
- 可以反复强调

**检验标准**：每个人都能复述规则，且复述结果一致

### 步骤2：设置可见后果（乃设鈇钺）

**约束既布后，立即**：
- 摆出刑具/惩罚工具
- 让所有人看到违规的后果
- 制造心理压力

**现代应用**：明确违规后果（量化）/ 让后果可见、可信 / 不是威胁，而是承诺（说了就要做到）

### 步骤3：三令五申

**为什么反复强调**：
- 确保每个人都听到
- 排除"不知道"的借口
- 强化重要性

**如何反复强调**：
- 不同方式：口头+书面+演示
- 不同时间：多次重复
- 确认理解：提问+回答

### 步骤4：首次违规——责己

**孙武反应**：

> "约束不明，申令不熟，将之罪也。"复三令五申而鼓之左，妇人复大笑。

**原理**：
- 第一次失败可能确实是沟通问题
- 责己展现公正
- 给第二次机会——但也是最后机会

**行动**：复三令五申，清除一切借口

**注意**：第一次责己是策略，不是软弱；第二次绝不能再让步

### 步骤5：二次违规——严惩

**孙武逻辑链**：

> "约束不明，申令不熟，将之罪也；既已明而不如法者，吏士之罪也。"乃欲斩左右队长。

- 第一次：可能是沟通问题→责己
- 第二次：已经明确，还违规→故意
- 处理：斩队长以徇（杀鸡儆猴，震慑整体）

**关键点**：不能再找借口 / 必须严格执行 / 以儆效尤

### 步骤6：顶住压力

**孙武面对**：
- 吴王从台上观，见且斩爱姬，大骇
- 吴王命令："愿勿斩也"

**孙武坚持**：

> "臣既已受命为将，将在军，君命有所不受。"遂斩队长二人以徇。

**原理**：规则一旦定下，就不能破例 / 破一次例，规则就废了 / 必须顶住所有压力

**前提**：你有这个权力吗？后果你能承担吗？值得吗？→三问都要先回答

### 步骤7：验证效果

**孙武做法**：
- 用其次为队长（继续执行，不因斩杀而中断）
- 复鼓之（重新测试）

**结果**：
> "妇人左右前后跪起皆中规矩绳墨，无敢出声。"

**后续**：阖庐知孙子能用兵，卒以为将。西破强楚，入郢，北威齐晋。

## 三大决策点

### 决策点1：是否真的要执行？

**必须先问自己**：
- 你说的惩罚能执行吗？
- 如果不能执行，不要说
- 说了必须做到

**陷阱**：说了不做→威信扫地→比不立规则更糟

### 决策点2：首次失败如何处理？

| 选项 | 优点 | 风险 |
|------|------|------|
| 立即严惩 | 快速建立威信 | 可能确实是沟通问题，显得不公正 |
| 责己再给机会（孙武选择）| 展现公正、给机会 | 可能被认为软弱 |

**结论**：选责己，但只有一次——第二次绝不能再让步

### 决策点3：遭遇上级阻力怎么办？

**前提**：你有权力吗？值得吗？后果你能承担吗？

**孙武选择**：顶住——"将在军，君命有所不受"

**适用条件**：有授权 + 军纪重于一切 + 必须建立威权

**注意**：不是所有情况都适用；要先确认自己有此权力；要确认后果可控

---

## 三个"必须"与三个"不能"

### 三个"必须"

1. **必须明确**：规则清晰，无歧义
2. **必须执行**：说到做到，无例外
3. **必须公正**：先责己，再责人

### 三个"不能"

1. **不能模糊**：规则模糊=给违规借口
2. **不能妥协**：一旦妥协，规则就废
3. **不能不公**：区别对待=失去威信

## 风险与代价

### 风险1：执行不坚决

**表现**：说了不做 / 做了不彻底 / 有例外、看人情

**后果**：威信扫地 / 规则形同虚设 / 比不立规则更糟

### 风险2：过于严苛

**表现**：没有给机会（第一次就严惩）/ 惩罚过重 / 不分情况机械执行

**后果**：显得不公正 / 失去人心 / 引发反弹

### 风险3：选错战场

**孙武的明智**：选择"用妇人练兵"这个低风险场景，不是在真实战争中立威，成本可控

**启示**：立威要选对时机和场景 / 不要在关键时刻才想起立规矩 / 提前演练、测试

## 现代应用

### 场景1：新经理上任

**团队散漫，缺乏执行力**：
1. 明确规则（开会准时、deadline必守）
2. 三令五申（反复强调）
3. 第一次违规：公开提醒+责己（"可能我没说清楚"）
4. 第二次违规：严格执行后果（扣绩效、公开批评）
5. 顶住压力（即使是核心员工也不例外）

### 场景2：项目管理

**团队不重视流程规范**：
1. 制定清晰流程（类似"前视心、左视左手"）
2. 培训确认（三令五申）
3. 首次违规：流程问题→改进流程（责己）
4. 二次违规：个人问题→追究责任
5. 不因人废规（核心成员也不例外）

### 场景3：组织变革

**推行新制度遭遇阻力**：
1. 充分沟通新规则（约束既布）
2. 设置明确后果（设鈇钺）
3. 给过渡期（三令五申）
4. 严格执行（斩队长）
5. 顶住高层压力（将在军，君命有所不受）

### 孙子吴起列传与司马穰苴列传：两种军事立威——孙武的"三令五申"与穰苴的"诛庄贾"

孙武的"三令五申斩队长"和司马穰苴的"诛庄贾立威"是中国军事史上最著名的两次"立威"案例——孙子吴起列传中孙武的核心是"程序正义"（先教后诛），司马穰苴列传中穰苴的核心是"以法治军"（违令必诛）——两篇合在一起，两种军事立威模式在不同场景下的适用性和共同原则才完整呈现。孙子吴起列传中，孙武教吴王阖闾的宫中美姬练兵——他用了"七步法"建立纪律：先明确规则（"前则视心，左视左手，右视右手，后即视背"——动作标准具体到身体部位）、再展示后果（"乃设鈇钺"——摆出刑具）、再三令五申（反复确认每个人都理解规则）、首次违规时责己（"约束不明，申令不熟，将之罪也"——把第一次失败的责任归于自己）、二次违规时才严惩（"既已明而不如法者，吏士之罪也。乃欲斩左右队长"）、面对吴王求情时坚持（"将在军，君命有所不受"）、最终令行禁止（"妇人左右前后跪起皆中规矩绳墨，无敢出声"）（[[065_孙子吴起列传|065-?]]）。孙武的方法是"先教后诛"——给了充分的警告和机会，惩罚时让人无话可说。司马穰苴列传中，穰苴的立威方式更加直接——他刚被任命为将军就请求齐景公派一位宠臣做监军（"愿得君之宠臣，国之所尊，以监军"），庄贾被派来后穰苴与庄贾约定"日中会于军门"，庄贾因为是宠臣而迟到（"亲戚左右送之，留饮。日中而贾不至"），穰苴在庄贾到后说"将受命之日则忘其家，临军约束则忘其亲，援枹鼓之急则忘其身。今敌国深侵，邦内骚动……君寝不安席，食不甘味。百姓之命皆悬于君，何谓相送乎！"然后问军正"法何如？"军正回答"当斩"——庄贾被斩首示众，齐景公派使者持节来救庄贾，穰苴说"将在军，君令有所不受"——直接把使者的仆人、马车拆毁以震慑全军（[[064_司马穰苴列传|064-?]]）。合观两篇，孙武的"三令五申"（程序正义——给了充分的警告和机会，让惩罚看起来公正合理）和穰苴的"诛庄贾"（后果威慑——用宠臣的死来建立不容置疑的权威）代表了立威的两种模式。孙武面对的是"从未受过训练的宫人"（弱兵——需要耐心教导），所以走了"先教后诛"的程序；穰苴面对的是"傲慢的宠臣"（权贵——需要立即震慑），所以走了"直接斩杀"的路径。两人的共通原则是"将在军，君命有所不受"——孙武对吴王说了同样的话（拒绝赦免队长），穰苴对齐景公的使者说了同样的话（拒绝赦免庄贾）。两篇并读，读者看到的是中国古代军事纪律的一条铁律——"立威必须先斩有权势者"——孙武斩的是吴王最宠爱的两个姬妾（宫廷中最有身份的人），穰苴斩的是齐景公最宠信的大夫（朝廷中最有权势的人）。立威的对象不能是普通人（杀普通人没有震慑效应），必须是"谁都没想到会杀的人"（杀宠臣/爱姬让全军知道——"将军连大王的爱姬/宠臣都敢杀，何况我们？"）。

### 孙子吴起列传与商君列传：军事立威与法律立威——孙武的"信赏必罚"与商鞅的"徙木立信"

孙武的"三令五申斩队长"和商鞅的"徙木立信"是战国时期"信任建设"的两个经典案例——孙子吴起列传中孙武用军事手段建立"规则的可信度"，商君列传中商鞅用法律手段建立"制度的可信度"——两篇合在一起，"信"作为一切制度的基础这一核心命题在不同领域的应用才完整呈现。孙子吴起列传中，孙武建立的不是单纯的"纪律"，而是"规则的可信度"——宫女们一开始以为孙武只是开玩笑（像吴王一样只是在玩），所以"复大笑"。孙武通过"说设鈇钺就真的设了、说斩队长就真的斩了"的行动，让所有人意识到"规则不是威胁而是承诺"——说了就一定会做。孙武的核心原则是"令出必行"——不是用恐怖统治，而是用"说到做到"建立信任。孙武斩了吴王的两个爱姬后，吴王虽然不高兴（"阖庐知孙子能用兵，卒以为将"——吴王虽然心痛但还是知道孙武有用），但孙武并不需要讨好吴王——他需要的是军队对他的绝对信任（"兵既整齐，王可试下观之，虽赴水火犹可也"——士兵信任孙武到了"即使赴汤蹈火也愿意"的程度）。商君列传中，商鞅在推行变法前面临同样的"信任问题"——秦国百姓对政府的新法令没有信心（因为长期以来秦国政令多变、法不责贵，百姓不相信法律会严格执行）。商鞅的"徙木立信"——"令既具，未布，恐民之不信己，乃立三丈之木于国都市南门，募民有能徙置北门者予十金"（[[068_商君列传|068-?]]）。一开始百姓觉得奇怪（"民怪之，莫敢徙"——没人敢相信这么简单的事会给这么多钱），商鞅把赏金涨到五十金，有一个人试着做了，果然得到了五十金——"以明不欺"（以此来表明不欺骗百姓）。徙木立信与三令五申的本质是相同的——都是"用一次公开的、可验证的行动来证明规则会严格执行"。合观两篇，孙子吴起列传中的"三令五申斩队长"（孙武说"前视心"就一定要做到"前视心"——用一次斩队长的行动证明规则会严格执行）和商君列传中的"徙木立信"（商鞅说"徙木者予五十金"就一定要给五十金——用一次奖励行动证明承诺会兑现）是"信"在军事领域和法律领域的两种表现形式。孙武面对的是"不相信规则会被执行的人"（宫女们觉得反正吴王会赦免），所以用"斩队长"来证明；商鞅面对的是"不相信法令会被执行的人"（百姓觉得政府说话不算话），所以用"徙木立信"来证明。两人的目标一致——建立制度的"可信度"（"信"——让人相信规则会严格执行）。两篇并读，读者看到的是"信赏必罚"的普遍原理——"赏"（徙木立信——说赏就赏）和"罚"（三令五申——说罚就罚）是建立制度信任的两条腿。孙武用"罚"建立信任（斩队长——你敢违规我就会罚你），商鞅用"赏"建立信任（徙木给金——你做到了我就会赏你）。赏和罚都必须是"说到做到"的——如果赏不能兑现（移了木不给钱），法律就失去了奖励的可信度；如果罚不能执行（违规了不惩处），规则就失去了威慑的可信度。

### 孙子吴起列传与伍子胥列传：孙武与伍子胥在吴国的不同角色——"军事天才"与"政治复仇者"的两种命运

孙武和伍子胥都是吴国崛起的关键人物——孙子吴起列传中孙武是"军事纪律的建立者"（他的贡献在于制度——为吴国训练了一支强大的军队），伍子胥列传中伍子胥是"政治复仇的执行者"（他的贡献在于谋略——为吴国设计了一整套争霸方案）——两篇合在一起，吴国从弱到强的过程中"制度贡献"和"谋略贡献"两种角色的不同分工才完整呈现。孙子吴起列传中，孙武在吴国的角色是"军事工程的奠基人"——他不仅写了《孙子兵法》十三篇（"孙子武者，齐人也。以兵法见于吴王阖庐"），还在吴国亲自训练军队、建立军事制度。孙武的贡献在于"如何打仗"——他建立了吴军的纪律体系（三令五申）、制定了吴军的战略战术（"西破强楚，入郢，北威齐晋，显名诸侯"——吴国的军事辉煌背后是孙武的兵法）。有趣的是，孙子吴起列传中孙武在吴国破楚入郢后就从历史记载中消失了（"孙子与有力焉"——一句话带过了孙武在破楚中的作用，然后就没有然后了）——没有人知道孙武的最后结局。孙武可能是"功成身退"的典范（训练完军队、写完兵法后就隐退了），也可能是"善终"（在吴国安享晚年）。无论哪种，孙武的结局是"神秘而安全的"——他没有陷入吴国后来的权力斗争中。伍子胥列传中，伍子胥在吴国的角色完全不同——他是"政治复仇者"（从楚国逃到吴国，目的是借吴国的力量报父兄之仇）。伍子胥帮助阖闾刺杀吴王僚夺取王位（"伍胥知公子光有内志……乃进专诸于公子光"——推荐专诸刺杀王僚），推荐孙武给阖闾（"孙子与有力焉"——孙武的入仕也离不开伍子胥的推荐），策划了破楚入郢的战争（"伍子胥谋伐楚"）。但伍子胥的结局非常悲惨——夫差即位后，伍子胥因反对与越国议和被疏远，最终被夫差赐剑自杀（"乃使使赐伍子胥属镂之剑"）。伍子胥死前说"必树吾墓上以梓，令可以为器；而抉吾眼县吴东门之上，以观越寇之入灭吴也"（[[066_伍子胥列传|066-?]]）——死后九年，越国果然灭了吴国。合观两篇，孙子吴起列传中孙武的"神秘退场"（训练完军队、写完兵法后就消失了——他的命运没有被政治斗争沾染）和伍子胥列传中伍子胥的"惨烈结局"（被夫差赐死——他的一生被政治复仇和权力斗争吞噬）构成了吴国两位功臣的截然不同的命运。孙武之所以能善终（或至少安全隐退），是因为他只做"军事"不做"政治"——他训练军队、写兵法，但不参与吴国的权力斗争（不站队、不结党）。伍子胥之所以被赐死，是因为他深度参与了吴国的政治（他推荐了专诸、参与了政变、主导了对楚对越的政策），一旦政局变化（阖闾死→夫差立），他的政治立场就成了他的催命符。两篇并读，读者看到的是"专业人才vs政治人才"在历史中的不同命运——孙武是"专业人才"（只提供军事技术服务，不参与政治决策——所以安全），伍子胥是"政治人才"（深度参与政治决策，承担政治风险——所以危险）。孙武在吴国破楚入郢后消失，留下了一部《孙子兵法》和一堆军事遗产；伍子胥在吴国灭亡前被处死，留下了一首诗（"日暮途远，故倒行而逆施之"）和一段血泪史。两种选择，两种结局。
