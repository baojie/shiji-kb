# 孙武式军事纪律建立：三令五申七步法与令出必行完整操作手册

孙武以宫中美姬练兵，一日建立完整军事纪律体系。七步操作法：明确规则（前视心/左视左手/右视右手）→设置可见后果（乃设鈇钺）→三令五申排除借口→首次违规责己（约束不明将之罪）→二次违规严惩（斩队长二人以徇）→顶住上级压力（将在军君命有所不受）→验证效果（妇人皆中规矩绳墨无敢出声）。三核心原则：令出必行（无例外）、责任分层（首责己次责人）、将在军君命有所不受。结果：西破强楚入郢，北威齐晋。

**来源**：《史记·孙子吴起列传》（卷065）

> "约束不明，申令不熟，将之罪也；既已明而不如法者，吏士之罪也。"
> "臣既已受命为将，将在军，君命有所不受。"遂斩队长二人以徇。
> "妇人左右前后跪起皆中规矩绳墨，无敢出声。"

## 七步操作流程

```
步骤1：明确规则（约束既布）
  → 提问确认理解："汝知而心与左右手背乎？"
  → 具体说明动作：前则视心，左视左手，右视右手，后即视背
  → 获得确认："诺"
↓
步骤2：设置后果（乃设鈇钺）
  → 摆出刑具，让所有人看到
  → 让后果可见、可信
↓
步骤3：三令五申
  → 口头+演示+反复确认
  → 排除一切"不知道"的借口
↓
步骤4：首次违规——责己（"将之罪也"）
  → "约束不明，申令不熟，将之罪也"
  → 复三令五申（再次强调，最后机会）
↓
步骤5：二次违规——严惩（"吏士之罪也"）
  → "既已明而不如法者，吏士之罪也"
  → 斩队长二人以徇（杀鸡儆猴）
↓
步骤6：顶住上级压力
  → 吴王命"勿斩"→孙武："将在军，君命有所不受"
  → 遂斩队长二人以徇
↓
步骤7：验证效果
  → 用其次为队长
  → 复鼓之
  → 妇人皆中规矩绳墨，无敢出声
```

**核心逻辑**：规则不是威胁，是承诺；说了就必须做到；一次妥协，规则全废

---

## 三大核心原则

### 原则1：令出必行——无例外

**不是**：
- 反复强调却不执行
- 说一套做一套
- 有例外、看情况

**而是**：
- 说到必须做到
- 违令必罚
- 无论对象是谁（即使是吴王宠爱的姬妾）

**孙武的果决**：说"设鈇钺"就真的设了 / 说"斩队长"就真的斩了 / 不是吓唬人

### 原则2：责任分层——公正执法

**第一次失败**：将之罪也
- 约束不明→责任在领导
- 申令不熟→责任在领导
- 处理：复三令五申（给第二次机会）

**第二次失败**：吏士之罪也
- 既已明而不如法→责任在执行者
- 处理：斩队长以徇（无可推诿）

**精妙之处**：第一次责己，展现公正、给机会；第二次严惩，排除一切借口

### 原则3：将在军，君命有所不受

**核心**：既已受命为将，军中事务由将决定，必要时可拒绝君主命令

**适用条件**：
- 涉及军纪军法
- 君主命令会破坏规则
- 维护威权高于一切

**孙武的判断**：已受命为将（有权力）/ 军法重于王命（值得）/ 必须建立威权（必要）

## 七步详解

### 步骤1：明确规则（约束既布）

**要点**：
- 规则要具体明确（"前则视心"而不是"向前看"）
- 不能有歧义
- 必须确认对方理解（提问+等待回答）
- 可以反复强调

**检验标准**：每个人都能复述规则，且复述结果一致

### 步骤2：设置可见后果（乃设鈇钺）

**约束既布后，立即**：
- 摆出刑具/惩罚工具
- 让所有人看到违规的后果
- 制造心理压力

**现代应用**：明确违规后果（量化）/ 让后果可见、可信 / 不是威胁，而是承诺（说了就要做到）

### 步骤3：三令五申

**为什么反复强调**：
- 确保每个人都听到
- 排除"不知道"的借口
- 强化重要性

**如何反复强调**：
- 不同方式：口头+书面+演示
- 不同时间：多次重复
- 确认理解：提问+回答

### 步骤4：首次违规——责己

**孙武反应**：

> "约束不明，申令不熟，将之罪也。"复三令五申而鼓之左，妇人复大笑。

**原理**：
- 第一次失败可能确实是沟通问题
- 责己展现公正
- 给第二次机会——但也是最后机会

**行动**：复三令五申，清除一切借口

**注意**：第一次责己是策略，不是软弱；第二次绝不能再让步

### 步骤5：二次违规——严惩

**孙武逻辑链**：

> "约束不明，申令不熟，将之罪也；既已明而不如法者，吏士之罪也。"乃欲斩左右队长。

- 第一次：可能是沟通问题→责己
- 第二次：已经明确，还违规→故意
- 处理：斩队长以徇（杀鸡儆猴，震慑整体）

**关键点**：不能再找借口 / 必须严格执行 / 以儆效尤

### 步骤6：顶住压力

**孙武面对**：
- 吴王从台上观，见且斩爱姬，大骇
- 吴王命令："愿勿斩也"

**孙武坚持**：

> "臣既已受命为将，将在军，君命有所不受。"遂斩队长二人以徇。

**原理**：规则一旦定下，就不能破例 / 破一次例，规则就废了 / 必须顶住所有压力

**前提**：你有这个权力吗？后果你能承担吗？值得吗？→三问都要先回答

### 步骤7：验证效果

**孙武做法**：
- 用其次为队长（继续执行，不因斩杀而中断）
- 复鼓之（重新测试）

**结果**：
> "妇人左右前后跪起皆中规矩绳墨，无敢出声。"

**后续**：阖庐知孙子能用兵，卒以为将。西破强楚，入郢，北威齐晋。

## 三大决策点

### 决策点1：是否真的要执行？

**必须先问自己**：
- 你说的惩罚能执行吗？
- 如果不能执行，不要说
- 说了必须做到

**陷阱**：说了不做→威信扫地→比不立规则更糟

### 决策点2：首次失败如何处理？

| 选项 | 优点 | 风险 |
|------|------|------|
| 立即严惩 | 快速建立威信 | 可能确实是沟通问题，显得不公正 |
| 责己再给机会（孙武选择）| 展现公正、给机会 | 可能被认为软弱 |

**结论**：选责己，但只有一次——第二次绝不能再让步

### 决策点3：遭遇上级阻力怎么办？

**前提**：你有权力吗？值得吗？后果你能承担吗？

**孙武选择**：顶住——"将在军，君命有所不受"

**适用条件**：有授权 + 军纪重于一切 + 必须建立威权

**注意**：不是所有情况都适用；要先确认自己有此权力；要确认后果可控

---

## 三个"必须"与三个"不能"

### 三个"必须"

1. **必须明确**：规则清晰，无歧义
2. **必须执行**：说到做到，无例外
3. **必须公正**：先责己，再责人

### 三个"不能"

1. **不能模糊**：规则模糊=给违规借口
2. **不能妥协**：一旦妥协，规则就废
3. **不能不公**：区别对待=失去威信

## 风险与代价

### 风险1：执行不坚决

**表现**：说了不做 / 做了不彻底 / 有例外、看人情

**后果**：威信扫地 / 规则形同虚设 / 比不立规则更糟

### 风险2：过于严苛

**表现**：没有给机会（第一次就严惩）/ 惩罚过重 / 不分情况机械执行

**后果**：显得不公正 / 失去人心 / 引发反弹

### 风险3：选错战场

**孙武的明智**：选择"用妇人练兵"这个低风险场景，不是在真实战争中立威，成本可控

**启示**：立威要选对时机和场景 / 不要在关键时刻才想起立规矩 / 提前演练、测试

## 现代应用

### 场景1：新经理上任

**团队散漫，缺乏执行力**：
1. 明确规则（开会准时、deadline必守）
2. 三令五申（反复强调）
3. 第一次违规：公开提醒+责己（"可能我没说清楚"）
4. 第二次违规：严格执行后果（扣绩效、公开批评）
5. 顶住压力（即使是核心员工也不例外）

### 场景2：项目管理

**团队不重视流程规范**：
1. 制定清晰流程（类似"前视心、左视左手"）
2. 培训确认（三令五申）
3. 首次违规：流程问题→改进流程（责己）
4. 二次违规：个人问题→追究责任
5. 不因人废规（核心成员也不例外）

### 场景3：组织变革

**推行新制度遭遇阻力**：
1. 充分沟通新规则（约束既布）
2. 设置明确后果（设鈇钺）
3. 给过渡期（三令五申）
4. 严格执行（斩队长）
5. 顶住高层压力（将在军，君命有所不受）
