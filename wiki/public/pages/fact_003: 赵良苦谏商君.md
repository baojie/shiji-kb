# fact_003: [[赵良苦谏商君]]

**类型**: 叙事性事实  
**主题**: 赵良对比五羖大夫与商君，预言商君之危  
**出处**: 《史记·商君列传》段落 [9]

---

## 背景

商君相秦十年，宗室贵戚多怨望者。赵良见商君，商君请与赵良交友。

---

## 对话过程

### 第一轮：赵良拒绝

**商君请求**：
- "鞅之得见也，从孟兰皋，今鞅请得交，可乎？"

**赵良回应**：
- "仆弗敢原也"
- 引孔子："推贤而戴者进，聚不肖而王者退"
- 自谦："仆不肖，故不敢受命"
- 担忧："非其位而居之曰贪位，非其名而有之曰贪名"

### 第二轮：商君追问

**商君问**：
- "子不说吾治秦与？"

**赵良回应**：
- 引虞舜："反听之谓聪，内视之谓明，自胜之谓彊。自卑也尚矣"
- 建议："君不若道虞舜之道，无为问仆矣"

### 第三轮：商君自夸

**商君自比**：
- "始秦戎翟之教，父子无别，同室而居"
- "今我更制其教，而为其男女之别，大筑冀阙，营如鲁卫矣"
- "子观我治秦也，孰与五羖大夫贤？"

**赵良反驳**：
- "千羊之皮，不如一狐之掖；千人之诺诺，不如一士之谔谔"
- "武王谔谔以昌，殷纣墨墨以亡"
- 请求："君若不非武王乎，则仆请终日正言而无诛，可乎？"

**商君同意**：
- "语有之矣，貌言华也，至言实也，苦言药也，甘言疾也"
- "夫子果肯终日正言，鞅之药也。鞅将事子，子又何辞焉"

## 五羖大夫与商君对比

### 五羖大夫（百里奚）

| 维度 | 表现 |
|------|------|
| **入仕方式** | 荆之鄙人，闻[[秦缪公]]之贤而原望见，行而无资，自粥於秦客，被褐食牛 |
| **晋升方式** | 期年，缪公知之，举之牛口之下，而加之百姓之上，秦国莫敢望焉 |
| **治国成就** | 相秦六七年，东伐郑，三置晋国之君，一救荆国之祸 |
| **影响力** | 发教封内，而巴人致贡；施德诸侯，而八戎来服。由余闻之，款关请见 |
| **个人作风** | 劳不坐乘，暑不张盖，行於国中，不从车乘，不操干戈 |
| **身后影响** | 功名藏於府库，德行施於后世 |
| **民众反应** | 五羖大夫死，秦国男女流涕，童子不歌谣，舂者不相杵 |

| 维度 | 表现 | 赵良批评 |
|------|------|----------|
| **入仕方式** | 因嬖人景监以求见秦王 | "非所以为名也" |
| **施政重点** | 不以百姓为事，而大筑冀阙 | "非所以为功也" |
| **执法方式** | 刑黥太子之师傅，残伤民以骏刑 | "是积怨畜祸也" |

| **个人作风** | 南面而称寡人，日绳秦之贵公子 | 引《诗》："相鼠有体，人而无礼，人而无礼，何不遄死" → "非所以为寿也" |
| **得人之道** | [[公子虔]]杜门不出已八年矣，君又杀祝懽而黥[[公孙贾]] | 引《诗》："得人者兴，失人者崩" → "非所以得人也" |
| **出行排场** | 后车十数，从车载甲，多力而骈胁者为骖乘，持矛而操闟戟者旁车而趋 | 引《书》："恃德者昌，恃力者亡" → "君之危若朝露" |

## 赵良的预言与建议

### 预言

**危险程度**：
- "君之危若朝露"
- "秦王一旦捐宾客而不立朝，秦国之所以收君者，岂其微哉？亡可翘足而待"

### 建议

**退出方案**：
1. 归十五都
2. 灌园於鄙
3. 劝秦王显岩穴之士
4. 养老存孤
5. 敬父兄
6. 序有功
7. 尊有德

**目的**：
- "可以少安"

### 商君反应

- "商君弗从"

---

## 预言应验

**时间**：后五月

**事件**：
1. [[秦孝公]]卒，太子立
2. 公子虔之徒告商君欲反
3. 发吏捕商君
4. 商君逃亡，[[作法自毙]]
5. 被魏国拒绝
6. 返秦起兵
7. 被杀於郑黾池
8. 车裂示众
9. 灭族

**应验对照**：
- "秦王一旦捐宾客而不立朝" → 孝公死，太子立
- "秦国之所以收君者，岂其微哉" → 车裂以徇，灭其家
- "亡可翘足而待" → 后五月即死

---

## 核心论点

### 赵良的核心观点

1. **入仕方式影响声名**：通过正道举荐 vs 依靠宠臣
2. **施政重点决定功效**：以民为本 vs 大兴土木
3. **执法方式影响民心**：教化为主 vs 严刑峻法
4. **个人作风决定寿命**：谦逊低调 vs 张扬霸道
5. **得人之道决定兴衰**：施德 vs 积怨

### 历史意义

赵良的谏言精准预测了商君的结局，体现了：
- 改革者需要考虑政治安全
- 严刑峻法积累怨恨的危险
- 依附权力者的脆弱性
- 德治与法治的平衡问题
