# 功高震主四策：张良萧何韩信曹参的生存决策树

当你的功劳、能力、影响力达到可能威胁上级时，如何自保？基于汉初四大功臣的不同选择，提炼"功高震主"的四种生存策略——功成身退（张良）、自污保身（萧何）、彻底自立（韩信应选而未选）、守成低调（曹参）——及其适用条件与操作方法。

**来源**：《史记·高祖功臣侯者年表》（卷018）及《萧相国世家》《留侯世家》《淮阴侯列传》《曹相国世家》

> "汉六年正月，封功臣。良曰：臣愿封留足矣，不敢当三万户。"（张良让封）

---

## 为何会"功高震主"？

**掌权者心理**：`我的权力 = 100% - 下属的威胁度`，下属越强，掌权者越不安。

**威胁三要素**：
| 要素 | 说明 | 案例 |
|------|------|------|
| 功劳显著 | 功劳太大，无法忽视 | 韩信：定魏赵齐，天下莫敢与当 |
| 实力强大 | 掌握兵权、财权、人脉 | 韩信：手握重兵 |
| 不可替代 | 掌权者离不开你 | 萧何：守关中，给军食 |

**危险指数公式**：
```
威胁度 = 功劳显著度 × 实力强度 × 不可替代性

韩信: 10×10×8 = 800（极度危险，被诛）
萧何: 10×6×10 = 600（很危险，需自污）
张良: 8×3×7 = 168（中等，功成身退）
曹参: 9×5×5 = 225（较低，萧规曹随）
```

## 四种生存策略

### 策略A：功成身退（最安全）——代表：张良

**核心操作**：
> "良曰：始臣起下邳，与上会留，此天以臣授陛下。臣愿封留足矣，不敢当三万户。"
> "留侯乃称曰病，不朝。从赤松子游。后八年卒，谥为文成侯。"

**四步操作**：
1. **主动让封**：三万户→仅封留（远低于贡献）
2. **称病不朝**：退出权力中心，不参与决策
3. **学黄老**：辟谷，轻身，表示无意政治
4. **善终**：八年后自然去世

**成功要素**：功劳主要是智谋（六奇计），不掌兵权；主动退让，让皇帝感激；退得彻底，学道远离政治

**适用条件**：已获得足够财富/地位；不掌握关键资源（兵权/财权）；有退路（学习、投资、享受生活）

### 策略B：自污保身（主动示弱）——代表：萧何

**萧何三次自污**：

**第1次让封示谦**：被推为第一时，多次推让，最终才被迫接受

**第2次买田自污**：
> "何乃令家人贱强买民田宅数千万。上闻之，怒曰：相国乃利民！何顿首曰：臣故为此，以自污，恐陛下奇臣。"

**第3次请苑被系**：替民请命废上林苑→皇帝怀疑收贿赂→被关进监狱→出狱后威胁降低

**深层逻辑**：皇帝怕臣子完美无缺（德才兼备+民望极高=造反可能）；主动暴露缺点（贪财），让皇帝放心（此人有弱点，不会造反）

**自污度控制**：30%-50%（太少无效，太多危险）；选择可控小错（贪财 ✅；谋反 ❌）；偶尔为之，不能过频

**适用条件**：掌握关键资源，不可替代；老板已经开始猜忌；你必须继续在位（不能退休）

### 策略C：彻底自立（最危险但可能最安全）——代表：韩信应选而未选

**蒯通劝韩信**：
> "今足下戴震主之威，挟不赏之功……莫若两利而俱存之，参分天下，鼎足而居，莫敢先动。"

**韩信汉四年时的实力**：军事（定魏赵齐，手握重兵✅）；地盘（齐国70余城✅）；时机（楚汉相持✅）

**韩信的实际选择**："汉王遇我甚厚，吾岂可以乡利倍义乎！"——拒绝自立

**结局**：被徙为楚王→降为淮阴侯→被诛

**韩信的教训**：如果不敢自立，就必须选策略A（功成身退）或策略B（自污保身）。**最差选择**：既不敢自立，又不甘心退让，最终"进退失据"被诛。

**适用条件**：老板已明显猜忌；你有能力自立；你已掌握足够资源（客户、技术、团队）

### 策略D：守成低调（适用于非顶尖功臣）——代表：曹参

**操作**：
> "参代何为汉相国，举事无所变更，一遵萧何约束。"
> "陛下观臣能孰与萧何贤？……今陛下垂拱，参等守职，遵而勿失，不亦可乎？"

**三步操作**：
1. **继承前任政策**："萧规曹随"，不创新
2. **无为而治**：日饮酒，不管事
3. **自知之明**：承认自己不如前任（萧何）

**适用条件**：你不是最顶尖人才；接替强势前任；组织已进入稳定期

## 失败案例分析

### 韩信（进退失据，被诛）

**错误1**：汉四年蒯通劝自立，韩信拒绝——错失最佳时机，此时韩信实力最强

**错误2**：被徙为楚王不甘心，被降为淮阴侯心怀不满——既然不敢自立，就应选A或B

**错误3**：淮阴侯时与陈豨谋反——此时实力已弱，谋反无胜算，纯属"送死"

**教训**：要么汉四年自立，要么徙楚王后功成身退；最差选择是两者都不做

### 彭越（谋反被诛，醢）

**错误**：梁王掌握兵权，功高震主，不知韬光养晦

**教训**：掌兵权者，必须选策略A（退）或策略B（自污），不可坐等被诛

## 核心原则

**原则1：识别时机，果断决策**——功高震主的最佳应对时机 = 老板刚开始猜忌时，不要等被边缘化才行动

**原则2：主动示弱，降低威胁**——"完美无缺" = 最大威胁；暴露可控缺点，让老板放心

**原则3：进退有据，不可两端**——要么彻底自立（策略C），要么彻底退让（策略A/B）；绝不可"进退失据"

**原则4：功成身退，善始善终**——功劳到顶点时，就是退出时机；不恋权，不贪位

## 检查清单

**你是否功高震主？**（3项以上为"是"→需选择策略）
- [ ] 我的功劳是否远超同侪？
- [ ] 我是否掌握关键资源（兵权/财权/客户/技术）？
- [ ] 老板是否开始疏远我？
- [ ] 老板是否旁敲侧击质疑我？
- [ ] 我是否不可替代？

**选择策略：**
- [ ] 我有退路吗？（财富/兴趣/家庭）→ 是→策略A
- [ ] 我不能退，但可自污吗？→ 是→策略B
- [ ] 我已被逼无路，有实力自立吗？→ 是→策略C
- [ ] 我非顶尖，接任强者？→ 是→策略D
- [ ] 以上都不是？→ ⛔ 危险！重新评估，避免"进退失据"
