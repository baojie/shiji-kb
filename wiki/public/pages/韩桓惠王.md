# 韩桓惠王

战国[[韩国]]君主，[[韩釐王]]之子，在位三十四年（约前272—前239年）。在位期间正值[[秦国]]全面扩张，韩国疆土屡遭蚕食：失陉、上党郡守降赵（直接引发[[长平之战]]）、失阳城负黍、城皋荥阳，乃至上党全境覆灭。卒后子[[韩安]]继位，韩安五年韩国被秦所灭。《史记》出现4处。

---

## 生平

### 继位，元年伐燕

韩釐王卒，华阳之战（华阳之战即魏赵联军被秦军击败的那场大战）之后，其子桓惠王继位：

> 〔韩世家〕是岁，韩釐王卒，子[[韩桓惠王]]立。（[[045_韩世家|045-30]]）

桓惠王元年，主动发兵伐燕：

> 〔韩世家〕桓惠王元年，伐燕。（[[045_韩世家|045-31]]）

### 秦国蚕食，韩地持续丧失

韩桓惠王在位的三十四年，是韩国全面衰落的时期，各年大事几乎均为秦国拔城、韩国割地：

| 年份 | 事件 |
| --- | --- |
| 九年 | 秦拔陉，城汾旁 |
| 十年 | 秦击韩于太行，上党郡守以上党郡降赵 |
| 十四年 | 秦拔赵上党，杀马服子（赵括）卒四十余万于长平 |
| 十七年 | 秦拔韩阳城、负黍 |
| 二十二年 | 秦昭王卒 |
| 二十四年 | 秦拔城皋、荥阳 |
| 二十六年 | 秦悉拔上党 |
| 二十九年 | 秦拔韩十三城 |

其中第十年的关键事件尤为重要：

> 〔韩世家〕十年，秦击我於太行，我上党郡守以上党郡降赵。十四年，秦拔赵上党，杀马服子卒四十馀万於长平。（[[045_韩世家|045-31]]）

上党郡守冯亭不愿降秦，将上党十七城转献赵国，由此引发秦赵之间惨烈的[[长平之战]]，赵括（马服子）战死，赵军被坑杀四十余万。这一事件的起点，正是桓惠王在位时韩国的地方官员为保全土地而做出的惊人抉择。

### 卒，子韩安立

桓惠王在位三十四年后卒：

> 〔韩世家〕三十四年，桓惠王卒，子[[韩安]]立。（[[045_韩世家|045-32]]）

[[韩安]]继位仅五年，秦国攻韩，韩安遣[[韩非]]出使，韩非被秦留置并杀，韩国随即亡于秦。

## 历史评价

韩桓惠王在位期间，韩国已无力抵御秦国扩张，所记大事无非割让、失地、守军降敌。《韩世家》对桓惠王的记录极为简略，没有任何外交功绩或政治举措，只是一年年地记录失地——这种"流水账式"的叙事本身，正是韩国末期历史的真实写照。
