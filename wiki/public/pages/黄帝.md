# 黄帝

[[黄帝]]（公孙轩辕，传说约前 2717 — 前 2599），**华夏文明的人文始祖**，五帝之首。
《史记》开篇第一帝，司马迁专门破例（始皇之前不以帝名），以"**五帝本纪第一**"为全书开端——
**黄帝既是历史起点，也是一切皇权合法性的终极溯源**。汉武帝封禅求仙多次以黄帝为标杆，
《史记》中半数引用出自[[012_孝武本纪|孝武本纪]]和[[028_封禅书|封禅书]]。

《史记》主叙见[[001_五帝本纪|001 五帝本纪]]（92 次，分布 24 篇）。

## 生平（依《五帝本纪》）

### 兴起与征讨

黄帝者，少典之子，姓公孙，名**轩辕**。生而神灵，幼而徇齐，长而聪明，成而聪明。

时**神农氏**衰，诸侯相侵伐。黄帝习用干戈，以征不享；诸侯咸来宾从。

**炎黄战阪泉**：与炎帝[[神农氏]]战于**阪泉**之野，三战得志（五帝本纪 001-003）。

**逐蚩尤涿鹿**：[[蚩尤]]作乱，黄帝乃征师诸侯，**与蚩尤战于涿鹿之野**，遂擒杀蚩尤。
（"**涿鹿之战**"，华夏神话最早的决定性战役。）

> *"诸侯咸尊轩辕为天子，代神农氏，是为黄帝。天下有不顺者，黄帝从而征之，平者去之。"*

### 文明创制

《五帝本纪》称黄帝"**迁徙往来无常处，以师兵为营卫，官名皆以云命，为云师**"——
游牧型政治形态。同时传说中黄帝开创：

| 发明/制度 | 传说之臣 |
| --- | --- |
| 文字 | [[仓颉]] |
| 历法 | [[大挠]] |
| 音律 | [[伶伦]] |
| 医学 | [[岐伯]]（黄帝问医，后世"岐黄之术"） |
| 舟车 | 传说中造车 |
| 蚕丝织造 | [[嫘祖]]（正妃） |

> *"黄帝顺天地之纪，幸四方，鬼神山川封禅，举风后、力牧、常先、大鸿以治民。"*

### 寿命与崩

黄帝践位百年而崩（传说寿 110 岁），葬于[[桥山]]（今陕西黄陵）——
**今黄帝陵**是华夏祭祖第一地。

## 汉武帝与黄帝

《史记》中黄帝的大量引用集中于汉武帝时代——因武帝热衷封禅、求仙，**以黄帝为"升天成仙"之先例**：

> *"黄帝铸鼎荆山下。……鼎既成，有龙垂胡**髯下迎黄帝。黄帝上骑，群臣后宫从上者七十余人，
> 龙乃上去**。"*（《封禅书》028-020）

方士们以黄帝"骑龙升天"劝武帝封禅、求不死——太史公暗含讽意。

## 谱系（五帝世系）

```
黄帝
├── 玄嚣 (青阳)
│   └── 蟜极 → 帝喾 → 尧
└── 昌意
    └── 颛顼 → (侧支) → 舜
```

《三代世表》整理了从黄帝到商周的完整谱系，为后世"**万世一系**"思想之祖。

## 典故与成语

- **涿鹿之战** — 黄帝擒蚩尤，华夏建立
- **阪泉之战** — 炎黄合并之役
- **岐黄之术** — 黄帝与岐伯问对，中医理论渊源
- **黄帝骑龙升天** — 汉武帝时封禅仙话的核心意象
- **轩辕** — 黄帝别称，车制发明的象征
