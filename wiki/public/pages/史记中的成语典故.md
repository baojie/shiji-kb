# 史记中的成语典故——100个来源于《史记》的汉语词汇

## 核心内容

《史记》是汉语成语和典故的最重要来源之一，数百个至今仍在使用的成语直接来源于《史记》中的故事。以下按主题分类整理最常用的成语典故：

---

### 一、项羽与楚汉相争

| 成语/典故 | 来源篇目 | 含义 |
|-----------|---------|------|
| **破釜沉舟** | 项羽本纪 | 下定决心，断绝退路，比喻意志坚决 |
| **作壁上观** | 项羽本纪 | 袖手旁观，不肯出力（诸侯"皆从壁上观"） |
| **沐猴而冠** | 项羽本纪 | 比喻外表像人而实为粗鄙，讽刺项羽 |
| **霸王别姬** | 项羽本纪 | 项羽与虞姬诀别，泛指英雄末路 |
| **四面楚歌** | 项羽本纪 | 被围困、孤立无援的绝境 |
| **楚河汉界** | 项羽本纪 | 楚汉对峙，泛指对立双方的分界 |
| **[[约法三章]]** | 高祖本纪 | 刘邦入关后与秦民约定的简单法规，泛指简明规定 |
| **运筹帷幄** | 高祖本纪 | 张良在帷帐中运筹谋划 |
| **高屋建瓴** | 高祖本纪 | 从高处往下倒水，比喻势不可挡 |

|-----------|---------|------|

| **负荆请罪** | 廉颇蔺相如列传 | 主动承认错误，诚心悔过 |
| **刎颈之交** | 廉颇蔺相如列传 | 可以同生共死的深厚友谊 |
| **纸上谈兵** | 廉颇蔺相如列传（赵括） | 只会理论，不能实际操作 |
| **[[毛遂自荐]]** | 平原君虞卿列传 | 自我推荐，主动请求担任职务 |
| **脱颖而出** | 平原君虞卿列传 | 比喻才能突出地显露出来 |
| **三人成虎** | 张仪列传 | 谣言重复多次会被误信为真 |
| **[[卧薪尝胆]]** | 越王勾践世家 | 刻苦忍耐，立志报仇雪耻 |

| **[[奇货可居]]** | 吕不韦列传 | 把稀缺的东西储存起来等待时机获利 |

### 四、人生哲理与处世

| 成语/典故 | 来源篇目 | 含义 |
|-----------|---------|------|
| **人固有一死，或重于泰山，或轻于鸿毛** | 报任安书 | 死亡的价值在于意义 |
| **士为知己者死** | 刺客列传（豫让） | 为了解自己价值的人而牺牲 |
| **燕雀安知鸿鹄之志哉** | 陈涉世家 | 眼界狭小的人无法理解有远大志向的人 |
| **王侯将相宁有种乎** | 陈涉世家 | 英雄不在乎出身，任何人都可以成就大事 |
| **大丈夫当如此也** | 高祖本纪（刘邦看秦始皇）| 表达对强者的羡慕和效仿之心 |
| **天下苦秦久矣** | 陈涉世家 | 长期的压迫积累了足够的反抗力量 |
| **明修栈道，暗渡陈仓** | 留侯世家 | 表面上做一件事，暗中却做另一件事 |
| **日暮途远，倒行逆施** | 伍子胥列传 | 时间紧迫，只能采取非常规手段 |
