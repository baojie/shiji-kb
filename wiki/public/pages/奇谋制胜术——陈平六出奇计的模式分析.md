## 技能概述

### 核心问题

**当正面硬打无胜算、道德牌又打不出时，如何用"奇计"以最小代价破局？**

陈平"六出奇计"提供了完整方法论：**诊断对手的结构性弱点、设计低成本高杠杆的操作、用一个动作引爆连锁反应**。这不是军事计谋，而是**政治—心理—外交的综合博弈**。

### 历史来源

本技能从陈平六出奇计中提炼：
1. **反间计破楚**（[8][9]）——一顿饭破坏项羽与范增的信任
2. **荥阳夜出**（[9]）——二千女子诱敌换刘邦脱困
3. **立韩信为齐王**（[10]）——蹑足止怒，留韩信于汉
4. **诈游云梦擒韩信**（[11]）——游猎之名擒楚王
5. **贿匈奴阏氏**（[12]）——借敌方内部关键人物脱围
6. **诈伏槛车载樊哙**（[15]）——不执行错误命令

共同特征：**用一个小动作**（换菜、一只脚、一顿饭、一辆槛车）**撬动一个大结果**（范增死、韩信稳、皇帝脱险、樊哙保命）。

## 方法步骤

### 步骤1：诊断对手的结构性弱点

**目标**：找到对手"看似强大但暗藏必然裂痕"的那一点

**操作**：
1. **列出对方的公开强项**——项羽恭敬爱人、匈奴武力强、韩信军功无双
2. **列出对方的公开短板**——项羽吝啬封赏、匈奴治理能力弱、韩信缺乏政治资源
3. **寻找"强项与短板的接缝处"**——这就是破绽：
   - 项羽"恭敬爱人"+"吝啬封赏"→骨鲠之臣"不过数人"→**"怀疑骨鲠之臣"是必然心理**
   - 匈奴武力强+治理能力弱→**"得汉地也管不了"的逻辑成立**
   - 韩信军功无双+政治依赖刘邦→**"不给齐王位就可能反"的威胁成立**
4. **验证这个破绽是"必然"还是"偶然"**——偶然破绽不能成为奇计基础

**注意事项**：
- 不要被对方的表象迷惑——"恭敬爱人"的项羽恰恰最容易怀疑亲信
- 破绽必须是"结构性"——偶然破绽次日就可能被修复
- 破绽找不到时不要硬上——这种情况下奇计反而失效

### 步骤2：设计最低成本的引爆动作

**目标**：找到一个"动作成本极低、后果极大"的操作点

**操作**：
1. **列出"理论上可行的方案"**——比如"让项羽怀疑范增"有多种做法
2. **评估每种方案的成本**：
   - 方案A：给范增写信（成本：伪造信件，风险：被识破）
   - 方案B：在楚军散布谣言（成本：黄金，风险：谣言被辨伪）
   - 方案C：**在楚使来时换菜**（成本：一顿饭，风险：极小）
3. **选择"杠杆比最大"的方案**——最小投入、最大产出
4. **设计"动作完成即触发连锁"的机制**——
   - 换菜→楚使报告→项羽疑亚父→亚父辞归→死
   - 一顿饭直接导致项羽失去最重要谋士——**杠杆率惊人**

**注意事项**：
- 不要选"表面上很猛"的方案——通常实际杠杆低
- "看起来不起眼的动作"往往杠杆最大——这违反直觉
- 必须想清楚引爆后的所有连锁——不要让第一步就没人接

### 步骤3：找到"关键非名义决策者"

**目标**：在对方内部找到"能影响决策但名义上不负责"的人

**操作**：
1. **画出对方决策链**——CEO、高管、配偶、助理、顾问、情人……
2. **区分"名义决策者"和"实际影响者"**——这两者常常不同
3. **选择影响者中"利益可重构"的那一个**：
   - 阏氏（单于夫人）——不是冒顿的决策者，但能私下劝说
   - 审食其（左丞相）——名义是相，实际听命吕后
   - 吕嬃（吕后妹）——名义无职位，实际谗言有效
4. **设计针对这一影响者的利益话术**——不是"让对方做什么"，而是"对方做什么符合其自身利益"

**注意事项**：
- 不要直接找"名义决策者"——那是正面对抗
- 影响者要有"能被利益驱动的柔性"——铁板一块的人不行
- 利益话术必须符合影响者的真实利益——不是表面好处

### 步骤4：一个动作完成后立即撤出

**目标**：不让对方看出是谁在操纵

**操作**：
1. **执行核心动作后立即断线**——不留连接痕迹
2. **不去"验证效果"**——你去验证就等于留下证据
3. **把功劳分给别人**——比如韩信封齐王时陈平"让张良册封"——自己不出面
4. **对外保持"与此事无关"的姿态**——事后也不声张
5. **只享受结果**——不宣扬过程

**注意事项**：
- 很多人计谋成功后爱"复盘炫耀"——这是灾难
- 陈平"六出奇计"至死不谈具体做法——"其计秘，世莫得闻"
- **不炫耀**不仅是低调——是保留下次使用的可能

### 步骤5：建立"奇计有报"的长期预期

**目标**：让主公知道"给你资源就能收获奇效"——形成正反馈

**操作**：
1. **每次奇计必须对主公利益有直接贡献**——破楚、擒韩、脱围、保命
2. **主公给资源时不讨价还价**——刘邦"出黄金四万斤，恣所为，不问其出入"——陈平立刻动作
3. **事后让主公看到"你的判断比我准"**——比如樊哙事件——陈平没斩樊哙是对的（后来高帝崩）
4. **长期积累"只有陈平能做到"的信誉**——形成不可替代性

**注意事项**：
- **不是每次都出奇计**——陈平平时低调、关键时刻才出手
- 资源使用要有回报——四万斤黄金不能白花
- 不可替代性要能"背后转正"——从"奇计大家"到"贤相"的转型

## 应用场景

### 历史案例分析

**案例：陈平反间计破范增**

**背景**：前204年楚围荥阳，刘邦绝境。陈平建议反间——离间项羽与范增。

**应用过程**：
1. **步骤1（诊断）**：项羽"恭敬爱人但吝啬封赏"——骨鲠之臣"不过数人"——必然怀疑
2. **步骤2（引爆动作）**：当楚使来汉时——**摆出太牢、看到楚使假装惊讶、换恶草具**——一顿饭的成本
3. **步骤3（关键人）**：不是找范增、而是找项羽——**让项羽自己怀疑**——借助楚使作为传话筒
4. **步骤4（撤出）**：陈平从不解释"这是我的计"——就让楚使回报、项羽自己消化
5. **步骤5（建立预期）**：刘邦赐黄金四万斤——陈平用这笔钱——**后来"益封"**——形成"奇计有封"的激励

**结果**：
- 范增怒辞归
- 未到彭城疽发背而死
- 项羽失去最重要谋士
- 楚汉战争走向转折

### 现代应用示例

#### 场景1：创业公司挖对方核心技术

**问题**：竞品公司有一个核心技术专家——直接挖难度大、薪资给不起——如何"让他主动来"？

**应用**：
1. **诊断结构性弱点**：对方公司对这个专家"重要但不让他主导"——形成内在不满
2. **低成本引爆动作**：不是开天价——而是**在业内公开发表一篇"某技术的未来方向"文章**（与该专家的理念一致）——让行业都看到"只有我们懂他的理念"
3. **关键人**：不是直接联系专家——而是与他的"学术好友"或"理念同道者"建立联系——**让同道者把消息传给他**
4. **不声张**：不主动联系专家——**等专家自己来打听**
5. **长期预期**：一旦专家加入，继续发表他主导的方向文章——**形成"只有我们是他理念的归处"的品牌**

**预期效果**：专家主动过来、薪资不高也愿意来、还带来他的学生和合作者

#### 场景2：对抗办公室政治陷阱

**问题**：某同事通过向老板"私下说你坏话"影响你的晋升——正面对质等于小题大做——如何破？

**应用**：
1. **诊断结构性弱点**：老板喜欢"听内部消息"——但不喜欢"听到同样一件事的两种版本"——**怀疑是必然心理**
2. **引爆动作**：不是反咬对方——而是**主动向老板汇报自己的工作时顺便提"某某同事也在做类似的事，有不同视角"**——老板会自己去听同事版本——发现两个版本不一致——**怀疑的种子种下**
3. **关键人**：老板——不是直接说"某人说我坏话"，而是让老板自己发现矛盾
4. **撤出**：不多说、不指控——让老板自己琢磨
5. **长期预期**：下次同事再"说坏话"，老板会自动打折——**一次操作保护长期**

**预期效果**：对方的"说坏话"不再有效——自己的名誉修复——且不留下"搞办公室政治"的痕迹

## 成功标准

### 过程指标

- [ ] 能清晰说出对方的"强项+短板+接缝处"
- [ ] 能列出至少三种备选方案并选出杠杆最大的
- [ ] 能找到至少一个对方的"关键非名义决策者"
- [ ] 执行动作后24小时内完全断线，不自吹自擂
- [ ] 事后不留下"我做的"证据

### 结果指标

- [ ] 用最小代价解决了最大问题
- [ ] 对方公开场合不知道是你干的
- [ ] 主公（或上级）得到信任度提升
- [ ] 自己成为"只有你能解决"的不可替代者
- [ ] 长期可以继续使用类似手段而不被识破

## 常见错误

### 错误1：用"奇计"解决"正道能解决"的问题

**表现**：明明可以直接沟通——却非要用反间、离间等手段
**原因**：把奇计当成"万能药"——或虚荣心作祟想显得聪明
**纠正**：**奇计只用于绝境**——陈平六出奇计都在生死关头——平时绝不用

### 错误2：引爆动作太大、留下痕迹

**表现**：动作很猛——但所有人都知道是谁干的——成了"阳谋"
**原因**：低估对方的分析能力、高估自己的隐身能力
**纠正**：**动作越小、痕迹越少**——换菜比伪造信件好

### 错误3：炫耀计谋过程

**表现**：事后到处说"多亏我这一计……"
**原因**：虚荣心、想要"谋士"名声
**纠正**：**"其计秘，世莫得闻"**——陈平六出奇计的具体做法至死不说——这样才能留下"奇计储备"给未来

### 错误4：对方的"结构性弱点"判断失误

**表现**：以为是破绽其实是误解——计策失败
**原因**：对对方了解不够深、用"自己的逻辑"推测对方
**纠正**：**反复观察验证**——破绽要确认是"必然"而非"偶然"——宁可不做也不要做错

### 错误5：忽视"引爆后的第二步"

**表现**：第一步动作完成——但没人接棒——事情没推进
**原因**：规划只到第一步、没想清楚整个链条
**纠正**：**所有连锁反应都要提前想好**——换菜→楚使报告→项羽疑亚父→范增辞归→死

## 进阶技巧

1. **"不打一次"的克制**：平时不出奇计——只在关键时刻——这样每一次都有最大权重
2. **"让主公花钱"的深度信任**：刘邦"恣所为，不问其出入"——陈平也从不贪——**信任换空间**
3. **"分功于他人"**：韩信封齐王时让张良去册封——**功劳不抢给自己**
4. **"预言自家结局"的道德清醒**：陈平说自己家会"世即废"——**承认代价反而让后世理解你是故意的**
5. **"日饮醇酒"的掩护**：奇计大家平时不该显得太聪明——要有"懒散"的掩护——否则对方时刻防备
6. **"给对方台阶"**：反间计让范增自己辞归——不是直接让项羽杀他——**给对方决策权**
