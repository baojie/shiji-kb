# 高风险行动时机选择：专诸曹沫豫让荆轲四维分析框架与窗口期识别法

专诸刺王僚成功（等九年：楚平王死∩二弟被困∩季子出使∩楚绝后路），曹沫劫盟成功（盟誓刚完成∩双方在场∩心理放松），豫让三次失败（目标警觉性持续升高），荆轲被催促仓皇行动失败。核心框架：窗口期=外部条件∩内部条件∩心理条件同时满足；错过后长期不再出现；被外部压力打乱节奏是最常见失败原因。四维评分：接近难度/防御强度/成功率/脱身可能。

**来源**：《史记·刺客列传》（卷086）

> "王僚可杀也。母老子弱，而两弟将兵伐楚，楚绝其后。方今吴外困于楚，而内空无骨鲠之臣，是无如我何。"（专诸判断）
> "往而不返者，竖子也！且提一匕首入不测之彊秦，仆所以留者，待吾客与俱。"（荆轲反驳催促）

## 四大案例对比

### 成功案例1：曹沫劫盟

**时机**："齐桓公与鲁庄公既盟于坛上"

| 维度 | 状态 | 分析 |
|------|------|------|
| 外部条件 | 公开盟誓坛上 / 双方及群臣在场 | 神圣仪式，齐桓公难以反悔 |
| 内部条件 | 曹沫作为将军有理由在场 / 携匕首有解释 | 接近零成本 |
| 心理条件 | 盟誓刚完成，最放松时刻 / 左右莫敢动 | 目标警觉性最低 |

**为何此时最佳**：盟约神圣性 + 公开性 + 管仲在场（可劝守约）+ 双方即将分离（最后机会）

### 成功案例2：专诸刺王僚

**等待九年，五条件同时满足**：

| 条件 | 状态 |
|------|------|
| 楚平王死（外部）| 楚无暇北顾 |
| 二弟被困楚（外部）| 无法回援 |
| 延陵季子使晋（外部）| 潜在威胁不在 |
| 楚绝吴将后路（外部）| 二弟真正无法动 |
| 公子光设宴（内部创造）| 最接近机会 |

**专诸的五维分析**：家庭（母老子弱）/ 军事（二弟被困）/ 内政（内空无骨鲠臣）/ 外交（外困于楚）/ 时间（宴饮警戒最低）

### 失败案例1：豫让三次刺杀

**第一次（涂厕）**：赵襄子刚灭智伯，警觉性极高；环境单一（厕所）易被困

**第二次（伏桥下）**：马惊暴露；襄子有了第一次经验，更警惕；桥下地点可预测

**第三次（击衣）**：已被包围；象征性完成心理闭环，但未达实际目标

**失败规律**：第一次失败→目标警觉性永久升高→后续机会几乎为零

### 失败案例2：荆轲刺秦

**准备不足就出发的代价**：
- 荆轲有所待，欲与某人同往
- 太子丹催促："日已尽矣"
- 荆轲愤而仓促出发
- 秦舞阳殿上色变（暴露异常）
- 秦王警觉性提高
- 剑长难拔（环境不利）
- 荆轲欲"生劫"而非径杀（执行犹豫）

**核心教训**：外部压力 ≠ 时机成熟；准备不足的行动不如不行动

## 窗口期识别框架

### 三维条件分析

**窗口期 = 外部条件 ∩ 内部条件 ∩ 心理条件**

**外部条件检查**：
- [ ] 目标的主要盟友/支持者不在场
- [ ] 外部环境对目标构成压力
- [ ] 救援力量无法及时到达
- [ ] 没有更强力量可以干预

**内部条件检查**：
- [ ] 己方武器/工具/计划准备充分
- [ ] 有合理的接近目标理由
- [ ] 有退路或后备方案
- [ ] 团队成员心理和技能准备好

**心理条件检查**：
- [ ] 目标处于放松状态（仪式、宴饮、日常）
- [ ] 目标警惕性降低
- [ ] 场合让目标难以大规模调动护卫

### 四维时机评分矩阵

| 时机选项 | 接近难度 | 防御强度 | 成功率 | 脱身可能 | 综合评分 |
|---------|---------|---------|--------|---------|---------|
| 出行路上 | 高 | 极高 | 低 | 低 | 2/10 |
| 正式朝会 | 中 | 高 | 低 | 低 | 3/10 |
| 祭祀仪式 | 中 | 中 | 中 | 低 | 5/10 |
| 宴饮聚会 | 低 | 中 | 高 | 低 | 8/10 |
| 盟誓时刻 | 低（已在场）| 低（仪式氛围）| 高 | 高（谈判）| 9/10 |

## 三大核心原则

### 原则1：等待条件同时成熟

**专诸的公式**：`楚平王死 ∩ 二弟被困 ∩ 季子出使 ∩ 楚绝后路 = 窗口期`

**曹沫的公式**：`盟誓完成 ∩ 双方在场 ∩ 心理放松 = 窗口期`

**错误做法**：只有一两个条件满足就出手→豫让/荆轲模式

### 原则2：不被外部压力打乱节奏

**太子丹催促荆轲的教训**：
- 太子丹："日已尽矣，丹请得先遣秦舞阳"
- 外部压力（秦兵进逼）打乱内部节奏
- 荆轲未准备好就出发

**启示**：区分"紧急"和"时机成熟"；紧急感是主观的，条件成熟是客观的

### 原则3：第一次失败后机会接近于零

**豫让的三次失败**：每次失败后目标警觉性倍增

**对策**：要么等条件完全成熟再出手；要么放弃，改变策略

## 创造时机 vs 等待时机

| 方式 | 典型案例 | 优点 | 缺点 |
|------|---------|------|------|
| 等待时机（被动）| 专诸等待九年 | 条件更成熟 | 时间成本高 |
| 创造时机（主动）| 公子光设宴邀王僚 | 时间可控 | 需要额外代价 |
| 利用仪式时机 | 曹沫利用盟誓 | 神圣性加持 | 机会稀少 |

**创造时机的方法**：利用仪式（曹沫利用盟誓）/ 创造场景（公子光设宴）/ 降低警惕（宴饮、托称足疾）

## 现代应用

### 产品发布时机

**对标专诸**：
- 外部条件：竞争对手产品有重大缺陷、行业热点出现
- 内部条件：产品打磨成熟、团队准备好
- 心理条件：市场对此类产品期待值高

**检查清单**：
- [ ] 竞争对手是否有弱点期（高管离职、产品召回）
- [ ] 行业是否有热点事件（新政策、新技术）
- [ ] 己方产品是否真的准备好（不被催促打乱节奏）
- [ ] 市场心理是否接受度高

### 商业谈判时机

**对标曹沫**：
- 外部条件：对方需要你（急需签约、有公开承诺压力）
- 内部条件：你有筹码（技术、资源、关系）
- 心理条件：对方处于放松或必须决策的时刻

### 市场进入时机

**窗口期公式**：`竞争对手失误 ∩ 政策利好 ∩ 资金到位 ∩ 团队Ready = 窗口期`

**反面案例（对标太子丹催促）**：投资人催促上市→产品未打磨→团队未准备→失败

## 延伸思考

**问题1：如果窗口期永不出现怎么办？**
答：创造时机（如公子光设宴）或改变策略

**问题2：外部压力很大（如太子丹的秦兵压境）怎么办？**
答：区分"紧急"和"时机成熟"，不被紧迫感绑架；准备不足时宁可放弃

**问题3：第一次失败，还有第二次机会吗？**
答：通常没有（豫让三次皆失败），所以第一次必须成功

## 检查清单

执行前：
- [ ] 外部条件：支持者/盟友不在场或无法救援
- [ ] 外部条件：环境对目标构成压力或限制
- [ ] 内部条件：武器/工具/计划准备充分
- [ ] 内部条件：有合理的接近目标的理由
- [ ] 内部条件：团队成员心理和技能准备好
- [ ] 心理条件：目标处于放松状态
- [ ] 心理条件：目标的警惕性降低到可接近程度
- [ ] 时机条件：这是短期内唯一或最佳窗口期
- [ ] 节奏控制：没有被外部压力打乱己方准备节奏

如有任一项未满足，重新评估是否应该推迟。
