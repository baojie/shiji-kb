# 秦国与楚国人物列表

秦、楚两国是战国最强的两股力量。此列表收录属于秦国**或**楚国的人物，并排除同时带两国标签的模糊归属者。

::: query
type: person
tags_any: [秦国, 楚国]
tags_not: [外邦]
sort: total_refs
order: desc
limit: 150
display: table
computed: {age: death_ce - birth_ce}
fields: [label, birth_ce, death_ce, age, tags, total_refs, total_chapters]
field_labels: {birth_ce: 生年, death_ce: 卒年, age: 寿命, total_refs: 引用次数, total_chapters: 涉及章节}
title: 秦国或楚国人物（OR，且排除外邦标签，按引用次数排序）
:::
