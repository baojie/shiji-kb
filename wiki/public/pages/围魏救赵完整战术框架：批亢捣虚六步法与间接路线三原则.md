# 围魏救赵完整战术框架：批亢捣虚六步法与间接路线三原则

孙膑桂陵之战核心方法："不控卷、不搏撠"——不直接救援，而是"批亢捣虚，形格势禁"——攻击敌人后方要害（大梁），迫使魏军自救撤兵，在有利战场歼灭回撤之敌。六步操作：识别要害→确认空虚→快速机动→控制要害→等待回救→选择有利战场。与马陵之战对比：桂陵是基本款围魏救赵，马陵是加入减灶诱敌+伏击的豪华版。

**来源**：《史记·孙子吴起列传》（卷065）

> "夫解杂乱纷纠者不控卷，救斗者不搏撠，批亢捣虚，形格势禁，则自为解耳。"（孙膑）
> "田忌从之，魏果去邯郸，与齐战于桂陵，大破梁军。"

---

## 核心原理：三个"不"与三个"要"

### 三个"不"原则

1. **不控卷**：不要硬扯（不直接对抗锋芒）
2. **不搏撠**：不要参与（不加入对方选择的混战）
3. **不（直接）救援**：不正面迎敌

### 三个"要"策略

1. **批亢**：打击敌人要害
2. **捣虚**：攻击敌人空虚之处
3. **形格势禁**：改变局势，迫使敌人按你的节奏行动（"则自为解耳"）

**核心洞察**：不在敌人选择的战场作战，而是把敌人引到你选择的战场

## 桂陵之战案例分析

**背景**：魏伐赵，赵急请救于齐

**传统方案（田忌）**：欲引兵之赵（直接救援）

**孙膑的分析链**：

| 分析维度 | 内容 |
|---------|------|
| 敌情 | 轻兵锐卒必竭于外，老弱罢于内（精锐在外，后方空虚）|
| 我方优势 | 机动性强，可攻击魏国本土 |
| 战略方案 | 引兵疾走大梁，据其街路，冲其方虚 |
| 预期效果 | 彼必释赵而自救→一举解赵之围→收弊于魏（魏军疲于奔命）|

**战果**：魏果去邯郸，与齐战于桂陵，大破梁军

## 六步操作法

### 步骤1：识别敌人要害

**判断标准**：
- 敌人不能失去的（失去=亡国级别）
- 比敌人当前正在攻击的目标**更重要**
- 威胁后敌人必然回救（不能是虚张声势）

**桂陵案例**：大梁（魏都）> 邯郸（盟友都城）→ 魏军必回救

**现代类比**：企业总部 / 核心产品线 / 关键供应链 / 品牌声誉

### 步骤2：确认敌后空虚

**孙膑分析**："轻兵锐卒必竭于外，老弱罢于内"

**验证方法**：
- 情报收集：敌人兵力分布
- 逻辑推理：进攻必然抽调主力，进攻越深入后方越空虚
- 时机判断：敌人进攻高峰期是后方最虚弱的时刻

**风险**：如果敌人有充足预备队防守后方→失败

### 步骤3：快速机动

**关键词**："引兵疾走大梁"

**要点**：速度快（趁敌人反应之前）+ 目标明确（直扑要害）+ 不中途纠缠

**现代应用**：快速抢占市场空白；突然发布革命性新品（改变战场）

### 步骤4：控制要害

**孙膑方案**："据其街路，冲其方虚"

**关键**：不一定真要攻下要害，而是造成威胁+显示能力→迫使回救

**两种策略**：
- **围点打援**（桂陵）：目的是消灭回救的援军
- **真攻要害**：目的是夺取要地

### 步骤5：等待回救

**预期**："彼必释赵而自救"

**心理分析**：都城被围 vs 继续攻赵→理性选择回救

**前提**：威胁必须真实，敌人必须相信你有能力也有意愿真正攻下要害

### 步骤6：选择有利战场交战

**孙膑计算**："是我一举解赵之围而收弊于魏也"

**优势**：
- 在魏国本土作战（魏军更着急，更容易犯错）
- 魏军疲于奔命（从赵国长途撤回）
- 齐军以逸待劳（提前布置好战场）

## 四大决策点

### 决策1：是否使用围魏救赵？

**必要条件（全部满足才用）**：
- ✓ 直接救援成本过高或风险太大
- ✓ 敌人后方确实空虚
- ✓ 有能力攻击敌人要害
- ✓ 敌人会理性回救

**禁忌条件（任何一个存在则不用）**：
- ✗ 敌人不在乎后方（如亡命之徒）
- ✗ 敌人有充足预备队防守后方
- ✗ 盟友支撑不到敌人回救
- ✗ 无法快速机动到敌后

### 决策2：攻击哪个要害？

**选项评估**：必须是敌人不能失去的 + 比当前目标更重要 + 你能攻击到

### 决策3：围点打援还是真攻？

取决于战略目标——解围援友为主→围点打援；获取要地为主→真攻

### 决策4：敌人不回救怎么办？

必须有Plan B：继续攻要害（真的拿下）或 先救援盟友（备选方案）

## 三大风险控制

| 风险 | 后果 | 预防方法 |
|------|------|---------|
| 敌人不回救 | 盟友被灭，进攻无效 | 确保威胁真实，选真正要害，备好真攻准备 |
| 敌人有预备队 | 进攻受阻，伤亡惨重 | 情报先行，快速试探，备Plan B |
| 盟友支撑不住 | 来不及解围 | 评估盟友承受力，加快速度，必要时先增援 |

## 现代四大应用场景

### 商业竞争

竞争对手正在抢你的客户 → **错误**：直接价格战 → **围魏救赵**：攻击对手核心市场，挖对手核心团队，发布革命性新品改变战场（对手被迫回防）

### 职场政治

同事在上司面前打压你 → **错误**：直接对抗或告状 → **围魏救赵**：与同事上级合作（攻其要害）；做出成绩让上司主动关注（改变局势）

### 谈判策略

对方在价格上死守 → **错误**：死磕价格 → **围魏救赵**：转移战场（提出付款方式、合作期限等其他议题），暗示有其他选择（形格势禁）

### 项目管理

团队陷入技术难题 → **错误**：硬啃难题，加班死磕 → **围魏救赵**：调整需求（攻其要害），改变技术方案（转移战场），寻找替代路径
