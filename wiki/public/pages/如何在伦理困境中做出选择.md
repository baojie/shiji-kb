# 如何在伦理困境中做出选择

**类型**：程序性知识（决策框架）
**应用场景**：多重伦理冲突、忠诚与责任的权衡、个人价值观选择
**核心案例**：聂政（孝道vs义气）、豫让（怀二心vs以死报恩）、田光（保密vs清白）
**来源**：《史记·刺客列传》（卷086）

---

## 适用场景

1. 面临多重道德义务冲突（如忠诚 vs 孝道）
2. 需要在不同关系中选择优先级（家人 vs 雇主）
3. 个人承诺与外部压力冲突
4. 必须做出不可逆的重大选择

**现代应用**：职业发展 vs 家庭照顾；公司利益 vs 个人道德；短期利益 vs 长期承诺

---

## 三种伦理困境类型与模型

### 类型1：孝道 vs 义气（聂政模型）

**聂政的两难**：老母需要照顾（孝道），严仲子千里求贤（义气）。

**决策框架（优先级序列）**：
```
1. 母在 > 一切（绝对优先）
2. 母死 + 姊嫁 → 可践江湖义
3. 姊在 → 保护姊姊（毁容自杀）
```

**决策过程**：
- 阶段1（母在）：严仲子求 → 聂政拒绝 → "老母在，政身未敢以许人也"
- 阶段2（母死后）：反思 → 主动西见严仲子 → "政将为知己者用"
- 阶段3（执行时）：独行（拒绝车骑），毁容自杀保护姊姊聂荣

---

### 类型2：普通待遇 vs 国士待遇（豫让模型）

**豫让的差异化报恩原则**：

> "臣事范、中行氏，范、中行氏皆众人遇我，我故众人报之。至于智伯，国士遇我，我故国士报之。"

```
待遇等级 → 回报等级
众人遇（普通雇佣） → 众人报（完成工作）
国士遇（顶级尊重） → 国士报（以死相报）
```

**拒绝"怀二心"的道德高地**：

朋友建议假装臣服襄子后刺杀，豫让拒绝：
> "既已委质臣事人，而求杀之，是怀二心以事其君也。且吾所为者极难耳！然所以为此者，将以愧天下后世之为人臣怀二心以事其君者也。"

选择极难方式（漆身吞炭），不用诈术，树立道德标杆。

---

### 类型3：保密 vs 证明清白（田光模型）

**困境**：太子丹嘱咐"国之大事，愿先生勿泄"——田光解读为被怀疑。

**田光的逻辑**："为行而使人疑之，非节侠也"——被怀疑无法用言语证明，只能以死证明。

**自刎的三重目的**：
1. 洗刷怀疑：以死证明不会泄密
2. 激励荆轲：用生命重量昭示任务严重性
3. 道德宣示："长者为行，不使人疑之"

---

## 三个决策框架

### 框架1：优先级排序法（聂政模型）

**步骤1**：列出所有相关伦理义务（孝道、手足之情、知遇之恩、自我保护）

**步骤2**：设定优先级（P1母在最高，P2次之）

**步骤3**：在不同阶段应用优先级（母在时P1生效，母死后P2生效，执行时P3生效）

---

### 框架2：对等原则法（豫让模型）

**步骤1**：评估对方的待遇等级（范氏=众人遇Level 1，智伯=国士遇Level 10）

**步骤2**：设定对等的回报等级

**步骤3**：一致性应用（对范氏不复仇，对智伯以死相报）

**一致性检验**：为范氏复仇 = 违反对等原则；不为智伯复仇 = 违反对等原则。

---

### 框架3：道德高地法（田光/豫让模型）

不选择最容易/最有效的方式，而选择道德上最无瑕疵的方式。

**豫让**：拒绝诈术，选择极难方式（漆身吞炭）→ 理由："将以愧天下后世之为人臣怀二心以事其君者"

**田光**：拒绝口头保证，选择以死证明 → 理由："长者为行，不使人疑之"

## 现代应用场景

### 场景1：职业发展 vs 家庭照顾（聂政框架）

```
阶段1：父母健在且需要照顾
→ 优先级：家庭 > 职业 → 拒绝海外机会

阶段2：父母身体好转/有其他人照顾
→ 可以考虑职业机会

阶段3：父母去世
→ 全力追求事业
```

关键：在阶段1明确拒绝并说明理由——雇主如果真惜才会等待（严仲子等到聂政母死）。

### 场景2：公司利益 vs 个人道德（豫让框架）

公司要求做违背个人道德的事时：
- 选项A（怀二心）：表面服从，实际破坏 → 豫让拒绝
- 选项B（直接）：明确说明道德底线，不采取"暗中破坏"的方式

### 场景3：跳槽 vs 忠诚（对等原则）

```
现雇主"众人遇"（市场价工资）→ 可接受更好offer
现雇主"国士遇"（培养、信任、超常待遇）→ 不应轻易离开
```

## 伦理困境中的沟通技巧

**技巧1：明确说明理由（聂政）**——"老母在，政身未敢以许人也" → 不是拒绝人，是时机不对，为未来合作留余地

**技巧2：区分不同待遇等级（豫让）**——逻辑自洽，对方理解（襄子认可其"义"）

**技巧3：承担责任而非指责（田光）**——不指责太子，客观陈述，用行动证明

---

## 常见错误

| 错误 | 案例 | 纠正 |
|------|------|------|
| 用单一标准衡量所有关系 | 以众人报之标准对待国士雇主 | 豫让对等原则 |
| 优先级未解除时强行行动 | 母在时就应诺严仲子 | 等母死除服后再应诺 |
| 用诈术代替直接面对 | 接受"假装臣服"建议 | 选择极难方式保持道德高地 |

---

## 检查清单

面临伦理困境时：
- [ ] 我列出了所有相关的伦理义务吗？
- [ ] 我为这些义务设定了明确的优先级吗？
- [ ] 我的优先级符合我的核心价值观吗？
- [ ] 我考虑了不同阶段的优先级变化吗？
- [ ] 我评估了对方对我的待遇等级吗？（对等原则）
- [ ] 我选择的方式在道德上无瑕疵吗？
- [ ] 我明确沟通了我的理由吗？
- [ ] 我为未来可能的变化留有余地吗？
