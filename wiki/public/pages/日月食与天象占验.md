# 日月食与天象占验

《天官书》专论日月异象（日食、月食、日晕、月道）的观测与占验，构建了一套将天文现象与人事祸福相对应的系统。日月作为"三光"中最显著的成员，其变化被视为国君将相命运的直接预兆。

## 基本框架：日君月臣

> 日蚀，国君；月蚀，将相当之。

这是[[中国]]古代天文占验中最重要的对应原则：
- **日蚀**：最高级异象，对应国君的存亡
- **月蚀**：对应将相的命运
- **月晕**：有战场胜负预判功能

## 日晕：战场判决法

两军对阵时，观测日晕是判断胜负的标准方法：

> 两军相当，日晕；晕等，力钧；厚长大，有胜；薄短小，无胜。

**晕的形态与方位**：

| 形态 | 预兆 |
|------|------|
| 晕等 | 两军力量均等 |
| 厚长大 | 有胜 |
| 薄短小 | 无胜 |
| 重抱 | 大破无害 |
| 抱 | 和谐 |
| 背 | 分离 |
| 直 | 立侯王，杀将 |
| 围在中 | 中军胜 |
| 围在外 | 外军胜 |

**时间节奏**：
- 气晕先至而后去：居军胜
- 先至先去：前利后病
- 后至后去：前病后利
- 后至先去：前后皆病，居军不胜

> 日晕制胜，近期三十日，远期六十日。

## 月道：月亮行度的预占

> 月行中道，安宁和平。

月亮行道的预占体系：

| 行道 | 预兆 |
|------|------|
| 中道 | 安宁和平 |
| 阴间 | 多水，阴事 |
| 阳间 | 骄恣 |
| 阴星 | 多暴狱 |
| 太阴 | 大水，兵 |
| 太阳 | 大旱丧 |
| 犯四辅 | 辅臣诛 |

## 月食与五星

月食五星是高级预占，按五星各有不同含义：

| 月蚀 | 预占 |
|------|------|
| 岁星 | 其宿地，饥若亡 |
| 荧惑 | 也乱 |
| 填星 | 下犯上 |
| 太白 | 强国以战败 |
| 辰星 | 女乱 |
| 大角 | 主命者恶之 |
| 心宿 | 内贼乱 |

## 月食周期

> 月食始日，五月者六，六月者五，五月复六，六月者一，而五月者五，凡百一十三月而复始。

[[司马迁]]记载了月食的113个月循环规律——这与现代天文学中的沙罗周期（约18年11天，含223个朔望月）相关联，反映了汉代天文学对月食周期已有较准确的认识。

## 历史案例

**[[平城]]之围**：汉高祖被匈奴围于平城时，"月晕参、毕七重"——七重月晕被记载为危难天象的史证，反映了天文占验在政治史书写中的重要作用。

## 日食的方位与分野

> 甲、乙：四海之外，日月不占。丙、丁：江、淮、海岱。戊、己：中州、河、济。庚、辛：[[华山]]以西。壬、癸：恒山以北。

通过发生日食的时辰（天干）可以判断所涉及的地理区域，再结合日食所在的星宿，进一步确认受影响的诸侯国。

## 阴阳理论框架

> 中国于四海内则在东南，为阳；阳则日、岁星、荧惑、填星；占于街南，毕主之。其西北则胡、貉、月氏诸衣旃裘引弓之民，为阴；阴则月、太白、辰星；占于街北，昴主之。

阳对应中国（文明），阴对应夷狄（边疆）：
- 日（阳）→主中国之事
- 月（阴）→主边夷之事

这一体系将天文观测与华夷地缘政治相结合，赋予了天象预占以政治含义。
