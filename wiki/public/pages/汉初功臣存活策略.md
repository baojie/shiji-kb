# 汉初功臣存活策略

汉初功臣存活策略，是《史记》通过汉高祖功臣的不同命运，总结出的高风险政治环境中保全自身的策略模式。高祖在位期间（前202—前195年）诛杀了大量异姓诸侯王与功勋大臣，但也有人得以善终。

## 历史背景

### 被诛的功臣

高祖时期，大量功臣遭到清洗：

| 人物 | 命运 |
|------|------|
| 韩信 | 废为淮阴侯，后以谋反罪被吕后诛，夷三族 |
| 彭越 | 被告谋反，醢（剁成肉酱） |
| 英布 | 被逼起兵，兵败被杀 |
| 韩王信 | 降匈奴后被杀 |
| 陈豨 | 起兵反叛，被杀 |
| 卢绾 | 燕王，逃往匈奴 |

### 得以善终的功臣

| 人物 | 策略 | 结局 |
|------|------|------|
| 萧何 | 自污保身，让封让财 | 善终，相国 |
| 张良 | 辞封退隐，从赤松子游 | 善终，深藏若虚 |
| 陈平 | 与周勃分工，适时谦让 | 善终，丞相 |
| 周勃 | 出镇地方，忠诚不疑 | 善终，太尉 |
| 曹参 | 继萧何之策，萧规曹随 | 善终，相国 |

## 主要策略

### 策略一：功成身退（张良）

**代表人物**：张良（留侯）

**张良的选择**：
> "今以三寸舌为帝者师，封万户，位列侯，此布衣之极，于良足矣。愿弃人间事，欲从赤松子游耳。"

行动：辟谷修道，不与俗事，彻底远离权力中心。

**逻辑**：功名已达极致，继续参与只有危险。主动放弃是最安全的选择。

**适用场景**：功绩极高、个人声望卓著者。

### 策略二：自污保身（萧何）

**代表人物**：萧何

**核心手段**：主动暴露"缺点"（贪财、置产），以证明自己无政治野心。

具体操作：
- 淮阴侯诛后：让封勿受，捐家财佐军
- 黥布反时：多买田地，贱贳贷以自污（故意做出贪财的样子）
- 上林苑事件：为民请苑，被下狱，旋赦，以证明"有错"

**逻辑**：功高则主疑，主疑则危。展示"弱点"（贪财），是告诉君主：我不是政治对手，我只是个庸人。

**适用场景**：身处要职、功劳极大且无法隐退者。

### 策略三：忠诚不疑（周勃）

**代表人物**：周勃（太尉）

周勃始终表现出对刘氏的绝对忠诚：
- 诛灭诸吕时，以"左袒为刘氏"公开表态
- 代王入继后，立即归还兵权
- 文帝即位后，出镇地方，不恋京城权力

**逻辑**：通过关键时刻的正确选择，建立不可替代的忠诚记录。

### 策略四：让权谦逊（陈平）

**代表人物**：陈平

陈平在高后之后的政治操作：
- 与周勃共同诛灭诸吕，功劳极大
- 文帝即位后，主动将丞相位让于周勃
- 说明让让的理由："勃功多于平"（不争功）

**逻辑**：不独占权力高位，以谦让换取安全。

## 失败案例分析

### 韩信：不知收敛的典型

韩信的致命错误：
- 功高之后，多次在政治上表现出不安分（为项羽旧将钟离昧庇护）
- 被废为侯后，"称病不朝"，"行游"诸将帅
- 与陈豨私下密谋（被告）

蒯通曾建议韩信在楚汉未决时拥兵自立，韩信以"汉王待我甚厚"拒绝。

**太史公评语**：
> "假令韩信学道谦让，不伐己功，不矜其能，则庶几哉，于汉家勋可以比周、召、太公之徒，后世血食矣。不务出此，而天下已集，乃谋畔逆，夷灭宗族，不亦宜乎！"

韩信的问题：不懂谦让，功高后仍不收敛，最终以"谋反"为由被诛。

### 彭越：被动等待的失败

彭越游击战功极大，封梁王。高祖出征陈豨时，召彭越从军，彭越称病，高祖大怒。

部下逃跑，被逮捕，告彭越谋反。高祖流放彭越，吕后主张诛杀，"醢彭越"。

**彭越的失败**：既没有主动求退（如张良），也没有自污展现忠诚（如萧何），在关键时刻的"称病"被解读为消极抵抗。

## 通用规律

《史记》汉初功臣案例揭示了高风险政治环境的生存规律：

1. **功高必疑**：功绩越大，君主猜疑越深，危险越高
2. **威胁感知**：君主的威胁感知比事实更重要——不是"你是否真的危险"，而是"你是否让君主觉得你危险"
3. **谦让降低威胁**：主动谦让（让封、让位、让财）是降低威胁感的有效手段
4. **退路的重要性**：有退路（辞封隐退）的人比没退路的人更安全
5. **忠诚需要表演**：忠诚不是内心感受，而是需要通过具体行动（自污、让权、公开表态）不断证明
