# 稷下学宫

稷下学宫，是战国时期齐国在临淄稷门附近建立的官方学术机构，由齐威王初创、齐宣王时达到鼎盛，聚集了战国时代大部分顶尖思想家，是百家争鸣的最重要制度平台，《史记·田敬仲完世家》有记载。

## 制度设计

齐宣王时期，稷下学宫核心成员达七十六人：

> "宣王喜文学游说之士，自如驺衍、淳于髡、田骈、接予、慎到、环渊之徒七十六人，皆赐列第，为上大夫，不治而议论。是以齐稷下学士复盛，且数百千人。"

**制度特征**：
- **"赐列第"**：赐予学者最高级别的住宅，物质保障充足
- **"为上大夫"**：赋予学者最高官阶，社会地位显赫
- **"不治而议论"**：不需要处理政务，只做学术研究——这是最关键的安排

"不治而议论"意味着学者可以自由批评政治而不承担执政后果，这种制度保障了学术自由。

## 主要学者

稷下学宫聚集的思想家涵盖多个流派：

| 学者 | 流派 |
|------|------|
| 邹衍 | 阴阳家 |
| 淳于髡 | 杂家 / 谋士 |
| 田骈 | 道家 |
| 接予（季真） | 道家 |
| 慎到 | 法家 |
| 环渊 | 道家 |
| 荀子（晚期） | 儒家 |
| 孟子（曾访齐） | 儒家 |

## 历史背景：田氏的合法性焦虑

稷下学宫的建立，有其政治逻辑：田氏以非正统途径取代姜氏成为齐君，文化软实力可以补足正统性的先天不足。聚天下贤士、建最高学府，是以文化制高点弥补政治正当性的一种方式。

## 运作机制

稷下学宫实质上是国家资助的"思想智库+大学"复合体：

- **上层核心**：七十六位上大夫级别的顶级学者，负责重大理论研究
- **中层学士**：数百到千人级别的各地来学之士，互相切磋辩论
- **与政治的距离**：保持相对独立，但重大决策时也会向齐王进言

著名例子：邹忌（稷下学者出身）以琴论劝齐威王接受劝谏，进入政界成为相国；田骈等人则始终保持"清议"身份。

## 政治价值

### 对齐国的意义

稷下学宫为齐国提供了无与伦比的"软实力"：
- 天下学者聚齐，齐国成为战国时代的思想中心
- 秦国"文化落后"的形象与稷下的繁荣形成对比
- 合法性焦虑转化为文化自信

### 对百家争鸣的意义

战国时代"百家争鸣"能够发生，稷下学宫是最重要的制度保障：
- 提供国家级别的资金支持
- 提供安全的辩论空间
- 提供跨学派的思想交流平台

如果没有稷下，各家学者散布在不同诸侯国，很难有如此系统的思想交锋。

## 与稷下相关的重要辩题

稷下学者留下的核心辩题：

1. **"精气说"（宋钘、尹文）**：心与气的关系，道德的生理基础
2. **"势"论（慎到）**：强调形势而非个人德行的政治论
3. **"阴阳五行"（邹衍）**：用天文历法解释历史更替
4. **"礼义之辩"（荀子）**：人性恶论与礼义建构

## 衰落

稷下学宫在齐湣王时期仍有活动，但楚灭宋后的连横围攻（前284年乐毅率五国联军攻齐）使齐国几乎覆灭，稷下学宫随之瓦解，学者星散各国。此后稷下虽有所恢复，但再未达到宣王时代的鼎盛。
