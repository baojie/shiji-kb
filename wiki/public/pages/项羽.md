# 项羽

::: infobox
native: 下相（今江苏宿迁）
title: 西楚霸王
office: 上将军 → 西楚霸王
note: 司马迁破格列入本纪，与帝王同列
:::

**西楚霸王**。姓项名籍字羽。出身楚国将门（祖父[[项燕]]，楚末将军战死于王翦之手），**力能扛鼎，才气过人**。
秦末随叔父[[项梁]]起兵[[会稽]]，三年破秦，分封十八诸侯而自号"霸王"；
与[[刘邦]]相争五年，败于垓下，自刎[[乌江]]。年三十一。

司马迁将这位**位不终而开纪**的失败者列入本纪（[[007_项羽本纪|项羽本纪]]），
与[[秦始皇|始皇]]、[[刘邦|高祖]]并肩，称其"近古以来未尝有也"——给他的是帝王之礼，
下的却是"自矜功伐"的史家判词。这种**矛盾评价**本身，就是司马迁史学精神的注脚。

*《史记》中共出现约 550 次，排名帝王/准帝王级人物第二（仅次于[[刘邦]]约 728 次）。本传 [[007_项羽本纪]]，正文 155 次；另见 [[008_高祖本纪]]（67次）、[[018_高祖功臣侯者年表]]（34次）等 37 章。*

## 生平

### 一、少年（前 232 — 前 209）

生于[[下相]]。**项氏世世为楚将**，累世食封于项，故以项为氏。少时学书不成，去；学剑又不成。
项梁怒之，项籍曰：

> *"书足以记名姓而已。剑一人敌，不足学；学万人敌。"*

遂学兵法——略知其意又不肯竟学。

始皇三十七年，秦始皇游[[会稽]]渡浙江，项梁项羽俱观。项羽看着始皇仪仗，脱口而出：

> **"彼可取而代也！"**

项梁掩其口："毋妄言，族矣！" 然心里奇之。**项羽身长八尺馀，力能扛鼎，才气过人**，
吴中子弟皆已畏之。

> *司马迁注："舜目盖重瞳子，又闻项羽亦重瞳子"——以舜的神异开篇，暗示此人命运不凡。*

### 二、起兵灭秦（前 209 — 前 206）

**会稽起兵** — 前 209 年秋，[[陈胜]]反于[[蕲]]。项梁、项羽斩会稽守起兵（007-005），
渡江北上。[[陈婴]]归附（007-007），项梁下[[范增]]之策，**立楚怀王孙心**为[[楚怀王]]以系楚人望（007-009）。

**项梁战死** — 前 208 年项梁轻秦，屡胜生骄，于[[定陶]]为[[章邯]]所破，战死（007-013）。
楚怀王欲乘势自立，夺项羽兵权。

**[[巨鹿之战|巨鹿破釜沉舟]]**（前 207） — 楚怀王命[[宋义]]为上将，项羽为次将北救赵。宋义行至[[安阳]]
"留四十六日不进"，项羽**斩宋义头**（007-015），怀王不得已拜项羽为上将军。

破釜沉舟 — 渡[[漳河]]时**皆沉船，破釜甑，烧庐舍，持三日粮**，以示士卒必死无还心。
九战九胜，大破[[王离]]、[[章邯]]（007-017）。战后诸侯将"**无不膝行而前，莫敢仰视**"（007-018）。

章邯降，项羽**新安坑杀降卒二十万**（007-022）。自此**"项羽"之名震天下**。

**入咸阳、[[鸿门宴]]** — 前 206 年十一月破[[函谷关]]入关。刘邦已先入，项羽怒，欲击。
[[项伯]]夜访[[张良]]通消息，遂有**[[鸿门宴]]**（007-027）：

> *"项庄舞剑，意在沛公。"* — 樊哙闯帐救主，[[刘邦]]脱身归营。

项羽入咸阳**屠城、烧秦宫**（"火三月不灭"）、东归分封。

**分封十八诸侯**（前 206 年春） — 自号**西楚霸王**，都彭城，以"先入关者王之"约逐刘邦，割地分封十八诸侯：

### 三、楚汉争霸（前 206 — 前 202）

[[刘邦]]还定[[三秦]]反击。五年间项羽屡胜却愈弱：

**彭城大捷**（前 205，[[彭城之战]]） — 刘邦联五诸侯，以五十六万众取彭城。项羽闻讯，急引三万精兵从鲁出胡陵，**晨时突袭**，日中大破汉军。汉卒走睢水，"汉军走，为楚所挤，**睢水为之不流**"（007-042）。刘邦父太公、妻吕后皆被俘。

> *丁公追急，刘邦回马谓丁公："两贤岂相厄哉！"丁公遂引兵还。汉得解去。*
> *然汉王得天下后，召诸侯见丁公，斩之，以"为项王臣不忠，使项王失天下"示天下。*（007-042）

**荥阳对峙** — [[陈平]]反间计，逼走[[范增]]（"吾欲竖子不足与谋"）；[[纪信]]诈降，刘邦脱身——项羽**陷入两线作战**，始终不能决定性击败汉军。

**成皋攻守** — 外黄城投降后，项羽欲坑杀男子。一少年谏曰："外黄之民，县上将复三日降，而君坑之……"项王乃赦之。（"**项王仁之**"——暴力与仁心之矛盾在此典型呈现）

**广武对峙三场景**（前 204—203）：

1. *烹太公威胁*：项王置刘邦之父于高俎，喝曰："今不急下，吾烹太公！"
   刘邦答：**"吾翁即若翁，必欲烹而翁，幸分我一杯羹。"**（007-141）
   项伯劝："天下事未可知，且为天下者不顾家。"项王遂罢。

2. *单挑请决*：项王曰："天下匈匈数岁者，徒以吾两人耳，**原与汉王挑战决雌雄，毋徒苦天下之民父子为也。**"
   汉王笑拒："吾宁斗智，不能斗力。"
   楚军楼上以一箭**射中汉王胸**——刘邦扪足诈言"虏中吾趾"以安军心。（007-144）

3. *鸿沟中分*（前 203）：楚汉约中分天下，以鸿沟为界，归汉太公、吕后。项羽东归，刘邦用张良、陈平计毁约追击——**约定立成陷阱**，是刘邦最终决定性的政治手段。

**固陵之败 → 垓下之围**（前 202）

### 四、垓下 · 乌江（前 202）

**四面楚歌** — 夜闻汉军四面皆楚歌，项羽惊曰：

> *"汉皆已得楚乎？是何楚人之多也！"*

**霸王别姬** — 项羽夜起饮帐中。有美人名[[虞]]，常幸从；骏马名[[骓]]，常骑之。乃悲歌慷慨，自为诗：

> ## 垓下歌
> **"力拔山兮气盖世，**
> **时不利兮骓不逝。**
> **骓不逝兮可奈何，**
> **虞兮虞兮奈若何！"**

*歌数阕，美人和之。项王泣数行下，左右皆泣，莫能仰视。*

**突围南走** — 率八百骑溃围南出。至[[阴陵]]迷道，陷大泽；至[[东城]]骑仅二十八。
谓骑曰："**此天之亡我，非战之罪也。**"三胜证之，斩汉将、刈旗。

**乌江自刎** — 至[[乌江]]。乌江亭长义船待，劝渡江东"地方千里众数十万亦足王也"。项羽笑曰：

> *"天之亡我，我何渡为！且籍与江东子弟八千人渡江而西，今无一人还。*
> *纵江东父兄怜而王我，我何面目见之？纵彼不言，籍独不愧於心乎？"*

乃**赐乌骓于亭长**，令骑下马步战。独项羽杀汉军数百人，身亦十馀创。顾见旧故[[吕马童]]，曰：

> *"若非吾故人乎？……吾闻汉购我头千金，邑万户，吾为若德。"*

**乃自刎而死**。[[王翳]]取其头，汉军分其尸，[[吕马童]]、[[王翳]]、[[杨喜]]、[[杨武]]、[[吕胜]]
五人各得一体，共会而验之皆是，**各封侯**（007-183）。年三十一。

**礼葬与尾声** — 楚地皆降，**独鲁不下**——鲁为礼义之邦，项羽旧封鲁公，民为主死节。
汉王引兵欲屠，以项王头示鲁父兄，鲁乃降。**刘邦以鲁公礼葬项羽于[[谷城]]，亲为发哀哭**（007-186）。
诸项氏不诛，[[项伯]]封射阳侯，其余赐姓刘。

## 他人眼中的项羽

**[[韩信]]评 · 妇人之仁 · 匹夫之勇**（[[092_淮阴侯列传|淮阴侯列传]]）：

> "项王喑噁叱咤，千人皆废，然不能任属贤将，此特**匹夫之勇**耳。
> 项王见人恭敬慈爱，言语呕呕，人有疾病，涕泣分食饮，至使人有功当封爵者，
> 印刓敝，忍不能予，此所谓**妇人之仁**也。"

**[[范增]]评** — 鸿门宴前："此其志不在小。"（见刘邦气象而预言其必为敌）
离楚时叹："**竖子不足与谋**！天下事大定矣，君王自为之！愿赐骸骨归卒伍。"

**[[陈平]]评** — *"项王为人，恭敬爱人，士之廉节好礼者多归之；至於行功爵邑，重之，士亦以此不附。"*（[[056_陈丞相世家|陈丞相世家]]）

**[[郦食其]]评** — *"项王有倍约之名，杀义帝之负。"*

**[[刘邦]]评**（汉五年论"何以得天下"）：

> "夫运筹策帷帐之中，决胜於千里之外，吾不如子房。镇国家，抚百姓，给馈馕，不绝粮道，吾不如萧何。
> 连百万之军，战必胜，攻必取，吾不如韩信。此三者，皆人杰也，**吾能用之，此吾所以取天下也。
> 项羽有一范增而不能用，此其所以为我擒也。**"（[[008_高祖本纪|高祖本纪]]）

## 性格矛盾：悲剧的两面

司马迁笔下的项羽是中国历史中最具张力的性格矛盾体：

| 一面 | 另一面 |
| --- | --- |
| 见伤者落泪（泣下沾襟）| 新安坑杀降卒二十万 |
| 鸿门宴手软放走刘邦 | 屠咸阳、烧秦宫（火三月不灭） |
| 赐骓马于亭长 | 暗杀义帝于江南 |
| 勇于单挑、不屑暗箭 | 义帝迁王、随即遣人杀之 |
| 乌江自刎以全气节 | 烹杀守城将周苛 |
| 分食于病卒 | 吝于封赏有功之士 |

这种矛盾不是简单的"好坏"，而是司马迁有意保留的**历史真实**：
项羽的"仁"是对个人的私情，他的"暴"是对敌方的工具。
当政治需要个人牺牲时，他两者皆失——**悲剧由此而生，而不由"天命"而来**。

## 太史公曰

司马迁在《项羽本纪》末给出史家最复杂的评价之一：

> *"吾闻之周生曰 **'舜目盖重瞳子'**，又闻项羽亦重瞳子。羽岂其苗裔邪？何兴之暴也！*
> *夫秦失其政，陈涉首难，豪杰蜂起，相与并争，不可胜数。然羽非有尺寸，**乘势起陇亩之中，
> 三年，遂将五诸侯灭秦，分裂天下，而封王侯，政由羽出，号为"霸王"，位虽不终，近古以来未尝有也**。*
> *及羽背关怀楚，放逐义帝而自立，怨王侯叛己，难矣。**自矜功伐，奋其私智而不师古**，
> 谓霸王之业，欲以力征经营天下，五年卒亡其国，身死东城，尚不觉寤而不自责，过矣。*
> *乃引 **'天亡我，非用兵之罪也'**，岂不谬哉！"*

四句史论：
1. **何兴之暴也** — 神奇的崛起
2. **近古以来未尝有也** — 空前的霸业
3. **自矜功伐，奋其私智而不师古** — 命定的败因
4. **"天亡我"岂不谬哉** — 否定项羽的天命论自辩

司马迁**把项羽列入本纪**（帝王传记），体例破格；又用这最后一问否定他的天命论——
此正是司马迁所谓"**究天人之际**"的实践：不是简单追随成王败寇，而是**诚实地面对复杂**。

## 与刘邦的"互见法"对照

司马迁把**项羽、刘邦的对手关系**作为《史记》戏剧张力的主线。两人诸多可比：

| 项目 | 项羽 | 刘邦 |
| --- | --- | --- |
| 出身 | 将门贵族 | 布衣亭长 |
| 性格 | 豪侠暴烈、重个人情义 | 豁达大度、善驾驭人 |
| 武力 | 能扛鼎、万人敌 | 不善战 |
| 将将 | 妇人之仁，吝于封赏 | 善将将，三杰为用 |
| 谋臣 | 只有一范增，离间而去 | 张良、陈平、萧何 |
| 对民 | 屠咸阳、坑降卒 | 约法三章、秋毫无犯 |
| 都城观 | "富贵不归故乡，如衣绣夜行"，回彭城 | 定都关中 |
| 结局 | 自刎乌江，年 31 | 称帝长安，年 62 |

司马迁借这对比**不直接评判**，让读者自悟"天命人事"之间的张力——**这正是互见法之妙**。
