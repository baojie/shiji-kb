# 叔孙通在秦末的生存策略：识时务者为俊杰

## 核心内容

### 秦廷的生死一言

陈胜起义时，二世召三十余名博士儒生问对策。多数人直言"人臣无将，将即反，罪死无赦"，触怒二世。叔孙通说这些人是"鼠窃狗盗"，不足为虑，二世大喜。

叔孙通出来后，同僚问他为何如此谄媚，他说："我几不脱于虎口！"

**这一场景的多重解读**：
- **生存智慧**：在专制君主面前，直言者被下狱，叔孙通选择说出君主愿意听的话，保全性命
- **道德代价**：同僚称其为"谀"（谄媚），叔孙通没有反驳
- **结果检验**：其后出逃，证明他并无幻想，只是在寻找逃跑时机

### 辗转多主的路径

```
秦二世（博士）→ 之薛（薛已降楚）→ 项梁 →
怀王（义帝）→ 留事项王 →
汉二年，汉王入彭城时，叔孙通降汉王 →
随汉王西，竟从汉
```

叔孙通事多主，但每次转投都在政治形势明确后。这是"时变"的具体体现：不提前赌注，等待局势清晰时再做选择。

### 拒绝推荐弟子的深意

降汉之后，叔孙通只向汉王推荐"群盗壮士"，不推荐自己的百余名儒生弟子。弟子们私下骂他。他解释：

> 汉王方蒙矢石争天下，诸生宁能斗乎？故先言斩将搴旗之士。诸生且待我，我不忘矣。

这不是遗忘弟子，而是深刻理解"时宜"：在打仗阶段，儒生没有用武之地；在守成阶段，才是儒生的舞台。他在等待正确的时机。

后来制定朝仪时，他兑现承诺，让所有弟子在朝仪参与后都得到了郎官职位。
