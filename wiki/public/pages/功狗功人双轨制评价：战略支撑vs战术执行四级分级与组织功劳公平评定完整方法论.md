# 功狗功人双轨制评价：战略支撑vs战术执行四级分级与组织功劳公平评定完整方法论

高祖论功行赏核心辩论：曹参攻城野战第一，萧何定为第一（功人>功狗）。完整方法论：功狗（执行者）vs功人（规划者）理论→万世之功vs一时之功→功劳评定三维度（战略价值×系统价值×持续价值）→双轨制评价（精神荣誉+物质奖励分开评定）→四级功劳分级（战略支撑/全程参战/关键贡献/局部战功）→三种量化方法（KPI量化/定性评估/反事实推理）。现代应用：销售vs研发评价困境解法，论文署名排序，组织绩效管理。

**来源**：《史记·高祖功臣侯者年表》（卷018）；参《萧相国世家》《曹相国世家》

> "夫猎，追杀兽兔者狗也，而发踪指示兽处者人也。今诸君徒能得走兽耳，功狗也。至如萧何，发踪指示，功人也。"（刘邦）
> 萧何：第一（精神荣誉），封8000户
> 曹参：野战第一，封户最多（10600+户）

## 核心辩论：谁功劳第一？

### 历史争议

**问题**：高祖分封，曹参"身被七十创，攻城略地，功最多"，按战功理应第一。但刘邦最终定萧何为第一。

**众将反对**：
> "平阳侯曹参身被七十创，攻城略地，功最多，宜第一。"

**刘邦的裁决逻辑**（功狗功人理论）：

```
狗的功劳：追杀野兽（曹参：攻城野战，直接战功）
人的功劳：发踪指示（萧何：战略规划，决定方向）

→ 功人 > 功狗
→ 萧何第一
```

**关隘**：诸将的反对被平阳侯鄂千秋化解："诸君知其一，未知其二。"

## 理论框架一：功狗 vs 功人

### 定义

| 类型 | 定义 | 特征 | 典型岗位 |
|------|------|------|---------|
| 功狗（执行者）| 直接完成任务 | 可量化/可见/即时 | 销售/前线将领/工程师 |
| 功人（规划者）| 设计系统/发现方向 | 难量化/隐性/长期 | 战略/后勤/架构师 |

### 为何功人 > 功狗

**逻辑链**：

```
没有方向（功人）→ 努力白费
有了方向 → 功狗的努力才有价值

因此：
功人的价值 = 决定功狗努力是否有效
功人是功狗的倍增器
```

**历史证据**：
- 萧何守关中：不仅提供粮草，还持续补充兵员（韩信之后"数使使劳苦汉王"）
- 无萧何 → 无法持续作战 → 曹参战功无从实现

### 识别功人 vs 功狗

**四个判断标准**：

| 标准 | 功人特征 | 功狗特征 |
|------|---------|---------|
| 可替代性 | 低（换一个后果截然不同）| 高（换一个照样完成）|
| 时间维度 | 长期持续（全程）| 短期爆发（某战役）|
| 影响范围 | 全局（影响所有战役）| 局部（某一场战）|
| 去掉后 | 整个系统崩溃 | 该任务失败但系统继续运行 |

## 理论框架二：万世之功 vs 一时之功

### 鄂千秋的关键论述

> "陛下与楚相距五岁，失军亡众，逃身遁者数矣。然萧何常从关中遣军补其处，非陛下所诏令召，而数万众会上之乏绝者数矣。"

**核心对比**：

| 维度 | 萧何（万世之功）| 曹参（一时之功）|
|------|--------------|--------------|
| 持续时长 | 全程七年 | 每次战役（短期）|
| 系统性 | 转漕关中=整个后勤系统 | 攻城掠地=单点突破 |
| 关键时刻 | 多次危机（数使使劳苦）| 多次战役 |
| 不可替代 | 极高（无人可代替守关中）| 较高（其他将领也能打仗）|

### 万世之功的三个特征

```
1. 时间跨度长：贯穿整个项目/战争
2. 系统支撑性：为他人的成功创造条件
3. 危机时不可缺失：失败时仍能提供支持
```

**萧何的三次"万世之功"**：

1. **初期建制**：为汉军建立法律/制度框架
2. **中期补给**：持续从关中补充粮草与兵员
3. **关键时刻**：彭城大败后补充数万兵，使刘邦得以继续作战

## 理论框架三：功劳评定三维度

### 评定公式

```
功劳得分 = 战略价值 × 系统价值 × 持续价值 × 贡献规模

战略价值：这个贡献对最终目标有多关键？（1-10分）
系统价值：这个贡献是否创造了可持续的系统？（1-10分）
持续价值：这个贡献维持了多长时间？（1-10分）
贡献规模：直接影响了多少人/多少资源？（1-10分）
```

**萧何 vs 曹参计算**：

| 维度 | 萧何 | 曹参 | 说明 |
|------|------|------|------|
| 战略价值 | 10 | 8 | 萧何决定了刘邦能打多久 |
| 系统价值 | 10 | 4 | 萧何建立了关中后勤系统 |
| 持续价值 | 10 | 5 | 萧何全程7年；曹参每次战役 |
| 贡献规模 | 9 | 9 | 两者均很大 |
| **总分** | **9000** | **1440** | 萧何约6倍于曹参 |

## 双轨制评价体系

### 核心设计

**问题**：萧何第一（荣誉），但封户不如曹参（物质）→ 如何平衡？

**解决方案**：精神荣誉 ≠ 物质奖励，分开评定

```
双轨制评价：
┌─────────────────────────────────┐
│ 轨道1：精神荣誉体系（排名/头衔）│
│   → 按战略价值/系统价值排名      │
│   → 萧何第一（功人优先）         │
├─────────────────────────────────┤
│ 轨道2：物质奖励体系（封户/金钱）│
│   → 按实际贡献规模/直接战功量化  │
│   → 曹参封户最多（10600+户）     │
└─────────────────────────────────┘
```

### 双轨制设计原则

| 原则 | 说明 | 操作 |
|------|------|------|
| 分离原则 | 荣誉≠金钱，可以不一致 | 第一名≠最高薪 |
| 透明原则 | 两个标准都公开说明 | 告诉众将为何萧何第一 |
| 补偿原则 | 在一轨道低的，在另一轨道补偿 | 萧何荣誉高→物质相对低；曹参物质高 |
| 激励原则 | 两类人都得到认可 | 功人和功狗都有奔头 |

### 特殊加分项

**案例：萧何剑履上殿**

> "今萧何未尝有汗马之劳，徒持文墨议论，不战，顾反居臣等上，何也？"

刘邦的加分逻辑：
- 基础分（战功）：第一
- 加分项：保全刘氏家族/关中百姓 → 超出职责范围的贡献
- 特殊标记：剑履上殿，入朝不趋（制度性荣誉）

**现代类比**：年度特别贡献奖，区别于常规绩效奖

## 四级功劳分级

### 分级标准

```
第一级：战略支撑型（功人中的功人）
  定义：为整场战争/项目提供不可或缺的系统性支撑
  特征：全程参与+系统建设+危机救援
  案例：萧何（关中后勤/法制建设/持续补兵）
  排名：第一位，精神荣誉最高

第二级：全程参战+独立贡献型（功人中的功狗）
  定义：从头到尾参与，且有独立、可量化的重大贡献
  特征：全程+多次关键战役胜利
  案例：曹参（身被七十创，攻城略地最多）
  排名：第二位，物质奖励可第一

第三级：关键时刻特殊贡献型（关键功狗）
  定义：在关键转折点有独特、无可替代的单次贡献
  特征：时机关键+贡献独特+可能非全程
  案例：张良（六奇计，汉中对策，鸿门宴）
  排名：视贡献重要性

第四级：局部战功型（普通功狗）
  定义：在某一具体战役/任务中有功劳
  特征：范围有限+可替代性较高
  案例：普通将领的攻城拔寨
  排名：按战功量化排名
```

### 分级决策树

```
Q1：是否全程参与（超过50%时间）？
  否 → 第三/四级（看关键性）
  是 → Q2

Q2：是否建立了可持续的系统（后勤/制度/网络）？
  否 → 第二级
  是 → Q3

Q3：系统是否在危机时仍发挥作用（别人失败时仍支撑）？
  否 → 第二级
  是 → 第一级（战略支撑型）
```

## 三种量化方法

### 方法A：战功型量化（KPI法）

**适用**：可量化的直接战果（攻城数/斩首数/行军里程）

**操作**：

```
曹参量化：
- 攻克城市：2个国，122个县（《曹相国世家》）
- 斩首/俘虏：柴将军以下27人
- 身被七十创：参与战役数量的体征

量化结果 → 封户数量：按贡献比例分配
```

**局限**：只能量化功狗，无法量化功人的系统价值

### 方法B：智谋型评估（定性法）

**适用**：难以量化的战略贡献（计谋/制度/文化）

**操作**：

```
萧何定性评估：
- 入咸阳第一时间：收秦律法图籍（前瞻性）
- 举荐韩信：一个人才决定了战争走向
- 守关中七年：数万兵员+持续粮草供应
- 法制建设：汉律基础

→ 无法精确量化，但定性评估：每项都是"不可或缺"
→ 功劳第一（定性结论）
```

**操作步骤**：
1. 列举所有贡献（不漏项）
2. 每项评估"如果没有这个贡献，结果会如何"（反事实）
3. 找出"不可替代的贡献"→ 权重最高
4. 综合定性排名

### 方法C：反事实推理法（假设法）

**适用**：评估某人缺失对整体的影响

**操作模板**：

```
"如果没有[某人]，[具体情景]会如何？"

萧何案例：
"如果没有萧何守关中，彭城大败后汉军会如何？"
→ 无粮草补充 → 无法重组 → 可能直接覆灭

"如果没有曹参攻城，汉军会如何？"
→ 扩张减慢，但仍可找其他将领代替

结论：萧何不可替代性 > 曹参不可替代性
```

**五步反事实评估**：

1. 假设该人不存在
2. 描述最可能的替代方案（谁来做/什么替代品）
3. 评估替代方案的效果差距
4. 量化差距（10%/50%/100%损失）
5. 差距越大 → 功劳越高

## 现代应用

### 场景1：销售 vs 研发的评价困境

| 职能 | 类比 | 常见评价偏差 |
|------|------|------------|
| 销售 | 曹参（功狗）| 数据好看，易拿奖金 |
| 研发/产品 | 萧何（功人）| 贡献难量化，常被低估 |
| 后台支撑 | 萧何（功人）| 最容易被忽视 |

**双轨制解决方案**：
- 荣誉评选：战略贡献型（研发/产品/后台）→ 组织奖/最佳团队奖
- 物质奖励：直接业绩型（销售/前线）→ 提成/绩效奖金
- 特别贡献奖：关键时刻救场者（危机处理/关键突破）

### 场景2：论文署名排序

**功劳分级对应**：

```
第一作者 = 直接执行者（功狗）：做实验、写论文
通讯作者 = 战略规划者（功人）：设计方向、提供资源
共同一作 = 全程参与+独立贡献者
致谢 = 局部贡献者（第四级）
```

**反事实法评估署名**：
- "如果没有这个人，这篇论文能发表吗？" → 不能 → 一作/通讯
- "如果没有这个人，论文会差很多吗？" → 差很多 → 共同一作
- "如果没有这个人，论文基本不变" → 致谢

### 场景3：组织绩效评价

**四级分级对应**：

| 员工类型 | 分级 | 评价方法 | 奖励设计 |
|---------|------|---------|---------|
| 战略规划者 | 第一级 | 定性+反事实 | 高级职称+期权 |
| 核心执行者 | 第二级 | KPI量化 | 高奖金+晋升 |
| 关键时刻贡献 | 第三级 | 事件分析 | 特别奖 |
| 日常执行 | 第四级 | KPI量化 | 绩效奖金 |

## 五大核心原则

### 原则1：识别功人，给予精神第一

**执行**：
- 评奖时明确说明："我们的第一名是战略贡献型"
- 向众人解释功人优先的逻辑
- 不要让"最显眼的成功"自动成为"第一名"

### 原则2：双轨制，两类人都有奔头

**执行**：
- 设计两个独立的评价轨道
- 功人型（战略/系统/规划）→ 荣誉轨道第一
- 功狗型（执行/量化/直接成果）→ 物质轨道第一
- 向所有人透明解释两个轨道

### 原则3：用反事实法评估不可替代性

**执行**：
- 评定每个人的功劳时，问："如果他不在，会差多少？"
- 答案决定了真实的功劳分级
- 越不可替代 → 越高分级

### 原则4：区分万世之功与一时之功

**执行**：
- 持续时间：全程 > 部分参与
- 系统建设：建立系统 > 使用系统
- 危机价值：危机时仍有效 > 顺风时有效

### 原则5：公开评定逻辑，不只公布结果

**执行**：
- 公布时说明评定标准
- 预防性解释：为何功人排第一（避免众将不满）
- 允许申诉，有据可查

## 决策清单

**评定功劳时**：
- [ ] 识别谁是功人（规划者/系统建设者）
- [ ] 识别谁是功狗（执行者/直接战功）
- [ ] 用反事实法评估不可替代性
- [ ] 区分万世之功与一时之功
- [ ] 设计双轨制（荣誉+物质分开）
- [ ] 公开解释评定逻辑

**设计组织评价体系时**：
- [ ] 是否有功人的评价轨道？
- [ ] 是否有功狗的评价轨道？
- [ ] 两个轨道是否都有足够吸引力？
- [ ] 反事实法是否纳入评估？
- [ ] 万世之功（全程系统性贡献）是否有权重加成？
