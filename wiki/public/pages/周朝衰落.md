# 周朝衰落

周朝衰落，是《史记·三代世表》记载的西周由盛转衰的历史进程，从昭王"南巡不返"到厉王出奔、共和行政，最终导致西周崩溃、平王东迁的全过程。

## 周朝天子世系与盛衰

| 序号 | 天子 | 在位特征 | 关键事件 |
|------|------|---------|---------|
| 武王发 | 开国 | - | 克商，建周 |
| 成王诵 | 幼年继位 | 周公辅政 |
| 康王钊 | **"刑错四十余年"** | 成康之治 |
| 昭王瑕 | 南巡不返 | **讳之（隐讳死因）** |

| 懿王坚 | 周道衰 | **诗人作刺** |
| 厉王胡 | "以恶闻过" | **出奔，死于彘** |
| 共和 | 二伯行政14年 | 周定公、召穆公执政 |
| 宣王静 | 中兴尝试 | 宣王中兴 |
| 幽王宫湦 | 烽火戏诸侯（传说） | **西周灭亡** |

## 成康之治

成王、康王是周朝鼎盛时期。

"成康之际，天下安宁，刑错四十余年不用。"

"刑错四十余年"——刑罚四十余年闲置不用，无须动用，是真正的盛世标志（类比汉代文景之治的"与民休息"）。

## 昭王之死：转折点

**事件**：昭王南巡，"南巡不返"——在南方死亡。

**"不赴，讳之"**：诸侯不报丧，朝廷隐讳死亡原因。

昭王死亡原因历来成谜：
- 可能被南方诸侯（楚）袭杀
- 可能在渡汉水时船只被破坏（"胶舟"之说）
- 朝廷"讳之"暗示死因不光彩

**历史意义**：周朝由盛转衰的转折点。南方诸侯不来报丧，标志离心加剧。

## 穆王时期：霸权衰退

**征伐西方**：穆王曾亲征西戎，作《甫刑》（吕刑）修订刑法。

**"荒服不至"**：远方诸侯不来朝贡，标志周王室实际控制力下降。

## 懿王时期：诗人批评

**"周道衰"**：懿王时代周朝衰落加剧。

**"诗人作刺"**：《诗经》中开始出现讽刺批评周朝政治的诗篇（如《大雅》《小雅》中的怨刺之作）。

诗人作刺意味着：公开批评政治成为风气，道义合法性下降。

## 厉王出奔：权威崩溃

### 厉王之政

厉王好利，任用荣夷公，垄断山川之利。

大夫芮良夫谏："王室其将卑乎？夫荣公好专利而不知大难。夫利，百物之所生也，天地之所载也，而或专之，其害多矣。天地百物皆将取焉，何可专也？所怒甚多，而不备大难，以是教王，王其能久乎？"

厉王不听，专利益深，"诗人作刺，王怒，得卫巫，使监谤者，以告则杀之"。

**防止批评**："国人莫敢言，道路以目"（路上相遇只用眼神交流，不敢说话）。

厉王自以为成功："吾能弭谤矣"（我能压制批评了）。

召公谏："是障之也。防民之口，甚于防川。川壅而溃，伤人必多，民亦如之……天子听政，使公卿至于列士献诗，……然后王斟酌焉。是以事行而不悖。今王塞民之口，其与能几何？"

### 国人暴动（前841年）

厉王压制言论三年后，"国人相与畔，袭厉王。厉王出奔于彘。"

这是中国历史上有明确纪年的最早政治事件——共和元年（前841年）。

## 共和行政（前841—前828年）

厉王出奔，太子靖被国人追索，召公以己子代之，藏匿太子。

**二伯执政**：周定公（周公）、召穆公（召公）二人共同执政，称"共和"，14年。

共和是中国历史进入有明确纪年的起点（《史记》以前841年为纪年开始）。

## 宣王中兴与西周末年

宣王继位（前827年），曾试图中兴，"宣王中兴"，但未能从根本上解决问题。

幽王时（前781—前771年），宠褒姒，废太子，最终被犬戎与申侯联军攻破镐京，幽王被杀，西周灭亡。

平王东迁（前770年），东周开始，进入春秋时代。

## 衰落规律

三代历史表明的衰落规律：

1. **三代定律**：开国后第三代往往出现衰落迹象（夏太康、商九世之乱、周昭王）
2. **防民之口**：压制批评必然导致政治危机的集中爆发
3. **专利自误**：利用权力垄断财富，必然失去民心
4. **谏言的价值**：厉王时代多次良谏（芮良夫、召公）均不被采纳
