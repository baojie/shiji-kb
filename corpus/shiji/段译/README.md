# 史记 段译（原文+白话对照）

本目录存放《史记》130 篇的**原文-白话段译对照**文本。

## 来源

- **项目**：hunterhug/china-history
- **仓库**：<https://github.com/hunterhug/china-history>
- **源路径**：`china-history/史记/{十二本纪,十表,八书,三十世家,七十列传}/*-段译.html`
- **导入方式**：`scripts/import_vernacular_shiji.py` 解析源 HTML，提取交替的
  `<p>` 原文 与 `<p style="color:#967d63;">` 白话译文段，逐段保留对应关系。

## 文件组织

- 130 个文件，命名为 `NNN_章节名_段译.txt`，编号与 `chapter_md/NNN_*.tagged.md` 对齐：
  - `001-012`：本纪（十二本纪）
  - `013-022`：表（十表）
  - `023-030`：书（八书）
  - `031-060`：世家（三十世家）
  - `061-130`：列传（七十列传）

## 文件格式

```
# NNN 章节名

【原文】
（文言段落）

【译文】
（对应白话译文）

---

【原文】
（下一段）
...
```

- 段间以 `---` 分隔
- 每个 `【原文】` 紧跟一个 `【译文】`

## 更新机制

当源仓库更新时，重新运行：

```bash
python scripts/import_vernacular_shiji.py
```

脚本会覆盖重新生成 130 个文件。
