## 曲沃代晋

### 起因
- 昭侯元年，封文侯弟成师于曲沃，号桓叔
- 曲沃邑大于翼（晋君都邑）
- 桓叔年五十八，好德，晋国之众皆附焉
- 君子曰："末大于本而得民心，不乱何待！"

### 冲突过程
| 年份 | 事件 |
|------|------|
| 昭侯七年 | 潘父弑昭侯，迎桓叔，桓叔败归曲沃 |
| 孝侯十五年 | 曲沃庄伯弑孝侯于翼 |
| 哀侯九年 | 曲沃武公虏哀侯于汾旁 |
| 小子四年 | 曲沃武公诱杀小子侯 |
| 晋侯缗二十八年 | 曲沃武公灭晋侯缗 |

### 结果
- 周釐王命曲沃武公为晋君，列为诸侯
- 自桓叔初封曲沃以至武公灭晋，凡六十七岁
- 武公代晋二岁而卒，子献公诡诸立
