| 节/字段 | 桓齮 | 庞暖 | 李牧 | 问题 |

| 所属国 | ✓（秦将） | ✓（赵将） | ✓ | OK |
| 对照表·阵营 | ✓ | ✓ | ✓ | OK |
| 对照表·关键战役 | ✓ | ✓ | ✓ | OK |
| 对照表·结局 | ✓ | ✓（未明） | ✓ | 庞暖结局不详 |
| 历史评析 | ✗（桓齮无） | ✓（简） | ✓ | 小人物可无 |

- 战国将领页面的核心叙事轴是"战役 → 胜负 → 结局"

- 合纵将领（庞暖）活跃于多国，应在对照表加"主要活跃诸侯国"行

**提案**：新建 `skills/templates/战国将领.md`（下轮执行）
