## 🎯 处理后的效果

### 示例展示

**处理前**（原文）:
```
袁盎者，楚人也，字丝。父故为群盗，徙处安陵。
```

**处理后**（标注版）:
```markdown
[1.1] @袁盎@者，&楚&人也，字@丝@。

[1.2] 父故为群盗，徙处=安陵=。
```

### 标注类型

- `@人名@` - 人物（如：@袁盎@）
- `=地名=` - 地点（如：=安陵=）
- `$官职$` - 官职（如：$丞相$）
- `~族群~` - 民族（如：~匈奴~）
- ... 等11种实体标注
