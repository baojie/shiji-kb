## 标注规则

### 11类实体标注

| 类型 | 符号 | 示例 | CSS类 |
|------|------|------|-------|
| 人名 | `@...@` | `@刘邦@` `@萧何@` | person |
| 地名 | `=...=` | `=长安=` `=咸阳=` | place |
| 官职 | `$...$` | `$丞相$` `$将军$` | official |
| 时间 | `%...%` | `%元年%` `%春%` | time |
| 朝代 | `&...&` | `&汉&` `&秦&` | dynasty |
| 族群 | `~...~` | `~匈奴~` | tribe |
| 制度 | `^...^` | `^礼^` `^法^` | institution |
| 器物 | `*...*` | `*金*` `*车*` | artifact |
| 动植物 | `🌿...🌿` | `🌿马🌿` | flora-fauna |
| 引文 | `"..."` | 对话引用 | quoted |
| 注释 | `> [NOTE]` | 太史公曰 | blockquote |
