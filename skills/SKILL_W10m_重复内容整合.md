# SKILL W10m: 重复内容整合（H12）

> "整页重复是 H1 的任务，节级重复是 H12 的任务。区别在粒度，不在原则：宁愿保留，不得删除有价值的内容。"

---

## 一、何时执行

| 触发场景 | 优先级 |
|---|---|
| 同一段引文出现在两个不同页面 | P1 |
| 两个概念页描述同一事物的不同方面（可互补） | P1 |
| W9 扫描发现跨页重复节 | P1 |

**与 H1 的区分**：
- H1（W10a）：整页重复（Jaccard 相似度 > 0.4）→ 整页去重合并
- H12（本文）：节级重复（特定 `##` 节内容重复）→ 节内整合

## 二、发现候选（扫描方法）

```bash
# 查找跨页重复的引文（长度 > 20 字）
python3 -c "
import glob, re
quotes = {}
for f in glob.glob('wiki/public/pages/*.md'):
    content = open(f).read()
    # 提取史记引文节的引文（> > 格式）
    for q in re.findall(r'^> (.{20,})', content, re.MULTILINE):
        q_norm = re.sub(r'\s', '', q)
        if q_norm not in quotes:
            quotes[q_norm] = []
        quotes[q_norm].append(f)

for q, files in quotes.items():
    if len(files) > 1:
        print(f'重复: {q[:30]}...')
        for f in files:
            print(f'  {f}')
"
```

## 三、处理流程

### 情形A：同一引文出现在两页

```
判断：两页都真正需要这条引文吗？

是（两页主题都高度相关）→ 两页均保留，加交叉链接说明来源重叠
否（一页是主要讨论者，另一页是顺带提及）→ 次要页保留引文，在引文后追加交叉链接注记
```

**⛔ 禁止操作**：不得删除次要页的 `## 史记引文` 节或其中的任何引文段落。

**✅ 正确操作**（追加注记，不删除）：

```bash
# 1. 确认哪页是主要讨论者
# 2. 在次要页引文节末尾追加一行交叉链接注记（不删除原引文）
# 3. 使用 edit_page.py 修改次要页
python3 wiki/scripts/butler/edit_page.py "次要页" /tmp/updated.md \
    --summary "w10m: H12 引文重复标注，次要页保留引文并加交叉链接注记" \
    --author "butler"
```

修改后的次要页引文节示例（**原引文保留，末尾追加注记**）：

```markdown
## 史记引文

> 原文引文内容……（保留不动）

（此引文亦见 [[主要页]] §史记引文节，该页为主要讨论处。）
```

### 情形B：两个概念页描述同一事物的互补方面

```
评估：是否可以合并为一个更完整的页面？

可以合并 → 判断哪页更适合作为规范页 → 进入 H1（W10a）流程
不适合整页合并（各有独特角度）→ 在两页的 ## 参见 节互链，说明互补关系
```

**互链操作**：

```bash
# 在页面A 的 ## 参见 节加入
# - [[页面B]]：从另一角度讨论同一主题
```

## 四、成功标准 / 完成条件

- [ ] 重复引文处理后，至少有一页保留原文，另一页改为交叉链接
- [ ] 两页互补的概念页均有 `## 参见` 节互链
- [ ] 未删除任何有价值的独有内容
- [ ] 整页重复的情形转移给 H1（W10a）处理
- [ ] diff ≤ 20 行/页

## 六、铁律：不得删除有价值内容

```
❌ 禁止：删除次要页的 ## 史记引文 节或其中任何引文段落
❌ 禁止：把次要页引文替换为"相关引文见[[主要页]]"（内容消失）
✅ 正确：两页引文重复时，次要页保留原引文，末尾追加交叉链接注记
✅ 正确：两页都有独特角度 → 互链，两份内容都保留
```

**例外**：整页去重（H1/W10a）和精品页升级（W2 premium-upgrade / W10u）属于授权范围，可能涉及内容迁移。H12 不在此例外内。
