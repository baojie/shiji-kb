# SKILL 07d: 溯源推理 — 史料来源可信度评估

> 追溯太史公司马迁获取历史信息的渠道，评估事实与事件记载的史源可信度。

---

## 一、核心问题

**太史公怎么知道的？**

对于《史记》中的每一个事实、事件和细节，推断其信息来源：
- 是档案文献？还是口述传闻？
- 是第一手材料？还是辗转相传？
- 是可靠史源？还是推测补白？

---

## 二、溯源推理框架

### 2.1 史源类型分类

| 史源类型 | 定义 | 可信度 | 识别特征 |
|---------|------|--------|---------|
| **档案文献** | 国家档案、诏令、奏疏、典籍 | 最高(A) | 太史公明确引用书名、档案 |
| **家传文献** | 世家族谱、私人记录 | 高(B) | "太史公曰"引"某家之传" |
| **目击口述** | 当事人或目击者口述 | 中高(B-C) | 太史公游历访谈记录 |
| **二手传闻** | 民间流传、道听途说 | 中(C) | "世传""闻之""人言" |
| **逻辑推测** | 太史公根据其他事实推断 | 中低(C-D) | 细节过于戏剧化、对话完整 |
| **文学虚构** | 为增强叙事而补充 | 低(D-E) | 私密对话、心理描写 |

### 2.2 推理依据层次

#### 第一层：明确标注
- 太史公明确说"见某书""读某传""访某人"
- 例：《史记·太史公自序》列举所依据的典籍
- **检测方法**：文本匹配"余读""余闻""见于"等模式

#### 第二层：时空可达性
- 太史公本人是否可能知道（时间 + 地理）
  - 汉代事件 → 可能有档案/口述
  - 战国事件 → 依赖文献传世
  - 上古事件 → 必定依赖更早文献
- **检测方法**：
  - 计算事件年份与太史公生年(约前145)的时间差
  - 查询太史公游历路线是否经过事件发生地

#### 第三层：信息性质分析
**公开事实 vs 私密细节**
- 公开：会盟、战争、诏令 → 可能有档案
- 私密：密谋对话、闺房之言 → 来源存疑

**宏观叙事 vs 微观细节**
- 宏观：某年某战 → 可查档案
- 微观："项王乃悲歌慷慨" → 如何得知？

**检测方法**：LLM判断事实的公开性与细节程度

#### 第四层：交叉验证
- 同一事件在不同章节的记载是否一致
- 利用已有的**矛盾检测**成果(SKILL_07 二节)
- 一致性高 → 可能有共同可靠史源
- 矛盾大 → 可能来源不同或推测成分多

#### 第五层：结构化知识辅助
**实体关系**：当事人与太史公是否有传承链
- 例：某人后代在汉朝为官 → 可能有家传
- 检测方法：查询人物后代关系链

**地点关系**：太史公游历路线是否经过
- 检测方法：地点匹配+游历记录

**时间关系**：事件距太史公时代多远
- 检测方法：事件年代索引计算时间差

## 三、置信度分级

| 等级 | 定义 | 史源推断 | 典型特征 |
|------|------|---------|---------|
| A | 明确档案/文献引用 | 太史公明确注明出处 | "余读《某书》""见《某传》" |
| B | 高度可能有可靠史源 | 公开事件+时间近+有传承链 | 汉代官方事件、有后代传承 |
| C | 可能有史源但存疑 | 时间较远或细节可疑 | 战国私密对话、戏剧化细节 |
| D | 来源不明或推测成分大 | 私密细节+无法验证 | 不可能知道的心理描写 |
| E | 几乎肯定是补白/虚构 | 不可能知道的细节 | 上古神话色彩、完整私密对话 |

## 四、溯源推理流程(Agent通用方法)

### 4.1 输入
- **事实/事件**：待评估的史料记载
- **上下文**：章节、原文引用、涉及实体
- **结构化知识**：事件索引、实体关系、矛盾检测结果

### 4.2 执行步骤

```
1. 提取基本信息
   - 事件时间(公元纪年)
   - 涉及人物/地点
   - 信息性质(公开/私密、宏观/微观)

2. 明确引用检查
   - 原文是否有"余读""见于"等标注
   - 三家注是否注明出处
   → 若有明确引用 → 史源置信度A

3. 时空可达性分析
   - 时间差 = 事件年份 - 太史公生年(约-145)
     - 时间差 < 50年 → 可能有口述/档案 → 加分
     - 时间差 > 500年 → 必依赖古文献 → 存疑
   - 查询太史公游历路线
     - 若到过事件发生地 → 可能有实地考察 → 加分

4. 信息性质判断(LLM)
   - 是公开事件还是私密对话？
   - 是宏观事实还是微观细节？
   - 细节是否过于戏剧化？
   → 私密+微观+戏剧化 → 降级

5. 交叉验证检查
   - 查询SKILL_07矛盾检测结果
   - 同一事件在其他章节记载是否一致
   → 一致性高 → 加分；矛盾大 → 降级

6. 传承链分析
   - 查询事件参与者后代
   - 是否在汉朝为官？是否可能有家传？
   → 有传承链 → 加分

7. 综合评级
   - 汇总各层证据
   - 输出史源置信度等级(A/B/C/D/E)
   - 输出推理依据说明

8. 生成溯源报告
   - 史源类型判断
   - 置信度等级
   - 支持/质疑的证据列表
   - 可用的结构化知识引用
```

### 4.3 输出
- **史源类型**：档案文献/家传文献/目击口述/二手传闻/逻辑推测/文学虚构
- **置信度等级**：A/B/C/D/E
- **推理依据**：列表形式的证据链
- **可用知识**：引用的结构化知识(事件ID、实体ID、关系链等)

## 五、典型案例：鸿门宴

### 案例概述
- **事件**：鸿门宴上的私密对话与细节
- **章节**：007项羽本纪
- **时间**：前206年
- **核心问题**：太史公如何得知宴会上的详细对话？

### 溯源推理过程

#### Step 1: 基本信息
- 时间：前206年
- 时间差：前206 - (-145) = 约61年
- 人物：项羽、刘邦、范增、张良、项伯、樊哙等
- 性质：**私密宴会**中的对话与暗斗

#### Step 2: 明确引用检查
- 原文无"余读""见于"等标注
- 三家注亦无明确史源引用
→ 无明确文献出处

#### Step 3: 时空可达性
- 时间差约61年 → 当事人后代可能在世
- 太史公游历是否到过鸿门？《自序》未明确提及

#### Step 4: 信息性质(LLM判断)
- **私密对话**：宴会内幕、项羽与范增眼色示意
- **戏剧化细节**："范增数目项王""项王默然不应"
→ 这些细节如何得知？

#### Step 5: 交叉验证
- 《高祖本纪》(008)亦载鸿门宴，但细节略有不同
- 矛盾程度：低，整体一致
→ 可能有共同史源

#### Step 6: 传承链分析
关键人物后代在汉朝情况：
- **张良**：汉初功臣，子孙封侯 → **可能有家传口述**
- **项伯**：项羽叔父，后降汉封射阳侯 → **极可能是重要史源**
- **樊哙**：汉初将军，有后人 → 可能提供樊哙视角
- **项羽**：自刎，但部下有降汉者
- **范增**：中途离开项羽，后代不详

**关键发现**：
- 项伯是宴会核心参与者，后归汉封侯
- 张良是刘邦方关键人物，汉朝显赫
→ **鸿门宴细节很可能来自项伯和张良的家传口述**

#### Step 7: 综合评级

| 要素 | 史源推断 | 置信度 |
|------|---------|--------|
| **鸿门宴发生** | 多源记载+时间近 | A |
| **基本经过** | 有传承链(项伯、张良) | B |
| **具体对话** | 传闻+文学加工 | C-D |
| **范增眼色细节** | 不可能确知/推测补白 | D |

#### Step 8: 溯源报告

```json
{
  "事件": "鸿门宴",
  "事件ID": "007-鸿门宴",
  "史源类型": ["家传口述(项伯、张良)", "二手传闻", "文学加工"],
  "整体置信度": "B-C",
  "分层评估": {
    "核心事实(鸿门宴发生)": {
      "置信度": "A",
      "理由": ["多章节记载一致", "时间距太史公仅61年", "有多个传承链"]
    },
    "基本经过(樊哙闯帐、张良斡旋)": {
      "置信度": "B",
      "理由": ["项伯封侯可能有家传", "张良后代显赫", "参与者众多难以完全虚构"]
    },
    "具体对话内容": {
      "置信度": "C-D",
      "理由": ["私密对话难以确知", "可能基于传闻润色", "符合太史公文学叙事风格"]
    }
  },
  "可用知识": [
    "人物关系: 项伯(项羽叔父) → 降汉封射阳侯",
    "人物关系: 张良 → 汉初三杰,留侯",
    "时间索引: 前206年,距太史公约61年",
    "矛盾检测: 007 vs 008 该事件记载基本一致"
  ],
  "研究价值": "典型的'核心事实可信+细节存疑'案例,展示太史公依据家传口述重建历史场景的方法"
}
```
