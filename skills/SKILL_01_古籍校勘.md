## 一、工序位置

```
archive/chapter/*.txt        底本（网络来源，可能有错）
        ↓ 校勘（SKILL_01b_多版本互校底本）
collation_base/*.txt         定本（多版本互校后）
        ↓ 同步到
archive/chapter/*.txt        定本（更新后的底本）
        ↓ 切分+编号（SKILL_古籍章节切分与编号）
chapter_md/*.tagged.md       标注文本
        ↓ 实体标注（SKILL_03a_实体标注）
        ↓ 完整性维护（SKILL_01a）← 确保标注未篡改原文
        ↓ 事件提取 ...
```

**核心原则：** 校勘只改原文txt，不改标注文件。标注文件应忠实反映定本。

**目录说明**：
- `archive/chapter/*.txt` — 原始底本（网络来源），校勘后更新为定本
- `collation_base/*.txt` — 多版本互校生成的定本（SKILL_01b 输出）
- `collation_reports/*.md` — 各章校对报告（记录异文判定过程）

**子技能**：
- [SKILL_01a_标注完整性维护](SKILL_01a_标注完整性维护.md) — 标注阶段的文本完整性验证与修复
- [SKILL_01b_多版本互校底本](SKILL_01b_多版本互校底本.md) — 多版本对校建立标准底本
- [SKILL_01c_表格校勘规范](SKILL_01c_表格校勘规范.md) — 年表/世家类章节的表格校勘规范
- [SKILL_01d_正音与拼音标注](SKILL_01d_正音与拼音标注.md) — 古音标注与拼音处理
- [SKILL_01e_繁简体处理与阅读切换](SKILL_01e_繁简体处理与阅读切换.md) — 繁简转换与阅读界面功能

## 三、常见底本错误类型

### 3.1 网站爬取垃圾
- 整章内容被替换为网站索引（如022章"中秋/怀古/爱国"）
- 混入HTML标签、导航文字

### 3.2 缺字与乱码
- 原文缺字显示为`？`（如`季？`→`季历`）
- PUA私用区字符（如`{单心}`→`惮`）

### 3.3 衍文与脱文
- 衍文（多余重复）：`蹇叔止蹇叔止臣`→`蹇叔止臣`
- 脱文（遗漏）：`马`→`马騺`

### 3.4 现代编辑附加
- 公元纪年数字（206、205...）不属于原文
- `公元前` 表头

### 3.5 异体字
- 原文使用古字/异体字时**保持原貌**，不做繁简转换
- 例：`太山`不改为`泰山`，`後`不改为`后`，`彊`不改为`强`

---

## 四、校勘原则

1. **保持原貌** — 异体字、古字、通假字保留，不做现代化
2. **有据可查** — 每处校改应有版本依据（ctext或其他权威版本）
3. **标注不改原文** — 标注过程中发现原文有误，应修正txt底本，而非在标注文件中悄悄改
4. **分段不改字** — 切分段落时只改换行位置，不改任何文字（包括引号层级`"`/`'`、标点`；`/`。`等）

## 五、校勘工具

### 5.1 lint完整性检查
```bash
python scripts/lint_text_integrity.py          # 全部130章
python scripts/lint_text_integrity.py 005 006  # 指定章节
```

**详细流程请参见子技能**：[SKILL_01a_标注完整性维护](SKILL_01a_标注完整性维护.md)

### 5.2 白名单
已确认的正确校勘记录在 `scripts/lint_whitelist.txt`，lint时自动跳过。

格式：`章节号<TAB>原文片段<TAB>标注片段<TAB>说明`

### 5.3 校勘参照

**参照版本优先级**：
1. **维基文库史记** (`archive/wikisource_shiji/*.html`) — 繁体，基于中华书局本，校勘质量高
2. **原始底本** (`archive/chapter/*.txt`) — 简体，网络来源
3. **在线资源** — ctext.org、国学网（辅助）

## 七、字典关联

校勘过程中遇到的古字、异体字、通假字，应关联到字典资源，为读者提供查阅入口。

### 7.1 关联场景

| 场景 | 示例 | 字典用途 |
|------|------|---------|
| **异体字** | `後`/`后`、`彊`/`强` | 查明字形演变，确认是否同字 |
| **通假字** | `饹`通`奔`、`蚤`通`早` | 查明通假关系，辅助校勘判断 |
| **古字** | `埶`（势）、`戹`（阸） | 查明古今字对应 |
| **疑难字** | PUA字符、罕见字 | 查明字义和读音 |

### 7.2 字典资源

| 字典 | 用途 | 访问方式 |
|------|------|---------|
| [漢典](https://www.zdic.net/) | 字义/字形/异体字查询 | `https://www.zdic.net/hans/X` |
| [教育部異體字字典](https://dict.variants.moe.edu.tw/) | 台湾教育部异体字权威 | 在线查询 |
| [說文解字](https://zh.wikisource.org/wiki/說文解字) | 许慎原典 | 维基文库 |
| [康熙字典](https://zh.wikisource.org/wiki/康熙字典) | 字形收录最全 | 维基文库 |

### 7.3 渲染集成

在HTML渲染中，校勘异体字可添加字典链接tooltip：

```html
<span class="variant-char" title="異體字：後，通行字：后">
  後
  <a href="https://www.zdic.net/hans/後" class="dict-link" target="_blank">📖</a>
</span>
```
