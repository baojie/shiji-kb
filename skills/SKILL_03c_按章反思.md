# SKILL 03c: 按章反思 — 实体标注逐章审查

> **使用方式**：复制 §一 提示词，替换 `NNN`/`章名`，提供给 Agent 执行。
> **适用场景**：单章语境误判、遗漏实体、章节特有错误。跨章系统性错误见 SKILL_03e_按类型反思.md。

---

## 一、按章反思提示词（即用）

```
你是《史记》实体标注质量审查员。请对章节 NNN_章名 的已标注文件执行按章反思。

## 任务说明

读取文件：chapter_md/NNN_章名.tagged.md

审查对象是**已标注文件**，不是原文重新提取——关注"标错了什么"和"漏标了什么"。
不修改原文字符，只修改标注符号。

---

## Step 1：格式扫描（旧格式残留）

列出所有出现位置：
- `〘X〙` → 应改为 `〖+X〗`（旧生物格式）
- `@X@`、`=X=` → 对应新格式（v1.0 对称格式）
- `〖=、〗`、`〖=。〗` → 去掉（句读符号误入标注）
- `<span style="...">` → 对应新格式（v1.0 HTML内联样式）

**格式破损**（AI生成时标注边界错误，用 grep `〖[^〗]*[。，；：]` 检测）：
- `孝王〖@。〗孝王` → 标注内含句读，改为 `〖@孝王〗。〖@孝王〗`
- `以相太〖@子而立〗之` → 动词短语误入标注，去标注
- 标注跨越多个词/短句 → 拆解或去标注

---

## Step 2：现有标注审查

逐段阅读。对每个标注 `〖TYPE X〗`/`《X》`/`〈X〉`/`【X】`/`〔X〕`/`〚X〛` 判断类型和边界是否正确。

### 类型误标速查

**官职 `〖;〗` 误标（→身份/不标）**
| 误标 | 正确 |
|------|------|
| 天子/皇帝/太后/皇后 | `〖#身份〗` |
| 诸侯（泛称）/处士/今王 | `〖#身份〗` |
| 父/母/兄/弟 | 不标注（亲属称谓） |
| 三公（泛指制度等级时） | `〖^制度〗` |

**时间 `〖%〗` 误标（→数量/不标）**
| 误标 | 正确 |
|------|------|
| X里/X人/X众（距离/人数） | `〖$数量〗` |
| 年X十（年龄格式） | 不标注 |
| 数十年/千馀岁（时长） | 不标注 |
| 百里/二百里/三百里（五服距离） | `〖$数量〗` |

**制度 `〖^〗` 误标（→刑法/礼仪/天文/思想）**
| 误标 | 正确 |
|------|------|
| 五刑/流宥五刑/官刑/教刑/赎刑/典刑/五服（刑制） | `【刑法】` |
| 祭祀/朝觐（作为具体行为） | `〈礼仪〉` |
| 日月（"所照/所至/历X迎送"语境） | `〖!天文〗` |
| 土德/水德/木德等五行之德 | `〔思想〕` |
| 封禅（具体礼仪行为） | `〈封禅〉`（典章制度时保留`〖^〗`） |

**地名 `〖=〗` 误标（→其他/不标）**
| 误标 | 正确 |
|------|------|
| 邦国名称（秦/汉/齐等政治实体） | `〖'邦国〗` |
| 太阴/昊天等天象 | `〖!天文〗` |
| 九鼎等器物 | `〖*器物〗` |
| 甘誓/夏小正等古籍（"作X"/"传X"语境） | `《典籍》` |
| 大白旗/小白旗等旗帜名（颜色+旗） | `〖*器物〗`（非天文） |
| 白马（"乘白马"语境） | `〖+生物〗` |
| 山川/海/野（泛指地貌，非专名） | 不标注 |
| 东北/西南等方位副词 | 不标注 |
| "成都"（"三年成都"= 成为都邑） | 不标注 |
| "道X"中的X（道=疏导动词） | `〖=地名〗`（X本身是地名） |

**人名 `〖@〗` 误标（→氏族/官职/不标/拆分）**
| 误标 | 正确 |
|------|------|
| "姓X"/"氏X"中的X | `〖&氏族〗` |
| 昭明（"百姓昭明"= 形容词） | 不标注 |
| 夔夔/婆娑等叠字副词 | 不标注 |
| "以X为Y"中Y（任命语境） | `〖;官职〗` |
| 帝颛顼高阳/帝喾高辛（帝号+氏族号） | 拆分：`〖@帝颛顼〗〖&高阳〗` |
| 长子/少子（亲属关系描述） | 不标注 |
| 维王/不虞（代词/动词短语） | 不标注 |

### 边界正确性

| 错误类型 | 示例 | 修正 |
|---------|------|------|
| 数字前缀漏标 | `三百六〖$十六日〗` | `〖$三百六十六日〗` |
| 实体被截断 | `〖@黄〗帝` | `〖@黄帝〗` |
| 标点符号入标注 | `〖=、〗` | 去掉标注 |
| 标注吞噬后续文字 | `〖*器物〗，...后续文字〗` | 找到错位的`〗`并移正 |
| 同一词重复标注 | `〖@黄帝〗〖@黄帝〗` | 删除重复 |
| 人名+氏族号未拆分 | `〖@帝颛顼高阳〗` | `〖@帝颛顼〗〖&高阳〗` |

### 语境判断（同一词汇，不同含义）

| 词汇 | 语境A（标注） | 语境B（不标或改类型） |
|------|------------|-----------------|
| 成都 | `〖=成都〗`（地名，四川成都） | 不标（"三年成都"= 成为都邑） |
| 日月 | `〖!日月〗`（天体，"日月所照"） | `〖^日月〗`（礼制规范，极少） |
| 山川 | `〖=山川〗`（具体山水专名） | 不标（泛指地貌） |
| 封禅 | `〖^封禅〗`（典章制度） | `〈封禅〉`（具体礼仪行为） |
| 共工 | `〖@共工〗`（人名，独立提及） | `〖;共工〗`（官职，"以X为共工"任命语境） |

## Step 3：未标注文字遗漏检查

扫描无标注的文字，重点关注：

| 句型/场景 | 典型遗漏 |
|---------|---------|
| `以X为Y`（任命） | Y 通常是官职 `〖;Y〗` |
| `至于X` / `居于X` | X 通常是地名 `〖=X〗` |
| `姓X` / `氏X` | X 通常是氏族 `〖&X〗` |
| 段落内已出现标注的人名 | 同段后文的未标出现（修饰语位/嵌入主语位） |
| 禽兽鱼鸟/动植物 | 漏标为 `〖+生物〗` |
| 《X》曰/《X》云 | X 漏标为 `《典籍》` |
| 星宿/历法名词 | 漏标为 `〖!天文〗` |

**高频遗漏位置（AI 系统性弱点）**：
- 修饰语位：`尧大祖也`（= 尧的大祖，"尧"漏标）
- 嵌入主语位：`於是尧妻之二女`（"尧"漏标，前有"於是/乃/遂"）

## 完成后操作

> ⚠️ **核心约束：只改标注符号，不改原文字符。**
> 反思修正只允许增删 `〖〗《》〈〉【】〔〕〚〛` 标注符号，**原文汉字一个都不能删除**。
> 例："皆贤妇人"中若 `〖;妇人〗` 标注有误，正确操作是去掉标注符号还原为"妇人"，而不是删掉"妇人"二字。

1. **应用修正**：将 B/C 中所有修正应用到 `chapter_md/NNN_章名.tagged.md`
2. **格式验证（必须执行）**：运行 lint，确认无格式错误
   ```bash
   python scripts/lint_markdown.py chapter_md/NNN_*.tagged.md
   ```
   若 lint 报错，修复后再继续。
3. **写反思报告**：修正统计和特有发现写入 `doc/entities/按章实体反思报告.md`
4. **写回 SKILL**：D 中的新规律写入 SKILL_03c §三或 SKILL_03a 对应小节
5. **消歧补录**（可选）：若发现同章多个同名同字人物、省称对应不明确，记录到 `kg/entities/data/disambiguation_map.json`
```

## 三、已积累的章节特有规律

### 日月三义（001章首发现）

同一词"日月"在史记中有三种用法，务必看语境：
- `〖!日月〗`：天体（"日月所照"/"历日月而迎送之"）
- `〖%日月〗`：时间纪年（"日月不居"）
- `〖^日月〗`：礼制规范（极少，仅指"日月之行"礼法规定时）

凡"所照/所至/历X迎送"语境均为天体。

### 官职任命中的职位名

"以X为Y"句型中，Y 是被授予的职位（`〖;官职〗`），但 Y 同时可能在文中以人名独立出现（如"共工"）。须区分：任命语境中是官职，独立提及时可能是人名。
