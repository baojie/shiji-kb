# 模板：忠义殉主者

## 适用判断

满足以下任意 2 条即适用：
- tags 含 `忠义`、`烹杀`、`殉主`、`殉葬` 之一
- 人物最重要的历史贡献是以死/牺牲护主
- 史记记载集中在"被俘/临死/殉葬"的一个核心场景

**亚类**：
| 亚类 | 代表人物 | 殉主方式 |
|---|---|---|
| 战场代死 | 纪信 | 被烧杀（冒充刘邦投降） |
| 被俘拒降 | 周苛 | 被烹杀（骂项羽） |
| 自请殉葬 | 公子高 | 自杀（骊山陪葬） |
| 将相殉国 | 冯去疾、冯劫 | 自杀（不受辱） |

---

## 必有节（缺一考虑补充）

### 生平（必须）
- 简要背景：身份、归属、与君主关系
- **殉主场景子节**（必须独立标题，含原文引文）

### 殉主场景（必须独立节，不混入生平）

格式范式：

```markdown
### 殉主场景（前 XXX）

[背景一句话：战场态势/政治处境]

> [原文引文，包含人物最后的话或行动]（[[NNN_章节名|NNN-PPP]]）

[一句话叙述结果]
```

### 历史评析（建议）
- 殉主行为的道义分析（是被迫还是自愿？是愚忠还是大义？）
- 后人/史官评价

### 对照表（必须）

必有行：
| 项目 | 内容 |
|---|---|
| 所效忠主公 | [[XXX]] |
| 殉主场景 | [地点/事件] |
| 殉主方式 | 被烧/被烹/自杀/殉葬/… |
| 死前话语 | "…"（原文）/ 无记载 |
| 史记出现 | X 次 / N 篇 |
| 主要传记 | [[NNN_章节名]] |

### 相关人物（必须）

必须链接：
- 所效忠的主公
- 同场景其他人物
- **同类忠义殉主者**（互链：如纪信↔周苛↔冯去疾）

---

## 加分项

- ➕ 有后人纪念/祠庙记载
- ➕ 有成语/典故传承（如"周苛骂贼"）
- ➕ 太史公曰 对此人有专评
- ➕ 历史上被敌方承认的尊重记载
- ➕ 与同类忠义人物横向比较分析

---

## 反面示例（避免）

- ❌ 殉主场景只有叙述，没有原文引文——这是此类页面的核心，必须有原文
- ❌ 没有"殉主方式"对照表行——各人死法不同，是类型内部区分的关键
- ❌ 同类人物互链缺失——读者需要能从纪信导航到周苛
