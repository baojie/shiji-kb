# SKILL 02j: 修辞标注

> 修辞标注是**语言层**标注，与实体层（〖〗）、事件层（⟦⟧）并列，标注文本自身的语言特征——固化短语、典故、谚语等不指称具体实体的语言单元。

---

## 一、设计原理

### 1.1 三层标注框架

| 层次 | 括号 | 本质 | 例子 |
|------|------|------|------|
| 实体层 | `〖〗` | 指称世界中的对象 | 〖@刘邦〗、〖=咸阳〗 |
| 修辞层 | `〘〙` | 语言本身的固化单元 | 〘※移风易俗〙 |
| 事件层 | `⟦⟧` | 世界中的行为动作 | ⟦◈攻⟧ |

**关键区别**：`移风易俗` 不指称任何历史人物、地点或制度，它是文言文词库中的一个固化单元（multiword expression），标注的是语言本身而非语言所指。

### 1.2 标注条件

成语标注有两个前提：
1. **未被其他实体标注占用**：如"名山大川"若在祭祀语境中已标为 `〖=名山大川〗`，不再重复标注
2. **未被动词标注占用**：固化动词短语若已有 `⟦TYPE⟧` 标注，不再重复

## 二、类型系统

### 当前启用（v1.0）

| 符号 | Unicode | 类型 | 定义 | 古籍出现 |
|------|---------|------|------|---------|
| `※` | U+203B | 成语 | 固化四字熟语及固定短语，意义不可完全从字面推导 | ❌ 绝不出现 |

**格式**：
- 基本：`〘※成语文本〙`
- 消歧：`〘※原文形式|规范形式〙`（原文字序/字形与现代标准成语不同时使用）

**消歧原则**：史记文本作为成语出处时，史记形式本身即为规范形，**不加消歧**。消歧只用于以下情形：

| 情形 | 示例 | 说明 |
|------|------|------|
| 字序颠倒 | `〘※同归而殊涂\|殊途同归〙` | 史记字序与现代通行形颠倒 |
| 字数简化 | `〘※讷於言，敏於行\|讷言敏行〙` | 原文有虚字，后世简化为4字 |
| 史记引用更早文本的异形 | `〘※X\|Y〙` | 史记引用的是前代文本的变体 |

**不消歧的情形**：
- 史记是该成语的出处（如 `失之豪釐，差以千里`、`风起云蒸`）→ 史记形式即规范形，不加 `\|`

**示例**：
```
〘※移风易俗〙                              ← 与现代标准相同，无需消歧
〘※同归而殊涂|殊途同归〙                   ← 字序不同
〘※风起云蒸|风起云涌〙                     ← 字形不同
〘※失之豪釐，差以千里|差之毫厘，谬以千里〙  ← 字形+结构均不同
〘※人给家足|家给人足〙                     ← 字序颠倒
```

### 预留扩展（暂不启用）

| 符号 | Unicode | 类型 | 备注 |
|------|---------|------|------|
| `✦` | U+2726 | 典故 | 引用前代具体事件/人物的固化表达 |
| `✧` | U+2727 | 谚语 | 民间固化格言，非书面四字成语 |
| `⁂` | U+2042 | 其他修辞 | 比喻、排比等句式修辞 |

## 三、成语标注规范

### 3.1 什么应该标注

✅ **标注**：
- 四字固化成语：`移风易俗`、`名山大川`、`桃李不言`
- 非四字但固化的熟语：`桃李不言，下自成蹊`（六字+补充）
- 在后世演变为成语、但在史记中是首见或早期用例的表达

❌ **不标注**：
- 普通四字短语（非固化，可逐字理解）：`遣使求和`、`斩将夺旗`
- 已被实体标注的：`〖◆秦〗`、`〖=名山〗` 等
- 已被动词标注的：`⟦◈攻〗` 等

### 3.2 互斥原则（核心规则）

**三层标注绝对互斥，不得嵌套**：

| 情况 | 规则 |
|------|------|
| 成语内部有人名/地名等实体 | ❌ 禁止在 `〘※〙` 内部加 `〖〗` |
| 成语内部有动词 | ❌ 禁止在 `〘※〙` 内部加 `⟦⟧` |
| 已标实体的文字 | ❌ 不再标成语 `〘※〙` |
| 已标动词的文字 | ❌ 不再标成语 `〘※〙` |

```
❌ 错误：〘※〖@管仲〗之交〙        （成语内嵌实体）
❌ 错误：〘※〖=名山〗大川〙        （成语内嵌实体）
✅ 正确：〘※管鲍之交〙             （整体作成语，不标内部人名）
✅ 正确：〘※名山大川〙             （整体作成语，不标内部地名）
```

**遇到内部有可标实体的成语，二选一**：
- 优先选择**成语**标注（整体固化意义更重要）
- 除非该实体在上下文中承担实质性指称功能

### 3.4 边界判断

**判断是否为成语的三问**：
1. 后世汉语中有此固化成语？→ 是：标注
2. 意义能否完全从字面推导？→ 不能完全推导：倾向标注
3. 在史记中是否为"引用前代"的固化用法？→ 是：标注

### 3.5 与实体标注的优先级

成语优先于地名：泛指性固化表达即使出现在地理语境中，仍标为成语而非地名。

```
〖{乐〗者，所以〘※移风易俗〙也    ← 成语
追本诸神〘※名山大川〙礼          ← 成语（泛指，不指具体地点）
〖=泰山〗封禅                    ← 具体地点，标地名
```

**判断标准**：能否替换为一个具体的专有名词？
- `名山大川` → 不能（泛指）→ 成语 `〘※〙`
- `泰山` → 是具体地点 → 地名 `〖=〗`

## 四、示例（第130章）

原文：
```
[62] 〖{乐〗者，所以移风易俗也。
[66] 追本诸神名山大川礼，作〖^封禅〗书第六。
```

标注后：
```
[62] 〖{乐〗者，所以〘※移风易俗〙也。
[66] 追本诸神〘※名山大川〙礼，作〖^封禅〗书第六。
```

> 注：`名山大川` 是成语（固化四字熟语），标为 `〘※〙`，不标为地名。虽然语境涉及祭祀地理，但"名山大川"不指称某个具体地点，是泛指性固化表达。

## 五、检查清单

- [ ] 标注前确认未被 `〖〗` 或 `⟦⟧` 占用
- [ ] 成语边界完整（四字成语不拆分）
- [ ] 使用 `※` 符号（U+203B），非 `*` 或 `＊`
- [ ] 括号为 `〘〙`（U+3018/3019），非 `【】` 或 `⟦⟧`
