# SKILL 02j: 修辞标注

> 修辞标注是**语言层**标注，与实体层（`〖〗`）、事件层（`⟦⟧`）并列，使用 `〘〙` 括号。本 Skill 限定于：符号、格式、互斥规则、HTML 呈现。
> **识别决策**（哪段原文算成语、如何处理实体冲突、如何反思迭代）见 [SKILL_02k 成语识别](SKILL_02k_成语识别.md)。

---

## 一、三层标注框架

| 层次 | 括号 | 本质 | 示例 |
|------|------|------|------|
| 实体层 | `〖〗` | 指称世界中的对象 | `〖@刘邦〗`、`〖=咸阳〗` |
| 修辞层 | `〘〙` | 语言自身的固化单元 | `〘※移风易俗〙` |
| 事件层 | `⟦⟧` | 世界中的行为动作 | `⟦◈攻⟧` |

**关键区别**：`移风易俗` 不指称任何历史对象，是文言文词库中的固化多词单元（multiword expression），标注的是语言本身。

---

## 二、类型系统

### 2.1 当前启用（v1.0）

| 符号 | Unicode | 类型 | 定义 |
|------|---------|------|------|
| `※` | U+203B | 成语 | 固化四字熟语及固定短语，意义不可完全从字面推导 |

**格式**：
- 基本：`〘※成语文本〙`
- 消歧：`〘※原文形式|规范形式〙`

### 2.2 消歧规则

**何时不加消歧**：史记是该成语的出处（如 `失之豪釐，差以千里`、`风起云蒸`）→ 史记形式即规范形。

**何时必加消歧**：

| 情形 | 示例 |
|------|------|
| 字序颠倒 | `〘※同归而殊涂\|殊途同归〙` |
| 字数不同 | `〘※讷於言，敏於行\|讷言敏行〙` |
| 字形不同 | `〘※风起云蒸\|风起云涌〙` |
| 史记引用更早文本的异形 | `〘※失之豪釐，差以千里\|差之毫厘，谬以千里〙` |
| 用户点校确立 | `〘※养虎自遗患\|养虎遗患〙`、`〘※扬扬甚自得\|扬扬自得〙` |

### 2.3 预留扩展（暂不启用）

| 符号 | Unicode | 类型 |
|------|---------|------|
| `✦` | U+2726 | 典故（引用前代具体事件/人物的固化表达）|
| `✧` | U+2727 | 谚语（民间格言）|
| `⁂` | U+2042 | 其他修辞（比喻、排比等句式）|

## 三、互斥原则

**三层标注绝对互斥，不得嵌套**：

| 情况 | 规则 |
|------|------|
| 成语内部有人名/地名等实体 | ❌ `〘※〙` 内不得嵌 `〖〗` |
| 成语内部有动词 | ❌ `〘※〙` 内不得嵌 `⟦⟧` |
| 已标实体的文字 | ❌ 不可重叠 `〘※〙` |
| 已标动词的文字 | ❌ 不可重叠 `〘※〙` |

```
❌ 〘※〖@管仲〗之交〙       （成语内嵌实体）
❌ 〘※〖=名山〗大川〙       （成语内嵌实体）
✓ 〘※管鲍之交〙            （整体作成语）
✓ 〘※名山大川〙            （整体作成语）
```

遇冲突如何抉择：见 [SKILL_02k §3.3 与实体/动词冲突时的优先级](SKILL_02k_成语识别.md)。

## 四、HTML 渲染

### 4.1 样式定义

成语的视觉呈现采用"朱批圈点"风格：
```css
.chengyu {
    color: #8b4513;               /* 赭石色 */
    font-weight: 600;
    position: relative;
}
.chengyu::before {
    content: '〘';
    color: #c9a56c;
    font-weight: normal;
}
.chengyu::after {
    content: '〙';
    color: #c9a56c;
    font-weight: normal;
}
```

### 4.2 消歧形式的展示

`〘※shiji_form|modern_form〙` 在 HTML 中：
- 章节页：显示 `shiji_form`（原文），`modern_form` 作 tooltip
- 成语索引：以 `modern_form` 为条目名，展示 "史记作 shiji_form"

### 4.3 成语索引页（`docs/special/chengyu.html`）

由 `scripts/render_chengyu_html.py` 生成，合并两份数据：
- `data/chengyu.json`：已标注条目（徽章无）
- `data/chengyu_summarized.json`：总结性条目（附徽章 `verbatim`/`partial`/`narrative`）

双轨数据结构与合并逻辑见 [SKILL_02k §5](SKILL_02k_成语识别.md)。

## 六、检查清单（形式合规）

- [ ] 使用 `※` 符号（U+203B），非 `*` 或 `＊`
- [ ] 括号为 `〘〙`（U+3018/3019），非 `【】` 或 `⟦⟧`
- [ ] 成语内部**无**任何 `〖〗` 或 `⟦⟧`
- [ ] 消歧分隔符为 `|`（ASCII 管道符）
- [ ] 成语边界完整（四字成语不拆分，多字熟语不截断）
- [ ] 去标注后的纯文本与原始 `.txt` 逐字一致（`lint_text_integrity.py` 通过）
