# SKILL 02f: 文本统计 — 定量分析与分布特征

> 对标注文本进行多维度定量统计，揭示《史记》的文本规模、结构特征和实体分布规律。

---

## 一、统计维度

### 1.1 基础规模

| 指标     | 数值             |
| -------- | ---------------- |
| 总章节数 | 130篇            |
| 纯汉字   | 576,915字        |
| 含标点   | ~706,000字       |
| 实体词条 | 12,380个（18类） |
| 标注次数 | 99,214次         |

### 1.2 五体分布

| 体裁   | 篇数   | 字数    | 占比   |
| ------ | ------ | ------- | ------ |
| 本纪   | 12     | 76,730  | 13.3%  |
| 表     | 10     | 70,525  | 12.2%  |
| 书     | 8      | 41,761  | 7.2%   |
| 世家   | 30     | 132,853 | 23.0%  |
| 列传   | 70     | 255,046 | 44.2%  |

### 1.3 可统计指标

| 类别     | 指标                     | 工具                        |
| -------- | ------------------------ | --------------------------- |
| 文本长度 | 章节字数、段落字数、句长 | `analyze_tagged.py`         |
| 词频     | 字频、实体频次、虚词频次 | `analyze_word_frequency.py` |
| 实体密度 | 每百字实体数、分类占比   | `analyze_tagged.py`         |
| 对话占比 | 引文字数/总字数          | `analyze_tagged.py`         |
| 标点密度 | 每百字标点数             | `analyze_tagged.py`         |

---

## 二、工具

```bash
python scripts/analyze_word_frequency.py          # 词频统计
python analyze_tagged.py                           # 标注文本分析
```
