## 标注规则说明

### 读音规则

1. **wéi（动词、系动词）**：成为、是、做、担任、认为（约85-90%）
   - **系动词**："是为XX"（这就是XX）
   - **任命动词**："立为"、"迁为"、"拜为"、"封为"、"谥为"（被任命为...）
   - **自主动词**："自立为"、"自以为"（自己成为、自己认为）
   - **判断动词**："以为"、"以为然"（认为）
   - **转变动词**："因祸为福"（变成）

2. **wèi（介词、被动）**：为了、替、给、被（约10-15%）
   - **目的介词**："为天下"、"为民"、"为君"（为了...）
   - **对象介词**："为之"、"窃为大王"（替、给）
   - **被动标志**："为XX所"（被...）
   - **疑问词**："何为"（为什么）

### 词表策略

**方案选择**: 正则表达式 + 高频固定词组

**正则规则**:
```javascript
// "是为XX"、"立为XX"、"XX为XX王/公/侯" 等结构读 wéi
if (/是为|[立迁拜封谥]为|自立为|以为/.test(context)) {
  return 'wéi';
}

// "为XX所"、"为之"、"为天下"、"窃为" 等结构读 wèi
if (/为\S{1,2}所|为之|为天下|窃为|何为/.test(context)) {
  return 'wèi';
}
```
