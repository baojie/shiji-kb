## 标注规则说明

### 读音规则

1. **jiàng（名词/动词）**：将军、将领、统帅、率领（约95%）
   - **名词**："大将军"、"车骑将军"、"秦将"、"诸将"
   - **动词**："将兵"（统率军队）、"所将"（所统率的）
   - **官名**：各种XX将军（大将军、骠骑将军、轻车将军等）

2. **jiāng（副词）**：将要、将会（约5%）
   - **时间副词**："将死"（将要死）、"将如何"（将要怎么办）
   - **特征**：通常后接动词，表示即将发生的动作

### 词表策略

**方案选择**: 使用正则表达式 + 少量固定词组

**正则规则**:
```javascript
// "XX将军"、"为XX将"、"将兵" 等结构读 jiàng
if (/将军|为\S{1,2}将|将兵|诸将|秦将|汉将/.test(context)) {
  return 'jiàng';
}

// "将XX"后接动词（如"将死"、"将何"）读 jiāng
if (/将[死亡归来至到何如]/.test(context)) {
  return 'jiāng';
}
```
