## 三、检查方法

```bash
# 1. 提取所有 @/; 标注中含"君"的实体
grep -oP '〖[@;][^〗]*君[^〗]*〗' chapter_md/*.tagged.md \
  | sed 's/.*://; s/〖[@;]//; s/〗//' | sort -u > all_jun_tags.txt

# 2. 与 jun_titles.json 已有条目比对，找出差集
# 3. 逐条查看原文上下文，判定性质
```
