# 韵文标题映射机制

## 目的

保护手工确定的诗歌和赋标题，防止被自动提取算法覆盖。

## 工作原理

1. **手工标题优先**：`yunwen_titles.json` 中定义的标题具有最高优先级
2. **自动提取备选**：如果没有手工标题，使用 `extract_yunwen.py` 的自动提取逻辑
3. **默认标题兜底**：如果都没有，使用通用标题（如"章节名诗"）

## 标题映射文件格式

文件：`data/yunwen_titles.json`

```json
{
  "章节号_韵文类型_序号": "标题",

  "006_诗歌_1": "泰山刻石",
  "006_诗歌_2": "琅邪台刻石",

  "084_赋_1": "怀沙赋",
  "084_赋_2": "吊屈原赋"
}
```

**说明**：
- **章节号**：3位数字，如 `006`、`084`
- **韵文类型**：`诗歌`、`赋`、`赞`
- **序号**：该章节中第N篇该类型的韵文，从1开始
- 以 `_` 开头的字段会被忽略（用于注释）

## 使用方法

### 添加新的手工标题

1. 确定章节号和韵文类型
2. 数清楚这是该章第几篇此类韵文
3. 在 `yunwen_titles.json` 中添加条目：
   ```json
   "章节号_类型_序号": "标题"
   ```
4. 重新运行提取脚本：
   ```bash
   python3 scripts/extract_yunwen.py
   python3 scripts/render_yunwen_html.py
   ```

### 修改已有标题

直接编辑 `yunwen_titles.json` 文件，然后重新运行提取脚本。

### 删除手工标题

删除 `yunwen_titles.json` 中的对应条目，系统会自动使用自动提取的标题或默认标题。

## 标题优先级

```
手工映射 (yunwen_titles.json)
    ↓ 如果没有
Markdown标题 (## 标题)
    ↓ 如果没有
上下文模式匹配 (地名、作品名等)
    ↓ 如果没有
默认标题 (章节名+类型)
```

## 注意事项

1. **序号从1开始**，不是从0
2. **章节号必须是3位**，如 `006` 而不是 `6`
3. **韵文类型必须精确匹配**：`诗歌`、`赋`、`赞`（简体中文）
4. **序号按出现顺序**：tagged.md 文件中从上到下的顺序
5. **赞一般不需要手工标题**：赞的标题通常能自动提取或使用默认标题

## 维护建议

- 每次发现自动提取的标题不准确时，立即添加到 `yunwen_titles.json`
- 定期检查 `data/yunwen.json`，查看是否有标题需要改进
- 保持 `yunwen_titles.json` 简洁，只存放真正需要手工维护的标题
