## 一、反思背景

022 汉兴以来将相名臣年表收录汉高帝元年（前 206 年）至成帝鸿嘉元年（前 20 年）共 187 年的三公（丞相/太尉→大司马、御史大夫）更替与名臣军功。表格以「纪年/大事纪/相位/将位/御史大夫位」五列呈现，人物密度极高。

前四轮基线：
- **第三轮**：D6/A58 年表军事/刑法动词（41 处）
- **第四轮**：A8-8 年号消歧（28 处）+ 深度反思（~80 处：邦国/氏族、A82a 复姓边界、皇子补名、温室钟等）
- **第四轮补充**：侯号切分专项（47 处：侯号+人名拆分、双字侯号合并、侯号类型更正）

但第五轮预扫描仍发现以下系统性漏标：

1. **严重错误**：`〖@李广〗利` —— "李广利"（贰师将军）被错拆成"李广"+"利"（r110、r117，2 处）
2. **严重错误**：`〖@冠军侯|霍去病〗〖@霍去病〗` —— 冠军侯的 canonical 形式与本名并列重复（r86）
3. **严重错误**：`〖[击〗〖@右贤〗` —— 应为匈奴右贤王（职位），误标为人名（r83）
4. **御史大夫列裸字姓名全章遗漏**：20+ 处 `〖;御史大夫〗X` 结构，X 为裸字未标：抵/蔡/宽/延广/卿/周/胜之/成/䜣/望之/霸/延年/定国 等
5. **大事纪列裸字状态动词主语**：`何薨`/`婴卒`/`嘉卒`/`蚡卒`/`泽免`/`弘卒`/`䜣卒`/`敞卒`/`义薨`/`相卒`/`霸薨`/`商免`/`禹卒` 等
6. **武帝朝名将省称未消歧**：将位列的 `青`（卫青）/`贺`（公孙贺）/`敖`（公孙敖）/`禹`（霍禹）/`安世`（张安世）/`充国`（赵充国）/`说`（韩说）等
7. **昭宣元成朝重要名臣漏标**：`王䜣`/`杨敞`/`蔡义`/`韦贤`/`辛武贤`/`陈万年`/`冯奉世`/`贡禹`/`薛广德`/`张谭`/`史丹`/`尹忠`/`张忠`/`王音`/`于永`/`张禹`/`薛宣`/`繁延寿` 等
8. **官职被错拆为人名**：`破羌将军`/`强弩将军`/`将屯将军`/`车骑将军` 等将军号被 `〖;将军〗` 截断
9. **薛广德裸字拆分错误**：`薛〖@广德〗` 应为 `〖@薛广德〗`
10. **上官舛/上官桀消歧**：r120 `〖@上官舛〗` 应补规范名（canonical 为上官桀）
11. **氏族"齐田"/"楚昭、屈、景"漏标**：r9 徙豪族于关中段
12. **误标方位词**：r13 `〖=北方〗`（在"长安城西北方"中按 D14 应去标）

## 五、新规律候选 / 发现

### 发现 1：22 年表"御史大夫位裸字姓名"模式

本章有近 20 处 `〖;御史大夫〗X` 结构，X 为一字或二字裸字姓名。该模式的主因是年表单元格内惯用省略（前文有全名时后段用字）或当时史料记录的简略。

**扫描命令**：
```bash
grep -nE '〖;御史大夫〗[^〖|]' chapter_md/NNN_*.tagged.md
```

**建议**：可作为"年表 X 位单字省称"专项的核心模式（A116 的年表版），覆盖其他职位如"相"、"太尉"、"将军"。

### 发现 2：将军号"XX 将军"被误拆为"XX〖;将军〗"

r74 五处连续将军号（护军/骁骑/轻车/将屯/材官）全部被 `〖;将军〗` 截断，只保留"将军"本身。这是标注粒度问题：应整体合并为 `〖;XX将军〗`。

**扫描命令**：
```bash
grep -nE '(护军|骁骑|轻车|材官|将屯|贰师|因杅|强弩|破羌|楼船|伏波|横海|游击|度辽|蒲类|虎牙|祁连|奋武|建威|复土|后土|重合|度辽|驃骑|骠骑)〖;将军〗' chapter_md/*.tagged.md
```

**建议**：将此扫描纳入新规律 D6.6 "年表/列传将军号合并扫描"。

### 发现 3：「年表主语裸字＋状态动词」模式

`X卒`/`X薨`/`X免`/`X罢`/`X坐`/`X老`/`X徙为` 等谓动词前的裸字主语是年表常见漏标模式，尤其在大事纪列。

**扫描命令**：
```bash
grep -nE '(^|[^〗])[一-龥]{1,3}(卒|薨|免|罢|坐|老|徙)[。，]' chapter_md/0(14|15|17|18|19|20|21|22)_*.tagged.md
```

**建议**：作为年表通用扫描，纳入新规律或 D6 子项。

### 发现 4：李广 vs 李广利 辨析

同章内"李广"（飞将军）与"李广利"（贰师将军）两位名将极易混淆。年表中"贰师将军李广利"若误拆为"贰师将军李广+利"将导致 canonical 指向错误人物（李广）。需以"贰师/贰师将军"作为"李广利"的强锚点判定。

**扫描命令**：
```bash
grep -nE '〖@李广〗利|贰师\s*〖;将军〗\s*〖@李广〗' chapter_md/*.tagged.md
```

建议：纳入 A109 同名异人消歧典型案例。

### 发现 5：官职+裸字姓名构造（〖;官职〗X 为）

年表中"〖;某官职〗X为〖;另一官职〗"结构里 X 几乎必是人名（在任官之际），但经常以裸字出现。

**扫描命令**：
```bash
grep -nE '〖;[^〗]+〗[一-龥]{1,3}为〖;' chapter_md/*.tagged.md
```
