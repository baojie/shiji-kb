## 三、修正详情

### 3.1 格式修正：L30 `:::` 注释独立行

**问题**：
```
[4.3] 对曰：::: 缪贤说辞
```
`:::` 段标记与原文 `对曰：` 混排在同一行，lint工具将 `:::缪贤说辞` 识别为"插入字符"（1处实质差异）。

**修正**：
```
[4.3] 对曰：
::: 缪贤说辞
```

**规范依据**：全章其他 `:::` 标记均独占一行（参考 L53 `相如斥责秦王`、L65 `设计归璧` 等格式），本处为格式异常。

### 3.2 A108 太史公曰段漏标：L373 "让颇"

**问题**：太史公曰段中 `退而让颇` 的"颇"字裸露，违反 A108：
> `::: 赞` / `::: 太史公曰` 段以单字代指主要人物时**100% 必须消歧**，不受 C3 豁免。

**修正**：
```
退而让〖@颇|廉颇〗
```

**说明**：赞段已有 `〖@颇|廉颇〗`（L383），太史公曰段此处补标保持全章一致性。

### 3.3 A4 封号类型一致性修正：L273 `〖@信平君〗` → `〖;信平君〗`

**问题**：
```
〖◆赵〗以〖=尉文〗⟦○封⟧〖@廉颇〗为〖@信平君〗，为假〖;相国〗。
```
廉颇在全章有全名 `〖@廉颇〗`，此处"信平君"是其封号，按 A4 规则：文中已有本名时封号引用作职衔 `〖;〗`。

**修正**：
```
〖◆赵〗以〖=尉文〗⟦○封⟧〖@廉颇〗为〖;信平君〗，为假〖;相国〗。
```

**一致性依据**：全章其他封号均使用 `〖;〗`：
- `〖;马服君〗`（L215、L227，赵奢封号）
- `〖;武安君〗`（L357，李牧封号）
- `〖;相国〗`（同行，官职 `〖;〗` 正确）

`〖@信平君〗` 为第四轮遗留的不一致标注，本轮修正。
