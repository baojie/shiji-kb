## 三、典型案例

### 案例 1：齐君本名链条系统性消歧（A87/A116）

032 齐世家的齐君本名有众多单字（如静/山/赤/潘/元/光/牙/荼/壬/骜），均为"谥号+公"的本名。按 A87 单字名 100% 消歧原则：

```
〖@静〗 → 〖@静|齐胡公〗        （哀公之弟静=齐胡公）
〖@山〗 → 〖@山|齐献公〗        （哀公同母少弟山=齐献公）
〖@赤〗 → 〖@赤|齐文公〗        （厉公子赤=齐文公）
〖@潘〗 → 〖@潘|齐昭公〗        （孝公弟潘=齐昭公）
〖@元〗 → 〖@元|齐惠公〗        （齐惠公本名元）
〖@光〗 → 〖@光|齐庄公〗        （灵公子光=齐庄公，太子光）
〖@牙〗 → 〖@牙|齐公子牙〗      （仲姬生子牙=齐公子牙）
〖@舍〗 → 〖@舍|齐公子舍〗      （昭公子舍，无谥号）
〖@荼〗 → 〖@荼|晏孺子〗        （景公少子荼=晏孺子）
〖@壬〗 → 〖@壬|齐简公〗        （悼公子壬=齐简公本名）
〖@骜〗 → 〖@骜|齐平公〗        （简公弟骜=齐平公本名）
```

涉及共 26 处修正，与 014 十二诸侯年表 L287 `〖@懿公|齐懿公〗`、L294 `〖@景公|齐景公〗` 等一致。

### 案例 2：崔杼家族 15 处单字名密集消歧（A116）

L258-260 记崔杼内乱始末，崔杼子"成、彊、明"及东郭氏"偃"六位人物单字名密集出现。按 A116 功臣传承链原则全部消歧：

```
〖@成〗 → 〖@成|崔成〗         ×6 （崔杼长子）
〖@彊〗 → 〖@彊|崔彊〗         ×4 （崔杼次子）
〖@明〗 → 〖@明|崔明〗         ×2 （崔杼幼子）
〖@偃〗 → 〖@偃|东郭偃〗       ×2 （东郭女之弟）
〖@杼〗 → 〖@杼|崔杼〗         ×1 （崔杼自称，臣名）
```

原保留 `〖@崔|崔杼〗` ×3 作为"〖@崔〗，宗邑"的氏族省称指向人 — 维持原形。

### 案例 3：田氏家族单字 6 处消歧 + 氏族/人名辨析

### 案例 4：L92 `〖@夷吾〗` 同名异人消歧（A87 + A109）

本章有两个"夷吾"：
- L92：`非〖@管夷吾〗不可。〖@夷吾〗所居国国重` → `〖@夷吾\|管仲〗`（管仲本名管夷吾）

### 案例 5：L117 `〖@昭王〗` 指周昭王（跨章/跨朝代）

齐桓公 35 年（651 BC）齐伐楚，管仲列举楚罪：

```
「〖◆楚〗贡〖•包茅〗不入，王祭不具，是以来责。〖@昭王〗南征不复，是以来问。」
〖;楚王〗曰：「贡之不入，有之，〖#寡人〗罪也，敢不共乎！〖@昭王〗之出不复，君其问之水滨。」
```

### 案例 6：范/中行 晋卿族修正（B6）

L285 记齐景公 55 年（497 BC）：「〖@范〗、〖@中行〗反其君於〖◆晋〗」指**范氏**与**中行氏**作为晋国世卿族反对晋定公。按 B6 "X 氏"一律氏族原则：
