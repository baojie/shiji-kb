| 原文 | 修正 | 位置 |

| `〖◆吴〗王从台上观` | `〖;吴王〗从台上观` | L5 |
| `〖◆吴〗王曰："〖;将军〗罢休就舍` | `〖;吴王〗曰："〖;将军〗罢休就舍` | L5 |

## 本章特点与发现

### 发现 1：吴王的多层次称谓体系

本章 L5 中吴王阖庐以多种形式出现：
1. `〖◆吴〗〖@王阖庐|姬阖庐〗`——邦国+全名，初次正式出现
2. `〖@阖庐|姬阖庐〗`——省称消歧（第四轮已修）
3. `〖;吴王〗`——邦国+裸王合并职位（第五轮修正）
4. `〖#王〗`——对话/叙事中的帝王通称（第五轮修正）

这四种形式各有适用场景，需严格区分。

### 发现 2：〖◆X〗王 vs 〖;X王〗 vs 〖#王〗 三分法

本章提供了清晰的实证：
- `〖◆吴〗〖@王阖庐〗` = 邦国 + 具体人名（B7 规则 2）
- `〖;吴王〗从台上` = 邦国+裸王合并（A113.1）
- `〖#王〗` = 叙事/对话中的帝王通称（A1）

三者不可混淆，且在同一 L5 段中均有出现。

### 发现 3：公族分类的边界

"公族"在不同章节可能有不同标注：
- 身份群体（宗室贵族）→ `〖#公族〗`（本章楚国公族）
- 宗法制度词语 → `〖^公族〗`（仅在讨论制度时）

本章"废公族疏远者"明确指人的群体（"废"的宾语是"公族（中）疏远的人"），判为 `〖#〗`。

### 发现 4：执行工具注意事项

本轮修复中，使用 Python 脚本 replace 时，若目标字符串含引号（`"` U+201C/U+201D），需注意 Python 字符串中的转义问题。建议优先用精确字符串匹配；切片替换时用字符位置确认。
