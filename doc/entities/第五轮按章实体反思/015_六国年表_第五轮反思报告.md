## 四、典型案例

### 案例 1：`〖@秦王|嬴政〗` 时代错位的系统性修正（B 类 5 处）

**背景**：秦王嬴政（秦始皇）即位于秦昭襄王孙庄襄王三年后（前246），在此之前近 60 年的秦国在位者是秦昭襄王。AI 标注器对单一"秦王"全局消歧至秦始皇，忽略了朝代语境。

**典型错误行（r164, L231）**：
```
六。〖◆秦〗来⟦○立⟧〖@公子政|嬴政〗为太子。与〖@秦王|嬴政〗会〖=临晋〗。
```
秦昭襄王 12 年（前295），嬴政尚未出生（嬴政生于前259），此处"公子政"为另一秦宗室（或史记异文），"秦王"即昭襄王。

**修正**：`〖@公子政|嬴政〗` → `〖#公子〗〖@政〗`（去错误消歧，拆为身份词+单字名），`〖@秦王|嬴政〗` → `〖@秦王|秦昭襄王〗`。

**新发现**：记入 **C5 新陷阱清单**——"秦王"在秦昭襄王朝（前306-前251）默认指秦昭襄王；在秦庄襄王三年（前246）后才指秦始皇嬴政。

### 案例 2：跨国同谥号"惠王"的同字异人陷阱（C 类 + D 类）

"惠王"之谥在战国至少 5 例：
- 周惠王（前676-前652）
- 魏惠王（前370-前319）
- 楚惠王（前488-前432）
- 秦惠文王（前337-前311，亦称惠王）
- 齐惠王（短命未继位）
- **赵惠文王**及其母"赵惠后"（吴娃）

L234 (r167) "立为惠王后" 实为赵武灵王立吴娃为"惠后"（吴氏后谥"惠"），与魏惠王毫无关系。但 AI 将"惠王"全局消歧至"魏惠王"，属典型跨国同字陷阱。

**C5 新增陷阱条目**：

| 字/词 | 候选 | 区分 |
|-------|------|------|
| 惠王 | 周惠王 / 魏惠王 / 楚惠王 / 秦惠文王 / 赵武灵王之惠后 | 按年份所在国别判定 |
| 秦王 | 秦昭襄王（前306-前251）/ 秦庄襄王（前249-前247）/ 秦始皇嬴政（前246-前210） | 按年份切换 |
| 魏王 | 魏惠王（前370-前319）/ 魏哀王（前318-前296）/ 魏昭王（前295-前277）/ 魏安厘王/魏景湣王/魏王假 | 按年份切换 |

### 案例 3：`〖@燕王|姬哙〗〖@哙|姬哙〗` 冗余格式统一（A 类 4 处）

第四轮报告明确指出该格式"待专项"，本轮作为第五轮漏标扫描的配套修正全部统一为 `〖;X王〗〖@Y|姓名〗` 格式（对标 L280 `〖;齐王〗〖@建|田建〗` 模板）：

```
原: 〖@燕王|姬哙〗〖@哙|姬哙〗〖%元年〗  (同一人两段标注，冗余消歧)
改: 〖;燕王〗〖@哙|姬哙〗〖%元年〗       (职位+本名，A113.1+A113.2 标准形)
```

同类修正：燕王喜（L290）、韩王安（L306）、魏王假（L317）。

**扫描方法**：对每处 `〖@X王|规范名〗` 核查该行所属年份是否落在规范名在位区间内。

#### 建议新增 A113 补充：`〖@X王|姓XY〗〖@Y|姓Y〗` 冗余格式

年表末段常见格式问题：
```
〖@燕王|姬哙〗〖@哙|姬哙〗〖%元年〗
```
应统一为：
```
〖;燕王〗〖@哙|姬哙〗〖%元年〗
```
对标 `〖;齐王〗〖@建|田建〗〖%元年〗` 模板。**A113.1 + A113.2 + A105 联用**。

#### 建议新增 C5 补充：历史上少见的"后"字跨国误消歧

原文"立为惠王后"指赵惠后（吴娃），与魏惠王无关。需先识别"惠后/宣后/庄后/华阳太后"等专名。A114 后宫称谓标注：具体王后名（吴娃→赵惠后）归 `〖@〗`，王后职位归 `〖;王后〗`。
