## 四、A86：裸亲属关系词补标（1 处）

### 4.1 问题位置

**位置**：L5  
**原文**：`魏昭王子少子而魏安釐王异母弟也`

### 4.2 当前标注问题

```
〖◆魏〗〖@昭王〗子〖#少子〗而〖@魏安釐王〗异〖#母〗〖#弟〗也
```

此处 `子` 字（在 `〖@昭王〗` 与 `〖#少子〗` 之间）是裸亲属关系词，表"昭王的儿子"之义，按 A86 应标 `〖#子〗`。

### 4.3 修正后

```
〖◆魏〗〖@昭王|魏昭王〗〖#子〗〖#少子〗而〖@魏安釐王〗异〖#母〗〖#弟〗也
```

**语义分析**：
- `〖#子〗`：昭王的儿子（亲属关系）
- `〖#少子〗`：排行最小（具体身份说明）

两者分别承担不同语义功能，前者是亲属词，后者是排行身份。
