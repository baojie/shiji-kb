
### 案例 2：L960 `〖•项羽〗` 物品标记错用

```
原: [49.2] 〖@汉王〗〖•项羽〗相与临〖=广武〗之间而语。
```

### 案例 5：L1413 `〖@魏公子|信陵君〗〖@无忌〗` 三段式拆分

```
原: 〖@魏公子|信陵君〗〖@无忌〗〖$五家〗。
```
