## 四、典型案例

### 案例 1：r34 十八王分封行——A113 职位化重构 6 处

r34 是项羽分封十八王的密集行，原标注将封号作整体 `〖@X王〗` 或 `〖◆X〗王` 处理，违反 A113 精神。本章样板重构：

```
原：
〖@汉王〗始，故〖@沛公〗。
王〖@韩广〗始，故〖@燕王〗。
王〖@魏豹〗始，故〖@魏王〗。

改：
〖@汉王|刘邦〗始，故〖@沛公|刘邦〗。     # 刘邦主人公消歧 × 2
王〖@韩广〗始，故〖;燕王〗。              # 前职位置
王〖@魏豹〗始，故〖;魏王〗。              # 前职位置
```

其中 `〖@汉王〗`/`〖@沛公〗` 按 007 canonical 作 `|刘邦` 消歧（因"汉王"/"沛公"特指刘邦，非通用职位），而 `〖@燕王〗`/`〖@魏王〗` 作 `〖;X王〗` 降级为职位（因指前燕王/魏王角色，非刘邦特指）。此两类判断构成 A113 的核心区分。

### 案例 2：r65 临江王+人名——A113.2 拆分并亲属词补标


- **A86 亲属词**：`〖@敖|共敖〗子` → `〗〖#子〗`（共驩是共敖之子）
