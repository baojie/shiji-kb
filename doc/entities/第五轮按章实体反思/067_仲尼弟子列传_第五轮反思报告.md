## 一、嵌套标注修复（L320）

**问题**：`〖#丈〖#夫〗〗`（严禁嵌套，违反 CLAUDE.md 红线规则）导致文本完整性检查报告"丈夫"字符丢失（1处实质差异）。

**修正**：
```
原：〖$五〗〖#丈〖#夫〗〗〖#子〗
改：〖$五〗〖#丈夫〗〖#子〗
```

**依据**：A61 嵌套刑法/亲属标注错误；A74 复合身份词整体标注；严禁任何嵌套 `〖TYPE〖...〗〗`。

**验证**：修复后 `python scripts/lint_text_integrity.py 067` → 实质差异 0 处（由 1 降至 0）。

---

## 二、复合亲属词漏标"昆"（L27）

**问题**：`昆〖#弟〗`——"昆弟"是"兄弟"的复合亲属词，"昆"字漏标，应整体标注。

**修正**：
```
原：人不间於其〖#父〗〖#母〗昆〖#弟〗之言
改：人不间於其〖#父〗〖#母〗〖#昆弟〗之言
```

**依据**：A74 "父子/兄弟/父兄/母子/夫妇/母弟等血缘关系复合词整体标 `〖#〗`，禁嵌套或拆分"；"昆弟"即兄弟，属同类复合词。

## 三、规范名修正（端木赐 → 端沐赐）

**问题**：子贡本名在原文（L34）写作"端沐赐"，但 L1 和 L116 的消歧规范名错误使用了"端木赐"（与原文不符）。

**修正**（共 3 处）：
```
L1:  〖@赐|端木赐〗 → 〖@赐|端沐赐〗
L116: 〖@赐|端木赐〗（×2）→ 〖@赐|端沐赐〗（×2）
```

**依据**：A87 "规范名优先本名原则"——消歧语法中规范名必须与原文一致；原文 L34 明确写"端沐赐"。

## 四、字号格式统一（字〖#子〗X → 字〖@子X〗）

**问题**：三处弟子字号定义用了 `字〖#子〗X`（将"子"独立标为身份词），与全章其他字号定义格式 `字〖@子渊〗` / `字〖@子骞〗` 等不一致。

**修正**（共 3 处）：

| 位置 | 原标注 | 修正 |
|------|-------|------|
| L73 南宫括 | `字〖#子〗容` | `字〖@子容〗` |
| L88 公伯缭 | `字〖#子〗〖@周|公伯缭〗` | `字〖@子周〗` |
| L107 公孙龙 | `字〖#子〗石` | `字〖@子石〗` |

**依据**：A87 字号整体标注；全章统一格式 `字〖@子X〗`；字号"子容"/"子周"/"子石"是不可分割的完整字号。

**注**：L88 公伯缭字号定义的"周"此前有消歧标注 `〖@周|公伯缭〗`，因字号整体化后消歧信息移入后续引用处（L89 `〖@周|公伯缭〗` 保持不变）。

## 五、传易链邦国误标修正（L82）

**问题**：`〖@疵|矫子庸疵〗传〖◆燕〗人〖◆周〗〖@子家〗〖@竖|周子家竖〗` 中，"周"被错误标注为邦国 `〖◆周〗`。实际上"周子家竖"是一个人名（燕人，复合姓名），"周"是姓氏首字，非邦国名。

**修正**：
```
原：〖◆燕〗人〖◆周〗〖@子家〗〖@竖|周子家竖〗
改：〖◆燕〗人〖@周子家竖〗
```

后续省称 `〖@竖|周子家竖〗` 保持不变。

**依据**：B1 单字邦国名误标人名；"周"在"燕人周子家竖"中是姓氏而非邦国（正文无"周国"语境）。

## 六、字号省称消歧补充（L132 子石）

**问题**：L132 `〖@子石〗请行` 是公孙龙（字子石）的字号引用，但在定义处（L347"公孙龙字子石"）**之前**出现，且无消歧标注。

**修正**：
```
原：〖@子张〗、〖@子石〗请行
改：〖@子张〗、〖@子石|公孙龙〗请行
```

**依据**：A87 "字号必须消歧"；公孙龙定义于 L347，L132 先于定义出现，消歧尤其必要。
