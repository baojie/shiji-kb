### 案例 1：吴君 19 世谱系亲属词大补标（L30 一行 11 处）

L30 是本章最长的一行（200+ 字符），列举周章—句卑 11 代世系：
```
〖@周章〗卒，子〖@熊遂〗立，〖@熊遂〗卒，子〖@柯相〗立。〖@柯相〗卒，子〖@彊鸠夷〗立。〖@彊鸠夷〗卒，子〖@馀桥疑吾〗立。〖@馀桥疑吾〗卒，子〖@柯卢〗立。…
```

### 案例 3：L235 "兄卒弟代立，今卒，其子当代" 三重亲属词

此处是吴人商议 "馀眛之子僚" 继位合法性的经典辞令：
- `兄卒弟代立`：兄（诸樊/馀祭/馀眛）卒，弟（下一位）代立
- `其子当代`：其（馀眛）子（僚）当代立

三个亲属词（兄、弟、子）均裸露漏标，按 A86 全补 `〖#〗`：
```
「〖#兄〗卒〖#弟〗代立，必致〖@季子|季札〗。〖@季子|季札〗今逃位，则〖;王〗〖@馀眛〗後立。今卒，其〖#子〗当代。」
```

### 案例 4：L248 "吾父兄弟四人" 重新切分

第四轮遗留问题：`〖#父兄〗弟〖$四人〗` 切分异常。第五轮按语义"吾父（诸樊）的兄弟共四人"重切：
```
吾〖#父〗〖#兄弟〗〖$四人〗
```

即 "父（诸樊）+ 兄弟（诸樊、馀祭、馀眛、季札共四人）"。对照 086 L34：
> "〖@光|公子光〗之父曰〖;吴王〗〖@诸樊〗。〖@诸樊〗弟〖$三人〗：次曰〖@馀祭〗，次曰〖@夷眛〗，次曰〖@季子札〗"

光 "真適嗣" 语境即光认为其父诸樊这一支（四兄弟嫡长）应先传完。切分合理。

### 案例 5：楚王省称 + 夹敖异写处理

```bash
grep -nP '，子〖@[^〗]+〗立' chapter_md/03[1-9]_*.tagged.md chapter_md/04[0-9]_*.tagged.md
```
