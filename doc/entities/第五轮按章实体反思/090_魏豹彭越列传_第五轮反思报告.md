## 二、重大发现与修复

### 2.1 A113.1 大规模邦国+王合并为职位（40处）

**规则**：`〖◆X〗` 紧跟裸"王"字（无人名/谥号）→ 合并为 `〖;X王〗`。

**问题背景**：第四轮反思对"君王"拆分、将军官职等问题进行了修复，但遗漏了全章最大规模的结构性问题：所有"邦国+裸王字"格式均未按A113.1转化为职位标注。

**修复明细**：

| 原格式 | 修复为 | 次数 | 说明 |
|--------|--------|------|------|
| `〖◆魏〗王` | `〖;魏王〗` | 11 | 魏国王位（含"立X为魏王"、"魏王乃使"等） |
| `〖◆梁〗王` | `〖;梁王〗` | 11 | 梁国王位（彭越封梁王后，全章"梁王"均改） |
| `〖◆汉〗王` | `〖;汉王〗` | 12 | 汉王刘邦（与091/092章`〖;汉王〗`格式对齐） |
| `〖◆齐〗王` | `〖;齐王〗` | 4 | 齐国王位（田荣、韩信） |
| `〖◆西魏〗王` | `〖;西魏王〗` | 2 | 西魏国王位 |
| **合计** | | **40** | |

**典型样例（修复后）**：
- L5: `⟦○立⟧〖@周市〗为〖;魏王〗`（立X为魏王）
- L12: `乃徙〖;魏王〗〖@豹|魏豹〗於〖=河东〗`（职位+人名，A113.2格式）
- L31: `今〖;西魏王〗〖@豹|魏豹〗亦〖;魏王〗〖@咎|魏咎〗从〖#弟〗也`（对话中引用多王）
- L38: `〖;齐王〗〖@信|韩信〗`（齐王韩信，职位+人名，正确拆分）
- L45: `〖;梁王〗称病`（梁王作主语，职位称谓）

**参考先例**：
- 092章：`〖;魏王〗〖@豹〗`
- 016章月表：`〖;魏王〗〖@咎|魏咎〗始`
- 100/091/008章：`〖;梁王〗〖@彭越〗`、`〖;汉王〗`

**验证**：无邦国+裸王字残留
```bash

# 结果：无输出（全部已转化）
```
