## 一、反思背景

030 平准书是《史记》书类财政经济文本，数量词密度应达 ~11%（D3 高密度特征）。第四轮已做 24 处修正（7 基础 + 17 地名邦国深度），但按第五轮主题再审，系统性漏标集中在以下维度：

1. **D3 数量词漏标**：钱数/重量/面积/度量衡/级数 大量未加 `〖$〗`
2. **数量词"碎片化"**：`〖$十馀〗锺` / `〖$百馀〗万匹` 等应合并为单一 tag
3. **D4 车字**：已标为 `〖•车〗`/`〖•牛车〗`/`〖•轺车〗`/`〖;楼船〗`，干净
4. **身份/群体漏标**：士/战士/使者/使/臣/商/卒/骑/吏卒/家属/农工商 等
5. **亲属词（A74/A86）**：弟/亲/子孙/子 等
6. **E1 格式残留**：`〖{春秋〗` 遗留
7. **邦国类型**：`〖=吴〗`（作邦国实体）→ `〖◆吴〗`
8. **D12 五帝判据**：武帝诏令中"五帝之教"并列"禹汤之法"，按圣王系列归 `@`

### D3 扩展建议：经济/财政章数量词类型清单

**规则**：经济/财政章（029/030/129）数量词应达 ~11% 密度。检查点：

| 类别 | 触发词 | 例 |
|------|-------|----|
| 重量 | 斤/两/铢/钧 | 一斤/八两/四铢/三十馀万斤 |
| 粮帛 | 石/锺/斛/斗/匹/束 | 十馀锺/百馀万匹/六百万石 |
| 面积 | 顷/亩 | 十顷/数百顷/百馀顷 |
| 长度 | 丈/里 | 十馀丈/千馀里/三千 |
| 人数 | 人/口/级/头 | 数万人/七十馀万口/万五千级/千馀头 |
| 车马 | 匹/乘 | 二万乘/数万匹 |
| 钱币 | 钱/算/金/巨万 | 万钱/一算/数十万金/百馀巨万 |
| 等级 | 品/等 | 三品/三等/二等 |

**合并规则**：`〖$数字〗+量词` → `〖$数字量词〗` 整体，避免碎片化。

