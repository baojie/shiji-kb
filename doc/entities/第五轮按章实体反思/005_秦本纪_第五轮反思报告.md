## 一、反思背景

本章是秦国 770 年兴衰史，30+ 代君主世系链长、跨国外交密集，典型漏标灾区为：

### 案例 1：L96 `〖@武公弟德公〗` 误合实体的复杂分解

```
原: 生子三人，长男〖@武公〗为太子。〖@武公弟德公〗，同母〖@鲁姬〗子。生〖@出子〗。

```
