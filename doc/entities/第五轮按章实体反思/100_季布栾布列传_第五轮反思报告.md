## 十、章节特有发现

### 10.1 Edit 工具引号转换风险

本轮证实 Edit 工具在涉及全角引号的行修改时，可能将 `"` (U+201C) 误转为 `"` (U+201D)。第四轮已发现并修复2处，本轮再次出现1处（L6修改 `其子→其〖#子〗子` 时被破坏）。

**建议**：每次使用 Edit 工具修改含引号的行后，立即检查：
```bash
python3 -c "
with open('chapter_md/NNN.tagged.md') as f: c=f.read()
print('LDQ:', c.count(chr(0x201C)), 'RDQ:', c.count(chr(0x201D)))
"
```

### 10.2 「弟」字动词/亲属词辨析

本章 L36 同行出现两个「弟」：
- `〖@季布〗〖#弟〗〖@季心〗`：亲属关系，标 `〖#弟〗`
- `弟畜〖@灌夫〗`：此处弟=副词/动词（像对待弟弟一样），不标

两处「弟」字用法不同，需逐字判断词性。

### 10.3 `~胡` 的标注依据

"胡"在汉代文献中作民族名（北方胡族、匈奴等）时，应标 `〖~族群〗`。本章两处均满足：作名词、指代北方异族群体，区别于地名。
