## 一、反思背景

本章为《史记》燕国世家（召公奭封燕—燕王喜亡国 800 余年），第四轮聚焦 E1 旧格式、A90 帝王自称、A1 类型一致性、C5 单字人名消歧，共 50 处修正。

第五轮按主题推进：

1. **B7 邦国/头衔六类规则**：本章有大量"邦国+王"、"谥号+王/侯"构型，需严格按 B7 #1/#2/#4 处理
2. **〖#X王〗→〖;X王〗 跨章一致性**：燕王/赵王/秦王/韩王/代王/周王/齐王 等"国名+王"**头衔** 一律 `〖;〗`（非 `〖#〗`），与 008/009/043/044/045 等全书主流对齐
3. **A113.1 `〖◆X〗王` 合并为 `〖;X王〗`**：本章此类 6 处
4. **A86 世系链亲属词**：16 处 `X卒，子〖@Y〗立` / `子为〖@X王〗` / `弟〖@X〗` 裸露漏标 `〖#子〗`/`〖#弟〗`
5. **B7 #1 邦国+谥号合并** vs **B7 #2 邦国+本名分标** 两处关键修复：`〖◆齐〗〖@宣王〗`→`〖@齐宣王〗`（#1 合并）；`〖@燕哙〗`→`〖◆燕〗〖@哙|姬哙〗`（#2 分标）
6. **赞段 A108 强制消歧**：L188-L192 赞诗"召伯/庄/惠/文公/易王/齐宣"单字或省称 6 处全量消歧
7. **他国君主 `〖#君〗X` 消歧**：崔杼弑庄公（齐庄公）/田常弑简公（齐简公）必须消歧（非燕主国）
8. **邦国漏标**：`赵〖@孝成王〗`/`赵〖@悼襄王〗` 前 `赵` 需升为 `〖◆赵〗`

### 案例 1：L103 的三重 B7 问题（一行三个规则）

原文 L103 同时出现 B7 #1 / #2 / A113.1 三种模式的典型错误：
```
及〖@苏秦〗死，而〖◆齐〗〖@宣王〗复用〖@苏代〗。〖@燕哙〗三年，与〖◆楚〗、三〖◆晋〗⟦◈攻⟧〖◆秦〗，不〖[胜〗而还。…〖#燕王〗问曰："〖◆齐〗王奚如？"
```

逐项修复：
- `〖◆齐〗〖@宣王〗` → `〖@齐宣王〗`（B7 #1 邦国+谥号合并）
- `〖@燕哙〗` → `〖◆燕〗〖@哙|姬哙〗`（B7 #2 邦国+本名分标，与本章 L97/L116 `〖◆燕〗〖@哙|姬哙〗` 保持一致）
- `〖#燕王〗` → `〖;燕王〗`（A1/跨章头衔类型）
- `〖◆齐〗王奚如` → `〖;齐王〗奚如`（A113.1 邦国+裸王合并）

四个修正聚于一行，体现 B7 六类规则的全谱覆盖。

### 案例 5：赞段 L190 `庄送霸主，惠罗宠姬` 双单字消歧

此行含两个单字 `庄`/`惠`，均为**燕本国**君主省称。A108 规定赞段单字 100% 必须消歧（不受 C3 豁免）：

- `庄` = 燕庄公（L47 "〖@齐桓公〗救〖◆燕〗，…〖;燕君〗送〖@齐桓公〗出境" 即此役，庄公送霸主齐桓公）

### 案例 6：L116 `是为燕〖@昭王|燕昭王〗` → `是为〖@燕昭王〗`

"是为 X" 格式中 X 整体是完整谥号名（参 032 "是为〖@桓公〗"、040 "是为〖@灵王|楚灵王〗"）。本章 L116 裸字 `燕` + `〖@昭王|燕昭王〗` 重复冗余，按 B7 #1 合并为 `〖@燕昭王〗`。同时清理消歧冗余（surface==canonical）。

### 案例 7：他国 `〖#君〗〖@X〗` 语境化消歧

本章 L57/L69 分别出现齐大夫崔杼弑齐君、田常弑齐君：
- L57 `〖◆齐〗〖@崔杼〗⟦◉弑⟧其〖#君〗〖@庄公〗`——崔杼为齐大夫，其君=齐庄公（非燕庄公）

## 六、新规律建议

### 建议 1：世家章节"国名+王/君"头衔类型系统化扫描

本章 `〖#燕王〗`×15 + `〖#燕君〗`×3 + `〖#赵王〗`×3 + `〖#秦王〗`×1 + `〖#韩王〗`×1 + `〖#周王〗`×1 = 24 处 `〖#〗` 误标，全书其他章节基本统一为 `〖;〗`。建议增加 lint 规则：

```bash
grep -nE '〖#([齐秦楚燕赵魏韩晋周鲁卫陈蔡宋郑吴越])王〗' chapter_md/*.tagged.md
grep -nE '〖#([齐秦楚燕赵魏韩晋周鲁卫陈蔡宋郑吴越])君〗' chapter_md/*.tagged.md
```

命中即应修正为 `〖;〗`（除非上下文明确是指"某国王族群体"而非单一君主头衔）。

### 建议 2：世家章节"X 卒，子 〖@Y〗 立"漏标扫描模板

本章世系链 10 行 15 处 `子〖@X〗立` 漏 `〖#子〗` 与 013 殷帝 21 条、031 吴君 19 世同构。建议固化扫描脚本：

```bash
grep -nE '(卒|薨)，子〖@[^〗]+〗' chapter_md/*.tagged.md
# 命中应补标 → 〖#子〗〖@X〗
```

预计可覆盖世家章 031-060 约 100-200 处漏标。

### 建议 3：A113.1 `〖◆X〗王` 纯头衔合并扫描

本章 `〖◆齐〗王`（×3）/`〖◆秦〗王`（×2）/`〖◆赵〗王`（×1）共 6 处裸王字，应合并为 `〖;X王〗`。建议扫描：

```bash
grep -nE '〖◆[齐秦楚燕赵魏韩周]〗王([^〖]|$)' chapter_md/*.tagged.md
```

本章提示：注意 `〖◆X〗王〖@name〗` 属 A113.2 拆分（转 `〖;X王〗〖@name〗`），与 A113.1 合并逻辑一致。
