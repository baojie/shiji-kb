## 四、典型案例

### 案例 1：礼之三本段系统性身份词补齐（7 处）
L199/L201/L203 三行构成严密的"天地-先祖-君师"三元对举结构。原标注仅 `〖!天地〗` 每处都标，却让"先祖""君师"空置。此为八书哲学段典型漏标：开头名词已标，后续并列同型名词被忽略。

```
原: 〖!天地〗者，生之本也；先祖者，类之本也；君师者，治之本也。
改: 〖!天地〗者，生之本也；〖#先祖〗者，类之本也；〖#君师〗者，治之本也。
```

跨章依据：024 乐书 `〖#先祖〗是听`（L255）、004 周本纪 `〖#先祖〗有德臣`（L99）、032 齐太公世家 `其〖#先祖〗尝为〖;四岳〗`（L5）等全库 `〖#先祖〗` 案例，确认 23 章应补。

### 案例 2：L35 礼乐对仗一致性
```
原: 〖◆周〗衰，〖^礼〗废乐坏
改: 〖◆周〗衰，〖^礼〗废〖^乐〗坏
```
"礼废乐坏"四字对偶，第三轮标了"礼"漏了"乐"。对照 024 乐书全章 `〖^礼〗` 与 `〖^乐〗` 成对出现 30+ 次，本章必须对齐。

### 案例 3：L280 赞文段四词集中补标
```
原: 礼因人心，非从天下 ... 以事宗社
改: 〖^礼〗因〖_人心〗，非从〖_天下〗 ... 以事〖^宗社〗
```
赞文压缩度高，本章 `〖^礼〗` 共 25 次全标、`〖_天下〗` 共 8 次全标、`〖_人心〗`在 024 乐书 4 次先例、`〖^宗社〗`= 宗庙+社稷合称制度，四词皆应标。这类赞文集中漏标是书类章节普遍特征。
