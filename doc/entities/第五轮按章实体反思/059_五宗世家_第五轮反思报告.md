## 主要发现与经验

### 1. 世系体裁时长标注是系统性遗漏

汉代宗室世家（050-060章）每个诸侯王均有"立X年卒"/"X年卒"/"立X年，某事"格式，全部属于时长。本章第四轮完全遗漏了此类标注，建议作为后续世系体裁章节的必查项。

**批量扫描命令**：
```bash
grep -nE '立〖%[^|〗]+(年|岁)〗[，卒⟦]|〖%[^|〗]+(年|岁)〗卒' chapter_md/NNN_*.tagged.md
```

### 2. 亲属词在世家宗室章节高度密集

本章记录十三王及其子孙关系，"父/母/兄/弟/子/女/姊"等亲属词频繁出现。对话语境（如中山靖王与赵王的对话）中的亲属词同样不豁免（A89），均需标注。

### 3. 时长与纪年的辨别要点

本章存在多处易混淆的情况：
- `〖@孝景〗帝前〖%二年〗` = 景帝前元二年（纪年，不加时长）
- `〖%十五年〗，〖@孝景〗帝崩` = 彭祖徙赵後第十五年（时长）

判断方法：查看"年"前方是否有"前/中/後"年号前缀 → 有则纪年；否则结合上下文判断"能否替换为第X年"。
