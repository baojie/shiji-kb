```
原:
〖@卢生〗说〖@始皇〗曰：
> "臣等求芝奇药〖?仙者〗常弗遇，类物有害之者。方中，〖;人主〗时为微行以辟恶〖?鬼〗，恶〖?鬼〗辟，〖?真人〗至。
```

### 案例 4：L1425《过秦论》思想词归类

```
原: 不信〖#功臣〗，不亲士民，废〖^王道〗，立私权，禁文书而酷刑〖^法〗，先诈力而後〖^仁义〗
改: ...废〖_王道〗，立私权，禁文书而酷刑〖^法〗，先诈力而後〖_仁义〗
```

"王道/仁义"是思想体系词（D12 `_`），与"法"（制度 `^`）对立；两者区分是先秦思想标注的经典判据。

### 案例 5：L337/345 "诸子" 类型反转

```
原: 请立〖_诸子〗...〖_诸子〗〖#功臣〗...
改: 请立〖#诸子〗...〖#诸子〗〖#功臣〗...
```

语境："请立诸子（为诸侯）"——指始皇的皇子们，是亲属群体，非思想学派"诸子百家"。该修正纠正了前轮对"诸子"一词的误归。
