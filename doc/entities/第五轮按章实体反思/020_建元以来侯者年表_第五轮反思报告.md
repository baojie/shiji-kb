## 五、新规律候选 / 发现

### 发现 1：官职"大司马"等被误标为人名的回扫模式

本章出现 `大〖@司马〗`（3 处）与 `为〖@司马〗`（1 处）—— 都是**官职被错误标成人名**。扫描模式：

```bash
grep -nE '(大)?〖@司马〗' chapter_md/*.tagged.md
grep -nE '[^,。、⟦]〖@[丞司御太长][^〗]*〗' chapter_md/*.tagged.md  # 官职前字被粘连
```

建议纳入 A 区"官职 vs 人名"的错标规律，作为下一轮审查重点模式。

### 发现 2：诏书/自序中裸字人名高发

r83 诏书内"封介子为义阳侯"（介子裸字），r98 宣帝叙事中"素与广汉出入相通"（许广汉误标为地名），r77 诏书内"蚩尤畔父，黄帝涉江"（虽标但作为诏书单字应强制消歧检查）。诏书语段单字人名需按 A108 赞语段同等优先级处理。
