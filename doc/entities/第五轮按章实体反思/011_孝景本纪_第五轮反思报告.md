## 四、典型案例

### 案例 1：L238 梁分五国段 4 子集群漏标

```
原: ⟦○立⟧〖@梁孝王〗子〖@明|刘明〗为〖;济川王〗，子〖@彭离|刘彭离〗为〖;济东王〗，子〖@定|刘定〗为〖;山阳王〗，子〖@不识|刘不识〗为〖;济阴王〗。
改: ⟦○立⟧〖@梁孝王〗〖#子〗〖@明|刘明〗为〖;济川王〗，〖#子〗〖@彭离|刘彭离〗为〖;济东王〗，〖#子〗〖@定|刘定〗为〖;山阳王〗，〖#子〗〖@不识|刘不识〗为〖;济阴王〗。
```

梁孝王武薨后，景帝分梁为五国，5 子中 4 子分别封王。前人名全部已用消歧 `〖@X|刘X〗`，但 4 处"子"全裸露漏标——与 009 章 L231 "〖@孝惠〗後宫子〖@彊〗...子〖@不疑〗...子〖@山〗...子〖@朝〗...子〖@武〗" 并列组漏标模式完全同构。一次补齐 4 处。

### 案例 2：L11 章首复合身份词 `〖#三男〗`/`〖#三子〗`

```
原: 〖@孝文〗在〖◆代〗时，前后有三男，及〖@窦太后〗得幸，前后死，及三子更死，故〖@孝景〗得立。
改: 〖@孝文〗在〖◆代〗时，前后有〖#三男〗，及〖@窦太后〗得幸，前后死，及〖#三子〗更死，故〖@孝景〗得立。
```

此段是孝景得以继位的关键：孝文帝在代地有先前王妃所生"三男"，窦太后得宠后三男相继死，故孝景（四子中序）立。A74 复合群体身份词整体标注——"三男"/"三子"是数字+身份词的固定复合搭配（与 009 章 `〖#两子〗`、005 章 `〖#三人〗〖#长子〗〖#中子〗〖#少子〗` 同构）。

**判定要点**：此处"三男""三子"**不是数量词 `〖$三男〗`**，因为上下文主语是"男/子"身份，数量作形容修饰；若标 `〖$〗` 则丢失"孝文三位先王男"的身份含义。本轮依 A74 选身份路径。

### 案例 3：L33 孝文太后跨章消歧对齐

```
原: 〖%四月壬午〗，〖@孝文太后〗⟦○崩⟧。
改: 〖%四月壬午〗，〖@孝文太后|薄太后〗⟦○崩⟧。
```

022 年表已作消歧 `〖@孝文太后|薄太后〗`；全书主称为"薄太后"（010/099/102/106/108/118 等 10+ 章），"孝文太后"仅本章 L33 单次出现（追谥简称）。按 A87"规范名优先本名/通称" 补消歧到 `薄太后`。

### 案例 4：L29 + L165 功臣后代承袭段"孙"漏标

```
L29 原: 封故〖;相国〗〖@萧何〗孙〖@系|萧系〗为〖;武陵侯〗。
L29 改: 封故〖;相国〗〖@萧何〗〖#孙〗〖@系|萧系〗为〖;武陵侯〗。

L165 原: 封故〖;御史大夫〗〖@周苛〗孙〖@平|周平〗为〖;绳侯〗，...
L165 改: 封故〖;御史大夫〗〖@周苛〗〖#孙〗〖@平|周平〗为〖;绳侯〗，...
```

景帝朝补录功臣后代封侯，"（高祖/文帝功臣）孙 + 本名"结构两次出现，两处"孙"均漏标。本补齐后与 002/084/096/103 章 `〖@X〗〖#孙〗〖@Y〗` 结构一致。

### 案例 5：L31 "男子"身份词

```
原: 男子〖$二十〗而得傅。
改: 〖#男子〗〖$二十〗而得傅。
```

"男子二十而得傅"是景帝二年改制：男子满 20 岁傅籍（古代徭役/兵役登记）。"男子"是性别+成年状态的**身份词**（对照"女子"），整体标 `〖#〗` 符合 A74/A85 群体身份规则。与 009 章 `〖#男女〗`/`〖#美人〗`/007 章 `〖#男子〗` 同构。
