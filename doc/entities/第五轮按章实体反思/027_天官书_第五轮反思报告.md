## 六、新发现规律（建议回写 D13/D14）

### D13 扩展建议：四象/四宫统一天文
**规则**：`〖!中宫〗/〖!东宫〗/〖!南宫〗/〖!西宫〗/〖!北宫〗` 全数归天文，即便行文中未显式提"七宿"亦从之。**禁用**：`〖=东宫〗`/`〖=北宫〗`（除非特指宫殿建筑语境）。

### D13 扩展建议：四象神兽归天文
**规则**：苍龙/朱雀/白虎/玄武 在 027 天官书/025 律书作为四象星官名 → `〖!〗`；避免 `〖?〗` 残留或 `〖+〗` 生物。

### D13 扩展建议：岁星 12 年名成体系
**规则**：摄提格岁"监德"、单阏"降入"、执徐"青章"、大荒骆"跰踵"、敦牂"开明"、叶洽"长列"、涒滩"大音"、作鄂"长王"、阉茂"天睢"、大渊献"大章"、困敦"天泉"、赤奋若"天皓" —— 皆为岁星在该岁之别名，一律 `〖!〗`。

### D13 扩展建议：五星单字省称扫描
**规则**：027 天官书/120 章中论星行/合/犯/入/守 时的单字"火/金/水/木/土" 必为 `〖!〗` 五星；仅在"曰东方木/南方火/中央土/西方金/北方水/主春/主夏…"方位主令语境保留 `〖^〗` 五行。

**扫描命令**：
```bash
grep -nE '[^〖^!〗](火|金|水|木|土)(犯|守|入|出|合|为|与|在|之|尤)' chapter_md/027_*.tagged.md
```

### D12 扩展建议：五方天神 @→_ 判据
**规则**：苍帝/赤帝/黄帝/白帝/黑帝 + 行德/主X/车舍/坐 语境 → `〖_〗`；仅"〖@黄帝〗生〖@昌意〗"等世系语境保留 `〖@〗`。

### A74 扩展建议：星官命名体例
- 官职-命名星 → `〖;〗`（上将/次将/贵相/司命/司中/司禄/执法/将/相）
- 身份-命名星 → `〖#〗`（贵人/诸侯/士大夫）
- 建筑/物象-命名星 → `〖!〗`（端门/掖门/郎位/将位）
- 神/方位-命名星 → 神格 `〖_〗`；星名本身 `〖!〗`（五帝坐/五帝车舍 整体作星官时为 `〖!〗`，但 027 中解作神之坐/车舍 → `〖_五帝〗+` 述语）
