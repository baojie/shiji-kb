## 一、反思背景

014 十二诸侯年表覆盖 `共和元年`(BC 841) 至 `鲁哀公十六年`(BC 479) 共 360+ 年、14 国谱系并列大表。前四轮已补 32 处（氏族识别 + 军事动词）+ 646 处邦国专项 + 300 处动词专项，基础层面基本完善。

本轮第五轮聚焦**细粒度漏标**：
1. **误合 @ 实体**（B7/A113）：`〖@庄公弟叔牙〗`/`〖@王弟带〗`/`〖@太子忽〗`/`〖@太子申生〗`/`〖@太子建〗`/`〖@孔子生〗`/`〖@三晋栾逞〗`/`〖@公孙〗固` 等复合名误合为单实体 —— 按 B7 #2 邦国+本名拆 / A113 太子+名拆 / B7 #3 氏族+名合
2. **裸露亲属词**（A74/A86 修订版）：年表表格中 "X公卒，弟/子 Y 立" 极简模式大量存在，`弟`/`子`/`母`/`孙`/`兄`/`妻`/`母弟`/`女` 等裸字应补 `〖#〗`
3. **X公/X侯/X王 跨章消歧**（A87/A110）：年表中春秋战国同名不同国君主极多，`〖@景公〗` / `〖@湣公〗` / `〖@殇公〗` / `〖@文侯〗` / `〖@孝侯〗` / `〖@庄王〗` / `〖@康王〗` / `〖@昭王〗` 等统一补 `〖@X|齐景公〗` 等消歧
4. **卿大夫单字名消歧**（A110）：`〖@杼|崔杼〗` / `〖@常|田常〗` / `〖@鲍|宋文公〗` / `〖@桓子|季桓子〗` / `〖@晏子|晏婴〗` / `〖@文子|季文子〗` / `〖@宣子|范宣子〗`
5. **〖;王〗/〖#公〗 等通称补标**：年表叙事中 "王奔/王入/纳王/公卒/公薨/公如X" 等
6. **地名补标**：`〖=蒲〗`/`〖=屈〗`/`〖=耿〗`/`〖=魏〗`/`〖=殽〗`/`〖=郓〗`/`〖=社宫〗` 等封邑/战地裸字
