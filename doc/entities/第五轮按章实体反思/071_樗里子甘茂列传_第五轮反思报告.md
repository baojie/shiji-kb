## 三、典型案例分析

### 案例1：A113.1 全章邦国+王整合（16处）

**现象**：全章共16处"〖◆X〗王"模式（楚/秦/赵/齐/魏），均为邦国名后紧跟裸"王"字，依据A113.1应合并为职位标注。

**判据**：
- `〖◆楚〗王怒` → `〖;楚王〗怒`：楚王是职位引用，不是"楚国"作主体行动
- `〖◆秦〗王曰` → `〖;秦王〗曰`：同理
- 区别于：`〖◆楚〗兵去`（楚=邦国主体行动，保留`〖◆〗`）

**注意**：本章已存在 `〖@楚怀王〗`、`〖@秦昭王〗` 等全名标注，短语"楚王"/"秦王"不应重复全名，正确处理为职位`〖;X王〗`。

### 案例2：A87/A110 省称消歧（重点：公仲侈9处）

`〖@公仲〗`（L62/72/73/76多处）：韩大夫公仲侈省称"公仲"，首次全名`〖@公仲侈〗`在L58，后续省称9处全部消歧。

`〖@张子〗`（L54）：甘茂对秦武王陈述"天下不以多张子而以贤先王"——"张子"是张仪的字号省称（"子"是敬称后缀），消歧为`〖@张子|张仪〗`。

### 案例3：〖@樗里〗 类型辨析（L41）

同一行内出现两处"樗里"，类型不同：

```
〖@樗里子〗疾室在於〖@昭王|秦昭王〗庙西〖=渭南〗阴乡〖=樗里〗，故俗谓之〖@樗里子〗。
至汉兴，〖=长乐宫〗在其东，〖=未央宫〗在其西，〖=武库〗正直其墓。
〖◆秦〗人谚曰："力则〖@任鄙〗，智则〖@樗里|樗里子〗。"
```

- 第一处：`阴乡樗里` → 里邑地名 → `〖=樗里〗`
- 第二处：`智则樗里` → 樗里子省称 → `〖@樗里|樗里子〗`
