## 四、规律应用与验证

### 4.1 A77：敬称"足下"系统性漏标

本章作为权谋论辩章节，蒯通/武涉/广武君与韩信的对话密集使用"足下"（25处），是A77规律的高密度违规案例。**建议后续章节优先扫描长篇对话中的"足下"**。

扫描命令：
```bash

```

### 4.2 A29：动词短语误标人名

"以威王此三人"中"威王"是谓语动词短语，典型的动词短语误标为人名案例。
- "威"=以威势，"王"=称王（使动用法）
- 此类结构"以X王/以X君/以X师"均不应标为人名

### 4.3 A87：省称消歧遗漏的系统性

第四轮只对主人公"信"进行了消歧，但本章其他出现次数较多的人物（萧何11次、陈豨5次、钟离昧5次等）均未消歧，是系统性遗漏。
