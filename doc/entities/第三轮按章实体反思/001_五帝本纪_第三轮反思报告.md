## 一、消歧标注规范

### 核心规则

**消歧标注语法**（规律A8）：
- **时长/持续时间**：`〖%XX年|时长〗` ← 必须加 `|时长`
- **纪年格式（年月日）**：`〖%XX年〗` ← 无需消歧
- **默认规则**：无消歧标注时默认为纪年（年月日）

**为什么需要消歧**：
- "九岁" 可以是 "九年的时间长度" 或 "某个纪年的第九年"
- 无消歧标注时默认按纪年理解，所以时长必须显式标注 `|时长`

**语境对比**：
```

```

**分析**：
- 四句并列结构："其X也Y"
- 前三句都是形容词：郁郁（颜色）、嶷嶷（品德）、时（适时）
- 第四句的"士"也应为形容词，意为"合宜、得体"
- 非身份词"〖#士人〗"，也非官职"〖;士〗（司法官）"

```markdown
#### A8. 时长vs纪年（008章归纳，001章细化）

**消歧标注规范**：
- **时长/持续时间**：`〖%XX年|时长〗` - 必须加 `|时长` 消歧
- **纪年格式（年月日）**：`〖%XX年〗` - 无需消歧标注

**默认规则**：没有消歧标注时，默认为纪年格式（年月日）

**时长类型**（必须加 `|时长`）：
1. **任期时长**：在位XX年/载、立XX年、践帝位XX年
2. **经过时长**：XX年而老/崩、用XX岁功用不成
3. **守丧时长**：三年丧毕、三年四方莫举乐
4. **制度周期**：X岁一考功、X岁一巡狩
5. **发展阶段**：一年成聚，二年成邑，三年成都
6. **已经时长**：XX年矣

**纪年格式**（无需消歧）：
1. **编年纪年**：元年、二年、三年（编年体中的某年）
2. **月份日期**：正月、八月、正月上日、八月丙子
3. **完整纪年**：XX年XX月XX日

**判断关键三问**：
1. 能否替换为"第X年"且不改变语义？→ 能则为时长，加 `|时长`
2. 有无"而/矣/毕"等表示经过的虚词？→ 有则通常为时长
3. 是否为历法/编年体的时间坐标点？→ 是则为纪年，无需消歧
```

## 九、经验总结

### 核心发现

1. **消歧标注是关键**：时长与纪年必须明确区分，时长必须加 `|时长` 消歧标注
2. **默认规则很重要**：无消歧时默认为纪年，因此时长必须显式标注
3. **系统性问题需要规律细化**：19处时长全部缺失消歧，说明规律A8需要补充消歧语法说明

### 判断技巧

**快速判断是否需要 `|时长`**：
- 问1：能否替换为"第X年"？能 → 时长
- 问2：后面有"而/矣/毕"等虚词？有 → 通常是时长
- 问3：前面有"在位/立/践帝位"？有 → 时长
- 问4：是"X岁一..."格式？是 → 周期，属于时长

**举例**：
- "〖@尧〗立七十年得〖@舜〗" → 能替换为"在位第七十年得舜" → 时长 ✓
- "〖%正月上日〗" → 不能替换为"第正月上日" → 纪年日期 ✗
