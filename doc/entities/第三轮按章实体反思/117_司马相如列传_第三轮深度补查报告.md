## 章节特有发现

### 1. 嵌套标注问题严重性

本章发现的嵌套标注问题（6处）反映了复姓人名处理的系统性错误：

```
错误：〖@司〖~马〗长卿〗
正确：〖@司马长卿〗

错误：〖;〖~车〗骑〗
正确：〖;车骑〗
```

**问题根源**：标注者可能误将"马"、"车"理解为族群名词，忽略了它们在复合词中的语义单元性。

### 2. 数量词与人名的混淆

"百里"、"千里"等距离数量词易被误标为人名（如"百里奚"），需要根据语境判断：

- "方九百里" → 面积数量，`〖$百里〗`
- "闻乎数百里之外" → 距离数量，`〖$百里〗`

### 3. 时长标注消歧的重要性

本章6处时长标注均未加`|时长`消歧，反映了A8规律执行不到位。根据规律：

- 任期时长："居数岁"、"为郎数岁"、"宿卫十有馀年"
- 经过时长："治道二岁"、"三年於兹"、"既卒五岁"

均需添加`|时长`消歧标注。

### 4. 原文完整性违规

发现1处将全角冒号`：`改为半角冒号`:`，违反标注铁律。这种错误会导致文本完整性验证失败，必须严格避免。
