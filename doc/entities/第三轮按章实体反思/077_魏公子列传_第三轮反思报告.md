## 二、典型修正案例

### 案例1：时长消歧标注遗漏（规律A8）

**问题类型**：时长类时间缺少 `|时长` 消歧标注

#### 修正1：L7 "十馀年"

**位置**：L7
**原文**："不敢加兵谋魏十馀年"

**修正前**：
```
不敢〖[加兵〗谋〖◆魏〗〖%十馀年〗。
```

**修正后**：
```
不敢〖[加兵〗谋〖◆魏〗〖%十馀年|时长〗。
```

**修正理由**：
- "不敢加兵谋魏十馀年"表示诸侯不敢攻打魏国的持续时长
- 属于规律A8中的"经过时长"类型
- 根据第三轮反思要求，时长类时间必须加 `|时长` 消歧

**规律依据**：SKILL-03c-rules.md A8（时长vs纪年消歧规则）

#### 修正2：L33 "十年"

**位置**：L33
**原文**："公子留赵十年不归"

**修正前**：
```
〖@公子〗留〖◆赵〗〖%十年〗不归。
```

**修正后**：
```
〖@公子〗留〖◆赵〗〖%十年|时长〗不归。
```

**修正理由**：
- "留赵十年"表示信陵君留在赵国的持续时长
- 属于规律A8中的"经过时长"类型
- 必须加 `|时长` 消歧

**规律依据**：SKILL-03c-rules.md A8

### 案例2：器物误标族群（汉赋专项检查A3e）

**位置**：L23
**原文**："臣乃市井鼓刀屠者"

**修正前**：
```
〖#臣〗乃市井鼓〖~刀〗屠者
```

**修正后**：
```
〖#臣〗乃市井鼓刀屠者
```

**修正理由**：
- "鼓刀"是动词短语，表示操刀屠宰，不是单独的器物
- `〖~刀〗` 误将"刀"标注为族群，应该去除标注
- 类似汉赋章节中"车骑之众"误标 `〖~车〗` 的错误模式
- "鼓刀"作为固定词组，表示职业动作（屠宰），不需要拆分标注

**规律依据**：SKILL-03c-rules.md A3e 第3.e节（汉赋章节专项检查：族群误标检查）

**相关案例对比**：
- ❌ `〖~车〗骑之众`（车是器物，非族群）
- ❌ `〖~弓〗不虚发`（弓是器物，非族群）
- ❌ `鼓〖~刀〗屠者`（刀在此处是动词短语的一部分，非单独器物）
- ✅ `鼓刀屠者`（正确，不标注）
