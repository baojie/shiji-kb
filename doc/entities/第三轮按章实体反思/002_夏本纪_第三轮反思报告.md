## 一、消歧标注规范回顾

### 核心规则（规律A8）

**消歧标注语法**：
- **时长/持续时间**：`〖%XX年|时长〗` ← 必须加 `|时长`
- **纪年格式（年月日）**：`〖%XX年〗` ← 无需消歧
- **默认规则**：无消歧标注时默认为纪年（年月日）

**为什么需要消歧**：
- "九年" 可以是 "九年的时间长度"（时长） 或 "纪年的第九年"（纪年）
- 无消歧标注时默认按纪年理解，所以时长必须显式标注 `|时长`

### 判断关键三问

1. **能否替换为"第X年"且不改变语义？** → 是：时长（需`|时长`）
   - "用鲧治水九年而水不息" ≈ "用鲧治水九年后水不息" → 时长 ✓
2. **有语法助词"而/矣/毕"修饰？** → 是：通常是时长
   - "十七年而帝舜崩"、"三年丧毕" → 时长
3. **是历法/编年体的时间坐标点？** → 是：纪年格式（无需消歧）
   - "元年"、"正月上日" → 纪年
