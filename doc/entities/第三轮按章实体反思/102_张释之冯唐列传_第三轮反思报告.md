## 二、典型修正案例

### 案例1：时长消歧标注遗漏

**位置**：L44
**原文**："岁馀"

**修正前**：
```
〖@张廷尉〗事〖@景帝〗〖%岁馀〗
```

**修正后**：
```
〖@张廷尉〗事〖@景帝〗〖%岁馀|时长〗
```

**修正理由**：
- "岁馀"表示任职时长，属于时长类
- 根据A8规律，时长类必须添加`|时长`消歧标注
- 与本章已有的`〖%十岁|时长〗`、`〖%五日|时长〗`、`〖%终日|时长〗`保持一致

**规律依据**：SKILL-03c-rules.md A8

### 案例2：群体身份"长者"补标

**位置**：L12
**原文**：'上曰："长者也。"'

**修正前**：
```
上曰："长者也。"
```

**修正后**：
```
上曰："〖#长者〗也。"
```

**修正理由**：
- "长者"是群体性身份称谓，表示"德高望重之人"
- 符合规律A21（群体性身份/官职）
- 与同段后文`〖#长者〗`、`称为〖#长者〗`保持标注一致性

**规律依据**：SKILL-03c-rules.md A21

### 案例3：引文中身份词补标

**位置**：L58
**原文**：'"上古王者之遣将也...寡人制之...将军制之"'

**修正前**：
```
上古王者之遣将也，跪而推毂，曰阃以内者，寡人制之；阃以外者，将军制之。
```

**修正后**：
```
上古〖#王者〗之遣将也，跪而推毂，曰阃以内者，〖#寡人〗制之；阃以外者，〖#将军〗制之。
```

**修正理由**：
- "王者"表示君主身份（群体性）
- "寡人"是君主自称，属于身份类（规律A1）
- "将军"是军事统帅的群体性称谓
- 这三个身份词在引文语境中需要标注，体现礼制和军政关系

**规律依据**：SKILL-03c-rules.md A1, A21

### 案例4：自谦称呼"鄙人"补标

**规律依据**：SKILL-03c-rules.md A21

### 案例5：数量误标为时间

**位置**：L69
**原文**："二君之所称诵"
