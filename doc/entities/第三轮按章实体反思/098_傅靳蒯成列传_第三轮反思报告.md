第三轮深度反思发现并修正以下系统性问题（参照 095_樊郦滕灌列传 规律）：

1. **爵级 `^` → `;`**（1 处）：五大夫——秦汉二十等爵，属爵位而非制度。
2. **时长误标为时间 `%` → 去标**（10 处）：五岁/四月/一月/二岁/二十四年/十二年/
   三十一年/二十一年——任职/在位时长，去标注；保留纪年（孝惠五年/高后五年/
   孝文后三年/高祖十二年/孝文五年/孝景中二年/元鼎三年）。
3. **谥号 `;` → `@`**（6 处）：景侯/顷侯/共侯/肃侯/贞侯 作谥号当人名用；
   孝景作帝号 → `@`；高后 作帝号 → `@`。
4. **亲属关系去标**（6 处）：子×6 均去标（亲属关系描述不标注）。
5. **封号作人名省称 `;` → `@`**（2 处）：[2]"属淮阴"指韩信（淮阴侯省称）；
   [8] 黥布（英布，刑名人称）。
6. **身份通称 `@`/`;` → `#`**（2 处）：今上/主上 属身份通称，非官职亦非特指人名。
7. **俸禄 `;` → `^`**（1 处）：二千石/五百石为俸禄制度。
8. **官职+人名复合拆分**（1 处）：`〖;楚王信〗` → `〖◆楚〗王〖@信〗`
   （对齐 095 章"楚王信"处理惯例）。
9. **数字前缀遗漏**（2 处）：`四千一百八〖$十人〗` → `〖$四千一百八十人〗`；
   `二千〖$四百人〗` → `〖$二千四百人〗`。
