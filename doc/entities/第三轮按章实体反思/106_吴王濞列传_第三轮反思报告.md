## 四、章节特有发现

1. **封王省称的双重用法**：`胶西/胶东/菑川/济南/济北` 既可作封王省称（"胶西王卬" → `@胶西王`），也可作邦国/地名（"胶西为渠率" / "使於胶西" / "徙王菑川"）。本章两种用法混用密集，需逐处判断：带 "王" 的封号省称保留 `@`，单独指国/地改 `◆`。
2. **七国之乱长篇的时长密度**：本章是全书少见的"时长"高频章节：`五十年/四十馀年/二十馀年×2/三十馀年×2/十馀日/三月/十月/万世` 均需 `|时长` 补齐；而 `孝景帝三年/前二年/七年/十一年/二月/正月/三月` 等作纪年保持 `〖%〗` 无 `|时长`。
3. **说辞中 "夫" 作发语词**：与 104、105 章发现一致（"且夫察见渊中鱼"），应去标。
4. **"诸吕" 应为氏族**：`〖@吕〗` → `〖&吕〗`，与 106 之前的汉初叙事（吕后本纪）标注规范一致。
5. **"百里" 在"地不过百里"中为数量**：虽然与先秦人物"百里奚"同形，此处当判数量 `〖$〗`。
6. **战场方位副词系统性误标**：`惊东南/备西北/从西北入` 等军事方位表达，应去标，与 A 类规律 `方位副词不标注` 一致。

- **A73（拟）**：封王号地名二义性——`胶西/胶东/菑川/济南/济北/淮南/长沙` 等"方位/地域+非'国'字"类短语，在 "与X"、"使於X"、"徙王X"、"X群臣" 等语境指邦国/地名，标 `〖◆〗`；在 "X王" 紧随其后或作封号省称时保留 `〖@X王〗`。
