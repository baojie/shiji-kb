## 二、典型修正案例

### 案例1：时长消歧标注缺失（规律A8）

**位置**：L71, L77, L87, L81, L109, L145
**问题**：时长类时间未添加`|时长`消歧标注

**修正前**：
```markdown
〖%数十年〗竟为〖◆秦〗所⟦◈灭⟧
〖%百有馀年〗，〖◆汉〗有〖@贾生〗
〖%二十馀年〗，〖_天下〗和洽
〖%一岁中〗至〖;太中大夫〗
〖%三年〗，有〖+鸮〗飞入
〖%百年〗之後，空悲吊〖=湘〗
```

**修正后**：
```markdown
〖%数十年|时长〗竟为〖◆秦〗所⟦◈灭⟧
〖%百有馀年|时长〗，〖◆汉〗有〖@贾生〗
〖%二十馀年|时长〗，〖_天下〗和洽
〖%一岁中|时长〗至〖;太中大夫〗
〖%三年|时长〗，有〖+鸮〗飞入
〖%百年|时长〗之後，空悲吊〖=湘〗
```

**修正理由**：
- 符合规律A8（时长vs纪年消歧规则）
- "数十年"="经过数十年"，表示时长
- "百有馀年"="一百多年"，表示时长
- "二十馀年"="二十多年"，表示时长
- "一岁中"="一年之内"，表示时长
- "三年"="（任职）三年"，表示任期时长
- "百年"="一百年之後"，表示时长

### 案例2：制度类型边界修正

**位置**：L87
**原文**："诸律令所更定"

**修正前**：
```markdown
诸〖^律令〗所更定
```

**修正后**：
```markdown
〖^诸律令〗所更定
```

**修正理由**：
- "诸律令"是一个整体概念，指"各种律令"
- "诸"是修饰词，应与"律令"一起标注为制度
- 类似于"诸侯"应标注为`〖#诸侯〗`而非"诸〖;侯〗"

**规律依据**：SKILL-03f（边界错误）修饰词+实体整体标注原则
