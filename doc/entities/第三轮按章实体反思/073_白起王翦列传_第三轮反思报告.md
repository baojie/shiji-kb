## 二、典型修正案例

### 案例1：时长消歧标注遗漏（规律A8）

根据规律A8，时长类时间必须添加 `|时长` 消歧标注，以区分纪年格式。

#### 修正1：持续天数
**位置**：L17
**原文**："不得食四十六日"

**修正前**：
```
〖=赵〗卒不得食〖%四十六日〗
```

**修正后**：
```
〖=赵〗卒不得食〖%四十六日|时长〗
```

**修正理由**：
- "四十六日"表示持续时间（四十六天），不是纪年日期
- 属于时长类第6类：已经时长
- 符合规律A8：时长必须加 `|时长` 消歧

**规律依据**：SKILL-03c-rules.md A8

#### 修正2：日久（时间久远）
**位置**：L21, L25（2处）

**修正前**：
```
〖_天下〗不乐为〖=秦〗〖#民〗之〖%日久〗矣
彼〖#诸侯〗怨〖=秦〗之〖%日久〗矣
```

**修正后**：
```
〖_天下〗不乐为〖=秦〗〖#民〗之〖%日久|时长〗矣
彼〖#诸侯〗怨〖=秦〗之〖%日久|时长〗矣
```

**修正理由**：
- "日久"表示时间持续很长，是时长表述
- 类似"岁馀"、"月馀"等表达方式
- 必须加 `|时长` 消歧标注

#### 修正3：三日三夜
**位置**：L35

**修正前**：
```
荆人因随之，〖%三日〗三夜不顿舍
```

**修正后**：
```
荆人因随之，〖%三日|时长〗三夜不顿舍
```

**修正理由**：
- "三日三夜"表示持续三天三夜，是时长表述
- 不是编年纪年的"三日"（第三天）
- 符合规律A8第2类：经过时长

#### 修正4：三世将
**位置**：L47（2处）

**修正前**：
```
为将〖%三世〗者必⟦◈败⟧
今〖@王离〗已〖%三世〗将矣
```

**修正后**：
```
为将〖%三世|时长〗者必⟦◈败⟧
今〖@王离〗已〖%三世|时长〗将矣
```

**修正理由**：
- "三世"表示三代的时间跨度，是时长概念
- "为将三世"="当了三代的将军"，是任期时长
- 不是编年纪年的"三世"（某朝代第三世）

**判断关键**：
- 能否替换为"第X年"？否（"为将第三世"不通）
- 是否表示持续/累计时间？是（三代累计）
- 因此是时长，需加 `|时长`

### 案例2：邦国并列表述（规律B1）

**位置**：L9
**原文**："韩赵为一，则可以当秦"

**修正前**：
```
〖=韩〗〖◆赵〗为一
```

**修正后**：
```
〖◆韩〗〖◆赵〗为一
```

**修正理由**：
- "韩赵为一"中的"韩"与"赵"并列，都指代国家政治实体
- "韩赵联合"="韩国和赵国联合"，两者语法地位相同
- 根据规律B1，单字邦国名在政治语境中应标注为邦国〖◆〗
- 此处"韩"与"赵"都是政治主体，应使用相同标注符号

**规律依据**：SKILL-03c-rules.md B1（单字邦国名判断）

### 案例3：赋体文本中的"荆"字邦国标注

**位置**：L63
**原文**：赞曰"递为秦将，拔齐破荆"

**修正前**：
```
递为〖=秦〗将，⟦◈拔⟧〖◆齐〗⟦◈破⟧荆
```

**修正后**：
```
递为〖=秦〗将，⟦◈拔⟧〖◆齐〗⟦◈破⟧〖◆荆〗
```

**修正理由**：
- 赞曰属于韵文体裁，"荆"在此指代楚国
- "齐"和"荆"并列，都是军事动词的受体
- "拔齐破荆"对仗工整，"齐"标注为〖◆齐〗，"荆"也应标注为〖◆荆〗
- 战国时期"荆"是楚国的别称（荆楚）

**判断依据**：
- 语法并列：拔X破Y，X、Y应使用同类标注
- 语义对应：齐国、楚国都是六国之一
- 韵文特征：对仗工整的赋体文本，实体标注应保持一致性
