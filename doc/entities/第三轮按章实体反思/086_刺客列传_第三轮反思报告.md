## 一、执行摘要

### 1.1 核心发现

086_刺客列传记五大刺客（曹沫/专诸/豫让/聂政/荆轲）及高渐离扑筑刺秦事。本章标注存在以下系统性问题：

1. **「太子丹」被误作官职合并标注（8 处）**：本章「太子丹」非官职而为「太子（身份）+丹（人名）」组合，按规律应拆分为 `〖#太子〗〖@丹〗`。原标注 `〖;太子丹〗` 共 8 处（L172、L207 ×3、L214、L227），全部修正。
2. **「太子」通称误标官职（14+ 处）**：`〖;太子〗` 统一修正为 `〖#太子〗`（身份），对齐 085 章规律。
3. **「燕王喜」的「喜」为人名而非官职（3 处）**：L207 ×2、L209 中 `〖;喜〗`（燕王名喜）全部修正为 `〖@喜〗`。
4. **合并标注拆分（3 处）**：
   - L191 `〖@中庶子蒙嘉〗` → `〖;中庶子〗〖@蒙嘉〗`（官职+人名）
   - L200 `〖;侍医夏无且〗` → `〖;侍医〗〖@夏无且〗`
   - L207 `〖;代王嘉〗` → `〖;代王〗〖@嘉〗`
5. **亲属关系词去标（5 处）**：
   - L174 `〖#父〗〖#母〗〖#宗族〗` → `父母宗族`
   - L237 `〖#弟〗` → `弟`（太史公赞四字句「彰弟哭市」）
   - L239 `〖#夫〗` → `夫`（「懦夫」中的名词 dweeb，非身份通称）
6. **发语词「夫」去标（1 处）**：L172 `〖#夫〗〖@樊〗将军` → `夫〖@樊〗将军`
7. **通称身份误标（3 处）**：`〖;宾客〗`→`〖#宾客〗`、`〖;勇士〗`→`〖#勇士〗`、`〖;庸保〗/〖;庸〗/〖;上客〗` → `〖#...〗`（L214 ×4）
8. **误标语录为天文（1 处）**：L221 `〖!甚矣吾不知人也〗` 明显误标——此为俗语/感叹，非天文术语，去标。
9. **时长消歧补充（1 处）**：L209 `後〖%五年〗` → `後〖%五年|时长〗`（秦灭燕在行刺后五年）
10. **半角引号修复（2 处）**：L227 `"天雨粟，马生角"` 含半角 `"`，修复为全角 `""`，与原文字符一致。

### 1.2 主要问题

- **人名识别疏漏**：本章涉及「太子丹」「燕王喜」「代王嘉」「侍医夏无且」「中庶子蒙嘉」等"官/号+名"结构，标注器普遍将其合并标为 `;官职`，未识别出其中的人名部分。
- **「太子丹」是本章高频错误**：第三轮反思专项提示「太子丹非官职而为人名」，实测 8 处需修正，占比最高。
- **太史公赞（L235-239）四字骈文中亲属/泛称词被误标**：赞文中「彰弟哭市」「懦夫增气」的「弟」「夫」被误作身份标注。
