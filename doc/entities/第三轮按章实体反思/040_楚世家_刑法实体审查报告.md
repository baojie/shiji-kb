## 三、典型案例分析

### 案例1：军事动词误标为刑法实体
**原标注**：
```
〖◆楚〗〖[伐〗〖◆随〗
〖[灭〗之
〖[败〗〖@子玉〗於〖=城濮〗
```

**问题分析**：
- "伐"、"灭"、"败"是军事动词，描述战争行为
- 不是刑名或法律术语
- 应改为`⟦◈〗`格式或不标注

**建议修正**：
```
〖◆楚〗⟦◈伐〗〖◆随〗
⟦◈灭〗之
⟦◈败〗〖@子玉〗於〖=城濮〗
```
或完全去标注（如果不强调战争行为）

## 四、修正建议

### 建议1：全部改为动词标注（保留语义信息）
将所有`〖[动词〗`改为`⟦◈动词〗`或`⟦◉动词〗`

**优点**：
- 保留了这些动词的语义标注
- 区分了动词与真正的刑名实体
- 便于后续知识图谱中区分"行为"与"刑罚"

**工作量**：
- 需要批量替换约350+处标注
- 脚本示例：`〖[` → `⟦◈`

---

### 建议2：完全去标注（简化处理）
移除所有军事动词和杀戮动词的标注，只保留人名、地名、邦国等核心实体

**优点**：
- 大幅简化标注密度
- 这些动词在史记中出现频率极高，标注意义有限
- 减少视觉干扰，聚焦核心实体

**缺点**：
- 丢失了部分语义信息
- 无法在知识图谱中查询"楚国伐过哪些国家"

**工作量**：
- 需要批量删除约350+处标注
- 脚本示例：`〖[伐〗` → `伐`

---

### 建议3：部分保留+部分去标注（推荐）
**保留标注**（使用`⟦◈〗`）：
- 特殊军事行为：灭、围、拔、刺（结果性强）
- 特殊杀戮行为：弑、诛（政治性强）

**去除标注**：
- 常规军事动词：伐、攻、击、败、破、取、救、袭、射
- 常规杀戮动词：杀、斩、虏

**优点**：
- 平衡了语义保留与简洁性
- 保留了最有价值的标注
- 减少了大量重复标注

**工作量**：
- 保留约60处特殊动词标注
- 删除约290处常规动词标注

import re

# 保留标注的动词列表
keep_verbs = ['灭', '围', '拔', '弑', '诛', '刺', '虏', '作乱', '相攻', '获罪', '僇', '劫']

# 去除标注的动词列表
remove_verbs = ['伐', '攻', '击', '败', '破', '取', '救', '袭', '射', '杀', '斩', '获', '执', '克']

def fix_criminal_entity(file_path):
    with open(file_path, 'r', encoding='utf-8') as f:
        content = f.read()

    # 保留的动词：〖[X〗 → ⟦◈X〗
    for verb in keep_verbs:
        content = content.replace(f'〖[{verb}〗', f'⟦◈{verb}〗')

    # 去除的动词：〖[X〗 → X
    for verb in remove_verbs:
        content = content.replace(f'〖[{verb}〗', verb)

    with open(file_path, 'w', encoding='utf-8') as f:
        f.write(content)

    print(f"✓ 修正完成：{file_path}")

if __name__ == '__main__':
