## 一、执行摘要

### 1.1 核心发现

088《蒙恬列传》篇幅较短（67 行），叙蒙氏三世仕秦、北逐匈奴、沙丘之谋、蒙毅蒙恬之死。系统性标注问题：

1. **通称身份 `;→#`（约 15 处）**：太子/君/大夫/陛下/良臣/忠臣/谋士 等敬称/通称统一改 `#`。
2. **合并标注拆分（5 处）**：`;公子胡亥` ×3 / `;公子扶苏` / `;御史曲宫` 全部拆分。
3. **谥号/帝号误作官职（5 处）**：`;成王` ×3、`;昭襄王`、`;二世皇帝` 改 `@`。
4. **"X王Y"合并拆分并改人名（3 处）**：`;赵王迁` / `◆燕 王;喜` / `◆齐 王;建` / `;吴王夫差` 统一改 `◆邦国`+`;王`+`@人名`。
5. **人名误作官职（2 处）**：`;子婴` → `@子婴`；`;后胜`（齐相）→ `@后胜`。
6. **时长消歧（4 处）**：`%十馀年`（暴师于外）/ `%日夜` / `%一旦` / `%三世`（积功三世）全部加 `|时长`。
7. **文学辨析（1 处）**：`;文学`（"书狱典文学"）→ `^文学`（学问类别，规律 A44）。
8. **"原大夫"去合并**：`;原大夫` → `原〖#大夫〗`（"原"=愿，非姓氏）。

- **家世段（L5）**：`;文学` → `^文学`。
