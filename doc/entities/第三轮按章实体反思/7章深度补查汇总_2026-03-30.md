## 二、主要发现

### 2.1 系统性问题（跨章节）

#### 问题1：国名误标为地名（242处）

**发现时间**: 批量扫描阶段  
**影响章节**: 051(16处)、078(226处)  
**问题描述**: 战国七雄及主要诸侯国被错误标注为地名`〖=国〗`而非国名`〖'国〗`

**修正示例**:
```
❌ 〖=楚〗攻〖'韩〗  
✓ 〖'楚〗攻〖'韩〗

❌ 〖=秦〗之所以重〖'楚〗者  
✓ 〖'秦〗之所以重〖'楚〗者
```

**根本原因**: 标注初期对国名vs地名的语境判断规则不明确

**解决方案**: 
1. 批量替换明确的国名列表（秦齐楚燕韩赵魏等17国）
2. 在SKILL规范中明确：诸侯国国号统一使用`〖'〗`

---

#### 问题2：国名标注完全缺失（15处）

**发现章节**: 063_老子韩非列传  
**问题描述**: 整章国名标注为0，所有邦国提及均未标注

**补充国名**:
- 楚(4): "楚苦县"、"楚人也"、"楚将"
- 周(2): "周守藏室史"、"周太史儋"
- 魏/汉/齐/郑/韩/秦/宋/卫: 各1处

**根本原因**: 思想家列传标注时过度关注思想词汇，忽视基础实体

---

#### 问题3："千乘"地名误标（5处）

**发现章节**: 067(2)、069(3)  
**问题描述**: "千乘"在军力描述语境中被误标为地名

**修正规则**:
```
语境1：国家规模
"〖$千乘〗之国" = 拥有千辆战车的国家（数量）
❌ 〖•车〗〖=千乘〗  
✓ 〖•车〗〖$千乘〗

语境2：地名
"临〖=千乘〗" = 千乘郡（地名，在今山东）
✓ 保持〖=千乘〗
```

---

#### 问题4：时长消歧遗漏（10处）

**发现章节**: 070_张仪列传  
**问题描述**: 第三轮反思引入`|时长`消歧规范后，部分时长标注未添加消歧符

**修正示例**:
```
❌ 〖%三月〗  
✓ 〖%三月|时长〗

❌ 〖%十五年〗  
✓ 〖%十五年|时长〗
```

**规律**: 外交说辞中时长描述密集，易遗漏消歧

#### 069_苏秦列传：远古帝王遗漏（6处）

**问题**: 外交辞令引用古代圣王（尧舜禹汤武纣）时未标注人名

**典型案例**:
```
❌ 尧无三夫之分  
✓ 〖@尧〗无三〖#夫〗之分

❌ 汤武之所以放弑而争也  
✓ 〖@汤〗〖@武〗之所以放⟦◉弑⟧而争也
```

**规律**: 外交辞令类章节易遗漏历史人物引用

---

#### 070_张仪列传："梁"字国名遗漏（9处）

**问题**: "梁"作为魏国别称（都城大梁）时未标注

**语境**:
- "从郑至梁" = 从郑国到魏国
- "梁南与楚" = 魏国南边与楚国接壤
- "秦驱韩梁攻齐" = 秦国驱使韩国魏国攻打齐国

**规律**: 地名与国名的转换需要语境判断（梁邑vs梁国）
