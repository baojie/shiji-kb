## 规律应用与新发现

### 应用的既有规律

1. **A49 - 天下语境判断**: 发现2处，1处修正为地名类，1处去标注
2. **A21/A48 - 群体称谓身份标注**: 发现18处遗漏，主要在辩论段落
3. **A10 - 思想概念标注**: 发现15处遗漏，商君列传思想辩论密集
4. **A17 - 易遗漏器物**: 发现12处，度量衡器具和物质奖励物品
5. **A44 - 典籍名称识别**: 发现3处，引用《诗》《书》

### 章节特有发现

#### 发现1: 变法理论辩论中的群体称谓密集

**现象**: [3]段变法讨论中，"知者""愚者""智者""常人""学者"等群体称谓高频出现，形成对比论述结构。

**示例**:
```
〖#知者〗见於未萌 / 〖#愚者〗闇於成事
〖#智者〗作法 / 〖#愚者〗制焉
〖#常人〗安於故俗 / 〖#学者〗溺於所闻
```

**规律提炼**: 法家理论辩论段落中，群体对比称谓需全覆盖标注，易形成8-12对的对比结构。

#### 发现2: 思想概念在劝谏文中的高密度

**现象**: [9]段赵良劝谏中，"义""道""贤""德""教""寿""功"等思想概念高频出现（15处）。

**规律提炼**: 道德劝谏类文字中，思想概念密度是普通叙事段落的3-5倍，需系统性扫描。

#### 发现3: 度量衡器具的完整标注

**现象**: "平斗桶权衡丈尺"中"斗""桶""权""衡"均需标注为器物。

**说明**: 度量衡改革是商鞅变法重要内容，每个度量器具都是实物，需完整标注。
