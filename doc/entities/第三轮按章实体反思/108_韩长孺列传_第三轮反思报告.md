## 三、典型修正案例

### 1. 时长消歧遗漏（规律A8）

**现象**：时长词缺少`|时长`消歧标注，可能与纪年混淆。

| 行号 | 原标注 | 修正后 | 上下文 | 说明 |
|------|--------|--------|--------|------|
| L11 | `〖%四岁〗` | `〖%四岁\|时长〗` | "安国为御史大夫四岁馀" | 担任官职的时间长度 |
| L11 | `〖%数月〗` | `〖%数月\|时长〗` | "安国病免数月" | 病免的时间长度 |
| L12 | `〖%明年〗` | `〖%明年\|次年〗` | "明年，匈奴大入边" | 指次年，非纪年 |
| L13 | `〖%数月〗` | `〖%数月\|时长〗` | "数月，病欧血死" | 距离事件的时间长度 |

**规律总结**：
- "X岁"、"X月"、"X日"后跟"馀/余"时，通常表示时长
- "明年"在叙事中通常指"次年"，应消歧为`|次年`
- 时长消歧是A8规律的核心要求，防止与纪年混淆

**修正方法**：
```bash
# 检测未消歧的时长词
grep -nE '〖%[^〗]*[岁年月日][馀余]〗' chapter_md/108_*.tagged.md | grep -v '|时长'
```

### 2. 语气词误标为身份（规律A1）

**现象**：发语词"夫"被误标为身份标注`〖#夫〗`。

| 行号 | 原标注 | 修正后 | 上下文 | 说明 |
|------|--------|--------|--------|------|
| L2 | `〖#夫〗〖%前日〗` | `夫〖%前日〗` | "夫前日吴楚七国反时" | "夫"是发语词，非身份 |
| L5 | `〖#夫〗〖@太上〗` | `夫〖@太上皇〗` | "夫太上、临江亲父子之间" | "夫"是发语词+"太上"应为完整人名 |

**辨析**：
- **身份标注**：`〖#夫〗` = 丈夫、成年男子（名词）
  - 示例：`〖#夫〗见之` = 她的丈夫看到了
- **发语词**：`夫` = 句首语气词，无实义（虚词）
  - 示例：`夫天下者，天下人之天下也` = （发语词）天下是天下人的天下

**判断依据**：
1. 看位置：句首的"夫"通常是发语词
2. 看语义：发语词可去掉不影响句意
3. 看搭配：发语词后常接时间词、判断句

**修正方法**：
```bash
# 检测可能的发语词误标
grep -nE '〖#夫〗〖%' chapter_md/108_*.tagged.md
```

## 五、新发现的规律

### 规律补充：发语词"夫"的判断标准

**现象**：本章发现2处"夫"字误标为身份`〖#夫〗`，实为发语词。

**判断标准**（完整版）：

| 判断维度 | 身份标注`〖#夫〗` | 发语词（不标注） |
|---------|----------------|---------------|
| **位置** | 句中、句尾 | 句首 |
| **语义** | 丈夫、成年男子（名词） | 无实义（虚词） |
| **搭配** | 后接动词、名词 | 后接时间词、判断句、论述 |
| **可删除性** | 不可删除（影响句意） | 可删除（不影响句意） |
| **典型示例** | "夫人曰"（夫人说） | "夫天下者"（发语词） |

**典型错误案例**（本章）：
- `〖#夫〗〖%前日〗吴楚七国反时` ❌
  - 正确：`夫〖%前日〗吴楚七国反时` ✅
  - 理由：句首发语词，可删除不影响句意

- `〖#夫〗〖@太上〗、〖@临江〗亲〖#父子〗之间` ❌
  - 正确：`夫〖@太上皇〗、〖@临江王〗亲〖#父子〗之间` ✅
  - 理由：句首发语词 + 人名应完整

**适用章节**：全书所有含"夫"字章节

**检测方法**：
```bash
# 检测可能的发语词误标
grep -rE '〖#夫〗〖[%@◆]' chapter_md/*.tagged.md
```
