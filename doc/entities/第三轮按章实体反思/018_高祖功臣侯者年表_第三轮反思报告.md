## 一、消歧标注规范(年表专用)

### 核心规则

**年表中的时间表达**:
- **任职时长**: `〖%XX岁/年|时长〗` ← 必须加 `|时长`
- **从属时间**: `〖%XX岁/年|时长〗` ← 表示跟随某人多久
- **纪年格式**: `〖%XX年〗` ← 无需消歧(如"六年"、"元年")

**判断标准**:
- 如果表示"持续了多久"、"任职了多久"、"跟随了多久" → 必须消歧为 `|时长`
- 如果是具体的年号、纪年 → 无需消歧
