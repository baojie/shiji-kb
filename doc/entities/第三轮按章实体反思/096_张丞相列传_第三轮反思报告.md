## 二、典型修正案例

### 案例1: 嵌套标注错误修正

**位置**: L25
**原文**: "持御史大夫印弄之"

**修正前**:
```
〖@高祖〗持〖•御史大夫#夫〗印〗弄之
```

**修正后**:
```
〖@高祖〗持〖•御史大夫印〗弄之
```

**修正理由**:
- 原标注错误地在器物标注内嵌套了身份标注`#夫`
- "御史大夫印"是完整的器物名称,应整体标注
- 嵌套导致标注符号损坏(多了一个`〗`)

**规律依据**: SKILL-03c Step 1.a(标注边界损坏)

### 案例2: 典籍标注格式错误

**位置**: L54(3处)、L89、L91(2处)
**原文**: "礼"、"诗"

**修正前**:
```
怠慢之〖{{礼〗
朝廷之〖{{礼〗
不为〖{{礼〗
明於〖{{诗〗
受〖{{诗〗
好〖{{诗〗
```

**修正后**:
```
怠慢之〖{礼〗
朝廷之〖{礼〗
不为〖{礼〗
明於〖{诗〗
受〖{诗〗
好〖{诗〗
```

**修正理由**:
- 典籍标注应使用单大括号`〖{`，而非双大括号`〖{{`
- 这是格式错误，可能是早期标注时的失误

**规律依据**: SKILL-03a实体标注规范(典籍类标注格式)

### 案例4: 误标人名去除

**位置**: L73、L75
**原文**: "得食其故国邑"

**修正前**:
```
得〖@食其〗故国邑
```

**修正后**:
```
得食其故国邑
```

**修正理由**:
- "食其"在此处不是人名(郦食其),而是"食+其"(享受+其)
- 语境："得食其故国邑"="享受其原封邑"
- 若为人名,应是"〖@郦食其〗",而非单独"食其"

**规律依据**: 语境判断,人名误标检查
