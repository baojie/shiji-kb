### 2. 族群误标（1处）

#### 修正3：L23 "鼓刀"中的"刀"误标族群

**行号**：L23
**原文**："臣乃市井鼓刀屠者"

**修正前**：
```markdown
〖#臣〗乃市井鼓〖~刀〗屠者
```

**修正后**：
```markdown
〖#臣〗乃市井鼓刀屠者
```

**修正类型**：族群误标
**规律依据**：A3e（汉赋章节专项检查：族群误标）
**修正说明**：
- "鼓刀"是动词短语，表示操刀屠宰
- `〖~刀〗` 误将"刀"标注为族群，应该去除
- 类似汉赋中"车骑"误标 `〖~车〗` 的错误模式
- "鼓刀"作为固定词组，表示职业动作，不拆分标注
