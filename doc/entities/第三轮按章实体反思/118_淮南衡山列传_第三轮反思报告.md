3. **时长消歧 `|时长`（12 处）**：`一世`/`千岁`/`百年`/`万世`/`一年`/`十馀日`/`三月 ×3`/`日夜 ×4` 全部加 `|时长`。
4. **单字"日"时点/量词去标（8 处）**：`前日`/`日五斤`/`一日晏驾`/`一日过成皋`/`一日发兵`/`会日`/`死之日`/`日益亲幸` 全部去 `%日` 标注（非纪年）；`冬`（"元年冬"后附季节词）亦去标。

## 章节特有发现（候选规律）

1. **"厉王"兼为谥号与称呼**：本章"厉王"出现 20+ 次，全部指代刘长（其人追尊为厉王）。统一归 `@人名` 而非 `;官职`，避免在同一段中一会儿指人一会儿指谥号造成的歧义。类似规律：任何"追谥为 X 王"后全章凡指代该人的"X 王"都应归人名。
2. **说辞长段中的"夫"多为发语词**：伍被谏辞连用 5 次"夫..."，全是发语词（不标）。规律：凡"夫"后接抽象主语（秦、吴王、吴越之众、纣、荆楚）或判断句，"夫"为发语词必去标。
3. **"一日"表示"某一天/有一天"是量词，非纪年**：本章多处"一日晏驾/一日过成皋/一日发兵"均为量词表达，与"日五斤"（每日）、"前日"（过去日子）一样应去 `%日` 标注。
4. **"王孙 + 人名"中的"王孙"是亲属称谓**：如"王孙建"指"王的孙子建"，"王孙"应去标（亲属称谓），只留 `@建`。
5. **"令 X 居 Y" 被误标地名"令居"的陷阱**：L101 "号曰将军，令居外宅" 中"令居"实为"使（孝）居住"，非地名令居县，应去标。属"两字组合误为地名"陷阱（与 117 章"下相"同理）。
6. **妾室通称 `美人/才人` 归身份**：`#美人`/`#才人`，与"陛下""天子"同类，非具体官职。
7. **"寿春庄芷"系"地名 + 姓氏 + 名"三段式**：应拆为 `〖=寿春〗〖&庄〗〖@芷〗`，寿春是县名、庄是姓、芷是名。本章原标注 `〖;寿春〗〖;庄〗〖@芷〗` 误地名为官职、误姓氏为官职。
