### 8.1 同形名在合传章节的典型风险

`〖@布〗` 在本章 16 处分指两人（季布、栾布），**超过 A87 规则"歧义风险"的所有场景**。合传章节（如「季布栾布列传」、「廉颇蔺相如列传」、「孙子吴起列传」等）必须：

1. **识别合传主人公的共字**（季布/栾布共"布"；颇/相如无共字）。
2. **明确分界段落**（以传记开头小节为界）。
3. **全章 `〖@共字〗` 逐一分辨**。

**建议加入规律库**（候选 A87a）：

> **A87a 候选**：合传章节主人公同字省称必须按所在传分别消歧。扫描命令：
> ```bash
> grep -oE '〖@[一-龥]〗' chapter_md/NNN_*.tagged.md | sort | uniq -c | sort -rn
> ```
> 出现 ≥ 5 次的单字 `〖@X〗` 须检查合传是否造成多人共称。

### 8.2 E3 扩展：全角引号方向也可错

本章证明 E3 扫描**不仅需查半角引号**，也需检查**全角引号的左右配对是否一致**。

**扫描命令建议扩展**：
```bash
python3 -c "
with open('chapter_md/NNN.tagged.md') as f: c=f.read()
ldq=c.count(chr(0x201c)); rdq=c.count(chr(0x201d))
lsq=c.count(chr(0x2018)); rsq=c.count(chr(0x2019))
print(f'LDQ={ldq} RDQ={rdq} 差={abs(ldq-rdq)}')
print(f'LSQ={lsq} RSQ={rsq} 差={abs(lsq-rsq)}')
"
```

若 LDQ ≠ RDQ 或 LSQ ≠ RSQ，需对照原文修复方向错误（非数量问题）。

**建议加入 E3**：
> **E3 扩展**：全角引号左右数量必须相等（LDQ == RDQ，LSQ == RSQ）；不等时须对照原文修复方向错误。
