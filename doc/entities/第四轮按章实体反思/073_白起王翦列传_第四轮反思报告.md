# 073 白起王翦列传 · 第四轮反思报告

## 一、章节概况

白起王翦列传记秦国两大名将白起、王翦之事迹。白起以伊阙之战、鄢郢之战、长平之战著称，王翦以灭楚六十万之师闻名。全篇以密集军事叙事为主，涉及大量邦国、将领、战功数字。

### 2.1 邦国升级（~32处）

全章所有七雄名（秦/楚/齐/魏/韩/赵/燕）及晋从 `〖=X〗`（地名）升级为 `〖◆X〗`（邦国），符合 A107 规则。

典型修正：
- `⟦◈攻⟧〖=韩〗` → `⟦◈攻⟧〖◆韩〗`（军事动词宾语）
- `〖=楚〗王亡去` → `〖◆楚〗王亡去`（政治主体）
- `三〖=晋〗将` → `三〖◆晋〗将`（邦国联称）

### 2.2 单字名消歧（~14处）

| 原标注 | 修正 | 说明 |
|--------|------|------|
| `〖@起〗` | `〖@起\|白起〗` | 白起省称 |
| `〖@昭王〗` | `〖@昭王\|秦昭王〗` | 秦昭王消歧 |
| `〖@穰侯〗` | `〖@穰侯\|魏冉〗` | 穰侯消歧 |
| `〖@武安君〗` | `〖@武安君\|白起〗` | 武安君消歧 |
| `〖@错〗` | `〖@错\|司马错〗` | 秦客卿 |
| `〖@茄〗` | `〖@茄\|赵茄〗` | 赵裨将 |

### 2.3 数量边界修正（~17处）

系统性问题：数字前缀在 `〖$〗` 标注外。

典型修正：
- `二〖$十四万〗` → `〖$二十四万〗`
- `七〖$十万人〗` → `〖$七十万人〗`
- `十〖$三万〗` → `〖$十三万〗`
- `二〖$万人〗` → `〖$二万人〗`

### 2.4 时间标注修正（~11处）

纪年（如"三十四年"）误标为 `|时长`。

典型修正：
- `〖%三十四年|时长〗` → `〖%三十四年〗`（纪年，非时长）
- `〖%四十三年|时长〗` → `〖%四十三年〗`
- `〖%四十四年|时长〗` → `〖%四十四年〗`

### 2.5 政治动词（4处）

- `封〖@` → `⟦○封⟧〖@`
- `谥〖@` → `⟦○谥⟧〖@`

### 2.6 段落拆分

密集战功段落换行，提升可读性（净增29行）。
