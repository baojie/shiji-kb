## 四、跨章一致性疑点

1. **大人赋神灵体系**：本章大人赋段有30+处神灵，属全书最密集的神灵标注区。已全部从 `〖?〗` 迁移至 `〖_〗`（神灵本体）/`〖+〗`（神兽）/`〖=〗`（仙界地名）。建议与 028 封禅书、012 孝武本纪的神灵标注交叉验证。

2. **赋体"君"字高频**：上林赋/封禅文中"君"字19处全部从 `〖;〗` 改为 `〖#〗`（泛称身份），需确认封禅书同类处理一致。

3. **杨雄/扬雄**：原文写"杨雄"（太史公曰段），这是《史记》原文用字，消歧为"扬雄"（规范名）。后续如有统一策略需回顾。

4. **"百里"人名误标**：本章2处"百里"实为数量（方九百里、数百里之外），被误标为 `〖@百里〗`。建议全书扫描 `〖@百里〗` 排查同类误标。
