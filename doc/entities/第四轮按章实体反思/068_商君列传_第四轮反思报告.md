### 8.2 商鞅名称链的 A87 系统性缺失

本章主人公在 60+ 处使用 4 种不同称谓（鞅/卫鞅/商君/公孙鞅）均未消歧,严重违反 A87 单字名/别名/封号必须消歧规则。第三轮仅验证"已标注",未对照 A87 加消歧。**建议此类多名称主人公列为 A87 必查类型**。

### 8.3 B6 邦国化在法家列传中的高频出现

本章涉及"秦国/晋国/荆国/殷/夏"等 9 处邦国实体被误标为地名,远超普通列传的 B6 违规密度。**建议对战国至秦汉列传加强 B6 专项扫描**。

### 8.4 复合身份词拆分的隐蔽性

"公子"这类复合身份词在 `公〖#子〗` 拆分形式下极易被忽略（grep 只能找 `〖#公子〗`,找不到拆分形式）。**建议增加 grep 模式**：

```bash
grep -nE '[公父兄母长少]〖#[子女兄弟父母妻夫]〗' chapter_md/NNN_*.tagged.md
```

### 8.5 "魏公子X"边界错误模式

"魏公子卬" 此类"邦国+公子+名"的结构,AI 倾向于标注 `〖@魏公子〗` 而排除末字（名）。**建议增加此类边界扫描**：

```bash
grep -nE '〖@[^〗]*公子〗[^〗〖]' chapter_md/NNN_*.tagged.md
```
