### 3.2 E3 嵌套引号错位修正（L15 一处）

**问题根因**：L15 原文作：

```
奏〖=未央宫〗。"〖;丞相〗...制曰：'...差也。"〖?高山〗仰之，〖?景行〗乡之"，朕甚慕焉...可。'臣〖@青翟〗...不可。"
```

含 **4 个双引号 + 2 个单引号**，属于嵌套结构（外层 `"..."` + 外层 `'...'` + 内层 `"..."`）。

pair-every-other 算法将 4 个双引号按 [开-关-开-关] 机械分配，结果为：

```
差也。"高山仰之，景行乡之"，朕甚慕焉      ← 内层被错配为 [右]高山...[左]
```

**手工修正**：将 `差也。"高山仰之，景行乡之"` 纠正为 `差也."高山仰之，景行乡之"`，即内层开关引号反转。修正后四引号正确配对：外层 `"...\"`（pos 20↔811）、内层 `"..."`（pos 193↔203）。
