## 八、章节特有发现

### 8.1 B6 邦国化在纵横家列传中密度极高

本章为战国纵横家列传，182 处邦国实体被误标为地名 `〖=〗`，远超普通列传。战国合纵连横语境下，七雄皆作为外交/军事主体出现，按 B6 铁律必须全部邦国化。建议对 065-075（战国四公子/纵横家）列传加强 B6 专项扫描。

### 8.2 A87 主人公多名称链 + 同名异人风险

孟尝君田文案例暴露 A87 第三轮系统性缺失：
- 主人公 "文" 27 处未消歧
- 065 章孙子吴起列传中也有 "田文"（魏相，与孟尝君同名异人）
- 第三轮仅按"单字名需消歧"表面规则，未考虑同名异人深层风险

建议新增 A87a 子规则："主人公单字名若在全书有同名异人（可通过 entity_index.json 查询），必须 100% 消歧，不受任何豁免条款减免。"

### 8.3 E3 半角引号"幽灵修正"延续

第三轮报告 075 章只讲 14 处修正，完全未提引号问题，但实际有 63 处半角引号 + 引号 L/R 不均衡。这是 068/048 类似"第三轮幽灵修正"的延续。

修复策略创新：采用"原文 L/R 序列对齐重写"法，一次性解决"半角违规 + 全角不均衡 + 顺序错乱"三大问题，不需成对配对逻辑。

### 8.4 原文完整性违规（第三轮 A70 违规案例）

第三轮将 "鸡齐鸣" 擅改为 "鸡皆鸣"，其理由"齐为皆通假"即使成立也不得改原文字符。这是 CLAUDE.md §3 禁改原文铁律的直接违规。建议 E3 扫描脚本同时校验原文完整性（`lint_text_integrity.py` 实质差异数 = 0）作为 Step 0 强制通过项。

### 8.5 "代" 字三义并存

本章"代"字出现 7 处，分属三种类型：
- 地名：0 处（实际无代地出现）
- 人名苏代：5 处（原全误为地名）
- 制度代舍：2 处（原全误为地名）

这是"单字多义且类型相同标签"的典型案例，建议 SKILL_03c1-rules 增加"单字多义类型辨析"条目。

### 8.6 冯驩 vs 冯暖异文

L74 赞段 "冯暖" 是 "冯驩" 的别名（冯暖/冯驩/冯谖，异文同人），按 A108 必须消歧对应到规范名"冯驩"。此属赞/太史公曰段特有的"文雅别名"，建议在 A87/A108 增加此类案例。

### 9.2 候选新规律

**候选 A87a：主人公多名称链同名异人强制消歧**

若主人公在全书存在同名异人（通过 entity_index.json 可确定），所有单字/省称必须消歧，不受任何豁免减免。典型案例：
- 田文：075 孟尝君 vs 065 魏相（同名异人）
- 韩信：092 淮阴侯 vs 094 韩王信（同名异人）

**候选 E3a：引号按原文 L/R 对齐重写法**

不再使用"成对替换"策略（该策略无法处理"半角+全角不均衡"混杂情况），改为：
1. 从 `corpus/archive/chapter/NNN_*.txt` 提取原文引号 L/R 序列
2. 标注文件所有引号按位置排序
3. 按原文 L/R 序列逐位重写

此法一次性解决三类问题：半角违规/左右不均衡/顺序错乱。

**候选 E3b：E3 扫描+原文完整性联合校验**

每轮 Step 0 除扫引号外，必须同时运行 `lint_text_integrity.py` 确认 0 处实质差异；两项都 0 方可进入 Step 1。本轮就发现第三轮 A70 违规（擅改原文）+ E3 遗留 63 处半角引号的并存问题。
