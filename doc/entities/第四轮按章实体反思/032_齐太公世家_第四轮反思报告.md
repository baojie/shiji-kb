## 典型案例

### 案例 1：齐作为本国的标注一致性（B6/B7）

**发现**：齐世家中"齐"65处标为地名 `〖=齐〗`，与其他世家惯例不一致。

**证据链**：
- 033 鲁周公世家：`◆鲁=64, =鲁=0`
- 037 卫康叔世家：`◆卫=39, =卫=0`
- 042 郑世家：`◆郑=109, =郑=1`
- 038 宋微子世家："〖◆宋〗之分野"（非 `〖=宋〗`）
- 078 春申君列传："吾適〖◆楚〗"（非 `〖=楚〗`）

**修正原则**：B6 "邦国指分封制度下的政治实体，有明确国界、君主、臣民、参与会盟征伐"；齐符合全部特征 → `〖◆齐〗`

**唯一保留**：复合地名 `〖=齐营丘〗`、`〖=齐安平〗`（地名一级结构）

### 案例 2：赞语段落中氏族 vs 人名

原标：`〖@陈氏〗专政，厚货轻收。〖@悼〗、〖@简〗遘祸，〖@田〗、〖@阚〗非俦`

问题分析：
- "陈氏"/"田氏" 是氏族名 → 应 `〖&〗`
- "悼"/"简" 是齐悼公/齐简公的谥号省称 → `〖@X|齐X公〗`
- "阚" 是简公宠臣"阚止"（即子我）单字省称 → `〖@阚|阚止〗`

修正后：`〖&陈氏〗专政...〖@悼|齐悼公〗、〖@简|齐简公〗遘祸，〖&田|田氏〗、〖@阚|阚止〗非俦`

### 案例 3：A21 "诸子长贤者"拆分

原标：`乃言原择诸〖@子长〗贤者为〖;太子〗`

问题分析：
- "子长"非人名（司马迁字子长，但此处是"诸子之长者"）
- 实际结构是 "诸 + 子 + 长 + 贤者"（诸子中年长且贤者）

修正：`乃言原择诸〖#子〗长〖#贤者〗为〖;太子〗`

## 新规律建议

### 规律 B6a（建议）：本世家主国邦国一致性

**规则**：世家/本纪类章节的主国名（如齐世家之齐、鲁世家之鲁、项羽本纪之楚），除复合地名（齐营丘/鲁东门）外，一律标 `〖◆〗`。

**适用语境**：
- "伐 X / 奔 X / 如 X / 在 X / 归 X" → ◆
- "X 人 / X 君 / X 兵 / X 师 / X 境 / X 国" → ◆
- "X 分野"（天文分野属国家地理代号） → ◆
- "X 乱 / X 政" → ◆

**唯一例外**：复合地名结构（`〖=XY〗`，如"齐营丘""鲁东门""秦长城"等）

### 规律 A87a（建议）：赞语/太史公曰段单字人名强制消歧

**规则**：史记篇末"赞"或"太史公曰"段常以单字人名代指主要人物（如"悼/简/田/阚"指齐悼公、齐简公、田氏、阚止）。该段落是全篇人物总结，单字名 100% 需要消歧。

**识别**：
- 位于 `::: 赞` 或 `::: 太史公曰` 段内
- 格式多为四字句对仗（"〖@A〗、〖@B〗遘祸"）
