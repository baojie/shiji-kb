## 典型案例

### 案例 A：吴国全章邦国化（最大规模修正）

第三轮因吴章视角内"吴"既有地理也有政治用法，保留 `〖=吴〗`。第四轮按 B6 严格判定：

> "**伐吴/奔吴/吴人/吴师/吴兵/吴之边邑/吴召鲁卫之君会於橐皋/吴於是始通於中国**"

——全部是"吴"作为分封政治实体的动作主体/对象/领属语境，应统一为 `〖◆吴〗`。仅保留 `〖=句吴〗`（古别号）、`〖=吴东门〗`（具体地点）。

### 案例 B：上古氏族方国区别

L397 伍子胥谏辞追溯夏史：
- `〖=斟灌〗` `〖=斟寻〗` → `〖&斟灌〗` `〖&斟寻〗`（B6：夏代带"斟"氏族，"斟灌氏""斟寻氏"省称）
- `〖=有虞〗` → `〖&有虞〗`（本主题 B6 显式列举的"有虞氏"）
- `〖=有仍〗` → `〖&有仍〗`（与"有虞"同类族号）

"〖=纶〗"（少康邑）是封邑地名，保留 `〖=〗`。

### 案例 C：A8 时长"哭三日"

L437 "吴王闻之，哭於军门外三日"：
- 三日在此是**连续持续时长**（夫差为齐悼公哀悼 3 天），而非纪时
- 按 A8-2 经过时长子类，需显式 `|时长` 消歧
- 与 L272 `〖%十二年冬〗`（纪年）、L359 `〖%九月〗`（月份）形成对照
