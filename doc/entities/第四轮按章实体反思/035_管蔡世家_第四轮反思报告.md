## 章节特有发现

### 1. 蔡国世系「谥号+本名」双标体例

蔡国早期封君普遍采用 `〖@谥号侯〗〖@本名〗` 双标（如 `〖@夷侯〗`、`〖@共侯〗〖@兴〗`、`〖@宣侯〗〖@措父〗`、`〖@桓侯〗〖@封人〗`、`〖@哀侯〗〖@献舞〗`）。本章存在 5 处混用 `〖;谥号侯〗` 形式，第四轮统一归 `〖@〗`。这是典型的 B7 规则 #5 场景。

### 2. 赞文/太史公曰人名消歧密度高

诗赞段（L130-137）和太史公曰（L124）共出现 9 处需消歧的单字/省称人名（胡/献舞/穆侯/共/负羁/伯阳/振铎/叔铎），因：
- 诗赞脱离世系段落上下文
- 多处为单字（共、胡）或裸谥号（穆侯）
- 跨「蔡」「曹」两国，需规范名区分

**规律**：世家类章节末尾的诗赞段，若涉及封国省称人名，必须全面消歧（参 A87）。

### 3. 管/蔡作为"人名省称"vs"封国"

同一字 `管`、`蔡` 在本章既作人名（管叔鲜、蔡叔度省称，L76「管蔡作乱」）又作封国（L10「封叔鲜於管」、L130 赞文「武王之弟管蔡霍」）。判别依据：
- 作乱/诛死/被诛 语境 → 人（@）
- 封/灭/弟/属/之地 语境 → 国（◆）

### 4. 时长识别扩展

本章发现两种**新的时长模式**（超出 A91 "享国 XX 年"）：
- **"留 N 岁/年"** — 被他国扣留时长（L33、L59）
- **"后 X 灭 N 年"** — 事件间隔时长（L68）

这些场景应补充到 A91 的适用范围（后续可在规则库中扩展）。
