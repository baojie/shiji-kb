## 四、典型案例

### 案例 1：北斗星名系统化标注
```
原: 〖!北斗〗〖!七星〗，所谓"旋、玑、〖!玉衡〗以齐〖^七政〗"。
改: 〖!北斗〗〖!七星〗，所谓"〖!旋〗、〖!玑〗、〖!玉衡〗以齐〖!七政〗"。
依据: 司马迁本章亲证"北斗七星，所谓旋、玑、玉衡以齐七政"——旋/玑/玉衡
      均为北斗之星。025律书先例已将此三者归 `!` 天体。
      七政（日月五星）与二十八舍同为天体总称，归 `!`。
```

### 案例 2：距离量词纠偏
```
原: 外北〖^三尺〗，阴星。北〖^三尺〗，〖!太阴〗... 水发，近〖^三尺〗，远〖$五尺〗。
改: 外北〖$三尺〗，阴星。北〖$三尺〗，〖!太阴〗... 水发，近〖$三尺〗，远〖$五尺〗。
依据: "三尺"在此为月距行道的距离量词（与"远〖$五尺〗"对应），应为 `$` 数量，
      而非 `^` 制度/名物。明显是早期批量误标。
```

### 案例 3：方位词占辞批量去标
```
原: 风从〖=南方〗来，大旱；西南，小旱；〖=西方〗，有兵；西北... 〖=北方〗...〖=东方〗
改: 风从南方来，大旱；西南，小旱；西方，有兵；西北... 北方... 东方
依据: 025 律书已形成深度反思先例——八风占辞中方位词非地名，统一去标。
      本章八风占、五星占、客星占、岁星占等多处方位词同理。
```
