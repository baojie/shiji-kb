### 3.1 L177 动词"王"误标为 `〖#今王〗`（高）

- **原文**：王陵引高帝白马之盟语"非刘氏而王，天下共击之"後接本句
- **修改前**：`〖#今王〗〖&吕氏〗，非约也。`
- **修改後**：`今王〖&吕氏〗，非约也。`
- **理由**：此处"今王"= "现在封（吕氏）为王"，"今"为时间副词，"王"为动词（封王），与前文"非刘氏而王"之"王"用法一致。不是"当今之王"的身份名词。应去标注。
- **对照**：L125 `〖#今王〗有七十馀城` —— 该处"今王"指齐王本人，是第二人称尊称，保留`〖#〗`正确。**同形异义**。

### 3.2 L183 动词"王"误标为 `〖#王子〗弟`（高）

- **修改前**：`〖@高帝〗定〖_天下〗，〖#王子〗弟，今〖@太后〗〖^称制〗，王昆弟诸〖&吕〗`
- **修改後**：`〖@高帝〗定〖_天下〗，王子弟，今〖@太后〗〖^称制〗，王昆弟诸〖&吕〗`
- **理由**：本句前後文形成"王子弟／王昆弟"的对仗结构，两个"王"同为动词"封王"，"子弟"与"昆弟诸吕"为宾语。将"王子"标为身份词 `〖#王子〗` 系语境误判，违反 A70 动词识别原则。
- **对照**：本章 L438 齐王檄文 `王〖_诸子〗弟` 为正确模板（"王"作动词，不标注；"诸子"作思想类概念）。此处保守处理为纯文本。

### 3.3 L601 泛称"人子"误标为 `〖#人子〗`（中）

## 五、跨章一致性疑点

1. **"王"作动词的识别**：009章首次出现"王"同一动词在同段内出现两次，一次被误标身份、一次未标注的混乱情况（L183）。建议在规律库中明确——**"王X"句式，X 为族群/人群/名号时，"王"必为动词**，类似  
   - "王子弟" / "王昆弟" / "王诸吕" / "王刘氏"

2. **"今王"的同形异义**：同章节 L125 与 L177 同形异义，应依语境分辨：
   - 「今王 + 数量词（有七十馀城）」→ 身份尊称`〖#〗`
   - 「今王 + 族群／人名」→ 动词，不标注

3. **"人子 / 人X"的身份词边界**：
   - `〖#王者〗〖#王子〗〖#人子〗` 等"X者/子"泛称 → 应严格区分"普通名词短语"与"身份泛称"
   - "他人子" ≠ "人之子" 身份词

## 六、新规律建议（供规律库扩充参考）

**A88 建议：动词"王"的句法识别**  
在"王 + [族群名 / 人名 / 名号 ]"结构中，"王"作及物动词"封……为王"，不得标注身份 `〖#〗`。典型误标形式：`〖#今王〗吕氏` / `〖#王子〗弟` / `〖#王昆弟〗`。  
例外："今王"+数量词／状态谓语（如"有七十馀城"）时，"今王"为身份尊称。

**A89 建议："他人子"类泛称不标注**  
"他人子 / 他人妇 / 他人家" 等"他人+N"结构为普通名词短语，不构成身份类型`〖#〗`。
