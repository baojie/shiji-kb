## 六、新发现规律建议

### 规律候选：A87+A108 在列传类章节的执行漏洞

**现象**：090 章第三轮报告声称"质量优秀仅 1 处修正"，却遗漏了全章 60 余处单字人名消歧。这暴露了第三轮反思工作流中 **"对话段 + 赞语段" 消歧覆盖不足** 的系统问题。

**建议检查流程**（用于其他列传章节第四轮排查）：

```bash
# Step 1: 列出本章所有单字 @X（必应 100% 消歧）
grep -oE '〖@[一-龥]〗' chapter_md/NNN_*.tagged.md | sort | uniq -c | sort -rn

# Step 2: 列出本章所有 @XY 封号（需判定是否需 |规范名）
grep -oE '〖@[^|〗]+王〗|〖@[^|〗]+公〗|〖@[^|〗]+侯〗' chapter_md/NNN_*.tagged.md | sort -u

# Step 3: 赞语/太史公曰段单独扫描（A108 100%）
awk '/::: 赞|::: 太史公曰/,/:::$/' chapter_md/NNN_*.tagged.md | grep -oE '〖@[^|〗]+〗'
```
