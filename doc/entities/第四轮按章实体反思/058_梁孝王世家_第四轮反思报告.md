## 五、A87 L9 梁王消歧错误纠正（1 处）

### 问题

L9 原标注：
```
以〖@胜|刘胜〗为〖@梁王|梁孝王〗
```

### 错误根源

L9 本句是文帝即位二年分封四子时的情况：刘胜被封为梁王。但此"梁王"是**梁怀王**（见 L11 紧接下句"而梁王胜卒，谥为梁怀王"），**不是梁孝王**（刘武，是后来改封过来的）。原消歧将初任梁王的刘胜（怀王）错指为梁孝王（刘武），属 A87 消歧错误。

### 修正

```
以〖@胜|刘胜〗为〖@梁王|梁怀王〗
```

### 验证

- L11 其后语境 `〖@怀王|梁怀王〗最〖#少子〗` 印证消歧正确
- L11 另 3 处"〖@梁王|梁孝王〗"（刘武改封）保持不变
