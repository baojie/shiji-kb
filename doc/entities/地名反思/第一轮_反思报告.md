## 批次 1 (条 1-10)

- **001 · 万里沙** → **水域** — "祷万里沙"（祭祀），《汉书·武帝纪》："东巡海上…过万里沙，祷焉"；实为东莱郡海边沙丘/沙洲（似水域边缘），暂归"水域"。
- **002 · 三危** → **山脉** — "迁三苗于三危"、"道黑水至于三危"；《书·禹贡》"三危既宅"；西部山名。
- **003 · 三户** → **城邑** — "楚虽三户，亡秦必楚"（泛指楚地遗民）/"引兵度三户"（漳水渡口城邑）；以后者更具体，归城邑。
- **004 · 三渚之浦** → **水域** — "三渚之浦"（浦=水边），水域。
- **005 · 三澨** → **水域** — "过三澨，入于大别"，《禹贡》水道。
- **006 · 三辅** → **区域** — 京兆/左冯翊/右扶风合称，汉代长安周边区域。
- **007 · 上假密** → **城邑** — "韩信击龙且军於上假密"（军事攻取地），城邑。
- **008 · 上兰** → **城邑** — 上下文不足（绛侯周勃世家，"大夫施…"），但"上兰"属上兰观/上兰县（上党郡），暂归城邑。
- **009 · 上原** → **城邑** — "城元氏，县上原"（置县）、"臣之上原"，城邑/县。
- **010 · 上庸** → **县** — "秦与魏、韩上庸地为一郡"—— 上庸本是县，后合并为上庸郡，归县级。

###### 反思 1（条 1-10）
- **规则候选 R1**：`祷〖=X〗`/`祠〖=X〗`（祭祀对象）→ 水域/山脉/建筑，需要结合具体语境，先不加强模式。
- **规则候选 R2**：后缀 `浦` / `澨` → 水域（水边/沙洲）。**可加入 SUFFIX_RULES**。
- **规则候选 R3**：三家注"X，X也"结构中"X 属齐" / "X 属某郡"这类地理归属信息 → X 为县/城邑。**需要升级 SANJIA_PATTERNS 提取"X在Y县/郡"**。

- **019 · 东南郊** → **建筑** — "祭泰一东南郊"、"立其祠长安东南郊"—— 祭坛位置，归建筑。

###### 反思 2（条 11-20）
- **规则候选 R4**：`陈、蔡、X` 或 `攻X、Y、Z` 中所有项同类；这已由 L5 peer 覆盖但传播阈值未满足时遗漏。**可把 peer 传播阈值调到 0.5**（当前 0.6），以让少数案例也能传播。
- **规则候选 R5**：`X郊` → 建筑（祭坛）或 城邑 (X 的近郊)，以 X 前有动词"祭/祠/立祠"时归建筑。**已部分实现**。
- **规则候选 R6**：方位词"东X"（东南/东北/东原）当后缀不是专名时（= 无具体地理实体）→ 区域。暂不精细化。

- **规则候选 R9**：`斩/刑/诛 + 〖=X〗`（刑场）→ 建筑（若 X 是市/朝）。不高频，暂不加规则。

## 批次 4 (条 31-40)

- **031 · 临济** → **县** — "攻定陶，取临济"/"魏咎临济"—— 魏豹王临济，县级城邑。
- **032 · 乌盭** → **山脉** — "骠骑将军…逾乌盭"（翻越），山脉/关。
- **033 · 乐徐** → **城邑** — "代地大动，自乐徐以西"—— 地名起点，城邑。
- **034 · 九防** → **水域** — "曲九防，决渟水致之海"（九条堤防/水道），水域/道桥。
- **035 · 乾封** → **建筑** — "黄帝时封则天旱，乾封三年"—— 乾封 ≠ 地名（"干燥封坛"之典故）。实际是**祭祀场景**，不是地点；实体索引可能误标。暂归建筑（封禅坛）。
- **036 · 乾时** → **水域** — "与鲁战于乾时"，齐国水名。
- **037 · 乾谿** → **水域** — "楚师次於乾谿"/"楚灵王乐乾谿"，水名/谿谷。
- **038 · 云梦** → **水域** — "云梦者方九百里"（楚泽），大泽，水域。
- **039 · 五城十二楼** → **建筑** — "黄帝时为五城十二楼以候神人"—— 神仙传说中的仙境建筑群。
- **040 · 五帝** → **建筑** — "宰我问五帝之德"—— "五帝"此处非地名（五帝 = 古代帝王），是实体索引误标。暂跳过或视作建筑（五帝坛）。

###### 反思 4（条 31-40）
- **重要发现**：实体索引中有误标 place 的情况（"乾封"/"五帝"实非地名）。**建议新增"非地名"标记或在反思时标记为待修正**。
- **规则候选 R10**：后缀 `谿` / `时`（有时指水，如"乾时"齐水）→ 水域。`谿` 可加入 SUFFIX_RULES。
- **规则候选 R11**：`逾〖=X〗`（越过）→ 山脉（翻越山）。**可加入 L3**。
- **规则候选 R12**：`过/自/以西 + X` 作为地名起止点，单字多为城邑/县。弱信号。

## 规则累积总览（献给 SKILL 03h 升级）

本次反思提炼出的关键经验：

1. **动词与附加名词是最可靠的分类信号** —— 超越字面后缀
2. **方位词 + 河/山/江 = 区域** —— 一眼看出但需显式列表
3. **三字名以"之"连接** —— 可通过后缀鉴别子类
4. **古城遗址的"虚/墟"与陵墓的"冢"要区分** —— 虚=城邑，冢=陵墓
5. **司马相如赋需要特殊处理** —— 修辞/虚指集中在 117 章，可按章节加特殊规则
6. **单字 + 动词命中 ≥ 2** —— 门槛降低但仍需多证据
