## 八、修正方法论总结

### 8.1 年表章节反思流程

1. **旧格式清理**: 优先使用sed批量替换`⟦⟧`→`〖〗`
2. **系统性扫描**: 按动词类型批量grep扫描
3. **批量正则修正**: 使用Python正则批量标注
4. **人工审查**: 重点检查批量修正可能的误伤
5. **完整性验证**: lint_text_integrity确保无实质差异

### 8.2 批量修正关键正则

```python
# 1. 动词+标注 模式
re.sub(r'([^〗〖])杀〖@', r'\1〖[杀〗〖@', content)
re.sub(r'([^〗〖])伐〖\'', r'\1〖[伐〗〖\'', content)

# 2. 动词+裸字 模式
re.sub(r'([^〗〖])伐(我|卫|齐|楚)', r'\1〖[伐〗\2', content)
re.sub(r'([^〗〖])围([^〖〗，。]{1,3})(，|。)', r'\1〖[围〗\2\3', content)

# 3. 负向后顾断言避免重复标注
re.sub(r'(?<!〖\[)杀(?!〗)', ...)  # 避免已标注的
```
