## 三、共性发现与规律

### 发现1：军事传记旧格式残留率高

**规律**：
- 5章平均旧格式残留：86.2处/章
- 110章（匈奴列传）最高：193处
- 原因：早期军事章节标注时使用⟦◈⟧/⟦◉⟧符号，后续格式升级未完成迁移

**建议**：对所有军事类章节（109-111、卫霍相关）系统性检查旧格式。

### 发现2：马邑之谋段落嵌套标注集中

**现象**：108章马邑之谋段落（L36-42），`〖=〖~马〗邑〗` 出现13次。

**原因分析**：
- "马邑"是汉代雁门郡县名，应标`〖=地名〗`
- 早期标注误将"马"标为`〖~异族〗`，导致嵌套
- 同一错误在关联段落重复出现

**规律**：特殊地名/人名在密集语境下，若首次标注错误，会导致批量错误传播。

### 发现3：身份类复合词嵌套高频

**高频词汇**：
- 兄弟、父子、子孙、妻子、夫人（家庭关系）
- 士大夫、君子（社会身份）

**原因**：这些词汇本身就有嵌套结构（如"兄"+"弟"），AI标注时容易误判为两个独立标注。

**新增规则建议**（写入SKILL_03c §三）：

#### A61. 复合身份词禁止嵌套（108-112章新增）

复合身份词作为整体标注，禁止拆分或嵌套。

**正确标注**：
- `〖#兄弟〗`、`〖#父子〗`、`〖#子孙〗`、`〖#妻子〗`
- `〖#士大夫〗`、`〖#君子〗`、`〖#夫人〗`

**错误示例**：
- ❌ `〖#〖#兄〗弟〗`
- ❌ `〖#〖#士〗大夫〗`
- ❌ `〖_〖;君〗子〗`（跨类型嵌套）

**检查方法**：
```bash
grep -E '〖#〖#[^〗]〗[^〗]+〗' chapter_md/NNN_*.tagged.md
```

### 发现4：异族地名与汉地名混淆

**典型案例**：
- `〖=〖~马〗邑〗` → `〖=马邑〗`（马邑是汉地，非异族）
- `〖;〖~车〗骑将军〗` → `〖;车骑将军〗`（车骑将军是官职，非异族）

**判断标准**：
- 异族地名`〖~〗`：匈奴/西域/南越等政权控制区域
- 汉地地名`〖=〗`：汉朝行政区域（郡县）
- "马邑"属雁门郡，是汉地边境县城 → `〖=〗`

**新增规则建议**：

#### A62. 异族标记使用边界（108-112章新增）

异族标记`〖~〗`仅用于异族政权、部落、非汉族群体：

**正确使用**：
- 政权：`〖~匈奴〗`、`〖~闽越〗`、`〖~东越〗`
- 部落：`〖~犬戎〗`、`〖~玁狁〗`
- 异族人：`〖#单于〗`（异族身份）

**禁止使用**：
- ❌ 汉地地名：马邑、雁门、上谷 → 用`〖=〗`
- ❌ 汉朝官职：车骑将军 → 用`〖;〗`
- ❌ 汉族器物：车、马 → 用`〖•〗`

### 发现5：官职+人名拆分标注不统一

**问题案例**：
- `〖;大中夫#夫〗公〖#孙〗敖〗` → 嵌套混乱

**正确模式**：
- `〖;大中大夫〗〖@公孙敖〗`

**规律**：官职与人名连用时必须拆分：
1. 官职部分：`〖;官职〗`
2. 人名部分：`〖@人名〗`
3. 两个标注相邻，不留空格

**参见**：SKILL_03c §A34（057章已归纳）
