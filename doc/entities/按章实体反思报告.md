### 章节特有发现

1. **日月三义问题**（→已写入SKILL §2.2.3）
   "历〖^日月〗而迎送之"和"〖^日月〗所照"中日月是天体，应为`〖!日月〗`。AI将"日月"全部归入制度，原因是日月出现在礼制语境中（历法/巡狩）。区分规则：凡"所照/所至/历X迎送"语境=天体`〖!〗`；"日月之行（作为礼制对象）"才可能是制度。

2. **制度^过度捕获刑法**（→已写入SKILL §2.2.3）
   五帝本纪第14.2段集中出现"典刑/流宥五刑/官刑/教刑/赎刑"5个刑法词，全部被归入制度。这是系统性问题，预计全书刑法词约有30-50处需要迁移（待按类型反思处理）。

3. **帝号+氏族号粘连**（→已写入SKILL §2.5）
   上古帝王"帝颛顼高阳""帝喾高辛"格式固定：帝+谥号（人名）+ 氏族号（氏族）。[26]中"〖@帝颛顼〗为〖&高阳〗"已正确标注，[5][7]开头介绍语中未拆分。

4. **官职任命句中的Y**（→已写入SKILL §2.5）
   "以垂为共工"：共工在[12][15]中是人名，但此处是舜任命垂担任的官职。同一词在不同语境下类型不同，需逐个判断。

5. **叠字副词**（→已写入SKILL §2.5）
   "夔夔唯谨"中"夔夔"（恭谨貌）被标为两个人名，是对已知人名的过度联想。叠字构成副词（礼貌/神态描述）时不标。

### 章节特有发现

1. **五服距离描述中的"百里"系列**（→已写入SKILL §2.2.2）
   "令天子之国以外五百里甸服：百里赋纳裛，二百里纳铚……"中，百里/二百里/三百里/四百里是行政区划距离，不是人名。AI将"百里"联想为人名（如百里奚），这是跨章联想导致的系统性误标。**识别规则**：五服制度语境（甸服/侯服/绥服/要服/荒服）中的距离表达均为`〖$数量〗`。

2. **器物^误捕典籍书名**（→已写入SKILL §2.2，可能有多章重复）
   甘誓/五子之歌/胤征/夏小正均为书名（古籍篇章），被标为器物`〖*〗`。凡"作X"/"传X"/"学X"语境中出现已知古籍名称，应为`《典籍》`。

3. **"道"字引导的地理名词**（→写入SKILL §2.2.4）
   "道九山"/"道九川"中"道"是动词（疏导/整治），九山/九川是地理概念（地名），不应整体标为器物。类似语境：凡"道X"结构，X为地名时单独标注为`〖=〗`。

### 章节特有发现

1. **殷本纪典籍误标大爆发**（→强化002章规律）
   全章有19处"作X"创作的古籍（帝诰/汤誓/伊训等）全部被标为器物`〖*〗`。殷本纪是史记中典籍引用最密集的章节之一。确认规律：**凡"作X"/"训X"/史书明确提到的古籍篇章名 → `《典籍》`**，不管有多短（"训"单字也是书名）。

2. **三公`〖^〗`→`〖;〗`**（→已补充进SKILL §2.2.1）
   "三公"在"以X、Y、Z为三公"和"三公咸有功"语境中是具体官职指称，应为`〖;〗`，而非制度。辨别规则：三公出现在任命语境或描述某人担任某职时 → `〖;官职〗`；仅泛指"三公制度"时可考虑`〖^制度〗`。

3. **亲属称谓+代词误标人名**（→已补充 §2.2.5）
   "帝乙长子/少子"（亲属称谓）、"小子"（谦称）、"维王/不虞"（文言代词/动词短语）均被标为人名。规律：文言文中"长子/少子/小子"等在亲属关系描述语境中 → 去标注；文言谦称/指代词 → 去标注。

### 章节特有发现

1. **甫刑段落刑法大规模误标**（→强化刑法规律）
   [26]甫刑全段（五刑/五罚/五过/黥辟/劓辟/膑辟/宫辟/大辟/墨罚/劓罚/膑罚/宫罚）共14处均被标为制度。这是一整段刑法典章，应全部用`【刑法】`。规律：法律文书段落内所有刑罚名词优先判断为刑法，而非制度。

2. **格式破损：标注符号错位**（→加入SKILL §2.1）
   [28]出现`孝王〖@。〗`和`燮〖@，是为〗夷王`这类标注内含句读的格式错误，是AI生成时将标注边界与文本边界搞混导致的。也出现`〖@子而立〗`这种将动词短语包入人名标注的情况。识别规则：若标注内含句读符号（。，；）或跨越多个词，则为格式破损。

3. **大白/小白旗→器物**（→新增§2.2.4）
   "持大白旗以麾诸侯""县其头小白之旗"中，大白/小白是旗帜名称（颜色+旗），应为`〖*器物〗`，被误标为`〖!天文〗`（可能因"白"字与天象相关联）。

4. **《大雅》《史记》→典籍**
   大雅（诗经部分）/史记（太史读的历史文献）均应用《》标注。本章首次出现"史记"作为典籍名被引用。

### 章节特有发现

1. **诗书礼乐被标为制度**（→强化典籍规律）
   [32]`〖^诗〗〖^书〗〖^礼〗〖^乐〗`，原因是被视为"礼法制度"，但诗书礼乐是具体典籍名称，应用《》标注。规律：单字书名（诗/书/礼/乐/易）用《》，与"礼制制度"区分。

2. **庶长+人名粘连**（→强化官职拆分规律）
   "庶长晁""庶长改""庶长疾""庶长章""庶长封"格式：庶长是官职，后接人名。全章多处将二者合并为`〖;庶长X〗`。应拆分为`〖;庶长〗〖@X〗`。

3. **初腊误标人名**
   "初腊"是"初次举行腊祭"，是事件描述，非人名。

4. **妻子（动词）语境**
   "秦妻子圉以宗女"中"妻"是动词（给…为妻），"子圉"是人名前缀，不应标为亲属称谓身份`〖#妻子〗`。
