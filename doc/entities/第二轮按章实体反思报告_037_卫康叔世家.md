## 三、章节特有发现

### 1. 春秋世家刑法词汇密度极高

本章记录卫国从西周到战国末期的历史，政变频繁：
- 弑君事件：桓公、隐公、殇公（3次）、哀侯、灵王、孺子、悼公、昭公、怀公，共11处
- 杀戮事件：遍布全章，共28处
- 诛罚事件：5处

**规律**：世家类章节比本纪类更注重具体政治斗争细节，刑法标注密度更高。

### 2. 氏族势力标注规则

在春秋战国背景下，"X氏"常指家族政治势力，应标注为〖&氏族〗而非〖@人名〗：
- 赵氏（晋国政治力量）
- 孔氏（卫国大夫家族）

判断标准：
- 作为政治集团/家族势力出现 → 氏族
- 指具体某个人的省称 → 人名

### 3. 器物"衣"的语境判断

"衣"字在不同语境需区别对待：
- "蒙衣而乘"：伪装用的衣物 → 器物〖•〗
- "衣食住行"：泛指生活需求 → 不标注
- "解衣推食"：抽象比喻 → 不标注

本章L120"蒙衣而乘"是具体伪装行为，衣物是实际器物。

### 4. "释之"消歧规则

"释之"有两义：
- 人名：张释之（汉朝廷尉）→ 〖@张释之〗
- 动词短语：释放之义 → 不标注

本章L80"郑降，复释之"明显是"释放之"，应去标注。

### 5. 身份标注"妾"的重复出现

同一身份在段落中多次出现需全部标注，不能只标第一次：
- L98: "有贱〖#妾〗...妾怪之" → 两处都要标注
- L84: 同段三次出现"妾" → 全部标注
