## 五、核心规律总结

### 5.1 旧格式残留规律

- **残留原因**: 之前使用过 `⟦◉⟧` 和 `⟦◈⟧` 作为刑法/军事标注
- **清理策略**: 使用sed批量替换最高效
- **验证方法**: `grep -E '⟦|⟧'` 确保零残留

### 5.2 刑法动词漏标规律

**高频漏标场景**：
1. **并列句式**: "X攻A，Y攻B，Z攻C" 中后续的"攻"易遗漏
2. **复杂句**: 主从句/状语较多时，核心动词易被忽略
3. **"相X"结构**: 如"相攻""相伐"等互动性动词
4. **"作乱"特殊性**: 作为成语性动词，易被当作普通动词忽略

**检测方法**：
```bash
grep -nE '(作乱|相攻|相伐)(?!〗)'
grep -nE '[^〗](攻|伐|败|破)[〖=〖']'
```

### 5.3 标注边界完整性规律

**"斩首X万"错误模式**：
- **错误认知**: 将"斩首"视为一个整体词
- **正确理解**: "斩"是动作，"首"是对象，"X万"是数量
- **标注原则**: 动词单独标注，数量单独标注，避免嵌套
