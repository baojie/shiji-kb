## 三、章节特有发现

### 1. 军事类世家刑法标注密度极高

**特征**:
- 040_楚世家全篇记载楚国600多年历史，战争频繁
- 平均每2-3行就有1处军事/刑法动词
- 第一轮标注几乎完全忽视了军事刑法动词

**高频场景**:
1. 战争描写：伐X、攻X、败X、灭X、围X
2. 权力斗争：弑君、杀弟、诛臣
3. 外交军事：袭、执、虏、囚

### 2. 楚国特有称谓

- 长子/中子/少子：不标注（亲属称谓）
- 句亶王/鄂王/越章王：标注为`〖;官职〗`（王号爵位）

### 3. 氏族与人名混淆

- 高阳/高辛：上古氏族号，易误标为人名
- 共工氏/昆吾氏/彭祖氏：氏族名，应标注为`〖&〗`

## 四、新增标注规则

### 1. 军事世家刑法扫描规则

**适用章节**: 世家类（尤其战国诸侯世家）、列传类（将相列传）

**扫描方法**:
```bash
grep -nE '(伐|攻|败|灭|围|救|击|袭|执|虏|破|克|射|弑|杀|诛|囚|劫|僇|斩)' chapter_md/NNN_*.tagged.md
```

**判断标准**:
- 单字军事动词 → 几乎都需标注`〖[刑法〗`
- 伐/攻/败在战争语境 → 100%标注
- 杀/弑在权力斗争语境 → 100%标注

### 2. 氏族号识别规则

**上古氏族号**（常见误标）:
- 高阳（帝颛顼）、高辛（帝喾）
- 有熊（黄帝）、有虞（舜）
- X氏（昆吾氏、共工氏、彭祖氏等）

**判断**:
- "XX者，帝X之Y也" → XX是氏族号
- "X氏作乱" → 氏族
- "X氏，Y之时尝为..." → 氏族
