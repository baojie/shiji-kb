## 二、核心发现

### 发现A：旧格式残留严重（138处，52.4%）

**问题描述**：
008章仍使用第一轮标注的旧格式`⟦◈⟧`（军事动词）和`⟦◉⟧`（刑法动词），共138处未转换。

**分布**：
- `⟦◈击⟧`、`⟦◈攻⟧`、`⟦◈破⟧`、`⟦◈围⟧`、`⟦◈战⟧`等军事动词：95处
- `⟦◉杀⟧`、`⟦◉诛⟧`、`⟦◉斩⟧`、`⟦◉弑⟧`等处决动词：43处

**修正方案**：
全部转换为新格式`〖[动词〗`，保留消歧说明（如`〖[取|攻取〗`、`〖[降|投降〗`、`〖[败|战败〗`）。

**影响**：
- 旧格式影响知识图谱抽取的准确性
- 此类错误在001-012章早期标注章节中普遍存在，需系统性排查

**建议**：
将"旧格式清理"作为按章反思的Step 0强制检查项。

### 发现B：本纪类章节刑法动词密度极高（114处遗漏）

**问题描述**：
本纪类章节记录帝王兴衰、战争征伐，刑法动词密度远超世家列传，但大量未标注。

**高频遗漏动词**：
1. **战争类**：破/战/击/攻/围/取/拔/下/定（共78处）
2. **处决类**：杀/诛/斩/屠（共20处）
3. **其他**：降/败/走/追/距/反/叛/虏/掠/阬/赦（共16处）

**典型遗漏句式**：
- "出与战，破之" → "出与〖[战〗，〖[破〗之"
- "大破之" → "大〖[破〗之"
- "皆多杀其" → "皆多〖[杀〗其"
- "引兵攻X" → "引兵〖[攻〗X"
- "未下" → "未〖[下〗"

**密度对比**：
- 008章（本纪）：刑法动词约占全章标注的45%
- 057章（世家）：刑法动词约占全章标注的56%
- 结论：本纪类章节刑法动词密度与军功世家相当，需系统性扫描

**检查方法**：
```bash
grep -nE '(杀|诛|斩|弑|击|攻|破|围|战|取|降|败|救|袭|拔|屠|灭|族|废|囚|虏)' chapter_md/NNN_*.tagged.md | grep -v '〖\['
```

### 发现C："有过"动词短语误标为氏族

**原标注**：`〖@吕后〗具言客〖&有过〗`
**修正**：`〖@吕后〗具言客有过`
**说明**："有过"=曾经来过（动词短语），非氏族名

**类似易误标词汇**：
- 有过、得意、不礼、出于、用事等动词短语
- 判断标准：能否替换为具体氏族名 → 否则为动词短语
