# 《史记》君主列表整理过程

## 项目概述

本文档记录了整理《史记》所有君主信息、建立 `kg/relations/rulers.json` 和 `rulers.csv` 数据集的完整过程，包括数据来源、提取方法、分类体系和已知局限。

---

## 一、数据集目标

**目标**：整理《史记》涉及的所有君主，涵盖：
- 上古圣王（黄帝至虞舜）
- 夏商周三代天子
- 春秋十二诸侯各国君主
- 战国七雄各国君主
- 秦末楚汉之际的诸王
- 汉代藩王（诸侯王）

**字段**：名、谥号、别名、父亲、上一任、下一任、出生年、死亡年、登基年、退位年、所属国、类型、备注

年份统一用**负整数**表示公元前年份（如 -221 = 公元前221年）。

---

## 二、数据来源

### 2.1 第一批：本纪 + 世家（手工提取）

**来源章节**：
- 001-012 本纪（历代天子）
- 031-049 世家（诸侯国世系）
- 017 汉兴以来诸侯王年表
- 050-060 汉代世家

**方法**：使用三个并行 Agent，分别读取各章事件索引（`kg/events/data/`），从中提取君主继承信息。三批数据格式不同（本纪是平铺列表，世家是按国分组的嵌套结构），提取后统一到同一 JSON schema。

**结果**：320 条君主记录

### 2.2 第二批：十表（自动解析）

**来源章节**：
- 014 十二诸侯年表（841-477 BCE，14个诸侯国）
- 015 六国年表（476-221 BCE，8列：周+六国）
- 016 秦楚之际月表（209-202 BCE）

**方法**：程序解析表格。年表中每一行是一年，列是各诸侯国。新君主继位通过"X元年"标记识别，例如：

```
| 825 | ... | 鲁武公敖元年 | 齐厉公无忌元年 | ... |
```

从中解析：
- 所属国 = 列名（去除标记符号 `&`）
- 谥号 = "元年"前的称号部分
- 名 = 谥号后的字（有时内嵌在称号中，如"武公敖"）
- 登基年 = 该行的年份（取负值）

**Agent 提取**：委托 general-purpose Agent 读取三个表格文件，解析所有"元年"条目，与现有320条做模糊匹配（按 国+谥号、国+名、国+年份±2 匹配），输出未重复的新条目。

**清理步骤**：
1. 删除误解析：016表格中"周 始皇帝"和"周 二世"（实为秦代，被归到周列是解析错误）
2. 删除重复：周威烈王、周赧王已在第一批数据中
3. 修正类型：016提取的临菑王、河南王、济北王、胶东王、淮南王（项羽所封）为"诸侯王"而非"汉诸侯王"

**结果**：197 条新记录

## 三、类型分类体系

### 3.1 四级分类

初始提取时使用三个类型（天子/主君、诸侯国君、汉诸侯王），后根据历史层级改为四级：

| 类型 | 说明 | 典型例子 |
|------|------|---------|
| 天子/皇帝 | 名义上统治天下的最高君主 | 上古圣王、夏商周天子、秦始皇、汉帝 |
| 诸侯王 | 封国内称王的诸侯 | 楚武王（首个称王的诸侯）、战国七雄诸王、西楚霸王 |
| 诸侯侯君 | 以公/侯/伯/子/君为爵称的诸侯 | 齐桓公、晋文公、鲁隐公、秦仲至秦孝公 |
| 汉诸侯王 | 汉代分封的藩王（臣属于汉帝） | 刘交（楚王）、刘肥（齐王）等 |

### 3.2 分类规则

**天子/皇帝**：
- 所属国为"上古"、"夏"、"商（殷）"、"周"、"西周"、"东周"、"汉" 的君主
- 秦国：谥号为"秦始皇帝"、"二世皇帝"、"子婴"者

**诸侯王**：
- 秦国：谥号含"王"且非皇帝（惠文王/武王/昭襄王/孝文王/庄襄王）
- 楚（西楚）：西楚霸王项羽
- 其他诸侯国：谥号含"王"（如楚武王、齐威王、燕昭王等）
- 吴国：从寿梦起称王（寿梦、诸樊、馀祭、馀眛、僚、阖庐、夫差），谥号本身不含"王"但历史上确为王爵

**诸侯侯君**：
- 其余诸侯（谥号含公/侯/伯等，或无爵称的早期诸侯）
- 秦国早期（秦仲至秦孝公）：历史上为诸侯公爵，不称王

**越国**：所有越国君主（允常/句践/鼫与/无彊）均为王爵，分类为诸侯王

## 五、已知局限

1. **父子关系稀疏**：十表中的新增君主（约197条）仅有登基年，父亲/上下任字段为空，需要从世家或列传中补充

2. **谥号名混用**：部分君主在 JSON 中"谥号"字段实际存的是通称（如"楚武王"），而"名"字段为空，格式不够统一

3. **吴国君主分类依赖硬编码**：寿梦起称王的规则是硬编码的名单，未来如有增删需手动维护

4. **年份精度不均**：本纪/世家的年份来自学术参考书（较精确），十表解析的年份直接来自表格行号（相对可靠），但两者精度和来源有差异

5. **秦末诸王**：016提取的临菑王/河南王等项羽所封诸侯，退位年均在 -202 年前后（楚汉战争结束），目前未填退位年

6. **重复风险**：模糊匹配以 国+称号/国+名/国+年份±2 为判据，同一君主在不同来源中称谓不同时可能有重复未被检出
