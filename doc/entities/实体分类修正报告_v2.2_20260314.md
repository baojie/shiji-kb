## 二、族群 / 氏族 / 邦国 重新分类

### 2.1 分类原则（确立三层体系）

| 类别 | 符号 | 定义 | 核心属性 |
|------|------|------|----------|
| 邦国 | `〖'〗` | 有组织的政治实体（统一王朝 + 列国 + 外邦政权） | 主权 / 领土 / 君主制度 |
| 氏族 | `〖&〗` | 以血缘为纽带的宗族/姓氏群体 | 血统 / 姓氏 / 传承谱系 |
| 族群 | `〖~〗` | 以族裔或文化认同为纽带的群体 | 族裔 / 文化 / 集体认同 |

**边界规则**：
- 有君主、领土、行政制度 → 邦国（含大夏、朝鲜等外邦）
- 以"氏/姓"为标志的姓氏宗族 → 氏族（含古代有X氏形式）
- 以族裔/文化名称出现、无明确政权结构 → 族群（匈奴、戎、狄、越、夷等）
- 君主称号（宛王、单于等）→ 官职

## 五、官职类别反思与重分类

### 5.1 分析背景

官职类（`〖;〗`）在迁移后含 **2306条** 实体，经词频统计发现大量误分类：
1. 时间词「元年」被标为官职
2. 具体皇帝/人物的专称被标为官职（应为人名）

### 5.2 官职→时间修正

| 误标词 | 原类别 | 正确类别 | 次数 |
|--------|--------|---------|------|
| 元年 | 官职`〖;〗` | 时间`〖%〗` | 1497 |

「元年」是纪年格式的固定组成部分（如「高祖元年」「武帝元年」），本质是时间表述，非官职名称。迁移中将所有 `〖;元年〗` 改为 `〖%元年〗`。

### 6.1 非地名词清理（29+30处）

| 误标词 | 错误类别 | 正确处理 | 次数 | 说明 |

## 八、实体索引页新增类型定义

在 `kg/entities/scripts/build_entity_index.py` 中新增 `ENTITY_DESCRIPTIONS` 字典，16类实体索引页开头均加入定义说明框，包含：

- 类型标签（带颜色）
- 标记符号（如 `〖@〗`）
- 文字定义（约50字，说明该类实体的范围和标准）

各类定义如下：

| 类别 | 定义摘要 |
|------|----------|
| 人名 | 历史人物，含帝王将相、诸侯公卿、学者方士等，有完整姓名及单字名/字/号/称谓 |
| 地名 | 地理名称，含国都城邑、山川关隘、道路地区，从上古至西汉 |
| 官职 | 官职名称与君主称号，含中央百官、地方长吏、军职武衔及外邦职称 |
| 时间 | 时间表述，含纪年、月份、季节及干支等历法用语 |
| 邦国 | 政治实体，含三代邦国、春秋战国列国、秦汉王朝及外邦政权 |
| 氏族 | 血缘宗族/姓氏群体，含王族、先秦贵族世家及古代氏族 |
| 族群 | 族裔/文化群体，含汉族以外各族及其部落联盟，也含"诸夏"等文化共同体 |
| 制度 | 政治经济军事制度及礼制规范，含各类法规、惯例和仪典规程 |
| 器物 | 有形物品，含礼器、兵器、车马器、宫室建筑、货币及珍宝异物 |
| 天文 | 天文历法概念，含星官、天象、历法术语及占星占候用语 |
| 神话 | 神话传说人物与神祇，含三皇五帝神话形象、各地祭祀神明及传说异兽 |
