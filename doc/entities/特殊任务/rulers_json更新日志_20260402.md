#### 1.2 武丁添加别名"高宗"、"商高宗"

**索引位置**：第130条记录

**更新前**：
```json
{
  "所属国": "商（殷）",
  "谥号": "武丁",
  "名": null,
  "别名": null
}
```

**更新后**：
```json
{
  "别名": ["高宗", "商高宗"]
}
```

**更新理由**：
- 数据来源：013 三代世表事件027"武丁雉升鼎，得傅说，称高宗"
- 庙号：高宗是武丁的庙号，史记明确记载"故庙号高宗"
- 史记惯例：商代君主常以庙号称呼（如"汤"也称"成汤"）

**执行逻辑**：
```python
# 1. 查找条件：谥号="武丁"
# 2. 初始化别名列表（原为null）
if ruler.get('别名') is None:
    ruler['别名'] = []
# 3. 添加庙号别名
ruler['别名'].extend(['高宗', '商高宗'])
```

**知识点**：
- 谥号 vs 庙号：谥号是死后根据行为评定（如"武"），庙号是祭祀时的称呼（如"高宗"）
- 商代庙号：商代开始使用庙号，武丁因中兴被尊为"高宗"

import json

# 构建上古圣王数据
ancient_rulers = [
    {
        "所属国": "上古",
        "谥号": "颛顼",
        "名": "高阳",
        "别名": ["高阳氏"],
        "类型": "天子/皇帝",
        "备注": "五帝之二，黄帝之孙"
    },
    # ... 其他3条
]

# 读取现有数据
with open('kg/relations/rulers.json', 'r', encoding='utf-8') as f:
    data = json.load(f)

# 检测并添加
existing_names = {r.get('谥号', '') for r in data['rulers']}
to_add = [r for r in ancient_rulers if r['谥号'] not in existing_names]

if to_add:
    data['rulers'] = to_add + data['rulers']  # 前置插入
    data['meta']['total'] = len(data['rulers'])

    # 更新类型统计
    for ruler in to_add:
        ruler_type = ruler['类型']
        data['meta']['by_type'][ruler_type] = \
            data['meta']['by_type'].get(ruler_type, 0) + 1

    with open('kg/relations/rulers.json', 'w', encoding='utf-8') as f:
        json.dump(data, f, ensure_ascii=False, indent=2)
