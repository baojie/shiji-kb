## 二、核心规律总结

### 2.1 遗漏类高发区（4大类）

#### A. 刑法类（占比88.4%，2471处）

**高频遗漏词汇**：
- **杀系**：杀(1003)、弑(21)、戮(15)、僇(8)
- **灭系**：灭(287)、族诛(17)、夷(12)
- **诛系**：诛(192)、伐(141)
- **特殊刑**：阬(58)、烹(28)、醢(7)、殛(3)

**遗漏原因**：
1. 刑法动词在叙事文本中高频出现，容易产生"麻痹效应"
2. 战争场景中"击、攻、破、败"等军事动词与刑法动词混淆
3. 部分刑法词汇（如"阬"）在现代汉语中不常用，易被忽略

**识别规律**：
```
刑法动词 + 人名/身份词 → 刑法
例：诛〖@孟说〗、杀〖#士卒〗、阬〖=马服〗降卒

刑法动词 + 之/其 → 刑法
例：诛之、杀之、阬之

刑法动词 + 邦国 → 刑法（灭国）
例：灭〖'赵〗、破〖'燕〗
```

**检查策略**：
```bash
# 刑法词汇一致性检查
grep -E '诛|杀|灭|阬|弑|烹|醢|殛|戮|僇|族' file.md | grep -v '〖\['
```

#### B. 人名省称类（占比39.2%，1097处）

**高频遗漏场景**：
1. **远古人名重复**：尧(21)、舜(18)、象(7)、禹(15)
2. **主人公省称**：项羽本纪中的"羽"(141)、韩信列传中的"信"(89)
3. **历史名人**：孔子→仲尼(62)、张良→子房(34)

**遗漏原因**：
1. 首次出现标注完整，后续省称易遗漏
2. 单字人名易被当作普通字词
3. 长段落中注意力衰减

**识别规律**：
```
段落首次：〖@尧〗立〖%七十年〗
后续省称：尧辟位凡... → 〖@尧〗辟位凡...

固定搭配：
- "尧+动词" → 人名（尧曰、尧立、尧辟位）
- "舜+处所" → 人名（舜耕历山、舜渔雷泽）
```

**检查策略**：
```python
# 章节主人公省称检查
chapter_name = "项羽本纪"
main_char = "羽"
grep -E '^[^〖]*羽[^〗]' file.md  # 查找未标注的"羽"字
```

#### C. 身份类（占比8.6%，239处）

**高频遗漏词汇**：
- **群体性**：群臣(47)、百姓(41)、士卒(32)、诸侯(21)
- **等级性**：公卿(18)、大夫(14)、诸生(12)、群儒(9)
- **关系性**：父老(11)、子弟(8)、父母(7)、妻子(6)

**遗漏原因**：
1. "太常见"导致忽视（群臣、百姓高频出现）
2. 专题章节产生"盲区"（如祭祀章关注礼器而忽略群体）
3. 身份vs官职判断犹豫（如"百官"）

**识别规律**：
```
群体+动词 → 身份
例：〖#群臣〗咸来、〖#百姓〗苦之、〖#士卒〗恐

修饰+群体 → 身份
例：〖=沛〗中〖#子弟〗、〖=关外〗〖#父老〗

判断标准：
- 百官/百工 → 官职〖;〗
- 诸侯/群臣/士卒 → 身份〖#〗
```


**检查策略**：

### 2.2 误标类高发区（4大类）

#### A. 同形异义词误判（占比3.2%，90处）

**典型案例**：

| 词汇 | 人名形式 | 词组形式 | 判断规则 |
|------|---------|---------|---------|
| 非子 | 〖@非子〗（秦非子） | 非子罪也（不是你的罪） | 看前文是否提及"非子"其人 |
| 得意 | 〖@得意〗（人名） | 得意之作（满意） | 独立出现且有身份描述→人名 |
| 释之 | 〖@张释之〗 | 释之不问（释放他） | 前文有"张释之"→人名省称 |
| 不礼 | 误标为人名 | 不以礼待（动词短语） | 动词+名词→不标注 |
| 中行 | 〖@中行偃〗 | 〖&中行氏〗 | 人名vs氏族判断 |

**识别规律**：
```
判断流程：
1. 前文是否提及此人？ → 是：人名省称
2. 是否有职衔/身份修饰？ → 是：人名（如"〖;太尉〗〖@得意〗"）
3. 是否为动词短语？ → 是：不标注（如"释之不问"）
4. 无法确定？ → 保守不标注
```

**案例对比**：
```
✓ 正确：〖@张释之〗为〖;廷尉〗...后文"释之曰" → 〖@释之〗曰
✗ 错误：不礼于〖@释之〗 → 不礼于释之（动词短语）

✓ 正确：〖@非子〗居〖=犬丘〗...后文"非子之功" → 〖@非子〗之功
✗ 错误：〖@非子〗罪也 → 非子罪也（=不是你的罪）
```

#### B. 谥号误标（占比2.4%，66处）

**主要问题**：年表中谥号误标为官职

| 原标注 | 正确标注 | 说明 |
|--------|---------|------|
| 〖;康侯〗 | 〖@康侯〗 | 谥号是人名的一部分 |
| 〖;夷侯〗 | 〖@夷侯〗 | 谥号用于指代已故者 |
| 〖;哀侯〗 | 〖@哀侯〗 | 谥号具有人名属性 |

**区分规则**：
```
谥号（人名）：康侯、夷侯、哀侯 → 〖@〗
封号（地名+爵位）：平阳侯、汝阴侯 → 〖=平阳〗侯
爵位（制度）：功侯、列侯 → 〖^〗或不标

判断标准：
1. 单独使用（如"康侯薨"） → 人名
2. 有地名前缀（如"平阳侯曹参"） → 地名+人名
3. 泛指爵位（如"封为列侯"） → 制度
```

**年表特殊规则**：
```
| 国名列 | 侯爵列 | 谥号列 |
|--------|--------|--------|
| 〖=平阳〗 | 侯 | 〖@康侯〗 |
| 〖=汝阴〗 | 侯 | 〖@夷侯〗 |

标注模式：
- 国名：〖=地名〗
- 侯爵：不标注或〖^侯爵〗
- 谥号：〖@谥号〗
```

#### C. 时间/时长混淆（占比0.9%，25处）

**误标案例**：

| 原标注 | 正确标注 | 规则 |
|--------|---------|------|
| 立〖%七十年〗 | 立〖$七十年〗 | "X年"作为时长 |
| 在位〖%二十岁〗 | 在位〖$二十岁〗 | "X岁"作为时长 |
| 〖%三世〗 | 〖$三世〗 | "X世"作为时长 |

**判断规则**（SKILL已明确）：
```
时间〖%〗：具体时间点
- 元年、二年、三月、甲子日
- 春、夏、秋、冬（季节）
- 明年、后年（相对时间点）

时长〖$〗：时间段长度
- X年、X岁、X月、X日、X世
- 动词+时长：立X年、在位X年、居X岁

识别关键词：
- 立/在位/居/历 + 数字+年/岁/世 → 时长
- 元/二/三 + 年/月 → 时间点
```

#### D. 礼仪/制度混淆（占比0.4%，10处）

**典型案例**：

| 类型 | 错误标注 | 正确标注 | 区分规则 |
|------|---------|---------|---------|
| 礼仪行为 | 王〖^冠〗 | 王〖:冠礼〗 | 行为→〖:〗 |
| 礼仪行为 | 〖^聘享〗 | 〖:聘享〗 | 外交礼节→〖:〗 |
| 制度体系 | 〖:井田〗 | 〖^井田〗 | 土地制度→〖^〗 |
| 制度体系 | 〖:封建〗 | 〖^封建〗 | 政治制度→〖^〗 |

**区分原则**：
```
礼仪〖:〗：具体礼节行为
- 冠礼、婚礼、丧礼、祭礼
- 朝觐、聘享、会盟、射礼

制度〖^〗：抽象体系规则
- 井田、封建、郡县、科举
- 宗法、嫡长子继承制

判断标准：
- 动词+名词 → 礼仪（行冠礼、行聘享）
- 名词+制度 → 制度（井田制、封建制）
- 可被"举行"→ 礼仪
- 可被"实行/推行"→ 制度
```

### 2.3 场景化遗漏模式

#### A. 战争场景（器物+刑法密集）

**特征**：
- 器物密集度：2-5个/句
- 刑法词汇集中：击、攻、破、杀、灭
- 遗漏率：战争场景比日常叙事高3倍

**典型段落结构**：
```
[战前准备] → 器物（车、甲、剑、弓、旗、鼓）
[战斗过程] → 刑法/军事动词（击、攻、破、败、杀）
[战后处理] → 刑法（诛、阬、夷、灭）
```

**检查策略**：
```python
# 战争场景专项扫描
import re

def check_battle_scene(text):
    battle_keywords = ['战', '击', '攻', '破', '围', '拔', '败']
    weapon_keywords = ['剑', '弓', '矢', '戟', '戈', '矛', '盾', '甲']
    vehicle_keywords = ['车', '骑', '乘']

    # 识别战争段落
    for match in re.finditer('|'.join(battle_keywords), text):
        context = text[match.start()-50:match.start()+150]

        # 检查器物标注
        for weapon in weapon_keywords:
            if weapon in context and f'〖•{weapon}〗' not in context:
                print(f"遗漏器物: {weapon} in {context[:30]}")
```

**案例**：周本纪第112段
```
原文：殊玉、轻剑、击以剑、玄钺、下车、罕旗、常车、执剑
```

### 2.4 语境判断规则（多义词）

#### A. "稷"字判断

| 语境 | 标注 | 规则 |
|------|------|------|
| 任命〖@XX〗为稷 | 〖;稷〗 | 官职（农官） |

| 让于稷 | 〖@稷〗 | 与人名并列→人名 |

**判断流程**：
```
1. 是否有"为/任/拜"等任命动词？ → 官职
2. 是否与契、皋陶等人名并列？ → 人名（后稷省称）
3. 是否独立作主语？ → 人名
4. 无法确定？ → 检查全章，统一标注
```

```
族 + 人名 → 刑法（族诛、灭族）
```


### 4.2 Python自动化检查脚本

```python
#!/usr/bin/env python3
"""实体标注一致性检查脚本"""

import re
from collections import defaultdict

class EntityChecker:
    def __init__(self, file_path):
        with open(file_path, 'r', encoding='utf-8') as f:
            self.content = f.read()
        self.errors = defaultdict(list)

    def check_punishment(self):
        """检查刑法词汇一致性"""
        punishments = ['诛', '杀', '灭', '阬', '弑', '烹', '醢', '殛', '戮', '僇', '族', '夷', '斩']
        for word in punishments:
            # 查找未标注的刑法词
            pattern = f'[^〖]{word}[^〗]'
            matches = re.finditer(pattern, self.content)
            for match in matches:
                context = self.content[match.start()-20:match.start()+30]
                self.errors['刑法遗漏'].append((word, context))

    def check_identity(self):
        """检查身份词一致性"""
        identities = ['群臣', '百姓', '士卒', '诸侯', '父老', '子弟', '公卿', '大夫']
        for word in identities:
            pattern = f'{word}(?!〗)'
            if re.search(pattern, self.content):
                self.errors['身份遗漏'].append(word)

    def check_artifact(self):
        """检查器物词一致性"""
        artifacts = ['金', '衣', '车', '剑', '弓', '矢', '帛', '琴', '鼓', '印']
        for word in artifacts:
            # 排除已标注的
            pattern = f'(?<!〖•){word}(?!〗)'
            matches = list(re.finditer(pattern, self.content))
            if matches:
                self.errors['器物遗漏'].append((word, len(matches)))

    def check_protagonist(self, name):
        """检查主人公省称"""
        # 查找已标注的完整人名
        full_pattern = f'〖@[^〗]*{name}[^〗]*〗'
        if re.search(full_pattern, self.content):
            # 检查未标注的省称
            short_pattern = f'(?<!〖@){name}(?!〗)'
            matches = list(re.finditer(short_pattern, self.content))
            if matches:
                self.errors['人名省称遗漏'].append((name, len(matches)))

    def check_battle_scene(self):
        """检查战争场景器物"""
        battle_keywords = ['战', '击', '攻', '破', '围', '拔', '败']
        weapon_keywords = ['剑', '弓', '矢', '戟', '戈', '矛', '盾', '甲', '车', '旗', '鼓']

        for keyword in battle_keywords:
            for match in re.finditer(keyword, self.content):
                context = self.content[match.start():match.start()+200]
                for weapon in weapon_keywords:
                    if weapon in context and f'〖•{weapon}〗' not in context:
                        self.errors['战争场景器物遗漏'].append((weapon, context[:50]))

    def generate_report(self):
        """生成检查报告"""
        print("=" * 60)
        print("实体标注一致性检查报告")
        print("=" * 60)
        for error_type, items in self.errors.items():
            print(f"\n{error_type}: {len(items)}处")
            for item in items[:10]:  # 显示前10个
                print(f"  - {item}")

# 使用示例
checker = EntityChecker('chapter_md/007_项羽本纪.tagged.md')
checker.check_punishment()
checker.check_identity()
checker.check_artifact()
checker.check_protagonist('羽')
checker.check_battle_scene()
checker.generate_report()
```

## 五、经验总结与建议

### 5.1 核心经验

1. **第一轮质量决定第二轮工作量**
   - 第一轮深度反思的章节，第二轮修正<20处
   - 第一轮浅度反思的章节，第二轮修正>50处

2. **grep是最有效的检查工具**
   - 刑法词汇检查：发现率95%
   - 身份词汇检查：发现率90%
   - 器物词汇检查：发现率85%

3. **场景化检查优于逐行扫描**
   - 战争场景专项检查：效率提升3倍
   - 表格体裁批量检查：效率提升5倍

4. **系统性错误需要自动化工具**
   - 年表谥号误标（66处）→ Python脚本一次修复
   - 刑法词汇遗漏（2471处）→ grep批量检查

5. **质量基线需灵活把握**
   - 不应机械追求每轮30-80处
   - 两轮合计达标即可
   - 短章节可适当降低基线
