## 五、新发现规律总结

1. 远古人名（尧、舜、象等）在段落中多次出现时，后续省称容易遗漏
2. 器物类（衣、车、琴、笠、仓廪、帛、金等）标注关注度不够
3. "稷"字需根据语境判断：任命语境→官职，与人名并列→人名
4. 数量标注需注意完整边界："二十有二人"应完整标注
5. 专题性章节（如祭祀主题）容易产生"主题盲区"：关注焦点实体而忽略常见词汇
6. 群体性身份词（群臣、百姓、群儒、诸生等）容易因"太常见"而被忽略

## 十、核心规律总结

### 1. 遗漏类高发区
- **身份类**: 群臣、百姓、诸侯、士卒等群体性身份词
- **器物类**: 金、衣、车、剑、印等常见器物
- **刑法类**: 诛、杀、灭、阬、弑等刑法动词
- **人名省称**: 历史名人在段落中重复出现时的省称

### 2. 误标类高发区
- **同形异义词**: 非子、得意、释之等可能是词组而非人名
- **谥号混淆**: 谥号误标为官职（XX侯）
- **时间/时长混淆**: X年/X岁的类型判断
- **礼仪/制度混淆**: 冠礼、聘享等行为vs制度

### 3. 场景化遗漏模式
- **战争场景**: 器物（车、剑、弓、矢、旗、鼓）密集
- **政治斗争**: 刑法词汇（诛、杀、灭）集中
- **祭祀场景**: 礼器、礼仪词汇容易因专题盲区忽略
- **表格体裁**: 年表中人名、地名、谥号大量重复

### 4. 语境判断关键点
- **稷**: 任命语境→官职，与人名并列→人名
- **族**: 动词族诛→刑法，名词宗族→氏族
- **王冠**: 成年加冠礼→礼仪，帽子→器物
- **网**: 捕猎工具→器物，抽象网络→不标注
