### 2.2 器物类（7条）

#### A17. 器物类高频遗漏（001第二轮）
- **易遗漏器物**: 衣/车/琴/笠/仓廪/瑞/服/鼎/弓/节/玺/冠
- **003章补充**: 捕猎工具（网）、财物类（宝玉/珠玉/金银）、服饰（宝玉衣）、标识（白旗/黄旗）、兵器（斧钺/弓矢/剑戟）
- **011章补充**: 布料等级（七布/十五布/粗布/细布）
- **区别**: 网在捕猎语境→器物；天罗地网抽象→不标注

#### A22. 典籍vs器物辨析（002第二轮）
- **发现**: 文献名称应标注为`〖{〗`而非`〖•〗`
- **常见错误**: ❌ `〖•五子作歌〗` ✅ `〖{五子之歌〗`

#### A23. 战争场景器物密集检查（004第二轮）
- **发现**: 战争场景器物密集，需逐句审查
- **关注动词**: 执/持/把/握/击/射/驰/下/佩/悬/挥/载/乘/骑/驱
- **高频器物**: 剑/车/弓/矢/旗/钺/戈/矛/盾/甲/戎车/兵车/罕旗/常车
- **检查方法**: `grep -nE '(执|持|把|握|击|射|驰|下|佩|悬|挥).*[剑车弓矢旗钺戈矛盾甲]' chapter_md/NNN_*.tagged.md`
- **典型段落特征**: 单段可能出现5-10处器物遗漏

#### A24. 旗名判断规则（004第二轮）
- **旗帜名称格式**: "颜色+旗"/"形状+旗"/"用途+旗"
- **器物标注**: 大白旗/小白旗/罕旗/常旗/黄旗/赤旗/黑旗/青旗 → `〖•器物〗`
- **避免误标**: 大白是旗名不是太白金星，带"旗"字明确→器物

#### A25. 器物省称识别（004第二轮）
- **发现**: 简洁语境（赞文/诗句/列举）中器物常用省称
- **常见省称**: 乐→乐器、车→戎车/兵车、兵→兵器/甲兵、旗→某某旗
- **判断方法**: 查看同章其他段落完整表达、根据语境推断、简洁文体中省称频率更高

#### A26. 礼器类词汇表（004第二轮）
- **礼器分类**:
  - 青铜器: 鼎/彝/爵/觚/尊/卣/簋
  - 玉器: 玉/璧/圭/璋/琮/琥
  - 祭品容器: 木主/神主/牌位
  - 祭祀用品: 明水/牲/粟/酒/血
  - 仪仗: 旗/车/剑/节
- **易混淆点**: 彝/明水/木主需区分器物vs制度

#### A52. 战前誓师器物固定句式（004章第二轮）
- **固定格式**: "称尔X，比尔Y，立尔Z"（X/Y/Z为器物）
- **004章案例**: "称尔〖•戈〗，比尔〖•干〗，立尔〖•矛〗"
- **类似句式**: "执X，持Y"、"左X，右Y"、"把X，秉Y"

### 2.3 人名省称类（3条）

#### A33. 主人公省称全覆盖原则（057章第二轮）
- **规律**: 世家/列传主人公的姓名省称应全覆盖，不论首次或后续
- **常见遗漏位置**: 动词前主语、介词后宾语、被动句施事、叙事句主语、动词宾语
- **057章统计**: 22处"勃"字省称遗漏
- **检查方法**: `grep -n '[^〗]主人公省称[^〗〖]' chapter_md/NNN_*.tagged.md`
- **注意**: 远古单名（尧/舜/禹等）同样适用（参考001章第二轮）

#### A46. 人名谱系传承中的单字名遗漏（062章第二轮）
- **发现**: 谱系传承语境（"X子Y，Y子Z"）中单字人名极易遗漏
- **062章案例**: "老子之子名宗，宗子注，注子宫，宫玄孙假，假之子解" → 全部需标注
- **检查方法**: `grep -nE '(之子|子名|玄孙|曾孙)' chapter_md/NNN_*.tagged.md`

#### A56. 人名边界切分错误检查（021章第二轮）
- **错误模式**: `〖;康侯〗刘〖@不害〗` → 错误；`〖;康侯〗〖@刘不害〗` → 正确
- **检查命令**: `grep -n '〖;.*侯〗刘〖@' chapter_md/NNN_*.tagged.md`
- **规律**: 汉代刘姓人名，"刘"必须包含在人名标注内

### 2.4 官职身份类（6条）

#### A21. 集体性身份/官职（002第二轮）
- **易遗漏的集体性称谓**:
  - 群后/百官/百工/子弟/父老/诸将
  - 群臣/百姓/众庶/黎民/百吏/众民/万民
- **检查方法**: `grep -nE '(群臣|百姓|众庶|黎民|百吏|众民|万民)' chapter_md/NNN_*.tagged.md`

#### A20. "今上"→身份标注（006第二轮）
- **规律**: "今上"指当今皇帝，是尊称→`〖#今上〗`，不是人名
- **类似词汇**: 今上/陛下/圣上（指当今皇帝）、先帝/先王（指已故帝王）

#### A23. 群体称谓的身份/官职判断（003章第二轮）
- **泛指群体且无具体指向**→身份或官职:
  - 群后/群臣/百吏 → `〖#〗`
  - 百工/百官 → `〖#〗`或`〖;〗`（语境判断）

#### A34. 官职+人名拆分标注（057章第二轮）
- **标注格式**: `〖;官职〗〖@人名〗`
- **修正示例**: `〖;南阳守齮〗` → `〖;南阳守〗〖@齮〗`
- **判断方法**: X+Y结构中X为职官名词→〖;〗，Y为姓名→〖@〗

#### A48. 世家群体称谓密集检查（034章第二轮）
- **高频群体称谓**: 贤者/贤士/士/士卒/壮士/诸将/百姓/众人/群臣/百官/百吏
- **034章统计**: 13处群体称谓遗漏（贤者/贤士6处，军事4处，民众5处）

#### A59. 年表侯号官职用法判断（022章新增）
- **判断标准**:
  - 侯号+人名+为+官职 → 侯号作官衔`〖;侯号〗`
  - 封+人名+侯号 → 侯号作封号`〖@侯号〗`
  - 侯号+人名（主语）→ 整体是人名`〖@侯号人名〗`
- **关键区分**: 年表职官列→侯号是身份标识→〖;〗

### 2.5 年表专项（4条）

#### A30. 年表"公"的标注豁免（014章第二轮）
- **决策**: 在年表/表类章节中，"公"作为通称可不标注，避免过度标注
- **适用条件**: 文体为年表、密度单行3次以上、语境明确指诸侯国君通称
- **说明**: 014章每行5-10处"公"字，全部标注会导致标注噪音过大

#### A31. 氏族标注在年表权力语境（014章第二轮）
- **规律**: 年表记录"X氏作乱"/"X氏专政"等权力斗争，X氏应标注为氏族类
- **判断标准**: X氏在权力、政治、家族势力语境中 → 氏族类`〖&〗`
- **014章统计**: 赵氏/崔氏/栾氏/田氏(3处)/季氏，共7处误标

#### A54. 年表侯号+人名组合检查（021章第二轮）
- **发现**: `〖;侯号〗刘X〖%元年〗` 格式的人名容易遗漏
- **检查命令**: `grep -n '〖;.*侯〗刘[^〗]' chapter_md/NNN_*.tagged.md`
- **修正模式**: `〖;X侯〗刘Y` → `〖;X侯〗〖@刘Y〗`
- **易错点**: 集中分封区域、复合名（如"狗彘"）更易遗漏

#### A55. 年表"侯X"结构词性判断（021章第二轮）
- **判断流程**:
  1. 查找前文是否有"X元年" → 有：人名 / 无：继续
  2. 检查X是否为常见副词（阴/阳/明/暗）→ 是：副词，不标注
  3. 检查X是否为动词 → 是：不标注
  4. 无法确定 → 保守不标注
- **021章案例**: `侯〖@阴〗不使人为秋请` → `侯阴不使人为秋请`（"阴"为副词"暗中"）

### 2.7 语境判断类（5条）

#### A16. 稷字语境判断（001第二轮）
- **任命语境**（"以X为稷"/"使X主稷"）→`〖;官职〗`
- **与人名并列**（"让於稷、契与皋陶"）→`〖@人名〗`（后稷省称）

#### A19. "得意"判断规则（006第二轮）
- **人名语境**: `〖@张得意〗`（明确指某个叫"得意"的人）
- **动词短语语境**: 不标注（表示满意、称心、实现意图）
- **类似易误标词汇**: 大业/大毕/昭明/天下大旱等

#### A28. "释之"消歧规则（014章第二轮）
- **情况A - 人名**: 张释之（汉朝廷尉）→ `〖@张释之〗`
- **情况B - 动词短语**: 释放之义（"肉袒谢，释之"）→ 不标注
- **判断流程**: 前文提及具体人物→人名；语境是释放/赦免→动词短语；无法确定→保守不标注

#### A44. 著述语境典籍名称识别（062章第二轮）
- **固定搭配**: "著书X篇"/"作X"/"X之书"/"X曰"/"为X书" → X是典籍
- **易混淆**: "著书二篇" → 数量"二"标注为〖$二〗，不是典籍
- **062章案例**: 孤愤/五蠹/内外储/说林/说难（韩非）、渔父/盗跖/胠箧/畏累虚/亢桑子（庄子）

#### A49. "天下"政治地理性质判断（034章第二轮，001章+002章第二轮补充）
- **政治地理实体**（授受对象）→ 〖=天下〗: "让天下於X"、"传天下於X"、"以天下授益"、"天下归X"
- **抽象概念/修辞** → 不标注: "天下大乱"、"天下皆知"、"天下属意"
- **判断标准**: 能否作为具体对象进行授受/传递 → 地名类；泛指天下人/局面/状态 → 不标注
- **典型语境**: 禅让对话、归附描述、名誉传播、战国合纵 → 政治地理实体

### 2.8 典籍思想类（1条）

#### A50. 典籍篇名vs制度概念辨析（034章第二轮）
- **典籍/篇名** → 〖{〗: 君奭(《尚书》)、甘棠(《诗经》)
- **制度/职官** → 〖^〗: 三公、六卿、从长

### 2.9 其他（9条）

#### A18. 数量标注完整边界（001第二轮）
- **错误**: `二十有〖$二人〗` → 数字前缀漏标
- **正确**: `〖$二十有二人〗` → 完整数量短语
- **其他示例**: `〖$三百六十六日〗`（不是`三百六〖$十六日〗`）

#### A19. 交通工具全覆盖（002第二轮）
- **易遗漏交通工具**: 船/舟/橇/暐/四载
- **标注规则**: ❌ `陆行〖•乘车〗` ✅ `陆行乘〖•车〗`（动词不纳入器物标注）
- **规律**: 动宾结构中，动词不纳入器物标注范围

#### A20. 测量工具识别（002第二轮）
- **准绳、规矩等测量工具**应标注为`〖•器物〗`
- **示例**: "左准绳，右规矩" → `左〖•准绳〗，右〖•规矩〗`

#### A21. 诗歌铭文实体标注（006第二轮）
- **规律**: 诗歌格式中的实体词汇也需完整标注，不因格式特殊而降低标注标准
- **器物类示例**: 戎兵/甲兵/章旗/备器/金石/珍宝/器械
- **思想类示例**: 仁义/礼法/王道

#### A29. "不礼"动词短语误标（014章第二轮）
- **错误**: `王〖@不礼〗`、`蔡〖@不礼〗`
- **正确**: 去标注（"不礼"=不以礼待之，动词短语）

#### A45. "子将"等复合结构的语法识别（062章第二轮）
- **错误**: `〖@子将〗隐矣`（误作人名）
- **正确**: 子将隐矣（"子"=您，"将"=将要）
- **识别规则**: "子+动词"结构 → "子"是尊称代词，不是人名
- **常见模式**: 子欲/子所/子言/子独/子将

#### A53. "出於"/"用事"动词短语误标（004章第二轮）
- **"出於"**: "所弃妖子出於路者" → 动词短语"出于路"，不标注
- **"用事"**: "以X为卿，用事" → 执政/掌权之义，不标注

#### A57. 干支纪年名称标注规则（026章新增）
- **天干古名（岁阳）**: 甲→焉逢、乙→端蒙、丙→游兆、丁→彊梧、戊→徒维、己→祝犁、庚→商横、辛→昭阳、壬→横艾、癸→尚章
- **地支古名（岁阴）**: 子→困敦、丑→赤奋若、寅→摄提格、卯→单阏、辰→执徐、巳→大荒落、午→敦牂、未→协洽、申→涒滩、酉→作鄂/作噩、戌→阉茂/淹茂、亥→大渊献/汁洽
- **标注规则**: 干名+支名组合均标注为〖!天文〗
- **易误标陷阱**: ❌ `〖@昭阳〗作鄂` → 误作人名；✅ `〖!昭阳〗〖!作鄂〗` → 干支古名

#### A60. 复合动词语序保持规则（022章新增）
- **原则**: 标注是语义层标记，不应改变文本表面形式
- **典型案例**: "弃某人市"（把某人扔到市场处决）
  - ❌ 错误：`〖[弃市〗〖@某人〗` → strip后"弃市某人"（语序错）
  - ✅ 正确：`〖[弃〗〖@某人〗〖[市〗` → strip后"弃某人市"（语序对）
- **判断方法**: 标注后想象去除标记，检查字序是否与原文一致
- **验证工具**: `python scripts/lint_text_integrity.py NNN` 应报告：0处实质差异
