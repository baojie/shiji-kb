# doc/ - 运行时生成的规范与日志

本目录包含项目运行过程中生成的文档、规范和日志，与静态参考资料（`resources/`）区分开来。

## 目录结构

```
doc/
├── entities/          # 实体标注反思报告（按轮次组织）
├── events/            # 事件相关文档（待整理）
├── methodology/       # 方法论文档
├── planning/          # 项目规划文档
├── predicate/         # 谓词/关系相关文档
├── reports/           # 各类分析报告
├── spec/              # 技术规范文档
├── workflow/          # 工作流文档
└── analysis/          # 数据分析结果

## 与 resources/ 的区别

- **doc/** - 运行时生成的动态文档（规范、日志、反思报告）
- **resources/** - 静态参考资料（出版物、论文、外部文献）

## 相关目录

- 静态参考资料：[`resources/`](../resources/)
- 每日工作日志：[`logs/daily/`](../logs/daily/)
- 项目结构总览：[`PROJECT_STRUCTURE.md`](../resources/help/PROJECT_STRUCTURE.md)
