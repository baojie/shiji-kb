## 小节命名特点

所有小节标题均符合以下要求：

✅ 使用史记文体风格
✅ 简洁明了（5-10字）
✅ 反映内容主题
✅ 按时间、事件或人物分类
✅ 禁止机械划分（如"段落1-10"）

## 划分原则

1. **按生平阶段**：早年经历 → 主要功绩 → 晚年结局
2. **按重大事件**：起因 → 过程 → 结果
3. **按时间顺序**：某时期 → 某事件
4. **按人物划分**：多人传记分别立节
