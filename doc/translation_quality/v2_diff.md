**替换为**：

```
**翻译要求**：
1. **准确性**：忠实原文含义，不增删内容。
   - **代词/单字简称必须展开**：原文的单字姓或字（如"斯"=李斯、"高"=赵高、"梁"=项梁、"商"=卜商、"通"=殷通）作主语或宾语时，**白话中一律展开为双字全名或姓+名**，禁止保留单字。
   - 若原文已消歧 `〖@X|Y〗`，以规范名 Y 决定展开形式。
   - 对话中第二人称 `公/君/子/若` 指代明确时译为"你"或"您"；第一人称自称（项羽说"籍…"）保留原字或译为"我"，**不可译为"项籍"**。
2. **流畅性**：使用现代汉语表达，符合当代阅读习惯。
   - **常见文言动词/句式必须改写**，译完通读一遍，凡不像现代汉语的一律替换：
     - `崩/薨/卒` → "逝世/去世"
     - `立/嗣/即位` → "继位"
     - `毋/勿` → "不要"
     - `畔` → "反叛/背叛"
     - `卻/郤` → "退却/撤退/击退"
     - `夷N族` → "诛灭N族"
     - `具N刑 / 论X刑` → "判处N刑"
     - `拜X为Y` → "任命X担任Y"
     - `下其议X / 议下X` → "把这个议题交给X讨论"
     - `收其士卒` → "收拢/收集他的士卒"
     - `於是/乃` → "于是/就"
     - `既已` → "已经"
   - **议论段落句式白话化**：`夫 X 者，Y 也` → "X，是 Y"；`…之所以…者也` → "…的原因是…"；连续 4 个以上 `者/也/则/而` 分句必须拆成两到三句短句。
3. **简洁性**：保持史记简练风格，避免冗长。
   - **不要扩写历史背景/情绪/括注**（如不要自加"享乐"/"家产万万钱"/"公元前X年"）。
   - 唯一例外是**语义消歧**（补说话人主语、补指代对象、补古今歧义地名的括号说明），这属必要而非扩写。
```

## 改动 2：在"翻译要求"末尾、"输出格式"之前新增 4 条

**原文（第 79–80 行之间的位置）**：当前条目到第 10 条（"表章仅译散文"）结束，之后就是 "**输出格式**:"。

**在第 10 条后、"**输出格式**:" 之前新增**：

```
11. **标点全角**：译文的所有标点**一律使用全角中文标点**（`，。；：！？"""''（）`）。**严禁半角** `,.;:!?"'()`。生成完必须自查；`scripts/lint_halfwidth_punct.py`（若存在）会强制校验。

12. **对话动词语气化**：原文 `曰` 根据前导情绪词选用白话动词，避免千篇一律的"说"：
    - `曰` → "说"（默认）
    - `怒曰` → "生气地说 / 怒道"
    - `笑曰` → "笑着说 / 笑道"
    - `问曰` → "问道"
    - `对曰 / 答曰` → "回答说"
    - `谏曰` → "劝谏说"
    - `让曰 / 责曰` → "责备说"
    - `谢曰` → "答谢说 / 道歉说"

13. **时间词语境区分**（关键）：
    - **年龄语境**（"少X N岁 / 长X N岁 / 年N岁"）：`岁` **保留为"岁"**，**绝不可**改为"年"。例：`少孔子三十九岁` → "比孔子小三十九岁"。
    - **时长/纪年语境**（"居N岁/岁馀/三岁而…"）：`岁` 译为"年"；`岁馀/月馀/N馀` 展开为"一年多/一个多月/N多"。
    - `X馀` 作数量时一律 "X多 / N几"（如"二十馀子" → "二十多个儿子"）。

14. **古今歧义地名**（`山东/山西/关东/关中/河南/河内/河外/中国/江南/北地/山之阳/山之阴`）：
    - 若原文已消歧 `〖=山东|...〗`，遵从规范名。
    - 未消歧且易被误读为今地名时，**首次出现**补括号说明，如 "山东（崤山以东）"、"河南（黄河以南）"、"中国（中原地区）"。
    - 已经是方位词（如"泰山之阳"）按"泰山的南面/南麓"处理，不加括号。

15. **"礼"不擅改**：除非原文明确标注为 `〖{礼〗`（典籍类），**严禁**把"礼"改写为"礼记"；用于礼义/礼节/礼仪/制度语境的"礼"保持不变。其他典籍同理：`〖{诗〗`→"诗经"，其他语境的"诗"保留。
```
