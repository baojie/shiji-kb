## 设计理念

### 1. 为什么标注动词

动词是历史叙事的核心要素，标注动词能够：

- **事件提取**：自动识别军事行动（伐/攻/击）、刑罚事件（杀/诛/斩）等历史事件
- **关系挖掘**：建立人物-动作-对象的三元组关系（如：秦伐楚、王诛臣）
- **语义检索**：按动词类型检索特定事件（如：所有"伐"的记录）
- **统计分析**：量化分析军事冲突频率、刑罚执行模式等历史规律

### 2. 动词与名词的区分

| 维度 | 名词实体 | 动词实体 |
|------|---------|---------|
| **外层符号** | `〖TYPE内容〗` (U+3016/3017 白色龟甲括号) | `⟦TYPE动词⟧` (U+27E6/27E7 数学双方括号) |
| **标注对象** | 静态实体（人名/地名/官职等） | 动态行为（军事/刑罚/政治等） |
| **TYPE符号** | 文本符号（@=;%&^等） | 几何符号（◈◉○◇） |
| **语义功能** | 事件参与者（主语/宾语） | 事件核心（谓语） |
| **示例** | 〖'楚〗〖@子常〗 | ⟦◈伐⟧⟦◉杀⟧ |

**核心原则**：使用不同的外层符号，确保动词与名词在视觉和语义上明确区分。

## 标注语法

### 1. 基本格式

```
⟦TYPE动词⟧
```

- **外层符号**：`⟦⟧` (U+27E6 MATHEMATICAL LEFT WHITE SQUARE BRACKET / U+27E7 MATHEMATICAL RIGHT WHITE SQUARE BRACKET)
- **TYPE**：动词类型符号（见下文分类体系）
- **动词**：单字或多字动词

### 2. 消歧格式

```
⟦TYPE动词|消歧说明⟧
```

- **消歧说明**：用于明确歧义词的具体含义
- **分隔符**：管道符 `|`（U+007C）

**消歧示例**：

```markdown
⟦◈取⟧〖=六〗、〖=潜〗               # 基本标注
⟦◈取|攻取⟧〖=六〗、〖=潜〗          # 消歧说明：攻取城邑

⟦◈走⟧〖'郑〗                       # 基本标注
⟦◈走|逃走⟧〖'郑〗                  # 消歧说明：逃亡之走，非行走之走

⟦◉死⟧罪                           # 基本标注
⟦◉死|处死⟧罪                      # 消歧说明：判处死刑，非自然死亡
```

### 3. 语法约束

**✅ 正确示例**：

```markdown
⟦◈伐⟧〖'随〗                       # 单字动词
⟦◉诛⟧〖@重黎〗而不尽               # 带宾语
⟦◈击|攻击⟧〖'楚〗于〖=城濮〗       # 消歧+地点
〖'吴〗⟦◈伐⟧⟦◈败⟧〖@子常〗         # 连续动词
```

**❌ 错误示例**：

```markdown
⟦⟦◈伐⟧⟧                          # 禁止嵌套
⟦◈〗                              # 禁止空标注
⟦◈伐|⟧                            # 消歧说明不能为空
〖◈伐〗                            # 错误的外层符号（应用⟦⟧）
⟦@伐⟧                             # 错误的TYPE符号（◈非@）
```

## 标注原则

### 1. 动词识别原则

**✅ 应标注的动词**：
- 历史事件的核心动作（军事/刑罚/政治等）
- 具有明确主语和宾语的及物动词
- 可用于事件提取和关系挖掘的关键动词

**❌ 不应标注的动词**：
- 一般性日常动词（如：行、走、坐、立、言、语）
- 助动词和连系动词（如：能、可、是、为）
- 趋向动词（如：来、去、上、下，除非表示军事机动）

### 2. 消歧原则

**需要消歧的情况**：

1. **同形异义**：一个字有多种动词含义
   ```markdown
   ⟦◈取|攻取⟧〖=城〗      # 军事之取
   取而代之                # 一般之取（不标注）

   ⟦◈走|逃走⟧奔〖'郑〗    # 逃亡之走
   走马观花                # 行走之走（不标注）
   ```

2. **跨类歧义**：同一字可属于多个动词类型
   ```markdown
   ⟦◈亡|逃亡⟧奔〖'郑〗    # 军事逃亡
   ⟦◉亡|逃罪⟧出奔        # 逃避刑罚
   ```

3. **与名词混淆**：避免与同形名词混淆
   ```markdown
   ⟦◈围⟧〖=邯郸〗         # 动词：包围
   解〖=邯郸〗之围         # 名词：包围圈（不标注）
   ```

### 3. 标注边界原则

**单字标注**：大部分动词为单字
```markdown
⟦◈伐⟧、⟦◈攻⟧、⟦◈击⟧、⟦◉杀⟧、⟦◉诛⟧
```

**多字标注**：复合动词可标注为整体（谨慎使用）
```markdown
⟦◈击破⟧〖'楚〗          # 击破作为一个整体
⟦◈击⟧⟦◈破⟧〖'楚〗      # 或拆分为两个动词（推荐）
```

### 4. 优先级原则

当一个词同时满足多个类型时，按以下优先级标注：

1. **军事动词** `⟦◈⟧` 优先于刑罚动词
   ```markdown
   ⟦◈杀⟧敌千人           # 战场杀敌用军事标注
   ⟦◉杀⟧〖@大臣〗        # 政治处决用刑罚标注
   ```

2. **刑罚动词** `⟦◉⟧` 优先于制度名词
   ```markdown
   ⟦◉斩⟧〖@颜良〗        # 动作：斩杀
   〖[斩首〗八万           # 名词：斩首刑
   ```

3. **特定上下文优先**：根据语境判断
   ```markdown
   〖'晋〗⟦◈救⟧〖'宋〗    # 军事救援
   救民于水火             # 一般救助（不标注）
   ```

| **外层符号** | `〖〗` 白色龟甲括号 | `⟦⟧` 数学双方括号 |
| **Unicode** | U+3016 / U+3017 | U+27E6 / U+27E7 |
| **TYPE符号类型** | 文本符号（@=;%等） | 几何符号（◈◉○◇） |
| **视觉区分** | 较圆润的括号 | 较方正的括号 |

### 2. 语义对比

| 对比项 | 名词实体 | 动词实体 |
|-------|---------|---------|
| **标注对象** | 静态实体（人、地、物） | 动态行为（动作、事件） |
| **句法功能** | 主语、宾语、定语 | 谓语 |
| **语义角色** | 事件参与者 | 事件核心 |
| **持久性** | 持续存在 | 瞬时发生 |

### 3. 混合标注示例

```markdown
〖'楚〗⟦◈伐⟧〖'随〗
 名词  动词  名词
 主语  谓语  宾语

〖@帝喾〗以〖%庚寅〗日⟦◉诛⟧〖@重黎〗
  主语    时间状语   动词    宾语

〖'吴〗⟦◈伐⟧⟦◈败⟧〖@子常〗，〖@子常〗⟦◉亡⟧奔〖'郑〗
 主语  动词1 动词2  宾语      主语    动词  趋向 宾语
```

## 标注示例

### 1. 军事动词示例

#### 示例1：单纯军事行动

**原文**：
```
楚伐随。楚灭邓。
```

**标注**：
```markdown
〖'楚〗⟦◈伐⟧〖'随〗。〖'楚〗⟦◈灭⟧〖'邓〗。
```

**解析**：
- `⟦◈伐⟧`：征伐，进攻类军事动词
- `⟦◈灭⟧`：消灭，交战类军事动词

#### 示例2：复杂战役叙述

**原文**：
```
吴伐败子常，子常亡奔郑
```

**标注**：
```markdown
〖'吴〗⟦◈伐|征伐⟧⟦◈败|击败⟧〖@子常〗，〖@子常〗⟦◈亡|逃亡⟧奔〖'郑〗
```

**解析**：
- `⟦◈伐|征伐⟧`：征伐，添加消歧说明
- `⟦◈败|击败⟧`：击败，明确为主动击败
- `⟦◈亡|逃亡⟧`：逃亡，区分于"死亡"之亡

#### 示例3：攻城战术

**原文**：
```
攻宛、舞阳，下城七十余，破秦军，斩首八万
```

**标注**：
```markdown
⟦◈攻⟧〖=宛〗、〖=舞阳〗，⟦◈下⟧〖=城〗七十余，⟦◈破⟧〖*秦军〗，〖[斩首〗八万
```

**解析**：
- `⟦◈攻⟧`：进攻城池
- `⟦◈下⟧`：攻下，占领
- `⟦◈破⟧`：击破军队
- `〖[斩首〗`：制度名词（非动词），表示斩首刑的执行结果

### 2. 刑罚动词示例

#### 示例1：政治处决

**原文**：
```
帝喾使重黎诛之而不尽。帝乃以庚寅日诛重黎
```

**标注**：
```markdown
〖@帝喾〗使〖@重黎〗⟦◉诛⟧之而不尽。帝乃以〖%庚寅〗日⟦◉诛⟧〖@重黎〗
```

**解析**：
- `⟦◉诛⟧`：诛杀，刑罚动词
- 两次出现，分别为对不同对象的诛杀

#### 示例2：弑君事件

**原文**：
```
商臣以宫兵围成王，成王请食熊蹯而死，商臣弑成王
```

**标注**：
```markdown
〖@商臣〗以宫兵⟦◈围|围困⟧〖@成王〗，〖@成王〗请食〖+熊蹯〗而⟦◉死|处死⟧，〖@商臣〗⟦◉弑|弑君⟧〖@成王〗
```

**解析**：
- `⟦◈围|围困⟧`：用宫兵包围，军事动词（虽用于政变，但行为本身是军事性的）
- `⟦◉死|处死⟧`：被迫处死
- `⟦◉弑|弑君⟧`：弑君，添加消歧说明明确性质

#### 示例3：刑罚制度混用

**原文**：
```
杀令尹子西，斩首八万。当死，赎为城旦
```

**标注**：
```markdown
⟦◉杀⟧〖;令尹〗〖@子西〗，〖[斩首〗八万。当⟦◉死⟧，〖[赎为〗〖[城旦〗
```

**解析**：
- `⟦◉杀⟧`：动词，杀害大臣
- `〖[斩首〗`：制度名词，斩首刑
- `⟦◉死⟧`：动词，判处死刑
- `〖[赎为〗`、`〖[城旦〗`：制度名词，刑罚制度术语

### 3. 军事+刑罚混合示例

#### 示例1：战争与处罚

**原文**：
```
楚伐随。楚灭邓。杀令尹子西
```

**标注**：
```markdown
〖'楚〗⟦◈伐⟧〖'随〗。〖'楚〗⟦◈灭⟧〖'邓〗。⟦◉杀⟧〖;令尹〗〖@子西〗
```

**解析**：
- 前两句为军事动词（国与国之间的战争）
- 第三句为刑罚动词（政治内部的处决）

#### 示例2：政变叙事

**原文**：
```
商臣袭杀父，自立为王，诛大臣三十人
```

**标注**：
```markdown
〖@商臣〗⟦◈袭⟧⟦◉杀⟧〖#父〗，自立为〖;王〗，⟦◉诛⟧〖;大臣〗三十人
```

**解析**：
- `⟦◈袭⟧`：袭击，军事动词（政变的军事行动）
- `⟦◉杀⟧`：杀害父亲，刑罚动词（弑父）
- `⟦◉诛⟧`：诛杀大臣，刑罚动词

#### 示例1：歧义词"取"

```markdown
# 军事之取（攻取）
⟦◈取|攻取⟧〖=六〗、〖=潜〗

# 政治之取（夺取）
⟦○取|夺取⟧〖;王位〗（未来扩展）

```


## 常见问题

### Q1: 如何区分军事动词和刑罚动词？

**A**: 根据上下文判断：

- **军事动词**：国与国、军队与军队之间的战争行为
  ```markdown
  〖'秦〗⟦◈伐⟧〖'楚〗        # 国家间战争
  ⟦◈杀⟧敌千人              # 战场杀敌
  ```

- **刑罚动词**：政治内部的惩处、处决行为
  ```markdown
  ⟦◉杀⟧〖;丞相〗〖@李斯〗  # 政治处决
  ⟦◉诛⟧〖@奸臣〗           # 刑罚诛杀
  ```

### Q2: 多字动词如何标注？

**A**: 推荐拆分为单字动词：

```markdown
# 推荐：拆分标注
⟦◈击⟧⟦◈破⟧〖'楚〗

# 可选：整体标注（谨慎使用）
⟦◈击破⟧〖'楚〗
```

**理由**：单字标注便于统计分析和语义检索。

### Q3: 如何判断是否需要消歧？

**A**: 以下情况需要消歧：

1. 同形异义词（如：走/取/下）
2. 在文本中频繁出现且含义不一致
3. 对理解历史事件有重要影响

**不需要消歧**：含义明确、上下文清晰的词。

### Q4: 动词标注是否会影响阅读体验？

**A**: 标注后可通过CSS控制显示：

- **默认模式**：仅高亮显示，不影响阅读
- **标注模式**：显示完整标注符号
- **悬停提示**：鼠标悬停显示动词类型

### Q5: 如何处理"引兵""发兵""起兵"等多字军事词？

**A**: 这些属于**军事名词短语**，而非单纯动词：

```markdown
〖@项羽〗〖[引兵〗渡〖=河〗     # 制度/术语名词
〖@陈胜〗〖[起兵〗于〖=大泽〗   # 制度/术语名词
```

如未来扩展，可标注为军事动词：
```markdown
〖@项羽〗⟦◈引兵⟧渡〖=河〗     # 军事动词（未来）
```

### Q6: 旧标注格式如何迁移？

**A**: 使用迁移工具 `migrate_verb_tags.py`：

```bash
# 预览迁移
python kg/entities/scripts/migrate_verb_tags.py --chapter 040 --dry-run

# 执行迁移
python kg/entities/scripts/migrate_verb_tags.py --all --backup

# 迁移结果
〖[伐〗 → ⟦◈伐⟧
〖[杀〗 → ⟦◉杀⟧
〖[斩首〗 → 保持不变（制度名词）
```

- 生成迁移报告

```bash
python kg/entities/scripts/migrate_verb_tags.py --all --backup

# 仅迁移军事动词
python kg/entities/scripts/migrate_verb_tags.py --all --type military

python kg/entities/scripts/migrate_verb_tags.py --all --type penalty

python kg/entities/scripts/migrate_verb_tags.py --all --report migration_report.md
```

### 4. 正则表达式模式

#### 匹配旧格式动词

```python
pattern_old = r'〖\[([^〗]+)〗'
```

#### 匹配新格式动词

```python
pattern_new = r'⟦([◈◉○◇])([^⟧|]+)(?:\|[^⟧]+)?⟧'
```

#### 提取动词类型

```python
import re

text = "〖'楚〗⟦◈伐⟧〖'随〗"
matches = re.findall(r'⟦([◈◉○◇])([^⟧|]+)(?:\|([^⟧]+))?⟧', text)

# matches = [('◈', '伐', None)]
# 类型符号: ◈ (军事)
# 动词: 伐
# 消歧说明: None
```
