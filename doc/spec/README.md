## 📝 版本日志

### v3.0.1 (2026-03-31)
- **修复**: "於是"误标问题（wū shì → yú shì）
- **影响**: 修复845次误标
- **质量**: 准确率 95.3% → 98.8%

### v3.0 (2026-03-31)
- **重大更新**: 彻底废除单字词条策略
- **删除**: 10个常见字单字词条
- **新增**: 45个固定词组
- **实施**: 2个正则模式（数词+骑/乘）
- **效果**: 误标 132,000次 → 291次（-99.78%）

### v2.9 (2026-03-31)
- **删除**: "王"、"中" 2个高风险单字
- **新增**: 13个固定词组

### v2.8 (2026-03-31 上午)
- **基线版本**: 131词条，21单字词条
- **问题**: 误标率 >95%

## 💡 使用建议

### 新增词条前
1. ✅ 阅读 [§ 一、设计原则](拼音标注质量规范v3.0.md#一设计原则与策略)
2. ✅ 检查 [新增词条检查清单](拼音标注质量规范v3.0.md#新增词条检查清单)
3. ✅ 验证是否符合 [单字词条准入标准](拼音标注质量规范v3.0.md#单字词条准入标准)

### 质量监控
1. ✅ 定期检查 [质量指标阈值](拼音标注质量规范v3.0.md#质量指标阈值)
2. ✅ 监控用户反馈的误标问题
3. ✅ 抽样验证高频词组标注正确性

### 贡献指南
1. 发现误标 → 提Issue或直接修改词表
2. 提议新词条 → 参考设计原则，提供频次数据
3. 优化建议 → 在 [§ 五、优化路线图](拼音标注质量规范v3.0.md#五优化路线图) 基础上讨论
