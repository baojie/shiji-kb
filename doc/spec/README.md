## 文档分类原则

根据 [SKILL_10e_文件组织与目录结构](../../skills/SKILL_10e_文件组织与目录结构.md)：

- **本目录（doc/spec/）** - 已定稿的技术规范和设计文档（内部使用）
- **docs/** - 已发布的静态网站文件（外部访问）
- **labs/planning/** - 实验性规划文档（未定稿）
- **labs/prototypes/** - 原型代码和POC
- **labs/research/** - 技术调研文档

## 文档维护

所有文档应遵循以下原则：
1. 使用中文命名，便于检索
2. 保持内容准确性和时效性
3. 更新时同步修改相关引用
4. 定期归档过时文档
