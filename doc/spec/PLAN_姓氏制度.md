# 先秦姓氏制度 — 史记知识库背景规范

> 为什么《史记》里"同姓"不等于"近亲"？为什么薛国君主不姓"薛"？本文档说明先秦姓氏制度的核心逻辑，及本知识库的数据建模方案。

---

## 一、核心问题

分清古文中"谁是谁"是阅读《史记》的最大障碍之一。先秦时代的"姓"与"氏"是两个完全不同的概念，但现代读者极易混淆。

**典型混淆案例：**

| 案例 | 文本表述 | 实际关系 | 常见误解 |
|------|---------|---------|---------|
| 屈原 | "楚之同姓也" | 芈姓，屈氏；与楚王室同血缘但已分立约350年 | 误以为是楚王室直系成员 |
| 薛国 | 薛公、薛人 | 任姓薛氏；统治者不姓"薛" | 误以为薛国君主姓薛 |
| 司马迁 | 太史令之家 | 姬姓司马氏；"司马"是官职来源的氏，非姓 | 误以为姓"司马" |

## 二、姓与氏的区别

### 2.1 定义

| 概念 | 汉字 | 含义 | 承传方式 | 用法 |
|------|------|------|---------|------|
| **姓** | xìng | 血缘标志，同祖同源 | 父系传代，极少改变 | 女性用姓，辨别婚配禁忌 |
| **氏** | shì | 分家标志，封地/官职/祖名 | 由封地、官职、祖先名等衍生 | 男性用氏，表示地位身份 |

### 2.2 核心规则

- 女子称姓不称氏（婚姻禁忌：**同姓不婚**）
- 男子称氏不称姓（日常标识、社会地位）
- 同姓可以不同氏（同一血缘中，各分家有不同氏）
- "同姓"只说明共同血缘始祖，**不代表近亲关系**

### 2.3 氏的三大来源

| 类型 | 说明 | 示例 |
|------|------|------|
| **以封地为氏** | 受封某地，后代以地名为氏 | 屈原：楚武王子屈瑕封于"屈"，后代以"屈"为氏 |
| **以官职为氏** | 先祖担任某官，后代以官名为氏 | 司马氏：先祖为周室司马，后代以"司马"为氏 |
| **以祖先名字为氏** | 以始祖的名或字为氏 | 孔氏：以先祖孔父嘉之"孔"为氏 |

### 2.4 秦统一之后

- 封建分封制被郡县制取代，"氏"失去政治意义
- 姓氏合流，统称"姓氏"
- 现代汉语中的"姓"实为古代的"氏"（如"姓刘"实为"刘氏"）
- 《史记》中秦汉人物已无需严格区分姓/氏

```json
      "zi": "原",
      "period": "pre-qin",
      "xing_heritage": "颛顼帝之后，芈姓，楚国王室同血缘",
      "shi_type": "封地",
      "shi_origin": "楚武王子屈瑕受封于屈邑，以邑名为氏，传至屈原约350年",
      "generations_from_founder": 9,
      "common_confusion": "《史记》'楚之同姓也'指芈姓同源，非直系；屈氏分家已逾350年",
      "confidence": "high",
      "evidence": [
        "《屈原贾生列传》：屈原者，名平，楚之同姓也",
        "《楚世家》：武王子瑕，封于屈，以邑为氏"
      ]
    }
  }
}
```

**字段说明：**

| 字段 | 类型 | 说明 | 必填 |
|------|------|------|------|
| `xing` | string | 姓（血缘，如姬/姜/芈/嬴/任） | 是 |
| `shi` | string | 氏（分家标志，可为空） | 否 |
| `ming` | string | 名 | 是 |
| `zi` | string | 字 | 否 |
| `period` | enum | `pre-qin` / `qin` / `han` | 是 |
| `xing_heritage` | string | 姓的血缘来源路径 | 推荐 |
| `shi_type` | enum | `封地` / `官职` / `祖名` / `其他` | 推荐 |
| `shi_origin` | string | 氏的形成过程 | 推荐 |
| `generations_from_founder` | int | 距氏创始人的代数 | 可选 |
| `common_confusion` | string | 常见误解及澄清 | 推荐 |
| `confidence` | enum | `exact` / `high` / `medium` / `low` / `legend` | 是 |
| `evidence` | array | 《史记》原文依据 | 推荐 |

## 五、典型案例：屈原

**证据链：**
1. `《屈原贾生列传》`："屈原者，名平，楚之同姓也" → 与楚王室同姓（芈姓）
2. `《楚世家》`：楚武王子瑕受命"屈瑕"，后代以"屈"为氏
3. 推算：楚武王约前741–前690年在位，屈原约前340–前278年，时间差约350年，约8–10代

**结论：**

> **屈原**：本名芈平，字原。芈姓，屈氏（以封地为氏）。与楚王室同姓（芈姓），但屈氏分家已逾350年，为远亲而非近戚。"楚之同姓也"指血缘同源，不意味着政治上的亲密关系。
