# 史记辞典规划

## 项目愿景

编一本史记辞典，就像唐诗鉴赏词典；可以很长，就像 TikZ 手册；方式可以创新，就像反向词典。

**提议人**：lifeng
**创建日期**：2026-03-31

## 核心理念

### 1. 内容深度：像唐诗鉴赏词典

- **全面性**：覆盖《史记》130篇中的重要词汇、典故、人物、地名、官职、制度等
- **鉴赏性**：不仅解释字面含义，还包括历史背景、文化内涵、司马迁的写作技巧
- **可读性**：每个词条都是一篇小文章，引人入胜

### 2. 篇幅自由：像 TikZ 手册

- **详尽性**：不限制篇幅，该详细的地方就详细
- **技术性**：可以包含大量示例、引文、对比分析
- **结构化**：分层次、分模块，方便查阅和学习

### 3. 创新方式：像反向词典

#### 参考：WantWords 反向词典 (https://wantwords.net/)

- **功能**：根据词语的意思搜索相关联的词语
- **技术**：基于大语言模型，支持语义搜索
- **创新点**：
  - 反向查询（描述→词语）
  - 语义聚类（相关词语集合）
  - 多维度关联（音韵、字形、语义）

#### 史记辞典的创新可能

1. **倒序索引**
   - 按词尾字符索引（如《现代汉语词典·倒序本》）
   - 方便发现词语规律、韵脚、句式模式

2. **语义检索**
   - 输入"刺杀秦王的刺客" → 找到"荆轲""专诸""聂政"等相关词条
   - 输入"汉初三杰的谋士" → 找到"张良""陈平"等

3. **多维度关联**
   - **时间线**：按历史时期组织词条（夏商周秦汉）
   - **人物关系网**：展示人物之间的师承、敌友、婚姻关系
   - **地理空间**：词条与地图关联
   - **文体特征**：按文体（本纪、世家、列传、书、表）分类

4. **AI辅助功能**
   - 基于知识图谱的智能问答
   - 词语用法的语境示例自动生成
   - 相似表达的自动推荐

## 内容结构

### 词条类型

1. **词汇类**
   - 实词：名词、动词、形容词等
   - 虚词：助词、连词、介词等
   - 成语典故：源自《史记》的成语及其出处

2. **专名类**
   - 人物：传主、重要人物
   - 地名：国名、郡县、山川
   - 官职：爵位、职官名称
   - 制度：礼制、法律、军事制度

3. **文献类**
   - 引文出处：《史记》引用的古文献
   - 异文校勘：版本差异、通假字
   - 注疏沿革：历代注家的解释

4. **主题类**
   - 历史事件：重大战役、政治事件
   - 思想主题：司马迁的史学思想
   - 文学手法：叙事技巧、修辞手法

### 词条结构（示例）

```markdown
## 【词条】韩信

### 基本信息
- **拼音**：hán xìn
- **异体**：韩信卢绾（与"卢绾"并称）
- **类别**：人物·汉初名将

### 释义
淮阴人，西汉开国功臣，军事家，汉初三杰之一。出身贫寒，曾受胯下之辱，后投奔项羽不被重用，转投刘邦，经萧何推荐被拜为大将军。

### 史记原文
> 淮阴侯韩信者，淮阴人也。始为布衣时，贫无行，不得推择为吏……

### 典故成语
- **胯下之辱**：韩信年轻时受市井无赖欺辱，从其胯下爬过
- **萧何月下追韩信**：萧何力荐韩信，连夜追回欲离去的韩信
- **成也萧何，败也萧何**：韩信因萧何得拜大将，最终也因萧何设计被诛

### 关联人物
- **刘邦**：主君，关系：君臣
- **萧何**：推荐人，关系：伯乐
- **张良**：同为汉初三杰
- **彭越**：同时期名将
- **英布**：同时期名将

### 历史评价
- **司马迁评**："国士无双""功高震主"
- **后世评价**：兵仙、谋战之神

### 参考章节
- [韩信卢绾列传](../../archive/史记093_韩信卢绾列传.txt) (卷九十三)
- [高祖本纪](../../archive/史记008_高祖本纪.txt) (卷八)

### 知识图谱链接
- 人物关系图：[查看](link)
- 时间线：公元前？年 - 公元前196年
- 地理位置：淮阴（今江苏淮安）

### 延伸阅读
- 《淮阴侯列传》赏析
- 韩信军事思想研究
```

### 阶段二：词条生成

2. **人工审校**
   - 补充历史背景
   - 添加鉴赏内容
   - 校对准确性

### 阶段三：索引系统
1. **传统索引**
   - 拼音顺序索引

   - 关系索引（人物关系网）

1. **静态版本**

2. **动态版本**
   - 在线检索系统
   - AI 问答助手
   - 反向词典功能（输入描述→找词条）

### 阶段五：AI 增强

   - 基于词条内容的向量检索
   - 支持自然语言查询

2. **智能推荐**

3. **生成功能**
