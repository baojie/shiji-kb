## 主要更新

### 1. CSS样式升级到v5.4

**版本路径**：`v5.3.2 → v5.3.3 → v5.4`

**核心改进**：
1. ✅ **朝代下划线加粗**：从1px升级到2px，突出政治重要性
2. ✅ **典籍添加波浪线**：使用传统书名波浪线 (2px + 3px offset)
3. ✅ **神话添加波浪线**：神话传说使用波浪线标识 (2px + 3px offset)
4. ✅ **思想下划线加粗**：点线从1px加粗到2px，改进颜色透明度

**设计原则**：
- 保持 v5.3.x 的 `text-decoration` 风格
- 保持统一的 `text-underline-offset: 3px`
- 应用 v6.0 方案B的精选视觉增强

#### 朝代（加粗下划线）
```css
.dynasty {
    color: #9370DB;
    background-color: #F0E6FF;
    text-decoration: underline solid #9370DB80;
    text-decoration-thickness: 2px;  /* ← v5.4新增 */
    text-underline-offset: 3px;
    padding: 0 3px;
    border-radius: 2px;
}
```

#### 典籍（波浪线）
```css
.book {
    color: #4B0082;

    text-underline-offset: 3px;  /* ← v5.4新增 */

}
```
