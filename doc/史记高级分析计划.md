## 一、社会网络分析（Social Network Analysis）

### 1.1 目标
构建《史记》人物关系网络图，揭示人物间的互动模式和社会结构。

### 1.2 方法
- **节点**：人物（1820个人名）
- **边**：关系类型
  - 君臣关系
  - 师友关系
  - 父子/家族关系
  - 敌对关系
  - 联盟关系
- **网络指标**：
  - 中心度（Centrality）：识别核心人物
  - 聚类系数（Clustering Coefficient）：发现人物群体
  - 路径长度：分析人物间的联系紧密度
  - PageRank：评估人物影响力

### 1.3 工具
- NetworkX（Python）
- Gephi（可视化）
- D3.js（交互式网页展示）

### 1.4 预期产出
- 人物关系网络图（整体）
- 分朝代网络图（春秋、战国、秦、汉）
- 核心人物分析报告
- 社会结构演变分析

### 1.5 研究价值
- 发现隐藏的历史人物联系
- 理解权力结构与社会网络
- 验证或挑战传统历史认知

## 二、时空分析（Spatio-temporal Analysis）

### 2.1 目标
结合地名（954个）和时间（390个）标注，绘制历史事件的时空分布。

### 2.2 方法
- **地理信息系统（GIS）**：
  - 标注地名坐标
  - 绘制历史地图
- **时间轴**：
  - 按年代排列事件
  - 标注关键时间节点
- **时空可视化**：
  - 动态地图：展示历史事件的时空演变
  - 热力图：显示不同时期的地理活动热点

### 2.3 工具
- QGIS（地理信息系统）
- Leaflet.js（交互式地图）
- TimelineJS（时间轴）
- Python GeoPandas

### 2.4 分析维度
- **空间维度**：
  - 战争分布：主要战场位置
  - 政治中心变迁：都城、封地
  - 人物活动范围：游历路线
- **时间维度**：
  - 历史事件密度：哪些时期事件最多
  - 朝代更替节点
  - 重大历史转折点

### 2.5 预期产出
- 历史事件时空分布地图
- 战争路线图
- 人物游历地图
- 政治中心变迁图

## 三、事件图谱（Event Knowledge Graph）

### 3.1 目标
构建历史事件关联网络，揭示事件间的因果关系。

### 3.2 事件要素
- **事件主体**：参与者（人名）
- **事件地点**：发生地（地名）
- **事件时间**：时间标注
- **事件类型**：
  - 战争
  - 政变
  - 改革
  - 外交
  - 文化活动

### 3.3 关系类型
- 因果关系：A导致B
- 时序关系：A在B之前
- 并发关系：A与B同时发生
- 对立关系：A与B冲突

### 3.4 提取方法
- 规则模板匹配
- 依存句法分析
- LLM辅助标注

### 3.5 预期产出
- 历史事件关联图
- 因果链分析
- 重大历史事件影响力评估

## 五、文体计量分析（Stylometric Analysis）

### 5.1 目标
量化分析《史记》五体（本纪、表、书、世家、列传）的语言特征差异。

### 5.2 分析指标

#### 5.2.1 基础统计
- 平均句长
- 平均段落长度
- 词汇丰富度（TTR: Type-Token Ratio）
- 字数分布

#### 5.2.2 词汇特征
- 高频词分布
- 虚词使用频率
- 实词/虚词比例
- 专有名词密度

#### 5.2.3 句法特征
- 句式分布（陈述、疑问、感叹）
- 特殊句式频率（判断、被动、倒装）
- 对话占比
- 引文占比

#### 5.2.4 修辞特征
- 对偶频率
- 排比频率
- 比喻频率
- 韵文占比

### 5.3 对比分析
- **本纪 vs 列传**：叙事风格差异
- **书 vs 其他体**：论述性强弱
- **表 vs 其他体**：简洁度对比

### 5.4 工具
- Python NLTK/jieba
- Stylo（R语言包）
- 自定义文本分析脚本

### 5.5 预期产出
- 五体语言特征对比报告
- 聚类分析：哪些篇章文风相似
- 作者归属分析（探讨是否有多作者）

## 六、主题建模（Topic Modeling）

### 6.1 目标
使用统计方法发现《史记》中的隐含主题。

### 6.2 方法
- **LDA（Latent Dirichlet Allocation）**：
  - 主题数量：10-20个
  - 主题词数量：每主题10-15个词
- **NMF（Non-negative Matrix Factorization）**
- **BERTopic**：基于BERT的主题建模

### 6.3 分析步骤
1. 文本预处理：分词、去停用词
2. 构建词-文档矩阵
3. 训练主题模型
4. 主题可视化
5. 主题解释与命名

### 6.4 预期主题（假设）
- 战争与军事
- 礼制与法律
- 经济与财政
- 外交与联盟
- 思想与文化
- 天文与历法
- 家族与世系
- 人物品德
- 自然灾害
- 改革变法

### 6.5 工具
- Gensim（Python）
- pyLDAvis（可视化）
- BERTopic

### 6.6 预期产出
- 主题分布图
- 主题-篇章关联
- 主题演变趋势
- 主题词云

## 九、修辞手法统计

### 9.1 修辞类型识别

#### 9.1.1 对偶
- 识别方法：句式结构、字数对称
- 预估数量：500-1000处
- 功能：增强节奏感

#### 9.1.2 排比
- 识别方法：三个及以上并列结构
- 预估数量：300-500处
- 功能：增强气势

#### 9.1.3 比喻
- 识别方法：比喻词（如、若、犹、似）
- 预估数量：200-400处
- 功能：形象生动

#### 9.1.4 夸张
- 识别方法：极端表达
- 预估数量：100-200处
- 功能：突出特征

#### 9.1.5 反问
- 识别方法：疑问词+语气
- 预估数量：50-100处
- 功能：强烈质问

#### 9.1.6 引用
- 识别方法：引用标记
- 预估数量：100-150处
- 功能：增加说服力
