## 一、社会网络分析（Social Network Analysis）

### 1.1 目标
构建《史记》人物关系网络图，揭示人物间的互动模式和社会结构。

### 1.2 方法
- **节点**：人物（4,142个人名）
- **边**：关系类型
  - 君臣关系
  - 师友关系
  - 父子/家族关系
  - 敌对关系
  - 联盟关系
- **网络指标**：
  - 中心度（Centrality）：识别核心人物
  - 聚类系数（Clustering Coefficient）：发现人物群体
  - 路径长度：分析人物间的联系紧密度
  - PageRank：评估人物影响力

### 1.3 工具
- NetworkX（Python）
- Gephi（可视化）
- D3.js（交互式网页展示）

### 1.4 预期产出
- 人物关系网络图（整体）
- 分朝代网络图（春秋、战国、秦、汉）
- 核心人物分析报告
- 社会结构演变分析

### 1.5 当前基础
- 已有：4,142人名词表、实体消歧映射、人物生卒年数据、家族关系初步提取
- 待做：大规模关系抽取、网络构建与分析

### 1.6 研究价值
- 发现隐藏的历史人物联系
- 理解权力结构与社会网络
- 验证或挑战传统历史认知

## 二、时空分析（Spatio-temporal Analysis）

### 2.1 目标
结合地名（1,820个）和时间（979个）标注，绘制历史事件的时空分布。

### 2.2 方法
- **地理信息系统（GIS）**：
  - 标注地名坐标
  - 绘制历史地图
- **时间轴**：
  - 按年代排列事件
  - 标注关键时间节点
- **时空可视化**：
  - 动态地图：展示历史事件的时空演变
  - 热力图：显示不同时期的地理活动热点

### 2.3 工具
- QGIS（地理信息系统）
- Leaflet.js（交互式地图）
- TimelineJS（时间轴）
- Python GeoPandas

### 2.4 分析维度
- **空间维度**：
  - 战争分布：主要战场位置
  - 政治中心变迁：都城、封地
  - 人物活动范围：游历路线
- **时间维度**：
  - 历史事件密度：哪些时期事件最多
  - 朝代更替节点
  - 重大历史转折点

### 2.5 当前基础
- 已有：3,092个事件（含时间、地点、人物要素）、事件时间索引、地铁图可视化
- 待做：地名坐标匹配、GIS地图生成、时空动态可视化

### 2.6 预期产出
- 历史事件时空分布地图
- 战争路线图
- 人物游历地图
- 政治中心变迁图

- 平均段落长度

- 字数分布

#### 5.2.2 词汇特征
- 高频词分布
- 虚词使用频率
- 实词/虚词比例

#### 5.2.3 句法特征

#### 5.2.4 修辞特征
- 对偶频率
- 排比频率
- 比喻频率
- 韵文占比

### 5.3 对比分析
- **本纪 vs 列传**：叙事风格差异
- **书 vs 其他体**：论述性强弱

## 六、主题建模（Topic Modeling）

### 6.1 目标
使用统计方法发现《史记》中的隐含主题。

### 6.2 方法
- **LDA（Latent Dirichlet Allocation）**：
  - 主题数量：10-20个
  - 主题词数量：每主题10-15个词
- **NMF（Non-negative Matrix Factorization）**
- **BERTopic**：基于BERT的主题建模

### 6.3 分析步骤
1. 文本预处理：分词、去停用词
2. 构建词-文档矩阵

4. 主题可视化
5. 主题解释与命名
