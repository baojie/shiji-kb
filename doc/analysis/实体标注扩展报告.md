## 三、各阶段详情

### Phase 1：单字人名省称推断

**算法**：扫描本章已标注全名 `@XX@`（≥2字），取末字作省称候选；在未标注文字中匹配句法框架（主语位：`X曰/X乃/X遂`；宾语位：`斩X/杀X/败X`）；子句边界约束防止误报。

**过滤**：AMBIGUOUS_CHARS（170+字）过滤高歧义末字（虚词/动词/国名字/官职后缀）

**结果**：
- 生成建议：79条（覆盖30章）
- 实际插入：**513处**
- 示例：留侯世家 `@张良@`→`@良@` 共插入20处

**质量控制**：仅在本章有全名标注的前提下推断（章内共指），不跨章。

---

### Phase 2：已知实体别名归一化补标

**算法**：加载 `entity_index.json`（13,478个词条，含所有类型正名+别名），对每章在未标注文字中执行贪心最长匹配，被覆盖位置不重复匹配。

**过滤机制**：
- **SKIP_WORDS 黑名单**（27词）：高频歧义词，含：
  - 人名误报：子之（"天子之德"中高频出现）、公子、大王、将军、而立等
  - 官职泛称：太子、诸侯、大臣、至于、陛下等
  - 地名泛称：天下（改由思想类处理）
  - 数量误报：一人、四人、十馀、万里、日夜
- **纯数字过滤**：全由汉字数词组成的词（二十/八十一等）跳过，消除乐律/年表类章节中的数学分数误报（entity_index中有80个纯数字time条目）

**结果**：
- 原始建议：32,102条 → 过滤后 21,975条（减少31%）
- 实际插入：**22,227处**（部分词已被Phase 1标注，去重后）

**主要误报来源与处置**：

| 误报词 | 原因 | 处置 |
|--------|------|------|
| 天下×1,170 | entity_index收为地名，实为概念词 | 加入SKIP_WORDS |
| 太子×469 | entity_index收为人名别名，实为官职称谓 | 加入SKIP_WORDS |
| 子之×140 | 燕臣子之，"天子之德"中出现频繁 | 加入SKIP_WORDS |
| 至于×103 | entity_index收为官职，实为介词/连词 | 加入SKIP_WORDS |
| 八十/二十等×80类 | entity_index时间类含纯数字条目 | 正则过滤 |

基于词表匹配，扫描全文未标注区域：

**词表设计原则**：
- 典籍：多字书名（≥2字），不含"诗/书/易/礼/乐"单字（歧义高）
- 思想：概念性词汇（君子、王道、天命等），不含天下（歧义，改为Phase 2处理）
- 刑法：刑罚名称（弃市/腰斩/宫刑等）和司法行为
- 礼仪：祭祀/典礼/朝觐等具体仪式活动
