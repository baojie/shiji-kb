## 一、矛盾检测案例

### 1.1 东城斩首数量矛盾（007 vs 008）

- 007项羽本纪：项羽以28骑对数千汉骑，斩一将、一都尉、**数十百人**
- 008高祖本纪：灌婴追杀项羽于东城，斩首**八万**
- 矛盾类型：DETAIL（数量差异>100倍）
- 可能解释：008的八万是垓下→东城战役全程总数，007聚焦最后一战微观细节

详见 [labs/contradiction_007_008.md](../../labs/contradiction_007_008.md)

### 1.2 下宫之难日期矛盾（039 vs 043）

- 039晋世家：赵同赵括之死记于**前583年**（景公十七年）
- 043赵世家：下宫之难（杀赵同赵括）记于**前597年**（景公三年）
- 矛盾类型：DATE（差14年）+ SEQUENCE（因果链不同）
- 赵世家版本因果链更完整：灭族→程婴救孤→隐匿15年→前583年复位

详见 [labs/contradiction_039_043.md](../../labs/contradiction_039_043.md)

### 1.3 太子丹之死（086 vs 006）

- 086刺客列传：**前222年**，燕王喜斩太子丹献秦
- 006秦始皇本纪：**前226年**，秦军"得太子丹之首"
- 矛盾类型：DATE（差4年）+ ATTRIBUTION（燕王主动 vs 秦军获取）

详见 [labs/contradiction_086_006.md](../../labs/contradiction_086_006.md)

## 二、沉默证据案例

### 2.1 秦始皇无皇后

推理链：
```
本体规则：君主 → 应有 → 皇后/王后
事实检查：
  黄帝↔嫘祖 ✓   禹↔涂山氏 ✓   帝喾↔简狄 ✓
  文王↔太姒 ✓   刘邦↔吕后 ✓
  秦始皇↔？ ✗ ← 异常
```

深层追问：
- 秦始皇在位37年，《史记》完全不记载其皇后
- 扶苏、胡亥的母亲也完全无记载
- 《外戚世家》（049）专记皇后事迹，秦代部分空白
- 已有学者讨论（李开元《秦谜》）

### 2.2 刘邦年龄与子女矛盾

李开元《秦崩》中的推理链：
```
已知事实：
  - 刘邦生于前256年（或前247年，两说）
  - 刘肥是长子，为曹氏所生
  - 刘盈（惠帝）生于前210年，为吕后所生
  - 刘邦前209年起兵时，刘盈年仅1岁
  - 刘肥前201年封齐王，需有一定年龄

推理：
  若生于前256年 → 起兵时47岁，30多岁才有长子？
  若生于前247年 → 起兵时38岁，更合理但与"亭长"中年形象矛盾

结论：刘邦出生年份两说都有矛盾，是开放研究课题
```

## 三、函谷关归因矛盾（007 vs 008）

- 007项羽本纪：函谷关有**关守卒**，项羽令当阳君攻破
- 008高祖本纪：**沛公**守函谷关不纳诸侯，项羽使黥布攻破
- 矛盾类型：ATTRIBUTION
- 分析：007淡化刘邦主动性（只说有兵守关），008明确说沛公主动封关。视角导致归因差异。
