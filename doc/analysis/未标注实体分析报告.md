## 四、新实体标注语法设计

基于现有11种标注符号（`@ = $ % & ^ ~ * ! ? 🌿`），为3类高价值新实体设计语法：

### 4.1 符号方案

| 实体类型 | 建议符号 | 示例 | HTML class |
|----------|----------|------|------------|
| **典籍/书名** | `《...》` | `《春秋》` | `book` |
| **礼仪** | `+...+` | `+封禅+` | `ritual` |
| **刑法** | `#...#` | `#弃市#` | `legal` |
| **思想/概念** | `\|...\|` | `\|天命\|` | `concept` |

**符号选择理由：**
- `《》`：与中文书名号完全一致，已是自然语言惯例，零学习成本
- `+`：ASCII符号，未被占用，视觉轻量，易输入
- `#`：联想"法条编号"有助记忆；仅出现在行内，不与Markdown标题冲突
- `|`：作为行内标注不与Markdown表格冲突

### 4.2 HTML渲染样式

```css
.book    { color: #8B4513; font-style: italic; }  /* 典籍：棕色斜体 */
.ritual  { color: #9932CC; }                       /* 礼仪：紫色 */
.legal   { color: #DC143C; }                       /* 刑法：深红 */
.concept { color: #4682B4; font-style: italic; }   /* 思想：蓝色斜体 */
```

### 4.3 实际标注示例

```markdown
# 原文
孔子序《春秋》，明王道，讥天子，退诸侯，讨大夫，以达王事。

# 标注后
@孔子@序《春秋》，明|王道|，讥&周&天子，退诸侯，讨&周&大夫，以达王事。
```

```markdown

@商鞅@变法，设^连坐^，犯者#族诛#，轻罪#肉刑#，重者#弃市#。
```

## 附：边界说明

### 礼仪 vs 制度 的区别

- **制度 `^`**：制度性安排、政治体制（郡县制、封禅制度作为制度）
- **礼仪 `+`**：具体礼仪动作/仪式（封禅作为具体行为、祭祀、朝聘）

### 刑法 vs 事件描述 的区别

- **刑法 `#`**：法律术语本身（弃市、腰斩、族诛）
- **不标注**：描述性动词短语（"将其斩首"中"斩首"视情况而定）
