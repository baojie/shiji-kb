# 史记知识库 - 动词标注规范v3.3扩展说明

**发布时间**: 2026-03-19
**规范版本**: v3.3
**扩展规模**: 新增39个历史核心动词

---

## 一、扩展背景

### 1.1 发现问题

在v3.2重新标注过程中，发现381处旧格式单字动词标注无法迁移到新格式。经分析，这些动词虽不在v3.2核心词表（53个），但大多数是《史记》中高频使用的**历史事件核心动词**。

### 1.2 数据支撑

通过对381处待审核动词的语境分析，识别出：
- **军事动词**: 61处（略、胜、烧、夺、渡、平等）
- **刑罚动词**: 70处（鞭、醢、刭、黥、徙、系等）
- **政治动词**: 28处（畔、王、拜、迁、赐、召等）

合计159处高频历史动词标注，占待审核动词的41.7%。

### 1.3 扩展决策

基于以下原则决定扩展v3.3：
1. **频次优先**: 选择频次≥1的历史动词
2. **语义明确**: 必须是历史事件核心动作
3. **分类清晰**: 符合军事/刑罚/政治三大类定义
4. **实用价值**: 能显著提升事件提取和关系挖掘能力

```

⟦○封⟧〖@功臣〗                # 册封事件
```

**v3.3新增提取能力**：
```

临城自⟦◉刭⟧                    # 自杀事件
⟦◉系⟧〖@蒙毅〗於〖=代〗          # 囚禁事件

```

### 4.2 关系挖掘深度增强

**军事关系链**：
```
攻取 → 夺取 → 占据 → 守卫 → 失守 → 收复
伐 → 攻 → 破 → 灭 → 取 → 据 → 守 → 失 → 收
```

**刑罚演变链**：
```
罪行 → 逮捕 → 囚禁 → 审判 → 处刑 → 赦免/流放
反 → 执 → 囚/系 → 诛/杀 → 赦/放/徙
```

**政治斗争链**：
```
叛乱 → 讨伐 → 平定 → 处置 → 任命
畔 → 讨 → 平 → 诛/免 → 拜/封
```

### 4.3 统计分析能力扩展

**v3.2可统计**：
- 各国征伐频次（伐、攻、击）
- 处决方式分布（杀、诛、斩）
- 册封记录（封、立）

**v3.3新增统计**：
- 军事策略偏好（略地 vs 守城）
- 刑罚残酷度量化（极刑频次：鞭、醢、刭、黥等）
- 政治斗争模式（叛乱频次、任命规律、谗言现象）
- 历史人物命运分析（被囚、被流放、被赎罪）

1. **数据驱动**: 基于381处实际标注的语境分析，而非主观臆断

3. **实用导向**: 优先选择高频、核心、明确的历史动词
4. **文档完善**: 提供详细的释义、示例和应用价值说明

- 从基础框架（v3.0-v3.2）到实用体系（v3.3）
