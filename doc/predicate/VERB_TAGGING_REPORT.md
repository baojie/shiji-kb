## 标注规范执行情况

### ✅ 已执行的规范

1. **外层符号**：全部使用 ⟦⟧ (U+27E6/27E7)
2. **TYPE符号**：军事用◈，刑罚用◉
3. **消歧标注**：为常见歧义词添加了消歧说明
   - 取 → ⟦◈取|攻取⟧
   - 降 → ⟦◈降|投降⟧
   - 败 → ⟦◈败|战败⟧
   - 走 → ⟦◈走|逃亡⟧
   - 死 → ⟦◉死|处死⟧
   - 反 → ⟦◉反|谋反⟧
   - 亡 → ⟦◉亡|逃亡⟧

4. **动词与名词区分**：
   - 刑罚动词用 ⟦◉杀⟧
   - 刑罚制度名词用 〖[斩首〗（已存在，未修改）

### ⚠️ 已修正的问题

1. **删除误标**：删除了"执剑"、"执牺牲"等非刑罚动词的误标
2. **格式修正**：统一了消歧格式

## 特殊情况说明

### 1. 未标注的动词类型
- **日常动词**：行、坐、立、言、语等（按规范不标注）
- **政治动词**：立、封等（预留为⟦○verb⟧，本次未标注）
- **经济动词**：赐、贡等（预留为⟦◇verb⟧，本次未标注）

### 2. 需要注意的边界情况

#### "下"
- 已标注：攻下、拿下等军事用法
- 未标注：岐下（地名）、之下（方位）

#### "定"
- 已标注：平定、安定等军事用法
- 未标注：定都、定居等非军事用法

#### "执"
- 已标注：执拿、逮捕等刑罚用法
- 未标注：执剑、执政等非刑罚用法
