## 二、发现的系统性误标模式

### 2.1 "诸X"模式（61处）

**核心问题**："诸"（各位、众）是修饰词，应与后面的身份词一起标注

| 错误模式 | 修复后 | 数量 | 原因 |
|---------|--------|------|------|
| 诸〖;侯〗 | 〖#诸侯〗 | 28 | "诸侯"是身份，非官职 |
| 诸〖;侯王〗 | 〖#诸侯王〗 | 20 | "诸侯王"是身份群体 |
| 诸〖;将〗 | 〖#诸将〗 | 11 | "诸将"是身份群体 |
| 诸〖^侯服〗 | 〖#诸侯〗服 | 2 | "侯服"误切分 |

**典型示例**：
```
007_项羽本纪:
  修复前: 当是时，诸〖;将〗皆慴服
  修复后: 当是时，〖#诸将〗皆慴服

052_齐悼惠王世家:
  修复前: 使使喻〖=齐〗王及诸〖;侯〗
  修复后: 使使喻〖=齐〗王及〖#诸侯〗
```

## 三、方法论总结

### 3.1 检测方法演进

| 阶段 | 方法 | 发现数量 | 问题 |
|-----|------|---------|------|
| 第1次 | 简单模式匹配 | 2处 | 规则太保守 |
| 第2次（用户提醒） | 扩展"诸X"规则 | 59处 | 发现系统性误标 |
| 第3次 | 全面检查"群X"等 | 14处 | 发现更多边界模式 |

### 3.2 关键经验

1. **用户反馈至关重要**：初步只发现2处，经质疑后发现75处（37.5倍）
2. **系统性错误成批出现**："诸X"、"群X"等修饰词+身份词的组合普遍误标
3. **需要语义理解**：不能只看形式，要理解词汇的语义（如"入臣"是动词）
4. **分类标注混淆**：身份(`#`)、官职(`;`)、制度(`^`)经常混淆

### 3.3 边界错误的本质

**边界错误分为两大类**：

1. **切分错误**（Segmentation Error）
   - 修饰词被切分：诸〖;侯〗 → 〖#诸侯〗
   - 复合词被切分：诸〖^侯服〗 → 〖#诸侯〗服

2. **分类错误**（Classification Error）  
   - 身份误标为官职：诸〖;将〗 → 〖#诸将〗
   - 身份误标为动词：入〖#臣〗 → 入臣（动词）
