# 特殊读音词表 v3.0.2 更新说明

## 更新日期
2026-03-31

## 版本变更
v3.0.1 → v3.0.2

## 问题描述

**用户反馈**：
> "复稍度河南与中国界於故塞"这里"於"的发音还是错误的 wū

**根本原因分析**：

默认拼音库（pinyin-pro）将"於"字标注为 **wū（阴平）**，这是错误的。

- **正确读音**：於 = yú（阳平），介词，相当于"在"、"到"、"对于"
- **错误读音**：於 = wū（阴平），叹词"於呼"（即"呜呼"），极罕见

**影响范围**：
- 史记中"於"字出现约 2,400 次
- v3.0.1 仅覆盖"於是"（845次）
- 剩余约 1,555 次"於"被误标为 wū

**典型误标示例**：
- 於故塞 → wū gù sāi ❌（应为 yú gù sāi）
- 於秦 → wū qín ❌（应为 yú qín）
- 於天下 → wū tiān xià ❌（应为 yú tiān xià）

## 解决方案

### 添加单字"於"词条（例外情况）

```json
{
  "text": "於",
  "pinyin": ["yú"],
  "context": "介词",
  "note": "介词，相当于'在'、'到'、'对于'，读yú（阳平），不读wū。此为例外单字词条，因默认拼音库错误标注为wū"
}
```

### 设计权衡

**为什么违反v3.0铁律添加单字词条？**

v3.0核心原则：
> 禁止使用常见字单字词条，必须使用固定词组或正则表达式

**本次例外的正当理由**：

1. **默认拼音库错误**：
   - pinyin-pro 将"於"标注为 wū，而不是正确的 yú
   - 这是库本身的bug，不是我们的设计问题

2. **使用率极高**：
   - "於"在史记中99%+的情况都读 yú（介词）
   - 仅极少数"於呼"读 wū（叹词）

3. **固定词组无法覆盖**：
   - "於"字搭配太多样：於秦、於齐、於楚、於赵、於天、於故塞...
   - 需要添加上百个词组才能覆盖，不现实

4. **误标风险极低**：
   - 添加单字"於（yú）"不会导致误标
   - 因为 99%+ 的情况确实应该读 yú

### 与v3.0原则的对比

| 场景 | v3.0策略 | 於字情况 | 是否例外 |
|-----|---------|---------|---------|
| 常见字多音 | 禁止单字 | 符合 | - |
| 特殊音占比 | <80%禁止 | >99%应为yú | ✅ 例外 |
| 误标风险 | 单字高风险 | yú无误标风险 | ✅ 例外 |
| 默认库问题 | 假设正确 | 默认库错误 | ✅ 例外 |
| 可替代性 | 应用固定词组 | 词组数量太多 | ✅ 例外 |

**结论**：添加单字"於（yú）"是**合理的例外**，用于修正默认拼音库的系统性错误。

## 技术实现

### 匹配优先级

根据 heading-pinyin.js 的实现：

1. ✅ 正则模式匹配（无相关模式）
2. ✅ **字典精确匹配**（按长度降序）
   - "於是"（2字）→ [yú, shì] ← 优先匹配
   - "於"（1字）→ [yú] ← 兜底匹配
3. ❌ 默认读音（"於" wū）← 被覆盖

### 测试验证

**测试用例**：
```javascript
// v3.0.1（错误）
"於是" → [yú, shì]  ✅ 正确（词表匹配）
"於秦" → [wū, qín]  ❌ 错误（默认库）
"於故塞" → [wū, gù, sāi]  ❌ 错误（默认库）

// v3.0.2（正确）
"於是" → [yú, shì]  ✅ 正确（优先匹配"於是"词条）
"於秦" → [yú, qín]  ✅ 正确（匹配"於"单字词条）
"於故塞" → [yú, gù, sāi]  ✅ 正确（匹配"於"单字词条）
```

### Longest-Match-First 保证

由于词表按长度降序排列：
- "於是"（2字）会优先于"於"（1字）被匹配
- 不会出现冲突

## 设计原则更新

### 新增例外规则

**v3.0.2 补充条款**：

单字词条准入条件（v3.0）：
1. ✅ 专有名词（姓氏、罕见人名/地名）
2. ✅ 特殊读音使用率 >95%
3. ✅ 不存在常见读音误标风险

**新增例外条款（v3.0.2）**：
4. ✅ **默认拼音库系统性错误**
   - 当默认库将常见读音标注错误
   - 且正确读音使用率 >99%
   - 且固定词组无法经济地覆盖
   - 可添加单字词条修正默认库错误

**当前符合例外条款的字**：
- ✅ 於（yú）：默认库错误标注为wū，实际99%+应为yú
