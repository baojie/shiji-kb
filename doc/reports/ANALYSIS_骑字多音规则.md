# "骑"字多音规则说明

## 版本信息
- **创建日期**: 2026-03-31
- **最后更新**: 2026-03-31
- **相关Issue**: 用户反馈"兒能骑羊"中"骑"应读qí不是jì

## 问题背景

"骑"是典型的**上下文相关多音字**：
- **动词**：骑马、骑羊 → 读 **qí**（阳平）
- **名词/量词**：骑兵、百骑 → 读 **jì**（去声）

最初词表错误地添加了单字条目：
```json
{
  "text": "骑",
  "pinyin": ["jì"],
  "note": "跨骑义时读jì（去声）"  // ❌ 错误
}
```

这导致所有"骑"字都被标注为jì，包括动词用法。

## 完整读音规则

### 1. 动词用法 → qí（默认，无需标注）

| 模式 | 示例 | 说明 |
|------|------|------|
| 骑 + 宾语 | 骑羊、骑马、骑龙 | 动宾结构 |
| 能愿动词 + 骑 | 能骑、善骑、习骑 | 骑乘能力 |
| 骑射 | 善骑射、习骑射 | 动词短语 |
| 骑 + 之 | 骑之亡归 | 骑它 |

**实例**（《史记·匈奴列传》）：
> 兒能**骑**羊，引弓射鸟鼠；少长则射狐兔。

## 当前实现方案

### 已实现：词表匹配（v2.6）

特殊读音词表包含：
- **官职名**：5个（骑都尉、骑将军、车骑将军、骑常侍、骠骑将军）
- **军事名词**：10个（骑士、骑郎、骑将、胡骑、楚骑、汉骑等）
- **人名**：1个（骑劫）

**工作机制**：
1. 从左到右扫描文本
2. 按词长从长到短匹配词表
3. 匹配成功 → 应用特殊读音（jì）
4. 未匹配 → 使用pinyin-pro默认读音（qí）

### 未实现：量词正则匹配

**原因**：
- 量词组合太多（30+种），穷举不现实
- 当前实现只支持固定词组匹配，不支持正则

**影响**：
- "从百骑" → "从百qí" ❌（应该是jì）
- "将万骑" → "将万qí" ❌（应该是jì）
- "独与一骑归" → "独与一qí归" ❌（应该是jì）

### 临时解决方案

如果量词误读严重，可以添加高频量词到词表：

```json
[
  {"text": "十骑", "pinyin": ["shí", "jì"]},
  {"text": "百骑", "pinyin": ["bǎi", "jì"]},
  {"text": "千骑", "pinyin": ["qiān", "jì"]},
  {"text": "万骑", "pinyin": ["wàn", "jì"]},
  {"text": "数十骑", "pinyin": ["shù", "shí", "jì"]},
  {"text": "百余骑", "pinyin": ["bǎi", "yú", "jì"]},
  {"text": "数万骑", "pinyin": ["shù", "wàn", "jì"]}
]
```

但这只能覆盖部分场景，不是根本解决方案。

## 理想解决方案（未来增强）

### 方案：JavaScript正则匹配

在 `matchSpecialPronunciation()` 函数中添加正则规则：

```javascript
function matchSpecialPronunciation(text, startIndex) {
  // 1. 尝试匹配量词模式：数词 + [余/馀] + 骑
  const pattern = /^([0-9一二三四五六七八九十百千万亿兆数]+[余馀]?)骑/;
  const substr = text.substring(startIndex);
  const match = substr.match(pattern);
  
  if (match) {
    const numText = match[1];
    const numPinyin = pinyinFn(numText, { type: 'array' });
    return {
      text: match[0],
      pinyin: [...numPinyin, 'jì'],
      length: match[0].length,
      note: '量词用法，骑读jì'
    };
  }
  
  // 2. 继续匹配固定词表...
  for (let entry of specialPronunciations) {
    // ...
  }
}
```

**优点**：
- 一条规则覆盖所有量词组合
- 不需要穷举词表

**缺点**：
- 需要修改JavaScript代码
- 需要测试正则性能
