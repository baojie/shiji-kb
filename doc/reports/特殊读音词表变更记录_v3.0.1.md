## 问题描述

**用户反馈**：
> "於是"被标注成了 wū shì

**问题分析**：
- "於"字有两个读音：
  - **yú（阳平）**：介词，相当于"在"、"到"、"对于"（常用，占99%+）
  - **wū（阴平）**：叹词"於呼"（即"呜呼"），或姓氏（罕见）

- "於是"是史记中高频词组（出现845次），应读 **yú shì**（阳平+去声）
- 某些拼音库默认将"於"标注为 wū，导致"於是"被误标为 wū shì

## 解决方案

### 新增词条（1条）

```json
{
  "text": "於是",
  "pinyin": ["yú", "shì"],
  "context": "连词",
  "note": "于是，'於'读yú（阳平），不读wū"
}
```

## 设计考量

### 为什么不添加单字"於"？

遵循 v3.0 设计原则：

1. **误标风险评估**:
   - "於"字在史记中99%+的用法都是介词 yú
   - 叹词"於呼"（wū）极少出现
   - **不存在大规模误标风险**

2. **v3.0铁律**:
   > 禁止使用常见字单字词条，必须使用固定词组或正则表达式。例外：仅限姓氏专有名词。

3. **精准修复原则**:
   - 仅针对发现的误标问题（"於是" wū shì）添加词条
   - 避免过度添加，保持词表精简

### 为什么只添加"於是"，不添加其他"於X"？

**观察优先原则**：
- 当前仅发现"於是"被误标
- 其他高频词组（於秦、於齐等）暂未发现误标
- 如果后续发现误标，再逐个添加

**避免过度优化**：
- 过早添加可能不必要的词条
- 增加词表维护成本
- 可能引入新的问题
