## 三、内容性质分析

### "正义"的文献身份

"正义"即《史记正义》，唐代张守节撰，与裴駰《史记集解》、司马贞《史记索隐》合称"史记三家注"。

- **成书时间**: 唐玄宗时期（约8世纪）
- **性质**: 官方认可的史记注释文献
- **特点**: 侧重地理、典章制度、名物训诂，此处为医学解剖数据注释

### 内容来源考证

此段"正义"内容与《黄帝内经·灵枢》《难经》等医学典籍数据高度吻合：
- 五脏重量数据（肝重四斤四两、心重十二两等）
- 经脉循行长度（手三阳五尺、足三阴六尺五寸等）
- 五脏与九窍对应关系

**文献价值**: 反映唐代对古代医学文献的理解和保存，具有医学史研究价值。

## 四、校勘结论

### 核心判断

**"正义"内容应当保留**，理由如下：

1. **文献真实性**: 四个独立来源均包含此内容，证明其为《史记》版本的固有组成部分
2. **历史价值**: 属于"史记三家注"之一《史记正义》的内容，是传统《史记》读本的标准配置
3. **版本一致性**: 从原始txt到维基文库各版本，均完整保留，无删节先例
4. **学术价值**: 对研究唐代医学知识、《史记》注释史有重要参考价值

### 内容归属建议

虽应保留，但需明确标识其性质：

**当前标注文件处理**:
```markdown
## 赞诗与正义

[50] 上池祕术，〖@长桑〗所传。
[51] 始候〖@赵简〗，知梦〖?钧天〗。
...
[54] 效验多状，式具于篇。
[55] 〖_正义〗胃大〖$一尺〗〖$五寸〗...（解剖数据）
[56] 手三阳之脉...（经脉数据）
[57] 肺气通於鼻...（五脏九窍对应）
```

**问题**:
- 章节标题"## 赞诗与正义"混淆了两个不同性质的内容
- [50]-[54]是"索隐述赞"（赞诗）
- [55]-[57]是"正义"（注释）

### 推荐处理方案

#### 方案A：拆分章节（推荐）

```markdown
## 赞诗

[50] 上池祕术，〖@长桑〗所传。
[51] 始候〖@赵简〗，知梦〖?钧天〗。
[52] 言占〖'虢〗嗣，尸蹶起焉。
[53] 〖;仓公〗〖[赎罪〗，〖@阳庆〗推贤。
[54] 效验多状，式具于篇。

## 正义

[55] 胃大〖$一尺〗〖$五寸〗，径〖$五寸〗...（去除〖_正义〗标注，因章节标题已说明）
[56] 手三阳之脉...
[57] 肺气通於鼻...
```

**理由**: 符合维基文库版本的结构划分，清晰区分"索隐述赞"与"正义注"。

#### 方案B：保持现状，优化标注

```markdown
## 赞诗与正义

### 索隐述赞
[50] 上池祕术...
...
[54] 效验多状，式具于篇。

### 史记正义
```
