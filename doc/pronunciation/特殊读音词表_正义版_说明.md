# 特殊读音词表（史记正义版）说明文档

## 概述

本词表从《史记集解三家注索隐正义》中提取特殊读音注释，重点收录古今读音差异明显、现代读者容易误读的地名、人名、官职和常用多音字。

**文件位置**: `docs/data/special-pronunciation-zhengyi.json`

## 数据来源

- **底本**: 维基文库《史记集解三家注索隐正义》
- **提取对象**: 《史记正义》中的读音注释
- **验证方法**: 所有词条均已验证在《史记》原文中确实出现

## 典型词条示例

### 1. 地名类

#### 阏与 (yān yǔ)
- **注释**: 赵奢破秦军处
- **出处**: 《史记·廉颇蔺相如列传》
- **正义**: "音焉与"
- **说明**: "阏"读yān，"与"读yǔ，今山西和顺县

#### 亢父 (gāng fǔ)
- **注释**: 在今山东济宁
- **出处**: 多处出现（31次）
- **说明**: "亢"读gāng，不读kàng

#### 朝鲜 (cháo xiǎn)
- **注释**: 古国名
- **正义**: "朝音潮"（如"潮汐"之潮）
- **说明**: 指箕子朝鲜、卫满朝鲜等古国

### 2. 官职类

#### 仆射 (pú yè)
- **说明**: 汉代官名，"仆"读pú，"射"读yè
- **职责**: 仆，主也；射，仆射也

#### 大行 (dà háng)
- **说明**: 掌宾客之官，"行"读háng
- **出处**: 《史记·张释之冯唐列传》等

### 3. 常用多音字

这些字在现代汉语中仍是多音字，但在古文中的具体用法容易混淆：

| 字 | 读音 | 用法 | 示例 |
|---|------|------|------|
| 王 | wàng | 称王、为王（动词） | "秦王欲以五百里地易安陵" |
| 将 | jiàng | 将军、大将（名词） | "廉颇者，赵之良将也" |
| 相 | xiàng | 宰相、丞相（名词） | "秦王坐章台见相如" |
| 行 | háng | 行列、道路（名词） | "大行令奏之" |
| 骑 | jì | 跨骑（动词） | "跨骑取弓" |
| 中 | zhòng | 射中、中试（动词） | "中石没镞" |
| 长 | zhǎng | 生长、年长（动词/形容词） | "长者加以金银" |
| 为 | wèi | 作为、成为（动词） | "为天下笑" |

## 与原有词表的区别

### 原有词表 (special-pronunciation.json)
- **来源**: 人工整理
- **重点**: 史记中确实出现的特殊人名、地名、族名
- **词条数**: 24个
- **示例**: 冒顿(mò dú)、单于(chán yú)、郦食其(lì yì jī)

### 正义版词表 (special-pronunciation-zhengyi.json)
- **来源**: 从《史记正义》系统提取
- **重点**: 正义注音标注的古今读音差异
- **词条数**: 30个
- **示例**: 阏与(è yǔ)、亢父(gāng fǔ)、仆射(pú yè)

### 互补性
两个词表内容有少量重叠（如"员"、"区"），但侧重点不同：
- 原词表侧重**专有名词**（人名、族名）
- 正义版侧重**易读错字词**（地名、官职、多音字）

### 3. 文本标注

- 提供准确注音

## 提取方法

从《史记正义》中提取读音注释的4种模式：

1. **反切注音**: "X音甲乙反"（如"朝音潮"）
2. **直接注音**: "X音Y"（如"骊音力知"）
3. **通假注音1**: "X读曰Y"（如"驯读曰训"）
4. **通假注音2**: "X读为Y"（如"施读为移"）

详细提取结果见: `doc/pronunciation/特殊读音注释提取结果.md` (694行)

## 验证标准

所有收录词条必须满足：
1. ✓ 在《史记》原文中确实出现
2. ✓ 有《史记正义》明确注音
3. ✓ 古今读音有显著差异或现代易误读
4. ✓ 有实用价值（常见词、重要地名人名）

## 维护说明

### 添加新词条

1. 在《史记正义》中找到明确注音
2. 使用 `grep` 验证在《史记》原文中出现
3. 按格式添加到 JSON 文件
4. 更新统计信息

### 格式规范

```json
{
  "text": "词条原文",
  "pinyin": ["拼", "音"],
  "context": "地名/人名/官职/动词/名词等",
  "note": "详细说明，引用正义原文"
}
```
