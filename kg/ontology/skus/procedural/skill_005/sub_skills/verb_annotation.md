# 动词标注 Agent Skill (Verb Annotation)

**SKU**: skill_005_verb_annotation
**分类**: 文本标注 / 知识库构建
**版本**: v3.0
**创建日期**: 2026-03-19

## 技能概述

### 目标

为《史记》文本中的动词进行语义标注，将历史叙事中的核心行为动作标记为结构化实体，用于：
- 事件提取与关系挖掘
- 历史动作模式分析
- 语义检索与知识图谱构建

### 适用场景

1. 为新章节添加动词标注
2. 验证和修复现有动词标注
3. 扩展新的动词类型（政治/经济动词）
4. 批量迁移和规范化动词标注

## 核心知识

### 1. 动词标注体系（v3.0）

**外层符号**: `⟦⟧` (U+27E6/27E7 数学双方括号)

**动词类型**:

| 类型 | 符号 | 格式 | 示例 | 核心词数 |
|------|------|------|------|---------|
| 军事动词 | ◈ | `⟦◈动词⟧` | `⟦◈伐⟧` `⟦◈攻⟧` | 17个 |
| 刑罚动词 | ◉ | `⟦◉动词⟧` | `⟦◉杀⟧` `⟦◉诛⟧` | 15个 |
| 政治动词 | ○ | `⟦○动词⟧` | `⟦○立⟧` `⟦○封⟧` | 预留 |
| 经济动词 | ◇ | `⟦◇动词⟧` | `⟦◇赐⟧` `⟦◇贡⟧` | 预留 |

**消歧语法**: `⟦TYPE动词|消歧说明⟧`

示例：
- `⟦◈伐|征伐⟧` - 军事征伐
- `⟦◈走|逃亡⟧` - 逃走（区分于行走）
- `⟦◉死|处死⟧` - 判处死刑（区分于自然死亡）

### 2. 军事动词核心词表（17个）

**进攻类**（5个）：
```
伐、攻、击、袭、侵
```

**交战类**（7个）：
```
战、破、败、灭、围、下、拔
```

**机动类**（5个）：
```
救、取、定、降、走
```

### 3. 刑罚动词核心词表（15个）

**处决类**（7个）：
```
杀、诛、斩、弑、族、戮、刺
```

**处罚类**（5个）：
```
废、囚、执、笞、刑
```

**其他类**（3个）：
```
反、亡、死
```

### 4. 刑罚动词 vs 刑罚制度名词

**关键区分**：

| 类别 | 标注格式 | 特征 | 示例 |
|------|---------|------|------|
| **刑罚动词** | `⟦◉verb⟧` | 单字动词，表示动作 | `⟦◉杀⟧` `⟦◉诛⟧` `⟦◉斩⟧` |
| **刑罚制度名词** | `〖[legal〗` | 2字+名词短语，表示制度 | `〖[斩首〗` `〖[腰斩〗` `〖[夷三族〗` |

**示例对比**：
```markdown
⟦◉杀⟧〖;令尹〗〖@子西〗，〖[斩首〗八万
 ↑动词：杀害           ↑名词：斩首刑

⟦◉斩⟧〖@颜良〗，〖[弃市〗
 ↑动词：斩杀   ↑名词：弃市刑
```

## 标注规则

### 规则1：识别原则

✅ **应该标注的动词**：
- 军事行动：伐/攻/击/战/破/败/灭等
- 刑罚处决：杀/诛/斩/弑/族等
- 明确的及物动词，有主语和宾语
- 可用于事件提取的关键动词

❌ **不应标注的动词**：
- 日常动词：行/坐/立/言/语
- 助动词：能/可/是/为
- 趋向动词：来/去/上/下（除非军事机动）
- 一般性动作：见/闻/知/思

### 规则2：消歧原则

**何时需要消歧**：
1. 同形异义（一字多义）
2. 跨类歧义（同字可属多类）
3. 与名词混淆

**示例**：
```markdown
⟦◈取|攻取⟧〖=城〗    # 军事攻取
取而代之              # 一般之取（不标注）

⟦◈走|逃亡⟧奔〖'郑〗  # 军事逃亡
走马观花              # 行走（不标注）

⟦◈围⟧〖=邯郸〗       # 动词：包围
解〖=邯郸〗之围       # 名词：包围圈（不标注）
```

### 规则3：标注边界

**单字优先**：
```markdown
⟦◈伐⟧ ⟦◈攻⟧ ⟦◈击⟧ ⟦◉杀⟧ ⟦◉诛⟧
```

**复合动词拆分**（推荐）：
```markdown
⟦◈击⟧⟦◈破⟧〖'楚〗    # 推荐
⟦◈击破⟧〖'楚〗        # 可选（谨慎使用）
```

### 规则4：语法约束

✅ **正确格式**：
```markdown
⟦◈伐⟧〖'随〗                    # 基本格式
⟦◈击|攻击⟧〖'楚〗于〖=城濮〗    # 消歧格式
〖'吴〗⟦◈伐⟧⟦◈败⟧〖@子常〗    # 连续动词
```

❌ **错误格式**：
```markdown
⟦⟦◈伐⟧⟧           # 禁止嵌套
⟦◈〗                # 禁止空标注
⟦◈伐|⟧             # 消歧说明不能为空
〖◈伐〗             # 错误的外层符号
⟦@伐⟧              # 错误的TYPE符号
```

## 工作流程

### 阶段1：文本分析

1. **阅读章节内容**，理解历史叙事
2. **识别关键事件**：战争、刑罚、政治、经济等
3. **标记候选动词**：圈出可能需要标注的动词
4. **确认上下文**：判断动词的具体含义

### 阶段2：动词分类

1. **军事判断**：是否涉及战争、军事行动？
   - 国与国冲突 → 军事动词
   - 军队机动 → 军事动词

2. **刑罚判断**：是否涉及处决、惩罚？
   - 政治处决 → 刑罚动词
   - 刑罚执行 → 刑罚动词
   - 刑罚制度名称 → 使用 `〖[legal〗`

3. **消歧判断**：是否需要消歧说明？
   - 有歧义 → 添加消歧
   - 含义明确 → 不需要

### 阶段3：标注执行

1. **套用格式**：
   ```markdown
   原文：楚伐随
   标注：〖'楚〗⟦◈伐⟧〖'随〗
   ```

2. **添加消歧**（如需要）：
   ```markdown
   原文：子常走奔郑
   标注：〖@子常〗⟦◈走|逃亡⟧奔〖'郑〗
   ```

3. **保持一致性**：同一动词在相似上下文中应使用相同标注

### 阶段4：质量检查

1. **运行验证脚本**：
   ```bash
   python kg/entities/scripts/validate_verb_tagging.py --chapter 040
   ```

2. **检查输出**：
   - 是否有未分类的动词？
   - 是否有格式错误？
   - 是否有遗漏的动词？

3. **修复问题**并重新验证

## 核心脚本

### 1. 查询脚本

**文件**: `kg/entities/scripts/query_verbs_by_type.py`

**功能**：
- 统计各类动词频次
- 按章节分析动词分布
- 导出动词-实体关系

**用法**：
```bash
# 统计军事动词
python kg/entities/scripts/query_verbs_by_type.py --type military

# 统计刑罚动词
python kg/entities/scripts/query_verbs_by_type.py --type penalty

# 分析指定章节
python kg/entities/scripts/query_verbs_by_type.py --chapter 040

# 生成完整报告
python kg/entities/scripts/query_verbs_by_type.py --all --report
```

### 2. 验证脚本

**文件**: `kg/entities/scripts/validate_verb_tagging.py`

**功能**：
- 检查未分类的标注
- 发现可能的标注错误
- 生成质量报告

**用法**：
```bash
# 验证所有章节
python kg/entities/scripts/validate_verb_tagging.py

# 验证指定章节
python kg/entities/scripts/validate_verb_tagging.py --chapter 040

# 严格模式
python kg/entities/scripts/validate_verb_tagging.py --strict

# 生成报告
python kg/entities/scripts/validate_verb_tagging.py --report validation_report.txt
```

### 3. 迁移脚本

**文件**: `kg/entities/scripts/migrate_verb_tags.py`

**功能**：
- 将旧格式迁移到新格式
- 自动分类军事/刑罚动词
- 生成迁移报告

**用法**：
```bash
# 预览迁移（不修改文件）
python kg/entities/scripts/migrate_verb_tags.py --chapter 040 --dry-run

# 迁移单个章节
python kg/entities/scripts/migrate_verb_tags.py --chapter 040 --backup

# 迁移所有章节
python kg/entities/scripts/migrate_verb_tags.py --all --backup

# 生成迁移报告
python kg/entities/scripts/migrate_verb_tags.py --all --report migration_report.md
```

### 系统提示（System Prompt）

```markdown
你是一个专门为《史记》文本进行动词标注的AI助手。你的任务是识别、分类和标注历史叙事中的核心动词，构建结构化的知识图谱。

**核心能力**：
1. 识别军事动词（伐/攻/击/战/破/败/灭等17个核心词）
2. 识别刑罚动词（杀/诛/斩/弑/族等15个核心词）
3. 区分动词与名词（如：⟦◉杀⟧ vs 〖[斩首〗）
4. 判断是否需要消歧（如：⟦◈走|逃亡⟧）
5. 保持标注一致性和准确性

**标注格式**：
- 军事动词：`⟦◈动词⟧` 或 `⟦◈动词|消歧说明⟧`
- 刑罚动词：`⟦◉动词⟧` 或 `⟦◉动词|消歧说明⟧`
- 外层符号：⟦⟧ (U+27E6/27E7 数学双方括号)

1. 只标注军事和刑罚动词（政治/经济动词暂时预留）
2. 单字动词优先，复合动词需谨慎
3. 有歧义时必须添加消歧说明
4. 刑罚制度名词使用 〖[legal〗，不要用 ⟦◉⟧
5. 日常动词不标注
```

## 常见错误及修正

### 错误1：混淆动词与名词

❌ **错误**：
```markdown
⟦◉斩首⟧八万
```

✅ **正确**：
```markdown
〖[斩首〗八万
```

**解释**：「斩首」是刑罚制度名词（2字+），应使用 `〖[legal〗` 而非动词标注。

### 错误2：过度标注日常动词

❌ **错误**：
```markdown
〖@项羽〗⟦行⟧至〖=鸿门〗
```

✅ **正确**：
```markdown
〖@项羽〗行至〖=鸿门〗
```

**解释**：「行」是日常动词，不需要标注。

### 错误3：缺少必要的消歧

❌ **错误**：
```markdown
〖@子常〗⟦◈走⟧奔〖'郑〗
```

✅ **正确**：
```markdown
〖@子常〗⟦◈走|逃亡⟧奔〖'郑〗
```

**解释**：「走」有多义（逃亡/行走），需要消歧说明。

### 错误4：错误的TYPE符号

❌ **错误**：
```markdown
⟦@伐⟧〖'随〗
```

✅ **正确**：
```markdown
⟦◈伐⟧〖'随〗
```

**解释**：军事动词应使用 `◈` 而非 `@`（人名符号）。

### 错误5：使用错误的外层符号

❌ **错误**：
```markdown
〖◈伐〗〖'随〗
```

✅ **正确**：
```markdown
⟦◈伐⟧〖'随〗
```

**解释**：动词应使用 `⟦⟧` (数学双方括号)，名词使用 `〖〗` (白色龟甲括号)。

## 性能指标

### 质量目标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| **准确率** | ≥95% | 正确标注的动词 / 总动词数 |
| **召回率** | ≥90% | 已标注的动词 / 应标注的动词 |
| **一致性** | ≥98% | 同类动词使用相同标注的比例 |
| **格式正确率** | 100% | 符合语法规范的标注比例 |
