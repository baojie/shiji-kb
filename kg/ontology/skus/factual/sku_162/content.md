## 六卿专权与三家分晋

### 六卿形成
- 晋作三军，后作三行，又作六军
- 六卿逐渐掌握实权，公室日衰

### 关键事件
| 时间 | 事件 |
|------|------|
| 景公十七年 | 诛赵同、赵括，韩厥复赵武为赵后 |
| 厉公八年 | 栾书、中行偃弑厉公 |
| 平公十四年 | 吴延陵季子预言晋国政归三家 |
| 顷公十二年 | 六卿灭祁傒、叔乡族，分其邑为十县 |
| 定公二十二年 | 晋败范、中行氏，二子奔齐 |
| 出公十七年 | 知伯与赵、韩、魏分范、中行地 |
| 哀公四年 | 赵襄子、韩康子、魏桓子共杀知伯 |

### 三家分晋
- 烈公十九年：周威烈王赐赵、韩、魏皆命为诸侯
- 静公二年：魏武侯、韩哀侯、赵敬侯灭晋后而三分其地
- 静公迁为家人，晋绝不祀
