## 三、SKU创建标准与模板

### 3.1 Facts创建标准

#### JSON格式（适用于结构化数据）
```json
{
  "name": "fact-name-in-english",
  "description": "事实描述（中文）",
  "data": {
    "分类1": {
      "子项1": "内容",
      "子项2": "内容"
    },
    "分类2": {
      ...
    }
  }
}
```

**适用场景**：
- 世系家谱
- 时间线
- 人物列表
- 地理数据
- 统计数据

#### Markdown格式（适用于叙事性内容）
```markdown
---
name: fact-name-in-english
description: 事实描述（中文）
---

# 标题

## 一级子标题

### 二级子标题

内容...
```

**适用场景**：
- 事件叙述
- 历史背景
- 政策措施
- 言论集萃
- 历史影响

### 3.3 Eureka创建标准

```markdown

**洞察统计**：N条
**来源章节**：章节名
**主题分布**：
- 主题1（X条）
- 主题2（X条）

**版本**：v1.0
**创建日期**：YYYY-MM-DD
```

**洞察质量标准**：
- 非显而易见（不是常识）

- 有反直觉的成分更佳
