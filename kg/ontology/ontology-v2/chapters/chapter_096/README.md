## 章节主题

### 核心主题

1. **丞相的职责与独立性**：汉初丞相如何在皇权与官僚体系之间维持自身立场
2. **廉洁的代价**：申屠嘉的刚直，最终以呕血而死告终——廉洁守节者的结局
3. **知识权威与政治权力**：张苍以知识权威压制公孙臣，最终被事实纠正
4. **进言的艺术**：周昌直谏 vs 赵尧智谋，两种进言方式的效果对比

### 关键人物

| 人物 | 身份 | 核心事件 |
|------|------|----------|
| **张苍** | 阳武人→北平侯→丞相 | 因外貌获救，精通律历制汉历，百余岁饮乳长寿 |
| **申屠嘉** | 梁人→御史大夫→丞相 | 持礼法处置邓通，欲诛晁错，呕血而死 |
| **周昌** | 沛人→御史大夫→赵相 | "期期不奉诏"直谏废太子，被调任赵相护赵王 |
| **赵尧** | 符玺御史→御史大夫 | 察情进言，为赵王安排周昌为相，升任御史大夫 |
| **邓通** | 汉文帝宠臣 | 朝廷失礼，被申屠嘉当众羞辱，叩首流血 |

## 太史公评价

> "张苍文学律历，为汉名相，而绌贾生、公孙臣等言正朔服色事而不遵……申屠嘉可谓刚毅守节矣，然无术学，殆与萧何、曹参、陈平异矣。"

司马迁认为：张苍有学问有能力，但在历法判断上犯了错；申屠嘉刚毅守节，但缺乏战略智慧，与萧曹陈平有本质差距。
