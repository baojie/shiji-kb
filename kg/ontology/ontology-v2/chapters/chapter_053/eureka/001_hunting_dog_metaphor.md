# Eureka: 功狗与功人——价值评估的范式转换

**Eureka ID**: 053_E001
**洞察类别**: 价值评估哲学
**来源**: 萧相国世家 [7]

## 原文引述

> 高帝曰："诸君知猎乎？"曰："知之。""知猎狗乎？"曰："知之。"高帝曰："夫猎，追杀兽兔者狗也，而发踪指示兽处者人也。今诸君徒能得走兽耳，功狗也。至如萧何，发踪指示，功人也。"

## 核心洞察

### 价值评估的两个维度

**显性价值（功狗）**:
- 直接可见的成果
- 容易量化：攻城数、杀敌数、负伤数
- 立即产生效果
- 例：曹参"身被七十创，攻城略地"

**隐性价值（功人）**:
- 战略规划与资源保障
- 难以量化：运筹、转漕、补充
- 持续产生效果
- 例：萧何"从关中遣军补其处，数万众会上之乏绝者数矣"

### 范式转换的关键

**从线性思维到系统思维**:

传统观点（群臣）:
```
战斗勇敢 → 杀敌多 → 功劳大
```

高祖观点（系统）:
```
战略规划 → 持续支持 → 战斗胜利 → 系统成功
     ↑______________|
```

## 现代应用

### 1. 企业价值评估

**错误案例：只看销售额**
```
销售冠军（功狗）：签单最多，提成最高
供应链（功人）：保证交付，无人关注

结果：产品质量问题频发，客户流失
```

**正确案例：系统评估**
```
产品经理（功人）：定义需求，规划路线图
研发团队（功狗）：实现功能，修复bug
运营团队（功人）：用户反馈，迭代优化

价值分配：考虑全链条贡献
```

### 2. 政府绩效评估

**传统模式**:
- GDP增长（功狗指标）
- 项目数量（功狗指标）
- 招商引资金额（功狗指标）

**系统评估**:
- 制度建设（功人工作）
- 营商环境（功人工作）
- 可持续发展能力（功人视角）

### 3. 学术评价

**旧体系：计数文化**
```
论文数量 → 影响因子 → 学术地位
（功狗思维）
```

**新思维：质量与影响**
```
问题定义 → 方法创新 → 长期影响
（功人贡献）
```

## 深层哲学

### 工具理性 vs 价值理性

**工具理性（功狗逻辑）**:
- 目标：追杀兽兔
- 评价：速度、咬合力
- 可替代性：高

**价值理性（功人逻辑）**:
- 目标：发踪指示
- 评价：判断力、战略眼光
- 可替代性：低

### 短期 vs 长期

**短期贡献（功狗）**:
- 立竿见影
- 容易衡量
- 群众认可度高

**长期贡献（功人）**:
- 持续产生价值
- "万世之功"
- 需要深度理解

## 历史验证

### 萧何的"万世之功"

**鄂君的论证**:
> "夫汉与楚相守荥阳数年，军无见粮，萧何转漕关中，给食不乏...此万世之功也。今虽亡曹参等百数，何缺於汉？汉得之不必待以全。柰何欲以一旦之功而加万世之功哉！"

**关键论点**:
1. **不可替代性**: "虽亡曹参等百数，何缺於汉"
2. **系统重要性**: "汉得之不必待以全"
3. **时间维度**: "一旦之功" vs "万世之功"

### 后世应验

**后代对萧何的尊崇**:
- "後嗣以罪失侯者四世，绝，天子辄复求何後，封续酂侯，功臣莫得比焉"
- 即使后代犯罪，皇帝仍寻找萧何后裔续封
- 体现了"万世之功"的真实性

## 认知陷阱

### 陷阱1：可见性偏误

**心理机制**:
- 人类大脑偏好可见、直接的因果关系
- 抽象的战略规划难以感知

**克服方法**:
- 建立系统思维框架
- 追踪长期影响
- 重视过程指标

### 陷阱2：归因错误

**群臣的逻辑**:
```
我们冲锋陷阵 → 敌人被击败 → 功劳归我们
```

**实际因果链**:
```
萧何保障后勤 → 军队有粮有兵 → 将士能战斗 → 敌人被击败
```

### 陷阱3：即时满足

**功狗的诱惑**:
- 立即获得奖赏
- 立即得到认可
- 立即看到成果

**功人的挑战**:
- 延迟满足
- 价值被低估
- 需要远见者认可

## 现代组织启示

### 绩效管理的重新设计

**传统KPI（功狗导向）**:
```
销售额、利润率、客户数
→ 鼓励短期行为
→ 忽视长期建设
```

**系统OKR（功人导向）**:
```
O: 建立行业领先的供应链体系
KR1: 交付准时率95%
KR2: 供应商满意度90%
KR3: 供应链成本降低15%
```

### 人才激励的平衡

**显性激励（功狗）**:
- 即时奖金
- 公开表彰
- 快速晋升

**隐性激励（功人）**:
- 战略项目机会
- 长期股权激励
- 组织信任与授权

## 相关理论

### 管理学理论

**彼得·德鲁克**:
> "效率是以正确的方式做事，而效能则是做正确的事。"
- 效率 = 功狗
- 效能 = 功人

**吉姆·柯林斯（Good to Great）**:
- Level 5领导力：系统建设者（功人）
- 执行力：战术执行者（功狗）

### 经济学理论

**交易成本理论（科斯）**:
- 制度建设降低交易成本（功人）
- 单次交易执行（功狗）

**可持续竞争优势理论**:
- 核心能力（功人构建）
- 战术优势（功狗获得）

## 终极洞察

### 组织成功的秘密

**表面**:
- 英勇的战士（功狗）
- 辉煌的胜利（结果）

**本质**:
- 卓越的后勤（功人）
- 正确的战略（系统）
- 持续的支持（保障）

### 个人成长的路径

**初级阶段**:
- 做好"功狗"：证明执行力
- 获得认可和信任

**中级阶段**:
- 转型"功人"：开始规划和协调
- 扩大影响范围

**高级阶段**:
- 成为"大功人"：系统建设者
- 创造"万世之功"

## 现代企业案例

### 成功案例：苹果公司

**功狗层面**:
- iPhone销售团队：创造销售纪录
- 工程师：开发新功能

**功人层面**:
- 乔布斯：定义产品哲学
- 蒂姆·库克：建立供应链体系
- 设计团队：创造设计语言

**价值分配**:
- 股权激励向"功人"倾斜
- 长期持有者获得最大收益

### 失败案例：柯达公司

**过度关注功狗**:
- 胶卷销售业绩
- 传统冲印店数量

**忽视功人**:
- 数字化转型战略
- 技术趋势预判
- 组织能力重构

**结果**:
- 短期业绩优秀，长期破产
