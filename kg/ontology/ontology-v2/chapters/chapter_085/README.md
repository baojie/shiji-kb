## 章节主题

### 核心主题

1. **政治投资**："奇货可居"——商人眼光在政治领域的应用与边界
2. **权力边界**：功高震主的陷阱，吕不韦vs范蠡的对比
3. **商人从政**：商业逻辑与权力逻辑的本质冲突
4. **知识资本**：《吕氏春秋》与悬金城门，知识权力的政治化

### 关键人物

| 人物 | 身份 | 作用 |
|------|------|------|
| **吕不韦** | 主角，阳翟商人→秦国相国 | 奇货可居的发现者与实施者，最终失败的政治投资者 |
| **子楚（庄襄王）** | 秦公子，后为秦王 | 吕不韦投资的对象，知恩图报，三年后去世 |
| **嬴政（秦始皇）** | 庄襄王之子，秦二世主 | 最终清算吕不韦的人 |
| **华阳夫人** | 安国君宠姬 | 被吕不韦说服，认子楚为养子 |
| **嫪毐** | 假宦官 | 吕不韦引入宫中，最终成为导致吕不韦失势的因素之一 |
| **赵姬（太后）** | 吕不韦之妾→子楚王后 | 连接吕不韦与秦王室的关键纽带 |
