## 章节概要

**主要人物**：
- **韩王信**：韩国王族后裔，封韩王，驻马邑防匈奴，被猜疑后投降匈奴，引发白登之围，后战死于参合
- **卢绾**：与刘邦同里同日生，最亲密的发小，封燕王，高祖晚年因恐惧吕后清洗异姓王而通谋匈奴，高祖崩后逃入匈奴，老死异乡
- **陈豨**：代相，通连匈奴和燕，最终被樊哙斩于灵丘

**核心主题**：
- "亲信背叛"的两种模式（功能性信任 vs 情感性信任）
- "内见疑强大，外倚蛮貊"的政治规律
- 白登之围的绝境外交
