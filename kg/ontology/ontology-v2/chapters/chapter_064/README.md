## 章节主题

### 1. 核心叙事

司马穰苴列传是一个完整的悲剧弧线：

**上升**：
- 晏婴荐举（外部发现）
- 斩庄贾立威（建立权威）
- 退敌收地（功业成就）
- 封大司马（权力顶峰）

**下降**：
- 田氏益尊 → 旧贵族嫉妒
- 被谗于景公 → 被罢免
- 发疾而死 → 悲剧结局

**余波**：
- 田乞、田豹怨旧贵族 → 田常弑简公、灭高国族
- 田和大放穰苴之法 → 诸侯朝齐
- 思想遗泽超越个人命运

### 2. 核心人物关系

```
齐景公（君主）
  ├── 晏婴（荐举穰苴）    ←→ 穰苴（被荐）
  ├── 庄贾（监军宠臣）    ──→ 被穰苴斩
  └── 鲍氏/高/国（旧贵族）──→ 谗毁穰苴

田氏系谱：
田完 → ... → 穰苴 → 田乞/田豹 → 田常 → 田和（齐威王）
```

### 3. 《司马兵法》线索

- 穰苴生前：以法治军，恩威并施
- 穰苴死后：被废黜，《司马兵法》散佚
- 齐威王时：整理古司马兵法，附穰苴入书
- 太史公评：闳廓深远，穰苴区区行师，未及揖让

## 跨领域洞察（Eureka）

本章提取出5条跨领域洞察：

1. **权威来自规则而非身份**：穰苴以军法而非个人魅力建立权威
2. **功高震主的必然下场**：成功本身成为失败的原因
3. **制度执行中的首违必惩原则**：第一次执法必须彻底
4. **兵法的价值在于思想而非具体战术**：原则比操作手册更持久
5. **临时团队的快速动员**：危机中通过规则+信任快速建立战斗力

## 核心名言

1. **将受命之日则忘其家，临军约束则忘其亲，援枹鼓之急则忘其身。**
   ——穰苴训庄贾，论将帅的使命担当

2. **将在军，君令有所不受。**
   ——穰苴拒景公赦令，论军事权威的独立性

3. **穰苴虽田氏庶孽，然其人文能附众，武能威敌。**
   ——晏婴的推荐语，简练概括穰苴能力

4. **余读《司马兵法》，闳廓深远，虽三代征伐，未能竟其义。**
   ——太史公对兵法的评价
