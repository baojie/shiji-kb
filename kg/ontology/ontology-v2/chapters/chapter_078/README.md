# Chapter 078: 春申君列传

## 章节概述

《春申君列传》记载战国四公子之一黄歇（春申君）的一生：从游学博闻的辩士，到说服秦昭王的外交家，再到舍身护送太子的忠臣，最终成为楚国权倾一时的相国。但在权力巅峰时，却因贪欲献女楚王、不听朱英预警，被门客李园伏杀灭族。

太史公评："初智之明，后制于李园。当断不断，反受其乱。"

## 核心人物

### 黄歇（春申君）
- **出身**：楚人，游学博闻
- **早期成就**：
  - 说秦昭王：阻止秦伐楚
  - 护送太子：以死当之，太子归立为考烈王
- **中期功业**：
  - 为相：考烈王元年为相，封春申君
  - 封地：初赐淮北十二县，后改封江东吴墟
  - 养士：三千余人，上客皆蹑珠履
  - 救赵：率兵救邯郸，秦兵去
  - 伐鲁：灭鲁，以荀卿为兰陵令
  - 合纵：为从长伐秦，至函谷关败走
- **晚期悲剧**：
  - 楚王无子，春申君忧虑
  - 李园献女，春申君幸之知有身
  - 献女楚王，生子立为太子
  - 李园养死士，朱英预警
  - 春申君不听，后17日被杀灭族

### 李园
- **身份**：赵人，春申君舍人
- **阴谋**：
  - 献女弟于春申君
  - 通过女弟说服春申君献女楚王
  - 女弟生子立为太子，李园用事
  - 阴养死士，伏杀春申君

### 朱英
- **身份**：观津人，春申君客
- **预警**：
  - 三种"无望"（福/祸/人）
  - 预见李园必杀春申君
  - 请为郎中杀李园
- **结局**：知言不用，恐祸及身，亡去

## 核心事件

### 1. 说秦昭王（早期辉煌）
**背景**：
- 秦败韩魏于华阳
- 秦欲令白起与韩魏共伐楚
- 黄歇出使秦国

**游说策略**：
1. **两虎相斗**：秦楚斗，驽犬（韩魏齐）受弊
2. **历史类比**：智氏、吴国的教训
3. **利益分析**：伐楚无得，善楚可制天下
4. **恐惧诉求**：齐将称帝
5. **愿景描绘**：地跨两海，四国皆服

**结果**：
- 秦王曰"善"
- 止白起、谢韩魏
- 赂楚为与国

### 2. 护送太子归楚（忠义担当）
**危机**：
- 楚太子质于秦数年
- 楚顷襄王病，太子不得归
- 楚有阳文君之子竞争继承

**策略**：
1. **说应侯**：归太子则事秦必重，不归则失与国
2. **劝太子逃**：变衣服为楚使者御出关
3. **黄歇留守**：以死当之
4. **坦白时机**：度太子已远，秦不能追
5. **应侯救歇**："出身以徇其主"，不如归之以亲楚

**结果**：
- 太子归，三月后顷襄王卒
- 太子立为考烈王
- 黄歇为相，封春申君

### 3. 李园之祸（悲剧结局）
**阴谋过程**：
1. **接近**：李园为春申君舍人
2. **献女**：进其女弟，幸于春申君，知有身
3. **说服**：通过女弟说春申君献女楚王
   - 楚王无子，将立兄弟
   - 新君必亲故人，春申君失势
   - 多失礼于王兄弟，祸且及身
   - 献女生子，则君之子为王，楚国尽可得
4. **执行**：春申君大然之，献女楚王
5. **养死士**：李园阴养死士欲杀春申君

**朱英预警**：
- **无望之福**：相少主可如伊尹周公（诱惑）
- **无望之祸**：李园养死士必杀君（威胁）
- **无望之人**：臣为君杀李园（救命稻草）

**春申君反应**：
- "李园，弱人也，仆又善之，且又何至此！"
- 不听朱英

**结局**：
- 后17日，楚王卒
- 李园先入，伏死士于棘门
- 刺杀春申君，斩首投棘门外
- 尽灭春申君之家

## 核心主题

### 1. 辩才智慧
- **说秦昭王**：长篇雄辩，层层递进
- **说应侯**：利益分析，说服放太子
- **战国四公子中辩才第一**

### 2. 忠义担当
- **舍身护主**：以死当之，护送太子
- **赌命换命**：留守秦国，等太子远后坦白
- **君臣信任**：太子立即封为相

### 3. 当断不断
- **贪欲**：被"楚国尽可得"诱惑
- **侥幸**："李园弱人也，仆又善之"
- **不听劝**：朱英预警后17日未行动
- **太史公评**："当断不断，反受其乱"

### 4. 四公子养士
- **规模**：三千余人
- **炫富**：上客皆蹑珠履，炫耀赵使
- **价值**：朱英预警（虽未听）、荀卿为兰陵令
