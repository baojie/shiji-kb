## 章节概述

苏秦是战国时期最著名的纵横家之一，也是合纵策略的主要倡导者。他的故事是中国历史上最具戏剧性的"逆袭"案例之一：早年游学失败、穷困潦倒，家人白眼相加；后来凭借不懈努力，悟得游说之术，佩六国相印，东而抗秦，位极人臣。

### 主要人物

- **苏秦**：洛阳人，师从鬼谷子（与张仪同门），合纵策略的主要倡导者
- **张仪**：苏秦同门，连横策略的代表，秦国重用，与苏秦在历史上形成对照
- **燕文侯、赵肃侯、韩宣王等**：苏秦游说的六国君主

### 历史背景

战国中期（约公元前340—前284年），秦国商鞅变法后实力大增，开始向东扩张。各国面临战略选择：
- **合纵**：六国联合共同抗秦（苏秦倡导）
- **连横**：各国单独与秦国结盟（张仪代表的秦国战略）

苏秦的历史意义在于，他提出并短暂实现了合纵策略，成功建立了六国联盟，使秦国"十五年不敢出函谷关"。

## 核心故事

### 1. 早年求学与挫败
苏秦游历多年，学成归来却未能得到任用。最落魄时，他返回洛阳，"妻不下纴，嫂不为炊，父母不与言"——妻子不理睬，嫂子不做饭，父母不说话。这是"锥刺股"苦读故事的历史背景（虽然"头悬梁锥刺股"一说，苏秦只对应"锥刺股"部分）。

### 2. 发奋研读《阴符》
苏秦发现了周书《阴符》，"读书欲睡，引锥自刺其股，血流至足"，刻苦钻研一年，终于悟得游说之道。

### 3. 游说六国
苏秦以"合纵"之策游说赵、燕、韩、魏、齐、楚六国，分析各国与秦国的力量对比，论证"六国合力大于秦国"的战略判断。他的游说辞令极具说服力，《史记》大量引用了他的游说原文。

### 4. 佩六国相印
苏秦最终说服六国，被任命为六国纵约长，同时担任六国相印。他衣锦还乡时，"父母闻之，清宫除道，张乐设饮，郊迎三十里"，形成与早年潦倒的鲜明对比。

### 5. 被刺与后续
苏秦在齐国遭到刺杀，临死前设计让齐王追查凶手，以车裂之刑惩处了刺客，显示其到死都在运用谋略。

### 6. 马王堆文献的新发现
1973年长沙马王堆汉墓出土的《战国纵横家书》对苏秦的历史记载提出了修正：其中的苏秦更多活动于燕、齐之间，时间比《史记》记载要晚，张仪实际上比苏秦早死。司马迁的记载可能存在将苏代、苏厉等人的事迹误记在苏秦名下的情况。这一发现使苏秦的历史形象更加复杂。
