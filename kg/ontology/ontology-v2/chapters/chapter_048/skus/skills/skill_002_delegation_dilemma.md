## 技能概述

### 核心问题

**组织快速扩张时，派出的人与自立的人如何区分？派出之后如何防止独立？**

陈胜称王后派出六路大军：周文、吴广、武臣、周市、邓宗、葛婴。六月之间，三路成功完成任务（周文死战到底、吴广攻荥阳至死、周市立魏咎），三路全部失控：武臣自立为赵王、葛婴擅立襄彊、邓宗占九江后信息中断。再加上秦嘉等地方势力直接不受节制，张楚的中央权威实际只能覆盖派出去的前三天。

本技能提炼的是**反面教训**：如何避免张楚的授权失控。

### 历史来源

从《陈涉世家》派遣+自立的多起事件集中提炼：
- [3.1] 以吴叔为假王，监诸将以西击荥阳；令武臣、张耳、陈馀徇赵地
- [3.2] 葛婴擅立襄彊，陈王诛葛婴
- [5.1] 武臣到邯郸，自立为赵王——柱国建议"不如因而立之"
- [6.1] 韩广自立为燕王
- [8.1] 田臧矫王令以诛吴叔
- [9.1] 秦嘉不受命自立为大司马

## 方法步骤

### 步骤1：派遣前的对象筛选

**目标**：只派能完成任务、不会独立的人

**操作**：
1. **优先派相对弱者**：能做事但难以独立自立的人——陈胜派周文（宾客出身无地盘）就比派武臣（陈地有根基）稳妥
2. **派有家眷在后方的人**：质子制度——陈胜扣武臣家属于宫中，是事后补救而非事前预防
3. **识别"功利型"与"野心型"**：功利型接受奖赏即可，野心型必会自立；武臣、韩广、田臧都是野心型而陈胜未识别
4. **不派有政治符号资源的人**：周文虽为陈人但无楚王室血统，稳定；立魏咎却因其有"甯陵君"封号，注定脱离

**注意事项**：
- 派遣前的筛选是**成本最低的控制环节**，一旦派出再控制代价陡增
- 野心型+有独立资源+距离远=必然自立，此组合绝对禁止

### 步骤2：派遣时的明确边界

**目标**：一次性把权力边界、任务目标、汇报节奏讲清

**操作**：
1. **任务边界明确**：不是"徇赵地"（无边界），而是"攻下X、Y、Z城后归报"
2. **权力边界明确**：可调兵，但不可立王、不可委任地方官、不可与他国结盟
3. **汇报节奏明确**：每X日一报，每Y里一驿
4. **资源边界明确**：粮草、兵员来源规定；不得就地扩军超过X

**注意事项**：
- 陈胜的最大失误是派出时"无任务边界"——武臣到邯郸无事可做便自立
- 边界越模糊，越依赖被派者的主观忠诚，越不可控

### 步骤3：派遣后的监督机制

**目标**：建立独立于被派者的信息与制衡通道

**操作**：
1. **插入监军/督察**：陈胜派武平君畔监郯下军——方向正确但层级不够
2. **多人派遣相互制衡**：张耳、陈余与武臣同行——初衷正确，但没有机制让其敢于举报武臣
3. **留守独立通讯**：派人独立驰报，不靠被派者自己汇报
4. **设计后勤牵制**：粮道、补给由中央掌握，派出单位有功能依赖

**注意事项**：
- 陈胜的监军失败在于：武平君畔被秦嘉杀了，没有上级支援机制
- 监督必须配套"监督者的安全保障"，否则监督者被收买或被杀

### 步骤4：自立事件的快速反应

**目标**：有人自立时，快速判断并采取相应对策

**操作**：
1. **区分"自立"与"擅立"**：
   - 擅立（立别人为王）：葛婴立襄彊——可以追诛
   - 自立（立自己为王）：武臣立自己为赵王——难以追诛
2. **区分有实力与无实力**：
   - 武臣有张耳陈余兵众，无法追诛，只能"因而立之"
   - 葛婴仅有小股兵力，可以召回诛杀
3. **预设"因而立之"的退路**：陈胜事发后被动接受武臣赵王，被迫承认
4. **立即切断连锁反应**：武臣自立后韩广跟进，必须阻断连锁链

**注意事项**：
- 差别对待"有实力的自立"与"无实力的擅立"：追诛无实力者可立威，追诛有实力者会生第二个秦
- 柱国房君"不如因而立之"的建议是现实主义的妥协，但也埋下张楚瓦解的伏笔

### 步骤5：用"授权递减"切分长线

**目标**：不一次性给全部授权，而是分阶段释放

**操作**：
1. **初始授权**：只给最小任务+最小资源
2. **阶段性增加**：完成目标再增加下一阶段授权
3. **资源与权力不挂钩**：兵可给多，但印玺不给
4. **临时性授权**：假王、假将军——冠"假"字明示临时，防止路径依赖

**注意事项**：
- 陈胜给吴广"假王"的设计是对的，但后续给田臧楚令尹印则错——田臧无背景，给最高文官印就给了他"反制假王"的合法性
- 一次给完所有授权等于放走人

## 应用场景

### 历史案例分析

**案例一：武臣自立为赵王（步骤1-2失败）**

**背景**：陈胜令陈人武臣、张耳、陈馀徇赵地，未设任务边界。

**问题分析**：
- **步骤1失败**：武臣是"陈人"（陈胜老乡），陈胜认为可信；但未看出武臣+张耳陈余有地方基础
- **步骤2失败**：任务只是"徇赵地"，无边界；武臣到邯郸即面临"继续前进还是自立"的选择
- **步骤3失败**：无监军，张耳陈余反成为自立的协助者
- **步骤4部分成功**：武臣自立后陈胜一度欲诛其家，柱国劝阻"因而立之"——正确决策但已失去先机
- **后果**：武臣自立后进一步派韩广徇燕，韩广再自立；自立链条成型

**教训**：派遣"陈人+张耳陈余"组合是陈胜最大的战略失误；无任务边界是第二大失误。

**案例二：葛婴擅立襄彊（步骤1失败，步骤4成功）**

**背景**：葛婴将兵徇蕲以东，至东城立襄彊为楚王。

**问题分析**：
- **步骤1失败**：葛婴是符离人，性格冲动，未经筛选
- **步骤4成功**：葛婴擅立（立别人）而非自立，无政治后盾；陈胜追诛得以立威
- **教训**：如陈胜对所有人都能如对葛婴一样果断，则不会有后续诸自立事件；但他做不到——因为其他人都有实力背景

### 现代应用示例

#### 场景1：创业公司分支机构的授权

**问题**：总部在A城，要开B城分公司，派遣首任总经理；如何防止分公司独立？

**应用**：
1. **步骤1**：选派非B城本地人担任总经理；其家庭关系在A城
2. **步骤2**：任务书详细：开设前X月重点客户、首年营收目标、扩张决策须上报
3. **步骤3**：财务总监独立汇报总部；派驻审计每季度到场
4. **步骤4**：预设"若分公司提出独立，先接受再内部清算"的应变预案
5. **步骤5**：分公司负责人只有经营权，无人事终决权、无业务方向权

**预期效果**：即便分公司做大，总部仍有制衡；即便出现独立诉求，不至于全面失控

#### 场景2：跨国公司海外子公司管理

**问题**：在X国设立子公司，经营多年后当地团队要求独立运营

**应用**：
1. **步骤1**：首任CEO应为本部派驻，非本地招募
2. **步骤2**：战略权归本部，执行权归子公司
3. **步骤3**：CFO、合规官、法务直接向本部汇报
4. **步骤4**：预设"部分让步 vs 全面脱离"的分级应对
5. **步骤5**：核心知识产权、品牌、客户关系由本部持有

**预期效果**：避免子公司借地缘/汇率优势脱离母公司控制

## 成功标准

### 过程指标

- [ ] 派遣对象经过"资源、背景、性格"三维筛查
- [ ] 任务书有明确的目标、权限、边界、汇报机制
- [ ] 派遣单位内部有至少一条独立于负责人的信息通道
- [ ] 负责人的家属/利益根基保留在总部可控范围
- [ ] 预设"自立事件"的分级应对预案

### 结果指标

- [ ] 派出单位完成任务比例高于70%
- [ ] 未发生派遣后脱离母体的情况
- [ ] 即便发生脱离，母体能在30天内重建控制或完成切割
- [ ] 其他派遣单位不因一次脱离事件产生连锁反应

## 常见错误

### 错误1：派遣即授权，全权不设限

**表现**：只说"你去X地做这件事"，不说具体权力边界
**原因**：领导者想简化沟通，或出于"信任"的姿态
**纠正**：派遣前必须有书面任务书；不规定禁止项的都算授予

### 错误2：监督机制形同虚设

**表现**：有监军但监军层级过低，或监军没有独立汇报通道
**原因**：低估被派者对监军的影响力
**纠正**：监军应由总部任命、总部汇报，被派者不得干预监军的行动与信息传递

### 错误3：先例处理不一致

**表现**：对同样行为（如自立）根据实力大小差别处理
**原因**：现实无奈
**纠正**：即便现实无法追究，必须对内表明立场，不让先例成为"可复制模式"；或至少让所有其他被派者知道"如果自立会被诛"

### 错误4：授权与信号混淆

**表现**：给"假王"印却又让其长期掌权，使"假"字失去意义
**原因**：急于激励
**纠正**："假"、"试"、"临时"等前缀必须短期（如3个月）内明确转正或撤回

## 进阶技巧

1. **派遣的"两人组合"设计**：派一位执行者+一位监督者，监督者薪酬由总部直接发放，且双方都知道彼此角色
2. **"功成则赏，但不立王"的激励**：陈胜失在"功宜为王"的逻辑开启；正确做法是立侯不立王——侯可以世袭但不能建国
3. **派遣者的"回召机制"**：设定某些条件下必须回召（如某城攻克后），用回召作为权力收束的节点
4. **"第二道保险"——家属质押**：自愿但有利益绑定，如"你的家属以贵宾身份在总部"——吕后对韩信之母的接待，比陈胜扣武臣家属更高明
