# Fact 005: 下城父之变——御者庄贾弑陈王

## 基本信息

- **章节**: 048 陈涉世家
- **类型**: 重大历史事件 / 权力崩溃事件
- **时间**: 秦二世元年腊月（前209年末/前208年初）
- **主角**: 陈胜（被弑者）、庄贾（行凶御者）、吕臣（复仇者）、章邯（外部压力）
- **地点**: 下城父（今安徽涡阳附近）
- **标签**: `#庄贾` `#弑君` `#隐王` `#腹心不与` `#首事之终`

---

## 事件背景

### 历史情境

张楚政权从称王到陈胜身死仅六月。陈胜之死的直接原因不是秦军主力，而是身边御者的反噬。这不是孤立的偶发事件，而是张楚内部信任体系彻底崩溃后的必然结果。

《太史公赞》点明："始亲硃房，又任胡武。夥颐见杀，腹心不与。庄贾何人，反噬城父！"——太史公把庄贾弑主与陈胜用人、杀故人、任酷吏连成一条因果链。

### 关键因素

1. **前线接连崩溃**：周文败亡 → 田臧矫命杀吴广 → 田臧败死 → 李归死 → 邓说军散 → 伍徐军散 → 章邯兵临陈城
2. **中央信任体系瓦解**：陈胜杀故人、任硃房胡武苛察诸将，"诸将不亲附"
3. **地方势力脱节**：武臣、韩广、田儋、秦嘉都不听节制；周市-魏咎系统太弱
4. **贴身护卫的政治忠诚问题**：御者（车夫）并非战斗人员，但却是贴身接触最多的人；庄贾的选任未经严格筛查

---

## 详细叙述

### 阶段一：前线的全面溃败

> 章邯已破伍徐，击陈，柱国房君死。章邯又进兵击陈西张贺军。陈王出监战，军破，张贺死。[10.1]

章邯从西线（周文）、荥阳（田臧、李归）、东线（邓说、伍徐）逐个击破后，兵锋直逼陈城。陈王亲出监战，张贺战死，陈王不得不撤退。

### 阶段二：向汝阴撤退

> 腊月，陈王之汝阴，还至下城父，其御庄贾杀以降秦。[10.2]

陈王撤退路线：陈 → 汝阴（方向西南）→ 下城父（折返东北）
此撤退动作极为仓皇：刚到汝阴又掉头返回，说明汝阴不安全；至下城父时，御者庄贾动手。

**为什么御者能杀王？**
- 御者驾车，与王同车独处
- 无其他护卫在侧（说明随行已溃散）
- 御者对路线最熟悉，可选择僻静处下手

### 阶段三：庄贾弑君降秦

庄贾的动机：
- **生存计算**：陈王败局已定，杀王降秦是免死的最佳选项
- **利益诱惑**：秦朝对降将有赏，杀王降秦赏最厚
- **惧怕秦法**：陈王的"御"是张楚官员，秦律对参与起义者诛族

庄贾杀陈王后降秦，秦军一度占领陈。

### 阶段四：吕臣复陈

> 陈王故涓人将军吕臣为仓头军，起新阳，攻陈下之，杀庄贾，复以陈为楚。[11.1]

- **吕臣身份**：陈王"涓人"（近侍）出身的将军，最忠于陈胜
- **仓头军**：以青头巾为标识的军队，是陈胜政治符号的延续
- **攻陈下之**：夺回陈城
- **杀庄贾**：复仇成功
- **复以陈为楚**：维持张楚国号

这是**陈胜死后张楚政权的最后坚守**。吕臣虽复陈，但很快秦左右校复攻陈下之，吕将军走；后来与鄱盗黥布之兵相收，破秦左右校于青波，再次复以陈为楚。最终被项梁系统吸纳。

### 阶段五：安葬与谥号

> 陈胜葬砀，谥曰隐王。[10.2]

- **葬砀**：砀地（今河南永城）是起义活动区的核心；非首都之葬
- **谥曰隐王**："隐"谥的含义复杂：
  - 《谥法》"隐拂不成曰隐"——志向未竟
  - 也可解为"哀怜之隐"——对其夭折的悼惜
  - 这是**张楚系统内部给予的谥号**，不是后世朝廷追封

### 阶段六：汉高祖的制度承认

> 陈胜虽已死，其所置遣侯王将相竟亡秦，由涉首事也。高祖时为陈涉置守冢三十家砀，至今血食。[15.1]

汉高祖立国后，主动为陈胜置守冢三十家——这是王级别的祭祀规格（汉初守冢制度：秦始皇二十家，孔子十家，与陈胜三十家相当），在司马迁时代（前90年代）仍血食不绝。

## 历史意义

### 直接影响

1. **张楚政权的实际终结**：虽然吕臣、黥布一度复陈，但陈胜死后的"楚"已不是张楚，而是项梁系的楚
2. **楚统的转移**：项梁借鉴陈胜教训立楚怀王心（非自立），以"战国血统"取代"功劳立王"
3. **反秦格局的重组**：从"陈胜首事—六国自立"转为"楚义帝—项羽十八王—楚汉相争"
4. **秦军的短暂喘息**：章邯攻下陈后，秦军获得一段喘息期，后又相继击破齐田儋、楚项梁

### 长远影响

1. **反噬概念的确立**：《太史公赞》"反噬城父"的表述，使庄贾成为中国政治史上"内贼"的原型
2. **"腹心不与"的警诫**：太史公借此事件强调：领导者的根本风险不是外敌，而是失去贴身人的忠诚
3. **谥号"隐"的特殊性**：以隐王谥平民起义者，是《史记》体例的大胆安排——承认陈胜的"王"地位
4. **汉初对首事之功的制度化承认**：守冢三十家血食，是汉政权对自己"继陈胜之志"的公开声明

---

## 现代启示

### 可借鉴之处

1. **贴身人员的筛选机制**：越是危急时刻，越需要严格的贴身保障；御者、秘书、司机不能只看技术
2. **败局下的撤退秩序**：仓皇撤退是领导人最脆弱的时刻，必须预设多重护卫和退路
3. **信任体系的外显化**：陈胜杀故人的后果之一就是失去了"可以真正信任的人"；危急时没有人愿意冒险救他
4. **"首事"身份的制度化固化**：陈胜身后由汉廷追认为"首事者"，说明正当性可以通过后世制度获得——企业早期创始人的历史地位也是如此

### 需警惕之处

1. **快速扩张中的信任透支**：陈胜派出六路大军后，周边"可以真正信任的人"已经所剩无几
2. **酷吏治理的反噬效应**：硃房胡武苛察使诸将不亲附，结果危急时无人救援
3. **公关符号不能取代实质保护**：张楚的"张"字、袒右、坛盟等符号再华丽，也无法抵挡一个御者的短剑
4. **信息隔断的致命危险**：陈胜败退至下城父时，很可能已经与外部失联——没有传令、没有接应、没有后勤——这种信息孤岛是弑君的温床
