# Fact 005 - 陈胜称王后的用人失误

**章节**: 048_陈涉世家
**来源段落**: [14.1]
**类型**: 叙事性事实

---

## 用人失误的三个层次

### 第一层：疏远故人

**事件背景**：
```
"其故人尝与庸耕者闻之，之陈，扣宫门曰：'吾欲见涉。'"
```

**处理过程**：
1. **初次接触**：宫门令欲缚故人（阻拦）
2. **陈胜召见**：载与俱归，入宫
3. **故人反应**："夥颐！涉之为王沈沈者！"（感叹宫殿之豪华）
4. **言多必失**：客出入愈益发舒，言陈王故情
5. **有人进谏**："客愚无知，颛妄言，轻威"
6. **陈胜决断**：斩之
7. **连锁反应**："诸陈王故人皆自引去，由是无亲陈王者"

**问题分析**：
- **未能平衡新旧关系**：新地位 vs 旧交情
- **缺乏容忍度**：不能容忍故人的坦率
- **杀鸡儆猴失败**：本想维护威严，却导致众叛亲离
- **失去信息渠道**：故人离去，失去了解民间疾苦的窗口

---

### 第二层：重用酷吏

**人事安排**：
```
"陈王以硃房为中正，胡武为司过，主司群臣。"
```

**职责定位**：
- **硃房（中正）**：监察群臣的忠诚与行为
- **胡武（司过）**：专门负责找错、处罚

**工作方式**：
```
"诸将徇地，至，令之不是者，系而罪之，以苛察为忠。
其所不善者，弗下吏，辄自治之。"
```

**工作特点**：
1. **苛刻细察**：以挑剔为忠诚
2. **越权执法**：不经司法程序，直接处理（"弗下吏，辄自治之"）
3. **无程序正义**：硃房、胡武说谁有罪就是谁有罪

**恶果**：
```
"陈王信用之。诸将以其故不亲附，此其所以败也。"
```

- **失去将领支持**：诸将因害怕被整，不再亲近陈王
- **军心涣散**：领兵在外的将领担心被清算
- **最终灭亡**：太史公明确指出"此其所以败也"

---

### 第三层：不控制权力扩散

**地方势力失控**：

| 人物 | 原职务 | 后续行为 | 结果 |
|------|--------|----------|------|
| 武臣 | 陈王派遣徇赵地 | 自立为赵王 | 陈王虽怒，但无力制止，只能承认 |
| 葛婴 | 陈王派遣徇蕲以东 | 擅立襄彊为楚王 | 陈王诛杀葛婴 |
| 秦嘉 | 地方势力 | 不受陈王节制，自立为大司马 | 矫王令杀武平君畔 |
| 田臧 | 陈王将军 | 矫王令诛吴叔 | 陈王虽赐田臧印，但军事失败 |

**权力失控的表现**：
- **中央指令不被执行**：秦嘉"不受命"
- **地方擅自立王**：武臣、襄彊
- **将领擅杀将领**：田臧杀吴叔
- **陈王被动应对**：要么追认（武臣），要么事后惩罚（葛婴）

## 对比：刘邦的用人策略

| 维度 | 陈胜 | 刘邦 |
|------|------|------|
| 对待故人 | 杀故友，故人皆去 | 封萧何、曹参，优待故人 |
| 用人标准 | 苛察为忠（硃房、胡武） | 各取所长（张良谋、韩信战、萧何治） |
| 权力控制 | 派出去就失控（武臣自立） | 先分封后削藩（白登之围后更谨慎） |
| 对待将领 | 诸将不亲附 | 与将士同甘共苦 |

---

## 太史公的评价

```
"陈王信用之。诸将以其故不亲附，此其所以败也。"
```

**核心观点**：
- 陈胜失败的根本原因是**失去人心**
- 用人失误 → 将领不亲附 → 军心涣散 → 最终败亡

---

## 历史教训

### 对创业者的启示

1. **不能疏远元老**
   - 创业成功后，不能抛弃早期追随者
   - 需要平衡新旧势力

2. **不能依赖酷吏**
   - 过度依赖监察、整肃，会失去团队信任
   - 需要制度建设，而非个人恐怖

3. **不能失控权力下放**
   - 派出去的人要有制衡机制
   - 不能放手不管

4. **要保持亲和力**
   - 领导者需要与核心团队保持信任
   - "诸将不亲附"是致命的
