## 两次舆论造势行动

### 第一次：鱼腹藏书

**操作步骤**：
```
1. 占卜询问：找卜者咨询，卜者理解意图
2. 制作预言：用丹砂在帛上写"陈胜王"
3. 植入证据：将帛书藏入鱼腹
4. 发现预言：士卒买鱼烹食，发现鱼腹中书
5. 舆论效果：士卒"固以怪之矣"
```

**核心要素**：
- **媒介**: 鱼（日常食物）
- **内容**: "陈胜王"（简洁明确的政治预言）
- **颜色**: 丹书（红色，象征神秘与权威）
- **受众**: 普通士卒
- **效果**: 制造神秘感，为后续起义埋下心理伏笔

---

### 第二次：狐鸣篝火

**操作步骤**：
```
1. 选择地点：驻地旁边的丛祠（有宗教氛围）
2. 选择时间：夜晚（增强神秘感）
3. 制造光效：篝火（营造超自然场景）
4. 模拟声音：狐鸣"大楚兴，陈胜王"
5. 舆论发酵：士卒夜惊恐，白天互相传说，都指目陈胜
```

**核心要素**：
- **地点**: 丛祠（借用宗教场所的神圣性）
- **时间**: 夜晚（利用黑暗与恐惧）
- **声音**: 狐鸣（动物声音，暗示天意）
- **内容**: "大楚兴，陈胜王"（明确的政治预言）
- **传播**: 口口相传 + 视觉确认（"皆指目陈胜"）

## 舆论战略分析

### 设计原理

1. **多感官刺激**
   - 视觉：丹书帛、篝火
   - 听觉：狐鸣
   - 触觉：鱼腹中的实物证据

2. **借助超自然力量**
   - 鱼腹藏书 → 暗示天意
   - 狐鸣 → 动物灵异，预示吉凶
   - 丛祠 → 借助宗教场所的神圣性

3. **分阶段建立认知**
   - 第一阶段：制造怪异（"固以怪之矣"）
   - 第二阶段：强化恐惧（"夜惊恐"）
   - 第三阶段：形成共识（"皆指目陈胜"）

4. **符合当时认知水平**
   - 秦代民众普遍相信天人感应
   - 相信动物异象预示重大事件
   - 相信文字具有神秘力量


- **Who**: 陈胜、吴广（信息发送者）
