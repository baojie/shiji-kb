## 概述

当王位继承出现危机（继承人无能/暴虐/早亡/权力真空）时，如何应对以避免政权崩溃。基于夏商周三代的成功/失败案例提炼。

## 三大典型危机

### 危机类型1：继承人无能（太康失国）

**症状识别**：
- 继承人奢侈腐化
- 政务荒废
- 民怨沸腾
- 外部势力蠢蠢欲动

**应对步骤**：

#### 步骤1：早期预警（继位前）
- 监督太子教育
- 定期考核能力
- 及时调整培养方案

#### 步骤2：紧急干预（继位后发现）
- **选项A**：摄政+培养（伊尹模式，见下文）
- **选项B**：劝谏+辅佐（贤臣辅佐）
- **选项C**：废立（风险高，需贵族共识）

#### 步骤3：防止外部填补
- ✗ 禁止：让外部势力（后羿）趁虚而入
- ✓ 允许：内部贵族临时执政
- ✓ 关键：保持家族统治的连续性

**太康失国的教训**：
- ✗ 无应急机制→外部势力夺权
- ✗ 无贤臣辅佐→继承人无人管教
- ✗ 制度不成熟→无废立程序

---

### 危机类型2：继承人淫乱（伊尹放太甲）

**症状识别**：
- 继承人道德败坏（淫乱）
- 违反祖宗制度
- 诸侯不满
- **但**：非完全无能，有可塑性

**伊尹模式步骤**：

#### 步骤1：决策（放逐vs废立）
**条件判断**：
- ✓ 如继承人年轻+可塑性→放逐培养
- ✗ 如继承人年老+无可救药→废立

**太甲案例**：
- 年轻（汤之孙，可能20-30岁）
- 淫乱但非暴虐（有可改造空间）
- 伊尹决定：放逐+培养

#### 步骤2：放逐地点选择
- **桐宫**（汤墓附近）
- 目的：让太甲反思祖先功德
- 环境：简朴，无奢侈条件

#### 步骤3：三年培养
**内容**：
- 学习祖先创业史（汤灭夏桀的正义性）
- 学习治国之道
- 反思自身错误

**监督**：
- 伊尹定期探访
- 观察悔过程度
- 判断是否可复位

#### 步骤4：复位或废黜
**太甲案例**：
- 三年后悔过自责
- 伊尹迎回复位
- 太甲成为贤君

**如果不悔改**：
- 继续放逐或废黜
- 另立贤者

**必要条件**：
1. **贤臣**：
   - 威望高（伊尹是汤之相）
   - 无私心（不篡位）
   - 有能力（培养太甲）

2. **君主可塑性**：
   - 年轻
   - 非本性恶毒
   - 愿意改过

3. **贵族支持**：
   - 认可摄政
   - 不趁机篡位

---

### 危机类型3：暴君出奔（厉王→共和）

**症状识别**：
- 君主暴虐（"以恶闻过乱"）
- 言论管制（道路以目）
- 民怨爆发（国人暴动）
- 君主被迫出奔

**共和模式步骤**：

#### 步骤1：君主出奔后立即稳定
**关键时间**：24-48小时
- 贵族紧急会议
- 宣布临时执政方案
- 稳定军队和官僚系统

#### 步骤2：建立集体执政机制
**周朝共和**：
- 执政者：周定公+召穆公（两位贵族）
- 权力：平等共享，共同决策
- 任期：至太子长成（14年）

**机制设计**：
- 重大决策：两人共同商议
- 日常政务：分工负责
- 诸侯朝贡：共同接见

#### 步骤3：保护太子
**关键**：
- 太子（后来的宣王）安全
- 秘密培养
- 避免被暴君（厉王）或野心家伤害

#### 步骤4：准备交接
**14年后**：
- 厉王死讯传来
- 太子已长成
- 平稳继位（宣王）

**成功因素**：
1. **贵族共识**：
   - 周定公、召穆公无篡位野心
   - 共同目标：保周朝

2. **礼制约束**：
   - 不能擅自称王
   - 尊重宗法

3. **分封制保障**：
   - 诸侯国稳定
   - 不趁机造反

4. **时间适中**：
   - 14年（可控范围）
   - 不超过20年（否则难以维持）

## 权力真空的四种解决方案对比

| 方案 | 案例 | 风险 | 成功率 | 必要条件 |
|------|------|------|--------|---------|
| **外部填补** | 太康→后羿 | 极高（政权更迭） | 低 | 无（被动） |
| **摄政培养** | 伊尹→太甲 | 中（贤臣篡位风险） | 高 | 贤臣+君主可塑性 |
| **集体执政** | 厉王→共和 | 中（贵族争权风险） | 中 | 贵族共识+礼制 |
| **内战** | 商九世乱 | 极高（国力耗尽） | 极低 | 无（最差结果） |

## 决策树：如何选择方案？

```
权力真空发生
  ↓
Q1: 是否有明确继承人？
  ├── 有 → Q2: 继承人是否有能力？
  │        ├── 有 → 正常继位
  │        └── 无 → Q3: 是否可培养？
  │                 ├── 可（年轻+可塑性） → 伊尹模式（摄政+培养）
  │                 └── 不可（年老+无可救药） → 废立（风险高）
  └── 无 → Q4: 是否有集体领导机制？
           ├── 有 → 共和模式（集体执政）
           └── 无 → 极高风险（可能内战或外部填补）
```

## 现代应用

### 应用1：企业CEO突然离职

**情景**：CEO突然去世/辞职/被解雇

**应对（共和模式）**：
1. 立即启动应急预案
2. 临时董事会集体决策
3. 寻找/培养新CEO
4. 3-6个月内完成交接

### 应用2：高管能力不足

**情景**：新任高管能力不足但未完全失败

**应对（伊尹模式）**：
1. 暂停职务（保留职位）
2. 送去培训（商学院/轮岗学习）
3. 3-12个月考察期
4. 能力改进→复职，否则→转岗或离职

### 应用3：领导暴虐/腐败

**情景**：领导严重失职，团队/股东不满

**应对（厉王模式）**：
1. 董事会/股东会弹劾
2. 强制辞职（"出奔"）
3. 临时集体领导
4. 选举/任命新领导

## 检查清单

### 危机识别阶段
- [ ] 症状是什么？（无能/淫乱/暴虐/早亡）
- [ ] 严重程度？（可挽救/不可挽救）
- [ ] 时间紧迫性？（立即处理/可延后）

### 方案选择阶段
- [ ] 是否有明确继承人？
- [ ] 继承人是否可培养？
- [ ] 是否有贤臣/贵族可摄政？
- [ ] 是否有集体执政机制？

### 实施阶段
- [ ] 快速决策（24-48小时）
- [ ] 稳定局面（宣布方案）
- [ ] 执行培养/等待（3年-14年）
- [ ] 准备交接

### 交接阶段
- [ ] 继承人是否成熟？
- [ ] 诸侯/团队是否认可？
- [ ] 仪式/程序是否完备？

## 常见错误

### 错误1：拖延决策
- **案例**：太康失国（无应急机制）
- **后果**：外部势力趁虚而入

### 错误2：贤臣篡位
- **风险**：伊尹模式如遇野心家
- **预防**：选择威望高+无私心的贤臣

### 错误3：集体执政时间过长
- **风险**：超过20年难以维持共识
- **预防**：明确时间限制（等太子长成）

### 错误4：无保护继承人
- **案例**：如厉王出奔后太子被杀
- **预防**：秘密保护+培养

## 预期结果

### 成功案例
- ✓ 伊尹放太甲：3年→太甲悔过→商朝稳定
- ✓ 共和行政：14年→宣王继位→宣王中兴

### 失败案例
- ✗ 太康失国：无应急→后羿夺权→失国数十年

### 关键成功因素
1. **快速反应**（24-48小时决策）
2. **贤臣/贵族共识**（无私心）
3. **合理时间**（3-15年可控）
4. **保护继承人**（秘密培养）
