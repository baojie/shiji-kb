## 章节简介

《三代世表》是司马迁编纂的第一张历史年表，系统梳理了从黄帝到周共和时期（约前2070-前841年，跨度1800年）的王朝世系。包括：
- 五帝世系（黄帝→颛顼→帝喾→尧→舜）
- 夏朝17位帝王（禹→启→...→桀）
- 商朝30位帝王（汤→...→纣）
- 周朝至共和10位天子（武王→成王→...→厉王→共和）
- 周朝11个诸侯国的早期世系（鲁、齐、晋、秦、楚、宋、卫、陈、蔡、曹、燕）

## 核心发现

### 1. 继承制度的演变
- **夏朝**：82%父子传位，12%兄弟 → 稳定但第3代失国（太康）
- **商朝**：60%父子传位，33%兄弟 → 能力强（太戊、武丁中兴）但九世之乱（300年）
- **周朝**：80%父子传位，10%兄弟 → 800年稳定（嫡长子制+分封+礼制）

### 2. "明确性-灵活性悖论"
- 规则越明确（父子继承）→ 稳定性↑，能力牺牲↑
- 规则越灵活（兄弟继承）→ 能力筛选↑，争夺成本↑
- **最优解**：周朝方案（明确主规则 + 应急机制）

### 3. "三代定律"
- 第1代（开国）：能力强，无需制度
- 第2代（巩固）：依靠父荫和元老
- **第3代（考验）**：制度化关键期
  - 成功→盛世（周康王"刑错四十年"）
  - 失败→失国（夏太康失国）

### 4. 权力真空的四种解决方案
- **外部填补**（太康→后羿）：最差，政权更迭
- **摄政培养**（伊尹→太甲）：最佳，但需贤臣
- **集体执政**（厉王→共和）：临时方案，需贵族共识
- **内战**（商九世乱）：最惨，300年混乱

### 5. 兄终弟及的"时间炸弹"
- 第1代：兄弟和睦
- 第2代：侄vs子矛盾
- 第3代：堂兄弟争夺
- **第4代：必然内战**
