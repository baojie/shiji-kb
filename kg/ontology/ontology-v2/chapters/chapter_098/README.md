## 章节主题

本传合并三位人物，核心主题是：

1. **封赏的多元标准**：汉初功臣封侯并非只有战功一条路，忠诚陪护也是重要路径。

2. **量化战功的制度力量**：精确到个位数的斩首记录，展示了汉初军功制度的精细化管理。

3. **"天授"论**：太史公认为傅宽、靳歙屡战屡胜、"未尝困辱"，有天命加持的成分，与其单纯的能力无关——这是司马迁对历史偶然性的哲学思考。
