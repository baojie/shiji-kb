# Chapter 070: 张仪列传

## 章节概述

《张仪列传》记载战国著名纵横家张仪的生平与事迹。张仪与苏秦同师鬼谷先生，苏秦行合纵以联六国抗秦，张仪则行连横为秦破合纵。本传集中展示了战国时代游说之士如何以口才、谋略、外交欺诈影响国际格局。

张仪最著名的事迹是以"商於六百里"欺骗楚怀王，使楚绝齐后秦楚两国交战，楚丧汉中之地。此事件揭示了利令智昏与忠言逆耳的永恒命题。

## 核心人物

### 张仪
- **出身**：魏人
- **师承**：鬼谷先生，同学苏秦
- **核心策略**：连横（使各国分别与秦结盟，破六国合纵）
- **著名事件**：欺楚怀王商於六百里、说魏王事秦、入楚被囚脱身
- **结局**：秦武王立，不重张仪；出走魏国，一年后卒于魏

### 苏秦（对比人物）
- 与张仪同师鬼谷
- 行合纵，联六国抗秦
- 恐秦攻诸侯败约，故激发张仪赴秦
- 评张仪："吾殆弗如也，今吾幸先用，而能用秦柄者，独张仪可耳"

### 楚怀王
- 轻信张仪商於六百里之诺
- 不听陈轸预警，群臣贺而独陈轸吊
- 绝齐后被秦所欺，两次伐秦皆败
- 最终入秦被扣，死于秦

## 核心事件

### 1. 苏秦激励张仪入秦
- 苏秦故意轻慢张仪，激其怒赴秦
- 暗中资助张仪，使其得见秦惠王
- 体现战略家的深远布局：用敌人替自己守护合纵

### 2. 张仪说秦伐蜀之辩（主张伐韩）
- 与司马错争论：张仪主张据三川挟天子，司马错主张伐蜀富国
- 秦惠王从司马错，先伐蜀
- 揭示连横思维的政治优先（周室）vs实用主义（土地）

### 3. 欺楚怀王商於六百里
- 张仪以六百里土地诱楚绝齐，实际毁约
- 楚兴兵伐秦，两战皆败，失汉中
- 陈轸预判全部应验，楚王不听

### 4. 张仪入楚被囚脱身
- 楚怀王欲得张仪换黔中地
- 张仪自请入楚，借靳尚→郑袖→楚王的人脉链脱身
- 展示险境外交官的生存智慧

### 5. 张仪去秦赴魏
- 秦武王立，不重张仪，张仪出走魏国
- 魏王亦不用，卒于魏
- 纵横之士的宿命：依附王权，无权则无用

## 核心主题

### 1. 连横破合纵
- 个别击破强于集体对抗
- 许诺未必兑现，欺诈是外交工具

### 2. 利令智昏
- 楚怀王见六百里之利，不听陈轸谏言
- "不原陈子闭口毋复言，以待寡人得地"
- 贪欲使人丧失理性判断

### 3. 辩才立身
- "卒不辱于辩"——张仪之妻早知其命运
- 纵横之士以口才为命，无权无势，只有言语

### 4. 游士的局限
- 仕秦有功则用，武王不喜则去
- 没有根基，只有才能，迁徙无定
- 战国游士的典型命运
