# 十二诸侯年表 — 知识图谱

《十二诸侯年表》记录从共和元年（前841）至孔子卒后（前479）的564年间，12个诸侯国的兴衰全景，是理解春秋时代权力格局演变、霸主政治、礼崩乐坏全过程的核心史料。

**SKU总数**：15个（4 Facts + 2 Skills + 8 Eureka + 1 占位） | **跨域洞察**：8条 | **版本**：v2.0

## 章节概览

### 时间跨度
- **共和元年**（前841）→ **孔子卒后**（约前476）
- 跨度：**564年**

### 核心内容

**太史公序言核心洞察**：
> "周道缺，诗人本之衽席，关雎作。仁义陵迟，鹿鸣刺焉。及至厉王，以恶闻其过，公卿惧诛而祸作，厉王遂奔于彘，乱自京师始，而共和行政焉。是后或力政，彊乘弱，兴师不请天子。然挟王室之义，以讨伐为会盟主，政由五伯，诸侯恣行，淫侈不轨，贼臣篡子滋起矣。"

**三大历史进程**：

1. **周王室衰落**（前841-前256）
   - 共和行政（前841-前828）：厉王被逐，召公周公共治14年
   - 平王东迁（前770）：失关中，仅有洛阳周边"方六百里"
   - 郑射周王（前707）：郑庄公射伤周桓王，天子威严扫地
   - 王室名存实亡（前550后）：诸侯完全无视周天子

2. **春秋五霸轮替**（前681-前550）
   - 齐桓公（前681-前643，38年）："九合诸侯，一匡天下"
   - 晋文公（前632-前628，9年）：城濮之战，践土之盟
   - 秦穆公（前659-前621，39年）：西向称霸，灭国12
   - 楚庄王（前613-前591，23年）：问鼎中原，邲之战大胜晋
   - 晋霸延续（前632-前550，82年）：晋悼公"九合诸侯"

3. **诸侯国兴衰**（前841-前476）
   - 早期灭亡：曹（前487）、陈（前478）
   - 战国灭亡：郑（前375）、鲁（前256）、宋（前286）等
   - 内部分裂：晋→韩赵魏（前403）、齐→田氏代姜（前489）
   - 最终统一：秦（前221，表外）

### Eureka（洞察型知识）：8条

详见 [eureka.md](eureka.md)

1. **分封制的"时间炸弹"——权力下放的不可逆性**
   - 数学模型：王室实力R（常数）vs 诸侯实力P(t) = P₀ × e^(rt)（指数增长）
   - 临界点：当P(Tc) > R时，权力倒转不可逆
   - 现代验证：苏联解体、企业分权、联邦制风险
   - 破局之道：秦朝郡县制、汉朝推恩令、现代矩阵式管理

2. **"四战之地悖论"——地缘劣势的生存智慧**
   - 悖论：地缘劣势国家（郑）发展出更高超策略 vs 地缘优势国家（秦）策略单一
   - 关键变量：改革能力 > 地缘优势 > 外交策略
   - 现代验证：以色列、新加坡、瑞士
   - 启示：地缘劣势可能是"伪装的祝福"（促进创新）

3. **"霸权更替的周期律"——130年的权力钟摆**
   - 霸权平均时长：30-40年（一代人）
   - 生命周期模型：H(t) = H₀ × e^(-λt)
   - 衰落原因：继承问题 + 新挑战者崛起 + 盟友背叛 + 内部衰落
   - 现代验证：美国霸权（1945-2025，80年）、英国霸权（1815-1914，99年）

4. **"礼崩乐坏的加速度"——从偶然越轨到常态失序**
   - 四阶段：初次越轨（郑射王）→ 越轨合理化（晋霸政）→ 越轨常态化（大夫专权）→ 越轨公开化（三家分晋）
   - 破窗理论：第一次越轨未被严惩 → 越轨行为蔓延 → 秩序崩溃
   - 沸腾点模型：秩序如水温，越轨如加热，临界点后不可逆
   - 现代验证：苏联解体、组织文化崩溃、2008金融危机

5. **"编年体的认知革命"——时间轴作为信息整合维度**
   - 创新：将不同国家、人物、事件投射到统一时间轴
   - 优势：多线索并行，时间同步，易于对比
   - 现代应用：甘特图（项目管理）、时间线（社交媒体）、时间序列图（数据可视化）
   - 认知科学：时间是最强大的整合维度，人类大脑天然理解时间顺序

6. **"地缘决定论的局限"——秦统一的必然性 vs 偶然性**
   - 修正模型：统一天下 = f(地缘优势, 改革能力, 战略正确性, 历史机遇)
   - 秦：35分 vs 齐：21分 vs 晋：16分 vs 楚：21分
   - 反事实推理：如果齐国商鞅变法？如果晋国未分裂？
   - 结论：地缘优势是必要条件，但非充分条件；改革能力 > 地缘优势

7. **"会盟政治的交易成本"——霸主制度的经济学解释**
   - 数学模型：会盟成本 C_霸主维持 + C_meeting × k < 战争成本 C_war × N(N-1)/2
   - 霸主提供的公共产品：安全保障、纠纷调解、秩序维护、对外征讨
   - 诸侯支付的费用：会盟成本、贡品、军事义务、主权让渡
   - 现代验证：联合国、北约、企业联盟（阿里巴巴生态）

8. **"孔子作《春秋》的元认知"——史官如何影响历史**
   - 元认知：孔子意识到"记录历史的方式会影响历史记忆，从而影响未来"
   - 褒贬法：通过用词选择表达褒贬（"弑"vs"卒"、"克"vs"杀"）
   - 写作即干预：记录历史 → 评价历史 → 理解历史 → 影响未来
   - 现代验证：媒体塑造历史记忆、社交媒体的"实时《春秋》"、AI生成历史的风险
