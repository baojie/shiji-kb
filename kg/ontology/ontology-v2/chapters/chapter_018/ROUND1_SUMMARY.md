## 已创建的Facts (4个)

### Fact_001: 萧何完整故事
**核心价值**: 展示**年表简记 vs 世家详述**的对比

**关键发现**:
1. **功劳三层次**: 收图书(勋)、举荐韩信(劳)、守关中转漕(功)
2. **功臣排序标准**: 一时之功 vs 万世之功、功狗 vs 功人
3. **功高震主的生存之道**: 让封、自污、以民为名
4. **知人善任**: 临终推荐曹参(素不相能,但知其贤能)

### Fact_002: 曹参vs萧何功劳排序
**核心价值**: 深入分析"功狗vs功人"理论

**关键发现**:
1. **功劳评定三维度**: 战略价值>战术价值、系统价值>局部价值、持续价值>一时价值
2. **封户vs排序差异**: 曹参10,600>萧何8,000(物质),但萧何第一>曹参第二(精神)
3. **守成者智慧**: 萧规曹随、法令既明遵而勿失
4. **高祖平衡术**: 用封户安抚武将,用排序明确战略价值

### Fact_003: 封侯原因与国除原因分类
**核心价值**: 143位功臣的完整统计分析

**关键发现**(基于前50条样本):
1. **从军起点**: 从起沛2人、从起砀7人、从项梁/项羽8人、其他26人
2. **从军身份**: 将军27人、都尉15人、舍人11人、中涓6人
3. **主要战功**: 击秦8人、定齐3人、守关中/蜀1人
4. **国除原因**: 有罪国除16人、无后国除6人、坐酎金5人、谋反被诛4人
5. **"百年存五"规律**: 97年后仅存5/143≈3.5%

### Fact_004: 韩信兴亡录
**核心价值**: 从王→侯→被诛的完整轨迹,揭示"功高震主"悲剧

**关键发现**:
1. **韩信悲剧五重分析**: 性格悲剧、战略悲剧、政治悲剧、时机悲剧、制度悲剧
2. **三次时机**: 汉四年(最佳,拒绝)、汉五年(次优,被动)、汉十年(最差,选择)
3. **韩信vs萧何对比**: 失败vs成功的两种生存策略
4. **"参分天下,鼎足而居"可行性分析**: 汉四年时韩信有实力自立

## 核心规律提炼

### 规律1: 功劳评定的层次理论

```
第1级: 战略支撑(万世之功)
├─ 萧何: 守关中、给军食、转漕
└─ 特点: 系统性、持续性、不可替代

第2级: 全程参战+独立战功
├─ 曹参: 下二国县122,身被七十创
├─ 郦商、周勃、灌婴
└─ 特点: 全程参与,有独立战绩

第3级: 特殊贡献
├─ 张良: 出六奇计(智谋)
├─ 韩信: 大将军,定魏赵齐(战术)
└─ 特点: 关键节点的决定性贡献

第4级: 局部战功
└─ 大多数功臣: 参与某些战役
```

| 第四代+ | 酎金、无后 | 32% | 元鼎五年大规模削藩 |
