# Fact 003: 合纵连横策略全景图

## 概述

"合纵连横"是战国时代最具代表性的外交策略，核心人物是苏秦（合纵）和张仪（连横）。这场博弈持续约40年（前333-293），最终秦国连横策略胜出，为秦统一奠定基础。

**太史公序言核心洞察**：
> "务在彊兵并敌，谋诈用而从衡短长之说起。矫称蜂出，誓盟不信，虽置质剖符犹不能约束也。"

## 合纵连横的本质

### 地缘政治基础

**东西对抗格局**：
- **秦**：关中平原，函谷关天险，地理位置偏西
- **六国**：齐、楚、燕、韩、赵、魏，地理位置偏东

**合纵（Vertical Alliance）**：
- **地理**：南北纵向联合（楚-韩-赵-魏-齐-燕）
- **目标**：六国联合抗秦
- **提出者**：苏秦
- **口号**："合六国之力，西向攻秦"

**连横（Horizontal Alliance）**：
- **地理**：东西横向联合（秦-某国）
- **目标**：秦国各个击破六国
- **提出者**：张仪
- **口号**："事秦则安，背秦则危"

## 苏秦合纵（前333-前317年）

### 第一次合纵（前333年）

**背景**：
- 秦国商鞅变法后日益强大（前356-338）
- 秦惠文王即位（前337），继续东扩
- 六国单独无法对抗秦

**苏秦策略**：
1. **燕文公二十八年（前334）**：苏秦首先说服燕昭王（年表记载："苏秦说燕"）
2. **游说六国**：燕→赵→韩→魏→齐→楚（南北纵向）
3. **赵武灵王为纵约长**：以赵国为盟主
4. **六国合纵成功**：佩六国相印

**成果**：
- 六国歃血为盟，约定不单独与秦议和
- **"秦十五年不敢出函谷关"**
- 年表记载：从前333到前318年，秦国确实未大规模东进

**苏秦巅峰**：
- 佩六国相印，是战国时代外交家的顶峰
- 《史记·苏秦列传》："苏秦既约六国从亲，归赵，赵肃侯封为武安君，乃投从约书于秦。"

### 第一次合纵瓦解（前318年）

**瓦解原因**：
1. **六国利益分歧**：
   - 齐、楚争霸，互不信任
   - 韩、魏、赵三晋虽同源，但也有矛盾
   - 燕在最北，秦威胁感最小
2. **秦国外交离间**：
   - 秦惠文王派使者分别游说六国
   - 许以小利，破坏合纵
3. **苏秦个人原因**：
   - 在齐国遭暗杀（前317），合纵失去主导者

**年表记录**：
- 秦惠文王更元八年（前317）："五国共攻秦，至盐氏不胜而去"
- 这次合纵攻秦失败，标志着第一次合纵瓦解

## 张仪连横（前328-前309年）

### 张仪出道（前328年）

**背景**：
- 张仪与苏秦同为鬼谷子学生
- 苏秦成功后，张仪入秦
- 秦惠文王拜张仪为相（前328）

**年表记录**：
- 秦惠文王更元十年（前315）："张仪相"
- 秦惠文王更元十三年（前312）："张仪与齐、楚会啮桑"

### 破楚：欺骗楚怀王（前313年）

**策略**：
1. **许诺商於六百里**：张仪对楚怀王说"楚若与齐绝，秦献商於六百里"
2. **楚怀王信以为真**：与齐闵王断交
3. **张仪反悔**：楚与齐绝后，张仪说"只有六里，不是六百里"
4. **楚怀王大怒**：两次伐秦，均败

**结果**：
- 楚国战败，割让汉中郡（秦占领巴蜀的门户）
- 齐楚联盟破裂，六国无法合纵
- **年表记录**：
  - 秦惠文王更元十三年（前312）："我与楚战，大破之丹阳，斩首八万"
  - 楚怀王六年（前313）："张仪来相"

### 破魏：诱魏献上郡（前330年）

**策略**：
1. **魏国地理劣势**：东有齐，西有秦，四战之地
2. **张仪游说**："魏若献上郡于秦，秦保魏安全"
3. **魏襄王同意**：献上郡15县

**结果**：
- 秦得上郡（今陕西北部），北方防线稳固
- 魏国失去战略要地，国力衰弱
- **年表记录**：
  - 秦惠文王更元八年（前317）："魏纳上郡"
  - 魏襄王七年（前312）："入上郡于秦"

### 破赵：利用三晋矛盾（前309年）

**策略**：
1. **挑拨三晋关系**：赵、魏、韩虽同源，但有领土争议
2. **秦赵结盟**：张仪说服赵国与秦结盟
3. **孤立韩魏**：赵国倒向秦，韩魏无法合纵

**结果**：
- 合纵彻底瓦解
- 秦国可以各个击破

## 连横策略分析

### 核心原理

**1. 利用不对称力量**：
- 秦：1个强国 vs 六国：6个弱国
- 秦单独与任何一国谈判，都占优势
- 六国联合才能与秦抗衡，但联合困难

**2. 离间六国**：
- 挑拨齐楚争霸
- 利用三晋矛盾
- 给予小国利益（如韩、魏）

**3. 许以小利，谋取大利**：
- 商於六百里（假）换楚国与齐绝交（真）
- 保护魏国安全（假）换上郡（真）

**4. 时间优势**：
- 合纵需要六国同时行动，协调困难
- 连横秦国主导，可以逐个击破

## 合纵为何失败

### 1. 结构性劣势

**合纵的困境**（博弈论分析）：
- **囚徒困境**：每个国家都想"搭便车"，让别国与秦作战
- **协调成本高**：六国利益不一致，决策慢
- **秦国只需要破坏一环**：六国中任何一国倒戈，合纵瓦解

**数学模型**：
```
合纵成功概率 = P1 × P2 × P3 × P4 × P5 × P6
（六国都不背叛的概率）

如果每国背叛概率20%，则：
合纵成功概率 = 0.8^6 = 26.2%

秦国只需破坏一国，成功概率 = 1 - 26.2% = 73.8%
```

### 2. 齐楚争霸矛盾

**地缘冲突**：
- 齐：东方霸主，控制山东、淮河流域
- 楚：南方霸主，控制长江中下游
- 两国争夺中原霸权，互不信任

**历史案例**：
- 徐州会盟（前334）：齐、魏相王，楚不满
- 垂沙之战（前301）：齐、韩、魏联合攻楚
- 年表记录：齐湣王九年（前292）："与魏会徐州，诸侯相王"

**张仪策略**：
- 优先破坏齐楚联盟（欺骗楚怀王与齐绝交）
- 合纵失去两个最强国家的支持

### 3. 三晋内部矛盾

**同源异心**：
- 韩、赵、魏同为晋国分裂而来（前403）
- 理论上应最团结，但实际矛盾重重
- 争夺河东、河内领土

**秦国策略**：
- 利用三晋矛盾，各个击破
- 先拉拢赵国，孤立韩魏
- 再拉拢魏国，孤立韩国
- 最后灭韩（前230）

### 4. 地理距离导致威胁感差异

**威胁感排序**：
1. 韩：四面受敌，威胁感最高，最想合纵
2. 魏：四战之地，威胁感高
3. 赵：西有秦，但有胡服骑射，威胁感中
4. 楚：南方大国，秦威胁感中
5. 齐：东方强国，秦威胁感低
6. 燕：最北方，秦威胁感最低

**结果**：
- 威胁感低的国家（齐、燕）不愿为韩、魏出力
- 秦国优先攻击威胁感高的国家（韩、魏），其他国家"隔岸观火"

### 第一阶段：苏秦合纵时期（前333-317）

```

前333-318年：秦十五年不敢出函谷关
前318年：五国共攻秦，至盐氏不胜而去（第一次合纵瓦解）
前317年：苏秦在齐被杀
```

### 第二阶段：张仪连横时期（前328-309）

```
前328年：张仪相秦（秦惠文王更元七年）
前318年：张仪将兵取陕（年表：秦惠文王更元十年）

前313年：张仪欺骗楚怀王，楚与齐绝交

前312年：魏献上郡于秦

前309年：张仪死（年表：魏襄王十年）
```

### 第三阶段：合纵最后一次尝试（前公元前293）

```
前318年：五国攻秦（楚、韩、赵、魏、燕），失败
前293年：伊阙之战，秦白起斩首24万（韩、魏联军）
从此六国无力合纵，秦国各个击破
```

## 远交近攻的兴起（范雎时代）

### 从连横到远交近攻（前266年后）

**背景**：
- 张仪连横成功，但秦国扩张仍缓慢
- 秦昭襄王前期（前306-266），秦国攻楚（南方）、攻齐（东方），战线过长
- 范雎入秦（前266），提出"远交近攻"

**远交近攻vs连横**：

| 维度 | 连横 | 远交近攻 |
|------|------|---------|
| 提出者 | 张仪 | 范雎 |
| 核心策略 | 六国各个击破 | 优先灭近邻，与远国结盟 |
| 攻击顺序 | 不固定（哪国弱攻哪国） | 固定（韩→赵→魏→楚→燕→齐） |
| 同盟对象 | 临时盟友 | 远国（如齐） |
| 战线 | 分散（南攻楚，东攻齐） | 集中（专注中原） |
| 成果 | 瓦解合纵，但未灭国 | 实际灭国（韩前230，赵前228，魏前225） |

**年表记录**：
- 秦昭襄王四十一年（前266）：范雎入秦，为客卿
- 秦昭襄王四十二年（前265）：范雎为丞相，封应侯
- 从此秦国战略从"连横"转向"远交近攻"

## 现代启示

### 1. 联盟的脆弱性（囚徒困境）

**历史规律**：
- 合纵需要六国同时配合，只要一国背叛，联盟瓦解
- 秦国只需破坏一环（离间一国），成本远低于六国协调成本

**现代应用**：
- **企业联盟**：行业联盟（如OPEC石油联盟）容易被破坏，需要强制执行机制
- **政治联盟**：北约、欧盟等需要核心国家（如美国、德国）维持，否则易瓦解
- **个人合作**：多人合作项目，只要一人退出，整体崩溃（需要契约约束）

**破局之道**：
- 建立"超国家机构"（如欧盟委员会）
- 签订严格的违约惩罚机制
- 文化认同（如"六国同文"运动，但战国未做到）

---

### 2. 不对称力量的外交策略

**历史规律**：
- 强国（秦）：连横（各个击破）
- 弱国联盟（六国）：合纵（联合对抗）
- 强国总是占优势，因为协调成本低

**现代应用**：
- **国际关系**：美国（强国）vs 第三世界联盟（弱国）
- **商业竞争**：阿里巴巴（强）vs 中小电商联盟（弱）
- **职场**：老板（强）vs 员工联盟（弱，如工会）

**弱国破局**：
- 建立制度化联盟（如欧盟、ASEAN）
- 拥有独特资源（如OPEC控制石油）
- 地理优势（如瑞士中立）

---

### 3. 外交的本质是"信息不对称+利益交换"

**历史规律**：
- 张仪成功的核心：信息不对称（欺骗楚怀王）+ 利益诱惑（商於六百里）
- 苏秦成功的核心：信息传递（让六国意识到秦威胁）+ 共同利益（抗秦）

**现代应用**：
- **商业谈判**：信息优势+利益交换
- **销售**：客户不了解产品（信息不对称）+许以利益（折扣）
- **个人关系**：情报优势+互惠原则

---

### 4. 地缘决定外交优先级

**历史规律**：
- 远国威胁感低，近国威胁感高
- 齐、燕对秦威胁感低，不愿为韩、魏出力
- 范雎"远交近攻"：与齐结盟，专攻韩、赵、魏

**现代应用**：
- **企业竞争**：优先打击近邻竞争者（同城、同行业）
- **个人职场**：优先处理身边关系（同部门），远处关系次要
- **国际关系**：中国"周边外交优先"
