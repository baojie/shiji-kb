# 汉兴以来诸侯王年表 — 知识图谱

《汉兴以来诸侯王年表》记录汉高祖元年至武帝太初年间(前206-前96)诸侯王的分封、迁徙、废立与削藩全过程,见证了从"分天下半"到中央集权的历史转折。

**SKU总数**: 2个 (1 Fact + 1 Skill) | **跨域洞察**: 6条 | **版本**: v2.0

---

## 章节概览

### 时间跨度
- **高祖元年**(前206) → **太初二年**(前103)
- 跨度: 约103年
- 涵盖: 高祖、惠帝、高后、文帝、景帝、武帝六朝

### 核心内容

**太史公序言**:
> "汉兴,序二等。高祖末年,非刘氏而王者,若无功上所不置而侯者,天下共诛之。高祖子弟同姓为王者九国,虽独长沙异姓,而功臣侯者百有余人。"

**三大历史阶段**:

1. **高祖分封**(前206-前195)
   - 初封异姓王:楚、齐、荆、淮南、燕、赵、梁等
   - 逐步剪除异姓王:韩信、英布、卢绾等
   - 确立"非刘氏不王"原则
   - **分天下半**:诸侯国占据半壁江山

2. **诸吕乱政**(前195-前180)
   - 高后专权,封诸吕为王
   - 文帝即位,诛诸吕,复刘氏
   - 淮南厉王骄恣杀人,被废徙死

3. **削藩与集权**(前180-前96)
   - **七国之乱**(景帝三年,前154):吴楚联合七国起兵,三月被平定
   - **推恩令**(武帝元朔二年,前127):诸侯王得推恩分子弟,诸侯国自动分化
   - 诸侯国逐步缩小:齐分为七,赵分为六,梁分为五,淮南分为三

### 表格结构

**横轴**: 年份(从高祖元年至太初年间)
**纵轴**: 诸侯国(楚、齐、荆/吴、淮南、燕、赵、梁、淮阳、代、长沙等)

**记录内容**:
- 初王某某元年
- 某王薨/废/诛
- 国除为郡/分为某国
- 来朝/反叛等重大事件

## 核心数据

### 诸侯王数量变化

**高祖时**:
- 初封异姓王: 7-8国(楚、齐、荆、淮南、燕、赵、梁等)
- 逐步剪除→仅余长沙1国
- 改封同姓王: 9国

**文帝时**:
- 淮南分为三(淮南、衡山、庐江)
- 齐分为七(齐、城阳、济北、济南、菑川、胶西、胶东)
- 诸侯国增至约20国

**景帝时**:
- 七国之乱平定后,国除者7国
- 复置部分诸侯国
- 梁分为五,赵分为六

**武帝时**:
- 推恩令后,诸侯国不断细分
- 数量增加,但面积缩小
- 中央集权逐步确立

### 七国之乱数据

**起兵时间**: 景帝三年正月甲子(前154)
**平定时间**: 三月(前154)
**持续时间**: 约3个月

**参与诸侯**:
1. 吴王刘濞(主谋,兵力二十余万)
2. 楚王刘戊
3. 赵王刘遂
4. 胶西王刘卬
5. 胶东王刘雄渠
6. 菑川王刘贤
7. 济南王刘辟光

**未参与但被诱**:
- 齐王后悔饮药自杀
- 济北王被劫,未发兵
- 淮南王相守城不听王令

**结局**:
- 七王皆死(自杀或被杀)
- 国除为郡或改封
- 汉廷完胜,中央集权强化
