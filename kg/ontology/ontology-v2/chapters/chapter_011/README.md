## 章节概述

《孝景本纪》是《史记》第十一篇本纪，记载了汉景帝刘启（前157—前141年在位）十六年的统治历程。刘启是汉文帝之子，继承并发展了文帝开创的"文景之治"，其统治期间最重要的政治事件是平定以吴楚七国为首的诸侯叛乱（七国之乱，前154年）。景帝在政治上削弱诸侯势力，在经济上继续推行轻徭薄赋政策，为武帝时期的鼎盛局面奠定了坚实基础。

本纪在史记知识体系中记载相对简略，但对于理解汉代中央集权化进程、诸侯国削藩、官制改革等问题具有重要价值。

---

## 主要人物和事件

**核心人物**：
- **汉景帝刘启**：主角，文帝之子，七国之乱的平定者
- **晁错**：景帝的主要谋臣，提出削藩策，后被腰斩以安抚诸侯
- **周亚夫**：汉军主帅，指挥平定七国之乱的军事行动
- **吴王刘濞**：七国之乱的主要发动者
- **王皇后（王娡）**：景帝皇后，武帝之母
- **栗太子刘荣**：景帝废太子，后被逼死

**核心事件**：
1. **七国之乱**（前154年）：吴楚等七国以"清君侧，诛晁错"为名发动叛乱，周亚夫三月平定
2. **诛晁错**：景帝在叛乱发生后腰斩晁错，试图平息叛乱（失败之举）
3. **削藩成功**：七国之乱平定后，景帝大力削弱诸侯王权力，汉廷中央权威大幅提升
4. **官制改革**：景帝多次更改官职名称，推进礼制规范化
5. **废立太子**：景帝废栗太子刘荣，改立王美人之子刘彻为太子
