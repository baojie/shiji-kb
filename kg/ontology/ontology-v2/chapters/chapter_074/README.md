# Chapter 074: 孟子荀卿列传

## 章节概述

《孟子荀卿列传》以孟子开篇，以荀卿收尾，中间穿插驺衍（阴阳家）、稷下先生群像（淳于髡、慎到、田骈、接子、环渊、驺奭等）及附传人物。

本传实质上是一部战国思想市场的群像记录：面对同一批诸侯"买家"，不同学派以不同方式争取接受与支持，最终各有不同命运。孟子以仁义之道游说诸侯而不得用，驺衍以宏大神秘学受尽礼遇，荀卿以综合批判立于稷下祭酒之位。

## 核心人物

### 孟子（孟轲）
- **出身**：驺人，受业于子思之门人
- **游说**：先事齐宣王（不用），后适梁（梁惠王不果所言），梁惠王问"何以利吾国"
- **被评为**：迂远而阔于事情
- **著作**：退而与万章之徒作《孟子》七篇

### 驺衍
- **出身**：齐（三驺子之一）
- **思想**：阴阳消息、五德转移、大九州说
- **待遇**：梁惠王郊迎，赵平原君侧行撇席，燕昭王拥彗先驱筑碣石宫
- **特点**：言语闳大不经，然归结于仁义节俭

### 荀卿（荀况）
- **出身**：赵人，年五十来齐游学
- **地位**：稷下学宫三为祭酒，最为老师
- **仕途**：春申君用为兰陵令，春申君死后废，因家兰陵
- **著作**：数万言（今存《荀子》32篇）
- **弟子**：李斯（后相秦）、韩非（法家集大成者）

### 淳于髡
- **出身**：齐
- **风格**：博闻强记，学无所主，承意观色为务
- **名言**：梁惠王心在驾驭或音声时沉默不言，待王心在时"壹语连三日三夜无倦"
- **结局**：惠王欲以卿相位待之，谢而去，终身不仕

### 其他稷下先生
- 慎到（赵）：黄老道德，著十二论
- 田骈、接子（齐）：黄老道德
- 环渊（楚）：黄老道德，著上下篇
- 驺奭（齐）：阴阳家，文辞华美（雕龙奭）

## 核心事件与主题

### 1. 孟子游说不遇
- 仁义之道与战国攻伐之时代精神根本冲突
- 梁惠王"迂远而阔于事情"的评语
- 退而著书，转向百年遗产策略

### 2. 驺衍的成功
- 将宇宙宏论包装成娱乐+神圣合法性
- 各国诸侯礼遇远超孔子、孟子
- 太史公暗示其或有更深层目的（牛鼎之意）

### 3. 稷下学宫的百家争鸣
- 齐国官方支持下的思想超市
- 各家学者同台，黄老/儒/阴阳/杂家并列
- 荀卿以综合批判姿态成为最权威学者

### 4. 荀卿的历史悖论
- 儒家学者，弟子却是韩非与李斯（法家人物）
- 荀卿主张礼法并重，弟子发展为纯粹法家
- 师徒思想的背离是战国思想演化的缩影

## 核心主题

### 1. 供需错位：真理与时代需求的张力
- 孟子的仁义是真理，但不是战国时代的迫切需求
- 驺衍的奇论满足了诸侯的精神需求（合法性叙事）
- 市场接受不等于价值高低

### 2. 两种遗产策略
- 著书传世（孟子）vs 体制内影响（荀卿）
- 各有优劣，取决于时代条件和个人选择

### 3. 太史公的感慨："利诚乱之始"
- 读孟子至"何以利吾国"叹息不已
- 认为好利是乱之根源，孟子对此的批判是深刻的
- 但时代选择了利，而非仁义

## 太史公评价

> 余读孟子书，至梁惠王问"何以利吾国"，未尝不废书而叹也。曰：嗟乎，利诚乱之始也！夫子罕言利者，常防其原也。故曰"放于利而行，多怨"。自天子至于庶人，好利之弊何以异哉！

太史公将孟子批利之语作为本传开篇的感慨，显示他对孟子价值判断的认同，也对时代以利为先的惋惜。
