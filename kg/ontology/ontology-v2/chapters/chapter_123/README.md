## 核心主题

### 1. 张骞凿空
- 第一次出使：十三年、百馀人出二人归、持汉节不失
- 第二次出使：战略构想（断匈奴右臂）、分遣副使至八国
- 历史地位："张骞凿空，其後使往者皆称博望侯"

### 2. 西域地理
- 八大国详述：大宛、乌孙、康居、奄蔡、大月氏、安息、条枝、大夏
- 水系：河源出于窴，盐泽，西海
- 交通要道：楼兰、姑师（当空道）

### 3. 李广利伐宛
- 第一次失败：补给线未建立、六千骑至郁成大败
- 第二次成功：六万人、断水围城、宛人杀王求和
- 战略成本：四岁、万馀人归还（vs 六万人出发）

### 4. 基础设施建设
- 军事：令居、酒泉郡、亭鄣至玉门、居延休屠
- 屯田：仑头田卒、使者护田积粟
- 外交：使者相望於道、联姻（江都翁主）

## 历史意义

### 丝绸之路开辟
- 张骞凿空：从无到有的探路
- 基础设施：从敦煌到玉门的补给线
- 外交网络：西北国始通於汉

### 汉匈战略态势改变
- 断匈奴右臂：浑邪王降，河西走廊开通
- 乌孙联盟：匈奴与汉竞争性联姻
- 西域威慑：伐宛后"宛以西皆自以远尚骄恣"态度转变

### 文化交流
- 汉朝输入：蒲陶、苜蓿、汗血马
- 汉朝输出：丝、漆、铸钱技术
- 文化展示：觳抵奇戏、府藏参观
