## 跨领域洞察(Eureka)

本章提取出8条跨领域洞察:

1. **知识断层的修复成本**:秦焚书导致经学近乎灭绝,汉初重建需要数代人努力;知识保护的边际成本远低于重建成本
2. **制度化传承vs个人魅力传承**:武帝设立博士弟子制度,将经学从个人师承转为国家制度,确保知识传承的稳定性
3. **权力与学术的博弈**:辕固生与窦太后的冲突显示,学术立场与政治权力的张力需要智慧化解,而非正面对抗
4. **考试制度的双重功能**:年度考试既是选拔机制,也是激励机制,将学习与职业晋升挂钩
5. **地方推荐的信息筛选**:郡国推荐+中央考试的两级筛选,解决大规模人才选拔的信息不对称问题
6. **学派竞争的积极作用**:齐诗、鲁诗、韩诗三家并立,促进经学解释的多元化和深化
7. **经典数量的幸存者偏差**:伏生29篇vs原有数十篇,后世所见经典并非全貌,而是劫难后的幸存样本
8. **师承谱系的声誉传递**:申公→王臧/赵绾→武帝重儒,师承关系成为知识权威的证明链条
