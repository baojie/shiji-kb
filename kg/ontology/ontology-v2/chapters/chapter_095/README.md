## 章节主题

### 核心主题

1. **草根崛起**：三位主角均出身平民（屠狗、马厩官、贩缯），汉初乱世提供了出身解耦的机会
2. **忠诚的层次**：樊哙（鸿门宴）和夏侯婴（救子女）展示了不同形态的忠诚——冲锋式忠诚 vs 结果忠诚
3. **骑兵时代**：灌婴的故事展示了骑兵在楚汉战争中的决定性作用
4. **功臣命运**：高祖晚年欲诛樊哙，陈平周勃未从命，展示了功臣在皇权下的脆弱处境
