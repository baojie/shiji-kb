## 章节概述

《吕太后本纪》是《史记》第九篇本纪，记载了汉高祖皇后吕雉（吕太后）在位期间（前195—前180年）的统治历程。吕太后是中国历史上第一位临朝称制的女性统治者，执政长达十五年。本纪详述了她如何在高祖驾崩后主导朝政、打压异姓功臣、残害戚夫人母子、大肆封立诸吕为王，直至诸吕被周勃、陈平等功臣集团诛灭的全过程。

本章在史记知识体系中具有特殊价值：它是研究汉初权力结构、外戚政治、女性执政的核心文献，也是理解汉初宫廷斗争与政治生态的关键节点。

## 主要人物和事件

**核心人物**：
- **吕太后（吕雉）**：主角，临朝称制，诸吕乱政的推手
- **戚夫人**：高祖宠妃，被吕太后砍去手足、挖眼做成"人彘"
- **刘如意（赵王）**：戚夫人之子，被吕太后毒杀
- **吕禄、吕产**：诸吕代表，被功臣集团诛灭
- **周勃、陈平**：汉初功臣，诛吕的主导者
- **代王刘恒**：诸吕被灭后即位，即汉文帝

**核心事件**：
1. **诛杀戚夫人**：吕太后将戚夫人砍断四肢、挖去眼睛、熏聋耳朵，置于厕所中，称为"人彘"
2. **诸吕擅权**：高后陆续封吕氏诸人为王，违背高祖"非刘氏不得王"的白马之盟
3. **七月政变**：高后驾崩后，周勃、陈平联合齐王发兵，迅速诛灭诸吕，迎立代王刘恒
