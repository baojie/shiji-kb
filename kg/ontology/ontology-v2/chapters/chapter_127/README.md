# 日者列传（第127篇）

## 章节概述

《日者列传》是《史记》中最短的列传之一，主要记录汉代占卜者（"日者"）的故事。核心内容是司马季主与宋忠、贾谊的一场辩论——两位儒家官员对占卜者抱着轻蔑态度，而司马季主则以犀利的辩驳反将一军，证明占卜者比所谓的"正直官员"更诚实、更有道德。

"日者"是古代以占卜为业的人，因其通过观察日辰（时日）推算吉凶而得名。

## 核心主题

1. **占卜的哲学辩护**：司马季主为占卜者的职业正名，将占卜视为在不确定性中寻求导引的合理方式
2. **对仕途的批评**：通过司马季主之口，批评汉代官僚体系中"仕而不能明君"的腐败现象
3. **知识与实践的张力**：读书人（宋忠、贾谊）的傲慢与占卜者的实践智慧的对比
4. **命运与选择**：占卜的本质问题——人能否通过预测来改变命运

## 主要人物

- **司马季主**：长安著名占卜者，博学多识，能以《易》理讲解占卜原理
- **宋忠**：汉代儒学博士，去拜访司马季主的官员之一
- **贾谊**：著名政论家，年轻时随宋忠一同造访司马季主（此处贾谊为年轻时的记录）

## 史学价值

《日者列传》保存了汉代占卜文化的珍贵记录，同时通过司马季主的辩论，展示了汉代士人对"正途"与"旁门"的价值观争论。司马迁将日者单独立传，本身就是对这一边缘群体的历史认可。
