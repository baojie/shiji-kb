## 历史背景

建元以来（前140-前90）是汉武帝对外大规模扩张时期。通过军功封侯激励机制，支撑了对匈奴、南越、东越、朝鲜的连年征战，汉朝疆域扩张至史上最大。这一时期封侯模式从"追封祖辈"转向"现役军功"，标志着从封建贵族制向军功官僚制的转型。
