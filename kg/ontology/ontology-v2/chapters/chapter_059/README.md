## 章节概述

《五宗世家》是《史记》第五十九篇，记述汉景帝十三个儿子封王的生平与国运。"五宗"指景帝诸子的五个母系：栗姬（刘荣、刘德、刘阏于）、程姬（刘馀、刘非、刘端）、贾夫人（刘彭祖、刘胜）、唐姬（刘发）、王夫人兒姁（刘越、刘寄、刘乘、刘舜）。

本章最核心的价值不在于单纯的世系记录，而在于：
1. **人物群像**：十三王性格各异，从好儒的献王到贼戾的胶西于王，构成汉代诸侯王的完整类型谱
2. **制度史**：太史公总论揭示吴楚之乱前后诸侯权力的根本性转变——从自治政权到仅食租税的"食利者"
3. **博弈案例**：赵王彭祖与胶西于王各以不同策略对抗朝廷监督，是研究权力博弈的生动案例

## 章节特点

**《五宗世家》的独特价值**：

1. **群像式写法**：篇幅最长的是赵王彭祖和胶西于王（都是反面典型），最短的是清河哀王（仅记国除），这种笔墨分配本身就是一种评价
2. **制度史意义**：太史公总论精炼地记录了汉代诸侯制度的根本转折，"诸侯贫者或乘牛车也"是对整个西汉诸侯命运的悲凉总结
3. **长沙王的伏笔**：以母微、封贫国的刘发，其后裔九世后出汉光武帝刘秀，史书中最著名的"弱者逆袭"之一
