# 佞幸列传（第125篇）

## 章节概述

《佞幸列传》是《史记》中最特殊的一篇列传——它专门记录以美貌和亲近关系获得帝王宠幸的人物，包括汉高祖的籍孺、汉惠帝的闳孺、汉文帝的邓通与赵同、汉武帝的韩嫣与李延年等。

司马迁在篇首明确点出："非独女以色媚，而士宦亦有之。"——佞幸不仅是女性的专利，男性同样可以以美色取悦君主。这是中国正史中对帝王与男宠关系最直接的历史记录。

## 核心主题

1. **以色事人的本质**：佞幸以容貌和亲近取悦君主，而非以才能或功绩立身
2. **佞幸的命运结构**：依附于一人，随宠主的喜怒或死亡而起落
3. **权力的性别政治**：汉代皇权中的私人亲密关系如何转化为政治资源
4. **司马迁的批评立场**：通过如实记录，暗含对这种权力运作方式的批评

## 主要人物

- **邓通**：汉文帝宠臣，被赐铜山（矿山），得以铸钱，富甲天下，后文帝死而立被抄家
- **韩嫣**：汉武帝少年时的宠臣，因与武帝射猎关系亲密而获赐大量财物
- **李延年**：音乐家，以音乐和姿容得宠，其妹李夫人成为武帝最爱的妃子

## 史学价值

《佞幸列传》是研究汉代皇权私人面向的重要文献，也是理解前现代权力结构中"亲近关系"如何转化为政治资源的经典案例。
