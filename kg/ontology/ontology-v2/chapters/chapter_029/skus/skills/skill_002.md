## 概述

《河渠书》虽短，却记录了完整的水利工程决策机制：从"上书建议"到"天子下问"、"专家论证"、"发卒施工"、"验收使用"、"失败废止"的全过程。本 skill 提炼汉代水利决策的制度性流程，既涵盖成功案例（郑国渠、漕渠、瓠子堵塞），也涵盖失败案例（河东渠田、褒斜漕道），为今日大型公共工程决策提供历史借鉴。

## 何时使用此 Skill

- 策划大型水利、基础设施工程
- 研究古代官僚制度如何运作
- 理解技术官员（"水工"）与朝廷的关系
- 学习项目决策中的风险评估

## 核心决策流程

### 第1步：问题提出

水利工程的触发条件：

#### 类型A：危机触发
- 黄河决口（酸枣、瓠子）
- 连年水旱灾害
- 河道淤塞

#### 类型B：机会触发
- 有新的技术可用（龙首渠井渠法）
- 有特殊政治契机（武帝封禅后）
- 有需要支援的军事行动（北方屯田）

#### 类型C：上书建议
- 水工、地方官提出（如郑国、番系、庄熊罴）
- 这是汉代最常见的触发方式

### 第2步：上书与受理

参考[7]郑当时、[8]番系、[9]张汤、[10]庄熊罴的案例：

1. **上书人身份**：大农（郑当时）、河东守（番系）、御史大夫（张汤）、民间（庄熊罴）
2. **上书内容**：必须包括问题、方案、预期效果、所需人力
3. **受理机关**：皇帝直接或相关大臣

### 第3步：天子下问/专家论证

典型流程：
> 天子以为然，令/发/拜...

**关键决策点**：
- 天子决定是否可行
- 如需专业咨询，下问相关部门
- 张汤"问其事"——即技术咨询

参考郑国渠案例（[4]）：
- 初发觉是间谍，欲杀
- 郑国以"渠成亦秦之利"说服
- "秦以为然，卒使就渠"
- 这一步是最关键的认知升级

### 第4步：确定主持人

主持人选拔：

| 案例 | 主持人 | 身份 |
|------|--------|------|
| 郑国渠 | 郑国 | 韩国水工（专业） |
| 漕渠 | 徐伯 | 齐人水工（专业） |
| 河东渠田 | 番系 | 河东守（行政） |
| 褒斜道 | 张卬 | 汉中守（行政，张汤之子） |
| 瓠子堵塞首次 | 汲黯、郑当时 | 朝臣（行政） |
| 瓠子堵塞二次 | 汲仁、郭昌 | 朝臣 |

**经验**：
- 专业水工主持的工程（郑国渠、漕渠、龙首渠）多成功
- 纯行政官员主持的（河东渠田、褒斜道）多失败
- 关键技术问题需要专业人员

### 第5步：动员与施工

人力动员的规模：

| 工程 | 人力规模 | 工期 |
|------|---------|------|
| 瓠子堵塞（二次） | 数万人 | - |
| 汉漕渠 | 数万人 | 3年 |
| 褒斜道 | 数万人 | - |
| 河东渠田 | 数万人 | 数年 |
| 龙首渠 | 万余人 | 十余年 |
| 郑国渠 | （不详） | （不详） |

**原则**：
- 一般工程数万人
- 长距离工程（龙首渠）需十余年
- 紧急工程（瓠子堵塞）可短期集中

### 第6步：天子亲临

最大型工程需要皇帝亲临：
- 武帝堵瓠子："还自临决河，沈白马玉璧于河，令群臣从官自将军已下皆负薪"
- 皇帝亲临的作用：
  - 打破官员推诿
  - 解决物资不足（砍淇园之竹）
  - 激励士气（群臣自将军以下负薪）

### 第7步：就地解决困难

工程中的应变：
- 瓠子工程薪柴不足 → 砍淇园之竹
- 龙首渠"岸善崩" → 凿井连通
- 漕渠工程复杂 → 齐人水工徐伯"表"（测量规划）

**灵活应变是成功的关键**。

### 第8步：效益评估

工程完成后的评估：

#### 评估维度

1. **技术通过**：水通、船通、渠通
2. **经济效益**：灌溉面积、漕运量
3. **社会效益**：民田获益、军粮充足
4. **政治效益**：国力彰显

#### 评估结果

- 郑国渠："用注填阏之水，溉泽卤之地四万馀顷，收皆亩一钟"——大成功
- 漕渠："以漕，大便利，其后漕稍多，而渠下之民颇得以溉田"——成功
- 褒斜道："道果便近，而水湍石，不可漕"——部分成功部分失败
- 河东渠田："河移徙，渠不利，则田者不能偿种"——失败
- 瓠子堵塞："梁、楚之地复宁，无水灾"——成功

### 第9步：失败工程的处理

失败工程如何处置：

- **河东渠田**：废，予越人，令少府以为稍入
- **褒斜道**：虽不可漕，但"少阪，近四百里"——部分保留
- **新垣平望气**：伪被诛——人受罚但制度保留

**原则**：失败人员追责，基础设施尽量保留利用。

### 第10步：纪念与制度化

重大工程后的纪念：
- 瓠子堵塞后："筑宫其上，名曰宣房宫"
- 武帝作《瓠子歌》——文学纪念
- 郑国渠"因命曰郑国渠"——以人名命名

制度化：
- 形成常设的水利管理机构
- 将工程经验写入史册（《河渠书》）
- 作为后世工程的范例

## 决策点

### 何时上马？

评估标准：
- 技术上可行吗？
- 经济上合算吗？
- 政治上必要吗？
- 民力能承担吗？

### 选谁主持？

- 纯技术工程：优先水工（专业）
- 军事相关：武将（执行力）
- 综合工程：行政官员（协调力）

### 失败如何止损？

- 损失过大立即废止（河东渠田）
- 部分有效保留（褒斜道）
- 个别关键人员替换（瓠子首塞失败后改换汲仁、郭昌）

### 利益相关者如何处理？

以田蚡阻挠堵塞为例：
- 识别出"鄃邑食奉"是其阻挠动机
- 必要时借皇帝亲临突破
- 不直接对抗而是等待政治时机（封禅后）

## 预期结果

- 工程成功完成，效益超过投入
- 或失败后及时止损，不累加损失
- 形成后世可参照的决策范式
- 积累水利工程的专业经验

## 潜在风险

### 风险1：规划不足
- 河东渠田：未预见黄河改道
- 褒斜道：未预见水文湍石

### 风险2：利益扭曲
- 田蚡阻塞瓠子堵塞：私邑利益凌驾天下
- 方士干扰：以"天事"为借口

### 风险3：技术错配
- 行政官员主持技术工程
- 没有专业水工指导

### 风险4：政治周期
- 朝廷注意力分散
- 皇帝不能亲临督工

## 成败因素总结

### 成功要素

1. 顺势（禹法：疏通分流）
2. 就地（淇园之竹）
3. 专业（水工主持）
4. 亲临（皇帝督工）
5. 灵活（工程中应变）

### 失败因素

1. 逆势（河东强引河）
2. 远距（褒斜百余里车转）
3. 外行（行政官主工）
4. 松懈（朝廷不督查）
5. 僵化（死守原定方案）
