## 章节概述

《孝武本纪》是《史记》第十二篇本纪，记载汉武帝刘彻（前141—前87年在位）的统治历程。然而，此篇在《史记》中颇为特殊：原文极为简短，主要记述封禅祭祀、求仙问道诸事，对武帝最重要的政治、军事、经济改革几乎一字未提，与武帝波澜壮阔的五十四年统治极不相称。学界普遍认为，司马迁因遭受宫刑之辱（与武帝有关），有意省略武帝功绩，后世由褚少孙等人补充完善。

尽管本纪文本简略，本章知识库通过整合《史记》其他相关传记（卫将军骠骑列传、平准书、封禅书等），对汉武帝时代进行了系统性知识提取，涵盖北击匈奴、张骞通西域、独尊儒术、盐铁官营等重大历史事件。

---

## 主要人物和事件

**核心人物**：
- **汉武帝刘彻**：主角，中国历史上最具影响力的皇帝之一
- **卫青、霍去病**：汉军名将，北击匈奴的主要统帅
- **张骞**：外交使节，两次出使西域，开拓丝绸之路
- **董仲舒**：儒家思想家，提出"独尊儒术、罢黜百家"
- **桑弘羊**：理财大臣，主持盐铁官营、均输平准等财政改革
- **司马迁**：本书作者，因李陵案受宫刑，与武帝恩怨复杂

**核心事件**：
1. **独尊儒术**（前136年）：接受董仲舒建议，罢黜百家，表彰六经，儒学成国家意识形态
2. **北击匈奴**：三次大规模军事行动，卫青、霍去病深入漠北，基本解除匈奴威胁
3. **张骞通西域**（前138年、前119年）：两次出使，建立汉朝与西域的政治经济联系
4. **封禅泰山**（前110年）：武帝五次封禅，彰显天命正统
5. **盐铁官营**：实行盐铁国家专卖，充实国家财政，支撑军事扩张
