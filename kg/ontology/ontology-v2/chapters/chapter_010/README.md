## 章节概述

《孝文本纪》是《史记》第十篇本纪，记载了汉文帝刘恒（前180—前157年在位）二十三年的统治历程。刘恒原为代王，诸吕被灭后以外藩之身入主长安，开创了中国历史上著名的"文景之治"。他以黄老思想为指导，推行无为而治、与民休息的政策，大幅减轻赋税徭役，废除肉刑，提倡节俭，使西汉王朝从立国之初的贫弱状态逐步走向富强。

本章是研究汉代仁政思想、刑法改革、皇权与功臣关系的核心文献，也是中国历史上"德治"理念最具代表性的实践案例。

---

## 主要人物和事件

**核心人物**：
- **汉文帝刘恒**：主角，以宽仁节俭著称，文景之治开创者
- **缇萦**：太仓令淳于意之女，上书请求以自身为奴换父赎罪，感动文帝废除肉刑
- **贾谊**：文帝时期著名政论家，提出诸多改革建议
- **周勃、陈平**：功臣集团代表，辅佐文帝稳定政局
- **窦皇后**：文帝皇后，崇尚黄老之学

**核心事件**：
1. **代王入主**：诸吕被灭，大臣迎立代王刘恒，登基为文帝
2. **缇萦救父**：缇萦上书感动文帝，文帝下诏废除黥、劓、斩趾等肉刑
3. **除肉刑**：文帝十三年废除肉刑，以笞刑代替，开刑法人道化先河
4. **节俭示范**：文帝在位期间不扩建宫室，露台议费百金而止，以身作则节俭
5. **与民休息**：多次减免田租，最低减至三十税一，促进农业恢复
