## 知识单元概览

### 核心对比：刘邦 vs 项羽

| 维度 | 刘邦 | 项羽 |
|------|------|------|
| 个人武力 | 普通 | 力能扛鼎 |
| 军事才能 | 屡战屡败 | 战无不胜 |
| 用人能力 | 知人善任（萧何、张良、韩信） | 优柔寡断，不善用人 |
| 战略眼光 | 长远规划，能屈能伸 | 短视，过度自信 |
| 处理失败 | 屡败屡战，越挫越勇 | 拒绝反思（"天亡我，非战之罪"） |
| 收买人心 | 约法三章，秦人大喜 | 屠城坑卒，秦人大失望 |
| 最终结果 | 建立汉朝，四百年基业 | 乌江自刎 |

### 关键洞察

**刘邦自评**："夫运筹策帷帐之中，决胜于千里之外，吾不如子房。镇国家，抚百姓，给馈饷，不绝粮道，吾不如萧何。连百万之军，战必胜，攻必取，吾不如韩信。此三者，皆人杰也，吾能用之，此吾所以取天下也。项羽有一范增而不能用，此其所以为我擒也。"
