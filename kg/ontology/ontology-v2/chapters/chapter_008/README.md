## 知识单元概览

### 核心对比：大汉系统 vs 西楚霸王

| 核心维度 | 刘邦（大汉系统） | 项羽（古典英雄） |
|------|------|------|
| **权力合法性原则** | 依靠政治与神权包装，脱离贵族血分 | 贵族后裔世代相传的基础 |
| **系统容错率** | **高**：萧何大后方补给，主帅无数次大败仍能重组 | **极低**：容不下一次战略失利，失败即自杀 |
| **利益分配与威权** | 极其慷慨地封侯裂土，利益最大化共同体 | 极其吝啬，威慑而非拉拢（印刓弊不忍予） |
| **对批评的容忍** | **极高理性容忍**：唾面自干，谁有理听谁（下邑画策） | **极高傲感**：不能容忍任何冒犯（亚父被离间） |
| **最终历史意义** | 终结血统门第，开启二千年官僚平民帝国基业 | 中国历史上最后一个纯粹的军事贵族挽歌 |
