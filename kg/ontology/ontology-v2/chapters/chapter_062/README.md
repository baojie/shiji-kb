## 核心主题

### 管仲部分

#### 1. 管鲍之交
- **五次误解与理解**：
  - 分财多取 → 非贪，知我贫
  - 谋事失败 → 非愚，知时运不济
  - 三仕三逐 → 非不肖，知我不遭时
  - 三战三走 → 非怯，知我有老母
  - 幽囚受辱 → 非无耻，知我耻功名不显

- **鲍叔举荐**：
  - 以身下之，主动让位
  - 子孙世禄，反因"知人"获美名
  - "天下不多管仲之贤而多鲍叔能知人也"

#### 2. 管仲改革
- **核心理念**：
  - 仓廪实而知礼节，衣食足而知荣辱
  - 四维不张，国乃灭亡
  - 下令如流水之原，令顺民心

- **政治策略**：
  - 善因祸而为福，转败而为功
  - 知与之为取，政之宝也
  - 贵轻重，慎权衡

- **外交智慧**：
  - 南袭蔡 → 转为伐楚责贡（私怨转公义）
  - 北征山戎 → 转为令燕修政（征服转指导）
  - 柯之会 → 守信诸侯（失地换信誉）

#### 3. 齐桓霸业
- 九合诸侯，一匡天下
- 管仲卒，齐国遵其政，常强于诸侯
- 后百余年而有晏子焉

### 晏婴部分

#### 1. 个人品德
- **节俭自律**：食不重肉，妾不衣帛
- **政治风格**：危言危行，节俭力行
- **朝廷表现**：
  - 君语及之，即危言
  - 语不及之，即危行
  - 国有道，即顺命；无道，即衡命

#### 2. 核心事迹

**赎越石父知礼**：
- 晏子赎人后未行礼
- 越石父教育："知己而无礼，固不如在縲绁之中"
- 晏子醒悟，延入为上客
- 启示：知己 = 理解 + 尊重

**御者自省**：
- 御者妻观察：晏子身高六尺却"志念深，常有以自下者"
- 对比：御者身高八尺却"意气扬找，自以为足"
- 御者自抑损，晏子荐为大夫
- 启示：自我认知需要外部镜像

#### 3. 历史评价
- 太史公："假令晏子而在，余虽为之执鞭，所忻慕焉"
- 三世显名于诸侯
- 进思尽忠，退思补过

- 管仲代表"功业型政治家"：改革、霸业、制度化
- 晏婴代表"道德型政治家"：节俭、直谏、个人美德
- 两者互补，共同构成齐国贤相典范

### 2. 知己主题
- 管鲍之交：理解动机而非评判结果
- 越石父：知己 = 理解 + 尊重
- 御者自省：自我认知需要外部镜像
- 三个故事从不同角度诠释"知己"

### 3. 政治智慧
- 管仲：转化策略（因祸为福、知与之为取）
- 晏婴：进谏艺术（危言危行、顺命衡命）
- 两种智慧适用不同场景

### 4. 现代应用价值
- 人才识别：鲍叔的根因分析法
