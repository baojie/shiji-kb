# 晏婴的品德与政治智慧

## 概述

晏平仲婴，莱之夷维人，历仕齐灵公、庄公、景公三朝，以节俭力行、智慧进谏、知人善任著称于诸侯，是继管仲之后齐国又一位贤相。

## 核心内容

### 基本信息

- **姓名**：晏平仲婴
- **籍贯**：莱之夷维（今山东莱州）
- **任期**：历仕齐灵公、庄公、景公三世
- **身高**：长不满六尺（约1.38米）
- **历史地位**："后百余年而有晏子焉"（管仲之后齐国贤相）

### 个人品德

#### 节俭自律

- **饮食**：既相齐，食不重肉（不吃两道肉菜）
- **家庭**：妾不衣帛（妾室不穿丝绸）
- **住房**：生活简朴，不以相国之位奢靡
- **持续性**：三世（历仕三君）显名于诸侯

#### 政治智慧

朝廷表现：
- **君语及之**：即危言（说正直的话）
- **语不及之**：即危行（做正直的事）
- **国有道**：即顺命（顺应时势）
- **国无道**：即衡命（权衡利弊以匡正）

### 核心事迹

#### 一、赎越石父知礼

**事件经过**：
1. **遇贤**：晏子出行遇越石父在囚禁中
2. **赎人**：解左骖（卸下车左侧的马）赎之
3. **失礼**：载归后弗谢（不道谢），直接入闺
4. **请绝**：越石父请求断交
5. **悟错**：晏子惊愕，整理衣冠谢罪
6. **纠正**：延入为上客

**越石父的理论**：
- 不知己时被囚，可以理解（"彼不知我也"）
- 既知己又赎我，是真知己（"夫子既已感寤而赎我，是知己"）
- 知己而无礼，还不如在狱中（"知己而无礼，固不如在縲绁之中"）
- 核心：君子"诎于不知己而信于知己者"

**启示**：知己的完整定义 = 理解（知我）+ 尊重（有礼）

#### 二、御者自省

**故事背景**：
- 晏子为齐相，御者（车夫）是其仆人
- 御者身高八尺（约1.84米），晏子身高不满六尺

**关键场景**：

1. **御者得意**：
   - 拥大盖（把持大伞盖）
   - 策驷马（驾驭四匹马）
   - 意气扬扬，甚自得

2. **妻子观察**：
   - 从门间窥视丈夫
   - 看到其自满姿态
   - 请求离去

3. **对比分析**（妻子的话）：
   - 晏子：长不满六尺 → 身相齐国，名显诸侯 → 志念深矣，常有以自下者
   - 丈夫：长八尺 → 为人仆御 → 意自以为足
   - 结论："妾是以求去也"

4. **自我修正**：
   - 御者"自抑损"（自我克制、谦逊）
   - 晏子发现异常询问
   - 御者如实相告
   - 晏子荐以为大夫

**核心洞察**：
- 真正的高度不在身高，在志向
- 真正的权力不在彰显，在内敛
- 自我认知需要外部镜像
- 谦逊是晋升的阶梯

### 政治风格特点

#### 直言敢谏

- 历仕三君而始终保持独立人格
- "危言危行"——不因位高而违背正直
- 能根据国家有道无道调整策略（顺命/衡命）

#### 知人善任

- 能从仆人的细微变化中发现人才（御者自省）
- 被越石父纠正后立即改正，延为上客
- 举贤不避仇，举贤不避亲

#### 内敛谦和

- 身高不满六尺，不以相位骄人
- "志念深矣，常有以自下者"
- 影响力来自品德而非权势

### 历史影响

#### 太史公评价

- "吾读《管氏》《牧民》《山高》《乘马》《轻重》《九府》，及《晏子春秋》，详哉其言之也"
- "至其谏说，犯君之颜，此所谓'进思尽忠，退思补过'者哉"
- "假令晏子而在，余虽为之执鞭，所忻慕焉"（太史公愿为晏子执鞭）

#### 与管仲对比

| 维度 | 管仲 | 晏婴 |
|------|------|------|
| 时代 | 齐桓公 | 齐灵公、庄公、景公 |
| 成就 | 九合诸侯，一匡天下 | 三世显名于诸侯 |
| 生活 | 富拟于公室，有三归、反坫 | 食不重肉，妾不衣帛 |
| 风格 | 转祸为福，贵轻重 | 危言危行，节俭力行 |
| 著作 | 《管子》（牧民、轻重等） | 《晏子春秋》 |
| 评价 | 孔子小之（未劝桓公王） | 太史公"所忻慕焉" |

### 传世名言

虽本章未直接引用晏子名言，但史记通过事迹传达的智慧：

- 知己的标准：理解+尊重
- 领导的品质：志念深远+常有以自下者
- 进谏的原则：进思尽忠，退思补过
- 识人的方法：观察细微变化，发现自省能力

## 历史意义

### 文化价值

1. **君子标准**：节俭、正直、谦和、智慧
2. **知己之道**：超越物质援助的精神认可
3. **自我修养**：通过外部反馈实现成长
4. **举贤机制**：从日常观察中发现人才

### 现代启示

1. **领导力**：真正的权力来自品德而非职位
2. **人才识别**：关注自省能力和成长潜力
3. **自我认知**：需要他人视角打破盲区
4. **知识分子风骨**：危言危行，三世不改
