# 屈原贾生列传 — 知识图谱

本目录包含《史记·屈原贾生列传》章节的结构化知识提取。

## 章节概述

本传以合传形式呈现两位跨越时代的悲剧文人，主题是才华横溢的忠臣如何被昏庸政治环境所摧毁。

**屈原**（约前340—前278年）：楚国大臣，以忠正著称，因被上官大夫谗毁而被楚怀王疏远，两度放逐。放逐期间作《离骚》《九歌》《天问》《九章》等不朽诗篇。公元前278年秦破郢都，屈原抱石投汨罗江自尽，以死殉道。

**贾谊**（前200—前168年）：西汉初期杰出政论家。18岁以能文著称，21岁任博士，23岁升太中大夫，被汉文帝极力赏识。因绛侯周勃、灌婴等权贵排挤，被贬为长沙王太傅。后召回，任梁怀王太傅。梁怀王坠马死，贾谊自责，33岁抑郁早逝。

司马迁将两人合传，并在传末以自身感受（李陵案受宫刑）寄情，表达对忠才被谗的深切同情。

## 核心人物

- **屈原**：楚国大臣、爱国诗人，中国文学史上第一位留名的大诗人
- **贾谊**：汉初政论家，《过秦论》作者，"不问苍生问鬼神"的典故主角
- **楚怀王**：屈原的君主，轻信谗言疏远屈原，最终客死秦国
- **汉文帝**：欣赏贾谊才华，却因政治压力未能充分重用，只召贾谊问鬼神而非国事
- **周勃、灌婴**：排挤贾谊的权贵，以"年少专权"为由攻击贾谊

## 核心故事

1. **屈原被放逐**：楚怀王信谗，屈原两度遭放逐，作《离骚》等诗篇寄托忠君爱国之情
2. **渔父对话**：屈原在江边遇渔父，渔父劝其随波逐流，屈原坚持"皓皓之白"不染尘埃
3. **投汨罗江**：郢都被破，屈原抱石自沉，端午节由此而来
4. **贾谊天才入仕**：21岁入仕，深受汉文帝赏识，拟委重任却遭权贵攻击
5. **贬谪长沙**：贾谊贬为长沙王太傅，作《鵩鸟赋》以道家达观面对命运
6. **贾谊之死**：梁怀王坠马，贾谊自责"哭泣岁余，亦死矣"，33岁早逝

## 关键概念

| 概念 | 解释 |
|------|------|
| 离骚 | 屈原最重要的长篇诗歌，"离"即罹，"骚"即忧愁，意为遭遇忧愁 |
| 过秦论 | 贾谊政论文，批评秦朝施政之失，核心论断"仁义不施而攻守之势异也" |
| 路漫漫其修远兮 | 《离骚》名句，比喻追求理想的漫长与执着 |
| 不问苍生问鬼神 | 后人对汉文帝召见贾谊只谈鬼神的批评，体现对贾谊才能的惋惜 |
| 攻守异势 | 《过秦论》核心：夺取天下与守卫天下所需策略截然不同 |
