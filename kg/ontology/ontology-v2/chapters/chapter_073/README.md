# Chapter 073: 白起王翦列传

## 章节概述

《白起王翦列传》记载秦国两位最著名的将领：白起与王翦。两人都为秦统一六国立下赫赫战功，但命运截然不同。白起军事天才，却因不懂政治而被赐死；王翦持重谨慎，以"请田自坚"消除猜忌而善终。

本传揭示了战国末期"功臣困境"的深层矛盾：军事才能越高，政治风险越大。

## 核心人物

### 白起（武安君）
- **出身**：郿人
- **服务君主**：秦昭王
- **军事成就**：
  - 伊阙之战：斩首24万
  - 攻楚：拔郢，楚王东走
  - 长平之战：坑杀赵降卒40万
  - 总计：战胜攻取70余城
- **战术特点**：料敌合变，出奇无穷
- **政治失败**：与应侯结隙，拒命被杀
- **道德负担**：临终自认"阬降卒四十万，是足以死"

### 王翦
- **出身**：频阳东乡人
- **服务君主**：秦始皇
- **军事成就**：
  - 灭赵：岁余拔赵，赵王降
  - 灭燕：燕王走辽东
  - 灭楚：虏荆王负刍
  - 子王贲灭魏、齐
- **战术特点**：持重坚壁，养士待机
- **政治智慧**：请田自坚，消除始皇猜忌
- **结局**：善终，子孙三代为将

## 核心事件

### 1. 长平之战（260 BC）
- **起因**：韩守冯亭献上党于赵，秦赵对峙
- **关键转折**：
  - 廉颇坚守（正确策略）
  - 秦反间，赵王换赵括（中计）
  - 白起诱敌深入，断粮道
- **结果**：
  - 围困46日
  - 赵括被射杀
  - 坑杀降卒40万
  - 前后斩首虏45万
- **影响**：赵国元气大伤，秦之暴虐形象

### 2. 白起拒命与被杀（250 BC）
- **背景**：
  - 长平后应侯劝秦王割地议和
  - 白起与应侯有隙
- **邯郸之围**：
  - 秦王欲使白起攻邯郸
  - 白起拒绝：邯郸难攻、诸侯必救、秦国内空
  - 预言全部应验（楚魏救赵，秦军失败）
- **处决**：
  - 免职迁徙
  - 赐死杜邮
  - 临终自省："阬降卒四十万，是足以死"
- **民间反应**："死而非其罪，秦人怜之，乡邑皆祭祀焉"

### 3. 李信惨败与王翦请田（223 BC）
- **李信出征**：
  - 始皇问兵，李信称20万足矣
  - 王翦称非60万不可
  - 始皇用李信，王翦归老
  - 李信初胜后惨败
- **王翦复出**：
  - 始皇亲请，王翦坚持60万
  - 出发前反复请田
  - 解释："秦王怚而不信人，今空秦国甲士而专委于我，我不多请田宅为子孙业以自坚，顾令秦王坐而疑我邪？"
- **灭楚**：
  - 坚壁不战，楚数挑战不出
  - 养士蓄锐，从"投石超距"判断士气
  - 楚军松懈引而东，王翦追击歼灭
  - 杀项燕，虏荆王，平楚地

## 核心主题

### 1. 功臣困境
- **症状**：功高震主，君主猜忌
- **白起案例**：七十余城，功比周召吕望，应侯担心其为三公
- **王翦案例**：空秦国甲士（60万），始皇"怚而不信人"

### 2. 生存策略
- **王翦式**：示弱求财，请田自保，利益绑定
- **白起式**：直言拒命，怏怏不服，情绪外露

### 3. 战略vs政治
- **白起**：战略判断正确（邯郸难攻），但政治失败（拒命被杀）
- **王翦**：战术持重成功（灭楚），但"偷合取容，以至筊身"

### 4. 道德负债
- **坑杀降卒**：白起临终自认罪当
- **长期代价**：秦之暴虐形象，为反秦埋下种子

## 太史公评价

> 鄙语云"尺有所短，寸有所长"。
>
> 白起料敌合变，出奇无穷，声震天下，然不能救患於应侯。
>
> 王翦为秦将，夷六国，当是时翦为宿将，始皇师之，然不能辅秦建德，固其根本，偷合取容，以至筊身。
>
> 及孙王离为项羽所虏，不亦宜乎！彼各有所短也。

**解读**：
- 白起短于政治（不能救患于应侯）
- 王翦短于道德担当（偷合取容）
- 两人都有所短，但各有所长
- 功臣宿命：政治智慧与道德担当的两难
