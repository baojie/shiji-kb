# 技能：持重用兵的智慧

## 核心原则

### 兵力务实判断
**反对虚荣冒进**：
- 李信："不过用二十万人"（年少壮勇，乐观）
- 王翦："非六十万人不可"（老将经验，务实）
- 始皇："王将军老矣，何怯也！"
- 结果：李信惨败，王翦成功

**启示**：
- 经验比锐气更可靠
- 老将的谨慎胜过少年的勇猛
- 坚持专业判断，不迎合上意

### 坚壁待时
**策略步骤**：
1. **初至**：楚悉国中兵拒秦
2. **王翦**：坚壁而守之，不肯战
3. **楚军**：数出挑战
4. **王翦**：终不出

**养士待机**：
- 日休士洗沐
- 善饮食抚循之
- 亲与士卒同食

**时机判断**：
- 使人问："军中戏乎？"
- 对曰："方投石超距"
- 王翦："士卒可用矣"

**决战**：
- 楚数挑战不出，乃引而东（敌方松懈）
- 王翦因举兵追之，令壮士击
- 大破楚军，杀项燕，虏荆王

## 实战要点

### 准备阶段
1. **兵力充足**：坚持足够兵力，不妥协（60万vs20万）
2. **养精蓄锐**：日休士洗沐，善饮食
3. **亲近士卒**：同食同乐，凝聚人心

### 对峙阶段
1. **坚壁不战**：敌挑战不出
2. **观察敌情**：等待松懈
3. **观察己方**：从"投石超距"判断士气可用

### 决战阶段
1. **抓住时机**：敌引而东（撤退）
2. **迅速追击**：举兵追之
3. **猛烈打击**：令壮士击，歼灭战

## 现代应用

### 企业竞争
- **市场进入**：充分准备（资金、团队、时机）
- **竞争策略**：不急于出手，养精蓄锐
- **时机把握**：观察竞争对手松懈时刻
- **决战时刻**：一击必杀，追求歼灭战

### 投资决策
- **资金储备**：坚持足够安全边际（王翦要60万）
- **耐心等待**：不被短期波动诱惑
- **观察信号**：市场情绪、估值水平
- **果断出击**：时机成熟，重仓进入

### 个人发展
- **技能储备**：充分准备再出手
- **职场策略**：不争一时锋芒
- **时机选择**：等待行业/公司机会窗口
- **抓住机遇**：准备好时，全力以赴

## 对比：李信的失败

### 失败原因
1. **兵力不足**：20万 vs 楚悉国中兵
2. **轻敌冒进**：初期破敌后立即追击
3. **疲惫松懈**：三日三夜不顿舍
4. **被敌利用**：楚人随之，大破李信军

### 教训
- 初期胜利不等于最终胜利
- 追击需要保持阵形和休整
- 年轻将领易犯轻敌冒进错误

## 历史案例

### 类似战例
- **廉颇坚守长平**：坚壁不出（正确策略）
  - 赵王不耐，换赵括（冒进）→ 惨败
- **白起vs廉颇**：白起未能直接击破廉颇
  - 等到赵括换将才得胜
- **王翦vs楚**：坚壁待时→全胜
  - 李信冒进→惨败

### 反例
- **赵括**：少年气盛，主动出击→中计被围
- **李信**：年少壮勇，兵力不足→三日惨败

## 核心智慧

### 兵法层面
1. **兵贵胜不贵久**：不急于求战，但求必胜
2. **养士蓄锐**：战前准备比战中勇猛更重要
3. **观察时机**：从细节判断（投石超距）
4. **一击必杀**：时机成熟，追击歼灭

### 人性层面
1. **老将经验**：经历过失败，更懂谨慎
2. **年轻冒进**：未经挫折，易轻敌
3. **君主心理**：喜欢听"容易"的答案（20万）
4. **专业坚持**：王翦坚持60万，不迎合
