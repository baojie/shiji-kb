## 章节主题

### 核心主题

1. **以少胜多**：田单以孤城残众，对抗燕国精锐，创造了战国史上最戏剧性的逆转
2. **心理战**：反间计除去乐毅、神师造势、激怒效应、火牛阵，层层精心设计
3. **创造性战术**：火牛阵是资源极度匮乏时创意最大化的典范
4. **领导力的两面**：守即墨的"共苦"成功，攻狄的"刚强"失败，被鲁仲连点出

### 关键人物

| 人物 | 身份 | 作用 |
|------|------|------|
| **田单** | 主角，即墨守将→安平君 | 复国英雄，心理战天才 |
| **乐毅** | 燕国统帅 | 最大的威胁，被田单反间计除去 |
| **骑劫** | 代乐毅者 | 轻敌，中火牛阵，阵亡 |
| **燕惠王** | 燕国国君 | 中反间计，换将，导致燕国功亏一篑 |
| **鲁仲连** | 齐国辩士 | 书谏田单，点出"共苦"之道 |
