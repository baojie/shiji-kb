# Chapter 071: 樗里子甘茂列传

## 章节概述

《樗里子甘茂列传》合传三位秦国名臣：樗里子、甘茂、甘罗（甘茂之孙）。三人分别代表不同的政治资本类型：樗里子以"骨肉"（王族血脉）加"智"立身；甘茂以布衣之才博得重用，以息壤之盟保全战功；甘罗年仅十二，以奇计出使赵国一举取五城，封为上卿。

本传集中体现战国政治的两个核心主题：谗言环境下如何保全功业，以及年少奇才如何以辩才在强者间游走。

## 核心人物

### 樗里子（严君）
- **身份**：秦惠王异母弟，母为韩国女子
- **外号**：智囊（秦人所赠）
- **成就**：攻曲沃、拔蔺、取汉中，多次立战功；历惠王、武王、昭王三朝
- **预言**：临终言身死百年后将有天子之宫夹其墓，汉代长乐宫建成应验

### 甘茂
- **出身**：下蔡闾巷之人（寒门）
- **入秦**：经张仪、樗里子引荐，仕于秦惠王
- **著名事迹**：以"息壤之盟"保障宜阳之役不受谗言动摇，拔宜阳斩首六万
- **结局**：遭向寿、公孙奭谗言，惧而出走，先奔齐后至楚，卒于魏，终不得归秦

### 甘罗
- **身份**：甘茂之孙
- **年龄**：十二岁
- **事迹**：说服张唐赴燕，出使赵国取五城，封上卿
- **意义**：战国少年奇才的极端案例，以逻辑比较法驱动强者行动

## 核心事件

### 1. 樗里子的智囊之名
- 秦惠王弟，骨肉与智能兼具
- 历三朝均受重用，惠王封严君
- 临终预言身后地：百年后天子之宫夹其墓（汉长乐宫应验）

### 2. 息壤之盟（甘茂攻宜阳）
- 秦武王欲攻宜阳以窥周室
- 甘茂预判：战事持久必遭谗言，请王提前盟约
- 以"曾参杀人三人疑"论证谗言的结构性存在
- 以魏文侯谤书一箧论证持久战中谗言是必然
- 武王与甘茂盟于息壤，承诺不受谗言动摇
- 五月不拔，樗里子、公孙奭果然来争
- 甘茂一语"息壤在彼"，武王大悉起兵，终拔宜阳

### 3. 甘茂亡秦
- 甘茂将武遂归韩，向寿、公孙奭谗言
- 甘茂惧，亡去奔齐
- 在齐楚间辗转，秦要求楚送回，楚谋臣范蜎阻止
- 甘茂竟不得复入秦，卒于魏

### 4. 甘罗十二岁出使取城
- 秦使张唐相燕，张唐以赵怨拒行
- 甘罗以比较逻辑说服张唐（比较功劳与文信侯权力）
- 出使赵国，以燕秦同盟威胁说服赵王割五城换安全
- 赵攻燕得三十城，秦得其十一
- 甘罗封上卿，复得甘茂田宅

## 核心主题

### 1. 谗言的政治生态
- 宫廷中谗言是结构性存在，非偶然
- 甘茂的防谗策略：事前建立盟约，赋予承诺道德约束力
- 甘茂最终仍未能逃脱谗言之害（失去向寿支持后流亡）

### 2. 骨肉与布衣的资本差异
- 樗里子：骨肉（王族）+ 智（个人能力），双重加持
- 甘茂：纯布衣起家，无血脉保护，功成后更脆弱
- 甘罗：靠家族声名（甘茂孙）+ 个人奇才，短暂爆发

### 3. 年少奇才的说服逻辑
- 以历史先例对抗年龄偏见（项橐七岁为孔子师）
- 以比较式恐惧分析驱动强者行动（张唐功不如白起，文信侯权大于应侯）
- 以利益交换设计让各方均受益的方案（赵割五城得三十城）

## 太史公评价

> 樗里子以骨肉重，固其理，而秦人称其智，故颇采焉。甘茂起下蔡闾阎，显名诸侯，重彊齐楚。甘罗年少，然出一奇计，声称后世。虽非笃行之君子，然亦战国之策士也。方秦之强时，天下尤趋谋诈哉。

评价分析：
- 樗里子：肯定其智，但点明骨肉是前提
- 甘茂：赞其从寒门崛起，显名于大国
- 甘罗：一奇计流芳，非道德君子，但是战国策士
- 最后感叹：秦强时天下趋谋诈——这三人正是此时代精神的体现
