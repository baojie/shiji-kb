# 廉颇蔺相如列传 — 知识图谱

本目录包含《史记·廉颇蔺相如列传》章节的结构化知识提取。

## 章节概述

本传以赵国两位重臣廉颇与蔺相如为主角，以完璧归赵、渑池之会、负荆请罪三大故事构成叙事核心，是《史记》列传中最广为传诵的篇章之一。

**廉颇**（约前327—前243年）：赵国最重要的武将，多次领兵大败秦军和齐军，以勇猛善战著称。长平之战前期坚守消耗秦军，被赵王替换为赵括后导致惨败（纸上谈兵）。晚年被猜忌，流亡魏国，临死前仍未能重返赵国。

**蔺相如**：出身宦者门客（宦者令缪贤舍人），因完璧归赵和渑池之会的外交功绩，两度升迁，最终位居廉颇之上，成为上卿。面对廉颇的挑衅，以"先国家之急而后私仇"为原则，主动退让，最终感化廉颇，两人结为刎颈之交。

## 核心人物

- **廉颇**：赵国大将，赵国军事支柱，以勇猛刚直著称，能知错就改（负荆请罪）
- **蔺相如**：外交家，凭智慧和胆略在秦国主场维护赵国尊严，以大局为重化解内部矛盾
- **秦昭王**：试图以和氏璧为名占便宜，先后在完璧归赵和渑池之会中被蔺相如挫败
- **赵惠文王**：赵王，蔺相如的后盾，信任蔺相如赴险
- **赵括**：廉颇替换后的赵国大将，长平之战全军覆没，"纸上谈兵"的原型

## 核心故事

1. **完璧归赵**：秦昭王以十五城换和氏璧，实为欺诈。蔺相如出使秦国，以机智和以命相搏的勇气，将和氏璧安全带回，"完璧归赵"
2. **渑池之会**：赵秦两国君主会盟，秦王令赵王鼓瑟以示羞辱，蔺相如逼秦王击缻还礼，在秦国大本营维护了赵国尊严
3. **负荆请罪**：蔺相如升位在廉颇之上，廉颇不服。蔺相如以国家利益为重，处处回避廉颇。廉颇悟其大义，负荆请罪，两人结为刎颈之交
