## 概述

《历书》的后半部分（第16-36节）是历术甲子篇，记录了太初元年（前104）至建始四年（前29）约75年的年历数据，每年列出月数、大余/小余（日）、大余/小余（月）和年名。这是汉代太初历的实际运行记录。

---

## 核心术语

| 术语 | 含义 |
|------|------|
| 大余 | 日余数（60甲子周期中的余数） |
| 小余 | 月余数 |
| 十二 | 该年有12个月（平年） |
| 闰十三 | 该年有13个月（闰年，有闰月） |
| 正北/正西/正南/正东 | 冬至所在的时辰（子/酉/午/卯时） |

## 步骤

### 步骤1：识别年份参数

每年的数据格式：
```
月数（12或闰13）
大余X，小余Y；大余A，小余B；
干支岁名 + 年号
```

**解读**：
- 第一组"大余X，小余Y"：该年日历的大余/小余
- 第二组"大余A，小余B"：该年月历的大余/小余

---

### 步骤2：解读岁名

**示例**：「焉逢摄提格太初元年」
- 焉逢 = 甲（天干）
- 摄提格 = 寅（地支）
- 合为甲寅年
- 公历：前104年

**示例**：「端蒙单阏二年」
- 端蒙 = 乙
- 单阏 = 卯
- 合为乙卯年
- 公历：前103年

---

### 步骤3：识别闰年

有"闰十三"标记的年份为闰年，月数为13。

**太初历闰年频率**：约每2-3年一闰，无固定间隔。

---

### 步骤4：识别冬至时辰标记

| 标记 | 冬至时辰 | 地支 |
|------|----------|------|
| 正北 | 子时（23:00-1:00） | 子 |
| 正西 | 酉时（17:00-19:00） | 酉 |
| 正南 | 午时（11:00-13:00） | 午 |
| 正东 | 卯时（5:00-7:00） | 卯 |

这四个标记在数据表中每隔若干年出现一次，标志着冬至时辰的轮转。

## 实用查询示例

**问：汉宣帝本始元年是什么甲子？**
- 查 fact_003.json 的"本始元年"条目
- 年名：端蒙作噩（乙酉年）
- 公历：前73年

**问：哪些年份有闰月？**
- 在历术表中找"闰十三"标记的年份

**问：太初元年的历法起点是什么？**
- 十一月甲子朔旦冬至（三合一完美起点）
