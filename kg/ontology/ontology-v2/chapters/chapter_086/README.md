## 核心主题

### 1. 刺客精神的演变

**曹沫（春秋）**：
- 动机：爱国主义（报鲁国之辱）
- 方式：劫持外交
- 结局：全身而退

**专诸（春秋晚期）**：
- 动机：个人忠诚（报公子光知遇）
- 方式：刺杀篡位
- 结局：当场牺牲，子封上卿

**豫让（战国早期）**：
- 动机：知己之恩（智伯国士遇我）
- 方式：三次复仇（为已死雇主）
- 结局：象征性完成，伏剑自杀

**聂政（战国中期）**：
- 动机：感恩图报（严仲子国士交）+ 家庭伦理（母在不许）
- 方式：独行刺杀 + 毁容保护
- 结局：成功击杀目标，姊姊彰名

**荆轲（战国末期）**：
- 动机：复合（知遇+救国+名节）
- 方式：淬毒匕首 + 樊於期首 + 地图
- 结局：失败但名垂千古，催化秦灭燕

**演变趋势**：
- 从国家利益 → 个人知遇关系
- 从劫持谈判 → 刺杀复仇
- 从全身而退 → 以死相报
- 从实际成果 → 精神价值

### 2. 知遇之恩的三个层级

| 层级 | 待遇特征 | 回报标准 | 案例 |
|-----|---------|---------|------|
| Level 1 | 众人遇（市场价） | 众人报（完成工作） | 豫让事范氏中行氏 |
| Level 5 | 优遇（超常待遇） | 愿意承担风险 | 严仲子待聂政 |
| Level 10 | 国士遇（命运共同体） | 国士报（以死相报） | 智伯待豫让、公子光待专诸 |

**核心洞察**: "众人遇我，众人报之；国士遇我，国士报之"

### 3. 刺杀成功的要素分析

#### 成功案例（曹沫、专诸、聂政）
✓ **时机选择**：盟誓时、内空虚时
✓ **武器隐蔽**：匕首、鱼炙藏刃
✓ **距离接近**：坛上、进食时、相府
✓ **出其不意**：仪式场合、宴饮时

#### 失败案例（豫让、荆轲）
✗ **助手失态**：秦舞阳色变
✗ **武器限制**：秦王剑长
✗ **环境设置**：柱子阻隔、群臣不得持兵
✗ **执行犹豫**：荆轲欲生劫而非径杀
