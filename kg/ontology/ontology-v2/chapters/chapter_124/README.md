## 核心主题

### 1. 游侠道德规范
- 言必信，行必果，已诺必诚
- 不爱其躯，赴士之厄困
- 不矜其能，羞伐其德（不居功）

### 2. 代表人物
- **朱家**（汉初）：藏匿季布，终身不见高祖，游侠精神的奠基者
- **剧孟**（景帝时）：母丧千乘，七国之乱时周亚夫视其立场决定大局
- **郭解**（武帝时）：汉代游侠的集大成者，最终被夷三族

### 3. 游侠的衰亡
- 汉武帝迁徙豪富入茂陵，切断游侠与地方社会的联系
- 郭解案：公孙弘以"大逆无道"定性，游侠权威本身成为罪行
- 历史意义：国家对社会秩序垄断权的确立

## 历史意义

### 社会史价值
- 游侠作为制度补充：填补官方秩序空白的民间力量
- 汉代社会结构的真实记录：豪强、游侠、地方官府的三角关系

### 思想史价值
- 司马迁为游侠辩护，与韩非子"侠以武犯禁"的对立立场
- 义（民间道德）vs 法（国家规范）的哲学张力
