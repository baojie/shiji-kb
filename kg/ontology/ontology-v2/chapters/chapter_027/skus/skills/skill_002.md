## 概述

skill_001 讨论"通过天象观测预判国事"的宏观战略。本 skill 专注于**实操方法**：如何按《天官书》原文的步骤，进行一次完整的"观候-占验-上报-修应"操作。参照司马迁对汉代观象家（唐都、王朔、魏鲜）工作的记载，以及对八风、云气、日旁观测的具体技术描述。

## 何时使用此 Skill

- 承担天文观测或占卜任务
- 需要从古代史料提取天人感应的实操线索
- 研究《史记》《汉书》天文志的读法
- 做制度史研究，分析"天象如何影响朝政"

## 核心步骤

### 1. 明确观测目标

首先判断此次观候的目的属于哪一类：

| 目的类型 | 主要对象 | 参考原文 |
|---------|---------|---------|
| 战事判断 | 日晕、蚩尤旗、彗星、云气 | [69]、[89] |
| 灾异预警 | 日蚀、月蚀、五星失行 | [70]-[73] |
| 年景预测 | 八风、岁始、云气 | [105]-[107] |
| 人主更迭 | 太白当出不出、彗星久现 | [53]、[118] |
| 诸侯关系 | 五星聚一舍、客星出天廷 | [49]、[124] |

### 2. 设立观测点与观测时

#### 空间选择

- **仰望**（附近300-400里）：观察近处的天气云气
- **平望**（桑榆上）：观察千里左右的云气
- **登高望**：观察下属地者三千里

#### 时间选择（[105]"四始"）

岁始观候的四个关键点：
- **冬至日**：产气始萌
- **腊明日**：人众卒岁，发阳气
- **正月旦**：王者岁首
- **立春日**：四时之始

其他关键观测时：
- **正月上甲**：看风与蚕候
- **仲春春分、仲夏夏至、仲秋秋分、仲冬冬至**：辰星四时出现的位置

### 3. 记录天象的"三维"信息

参照[69]日晕观测的范例，完整记录包括：

- **位置**：出现在何宿、何方
- **形态**：厚薄、大小、颜色、形状（圜/兑/钩/杼/轴...）
- **时间**：出现的时辰、持续时长、消失方向
- **伴随**：有无其他天象同时出现

### 4. 按五行-五方-五时对应

根据五星原则判断天象所"失"：

```
岁星异 → 义失
荧惑异 → 礼失
填星异 → 德失（主德女主象）
太白异 → 杀失
辰星异 → 刑失
```

### 5. 查分野对应（[67]）

确定天象涉及的地理区域：

| 宿 | 州域 |
|-----|------|
| 角、亢、氐 | 兗州 |
| 房、心 | 豫州 |
| 尾、箕 | 幽州 |
| 斗 | 江、湖 |
| 牵牛、婺女 | 杨州 |
| 虚、危 | 青州 |
| 营室至东壁 | 并州 |
| 奎、娄、胃 | 徐州 |
| 昴、毕 | 冀州 |
| 觜觿、参 | 益州 |
| 东井、舆鬼 | 雍州 |
| 柳、七星、张 | 三河 |
| 翼、轸 | 荆州 |

### 6. 适用"过度乃占"原则（[123]）

关键决策：本次天象是否"过度"？

- 是常规五星运行吗？——否
- 是常规日月节气吗？——否
- 是过去百年内见过的类似异象吗？——参考先例占
- 从未见过？——记录并慎重占验

不要对每一次日月异动都惊恐——必须"过度"才占。

### 7. 选择应对级别（[123]）

根据天变性质选择应对方式：
- 日变 → 修德（帝王反省）
- 月变 → 省刑（减刑）
- 星变 → 结和（外交和解）

分四级递降：
1. **修德**：最高层级，帝王反省内修
2. **修政**：修正具体政策
3. **修救**：救济受灾百姓
4. **修禳**：最后选择，祭祀禳除

### 8. 上报与记录

参考《天官书》[122]司马迁"考行事"的方法：
- 记录天象细节
- 对照历史先例
- 汇报给朝廷
- 长期保存以供"古今验证"

## 具体案例实施：八风占（[106]）

以魏鲜"集腊明正月旦决八风"为例：

### 实施步骤

1. **选日**：腊日明日、正月旦
2. **观风向**：判断风从哪方来
3. **对照表格**：

| 风向 | 预占 |
|------|------|
| 南 | 大旱 |
| 西南 | 小旱 |
| 西 | 有兵 |
| 西北 | 戎菽为，小雨，趣兵 |
| 北 | 中岁 |
| 东北 | 上岁 |
| 东 | 大水 |
| 东南 | 民有疾疫，岁恶 |

4. **判强弱**：八风各与其冲对，课多者为胜。多胜少，久胜亟，疾胜徐。
5. **对作物**：不同时段风向对应不同作物的丰歉
   - 旦至食：麦
   - 食至日昳：稷
   - 昳至餔：黍
   - 餔至下餔：菽
   - 下餔至日入：麻
6. **结合云日**：
   - 日当其时者，深而多实
   - 无云有风日，当其时，浅而多实
   - 有云风，无日，当其时，深而少实
   - 有日，无云，不风，当其时者稼有败

## 决策点

### 本次观测是"占"还是"记"？

- 占：过度天象、影响人事重大
- 记：常规异动、作为数据积累

### 当先例与当前矛盾时

参考[122]司马迁的方法：**考行事**——通过更多史料验证。不轻易否定先例，也不盲从。

### 当多种天象同时出现

优先级：
- 日食 > 月食（[73] 日蚀为不臧，月蚀为常）
- 客星 > 五星（客星更"过度"）
- 五星聚 > 单星行（聚合事大）
- 彗星竟天 > 短彗星

### 当预占与政局矛盾

这是最敏感的决策点。参考[7]晁错之例的教训——不要以天象强行推动过激政策；也参考[123]的"四级应对"——不能"正下无之"（毫无作为）。

## 预期结果

- 形成一份规范的观测记录：包含天象三维信息、分野对应、占验结论、应对建议
- 建立长期"实录"数据库——为后世"考行事"提供基础
- 避免过度解读（过度乃占）
- 避免单一维度误判（常气 vs 客气分工）
- 推动从"禳"向"德政"的治理升级

## 潜在风险

- **过度解读**：把常规天象解读为凶兆，引发朝廷恐慌
- **迎合现实**：先定人事再强解天象（"禨祥"的危险）
- **失之于迂**：引据过于古远，失去可信度
- **技术失准**：观测点、时间、仪器误差导致记录不准
