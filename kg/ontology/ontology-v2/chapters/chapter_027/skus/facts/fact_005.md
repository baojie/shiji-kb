## 概述

《天官书》[95]-[107]是一套完整的云气观测方法论。司马迁指出，云气不同于星辰，属"天之客气"——变化迅疾，占验亦更与人事密切相关。本 fact 还原原文的云气占候体系：观测视距、各方地气、军事云、地方云、征候云。

## 云气观测的基本方法

### 视距公式

原文[95]：
> 凡望云气，仰而望之，三四百里；平望，在桑榆上，千馀二千里；登高而望之，下属地者三千里。

不同观测角度对应不同视距：
- **仰望**：300-400里
- **平望**（桑榆上）：1000-2000里
- **登高望**（下属地者）：3000里

### 云气有动物形象

> 云气有兽居上者，胜。

云气上若有兽形，主胜——这是军事判断的基本法则。

## 各方地气的分野

原文[96]：
> 自华以南，气下黑上赤。嵩高、三河之郊，气正赤。恒山之北，气下黑下青。勃、碣、海、岱之间，气皆黑。江、淮之间，气皆白。

| 地区 | 气色 |
|------|------|
| 华山以南 | 下黑上赤 |
| 嵩高、三河 | 正赤 |
| 恒山以北 | 下黑下青 |
| 勃、碣、海、岱之间 | 皆黑 |
| 江、淮之间 | 皆白 |

这是一张按地理区位的"地气色谱"——每个区域的正常地气颜色不同，异常则有预占。

## 军事云气（[97]）

### 兵种云气

> 徒气白。土功气黄。车气乍高乍下，往往而聚。骑气卑而布。卒气抟。

| 兵种/活动 | 气貌 |
|-----------|------|
| 徒（步兵） | 白 |
| 土功 | 黄 |
| 车 | 乍高乍下，往往而聚 |
| 骑 | 卑而布 |
| 卒 | 抟 |

### 速度判断

- **前卑而后高者**：疾（快）
- **前方而后高者**：兑（锐利）
- **后兑而卑者**：却（退却）
- **气平者**：其行徐（缓）
- **前高而后卑者**：不止而反（会折返）

### 气相遇法则

> 气相遇者，卑胜高，兑胜方。

- 低气胜高气
- 锐气胜方气

### 时距对应表

| 气高 | 预期时间 | 距离 |
|------|---------|------|
| 来卑循车通 | 3-4日 | 5-6里见 |
| 高7-8尺 | 5-6日 | 10余里见 |
| 高丈馀至2丈 | 30-40日 | 50-60里见 |

## 阵云形态（[98]）

原文列举七种形态的云：

- **阵云**：如立垣（像直立的墙）
- **杼云**：类杼（像织布梭子）
- **轴云**：抟两端兑（圆长两端尖）
- **杓云**：如绳者，居前互天（像绳索，前方贯天）
- **钩云**：句曲（弯曲如钩）
- **稍云**：精白者——其将悍，其士怯
- **大根而前绝远**：当战

### 颜色预占

- 青白，其前低者：战胜
- 其前赤而仰者：战不胜
- 各以五色合占

### 出现即有应验

> 兵必起，合其直。

## 日旁云气（[99]）

> 王朔所候，决于日旁。日旁云气，人主象。皆如其形以占。

- 专家：王朔
- 观测点：日旁
- 象征：人主
- 方法：以云气形状为象，按形占断

## 地方云气（[100]）

### 四夷之气

- **北夷之气**：如群畜穹闾（像游牧营地）
- **南夷之气**：类舟船幡旗（像船只旗帜）

### 特殊场所的气

- 大水处：有气
- 败军场：有气
- 破国之虚：有气
- 下有积钱金宝之上：皆有气

### 海旁气

> 海旁蜄气象楼台；广野气成宫阙然。

——这是"海市蜃楼"的最早记载。

### 山川人聚

> 云气各象其山川人民所聚积。

## 察国气的方法（[101]）

入国邑察看的要素：
1. **封疆田畴之正治**（耕田秩序）
2. **城郭室屋门户之润泽**（建筑状态）
3. **次至车服畜产精华**（车服牲畜）

原则：
> 实息者，吉；虚秏者，凶。

## 特殊气象

### 卿云（[102]）

> 若烟非烟，若云非云，郁郁纷纷，萧索轮囷，是谓卿云。卿云，喜气也。

### 甲气（[102]）

> 若雾非雾，衣冠而不濡，见则其域被甲而趋。

### 阳气之动（[103]）

> 雷电、虾虹、辟历、夜明者，阳气之动者也，春夏则发，秋冬则藏。

## 地象物候（[104]）

除了天象、云气外，地象也是观测要素：
- 天开县物，地动坼绝
- 山崩及徙，川塞谿垘
- 水澹地长
- 城郭门闾、闺臬枯、宫庙邸第
- 谣俗车服，观民饮食
- 五穀草木，观其所属
- 仓府厩库
- 六畜禽兽，鱼鳖鸟鼠

这些物候合起来构成"综合观测"——天地一体而观。

## 岁始候岁（[105]）

原文指出"候岁美恶"的四个时间节点（四始）：
- **冬至**：产气始萌
- **腊明日**：人众卒岁，发阳气
- **正月旦**：王者岁首
- **立春日**：四时之始

这四日的天象、风向、云气都被视为预测全年吉凶的关键。

## 魏鲜八风法（[106]）

原文记载魏鲜"集腊明正月旦决八风"的方法：

| 风向 | 预占 |
|------|------|
| 南 | 大旱 |
| 西南 | 小旱 |
| 西 | 有兵 |
| 西北 | 戎菽为，小雨，趣兵 |
| 北 | 中岁 |
| 东北 | 上岁 |
| 东 | 大水 |
| 东南 | 民有疾疫，岁恶 |

### 风力规则

> 八风各与其冲对，课多者为胜。多胜少，久胜亟，疾胜徐。

### 时段对应作物

- 旦至食：麦
- 食至日昳：稷
- 昳至餔：黍
- 餔至下餔：菽
- 下餔至日入：麻

### 云风日组合

- 日当其时者，深而多实
- 无云有风日，当其时，浅而多实
- 有云风，无日，当其时，深而少实
- 有日，无云，不风，当其时者稼有败

## 云气占的理论总结

云气占验不同于星辰占验的关键：

原文[123]：
> 夫常星之变希见，而三光之占亟用。日月晕適，云风，此天之客气，其发见亦有大运。然其与政事俯仰，最近人之符。

常星变化稀少（大运），但日月晕適、云气、风向这些"客气"——与政事直接对应，是"最近人之符"。
