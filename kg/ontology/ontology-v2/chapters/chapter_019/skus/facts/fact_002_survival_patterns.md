# Fact 002: 侯爵存续模式与消亡原因

## 核心数据

**总体存续率**：
- 封侯时（惠帝-景帝）：93个
- 至景帝末年：约30个（32%）
- 至武帝建元六年：约10个（11%）
- 平均存续代数：2.3代（约50-60年）

**存续时间分布**：
- 1代即除：约30%
- 2代传承：约40%
- 3代以上：约30%
- 5代最长：仅长沙王世系

## 国除原因统计

### 1. 坐法（约50%）

**酎金罪（最常见）**：
- 定义：祭祀宗庙时献金不足或不合格
- 集中爆发：元鼎五年（前112）
- 案例：
  - "便侯广志坐酎金，国除"
  - "平定侯昌坐酎金，国除"
  - "梧侯戎奴坐酎金，国除"

**谋反（最严重）**：
- 吕氏事件（前180）：13个吕氏侯爵全部诛除
  - "扶柳侯吕平坐吕氏事诛，国除"
  - "郊侯吕产坐吕氏事诛，国除"
- 七国之乱余波：
  - "管侯戎奴反，国除"
  - "瓜丘侯偃反，国除"

**不敬/不法**：
- 隆虑侯𫊸："坐母长公主薨未除服，奸，禽兽行，当死，自杀，国除"
- 山都侯当："坐与奴阑入上林苑，国除"
- 章武侯常："坐谋杀人未杀罪，国除"

### 2. 无后（约25%）

**案例**：
- "波陵侯魏驷薨，无后，国除"
- "缾侯孙单薨，无后，国除"
- "昌武侯充国薨，亡后，国除"

**特点**：
- 多发生在2-3代
- 缺乏生育或早夭
- 无法继承爵位

### 3. 吕氏事件（13个，集中于高后八年-文帝元年）

**背景**：
- 吕后专权，封吕氏宗族13侯
- 文帝即位，周勃、陈平等诛除诸吕
- "吕氏事"成为集体清算

**名单**：
1. 扶柳侯吕平
2. 郊侯吕产
3. 南宫侯张买
4. 博成侯冯代
5. 沛侯吕种
6. 赘其侯吕胜
7. 俞侯吕它
8. 滕侯吕更始
9. 吕成侯吕忿
10. 锤侯吕通
11. 祝兹侯吕荣
12. 东平侯吕庄
13. （还有其他相关者）

### 4. 改封（约10%）

**升为王**：
- "襄成侯义为常山王，国除"
- "轵侯朝为常山王，国除"
- "朱虚侯章为城阳王，国除"

**特点**：
- 多为宗室子弟
- 侯爵是过渡，最终目标是王爵

## 存续成功案例分析

### 长沙王世系（唯一五代传承）

**传承轨迹**：
1. 芮（开国文王）→ 2. 臣（成王）→ 3-5代

**成功原因**：
> "昔高祖定天下，功臣非同姓疆土而王者八国。至孝惠帝时，唯独长沙全，禅五世，以无嗣绝，竟无过，为籓守职，信矣。"

**关键要素**：
1. **无政治错误**："竟无过"
2. **忠于职守**："为籓守职"
3. **远离权力中心**：长沙偏远，不参与朝廷斗争
4. **传承稳定**：五代都有合法继承人

### 轵侯薄昭（文帝舅，传三代）

**传承**：薄昭 → 戎奴 → 梁

**成功原因**：
1. 外戚身份，但不干政
2. 薄太后节俭，不奢靡
3. 低调处事

### 犁侯召奴（传四代）

**传承**：召奴 → 泽 → ? → 延

**成功原因**：
1. 齐相之后，有家族基础
2. 远离朝廷权力斗争
3. 传承稳定

## 存续规律总结

### 高危因素

| 因素 | 风险 | 案例 |
|-----|------|------|
| 吕氏宗族 | 100%国除 | 13个吕氏侯全灭 |
| 参与谋反 | 100%国除 | 七国之乱相关侯 |
| 酎金不足 | 90%国除 | 元鼎五年大规模除国 |
| 无子嗣 | 100%国除 | 无法传承 |
| 外戚干政 | 高风险 | 隆虑侯等 |

### 保护因素

| 因素 | 效果 | 案例 |
|-----|------|------|
| 远离权力中心 | 降低政治风险 | 长沙王 |
| 低调守法 | 避免坐法 | 轵侯薄昭 |
| 有合法继承人 | 避免无后 | 多数传三代以上者 |
| 不参与朝廷斗争 | 政治安全 | 犁侯召奴 |
| 功臣世家 | 有保护伞 | 部分功臣后代 |

## 对比分析：异姓vs同姓

### 异姓侯（高祖功臣后代）

**特点**：
- 初期较多（约30%）
- 逐步减少（七国之乱后更甚）
- 坐法比例高

**原因**：
- 缺乏宗室保护
- 容易被猜忌
- 政治风险大

### 同姓侯（宗室外戚）

**特点**：
- 后期比例增加
- 改封为王者多
- 相对稳定（但也有风险）

**原因**：
- 皇帝信任
- 宗室保护
- 但如参与权力斗争，风险极大（如霍氏）

## 惠景间 vs 高祖功臣侯者的对比

| 维度 | 高祖功臣侯 | 惠景间侯 |
|-----|----------|---------|
| 封侯原因 | 军功为主 | 追封、外戚、平叛 |
| 政治地位 | 开国元勋 | 后续封赏 |
| 存续率 | 更低（"太初百年之间，见侯五"） | 相对较高 |
| 坐法原因 | 谋反为主 | 酎金、不敬为主 |
| 传承代数 | 平均2代 | 平均2.3代 |

## 现代启示

### 权力传承的风险管理

1. **政治风险**：
   - 参与权力斗争→高风险（吕氏全灭）
   - 远离权力中心→低风险（长沙五世）

2. **合规风险**：
   - 违法乱纪→必然清算（酎金事件）
   - 守法低调→长期生存

3. **继承风险**：
   - 无子嗣→家族断绝
   - 培养继承人→传承延续

### 家族企业的传承启示

**长沙王模式**（五世传承）：
1. 不参与公司政治斗争
2. 专注业务，忠于职守
3. 有清晰的继承规划
4. 低调处事，不张扬

**吕氏失败教训**：
1. 过度参与权力斗争
2. 依赖强人（吕后），强人倒台则全军覆没
3. 缺乏合法性基础

**功臣后代模式**：
1. 依靠祖宗功业
2. 但需持续证明价值
3. 否则逐步边缘化
