# 分封建制：平衡中央与地方的方法

## 概述

分封建制是周代为了"以封建亲戚以藩屏周"而创立的核心政治制度。它在《周本纪》[17]武王分封、[21]周公改封、[22]成王继封等多个节点逐步成型，将"天子—诸侯—卿大夫"三级权力体系、"同姓—异姓—先王之后—殷后"四类封建对象、"王畿—侯服—绥服—要服—荒服"五层地理差序，整合为一套长时段稳定的政治架构。

本Skill从《周本纪》实例中提炼周代分封的**分类、选址、授权、监管、问题**五个维度的方法论，供后世处理"中央—地方"、"总部—分部"、"主公—诸侯"等类似结构参考。

## 核心原则

### 原则一：分类封建，各司其职

周代分封对象分四类：

| 类别         | 代表        | 目的                     |
| ------------ | ----------- | ------------------------ |
| 同姓王叔     | 周公、康叔  | 血缘藩屏，防止异姓坐大   |
| 异姓功臣     | 师尚父（齐）| 酬功激励，集聚人才       |
| 先圣王之后   | 焦祝蓟陈杞  | 尊古怀柔，收人心         |
| 前朝遗裔     | 武庚→微子   | 续祀安民，化解复辟风险   |

### 原则二：战略要地由同姓或功臣把守

- 中心要地（如鲁、卫、晋、齐）给可信赖者
- 边缘之地（如陈、杞）给前朝之后——荣誉性封建，不涉实权

### 原则三：远近有别，责任有差

- 甸服重赋、荒服轻王——五服贡赋制度与分封体系同步
- 天子与诸侯之间保留"朝聘、贡赋、征伐"三项关键联结

### 原则四：监管到位，但不过度干预

- 武王克殷后对武庚"使弟管叔鲜、蔡叔度相禄父治殷"——建立"三监"式监管
- 管蔡叛乱后，周公改用"直接册立同姓诸侯（卫康叔）治殷故地"——监管升级为接管

### 原则五：制度要通过实战检验，不怕迭代

- 武王初封以"监殷"解决问题
- 武庚之乱后改以"康叔代殷"
- 这说明分封制度不是一次性静态结构，而是随着实践反馈持续调整的动态系统

## 具体步骤

### 第一步：明确分封的政治目的

**武王克殷后的五重目的**（[17]）：

1. **安殷遗民**：封武庚、散鹿台之财、发钜桥之粟
2. **表敬古圣**：封神农、黄帝、尧、舜、禹之后
3. **酬功激励**：首封师尚父于齐
4. **藩屏王室**：封周公、召公、叔鲜、叔度
5. **宣示正统**：展九鼎、保玉、作《分殷之器物》

**启示**：分封不是简单的"土地分配"，而是多重政治目的的综合工具。应先明确要解决哪些问题，再决定封谁、封在哪里、给什么权力。

### 第二步：选定封地要考虑战略价值

**周代封地的战略考量**：

- **营丘（齐）**：东海之滨，控制东方夷人——所以首封功勋最重的师尚父
- **曲阜（鲁）**：商奄故地，需要强大的宗亲力量——所以封周公（实际由伯禽就国）
- **燕**：北方防线，防御戎狄——封同姓召公
- **管、蔡**：殷故地附近，监视武庚——封同母之弟
- **卫**：殷故地核心——平乱后封同姓幼弟康叔
- **陈、杞等**：非战略要地，给前朝之后为荣誉性封建

**方法**：以地理+战略+文化三维评估封地，把最可信之人放在最关键之地。

### 第三步：授权要包含经济、军事、祭祀三项

**受封者的典型权力**（贯穿《周本纪》多处）：

- **经济**：征税、统领百姓、使用当地物产
- **军事**：设立军队、受天子征召、有限的独立征伐
- **祭祀**：续先人之祀、祭封地山川
- **赐命器物**：弓矢斧钺（授权征伐，如[8]殷赐西伯、[47]襄王赐晋文公）、珪鬯（祭祀）、胙肉（承认正统）

**方法**：权力授予要完整（三维缺一不可），但权力行使要受约束（如征伐需请命、祭祀有定式）。

### 第四步：监管制度必须双轨

**武王初建的监管双轨**：

1. **血缘监管**：管叔鲜、蔡叔度相禄父治殷——同姓近亲监视异姓降臣
2. **文化监管**：令武庚"修行盘庚之政"——以殷人自己的中兴传统约束其行为

**周公改进的监管升级**：

- 三监叛乱后，不再使用"监而不管"的间接方式
- 改为"收殷馀民，封武王少弟封为卫康叔"——由同姓幼弟直接统治殷故地
- 另以微子（纣庶兄，原三仁之一）代武庚续殷祀于宋，分割殷遗之政治与祭祀

**方法**：当间接监管失败时，要敢于直接接管；但接管不等于消灭，应保留"祭祀延续"这一文化纽带。

### 第五步：建立朝聘与贡赋的常态联结

**周代朝聘制度的关键点**（综合[25]祭公谋父引述）：

- **甸服者祭、侯服者祀、宾服者享、要服者贡、荒服者王**
- 不同层级的贡赋有频率差：日祭、月祀、时享、岁贡、终王

**惩戒层级**：

- 有不祭则脩意（修省心意）
- 有不祀则脩言（整理言辞）
- 有不享则脩文（修订文告）
- 有不贡则脩名（校正名分）
- 有不王则脩德（提升德政）
- 序成而有不至则脩刑——最后才用兵

**方法**：中央与地方的联系不能只靠税收和军事，应建立礼仪性的、渐进式的联络节奏。违约处理也要按梯度升级，而非一言不合即用兵。

### 第六步：封建要随时代更新

**周代分封的三次重大调整**：

1. **武王初封**：重酬功臣、监殷遗民、尊先圣王
2. **周公改封**：平乱后以同姓代殷后、以微子代武庚
3. **成王继封**：营洛邑、封康叔于卫、封唐叔于晋

每次调整都解决了上一阶段暴露的问题：

- 初封的问题是三监监而不管 → 周公改为同姓直治
- 周公改封的问题是洛邑与丰镐两都分立 → 成王继续营洛邑
- 成王继封的问题是太子不任 → 顾命制度补充

## 决策点

### 决策点1：哪些地区应封谁？

**优先级**：
1. 战略要地 → 同姓王叔或首功之臣
2. 次战略地 → 同姓亲属或重要功臣
3. 敏感地带（前朝故地） → 同姓直治 或 前朝之后续祀
4. 边缘地带 → 前朝之后、异姓功臣、荣誉性封建

### 决策点2：给多大权力？

**最小权力原则**：

- 能用祭祀联结的，不用军事
- 能用贡赋约束的，不用征伐
- 能用道义申告的，不用刑罚

从"脩意"到"脩刑"的六级处理梯度——体现的就是最小权力原则。

### 决策点3：监管失败怎么办？

**升级方案**：

- 间接监管失败 → 直接接管（如武庚乱后派康叔）
- 直接接管失败 → 扶植替代势力（如以微子代武庚）
- 全面失控 → 以霸权诸侯代理（如东周后期以晋、齐诸公为"方伯"）

### 决策点4：前朝遗裔怎么处理？

**三步法**：
1. **封以续祀**：不绝前朝之祀，争取遗民
2. **限其势力**：封于非战略地带
3. **必要时换人**：若遗裔反叛（如武庚），改封另一位前朝后人（如微子）

### 决策点5：何时应用刑罚？

只有在"脩意、脩言、脩文、脩名、脩德"五级处理无效后，才动用"脩刑"与兵戎。《周本纪》[25]祭公谋父反对穆王轻征犬戎，即是此原则的经典实践。

## 预期结果

### 短期结果（武王—成王时期）

- 殷遗民被有效安置
- 八百诸侯宗周
- 同姓十余国建立
- 先圣王之后得到册封
- 九鼎居洛邑

### 中期结果（成康时期）

- 刑错四十馀年不用
- 天下安宁
- 礼乐制度奠基

### 长期结果（到春秋战国）

- 周室八百年国祚
- 形成以齐、鲁、晋、燕、秦、楚等为代表的诸侯网络
- 为汉代"郡国并行"、唐代"节度使"、明代"藩王"等后续"中央—地方"制度提供参照
- 分封失控时的表现（襄王之乱、王子朝之乱、三晋分立）也成为历代"削藩"政策的反面教材

## 现代应用

### 一、大型企业的事业部制

- 总部（王室）+ 事业部（诸侯）+ 区域分公司（卿大夫）
- 创始人核心成员（同姓王叔）掌握战略事业部
- 外部引入的职业经理人（异姓功臣）掌握具体业务线
- 合并重组的公司前主（前朝遗裔）给予保留地位，但不给实权

### 二、集团公司的海外扩张

- 战略市场由总部直接派员（同姓）
- 次战略市场由本地合作伙伴（异姓功臣）
- 新兴市场保留当地传统品牌（续祀）

### 三、联邦制政治的管理

- 中央与地方的权力分配要按"经济+军事+祭祀（文化认同）"三维平衡
- 中央对地方的约束要有"渐进式升级"的处理机制
- 地方代表要有"朝聘"式的定期联络机制

## 常见误区

### 误区1：只封不监

**警示**：武庚被封后若无"三监"，早已独立；但仅有"三监"而管叔、蔡叔反叛——说明监管者本身也要被约束。

### 误区2：只管不封

**警示**：秦并六国后废分封行郡县，虽然短期强力集权，但秦亡后汉高祖又恢复部分分封——说明极端中央集权面临"二世而亡"的系统性风险。

### 误区3：封建不随时代调整

**警示**：周穆王不听祭公谋父之谏，强征犬戎，"自是荒服者不至"——五服体系因僵化执行而首先瓦解。

### 误区4：用刑太急

**警示**：周幽王不听警告、厉王禁言杀人——王权反被民意所反噬。

### 武王封建（分类齐全）

《周本纪》[17]：

> "武王追思先圣王，乃襃封神农之後於焦，黄帝之後於祝，帝尧之後於蓟，帝舜之後於陈，大禹之後於杞。於是封功臣谋士，而师尚父为首封。封尚父於营丘，曰齐。封弟周公旦於曲阜，曰鲁。封召公奭於燕。封弟叔鲜於管，弟叔度於蔡。馀各以次受封。"

### 监殷

《周本纪》[17]：

> "封商纣子禄父殷之馀民。武王为殷初定未集，乃使其弟管叔鲜、蔡叔度相禄父治殷。"

### 周公改封

《周本纪》[21]：

> "以微子开代殷後，国於宋。颇收殷馀民，以封武王少弟封为卫康叔。"

### 五服朝聘

《周本纪》[25]祭公谋父：

> "夫先王之制，邦内甸服，邦外侯服，侯卫宾服，夷蛮要服，戎翟荒服。甸服者祭，侯服者祀，宾服者享，要服者贡，荒服者王。日祭，月祀，时享，岁贡，终王。"

### 渐进处理

《周本纪》[25]：

> "有不祭则脩意，有不祀则脩言，有不享则脩文，有不贡则脩名，有不王则脩德，序成而有不至则脩刑。"

### 授权器物

> "襄王乃赐晋文公珪鬯弓矢，为伯，以河内地与晋。"
