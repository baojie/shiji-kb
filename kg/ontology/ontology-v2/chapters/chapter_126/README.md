# Chapter 126：滑稽列传

## 章节概要

《滑稽列传》是《史记》一百三十篇中独具特色的一篇，专门为以幽默、机智、讽谏著称的人物立传。"滑稽"在先秦汉代是一个中性甚至正面的词，指能言善辩、言语圆转、以幽默方式化解难题的才能。

本篇收录了春秋至汉初四位代表人物：楚国俳优优孟、战国齐辩士淳于髡、秦代俳优优旃、汉代文人东方朔，勾勒出"滑稽"传统在中国早期历史中的演变轨迹。
