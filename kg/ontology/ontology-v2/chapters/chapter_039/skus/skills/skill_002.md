## 概述

基于晋文公4年内从入晋到称霸的案例，提炼出称霸的完整战略框架。核心：尊王+攘夷+树信+联盟。

## 核心策略

### 三步称霸法

1. **尊王**：获得道德制高点
   - 文公2年：入周襄王于周
   - 赵衰：求霸莫如入王尊周
   - 效果：获周王赐河内阳樊之地

2. **攘夷**：建立军事威信
   - 文公5年：城濮之战败楚
   - 策略：退避三舍（兑现承诺）
   - 效果：楚兵大败，子玉自杀

3. **受命**：获得合法性
   - 文公5年5月：献俘于周，受命为伯
   - 周王赐：大辂、彤弓矢百、玈弓矢千、秬鬯、珪瓚、虎贲300人
   - 践土会盟：王子虎盟诸侯于王庭

### 战略决策树

```
欲称霸
├── 是否先尊王？
│   ├── 是（晋文公）→ 获道德制高点 → 继续
│   └── 否（齐桓公晚年）→ 失人心 → 衰落
├── 选择什么对手树威？
│   ├── 大国（楚）→ 效果好，风险高
│   └── 小国 → 效果差
├── 如何战胜对手？
│   ├── 正面硬刚 → 风险高
│   └── 迂回策略（伐曹卫解宋围）→ 风险低
└── 胜后如何巩固？
    ├── 骄傲自满 → 失败
    └── 谦逊敬畏（文公："子玉犹在，庸可喜乎"）→ 成功
```

## 步骤

### 1. 尊王攘夷，获道德制高点

**行动**：入周襄王于周
**时机**：秦军驻河上，将入王
**决策**：晋不先入王，后秦入之，无以令于天下

### 2. 救难扶危，建立信誉

**行动**：救宋
**策略**：不直接救宋，而是伐曹卫，迫楚自退
**效果**：楚成王引兵归

### 3. 兑现承诺，树立威信

**行动**：城濮退避三舍
**承诺**：19年前对楚成王许诺
**效果**：信誉树立，天下信服

### 4. 献俘受命，巩固地位

**行动**：献楚俘于周，受命为伯
**效果**：获周王合法性认可

## 预期结果

- 短期：军事胜利
- 中期：诸侯归附
- 长期：霸业确立
