## 概述

晋献公两次假道于虞以伐虢，虞大夫宫之奇预言"唇亡齿寒"，虞公不听，最终晋灭虢后回师袭灭虞。这是春秋时期战略依存关系的经典案例，成语"唇亡齿寒"的出处。

## 详细内容

### 第一次假道伐虢（献公19年）

```json
{
  "背景": "先君庄伯、武公诛晋乱，而虢常助晋伐我，又匿晋亡公子",
  "方法": "使荀息以屈产之乘（名马）假道于虞",
  "结果": "虞假道，遂伐虢，取其下阳以归"
}
```

### 第二次假道伐虢（献公22年）

#### 宫之奇谏言

```json
{
  "谏言": {
    "核心": "虞之与虢，唇之与齿，唇亡则齿寒",
    "虞公反驳": "晋我同姓，不宜伐我",
    "宫之奇反驳": [
      "太伯、虞仲，太王之子也，太伯亡去，是以不嗣",
      "虢仲、虢叔，王季之子也，为文王卿士，其记勋在王室，藏于盟府",
      "将虢是灭，何爱于虞？",
      "且虞之亲能亲于桓、庄之族乎？桓、庄之族何罪，尽灭之"
    ]
  },
  "虞公不听": "遂许晋",
  "宫之奇": "以其族去虞"
}
```

#### 晋灭虢灭虞

| 时间 | 事件 |
|------|------|
| 冬 | 晋灭虢，虢公丑奔周 |
| 还师 | 袭灭虞，虏虞公及其大夫井伯、百里奚 |
| 百里奚 | 以媵秦穆姬（后为秦相） |
| 修虞祀 | 晋修虞祀（象征性安抚） |

#### 献公的玩笑

```json
{
  "荀息": "牵曩所遗虞屈产之乘马奉之献公",
  "献公笑": "马则吾马，齿亦老矣！",
  "寓意": "借出的马收回，只是老了；虞国借道，国已亡"
}
```

## 洞察

1. **唇亡齿寒**：相互依存的盟友，一方被灭另一方难存
2. **同姓不可靠**：虞公迷信"晋我同姓"，宫之奇举桓庄之族为例反驳
3. **贪小利失大局**：虞公贪屈产之乘，失国
4. **明哲保身**：宫之奇谏不听，以其族去虞
5. **战略依存**：现代国际关系、商业联盟中的依存关系同理
