## 核心主题

### 1. 音乐本体论

**三阶段生成**：
- 人心感物 → 声（原始）
- 声相应变成方 → 音（结构化）
- 比音加器乐舞蹈 → 乐（艺术完整体）

**知识层级**：禽兽知声 < 众庶知音 < 君子知乐

### 2. 音乐政治学

**诊断功能**：审乐以知政
- 治世之音：安以乐
- 乱世之音：怨以怒
- 亡国之音：哀以思

**五音映射**：宫（君）/商（臣）/角（民）/徵（事）/羽（物）

### 3. 礼乐哲学

**核心对立**：乐为同，礼为异
**宇宙论**：乐对应天地之和，礼对应天地之序
**治理论**：礼乐刑政四达，王道备矣

### 4. 历史叙事

**秦代**：李斯/赵高之争 → 二世选娱乐
**汉代**：高祖→惠文景无所更→武帝制十九章
**典故**：师旷识亡国之声，晋国大旱三年
