## 章节主题

### 核心主题

1. **辩士精神**：鲁仲连是战国游士中最纯粹的"义士"，以才能帮人，不受报酬
2. **书信说服**：鲁仲连射书聊城、邹阳狱中上书，均以文字达成了军队无法做到的事
3. **历史引证**：邹阳上书的论证方式——以历史案例为核心论据——是中国论辩文学的典范
4. **人际理解**："白头如新，倾盖如故"——了解一个人不取决于时间，而取决于真心

### 关键人物

| 人物 | 身份 | 代表事迹 |
|------|------|---------|
| **鲁仲连** | 主角，齐国辩士 | 义说辛垣衍、射书聊城、两次辞谢封赏 |
| **辛垣衍** | 魏国将领 | 被鲁仲连说服，放弃劝赵尊秦 |
| **平原君** | 赵国贵族 | 欲封赏鲁仲连，被拒 |

| **梁孝王** | 梁王刘武 | 读邹阳书信后，释放邹阳 |
