# 孟尝君列传 — 知识图谱

本目录包含《史记·孟尝君列传》章节的结构化知识提取。

## 章节概述

孟尝君田文是战国四公子之一（与平原君赵胜、信陵君魏无忌、春申君黄歇并称），以养士三千著称于世。其父田婴为齐威王之子、齐宣王之弟，田文因五月五日出生，父亲认为"五月子长与户齐，将不利其父母"而不欲抚养，田文母亲偷偷将其抚养成人，后田文以"天命论"说服父亲，展现其早年的才辩。

孟尝君门客来自各国、不分贵贱，甚至包括鸡鸣狗盗之人，其核心理念是"无贵贱，一与文等"。正是这些"卑微"的门客，在关键时刻救了孟尝君的性命。

## 核心人物

- **田文（孟尝君）**：主角，齐宣王之孙，战国四公子之一，养士三千
- **冯驩（冯谖）**：门客中最重要者，初来时弹铗而歌（冯谖三弹铗），后为孟尝君市义（焚薛地债券），营造三窟
- **秦昭王**：曾招孟尝君入秦为相，后又欲杀之
- **函谷关守将**：被门客学鸡叫骗过，放行孟尝君一行

## 核心故事

1. **五月子立为继承人**：田文五月出生被父嫌弃，以智辩说服父亲，确立继承权
2. **鸡鸣狗盗脱秦难**：在秦国被囚，门客学狗盗偷回白狐裘献秦王宠姬，又学鸡鸣骗开函谷关，深夜逃脱
3. **冯谖弹铗三歌**：冯谖初来，三次弹铗而歌，要求鱼、车、养家，孟尝君皆满足之
4. **焚券市义**：冯谖受命赴薛地收债，自作主张焚烧债券，为孟尝君"市义"，薛地百姓感恩戴德
5. **营造三窟**：冯谖为孟尝君策划三处保身之策（薛地为根据地、宗庙保平安、恢复相位），确保其政治安全

## 历史评价

太史公评：孟尝君招致天下任侠奸人，善人不称于孟尝，然世未有能及之者。

## 关键概念

- **养士文化**：战国特有的贵族-门客制度，以物质供养换取智力与武力支持
- **鸡鸣狗盗**：比喻微小技能在关键时刻发挥意外作用，也指旁门左道的才能
- **狡兔三窟**：比喻多做准备、多留退路的处世智慧
- **无贵贱一与文等**：孟尝君待客原则，不以门第高低区别对待
