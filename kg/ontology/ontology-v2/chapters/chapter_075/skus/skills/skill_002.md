# 冯驩复相术：利用竞争关系制造稀缺价值

## 概述

孟尝君被齐湣王废黜，门客皆去，处于政治低谷。冯驩以一车一人出发，先说秦王，再说齐王，利用秦齐两国的战略竞争关系，将孟尝君的"被废"变成两国争夺的资产，最终逼迫齐王主动恢复孟尝君相位并增封千户。这是战国时期最精妙的价值重塑案例之一。

## 步骤

### 1. 分析局势：找到可以利用的矛盾

**冯驩的战略分析**：

孟尝君被废的根本原因：齐惑于秦楚之毁，认为孟尝君名高于主，威胁王权。

冯驩看到的机会：
- 秦齐是天下"雄雌之国"，势不两立
- 孟尝君被废，且心中怀怨，若入秦，等于将齐国情报拱手相让
- 这一威胁对秦是机遇，对齐是危险

**关键洞察**：孟尝君的"弱势"（被废）恰好是其潜在价值最高的时刻——他手握齐国内情，任何一方得到他，另一方就处于战略劣势。

### 2. 第一步行棋：向秦王展示机会

**冯驩说秦王**：

> "天下之游士冯轼结靷西入秦者，无不欲强秦而弱齐；冯轼结靷东入齐者，无不欲强齐而弱秦。此雄雌之国也，势不两立为雄，雄者得天下矣。"

**论证逻辑**：
1. 建立框架：秦齐是雄雌之争，必有一胜
2. 揭示机会：孟尝君被废，怀怨必背齐
3. 计算收益：孟尝君入秦→齐国情报尽入秦手→齐地可得
4. 制造紧迫感：机不可失，需急速行动

**秦王反应**：大悦，遣车十乘黄金百镒以迎孟尝君。

**冯驩的真实目的**：
- 不是真的让孟尝君入秦
- 而是制造"秦已出手"这一事实，向齐国施压

### 3. 第二步行棋：向齐王展示危险

冯驩先行离秦返齐，说齐王：

> "今臣窃闻秦遣使车十乘载黄金百镒以迎孟尝君。孟尝君不西则已，西入相秦则天下归之，秦为雄而齐为雌，雌则临淄、即墨危矣。王何不先秦使之未到，复孟尝君，而益与之邑以谢之？"

**论证逻辑**：
1. 陈述事实：秦已派使者迎孟尝君（属实）
2. 预测后果：若孟尝君入秦，天下归秦，齐成为雌国
3. 提供方案：在秦使到达前主动复召，加封以谢
4. 结尾点睛：秦虽强国，岂可以请人相而迎之！若齐主动召，折秦之谋，绝其霸强之略

**齐王反应**：
- 曰"善"，乃使人至境候秦使
- 秦使车适入齐境，齐王已召回孟尝君，恢复相位，加封千户
- 秦使闻孟尝君复相，还车而去

### 4. 结果验收：双重成功

**表层结果**：
- 孟尝君复相，加封千户

**深层结果**：
- 秦的拉拢行动被化解（折秦之谋）
- 齐王主动示好，孟尝君地位更稳
- 冯驩以一车之力，完成了外交上的多步棋局

## 决策点

### 为何先说秦，后说齐？

**顺序的必要性**：
1. 必须先让秦王出手（派使者），才能有"秦已行动"这一事实作为依据
2. 没有秦王的实际行动，冯驩在齐王面前的威胁只是空话
3. 顺序颠倒则失效：若先说齐王，秦王不行动，齐王不会当真

### 如何确保秦王一定出手？

冯驩的说辞足够有吸引力：
- 得孟尝君 = 得齐国情报 = 齐地可得
- 对秦王来说，遣使代价极低（十辆车百镒金），潜在收益极大
- 秦王没有理由不出手

### 如何保证时机精准？

冯驩"辞以先行"——在秦使出发前先行离秦。计算好双方的路程，确保在秦使抵达齐境之前，自己已经说服齐王，但秦使还没有到达（确保齐王能在秦使抵达前行动）。

## 预期结果

**三层预期**：
1. 孟尝君复相（基本目标）
2. 加封邑，关系改善（进阶目标）
3. 折秦之谋，孟尝君在齐地位更稳（战略目标）

**实际结果**：三层目标全部实现。

## 现代应用

### 谈判策略："竞争性报价"

1. **制造备选项**：让A方知道B方有意向，价值立即提升
2. **顺序设计**：先获得B方的具体行动（报价），再拿来向A方施压
3. **截止时间**：利用"如果不快，B方就得手"制造紧迫感

### 职场价值重塑

1. **失业时是价值最高时机**：你手握的资源和人脉对新雇主最有价值
2. **利用竞争对手**：让A公司知道B公司在招你，倒逼A公司提高待遇
3. **先行动，后告知**：先得到竞争方的具体offer，再拿去谈判，比空口声称"有人要我"更有说服力

### 品牌公关

1. **被竞争对手看中 = 最好的背书**：某某想挖走你，证明你的价值
2. **制造竞争叙事**：让目标方觉得"不快点，就被别人抢走了"
