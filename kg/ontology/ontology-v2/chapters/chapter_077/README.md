# 魏公子列传（信陵君） — 知识图谱

本目录包含《史记·魏公子列传》章节的结构化知识提取。

## 章节概述

信陵君魏无忌（?—前243年）是魏安釐王之弟，战国四公子中最具军事才能者，也是司马迁最为推崇的一位。太史公亲赴大梁寻访信陵君遗迹，感叹"名冠诸侯，不虚耳"。汉高祖刘邦每过大梁，必亲自为信陵君扫墓，并置守冢五户，世代祭祀。

信陵君的核心精神是**礼贤下士**——他能够真正放下贵族身段，以平等甚至谦卑的态度对待有才能的平民（侯嬴、朱亥）。这种真诚的尊重，换来了对方以死相报的忠诚，也成就了"窃符救赵"这一历史壮举。

## 核心人物

- **魏无忌（信陵君）**：魏安釐王之弟，战国四公子之一，窃符救赵的主导者，五国联军破秦的统帅
- **侯嬴**：大梁城门守门人，七十岁，信陵君以上宾之礼待之。为信陵君谋划窃符救赵全策，并在信陵君出发后自刎，以死谢知遇之恩
- **朱亥**：市井力士，信陵君多次拜访、侯嬴力荐，关键时刻以大铁锥击杀晋鄙，帮助信陵君夺得兵权
- **如姬**：魏王宠姬，其父被人所杀，信陵君曾为其报仇。应信陵君所请，从魏王枕边盗取兵符
- **晋鄙**：魏国大将，奉王命率军十万驻扎，不肯出兵救赵，被朱亥以铁锥击杀

## 核心故事

1. **礼贤侯嬴**：信陵君知侯嬴贤，亲自驾车迎接，执辔御者，侯嬴故意让信陵君久候，亲眼验证信陵君的诚意，最终成为信陵君的首席谋士
2. **窃符救赵**（前257年）：秦围邯郸，魏王观望。信陵君请如姬盗兵符，持符见晋鄙，晋鄙疑惑不从，朱亥用大铁锥击杀晋鄙，信陵君率八万精兵救赵，秦军撤退，邯郸解围
3. **侯嬴自刎**：信陵君出发时，侯嬴说："我当随你同去，但年老体衰，不能相随。计算你的行程，当你击杀晋鄙之日，我将面北自刎，以此报答知遇之恩。"届时侯嬴果然自刎
4. **留赵十年**：窃符救赵后，信陵君畏魏王之怒，留赵不归，与平原君为宾客，结交赵国贤士（毛公、薛公）
5. **五国破秦**：赵国危急，赵使告知信陵君，信陵君以计夺回魏国兵权，统帅五国联军，大破秦军于河外
6. **晚年被谗**：秦国施反间计，魏王信谗言，免去信陵君兵权。信陵君心灰意冷，日夜沉湎酒色，四年后抑郁而卒，年约五十六岁

## 历史评价

**太史公**："吾过大梁之墟，求问其所谓夷门……名冠诸侯，不虚耳。高祖每过之而令民奉祠不绝也。"

**汉高祖刘邦**：每过大梁，常祠公子，置守冢五家，世世岁以四时奉祠。

司马迁在战国四公子中，对信陵君给予了最高的历史评价，认为他是真正做到礼贤下士、心怀天下而非一己私利的贵族典范。

## 关键概念

- **窃符救赵**：在君命与道义冲突时选择道义，以非常手段行正义之事
- **礼贤下士**：真诚尊重有才能的平民，不因身份差异而有所轻慢
- **功高震主**：功勋太大导致君主猜忌，进退两难的政治困境
- **道义与法统的冲突**：信陵君违抗王命救赵，是法律上的叛逆、道义上的英雄
