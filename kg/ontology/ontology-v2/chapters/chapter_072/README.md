# Chapter 072: 穰侯列传

## 章节概览

**主角**: 魏冉（穰侯），秦昭王母宣太后异父长弟  
**时代**: 战国晚期（秦昭王时期，约公元前306-前251年）  
**核心主题**: 外戚权力的建立、维持与衰落；远交近攻战略的诞生  
**历史地位**: 外戚专权的典型案例，秦国战略转型的关键节点

## 核心人物

### 魏冉（穰侯）
- **身份**: 宣太后异父长弟，秦昭王舅父
- **职位**: 四次为相国、将军
- **封地**: 穰、陶
- **成就**: 举白起，伊阙之战斩首24万，华阳之战斩首10万，拔楚郢
- **结局**: 因"富于王室"被范雎进谏，出关就封，卒于陶

### 宣太后
- **身份**: 秦昭王母，楚国羋氏
- **权力**: 昭王少时临朝称制，任魏厓为政
- **家族**: 魏冉、羋戎、高陵君、泾阳君

### 范雎
- **身份**: 魏人，自称张禄先生
- **贡献**: 提出远交近攻战略，批判穰侯越三晋伐齐
- **成就**: 促使昭王削外戚，自己上位为相

### 须贾、苏代
- **身份**: 战国纵横家
- **成就**: 经典游说案例（罢围大梁、止伐齐）

## 核心事件

### 1. 外戚擁立昭王（权力建立）
武王卒，诸弟争立，魏厓力立昭王→诛季君之乱、逐武王后→威振秦国

### 2. 举白起四拜相（权力维持）
- 第一次：举白起，伊阙之战
- 第二次：封穰侯，攻魏献河东
- 第三次：拔楚郢，置南郡
- 第四次：任相国，围大梁、华阳之战

### 3. 须贾说穰侯罢围大梁（游说经典）
五层论证法：历史类比→本质揭示→风险推演→技术论证→替代方案

### 4. 苏代止伐齐（战略萌芽）
五论破局：肥敌→速胜→两难→反噬→格局，提出"取安邑"替代"伐齐"

### 5. 范雎奸说与外戚削藩（权力崩溃）
批判穰侯越三晋伐齐+外戚专权→昭王削藩→穰侯出关

## 核心智慧

### 1. 权力的生命周期
- **建立**: 血缘+拥立+军功
- **维持**: 循环任相+经济基础+人事布局+母后支持
- **崩溃**: 战略失误+君主成熟+外部批评+替代者+财富外显

### 2. 游说的五层论证法
- 历史类比（建立参照系）
- 本质揭示（打破幻想）
- 风险推演（制造紧迫感）
- 技术论证（用数据说话）
- 替代方案（给出路）

### 3. 远交近攻的战略转型

- 创新：范雎提出"远交近攻"
