## 章节概要

**主要人物**：
- **田儋**：齐国田氏后裔，狄人，诈杀狄令自立齐王，出兵救魏被章邯夜袭战死于临济
- **田荣**：田儋从弟，收残兵，驱逐楚所立诸王，独吞三齐，第一个公开反楚，兵败后被平原人所杀
- **田横**：田荣之弟，辅立田广，坚守齐地；齐国被韩信攻破后，率五百壮士入海；奉召入朝途中自刎，五百壮士闻讯皆殉死

**核心主题**：
- 齐国田氏的复国抗争（三代人的悲剧弧线）
- "以死全义"的精神遗产
- 民心与强硬策略的张力
