## 章节主题

### 核心主题

1. **儒生文士在武功时代的位置**：郦食其（殉道型）vs 陆贾（智慧型），两种文士生存策略的对比
2. **口才外交的力量与限制**：言辞可以改变认知，但无法阻止既成的军事行动
3. **武力与文治的转型**：陆贾"马上得之不能马上治之"的思想，是汉朝治国理念转型的关键时刻
4. **"数见不鲜"的关系哲学**：陆贾晚年安排展示了对人际关系的深刻理解

## 太史公评价

《史记》将郦食其和陆贾合为一传，揭示了司马迁对"文士"这一类型的分类思考：两人都是以口辩为核心能力的"说客"，但命运迥异——郦食其以正面冲锋的方式建立功业，最终殉于外交背叛；陆贾以迂回智慧影响历史，终于安泰。
