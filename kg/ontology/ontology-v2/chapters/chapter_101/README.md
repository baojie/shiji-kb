## 章节主题

1. **改革者的结构性困境**："不改革等死，改革找死"——晁错案例是中国历史上改革悲剧的经典范本。

2. **谏言艺术与风险**：袁盎展示了高超的谏言技巧，但也展示了"好声矜贤"带来的致命风险。

3. **政策与人的分离**：好的政策不一定保护推行者。晁错的正确判断无法保护他免于被政治博弈牺牲。

4. **仇家共处一传**：两人"未尝同堂语"，却在七国之乱中命运交汇，太史公以"悲彼二子，名立身败"总结两人的共同悲剧。

## 跨领域洞察（Eureka）

本章提取出5条跨领域洞察：

- 不改革等死，改革找死——改革者的结构性困境
- "清君侧"作为反抗正当性的制造机器
- 谏言者的私怨与公义不能混用（晁错七国之乱时的致命失误）
- 声誉的负面效应——"好声"者更容易树敌
- 改革政策的"遗孤效应"——推行者死后政策反而更有生命力
