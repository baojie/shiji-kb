## 章节概述

《三王世家》是《史记》第六十篇，记述汉武帝元狩六年（前117年）封三子为王的过程及其命运。"三王"指齐王刘闳、燕王刘旦、广陵王刘胥，三人同日于太庙接受封策，同时获得针对各自封地特点的训诫册书。

本章最独特的价值在于：
1. **保存了完整的奏议文本**：从霍去病首倡到群臣三度奏请，到武帝三度制书，是汉代奏议制度的珍贵实例
2. **三份差异化册书**：武帝针对齐、燕、广陵各地人文特点，为三子分别写了完全不同的训诫，展示精准的"因地制宜"领导风格
3. **册书即预言**：三王后来的命运与册书训诫高度吻合，形成独特的"预言—验证"结构
4. **文献特殊性**：太史公原作已亡佚，现存文本为褚少孙访得封策书编列而成，是《史记》中最具文献学趣味的篇章

## 章节特点

**《三王世家》的独特价值**：

1. **汉代政治文书的标本**：三份封王册书文辞优美，被司马迁（及褚先生）评价为"文辞烂然，甚可观也"，是研究汉代政治修辞的重要文本
2. **决策程序的完整记录**：四轮奏议过程是汉代议政制度运作的最详尽实例
3. **自我实现的预言**：武帝已知三子性格弱点，在册书中预先警告，三王最终如其所戒（或所不戒）而命运分叉
4. **文献学价值**：褚先生的补记说明，揭示了《史记》文本流传中亡佚与辑录的复杂历史
