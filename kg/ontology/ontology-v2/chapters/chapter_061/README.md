## 章节特点

**《伯夷列传》是《史记》列传之首，具有特殊意义**：

1. **哲理性强**：不是简单的人物传记，而是对天道、正义、名声的哲学反思
2. **批判性强**：质疑"天道无亲，常与善人"的传统观念
3. **方法论价值**：展示太史公的史学考证方法
4. **价值观表达**：通过伯夷叔齐表达对暴力革命的保留态度

**为何将伯夷列传置于列传之首**：
- 伯夷代表了坚守原则、不随时俗的"义"
- 体现了太史公"究天人之际，通古今之变，成一家之言"的史学追求
- 为全书定下批判性思考的基调
