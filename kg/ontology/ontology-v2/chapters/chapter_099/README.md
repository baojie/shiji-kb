## 章节主题

本传核心主题：

1. **实用主义作为治国原则**：刘敬和叔孙通都不拘泥于理想方案，而以"可行性"为第一标准。

2. **时机的重要性**：娄敬的谏言、叔孙通的礼制改革，都是在正确时机做出正确行动——早一步或晚一步都不会成功。

3. **逆势判断的价值**：刘敬在"十使者皆报可击"时独持异议，被械系但被历史证明正确。
