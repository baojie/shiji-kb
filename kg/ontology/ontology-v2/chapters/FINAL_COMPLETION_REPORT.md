## 🛠️ 技术实现

### 方法论

**a2o-lite核心原则**：
1. **MECE原则**：知识单元相互独立、完全穷尽
2. **结构化保留**：世系表、年表等使用JSON格式
3. **事实技能分离**：Facts回答"是什么"，Skills回答"怎么做"
4. **质量重于数量**：每个SKU都有明确使用场景
5. **跨领域洞察**：Eureka必须是深层原理，可应用现代场景

### 工具链

1. **知识提取引擎**：Claude Code (Anthropic Claude Sonnet 4.5)
2. **方法论来源**：a2o-lite (pdf2skills.memect.cn)
3. **批处理工具**：Python脚本（10个）
4. **并行处理**：Task Agent（最多6个并发）
5. **版本控制**：Git
6. **格式标准**：Markdown + JSON + YAML frontmatter

- **完整示例章节平均扩展率**：2.5倍
  - 001章：6000字 → 12000字（2倍）
