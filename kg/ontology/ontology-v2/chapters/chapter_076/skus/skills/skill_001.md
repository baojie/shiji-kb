# Skill 001: 自我推销与机会争取

## 毛遂自荐的五个要素

1. **识别缺口**: 平原君需20人，只得19人
2. **自信表达**: "臣乃今日请处囊中耳"
3. **合理化**: "使遂蚤得处囊中，乃颖脱而出"
4. **把握时机**: 在出发前（最后时刻）
5. **证明价值**: 到楚国后立即表现

## 现代应用

### 求职面试
- 不要等待被发现，主动展示自己
- "锥处囊中"理论的局限：不是所有人都能被看见

### 创业融资
- 毛遂式pitch：直接指出投资人的需求缺口
- 到楚国的表现：路演后立即证明价值

### 职场晋升
- 主动请缨承担关键项目
- 不要等待"伯乐"，要做自己的伯乐
