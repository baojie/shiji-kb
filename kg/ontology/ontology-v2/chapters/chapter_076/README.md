# 平原君虞卿列传 — 知识图谱

本目录包含《史记·平原君虞卿列传》章节的结构化知识提取。

## 章节概述

本传合传两位赵国重要人物：平原君赵胜与谋士虞卿。

**平原君赵胜**（前308年—前251年）是赵惠文王之弟，战国四公子之一，以养士著称，门下食客数千人。其最著名的事迹是毛遂自荐——在与楚国谈判合纵时，门客毛遂主动请缨，凭借胆识和辩才说服楚王签订合纵盟约，解了邯郸之围。

**虞卿**是赵国著名谋士，曾多次在关键时刻为赵国出谋划策。长平之战后，虞卿力主抗秦、拒绝割地求和，但未被赵王采纳。后因援助魏国信陵君，受到平原君牵累而出走，流亡梁国，著《虞氏春秋》。

## 核心人物

- **赵胜（平原君）**：赵惠文王之弟，战国四公子之一，养士三千，为赵国重要政治人物
- **毛遂**：平原君门客，自荐随行出使楚国，凭"锥处囊中"之喻和刚烈辩才迫使楚王歃血为盟
- **虞卿**：赵国谋士，善于分析国际形势，力主抗秦，著《虞氏春秋》
- **楚考烈王**：被毛遂说服，与赵国签订合纵盟约
- **李同**：赵国奇士，说服平原君拿出家财犒军，自率三千敢死之士出城突击，战死

## 核心故事

1. **毛遂自荐**：秦围邯郸，赵使平原君赴楚求救。平原君选20名门客随行，差一人，毛遂自荐。楚王犹豫，毛遂以剑逼王，以"锥处囊中"之言陈明利害，楚王歃血，合纵成立。
2. **李同敢死解邯郸**：邯郸被围，粮尽，李同说服平原君散家财、出私兵，三千敢死之士出城突击，终于等到援军到来。
3. **虞卿割地辩论**：长平之战后，秦使来割地，虞卿力主先联齐拒秦，再以齐国加入为筹码与秦谈判，可以少割地甚至不割地。赵王不从，结果割地后秦仍不满，继续进攻。
4. **虞卿出走著书**：虞卿因援助魏国信陵君之事受牵连，出走梁国，穷困潦倒，著《虞氏春秋》传世。

## 历史评价

太史公评：平原君翩翩浊世之佳公子也，然未睹大体。鄙语曰"利令智昏"，平原君贪冯亭邪说，使赵陷长平兵四十余万众，邯郸几亡，是其咎也。

## 关键概念

- **毛遂自荐**：主动请求参与、展示才能以获得机会，比喻自我推荐
- **锥处囊中**：毛遂的比喻——真正的才能终究会显露，就如锥放囊中必然刺破
- **利令智昏**：因贪利而丧失判断力，平原君贪冯亭的利益导致长平之战
