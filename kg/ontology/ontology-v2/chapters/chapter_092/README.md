## 章节概要

**韩信**（?—前196年），淮阴人，汉初三杰之一，历史上最伟大的军事天才之一。

**人生轨迹**：
- 早年：胯下之辱、漂母之恩——隐忍与知恩图报
- 汉中：月下出走，被萧何追回，拜大将
- 军旅：明修栈道暗度陈仓，背水一战，连克魏赵燕齐
- 封王：自请齐王，拒绝蒯通三分建议
- 垓下：十面埋伏，灭楚，徙封楚王
- 被贬：被陈平计策擒获，降封淮阴侯
- 被诛：联络陈豨谋反，被吕后与萧何设计斩于长乐钟室

**核心矛盾**：军事天才的巅峰才能 vs 政治上的天真与自负，最终"功高震主，谋臣亡"。
