## 知识单元概览

### 事实性知识（Facts）

**fact_001**：蒙恬生平与功绩
- 将门世家（祖蒙骜、父蒙武）
- 伐齐显功，北击匈奴三十万众，收河南地
- 修长城起临洮至辽东，万余里
- 传说发明毛笔（太史公存疑）

**fact_002**：沙丘政变与矫诏赐死
- 秦始皇崩于沙丘，赵高、胡亥、李斯密谋
- 矫改遗诏：立胡亥，赐死扶苏与蒙恬
- 扶苏盲从自杀，蒙恬质疑抵制，被囚后服毒
- 此政变为秦朝速亡的直接根源

**fact_003**：太史公历史评价
- 蒙恬"阿意兴功"，不谏始皇之暴政
- 功高者有谏责，沉默即失德
- "死有遗恨"：功勋与名节之间的缺失

### 程序性知识（Skills）

**skill_001**：矫诏辨伪与抵制策略
- 判断命令来源合法性的三步框架
- 扶苏（盲从）vs 蒙恬（质疑）两种范本
- 即使结局已定，抵制的历史价值

**skill_002**：功勋将领的进谏困境
- 名将的双重职责：军功 + 谏责
- 进谏最佳时机与策略选择
- 蒙恬反例：沉默不能保全自身

## 核心主题

1. **矫诏政变**：权力真空期的信息操控与合法性伪造
2. **功臣困境**：功高者的道义责任与自保本能的冲突
3. **谏责失位**：有条件进谏却沉默，是太史公批评的核心
4. **秦亡根因**：扶苏蒙恬之死是秦朝快速覆灭的关键节点
