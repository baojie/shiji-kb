## 核心主题

### 1. 复仇的人生逻辑
太史公说："弃小义，雪大耻，名垂于后世。"伍子胥的一生是围绕复仇构建的——这个目标既是动力，也是局限。

### 2. 借力的战略智慧
无力量时，不蛮干；识别有力量的盟友，以自身价值换取其支持，将个人目标融入对方叙事。

### 3. 君臣关系的核心变量
阖庐用人不疑（成功的君臣）；夫差听信谗言（失败的君臣）。君主的用人品质决定了一切。

### 4. 谗言的普遍性
费无忌→伍奢；伯嚭→伍子胥。谗言不是偶然事件，而是权力关系中的系统性风险。
