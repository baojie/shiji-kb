# 围魏救赵战术

## 概述

"围魏救赵"是孙膑在桂陵之战中创造的经典战术，核心是**不直接对抗敌人锋芒，而是攻击其后方要害，迫使其自救**。

## 适用场景

- 敌人正在进攻第三方
- 直接救援成本高、风险大
- 敌人后方空虚
- 你有能力攻击敌人要害

## 战术原理

### 核心理论

**孙膑原文**：
"夫解杂乱纷纠者不控卷，救斗者不搏撠，批亢捣虚，形格势禁，则自为解耳"

**翻译**：
- 解开乱绳不能硬扯（会越扯越乱）
- 劝架不能自己参与（会越打越乱）
- 批亢捣虚（打击要害，攻击空虚）
- 形格势禁（局势改变，自然解除）

### 三个"不"原则

1. **不控卷**：不要硬扯（不直接对抗）
2. **不搏撠**：不要参与（不加入混战）
3. **不救援**：不要直接救（不正面迎敌）

### 三个"要"策略

1. **批亢**：打击敌人要害
2. **捣虚**：攻击敌人空虚之处
3. **形格势禁**：改变局势，迫使敌人自救

## 战场分析

### 桂陵之战案例

**背景**：
- 魏伐赵，赵急，请救于齐

**传统方案（田忌）**：
- 欲引兵之赵（直接救援）

**孙膑分析**：
1. **敌情**：
   - 今梁赵相攻
   - 轻兵锐卒必竭于外（精锐在外）
   - 老弱罢于内（内部空虚）

2. **我方优势**：
   - 机动性强
   - 可攻击魏国本土

3. **战略方案**：
   - 君不若引兵疾走大梁（直扑魏都）
   - 据其街路（控制交通要道）
   - 冲其方虚（趁其空虚）

4. **预期效果**：
   - 彼必释赵而自救（魏军必撤）
   - 一举解赵之围（赵国得救）
   - 收弊于魏（在魏国本土交战，魏军疲惫）

### 战果

- 田忌从之
- 魏果去邯郸（魏军撤离赵都）
- 与齐战于桂陵，大破梁军（在魏国境内大胜）

## 操作步骤

### 步骤1：识别敌人要害

**问题**：什么是敌人必救之地？

**桂陵案例**：
- 大梁（魏都）
- 魏军精锐在外，都城空虚
- 失去都城=亡国

**现代类比**：
- 企业总部
- 核心产品线
- 关键供应链
- 品牌声誉

**判断标准**：
- 敌人不能失去的
- 比当前目标更重要的
- 威胁后必然回救的

### 步骤2：确认敌后空虚

**孙膑分析**：
"轻兵锐卒必竭于外，老弱罢于内"

**验证方法**：
- 情报收集：敌人兵力分布
- 逻辑推理：进攻必然抽调主力
- 时机判断：敌人进攻越深入，后方越空虚

**风险**：
- 如果敌人有充足预备队→失败
- 如果敌人不回救→白费力气

### 步骤3：快速机动

**关键词**：疾走

**孙膑方案**：
"引兵疾走大梁"

**要点**：
- 速度快（趁敌人反应之前）
- 目标明确（直扑要害）
- 不拖泥带水（不在路上纠缠）

**现代应用**：
- 抢占市场空白
- 快速并购
- 突然发布新产品

### 步骤4：控制要害

**孙膑方案**：
"据其街路，冲其方虚"

**不是**：
- 真的要攻下都城
- 长期占领

**而是**：
- 造成威胁（据其街路）
- 显示能力（冲其方虚）
- 迫使回救

**现代应用**：
- 不一定真要抢客户
- 但要显示出你能抢
- 迫使对手分心

### 步骤5：等待敌人回救

**预期**：
"彼必释赵而自救"

**心理分析**：
- 都城被围vs继续攻赵
- 理性选择：回救都城
- 如果不回救，就真的攻下都城

**关键**：
- 威胁必须真实
- 敌人必须相信你会做
- 不能是虚张声势

### 步骤6：选择有利战场

**孙膑计算**：
"是我一举解赵之围而收弊于魏也"

**优势**：
- 在魏国本土作战（魏军更着急）
- 魏军疲于奔命（从赵国撤回）
- 齐军以逸待劳（提前布置）

**结果**：
桂陵大破梁军

## 决策点

### 决策点1：是否使用围魏救赵？

**必要条件**：
- ✓ 直接救援成本过高
- ✓ 敌人后方确实空虚
- ✓ 有能力攻击敌人要害
- ✓ 敌人会理性回救

**禁忌条件**：
- ✗ 敌人不在乎后方（如亡命之徒）
- ✗ 敌人有充足预备队防守
- ✗ 盟友支撑不到敌人回救
- ✗ 无法快速机动到敌后

### 决策点2：攻击哪个要害？

**选项评估**：
- 必须是敌人不能失去的
- 必须比当前目标更重要
- 必须是你能攻击到的

**桂陵案例**：
- 大梁（都城）>邯郸（盟友都城）
- 所以魏军必回救

### 决策点3：是否真的攻下？

**孙膑选择**：
- 不一定真要攻下大梁
- 目的是迫使魏军回救
- 然后在有利地形交战

**两种策略**：
1. 围点打援（目的是打援军）
2. 真攻要害（目的是夺取要地）

**桂陵是第一种**

## 变种：马陵之战

### 区别

**桂陵**：
- 纯粹的围魏救赵
- 攻大梁迫使魏军回救

**马陵**：
- 围魏救赵+减灶诱敌+伏击
- 更复杂的组合战术
- 目标是全歼而非击退

### 进化

从桂陵到马陵，孙膑的战术在进化：
1. 桂陵：围魏救赵（战略级）
2. 马陵：围魏救赵+心理战+伏击（战役级组合）

## 现代应用

### 1. 商业竞争

**场景**：竞争对手正在抢你的客户

**错误做法**：
- 价格战（直接竞争）
- 跟随策略（模仿对手）

**围魏救赵做法**：
- 攻击对手核心市场
- 挖对手核心团队
- 起诉对手侵权（迫使分心）
- 发布革命性新品（改变战场）

**案例**：
- 对手攻你三四线市场
- 你突然进入对手一二线核心市场
- 对手必须回防

### 2. 职场政治

**场景**：同事在老板面前给你穿小鞋

**错误做法**：
- 直接对抗（控卷）
- 向老板告状（搏撠）

**围魏救赵做法**：
- 找同事的上级合作（攻其要害）
- 做出成绩让老板主动关注（改变局势）
- 建立其他资源网络（降低依赖）

### 3. 谈判策略

**场景**：对方在价格上咬死不放

**错误做法**：
- 在价格上死磕（控卷）

**围魏救赵做法**：
- 提出付款方式问题（转移战场）
- 提出合作期限问题（攻其要害）
- 暗示有其他选择（形格势禁）

### 4. 项目管理

**场景**：团队陷入技术难题

**错误做法**：
- 硬啃难题（控卷）
- 加班加点（搏撠）

**围魏救赵做法**：
- 调整需求（攻其要害）
- 改变技术方案（转移战场）
- 寻找替代方案（批亢捣虚）

## 风险与限制

### 风险1：敌人不回救

**如果**：
- 敌人认为你是虚张声势
- 敌人决定先拿下当前目标
- 敌人有预备队防守

**后果**：
- 盟友被灭
- 你的进攻无效
- 战略失败

**预防**：
- 确保威胁真实
- 选择敌人真正的要害
- 做好真攻的准备

### 风险2：敌人有预备队

**如果**：
- 敌人早有准备
- 后方并不空虚

**后果**：
- 进攻受阻
- 伤亡惨重
- 浪费时机

**预防**：
- 情报先行
- 快速试探
- 准备Plan B

### 风险3：盟友支撑不住

**如果**：
- 盟友在敌人回救前就被击溃
- 来不及解围

**后果**：
- 救援失败
- 战略目标落空

**预防**：
- 评估盟友承受力
- 加快行动速度
- 必要时先增援一部分兵力

## 核心洞察

### 1. 间接路线的智慧

**利德尔·哈特**（现代军事理论家）总结：
"历史上的决定性胜利，几乎都是通过间接路线取得的"

**围魏救赵的本质**：
- 不在敌人选择的战场作战
- 把敌人引到你选择的战场
- 以我之长，击彼之短

### 2. 攻其必救

**不是所有攻击都有效**

**有效攻击**：
- 敌人必须救
- 敌人来不及救
- 敌人救不了

**围魏救赵选择**：
- 大梁（必须救）
- 疾走（来不及准备）
- 精锐在外（老弱难守）

### 3. 形格势禁

**改变局势才是关键**

**不是**：
- 消灭敌人所有兵力
- 完全控制战场

**而是**：
- 改变敌人的优先级
- 迫使敌人按你的节奏
- "则自为解耳"（自然化解）
