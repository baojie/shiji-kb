## 核心主题

### 1. 军事智慧

**孙武**：
- 令出必行（斩美姬）
- 规则威权（将在军，君命有所不受）
- 西破强楚、北威齐晋

**孙膑**：
- 围魏救赵（批亢捣虚、攻其必救）
- 诱敌伏击（减灶、马陵）
- 心理战（树上预言）

### 2. 领导力反思

**吴起的矛盾**：
- 理论高明：在德不在险
- 实践背离：刻暴少恩
- 知行不一：知道但做不到

**专家vs协调者**：
- 吴起强于田文（单项能力）
- 田文居吴起之上（综合协调）
- 职位越高，需要的能力越不同

### 3. 改革困境

**吴起变法**：
- 方向正确（富国强兵）
- 手段强硬（废公族）
- 代价惨重（被杀）

**改革者宿命**：
- 依赖君主支持
- 君主一死即亡
- 未形成社会共识

### 4. 人性洞察

**庞涓的嫉妒**：
- 自以为能不及孙膑
- 陷害同门受膑刑
- 马陵自刭成全孙膑之名

**吴起的情感操控**：
- 对士兵：吮疽、同甘共苦
- 对家人：杀妻、母死不归
- 工具理性的极致

## 使用建议

### 1. 按场景查找

使用 `mapping.md` 快速定位：
- 问题类型（团队管理、战略竞争、组织变革等）
- 历史人物（孙武、孙膑、吴起）
- 应用领域（企业管理、商业竞争、个人发展）
- 决策场景（是否严惩、是否直接对抗等）

### 2. 按知识类型学习

- **方法论**：优先阅读4个Skills
- **案例**：深入理解9个Facts
- **洞察**：拓展思维，查看15条Eureka

### 3. 深度学习路径

**路径1：纪律管理**
→ `skill_001` → `fact_001` → `eureka`（令出必行）

**路径2：间接策略**
→ `skill_002` → `skill_003` → `fact_003` → `eureka`（围魏救赵）

**路径3：领导力反思**
→ `skill_004` → `fact_006-009` → `eureka`（在德不在险、改革者宿命）
