## 按主题查找SKU

### 主题1：史学方法论

**核心问题**: 如何理解史记？如何阅读史记？

**相关SKU**:
- fact_001：提供哲学框架（司马迁的史学思想）
- skill_001：提供操作方法（七个步骤）

**使用建议**:
1. 史记入门者：先读 skill_001（获得方法） → 开始读史记原文 → 回到 fact_001（深化理解）
2. 研究者：先读 fact_001（理解框架） → 对照ontology-v1的具体案例 → 验证理论

---

### 主题2：史记的价值

**核心问题**: 为什么要读史记？史记有什么独特价值？

**相关SKU**:
- fact_001：第三维度"成一家之言"（纪传体的创新、对后世的影响）
- fact_001：跨域洞察（失败者的史学、纪传体的深层逻辑、史家的孤独）

**核心论述**:
1. **史学创新**：发明纪传体，以"人"为中心的历史叙事
2. **思想深度**："究天人之际"的哲学追问，超越简单的善恶判断
3. **文学价值**：戏剧性叙事（鸿门宴、荆轲刺秦）、悲剧精神（项羽、李广）
4. **社会视角**：不仅记录帝王将相，还记录游侠、刺客、商人、医家（《货殖列传》《游侠列传》）

---

### 主题3：司马迁的历史观

**核心问题**: 司马迁如何看待历史？如何看待天道与人事？

**相关SKU**:
- fact_001：第一维度"究天人之际"
- fact_001："天道的不确定性"+"人事的主动性"

**核心观点**:
1. **天道存疑**："余甚惑焉，傥所谓天道，是耶非耶？"（《伯夷列传》）
2. **人事为本**：历史的主角是人，历史变化的动力是人的行动
3. **史家使命**：通过记录，让善人留名，让恶人遗臭（部分"修正"天道的不公）
4. **实录精神**：忠实记录史料，但不放弃"一家之言"

---

### 主题4：历史演变的规律

**核心问题**: 历史如何变化？有规律吗？

**相关SKU**:
- fact_001：第二维度"通古今之变"
- fact_001："从三代到秦汉的结构性变化"

**核心观点**:
1. **制度演变**：从封建制到郡县制（历史必然性）
2. **权力逻辑**：集权vs分权的永恒张力
3. **历史循环**：分久必合，合久必分（但司马迁未明确提出）
4. **当下判断**：汉武帝的风险（可能重蹈秦始皇覆辙）

## 与ontology-v1的配合使用

### 互补关系

| 问题类型 | 推荐使用 | 示例问题 |
|---------|---------|---------|
| **事实查询** | ontology-v1 | "项羽是谁""鸿门宴发生了什么""商鞅变法的内容" |
| **深度理解** | ontology-v2/v1 | "为什么要读史记""如何理解司马迁的历史观" |
| **方法指导** | ontology-v2/v1 | "如何开始读史记""读史记的顺序是什么" |
| **综合主题** | 两者结合 | "春秋五霸的兴衰"（v1提供具体SKU，v2提供综合分析——待v2补充） |

### 推荐工作流

**场景1：史记入门**
```
ontology-v2/v1 skill_001（阅读指南）
    ↓
按照skill_001的建议读史记原文
    ↓
遇到具体人物/事件 → 查询ontology-v1
    ↓
读完30篇 → ontology-v2/v1 fact_001（深化理解）
```

**场景2：研究某个主题**（如"司马迁的史学思想"）
```
ontology-v2/v1 fact_001（理论框架）
    ↓
ontology-v1 具体案例（如sku_433 太史公自序、sku_003 司马迁传记）
    ↓
史记原文（《太史公自序》《报任安书》《伯夷列传》）
    ↓
回到 ontology-v2/v1 fact_001（验证理论）
```

**场景3：教学/讲座**
```
使用 ontology-v2/v1 fact_001 作为"史记导论"的理论框架
    ↓
使用 ontology-v1 的具体SKU作为案例（如刺客列传、货殖列传）
    ↓
使用 ontology-v2/v1 skill_001 作为"阅读指导"讲义
```
