# 动词标注分类体系 v3.0

## 设计原则

动词标注采用与名词实体类似的结构，使用不同的外层符号以区分：

- **名词实体**: `〖TYPE内容〗` (U+3016/3017 白色龟甲括号)
- **动词实体**: `⟦TYPE动词⟧` (U+27E6/27E7 数学双方括号)

## 标注语法

### 基本格式

```
⟦TYPE动词⟧
⟦TYPE动词|消歧说明⟧
```

### TYPE符号定义

| TYPE符号 | 类型 | 说明 | 高频词数 |
|---------|------|------|---------|
| `⟦◈verb⟧` | 军事动词 | 战争、军事行动相关 | 27个核心词 |
| `⟦◉verb⟧` | 刑罚动词 | 刑罚执行、政治斗争 | 23个核心词 |
| `⟦○verb⟧` | 政治动词 | 政治行为、外交活动 | 2个核心词 |
| `⟦◇verb⟧` | 经济动词 | 经济活动、贸易行为 | 待扩展 |

注：TYPE符号使用几何图形，与名词实体的符号类型(@=;#%等)形成区分。

## 标注示例

### 示例1：040_楚世家片段

**原标注**（错误混用）:
```
〖@帝喾〗使〖@重黎〗〖[诛〗之而不尽。帝乃以〖%庚寅〗日〖[诛〗〖@重黎〗
```

**新标注**（正确区分）:
```
〖@帝喾〗使〖@重黎〗⟦◉诛⟧之而不尽。帝乃以〖%庚寅〗日⟦◉诛⟧〖@重黎〗
```

### 示例2：军事+刑罚混合

**原标注**:
```
〖'楚〗〖[伐〗〖'随〗。〖'楚〗〖[灭〗〖'邓〗。〖[杀〗〖;令尹〗〖@子西〗，〖[斩首〗八万。
```

**新标注**:
```
〖'楚〗⟦◈伐⟧〖'随〗。〖'楚〗⟦◈灭⟧〖'邓〗。⟦◉杀⟧〖;令尹〗〖@子西〗，〖[斩首〗八万。
```

说明：
- `⟦◈伐⟧` `⟦◈灭⟧` - 军事动词
- `⟦◉杀⟧` - 刑罚动词
- `〖[斩首〗` - 刑罚制度名词（2字以上）

### 示例3：带消歧说明

**原标注**:
```
〖'吴〗〖[伐〗〖[败〗〖@子常〗，〖@子常〗亡奔〖'郑〗
```

**新标注**:
```
〖'吴〗⟦◈伐|征伐⟧⟦◈败|击败⟧〖@子常〗，〖@子常〗⟦◈亡|逃亡⟧奔〖'郑〗
```

说明：使用管道符 `|` 添加消歧说明，明确动词具体含义。
