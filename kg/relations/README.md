## 使用说明

### 关系类型说明

- **父子关系**：记录父亲→子女的直系关系
- **母子关系**：记录母亲→子女的直系关系
- **兄弟关系**：记录兄弟间的平行关系
- **夫妻关系**：记录婚姻关系（包括妻、妃等）
- **祖孙关系**：记录隔代的直系关系
- **叔侄关系**：记录旁系血亲关系

- 人物1（起点）
- 关系类型
- 人物2（终点）
- 出处章节
