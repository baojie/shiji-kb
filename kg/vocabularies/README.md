## 与命名实体索引的关系

本词表与 `kg/entities/data/entity_index.json`（命名实体索引）数据来源相同，但统计口径不同：

| | 词表（本目录） | 命名实体索引 |
|--|--------------|------------|
| 统计单位 | **表面形式**（每个不同写法独立计数） | **规范实体**（别名合并后的唯一实体） |
| 人名示例 | 沛公、汉王、高祖、高帝 各为独立词条 | 合并为「刘邦」一个条目 |
| 用途 | 词汇表/字典，查看每个表面形式的用法 | 实体目录，查询同一实体的所有称谓和出处 |
| 生成脚本 | `kg/vocabularies/scripts/build_vocabularies.py` | `kg/entities/scripts/build_entity_index.py` |

因此词表的词条数通常 ≥ 索引的条目数（人名差异最大，因古人别名众多）。

## 使用说明

每个词表文件包含以下信息：

- **词条名称**：提取的实体词汇
- **出现次数**：该词条在所有章节中出现的总次数
- **出现章节**：包含该词条的章节列表
- **典型用例**：该词条的实际使用示例（带上下文）
