# 生物实体 (Biology)

《史记》中出现的动植物实体的标注指南和词表。

## 数据文件

| 文件 | 说明 |
|------|------|
| `biology_wordlist.json` | 分类词表（家畜/野兽/鸟类/植物/神话生物等） |
| `FLORA_FAUNA_TAGGING_GUIDE.md` | 按章节的标注指南 |
| `FLORA_FAUNA_PROJECT_SUMMARY.md` | 项目概述（43种动物、27种植物） |

## 标注格式

`〖+X〗`（v2.4 迁移自旧格式 `〘X〙`）

## 分类

- 动物：家畜（马/牛/羊）、野兽（虎/狼/鹿）、鸟类（雉/乌）、神话（龙/凤凰）
- 植物：五穀（稻/麦/菽）、蔬菜、木材（桐/柏）、纤维（麻）
