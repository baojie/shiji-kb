## 核心价值与创新

### 1. 知识结构化

- **Facts（事实性知识）**：将历史事件、人物关系、制度设计转化为结构化数据
- **Skills（程序性知识）**：提取可操作的方法论、决策流程、实施步骤
- **Eureka（跨领域洞察）**：发现古代智慧的现代应用价值

### 2. 多格式支持

- **JSON格式**：适合表格化、列表化的结构数据（世系、地理、制度）
- **Markdown格式**：适合叙事性、流程性内容（方法、洞察、评价）
- **统一元数据**：所有文件包含YAML frontmatter（name、description）

### 3. 现代应用导向

每个Skill都包含"现代应用"章节，连接古今智慧：
- 因势利导 → 工程管理、组织变革
- 资源调配 → 项目管理、危机应对
- 九德人才 → 招聘面试、绩效评估

### 4. 知识复用机制

- **mapping.md**：将SKU映射到用户查询场景
- **README.md**：提供快速查询指南和概览
- **MECE原则**：知识单元相互独立、完全穷尽
