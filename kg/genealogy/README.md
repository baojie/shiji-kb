# 帝王家谱 (Imperial Genealogy)

从《史记》本纪和世家中提取的帝王世系，覆盖七个朝代。

## 数据文件

| 文件 | 说明 |
|------|------|
| `五帝朝帝王家谱.md` | 五帝世系 |
| `夏朝帝王家谱.md` | 夏朝世系 |
| `商朝帝王家谱.md` | 商朝世系 |
| `周朝帝王家谱.md` | 周朝世系 |
| `秦朝帝王家谱.md` | 秦朝世系 |
| `秦末朝帝王家谱.md` | 秦末诸王 |
| `汉朝帝王家谱.md` | 汉朝世系 |

## 统计

- 7 个朝代，868 位帝王
- 254 条家族关系（父子 85.4%、继承 12.2%、祖孙 2%、母子 0.4%）

## 脚本

| 脚本 | 功能 |
|------|------|
| `extract_imperial_genealogy.py` | 从标注文本提取帝王世系 |
| `extract_family_relations.py` | 通用家族关系提取 |
