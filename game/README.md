# 史记争霸 - 网页版游戏原型

基于《史记》历史技能的策略卡牌游戏原型。

## 🎮 如何运行

### 方法一：直接打开（推荐）
1. 在文件管理器中找到 `game` 文件夹
2. 双击 `index.html` 文件
3. 游戏将在默认浏览器中打开

### 方法二：使用本地服务器
```bash
cd /home/baojie/work/shiji-kb/game
python3 -m http.server 8000
```
然后在浏览器中访问: `http://localhost:8000`

## 📁 文件结构

```
game/
├── index.html      # 主HTML文件
├── styles.css      # 样式表
├── game.js         # 游戏逻辑
└── README.md       # 说明文档
```

## 🎯 当前功能

### ✅ 已实现
- **主菜单系统**: 历史重演、技能图鉴、游戏教程
- **技能卡牌系统**: 8个核心技能卡
  - 破釜沉舟（军事）
  - 鸿门脱险（军事）
  - 约法三章（政治）
  - 知人善任（领导）
  - 牧野之战（军事）
  - 废分封行郡县（政治）
  - 皇帝尊号制（政治）
  - 城池劝降（军事）
- **资源管理系统**: 财富、民心、军力、威望
- **技能图鉴**: 查看所有技能详情和《史记》原文
- **剧情模式**: 第一章"商周之变"（2个场景）
- **技能详情弹窗**: 显示技能效果和历史典故
- **回合制系统**: 基础的回合管理

### 🎨 视觉特色
- 传统中国风格设计（楷体字体、水墨色调）
- 金色和棕色主题（模拟竹简和古籍）
- 卡牌悬停动画效果
- 响应式设计（支持移动端）

## 🎮 游戏玩法

### 基础操作
1. **查看技能**: 点击任意技能卡查看详情
2. **使用技能**: 在技能详情弹窗中点击"使用技能"
3. **剧情选择**: 在故事模式中选择不同的行动
4. **资源管理**: 注意四大资源的平衡

### 技能系统
- 每个技能都有资源消耗
- 使用技能后会进入冷却期
- 技能效果基于真实历史事件
- 每张卡都附有《史记》原文

### 资源说明
- 💰 **财富**: 用于招募人才、维持军队
- 👥 **民心**: 影响政策推行效果
- ⚔️ **军力**: 决定战争胜负
- 📜 **威望**: 影响外交和人才吸引

## 🔧 技术栈

- **HTML5**: 页面结构
- **CSS3**: 样式和动画
- **Vanilla JavaScript**: 游戏逻辑（无框架依赖）
