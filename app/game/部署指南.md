# 《史记争霸》在线部署指南

本游戏是纯静态网页应用（HTML + CSS + JavaScript），可以部署到任何静态网站托管服务。

---

## 🚀 推荐部署方案

### 方案一：GitHub Pages（推荐）⭐⭐⭐⭐⭐

**优点**：
- ✅ 完全免费
- ✅ 自动HTTPS
- ✅ 全球CDN加速
- ✅ 支持自定义域名
- ✅ 与Git集成，方便更新

**步骤**：

#### 1. 创建GitHub仓库
```bash
cd .
git init
git add game/
git commit -m "Initial commit: 史记争霸游戏"
```

#### 2. 推送到GitHub
```bash
# 在GitHub上创建新仓库 shiji-game
git remote add origin https://github.com/你的用户名/shiji-game.git
git branch -M main
git push -u origin main
```

#### 3. 启用GitHub Pages
1. 进入仓库设置（Settings）
2. 找到"Pages"选项
3. Source选择"main"分支
4. 文件夹选择"/game"
5. 点击Save

#### 4. 访问网站
几分钟后，访问：
```
https://你的用户名.github.io/shiji-game/
```

**自动化脚本**：
```bash
#!/bin/bash
# deploy_github.sh

cd .

# 如果还没有初始化git
if [ ! -d ".git" ]; then
    git init
    git add game/
    git commit -m "Initial commit: 史记争霸游戏"
fi

# 添加所有更改
git add game/
git commit -m "Update: $(date '+%Y-%m-%d %H:%M:%S')"

# 推送到GitHub
git push origin main

echo "✅ 部署完成！"
echo "访问地址：https://你的用户名.github.io/shiji-game/"
```

**优点**：

```
https://shiji-game.vercel.app
```


## 📦 部署前准备

### 1. 创建部署配置文件

创建 `game/vercel.json`：
```json
{
  "version": 2,
  "builds": [
    {
      "src": "**/*",
      "use": "@vercel/static"
    }
  ],
  "routes": [
    {
      "src": "/(.*)",
      "dest": "/$1"
    }
  ]
}
```

创建 `game/netlify.toml`：
```toml
[build]
  publish = "."
  
[[redirects]]
  from = "/*"
  to = "/index.html"
  status = 200
```

### 2. 优化文件结构

确保文件结构清晰：
```
game/
├── index.html
├── styles.css
├── game.js
├── README.md
├── 测试地址.md
└── 更新日志.md
```

### 3. 检查文件路径

确保所有资源路径都是相对路径，不要使用绝对路径。

## 🔧 一键部署脚本

创建 `deploy.sh`：
```bash
#!/bin/bash

echo "🚀 史记争霸 - 一键部署脚本"
echo "================================"
echo ""
echo "请选择部署平台："
echo "1) GitHub Pages"
echo "2) Vercel"
echo "3) Netlify"
echo "4) Cloudflare Pages"
echo ""
read -p "请输入选项 (1-4): " choice

case $choice in
    1)
        echo "📦 部署到 GitHub Pages..."
        cd .
        git add game/
        git commit -m "Deploy: $(date '+%Y-%m-%d %H:%M:%S')"
        git push origin main
        echo "✅ 部署完成！"
        echo "访问：https://你的用户名.github.io/shiji-game/"
        ;;
    2)
        echo "📦 部署到 Vercel..."
        cd game
        vercel --prod
        ;;
    3)
        echo "📦 部署到 Netlify..."
        cd game
        netlify deploy --prod
        ;;
    4)
        echo "📦 部署到 Cloudflare Pages..."
        cd game
        wrangler pages publish . --project-name=shiji-game
        ;;
    *)
        echo "❌ 无效选项"
        exit 1
        ;;
esac
```

使用方法：
```bash
chmod +x deploy.sh
./deploy.sh
```

## 🌍 自定义域名

### GitHub Pages
1. 在仓库根目录创建`CNAME`文件
2. 写入你的域名：`game.yourdomain.com`
3. 在域名DNS设置中添加CNAME记录指向`你的用户名.github.io`

### Vercel
1. 进入项目设置
2. Domains → Add Domain
3. 输入域名并按提示配置DNS

### Netlify
1. Site settings → Domain management
2. Add custom domain
3. 按提示配置DNS

## 📈 性能优化

部署后可以进行的优化：

1. **启用CDN**：大部分平台自动提供
2. **压缩资源**：
```bash
# 压缩CSS
npx csso styles.css -o styles.min.css

# 压缩JS
npx terser game.js -o game.min.js -c -m
```

3. **添加缓存策略**：
在`vercel.json`中添加：
```json
{
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        {
          "key": "Cache-Control",
          "value": "public, max-age=31536000, immutable"
        }
      ]
    }
  ]
}
```

### Q: 可以使用自己的域名吗？
