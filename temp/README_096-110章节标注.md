# 史记096-110列传章节实体标注项目

## 项目概述

本项目对《史记》096-110列传章节进行实体标注,采用11类实体标记和圣经式段落编号,为后续知识图谱构建和学术研究提供基础数据。

## 标注体系

### 11类实体标注

| 类型 | 标记符号 | 示例 | 说明 |
|------|---------|------|------|
| 人名 | @@ | @张苍@, @郦食其@ | 所有人物姓名 |
| 地名 | == | =长安=, =荥阳= | 郡县城邑、山川河流 |
| 官职 | $$ | $丞相$, $御史大夫$ | 所有官职名称 |
| 时间 | %% | %五年%, %十月% | 年月日时、时长 |
| 朝代 | && | &汉&, &秦&, &楚& | 朝代国名 |
| 制度 | ^^ | ^礼^, ^律历^, ^法^ | 礼乐制度、法律等 |
| 族群 | ~~ | ~匈奴~, ~蛮夷~ | 民族部落名称 |
| 器物 | ** | *衣*, *剑*, *车* | 日用品、礼器、兵器 |
| 天文 | !! | !北斗!, !彗星! | 天文现象(本批次较少) |
| 神话 | ?? | ?尧?, ?舜?, ?禹? | 神话人物、传说 |
| 动植物 | 🌿🌿 | 🐯虎🐯, 🌿樱桃🌿 | 动植物名称 |

### 段落编号规范

采用圣经式编号:
- `[0]` - 标题
- `[1]`, `[2]`, `[3]` - 主要段落
- `[1.1]`, `[1.2]` - 子段落
- `[1.1.1]` - 更细分段落(视情况使用)

## 质量要求

### 一致性原则
1. **同人异称:** 同一人物的不同称呼都要标注
   例: @张苍@, @苍@, $@张@丞相$

2. **地名完整:** 所有地名不论大小都要标注
   例: =长安=, =汉中=, =霸上=

3. **时间细致:** 各种时间表达都要标注
   例: %五年%, %十月%, %一日一夜%, %万岁%

### 准确性原则
1. **官职准确:** 完整标注官职全称
2. **不添加解释:** 仅标注,不在正文中添加解释
3. **尊重原文:** 保持原文不变,仅添加标记

### 可读性原则
1. **标记紧凑:** 标记符号紧贴实体,不含空格
2. **避免过度:** 不标注虚词、语气词
3. **层次清晰:** 段落编号体现文章结构
