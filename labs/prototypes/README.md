# 原型与界面实验

前端交互原型，探索《史记》知识库的语义排版与可视化交互方式。

## 实验一：右侧批注 + 语义排版

**原型文件**: [prototype_marginal_notes.html](prototype_marginal_notes.html)

### 基于数据
- [001-notes.json](001-notes.json) - 段落级批注数据
- [001-sentences.json](001-sentences.json) - 句子级基础数据
- [001-note-sentence-map.json](001-note-sentence-map.json) - 批注与句子映射
- [001-structural-semantics.json](001-structural-semantics.json) - 结构化语义数据

### 功能特性
- 正文左侧，批注右侧布局
- 语义化排版
- 批注与正文联动高亮
- 基于：五帝本纪（001）

### 状态
原型验证完成

## 实验二：句间关系排版

**原型文件**: [排版实验_弋射说楚王.html](排版实验_弋射说楚王.html)

### 功能特性
- 因果关系可视化
- 并列/递进/总结关系标注
- 色块区分不同语义关系
- 基于：楚世家片段

### 状态
原型验证完成

## 设计理念

1. **语义优先** - 排版服务于语义理解
2. **关系可视** - 让文本关系可见
3. **交互探索** - 支持主动探索学习
4. **古今融合** - 传统文本 + 现代交互

## 后续计划

- 整合批注系统
- 支持更多篇章
- 移动端适配
- 交互优化
