## 实验三：三家注侧边栏展示

### v1 基础版
**原型文件**: [001_sanjia_sidebar.html](001_sanjia_sidebar.html)

**功能特性**:
- 正文左侧，三家注侧边栏右侧布局
- 显示集解（绿色）、索隐（粉色）、正义（棕色）三种注释
- 响应式设计，移动端自动调整布局
- 粘性定位侧边栏，方便阅读

**状态**: ✅ 基础版完成

### v2 动态连线增强版 ⭐推荐
**原型文件**: [001_sanjia_sidebar_v2.html](001_sanjia_sidebar_v2.html)

**新增功能**:
- ✨ **动态SVG连线**：锚点与注释之间绘制贝塞尔曲线，可切换显示/隐藏
- ✨ **多彩编号系统**：使用10种不同颜色区分不同的注释，连线颜色与编号颜色匹配
- ✨ **双向高亮**：鼠标悬停或点击锚点/注释时，双向高亮并绘制连线
- ✨ **自动滚动**：点击锚点自动滚动到对应注释，反之亦然
- ✨ **注释筛选**：支持全部/仅集解/仅索隐/仅正义
- ✨ **视觉反馈**：高亮时带阴影效果，提升可读性

**交互说明**:
- **悬停锚点/注释**：临时高亮并显示连线
- **点击锚点/注释**：锁定高亮状态，并滚动到对应位置
- **点击页面空白**：取消锁定状态
- **顶部控制面板**：切换连线显示和注释类型筛选

### 基于数据
- `../../data/notes/001-notes.json` - 从维基文库提取的三家注数据（集解、索隐、正义）
- `../../docs/original_text/001_五帝本纪.txt` - 原文文本

### 如何运行
```bash
# 在项目根目录运行
./labs/prototypes/serve.sh
# 然后在浏览器访问：
# v1版: http://localhost:8000/labs/prototypes/001_sanjia_sidebar.html
# v2版: http://localhost:8000/labs/prototypes/001_sanjia_sidebar_v2.html (推荐)
```

### 状态
✅ v1 基础版完成
✅ v2 动态连线增强版完成

### 技术实现
- SVG动态绘制贝塞尔曲线
- 实时计算元素位置并响应滚动
- 使用事件委托优化性能

## 实验四：三家注行下展示

**原型文件**: [001_sanjia_inline.html](001_sanjia_inline.html)

### 基于数据
- `../../data/notes/001-notes.json` - 从维基文库提取的三家注数据（集解、索隐、正义）
- `../../docs/original_text/001_五帝本纪.txt` - 原文文本

### 功能特性
- 传统纸质书籍注释风格：注释显示在段落下方
- 用上标编号[1][2][3]标记anchor位置
- 点击编号双向跳转：正文编号↔注释条目
- 高亮联动：点击后anchor和note同时高亮
- 注释筛选：支持全部显示/仅集解/仅索隐/仅正义
- 显示context（前后文），便于定位
- 三种注释用不同颜色区分：集解（绿色）、索隐（粉色）、正义（棕色）
- 基于：五帝本纪（001）

### 交互说明
- **点击正文编号[N]**：跳转到对应注释并高亮
- **点击注释编号[N]**：跳转到对应正文位置并高亮
- **使用顶部筛选按钮**：切换显示不同类型的注释

### 如何运行
```bash
# 在项目根目录运行
./labs/prototypes/serve.sh
# 然后在浏览器访问: http://localhost:8000/labs/prototypes/001_sanjia_inline.html
```

### 状态
✅ 原型完成，交互功能完整

### 与其他展示方式对比
| 展示方式 | 优点 | 缺点 | 适用场景 |
|---------|------|------|---------|
| **侧边栏** | 不打断阅读流，可同时看到正文和注释 | 占用屏幕空间，移动端体验差 | 桌面端深度阅读 |
| **弹出** | 按需显示，不占空间 | 需要额外操作，不便于对比多条注释 | 快速查阅，移动端 |
| **行下** | 接近传统书籍，便于连续阅读注释 | 拉长页面，打断正文连贯性 | 学术研究，详细注解 |
