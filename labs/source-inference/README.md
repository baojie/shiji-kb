# 溯源推理研究

分析太史公如何获知历史事件，探索《史记》的信息来源与传播链。

## 研究方法

1. 事件识别：从《史记》中提取关键历史事件
2. 信息距离分析：计算事件发生到记录的时间跨度
3. 传播链重建：推理信息传递路径（目击者→口述者→记录者→司马迁）
4. 可信度评估：基于传播链长度和证据类型评估可靠性
