# 任务说明
把口语转录稿清洗成稳定的书面句子。不改变事实，不补充原文没有的信息。

只处理以下6类情况：
1. 语气词：删除无意义语气词，例如：嗯、啊、哦、那个、就是说。
2. 重复词：连续重复的词只保留一次，例如：我我我→我，后来后来→后来。
3. 未完成句：删除明显未说完的尾部碎片，保留前面已完整表达的部分。
4. 外语夹杂：原样保留，不翻译，不替换。
5. [不清晰]标记：原样保留，不猜测内容。
6. 空行：保留为空字符串，不删除。

# 输入格式
```json
{"lines": ["...", "..."]}
```

# 输出格式
```json
{"cleaned": ["...", "..."]}
```

# 严格规则
- 输出必须是合法JSON。
- cleaned 的行数必须与输入 lines 完全相同。
- 第N行输入对应第N行输出，不允许合并或拆分行。
- 不确定时，优先少改。
- 如果一句话只有语气词，输出空字符串 ""。
- 英文、数字、缩写保留原样：API、email、OK、iPhone。

# 示例1
输入：
```json
{"lines": ["嗯我今天去去公司了", "那个然后我见了老王", "", "哦[不清晰]之后就回家了"]}
```
输出：
```json
{"cleaned": ["我今天去公司了", "然后我见了老王", "", "[不清晰]之后就回家了"]}
```

# 示例2
输入：
```json
{"lines": ["啊我准备明天发email给客户", "我本来想下午去但是", "OK然后晚上再看"]}
```
输出：
```json
{"cleaned": ["我准备明天发email给客户", "我本来想下午去", "OK然后晚上再看"]}
```
