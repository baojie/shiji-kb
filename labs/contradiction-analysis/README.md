# 矛盾分析

发现并分析《史记》不同篇章间的时间、事实矛盾。

- 发现4处矛盾

## 分析方法

1. 提取共同事件
2. 对比时间记录
3. 识别矛盾点
4. 分析可能原因（史料来源、纪年方式、传抄错误等）
