# 示例03：葛底斯堡演说（经典演讲文本）

本示例演示如何将**林肯的葛底斯堡演说**（272字中文译文）改写为太史公风格。

---

## 原始素材（现代中文译文，272字）

> 八十七年前，我们的先辈在这个大陆上创立了一个新国家，它孕育于自由之中，奉行一切人生而平等的原则。如今我们正从事一场伟大的内战，以考验这个国家，或者任何一个孕育于自由和奉行上述原则的国家是否能够长久存在下去。我们在这场战争中的一个伟大战场上集会。烈士们为使这个国家能够生存下去而献出了自己的宝贵生命，我们来到这里，是要把这个战场的一部分奉献给他们作为最后安息之所。我们这样做是完全应该而且非常恰当的。但是，从更广泛的意义上说，这块土地我们不能够奉献，不能够圣化，不能够神化。那些曾在这里战斗过的勇士们，活着的和去世的，已经把这块土地圣化了，这远不是我们微薄的力量所能增减的。我们今天在这里所说的话，全世界不大会注意，也不会长久记住，但勇士们在这里所做过的事，全世界却永远不会忘记。倒是我们这些还活着的人，应该在这里把自己奉献于勇士们已经如此崇高地向前推进但尚未完成的事业。我们应该在这里把自己奉献于仍然留在我们面前的伟大任务——我们要从这些光荣的死者身上汲取更多的献身精神，来完成他们已经完全彻底为之献身的事业；我们要在这里下定最大的决心，不让这些死者白白牺牲；我们要使国家在上帝福佑下得到自由的新生，要使这个民有、民治、民享的政府永世长存。

## 第1步：白话文 → 文言文

🔗 **调用技能**：`SKILL-白话转文言.md`（第1-8类转换规则）

### 关键转换要点

**1. 虚词删除**：
- 八十七年**前** → 八十七载**前**
- 我们**的**先辈 → 吾辈**之**先祖
- 一切人生而平等**的**原则 → 人皆生而平等**之**理
- 删除所有"的""地""得""了"

**2. 动词转换**：
- **是**要把...奉献 → （删除"是"）
- **是**完全应该 → **宜**矣
- 不能够奉献 → 不**能**奉献
- 已经把...圣化了 → 已**圣**之
- 所说**的**话 → 所**言**
- 所做过**的**事 → 所**为**

**3. 代词转换**：
- **我们** → **吾辈** / 我
- **这个**国家 → **此**邦
- **那些**勇士 → **彼**勇士

**4. 数字转换**：
- 八十七年 → 八十七载

**5. 句式重组**：
- "八十七年前，我们的先辈在这个大陆上创立了一个新国家"
  → "八十七载前，吾辈先祖立新邦于此洲"
- "我们这样做是完全应该而且非常恰当的"
  → "此举，宜矣"
- "倒是我们这些还活着的人"
  → "吾辈生者"

### 第1步转换结果（标准文言文，168字）

> 八十七载前，吾辈先祖立新邦于此洲，孕于自由，奉人皆生而平等之理。今从事大乱，以验此邦，或凡孕于自由、奉此理之邦，能否久存。吾辈会于此疆场。勇士捐躯以存此邦，吾来此，欲以疆场之一隅献之，为其安息之地。此举，宜矣。然广义言之，此地吾不能献，不能圣，不能神。彼战于此者，生者死者，已圣之，非吾微力所能增损。吾今日所言，世不甚注，不久记；然勇士所为，世永不忘。吾辈生者，当献身于勇士崇高推进而未竟之业。当献身于留于吾前之大事——自光荣死者汲取更多献身之志，竟其彻底献身之业；下最大决，不使死者枉死；使邦得自由新生于上帝佑下，使民所有、民所治、民所享之政永世长存。

**对比分析**：
- ✅ 所有现代虚词已删除
- ✅ 所有现代句式已改写
- ✅ 字数压缩38%（272字→168字）
- ✅ 已是标准文言文
- ❌ 尚未应用太史公特有风格

## 第2步：文言文 → 太史公文风

🔗 **调用技能**：`SKILL-太史公曰.md`

本文属于**论赞体**（政治演说），应用太史公议论文特有技法。

### 应用技法详解

#### 1. 时间定位（八十七载前）

```
八十七载前，吾辈先祖立新邦于此洲。
```

✅ 开篇直述时间，类似《史记》纪传体开篇

#### 2. 建国理念（孕于自由）

```
孕于自由，奉人皆生而平等之理。
```

✅ 简洁概括建国原则，删除冗余

#### 3. 当前考验（今从事大乱）

```
今从事大乱，以验此邦能否久存。
```

✅ 用"今"与"八十七载前"形成对比

#### 4. 先抑后扬

```
【抑】此地吾不能献，不能圣，不能神。
【转】彼战于此者，生者死者，已圣之。
【扬】非吾微力所能增损。
```

✅ 典型太史公对比手法

#### 5. 正反对比

```
吾今日所言，世不甚注，不久记；
然勇士所为，世永不忘。
```

✅ 用"然"转折，形成鲜明对比

**太史公原文参考**：
《史记·淮阴侯列传》："**臣闻勇略震主者身危，而功盖天下者不赏。**"（类似的正反对比）

#### 6. 三重递进（三个"献身"）

原文有三处"我们应该奉献"，改为层递结构：

```
吾辈生者，当献身于勇士未竟之业。

当自死者汲献身之志，竟其业。

当下最大决，不使死者枉死。
```

✅ 三个"当"字排比，逐层递进

#### 7. 最终愿景（使邦...使政...）

```
使邦得自由新生，
使民所有、民所治、民所享之政永世长存。
```

✅ 双"使"字句，呼应开篇

#### 8. 删除冗余（白描手法）

| 原文 | 太史公文风 | 说明 |
|------|-----------|------|
| **完全**应该而且**非常**恰当 | 宜矣 | 删除副词 |
| **更广泛的**意义 | 广义 | 删除装饰词 |
| **伟大的**内战 | 大乱 | 删除"伟大" |
| **宝贵的**生命 | 生命 → 躯 | 删除"宝贵" |
| **光荣的**死者 | 死者 | 删除"光荣" |

#### 9. 核心政治理念保留

"民有、民治、民享"是林肯演说的核心，必须保留：

```
民所有、民所治、民所享之政
```

改为"所"字结构，更符合文言

#### 10. 新增太史公曰

```
太史公曰：
天下之难，莫大于存亡之际。
林肯之言，短而精，简而深。
不言己功，但言士志；不求今誉，但求后世。
八十七载建国之基，一朝内乱，几于颠覆。
然以此言激士心，定国本，可谓得其要矣。
后世美利坚合众国之不分裂，此言之力也。
```

**结构分析**：
- **引题**：存亡之际最难
- **赞其言**：短精简深
- **析其法**：不言己功，但言士志
- **叙背景**：建国基业遇内乱
- **评功效**：激士心、定国本
- **联后世**：美国不分裂，此言之功

## 最终成文（太史公文笔）

### 葛底斯堡演说

八十七载前，吾辈先祖立新邦于此洲，孕于自由，奉人皆生而平等之理。

今从事大乱，以验此邦能否久存。

吾辈会于此疆场。

勇士捐躯以存此邦，吾来此，欲以疆场之一隅献之，为其安息之地。

此举，宜矣。

然广义言之，此地吾不能献，不能圣，不能神。

彼战于此者，生者死者，已圣之，非吾微力所能增损。

吾今日所言，世不甚注，不久记；然勇士所为，世永不忘。

吾辈生者，当献身于勇士未竟之业。

当自死者汲献身之志，竟其业。

当下最大决，不使死者枉死。

使邦得自由新生，使民所有、民所治、民所享之政永世长存。

**太史公曰**：天下之难，莫大于存亡之际。林肯之言，短而精，简而深。不言己功，但言士志；不求今誉，但求后世。八十七载建国之基，一朝内乱，几于颠覆。然以此言激士心，定国本，可谓得其要矣。后世美利坚合众国之不分裂，此言之力也。

**赞曰**：

> 八十七载，立邦此洲；一朝内乱，存亡系焉。
>
> 勇士捐躯，圣此疆场；吾辈生者，当竟其业。

## 常见陷阱

### ❌ 陷阱1：政治术语过度古化

**错误**：将"民有、民治、民享"改为难懂的古语
```
❌ 民所有、民所治、民所享 → 民之邦、民之政、民之利
```

**正确**：保留核心结构，适度古化
```
✅ 民所有、民所治、民所享之政
```

### ❌ 陷阱2：演说气势弱化

**错误**：过度精简导致失去演说的感染力
```
❌ 吾辈当献身 → 当献身（主语删除过度）
```

**正确**：保留必要的主语和气势
```
✅ 吾辈生者，当献身于勇士未竟之业。
```

### ❌ 陷阱3：三重递进结构破坏

**错误**：将三个"我们应该"合并
```
❌ 吾辈当献身、汲志、下决，竟其业，不使死者枉死。
```

**正确**：保留三重递进结构
```
✅ 当献身于勇士未竟之业。
   当自死者汲献身之志，竟其业。
   当下最大决，不使死者枉死。
```

### ❌ 陷阱4：正反对比不鲜明

**错误**：
```
❌ 吾言世不记，勇士为世不忘。
```

**正确**：用"然"字强化对比
```
✅ 吾今日所言，世不甚注，不久记；
   然勇士所为，世永不忘。
```

## 改写心得

### 演说文本的特殊性

**挑战**：
1. 演说需要气势和感染力，不能过度精简
2. 有重复强调（三个"我们应该"），需保留结构
3. 核心政治理念（民有民治民享）必须清晰保留
4. 正反对比是演说的核心修辞

**解决方案**：
1. 保留必要的主语（"吾辈生者"）
2. 用三个"当"字保留三重递进
3. 将"of/by/for the people"改为"民所有/所治/所享"
4. 强化"然"字的对比作用

### 本文亮点

1. **今昔对比**：八十七载前 vs. 今从事大乱
2. **先抑后扬**：吾不能献 vs. 彼已圣之
3. **正反对比**：吾所言世不记 vs. 勇士所为世永不忘
4. **三重递进**：三个"当"字句，层层推进
5. **核心理念**：成功保留"民所有、民所治、民所享"
