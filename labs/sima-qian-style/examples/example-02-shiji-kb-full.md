#### 2. 列举式（所为有二：其一...其二...）

```

其一，以智能之术化《史记》百三十篇、五十七万言为条理之学。
```

✅ 已应用，符合太史公列举事项的典型句式

**为何用"所为有二"而非"做了两件事"？**
- "所为"是文言常用表达（所做之事）
- "有二"比"两件"更简洁古雅
- 参考《史记·项羽本纪》："项籍少时，学书不成，去；学剑，又不成。项梁怒之。籍曰：'书足以记名姓而已。剑一人敌，不足学，学万人敌。'於是项梁乃教籍兵法，**籍大喜，略知其意，又不肯竟学**。"（类似的列举结构）

#### 3. 层递并列（短句排比）

```
类十有八，注近十万；
事三千馀，系七千馀，
学条六百馀。
```

**节奏分析**：
- 三段式数据排比
- 句式相似：[名词][数量]
- 分号断句，节奏感强
- 最后一句不加"粲然备矣"，因为后文还有"靡不具也"

**太史公原文参考**：
《史记·货殖列传》："故曰：**陆地牧马二百蹄，牛蹄角千，千足羊，泽中千足彘**，水居千石鱼陂...各千金。"（类似的数据排比）

#### 4. 简短断言

```

新学府立成。（新学府立即建成）
竟其事。（完成了）
```

**特征**：

- 节奏紧凑有力

**太史公原文参考**：

## 常见陷阱

### ❌ 陷阱1：第0步遗漏现代专业术语

**错误**：保留"知识库构造方法论""管线""参数"等现代术语
```
❌ shiji-kb者，开源古籍知识库也。以AI化《史记》为结构化知识...
```

**正确**：必须先古化所有现代专业名词
```
✅ shiji-kb者，开源古籍学问事业也。以智能之术化《史记》为条理之学...
```

### ❌ 陷阱2：现代比喻词未处理

**错误**：保留"瓶颈""挖掘""初心"等现代比喻和流行语
```
❌ 古之治文史者，困於人力瓶颈。宝库未被充分挖掘。
```

**正确**：删除或改为文言表达
```
✅ 古之治文史者，困於人力。宝库深藏，未尽其用。
```

### ❌ 陷阱3：大数不用概数

**错误**：`三千一百八十五个事件，七千六百五十二条关系`

**正确**：`事三千馀，系七千馀`（太史公常用"馀"表约数）

### ❌ 陷阱4：列举式不统一

**错误**：
```
❌ 做了两件事：一是...，二是...
❌ 有两件事：第一...第二...
```

**正确**：
```
✅ 所为有二：其一...其二...
```

### ❌ 陷阱5：保留过渡性表达

**错误**：`这个项目最开始的初心是要回答一直以来的问题`

**正确**：`尝自问`（跳跃式剪辑，删除冗余）
