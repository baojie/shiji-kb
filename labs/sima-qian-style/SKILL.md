# 太史公曰（Taishi Gong Yue - Sima Qian's Style）

将现代素材以司马迁《史记》的文笔风格重写。

---

## 技能分层

本技能采用**三层架构**：

1. **预处理层**：调用 `references/modern-terms.md` 将现代名词转为古风表达（如：知识图谱→学府、AI→智能之术）
2. **基础层**：调用 `references/classical-chinese.md` 完成现代汉语→文言文转换
3. **风格层**：在文言基础上，应用太史公特有句法和修辞（参考 `references/style-features.md`）

---

## 工作流（三步转换法）

### 第一步：现代名词 → 古风名词

**🔗 调用技能**：`references/modern-terms.md`

将现代专有名词转为古风雅称。完整词典见该文档（7大类名词转换）。

**核心要点**：
- ✅ 技术概念：知识图谱→学府、AI→智能之术、GitHub→代码库
- ✅ 现代职业：程序员→工程师、产品经理→运筹官
- ✅ 现代地名：硅谷→矽谷、纽约→纽约
- ✅ 现代物品：电脑→计算机、手机→通讯器
- ✅ 抽象概念：创新→革新、迭代→更迭

**结果**：专有名词古化，但语法仍为现代汉语

---

### 第二步：现代汉语 → 文言文

**🔗 调用技能**：`references/classical-chinese.md`

将现代白话彻底转换为文言文。完整规则见该文档（9大类转换规则、5大句式、自检清单）。

**核心要点**：
- ✅ 删除所有"的、地、得、了、吗、呢、着"
- ✅ 替换所有现代动词（是→为，有→具，被→为...所...）
- ✅ 改写所有现代句式（被动句、疑问句、判断句）
- ✅ 数字汉字化：130 → 百三十，3,185 → 三千馀（概数化）
- ✅ 使用文言专用连词、副词、代词

**结果**：完全文言化的文本，无任何现代白话痕迹

---

### 第三步：文言文 → 太史公文风

**核心原则**：在文言基础上，应用太史公特有句法和修辞

#### 1. 解析与分流

识别素材类型，选择对应文体：

| 素材类型 | 文体 | 特征 |
|---------|------|------|
| 人物传记 | **列传体** | 以人物为中心，突出关键转折，惜墨如金 |
| 历史事件 | **本纪体** | 编年叙事，简洁记事，跳跃式剪辑 |
| 评论议论 | **论赞体** | 引经据典，排比对仗，寓褒贬于叙事 |

#### 2. 应用太史公句式

参考 `references/style-features.md`，应用以下核心句式：

##### 2.1 开篇定式

**模式**：`X者，Y也`

```
史记知识库者，开源古籍学问事业也。
西瓜者，名鲍捷，治智能知识工程三十载，好文史。
```

**变体**：
- 籍贯式：`X者，Y人也`（项籍者，下相人也）
- 职业式：`X者，为Y也`

##### 2.2 列举式

**模式**：`其所为有二：其一...其二...`

```
其所为有二：
其一，以智能之术，化《史记》百三十篇...
其二，沉淀其法为技能文档二十六篇...
```

**注意**：用"其一/其二"而非"一曰/二曰"（后者常用于引述对话）

##### 2.3 层递并列

**模式**：`A，B，C`（动词连用，省略主语，节奏渐强）

```
类十有八，标近十万；事三千馀，系七千馀，单元六百馀
（数据排比，节奏感强）

学书不成，去学剑，又不成。
（反复强调，凸显志向）

易一书，线不改，调参数，新库立成。
（四短句，层层递进）
```

##### 2.4 因果承转

**模式**：`A，故B` / `A，以是B` / `A，然B`

```
古之治文史者，困於人力。（A）
华夏典籍以亿言计，宝库深藏，未尽其用。（进一步说明A）

以其笃学，故绩甚佳。（以A，故B）

虽难，然必为之。（A，然B）
```

##### 2.5 简短断言

**特征**：单句陈述，不加修饰，直击要害，常作收束

```
粲然备矣。（华丽完备了）
靡不具也。（无不具备）
新库立成。（新库立即成功）
竟其功。（完成了任务）
```

##### 2.6 设问自答

**模式**：问题（常用连续两问，递进强化）+ 简短回答

```
尝自问：智能之术，能压皓首穷经之数十载为旬月乎？能令知识活用乎？
（连续两问，递进）

今以一人携群智能体，竟其功。
（简短回答：完成了）
```

#### 3. 应用太史公修辞

##### 3.1 反复强调

**技法**：重复关键词，递进加强

```
学书不成，去学剑，又不成。
（"不成"重复 → 凸显项羽志向之大）

能...乎？能...乎？
（连续设问 → 强化疑问）

命也夫，命也夫！
（"命也夫"重复 → 司马谈的悲愤）
```

##### 3.2 先抑后扬

**模式**：`A则B，然其C`

```
古之治文史者，困於人力。（抑：古人困境）
↓
今以一人携群智能体，竟其功。（扬：今人突破）

儒者博而寡要，劳而少功；然其序君臣父子之礼，不可易也。
（先抑：博而寡要 → 后扬：礼不可易）
```

##### 3.3 排比对仗

**技法**：句式整齐，音律工整，常用于总结性论述

```
类十有八，标近十万；
事三千馀，系七千馀，
单元六百馀，粲然备矣。
（数据排比，三段式，节奏分明）

太史公曰（对仗）：
古人穷经，皓首一生，不过数卷。
今藉智能，旬月可竟全史。
（古 vs 今，皓首一生 vs 旬月，数卷 vs 全史）
```

##### 3.4 白描手法

**原则**：删除所有装饰性形容词，只写动作和结果

```
❌ 繁冗：项籍看到秦始皇威严的仪仗队浩浩荡荡地过江，心中顿时涌起无限豪情...
✅ 太史公：秦始皇帝游会稽，渡浙江，梁与籍俱观。籍曰："彼可取而代也。"

❌ 装饰：非常重要的核心成果
✅ 白描：成果（直接删除"非常重要的核心"）
```

##### 3.5 对话场景的动作穿插

**原则**：《史记》对话场景 = 对话 + 动作 + 情绪 + 氛围，参考《鸿门宴》

**模板**：
```
[人物A][动作]，曰："[对话]。"

[人物B][情绪/反应]。

[时间标记]，[人物C][位置标记]曰："[对话]。"

[氛围标记]。[人物D][动作]曰："[对话]。"
```

**常用元素**：
- **动作**：起、坐、出、入、归、视、拔剑、举杯
- **情绪**：默然（沉默）、怒、喜、惧
- **时间**：顷之（过一会儿）、良久、俄而、会罢
- **位置**：从旁、前、退、在侧
- **氛围**：坐皆默、左右皆泣、莫敢仰视

**示例**（会议争论场景）：
```
会议。

产品起，曰："以此为枢机，竟其环。"

吾曰："架构不能承。"产品默然。

顷之，复言曰："当专其要，速益之。"

行人从旁曰："当积法，为可传之策。"

工师张怒，斥曰："虚需也，未思明！"

坐皆默。上徐曰："先为简版，试而改之。"

会罢，吾归，视屏良久，书周报四字："本周对齐。"
```

**要点**：
- 不能只有对话，必须穿插动作/情绪/氛围
- 动作展现人物状态（起=发言前、默然=被驳倒、怒=不满）
- 时间标记控制节奏（顷之=缓一缓、会罢=结束）
- 氛围标记渲染场景（坐皆默=尴尬冷场）

##### 3.6 跳跃式剪辑

**技法**：省略过渡，直接跳到下一场景，时间跨度可能数年

```
[场景1] 项梁杀人，与籍避仇於吴中。
[场景2] 吴中贤士大夫皆出项梁下。
[场景3] 秦始皇帝游会稽...

（省略中间数年经历，不交代如何在吴中建立声望）
```

#### 4. 人物刻画技法

##### 4.1 关键细节定性格

**原则**：一个细节胜过千言万语

```
项籍曰："书足以记名姓而已。剑一人敌，不足学，学万人敌。"
→ 性格：雄心勃勃、不拘小节（不直说，让对话说话）

西瓜者，名鲍捷，治智能知识工程三十载，好文史。
→ 能力+爱好（简洁勾勒人物）
```

##### 4.2 行为即性格

**原则**：不评价，让行为说话

```
❌ 说教：项羽是个骄傲自大的人
✅ 太史公：（后文）项王不肯竟学兵法、分封诸侯失当等行为
→ 读者自己得出结论
```

##### 4.3 他人之口借力

**原则**：用旁人评价暗示主角地位

```
吴中贤士大夫皆出项梁下。
（侧面写项梁声望，不直说"项梁很厉害"）

梁以此奇籍。
（借项梁之眼，写项羽不凡）
```

#### 5. 太史公曰（可选）

**结构**：引古（可选）→ 核心论断（极简）→ 升华哲理（可选）

**精简原则**（重要）：
- 太史公曰应该极简，**50-80字**为宜
- 删除正文已详述的内容（不重复）
- 删除冗余修饰（直接切入）
- 删除多余说明（点到为止）
- 保留精华对仗（简洁有力）

**示例1**（完整版，带引古和升华）：
```
太史公曰：
古人穷经，皓首一生，不过数卷。        （引古）
今藉智能，旬月可竟全史，奇哉！        （联今）
然工具虽利，志在人也。                （评判）
西瓜三十年磨一剑，方得此业。          （具体化）
技可授，志不可夺也。                  （升华哲理）
```

**示例2**（精简版，直接论断）：
```
太史公曰：

四家争执，各执其词，莫知所从。

上决之以"速试"，此非决也，推责耳。

散会书报，止五字而已。

言愈繁，行愈寡；会愈多，事愈滞。
```
（52字，无引古，直击要害）

**删减示例**：

❌ **冗长版**（106字）：
```
天下会议之乱，莫过于各执一词而无定见。
产品言抓手闭环，技言承载之困，运营言方法论，研发斥虚需。四家纷纭，谁得其实？
终赖长官一言决之：作简版，速试错。
此非真决也，乃推责于试错之名耳。
及散会，牛马开周报，书五字应付，此职场之常态也。
言之愈繁，行之愈寡；会之愈多，事之愈滞。
可不慎哉！
```

✅ **精简版**（52字）：
```
四家争执，各执其词，莫知所从。
上决之以"速试"，此非决也，推责耳。
散会书报，止五字而已。
言愈繁，行愈寡；会愈多，事愈滞。
```

**删减要点**：
1. 删除重复（"产品言抓手闭环"等正文已述）
2. 删除冗余开头（"天下会议之乱，莫过于"→"四家争执"）
3. 删除多余说明（"此职场之常态也"）
4. 删除疑问句（"谁得其实？"）
5. 删除感叹收束（"可不慎哉！"）

#### 6. 自检与润色

- [ ] 确保无现代白话痕迹（"的""了""吗""很""非常"等）
- [ ] 对话只截取关键一句
- [ ] 用行为代替心理描写
- [ ] 句式简短有力（平均<15字/句）
- [ ] 应用至少3种太史公句式（开篇定式、层递并列等）
- [ ] 应用至少2种太史公修辞（白描、先抑后扬等）
- [ ] **严禁**：直接复用《史记》原文语句

#### 7. 输出成文

- 仅输出最终文本（Markdown格式）
- 可选章节小标题（### 格式）
- 末尾可附"太史公曰"式评论段落（可选）

## 注意事项

### 1. 语言时代

- 使用文言文（非现代白话）
- 避免网络用语、新词
- 年份可用公元纪年（如"公元1976年"）或传统纪年（如"癸丑年"）

### 2. 价值中立

- 不强加现代价值观
- 不美化或丑化历史人物
- 保持太史公的"实录精神"（客观记事，寓褒贬于细节）

### 3. 篇幅控制

- 根据素材量适度扩充或压缩
- 典型列传体: 300-800字
- 短评（太史公曰）: 50-150字

### 4. 信息密度

- 文言文信息密度高，字数通常压缩30-40%
- 240字现代白话 → 约160字文言文
