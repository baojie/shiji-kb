# 白话转文言（Modern Chinese to Classical Chinese）

将现代汉语（白话文）彻底转换为文言文。

---

## 核心原则

**完整的文言化，不保留任何现代白话痕迹**

- ✅ 删除所有现代虚词（的、地、得、了、吗、呢、着）
- ✅ 替换所有现代动词（是、有、被、把、让、使）
- ✅ 改写所有现代句式（被动句、把字句、疑问句）
- ✅ 数字全部汉字化并适度倒装
- ✅ 使用文言专用连词、副词、代词

---

## 转换规则（9大类）

### 0. 现代名词古化（调用子SKILL）

**🔗 调用技能**：`SKILL-现代名词古化.md`

在进行虚词、动词、句式转换之前，**首先**要将现代学术、科技、社会名词转化为古代汉语表达。

**必须古化的现代名词**：
- 学术类：知识库、知识图谱、知识体系、数据、结构化、标注、实体、关系、类型...
- 科技类：人工智能、AI、算法、模型、系统、框架、工程、项目、技术、平台...
- 管理类：团队、成员、领导、管理、组织、机制、流程、标准...

**快速参考表**（详见`SKILL-现代名词古化.md`）：

| 现代名词 | 古化方案 | 示例 |
|---------|---------|------|
| 知识库 | 学府 / 书藏 / 典库 | 知识库 → 典库 |
| 知识图谱 | 知识经纬 / 学问图志 | 知识图谱 → 知识经纬 |
| 知识体系 | 学统 / 典章 | 知识体系 → 学统 |
| 数据 | 数 / 记录 / 簿籍 | 数据 → 数 |
| 结构化 | 条理化 / 编次 | 结构化知识 → 条理之学 |
| 标注 | 注 / 疏 / 笺 | 标注 → 注 |
| 实体 | 名物 / 物事 | 实体类型 → 名物之类 |
| 关系 | 系 / 联 / 缘 | 关系 → 系 |
| 事件 | 事 / 故 / 变 | 事件 → 事 |
| 人工智能 | 智能之术 / 机巧之学 | AI → 智能之术 |
| AI Agent | 智能体 / 机灵 | AI Agent → 智能体 |
| 算法 | 算术 / 算法 | 算法（古已有之） |
| 工程 | 事 / 业 / 作 | 知识工程 → 治学之事 |
| 项目 | 事 / 业 / 工 | 项目 → 事 |
| 技能文档 | 技艺文 / 技法篇 | 技能文档 → 技艺文 |
| 研究 | 治 / 究 / 考 | 研究历史 → 治史 |

**使用流程**：
1. 扫描原文，识别所有现代名词
2. 查阅`SKILL-现代名词古化.md`词典
3. 选择合适的古化方案（优先"古代对应物"，次选"功能描述"）
4. 替换现代名词
5. 进行后续的虚词、动词、句式转换

**示例**：
```
原文：这是一个开源的古籍知识工程项目。用AI Agent把《史记》转化成结构化知识图谱。

第0步（现代名词古化）：
- 知识工程项目 → 治学之事
- AI Agent → 智能体
- 结构化知识图谱 → 条理之知识经纬 → 知识经纬

初步古化：这是一个开源的古籍治学之事。用智能体把《史记》转化成知识经纬。

（然后进行第1-8步的虚词、动词、句式转换...）
```

### 1. 虚词转换

#### 1.1 助词

| 现代白话 | 文言文 | 说明 |
|---------|--------|------|
| **的** | 之 / 删除 | "他的书" → "其书"（删除）<br>"美丽的花" → "美花"（删除）<br>"中国的历史" → "中国之史"（保留"之"） |
| **地** | 删除 | "快速地跑" → "疾走"<br>"认真地学习" → "笃学" |
| **得** | 删除 / 以 | "跑得很快" → "走甚疾"<br>"做得好" → "为之善" |
| **了** | 矣 / 已 / 删除 | "完成了" → "竟矣" / "既竟"<br>"他走了" → "其去矣" |
| **着** | 删除 / 焉 | "看着书" → "观书"<br>"坐着" → "坐焉" |
| **吗/呢** | 乎 / 哉 / 耶 | "好吗？" → "善乎？"<br>"是吗？" → "然乎？" |

**规则**：
- "的"：90%情况直接删除，仅在歧义时改"之"
- "地/得"：100%删除
- "了"：句尾可改"矣"，句中删除
- "着"：100%删除
- "吗/呢"：疑问句改"乎/哉/耶"

#### 1.2 语气词

| 现代白话 | 文言文 |
|---------|--------|
| 啊、呀 | 哉、乎、矣 |
| 吧 | 也、焉 |
| 吗 | 乎、耶 |
| 呢 | 乎、哉 |

### 2. 动词转换

#### 2.1 判断动词

| 现代白话 | 文言文 | 示例 |
|---------|--------|------|
| **是** | 为 / 乃 / 也 | "他是学生" → "其为学生" / "学生也" |
| **有** | 具 / 持 / 存 / 删除 | "有三个人" → "三人" / "人三"<br>"有很多书" → "书甚多" |
| **没有** | 无 / 莫 / 未 | "没有钱" → "无钱"<br>"没有去" → "未去" |

#### 2.2 使役动词

| 现代白话 | 文言文 | 示例 |
|---------|--------|------|
| **被...** | 为...所... / 见...于... | "被打败" → "为敌所败"<br>"被抓" → "见执于吏" |

| **让...** | 使.../ 令... | "让他去" → "使之去" / "令其往" |
| **使...** | 使.../ 令... | 保留（文言也用"使"） |

### 7. 数字转换

#### 7.1 基本规则

| 现代写法 | 文言写法 | 说明 |
|---------|---------|------|
| 阿拉伯数字 | 汉字 | 130 → 百三十 / 一百三十 |
| 小数字 | 汉字小写 | 1-10 → 一、二、三...十 |
| 大数字 | 汉字+单位 | 10000 → 万 / 一万 |
| 序数 | 删"第" | 第一 → 一 / 其一 |

#### 7.2 特殊表达

| 现代写法 | 文言写法 | 说明 |
|---------|---------|------|
| 18类 | 类十有八 | **倒装**（更古雅） |
| 10万次 | 十万 / 十万度 | 删除量词"次" |
| 3,185个 | 三千馀 | **概数化**（太史公常用"馀"） |
| 130篇 | 百三十篇 / 一百三十篇 | |
| 50% | 半 / 过半 | 百分比改描述 |
| 1000+ | 千馀 / 逾千 | |

**概数规则**：
- 精确数 → 概数（3,185 → 三千馀）
- 用"馀""许""左右""上下"表约数
- 太史公文风特别爱用"馀"

#### 7.3 倒装规则

| 现代语序 | 文言倒装 |
|---------|---------|

| 57万字 | 五十七万言 / 字五十七万 |
| 三个朋友 | 友三 / 朋三人 |

**何时倒装**：
- 强调数量时，数字后置
- 更符合古文节奏

## 句式转换（5大类）

### 1. 判断句

| 现代白话 | 文言文 | 类型 |
|---------|--------|------|
| 他**是**老师 | 其**为**师 / 师**也** | "是" → "为" / "也" |
| 这**是**书 | 此**乃**书 / 书**也** | "是" → "乃" / "也" |
| 我**是**学生 | 余学生**也** | "是" → "也"（放句尾） |

**规则**：
- "是" → "为" / "乃"（动词）
- "是" → "也"（助词，放句尾）
- "是" → 直接删除，用"者...也"结构

### 2. 被动句

| 现代白话 | 文言文 | 类型 |
|---------|--------|------|
| **被**打败 | **为敌所**败 | 为...所... |
| **被**抓住 | **见执于**吏 | 见...于... |
| **让**他去 | **使之**往 | 使令句 |
| **把**书拿来 | **取书**来 | 改语序，删"把" |

**规则**：
- "被" → "为...所..."（最常用）
- "被" → "见...于..."（较少用）
- "把" → 直接删除，改语序（宾语前置）

### 3. 疑问句

| 现代白话 | 文言文 | 疑问词 |
|---------|--------|--------|
| 好**吗**？ | 善**乎**？ | 乎 |
| 是**吗**？ | 然**乎**？/ 然**耶**？ | 乎 / 耶 |
| 对**吗**？ | 可**乎**？ | 乎 |
| 去**吗**？ | 往**乎**？ | 乎 |
| 怎么办？ | **奈何**？/ **如何**？ | 奈何 / 如何 |
| 什么事？ | **何**事？ | 何 |
| 在哪里？ | 安在？/ 在**焉**？ | 安 / 焉 |

**规则**：
- "吗/呢" → "乎"（最常用）
- "吗" → "耶"（表反问）
- "吗" → "哉"（表感叹）

### 4. 否定句

| 现代白话 | 文言文 | 否定词 |
|---------|--------|--------|
| **不**去 | **不**往 / **弗**往 | 不 / 弗 |
| **没有**去 | **未**往 / **不**往 | 未（表未然） |
| **没有**钱 | **无**钱 | 无 |
| **不要**去 | **毋**往 / **莫**往 | 毋 / 莫（禁止） |
| **不能** | **不能** / **弗能** | 保留或改"弗" |

**规则**：
- "不" → "不"（可保留）/ "弗"（更古）
- "没有" → "无"（表没有）/ "未"（表未曾）
- "不要" → "毋" / "莫"（表禁止）

### 5. 倒装句

#### 5.1 宾语前置

| 现代语序 | 文言倒装 |
|---------|---------|
| 不知道**什么** | **何**不知 |
| 喜欢**音乐** | **音乐**，其好之 |
| 他很喜欢**读书** | **读书**，其甚好之 |

**标志**：疑问句、否定句、强调句

#### 5.2 主谓倒装

| 现代语序 | 文言倒装 |
|---------|---------|
| **传统文史研究**的核心困境**是**人力瓶颈 | 古之**治文史者**，**困於**人力 |

#### 5.3 定语后置

| 现代语序 | 文言倒装 |
|---------|---------|
| **开源的**古籍工程 | 古籍工程**之开源者** |
| **可复用的**方法 | 方法**之可复用者** |

**规则**：用"之...者"结构，但**太史公文风倾向直接删除，不倒装**

## 工作流

1. **通读原文**
   - 识别所有现代白话标记词

2. **第一遍：删除虚词**
   - 删除所有"的、地、得、了、着、吗、呢"
   - 疑问句改"乎、哉、耶"

3. **第二遍：替换动词**
   - "是" → "为/乃/也"
   - "有" → "具/持" 或删除
   - "被/把/让" → 改为文言被动句

4. **第三遍：替换代词/连词/副词**
   - "我/你/他" → "余/汝/其"
   - "这/那" → "此/彼"
   - "很/非常" → "甚/极" 或删除
   - "因为...所以..." → "以...故..."

5. **第四遍：数字汉字化**
   - 阿拉伯数字 → 汉字
   - 考虑是否倒装（18类 → 类十有八）
   - 考虑是否概数化（3,185 → 三千馀）

6. **第五遍：调整句式**
   - 检查是否有被动句未改
   - 检查是否有疑问句未加"乎"
   - 检查句子是否过长（需断句）

7. **第六遍：检查现代痕迹**
   - 全文搜索"的、地、得、了、吗、呢、着、很、非常、是、有"
   - 确保100%删除

## 示例

### 示例1：简单句

**原文**：
```
他是一个很聪明的学生，学习非常努力。
```

**第一遍（删虚词）**：
```
他是一个很聪明学生，学习非常努力。
（删"的"）
```

**第二遍（改动词）**：
```
他为一很聪明学生，学习非常努力。
（"是" → "为"）
```

**第三遍（改代词/副词）**：
```
其为一甚聪明学生，学习极努力。
（"他" → "其"，"很" → "甚"，"非常" → "极"）
```

**第四遍（删量词）**：
```
其为甚聪明学生，学习极努力。
（删"一个"）
```

**第五遍（优化）**：
```
其为聪明学生，笃学。
（删"甚""极"，"学习努力" → "笃学"）
```

**最终**：
```
其为聪明学生，笃学。
```

### 示例2：复杂句

**原文**：
```
因为他很努力学习，所以成绩非常好，老师都很喜欢他。
```

**转换过程**：
1. 删虚词："因为他很努力学习，所以成绩非常好，老师都很喜欢他。"
2. 改连词："以他很努力学习，故成绩非常好，老师都很喜欢他。"
3. 改代词："以其很努力学习，故成绩非常好，老师都很喜欢其。"
4. 改副词："以其甚努力学习，故成绩极好，老师皆喜其。"
5. 简化："以其笃学，故绩甚佳，师皆好之。"

**最终**：
```
以其笃学，故绩甚佳，师皆好之。
```

## 常见错误

### ❌ 错误1：保留"的"的痕迹

**错误**：`开源之古籍知识工程`
**正确**：`开源古籍知识工程`（直接删"之"）

### ❌ 错误2：不改"是"

**错误**：`他是学生`
**正确**：`其为学生` / `学生也`

### ❌ 错误3：不改"有"

**错误**：`有三个人`
**正确**：`三人` / `人三`

### ❌ 错误4：保留"很""非常"

**错误**：`很好`
**正确**：`甚善` 或直接删除改为`善`

### ❌ 错误5：不改被动句

**错误**：`被打败`
**正确**：`为敌所败`

### ❌ 错误6：数字不转汉字

**错误**：`130篇`
**正确**：`百三十篇`

### ❌ 错误7：保留量词"个"

**错误**：`三个人`
**正确**：`三人`

### ❌ 错误8：不改疑问句

**错误**：`好吗？`
**正确**：`善乎？`
