# 太史公文笔风格核心特征

本文档提炼司马迁《史记》的核心文笔风格，供AI写作参考。

**核心原则**：
- ❌ 不复述《史记》原文内容
- ❌ 不模拟司马迁的历史观点
- ❌ 不套用《史记》名句
- ✅ 学习语言形式、句法结构、修辞节奏

---

## 一、句法结构

### 1.1 人物开篇定式

**模式**: `X者，Y人也，字Z。`

```
〖@项籍〗者，〖=下相〗人也，字〖@羽〗。
〖@廉颇〗者，〖'赵〗之〖;良将〗也。
```

**变体**:
- 籍贯 + 职业: `X者，Y人也，为Z。`
- 籍贯 + 家世: `X者，Y人也，其先Z。`

### 1.2 简短断言式

**特征**: 单句陈述，不加修饰，直击要害

```
初起时，年二十四。
学书不成，去学剑，又不成。
项籍长八尺馀，力能扛鼎，才气过人。
```

**应用**: 人物早年经历、关键转折、核心能力

### 1.3 层递并列式

**模式**: `A，B，C，...` (节奏渐强)

```
学书不成，去学剑，又不成。
上会稽，探禹穴，闚九疑，浮於沅湘。
序君臣父子之礼，列夫妇长幼之别。
```

**特征**:
- 动词连用，省略主语
- 由浅入深、由小到大
- 无连词，靠标点控制节奏

### 1.4 因果承转式

**模式**: `A，故B。` / `A，以是B。`

```
项氏世世为楚将，封於项，故姓项氏。
项梁常为主办，阴以兵法部勒宾客及子弟，以是知其能。
```

**应用**: 解释缘由、推导结论

## 二、叙事节奏

### 2.1 惜墨如金的白描

**特征**: 删除一切装饰性形容词，只写动作和结果

```
❌ 繁冗: 项籍看到秦始皇威严的仪仗队浩浩荡荡地过江，心中顿时涌起无限豪情...
✅ 太史公: 秦始皇帝游会稽，渡浙江，梁与籍俱观。籍曰："彼可取而代也。"
```

### 2.2 对话的精准截取与动作穿插

**模式**: 只录关键一句，并穿插动作、情绪、氛围

```
籍曰："彼可取而代也。"
梁掩其口，曰："毋妄言，族矣！"
梁以此奇籍。
```

**要点**:
- 不交代对话背景（观众自推）
- 不描写神态动作（"掩其口"是动作，非描写）
- 用行为暗示后果（"以此奇籍"）

**进阶技法：动作与对话穿插**（参考《鸿门宴》）

《史记》对话场景不是纯对话，而是对话+动作+情绪+氛围的混合体。

**《鸿门宴》经典示例**：
```
范增数目项王，举所佩玉玦以示之者三，项王默然不应。

范增起，出，召项庄，谓曰："君王为人不忍。若入前为寿，寿毕，请以剑舞，因击沛公於坐，杀之。"

庄则入为寿。寿毕，曰："君王与沛公饮，军中无以为乐，请以剑舞。"

项王曰："诺。"

项庄拔剑起舞。项伯亦拔剑起舞，常以身翼蔽沛公，庄不得击。
```

**核心要素**：
1. **动作标记人物状态**：起、出、入、拔剑、默然
2. **情绪通过动作展现**：数目（急切）、默然（犹豫）、翼蔽（保护）
3. **时间节奏标记**：数...者三（反复）、寿毕（时间点）
4. **空间位置标记**：出、入、前
5. **氛围描写极简**：不说"气氛紧张"，但"默然不应""常以身翼蔽"已传达

**应用模板**：
```
[人物A][动作]，曰："[对话]。"

[人物B][情绪/动作]。

[时间标记]，[人物C][位置标记]曰："[对话]。"

[人物D][动作]，[人物E][反应动作]。
```

**常用动作词**：
- 起（站起）、坐（坐下）、出（离开）、入（进入）、归（返回）
- 默然（沉默）、怒（愤怒）、喜（高兴）、惧（害怕）
- 拔剑、举杯、掩口、视（注视）

**常用时间标记**：
- 顷之（过了一会儿）、良久（很久）、俄而（不久）
- 寿毕（祝寿完）、会罢（会议结束）、日暮（天黑了）

**常用位置标记**：
- 从旁（从旁边）、前（上前）、退（后退）、在侧（在旁边）

**常用氛围标记**：
- 坐皆默（在座的人都沉默了）
- 左右皆泣（周围的人都哭了）
- 莫敢仰视（没人敢抬头看）

### 2.3 跳跃式剪辑

**特征**: 省略过渡，直接跳到下一场景

```
[场景1] 项梁杀人，与籍避仇於吴中。
[场景2] 吴中贤士大夫皆出项梁下。
[场景3] 秦始皇帝游会稽...
```

**时间跨度**: 可能数年，不交代中间经历

## 三、修辞手法

### 3.1 反复强调式

**模式**: 重复关键词，递进加强

```
学书不成，去学剑，又不成。
（"不成" 重复 → 凸显项羽志向之大）

命也夫，命也夫！
（"命也夫" 重复 → 强化司马谈的悲愤）
```

### 3.2 先抑后扬 / 先扬后抑

**模式**: A则B，然其C

```
儒者博而寡要，劳而少功，是以其事难尽从；然其序君臣父子之礼，列夫妇长幼之别，不可易也。
（先抑：博而寡要 → 后扬：礼不可易）

法家严而少恩；然其正君臣上下之分，不可改矣。
（先抑：少恩 → 后扬：分不可改）
```

### 3.3 排比对仗式

**特征**: 句式整齐，音律工整

```
易著天地阴阳四时五行，故长於变；
礼经纪人伦，故长於行；
书记先王之事，故长於政；
诗记山川谿谷禽兽草木牝牡雌雄，故长於风；
乐乐所以立，故长於和；
春秋辩是非，故长於治人。
```

**应用**: 总结性论述、评价多个对象

### 3.4 设问自答式

**模式**: 问题 + 回答（常用于议论）

```
昔孔子何为而作春秋哉？
余闻董生曰：'周道衰废，孔子为鲁司寇...'
```

## 四、人物刻画

### 4.1 关键细节定性格

**原则**: 一个细节胜过千言万语

```
项籍曰："书足以记名姓而已。剑一人敌，不足学，学万人敌。"
→ 性格: 雄心勃勃、不拘小节

项籍长八尺馀，力能扛鼎，才气过人。
→ 能力: 武力 + 气魄

吴中子弟皆已惮籍矣。
→ 影响力: 侧面烘托
```

### 4.2 行为即性格

**原则**: 不评价，让行为说话

```
❌ 说教: 项羽是个骄傲自大的人
✅ 太史公: （后文）项王不肯竟学兵法、分封诸侯失当等行为
```

### 4.3 他人之口借力

**原则**: 用旁人的评价暗示主角地位

```
吴中贤士大夫皆出项梁下。
（侧面写项梁声望）

梁以此奇籍。
（借项梁之眼，写项羽不凡）
```

## 五、议论特征

### 5.1 太史公曰模式

**结构**:
1. 引典（历史事例 / 经典语录）—— **可选，非必需**
2. 核心论断（极简，直击要害）
3. 升华哲理（可选）

**示例**（从太史公自序）:
```
[引典] 孔子知言之不用，道之不行也
[联系] 今汉兴，海内一统，明主贤君忠臣死义之士
[评判] 余为太史而弗论载，废天下之史文，余甚惧焉
```

**精简原则**（重要）：

太史公曰应该极简，避免啰嗦重复。

❌ **冗长版本**（106字）：
```
太史公曰：

天下会议之乱，莫过于各执一词而无定见。

产品言抓手闭环，技言承载之困，运营言方法论，研发斥虚需。四家纷纭，谁得其实？

终赖长官一言决之：作简版，速试错。

此非真决也，乃推责于试错之名耳。

及散会，牛马开周报，书五字应付，此职场之常态也。

言之愈繁，行之愈寡；会之愈多，事之愈滞。

可不慎哉！
```

✅ **精简版本**（52字）：
```
太史公曰：

四家争执，各执其词，莫知所从。

上决之以"速试"，此非决也，推责耳。

散会书报，止五字而已。

言愈繁，行愈寡；会愈多，事愈滞。
```

**删减要点**：
1. **删除重复陈述**：正文已详述的内容（如"产品言抓手闭环"等），太史公曰不再重复
2. **删除冗余修饰**："天下会议之乱，莫过于"→"四家争执"（直接切入）
3. **删除多余说明**："此职场之常态也"→删除（点到为止）
4. **保留精华对仗**："言愈繁，行愈寡"（简洁有力）
5. **删除疑问句**："谁得其实？"→删除（反问影响节奏）
6. **删除感叹收束**："可不慎哉！"→删除（对仗句已是收束）

**字数控制**：
- 太史公曰正文：**50-80字**为宜
- 超过100字视为过于啰嗦，需精简

### 5.2 褒贬寓于叙事

**原则**: 不明说好坏，通过细节暗示立场

```
中性叙述: 项王乃疑范增与汉有私，稍夺之权。
暗含批判: （不明说"刚愎自用"，但读者自知）
```

## 六、文体分类建议

根据内容类型，太史公采用不同风格：

| 文体 | 特征 | 典型篇目 |
|------|------|----------|
| **本纪** | 编年体，简洁记事，夹叙夹议 | 项羽本纪 |
| **列传** | 以人物为中心，详略有致，重关键转折 | 李将军列传 |
| **自序/论赞** | 议论为主，引经据典，排比对仗多 | 太史公自序 |

## 七、自检清单

使用太史公文笔时，请确保：

- [ ] 是否删除了所有装饰性形容词？
- [ ] 对话是否只截取关键一句？
- [ ] 有无用行为代替心理描写？
- [ ] 句式是否简短有力（避免从句套从句）？
- [ ] 有无通过他人之口烘托主角？
- [ ] 议论是否做到"寓褒贬于叙事"？
- [ ] 有无直接复用《史记》原句？（❌禁止）

## 八、常用句式模板库

### 开篇定式
- `X者，Y人也。`
- `X，其先Y。`
- `X少时...及壮...`

### 转折承接
- `A，B，故C。`
- `A，以是B。`
- `A，然B。`

### 评价结论
- `以此知X。`
- `X以此奇Y。`
- `皆出X下。`（侧面烘托）

### 议论收束
- `A，B，C，未必然也。`（欲扬先抑）
- `虽百家弗能X也。`（强调不可撼动）
- `此X之大者也。`（总结提升）
