# 史记知识库测试问题集

## 概述

本目录包含用于测试各种知识库对《史记》知识覆盖度的标准化问题集。

## 设计原则

1. **全面性**：10组问题，每组100个，覆盖史记的所有知识维度
2. **均衡性**：每组问题均衡覆盖130个章节，避免偏向特定章节
3. **互斥性**：10组问题集相互无交集，可用于不同测试场景
4. **可验证性**：每个问题都有明确的答案和原文出处

## 10组问题分类

### 第1组：人物基本信息 (Person-Basic)
**目标**：测试对人物姓名、字号、籍贯、家世、生卒年等基本信息的掌握
- 问题类型：人物的基本属性查询
- 覆盖范围：本纪、世家、列传中的核心人物
- 答案格式：明确的文本信息 + 章节出处

**示例问题**：
- Q: 孔子的字是什么？
- A: 仲尼 (出处: 047_孔子世家)

### 第2组：人物关系网络 (Person-Relations)
**目标**：测试对人物间复杂关系的理解
- 问题类型：君臣、师徒、亲属、朋友等关系查询
- 覆盖范围：跨章节的社会网络
- 答案格式：关系类型 + 相关人物 + 章节出处

**示例问题**：
- Q: 韩信与萧何是什么关系？
- A: 萧何向刘邦推荐韩信为大将，是推荐人关系 (出处: 092_淮阴侯列传、053_萧相国世家)

### 第3组：重大历史事件 (Events-Major)
**目标**：测试对战争、政变、改革等重大事件的掌握
- 问题类型：事件的起因、经过、结果、影响
- 覆盖范围：跨章节的历史事件
- 答案格式：事件描述 + 时间 + 章节出处

**示例问题**：
- Q: 垓下之战的结果是什么？
- A: 项羽兵败自刎，刘邦统一天下 (出处: 007_项羽本纪、008_高祖本纪)

### 第4组：时间与年代 (Chronology)
**目标**：测试对时间信息的理解和推断能力
- 问题类型：具体年份、朝代更替、在位时长、事件时序
- 覆盖范围：纪年系统、历法、时间推断
- 答案格式：时间信息 + 推断依据 + 章节出处

**示例问题**：
- Q: 秦始皇统一六国是哪一年？
- A: 秦王政二十六年(公元前221年) (出处: 006_秦始皇本纪)

### 第5组：地理信息 (Geography)
**目标**：测试对地理空间信息的掌握
- 问题类型：地名、都城、封地、战场、行政区划
- 覆盖范围：空间维度的知识
- 答案格式：地理位置 + 相关描述 + 章节出处

**示例问题**：
- Q: 项羽分封十八诸侯时，刘邦被封在哪里？
- A: 汉中，为汉王 (出处: 007_项羽本纪、008_高祖本纪)

### 第6组：制度与官职 (Institutions)
**目标**：测试对古代制度体系的理解
- 问题类型：官职体系、礼仪制度、法律刑罚、经济制度
- 覆盖范围：社会制度层面的知识
- 答案格式：制度描述 + 功能说明 + 章节出处

**示例问题**：
- Q: 秦朝的"三公九卿"制度中，"三公"指的是哪三个官职？
- A: 丞相、太尉、御史大夫 (出处: 006_秦始皇本纪)

### 第7组：思想与文化 (Culture-Ideology)
**目标**：测试对思想文化内容的理解
- 问题类型：哲学思想、儒道法墨、典籍引用、太史公曰
- 覆盖范围：思想文化维度
- 答案格式：思想内容 + 代表人物 + 章节出处

**示例问题**：
- Q: 老子的核心思想是什么？
- A: 道家思想，主张"无为而治" (出处: 063_老子韩非列传)

### 第8组：器物与自然 (Objects-Nature)
**目标**：测试对物质文化和自然知识的掌握
- 问题类型：器物、生物、天文、神话、数量统计
- 覆盖范围：物质文化和自然知识
- 答案格式：器物/自然描述 + 用途/特征 + 章节出处

**示例问题**：
- Q: 秦始皇陵中有哪些特殊的陪葬品？
- A: 兵马俑、珍宝器物、水银江河等 (出处: 006_秦始皇本纪)

### 第9组：成语与典故 (Idioms-Allusions)
**目标**：测试对史记文学价值和文化传承的理解
- 问题类型：成语来源、典故出处、名言警句
- 覆盖范围：文学价值和文化传承
- 答案格式：成语/典故 + 故事原文 + 章节出处

**示例问题**：
- Q: 成语"破釜沉舟"出自哪个历史事件？
- A: 项羽在巨鹿之战前，破釜沉舟以示必死决心 (出处: 007_项羽本纪)

### 第10组：综合推理 (Reasoning-Complex)
**目标**：测试知识图谱的深度关联和推理能力
- 问题类型：需要跨章节、多维度综合推理的复杂问题
- 覆盖范围：知识图谱的深度关联能力
- 答案格式：推理过程 + 结论 + 多个章节出处

**示例问题**：
- Q: 为什么刘邦能够在楚汉之争中战胜项羽？
- A: 综合多方面因素：善用人才(韩信、张良、萧何)、战略得当、项羽刚愎自用失人心等 (出处: 007_项羽本纪、008_高祖本纪、092_淮阴侯列传等多章节)

## 章节覆盖策略

### 史记130章分类

- **本纪**：12篇 (001-012)
- **表**：10篇 (013-022)
- **书**：8篇 (023-030)
- **世家**：30篇 (031-060)
- **列传**：70篇 (061-130)

### 覆盖平衡策略

每组100个问题的章节分布：
- 本纪：~10个问题 (覆盖10-12章)
- 表：~5个问题 (覆盖5-8章，表类章节知识点较少)
- 书：~8个问题 (覆盖6-8章)
- 世家：~25个问题 (覆盖20-25章)
- 列传：~52个问题 (覆盖55-65章)

### 均衡性验证

使用Python脚本验证每组问题的：
1. 章节覆盖率：至少覆盖100/130个章节
2. 章节分布方差：避免过度集中在某些章节
3. 问题类型分布：每组内部也要有子类型的均衡
4. 难度分布：简单/中等/困难 = 4:4:2

## 质量控制

### 问题设计原则

1. **明确性**：问题表述清晰，不产生歧义
2. **可验证性**：答案可在原文中找到明确依据
3. **代表性**：覆盖该类别的典型知识点
4. **难度梯度**：简单、中等、困难问题的合理分布

### 验证检查清单

- [ ] 10组问题总计1000个，每组正好100个
- [ ] 每组问题覆盖至少100个章节
- [ ] 10组问题之间完全无交集
- [ ] 每个问题都有明确的答案和出处
- [ ] 章节分布方差在合理范围内
- [ ] 难度分布符合4:4:2比例
- [ ] 所有答案都可在原文验证

## 使用方法

### 生成问题集

```bash
# 生成第1组问题（示例）
python scripts/generate_questions_set01.py

# 验证章节覆盖率
python scripts/validate_coverage.py --set set01

# 验证所有10组的互斥性
python scripts/validate_uniqueness.py --all
```

### 测试知识库

```bash
# 使用问题集测试知识库
python scripts/test_kb.py --kb your_kb --questions questions/set01_person_basic.json
```
