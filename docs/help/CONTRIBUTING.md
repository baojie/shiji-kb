# 参与指南

欢迎参与《史记》知识库项目！

## 反馈与讨论

- **[Issues](https://github.com/yourrepo/shiji-kb/issues)** - 提交 Bug 或功能建议
- **[Discussions](https://github.com/yourrepo/shiji-kb/discussions)** - 分享想法、提问、交流

## 通过 Pull Request 贡献

### 1. Fork 项目

在 GitHub 项目页面点击 `Fork` 按钮，将项目复制到你的账号下。

### 2. 克隆到本地

```bash
git clone https://github.com/你的用户名/shiji-kb.git
cd shiji-kb
```

### 3. 创建分支

```bash
git checkout -b fix-something
```

分支名建议：
- `fix-xxx` - 修复错误
- `add-xxx` - 新增功能
- `update-xxx` - 更新内容

### 4. 修改并提交

```bash
git add 修改的文件
git commit -m "简短描述你的改动"
```

### 5. 推送到 GitHub

```bash
git push origin fix-something
```

### 6. 创建 Pull Request

1. 访问你 Fork 的项目页面
2. 点击 `Pull Request` 按钮
3. 选择你的分支，填写改动说明
4. 提交 PR，等待审核

## 常见贡献类型

- **修正知识图谱错误**：人物、事件、地点等标注错误
- **补充缺失信息**：添加遗漏的实体或关系
- **改进文档**：完善说明文档
- **技术优化**：改进脚本和工具

## 注意事项

- 一个 PR 只做一件事
- 提交前检查格式是否正确
- 详细描述改动原因
