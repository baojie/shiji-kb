太史公序论

礼仪之本

1.1 太史公曰：洋洋美德乎！宰制万物，役使群众，岂人力也哉？

1.2 余至大行礼官，观三代损益，乃知缘人情而制礼，依人性而作仪，其所由来尚矣。

礼制之用

2.1 人道经纬万端，规矩无所不贯，诱进以仁义，束缚以刑罚，故德厚者位尊，禄重者宠荣，所以总一海内而整齐万民也。

2.2 人体安驾乘，为之金舆错衡以繁其饰；目好五色，为之黼黻文章以表其能；耳乐钟磬，为之调谐八音以荡其心；口甘五味，为之庶羞酸咸以致其美；情好珍善，为之琢磨圭璧以通其意。

2.3 故大路越席，皮弁布裳，硃弦洞越，大羹玄酒，所以防其淫侈，救其彫敝。

2.4 是以君臣朝廷尊卑贵贱之序，下及黎庶车舆衣服宫室饮食嫁娶丧祭之分，事有宜適，物有节文。

2.5 仲尼曰：

"禘自既灌而往者，吾不欲观之矣。"

礼崩乐坏

周衰礼废

3.1 周衰，礼废乐坏，大小相逾，管仲之家，兼备三归。

3.2 循法守正者见侮於世，奢溢僭差者谓之显荣。

3.3 自子夏，门人之高弟也，犹云：

"出见纷华盛丽而说，入闻夫子之道而乐，二者心战，未能自决。"

3.4 而况中庸以下，渐渍於失教，被服於成俗乎？

3.5 孔子曰"必也正名"，於卫所居不合。

3.6 仲尼没後，受业之徒沈湮而不举，或適齐、楚，或入河海，岂不痛哉！

秦汉礼制

秦制礼仪

4.1 至秦有天下，悉内六国礼仪，采择其善，虽不合圣制，其尊君抑臣，朝廷济济，依古以来。

汉初承秦

5.1 至于高祖，光有四海，叔孙通颇有所增益减损，大抵皆袭秦故。

5.2 自天子称号下至佐僚及宫室官名，少所变改。

孝文罢礼

6.1 孝文即位，有司议欲定仪礼，孝文好道家之学，以为繁礼饰貌，无益於治，躬化谓何耳，故罢去之。

孝景削藩

7.1 孝景时，御史大夫晁错明於世务刑名，数干谏孝景曰：

"诸侯籓辅，臣子一例，古今之制也。今大国专治异政，不禀京师，恐不可传後。"

7.2 孝景用其计，而六国畔逆，以错首名，天子诛错以解难。

7.3 事在袁盎语中。

7.4 是後官者养交安禄而已，莫敢复议。

今上定礼

8.1 今上即位，招致儒术之士，令共定仪，十馀年不就。

8.2 或言古者太平，万民和喜，瑞应辨至，乃采风俗，定制作。

8.3 上闻之，制诏御史曰：

"盖受命而王，各有所由兴，殊路而同归，谓因民而作，追俗为制也。议者咸称太古，百姓何望？汉亦一家之事，典法不传，谓子孙何？化隆者闳博，治浅者褊狭，可不勉与！"

8.4 乃以太初之元改正朔，易服色，封太山，定宗庙百官之仪，以为典常，垂之於後云。

礼之本义

礼之起源

9.1 礼由人起。

9.2 人生有欲，欲而不得则不能无忿，忿而无度量则争，争则乱。

9.3 先王恶其乱，故制礼义以养人之欲，给人之求，使欲不穷於物，物不屈於欲，二者相待而长，是礼之所起也。

9.4 故礼者养也。

礼之养性

10.1 稻粱五味，所以养口也；椒兰芬茝，所以养鼻也；钟鼓管弦，所以养耳也；刻镂文章，所以养目也；疏房床笫几席，所以养体也：故礼者养也。

礼之辨等

11.1 君子既得其养，又好其辨也。

11.2 所谓辨者，贵贱有等，长少有差，贫富轻重皆有称也。

11.3 故天子大路越席，所以养体也；侧载臭茝，所以养鼻也；前有错衡，所以养目也；和鸾之声，步中武象，骤中韶濩，所以养耳也；龙旂九斿，所以养信也；寝兕持虎，鲛韅弥龙，所以养威也。

11.4 故大路之马，必信至教顺，然后乘之，所以养安也。

11.5 孰知夫出死要节之所以养生也。孰知夫轻费用之所以养财也，孰知夫恭敬辞让之所以养安也，孰知夫礼义文理之所以养情也。

儒墨之分

12.1 人苟生之为见，若者必死；苟利之为见，若者必害；怠惰之为安，若者必危；情胜之为安，若者必灭。

12.2 故圣人一之於礼义，则两得之矣；一之於情性，则两失之矣。

12.3 故儒者将使人两得之者也，墨者将使人两失之者也。是儒墨之分。

礼之功用

治国之本

13.1 治辨之极也，彊固之本也，威行之道也，功名之总也。

13.2 王公由之，所以一天下，臣诸侯也；弗由之，所以捐社稷也。

楚国之鉴

14.1 故坚革利兵不足以为胜，高城深池不足以为固，严令繁刑不足以为威。由其道则行，不由其道则废。

14.2 楚人鲛革犀兕，所以为甲，坚如金石；宛之钜铁施，钻如蜂虿，轻利剽，卒如熛风。

14.3 然而兵殆於垂涉，唐昧死焉；庄蹻起，楚分而为四参。

14.4 是岂无坚革利兵哉？其所以统之者非其道故也。

14.5 汝、颍以为险，江、汉以为池，阻之以邓林，缘之以方城。

14.6 然而秦师至鄢郢，举若振槁。是岂无固塞险阻哉？其所以统之者非其道故也。

纣王之鉴

15.1 纣剖比干，囚箕子，为砲格，刑杀无辜，时臣下懔然，莫必其命。

15.2 然而周师至，而令不行乎下，不能用其民。是岂令不严，刑不鷟哉？其所以统之者非其道故也。

古之善治

16.1 古者之兵，戈矛弓矢而已，然而敌国不待试而诎。

16.2 城郭不集，沟池不掘，固塞不树，机变不张，然而国晏然不畏外而固者，无他故焉，明道而均分之，时使而诚爱之，则下应之如景响。

16.3 有不由命者，然後俟之以刑，则民知罪矣。

16.4 故刑一人而天下服。罪人不尤其上，知罪之在己也。

16.5 是故刑罚省而威行如流，无他故焉，由其道故也。

16.6 故由其道则行，不由其道则废。

尧之善治

17.1 古者帝尧之治天下也，盖杀一人刑二人而天下治。

17.2 传曰：

"威厉而不试，刑措而不用。"

礼之三本

天地祖宗君师

18.1 天地者，生之本也；先祖者，类之本也；君师者，治之本也。

18.2 无天地恶生？无先祖恶出？无君师恶治？三者偏亡，则无安人。

18.3 故礼，上事天，下事地，尊先祖而隆君师，是礼之三本也。

祭祀之分

19.1 故王者天太祖，诸侯不敢怀，大夫士有常宗，所以辨贵贱。贵贱治，得之本也。

19.2 郊畴乎天子，社至乎诸侯，函及士大夫，所以辨尊者事尊，卑者事卑，宜钜者钜，宜小者小。

19.3 故有天下者事七世，有一国者事五世，有五乘之地者事三世，有三乘之地者事二世，有特牲而食者不得立宗庙，所以辨积厚者流泽广，积薄者流泽狭也。

礼之仪节

祭祀之礼

20.1 大飨上玄尊，俎上腥鱼，先大羹，贵食饮之本也。

20.2 大飨上玄尊而用薄酒，食先黍稷而饭稻粱，祭哜先大羹而饱庶羞，贵本而亲用也。

20.3 贵本之谓文，亲用之谓理，两者合而成文，以归太一，是谓大隆。

礼仪之同

21.1 故尊之上玄尊也，俎之上腥鱼也，豆之先大羹，一也。

21.2 利爵弗啐也，成事俎弗尝也，三侑之弗食也，大昏之未废齐也，大庙之未内尸也，始绝之未小敛，一也。

21.3 大路之素幬也，郊之麻絻，丧服之先散麻，一也。

21.4 三年哭之不反也，清庙之歌一倡而三叹，县一钟尚拊膈，硃弦而通越，一也。

礼之三境

22.1 凡礼始乎脱，成乎文，终乎税。

22.2 故至备，情文俱尽；其次，情文代胜；其下，复情以归太一。

22.3 天地以合，日月以明，四时以序，星辰以行，江、河以流，万物以昌，好恶以节，喜怒以当。

22.4 以为下则顺，以为上则明。

太史公再论

礼之至极

23.1 太史公曰：至矣哉！立隆以为极，而天下莫之能益损也。

23.2 本末相顺，终始相应，至文有以辨，至察有以说。

23.3 天下从之者治，不从者乱；从之者安，不从者危。小人不能则也。

礼之深广

24.1 礼之貌诚深矣，坚白同异之察，入焉而弱。

24.2 其貌诚大矣，擅作典制褊陋之说，入焉而望。

24.3 其貌诚高矣，暴慢恣睢，轻俗以为高之属，入焉而队。

礼之标准

25.1 故绳诚陈，则不可欺以曲直；衡诚县，则不可欺以轻重；规矩诚错，则不可欺以方员；君子审礼，则不可欺以诈伪。

25.2 故绳者，直之至也；衡者，平之至也；规矩者，方员之至也；礼者，人道之极也。

礼之层次

26.1 然而不法礼者不足礼，谓之无方之民；法礼足礼，谓之有方之士。

26.2 礼之中，能思索，谓之能虑；能虑勿易，谓之能固。能虑能固，加好之焉，圣矣。

26.3 天者，高之极也；地者，下之极也；日月者，明之极也；无穷者，广大之极也；圣人者，道之极也。

礼之隆杀

27.1 以财物为用，以贵贱为文，以多少为异，以隆杀为要。

27.2 文貌繁，情欲省，礼之隆也；文貌省，情欲繁，礼之杀也；文貌情欲相为内外表里，并行而杂，礼之中流也。

27.3 君子上致其隆，下尽其杀，而中处其中。

27.4 步骤驰骋广骛不外，是以君子之性守宫庭也。

27.5 人域是域，士君子也。外是，民也。於是中焉，房皇周浃，曲得其次序，圣人也。

27.6 故厚者，礼之积也；大者，礼之广也；高者，礼之隆也；明者，礼之尽也。

赞

28 礼因人心，非从天下。合诚饰貌，救弊兴雅。
以制黎甿，以事宗社。情文可重，丰杀难假。
仲尼坐树，孙通蕝野。圣人作教，罔不由者。
