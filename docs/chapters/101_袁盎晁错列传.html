0 袁盎晁错列传 原文

袁盎

出身与早年

1.1 袁盎者，楚人也，字丝。

1.2 父故为群盗，徙处安陵。

1.3 高后时，盎尝为吕禄舍人。

1.4 及孝文帝即位，盎兄哙任盎为中郎。

论绛侯

2.1 绛侯为丞相，朝罢趋出，意得甚。

2.2 上礼之恭，常自送之。

2.3 袁盎进曰：

"陛下以丞相何如人？"

2.4 上曰：

"社稷臣。"

2.5 盎曰：

NOTE: 袁盎论社稷臣与功臣之别
"绛侯所谓功臣，非社稷臣，社稷臣主在与在，主亡与亡。方吕后时，诸吕用事，擅相王，刘氏不绝如带。是时绛侯为太尉，主兵柄，弗能正。吕后崩，大臣相与共畔诸吕，太尉主兵，適会其成功，所谓功臣，非社稷臣。丞相如有骄主色。陛下谦让，臣主失礼，窃为陛下不取也。"

2.6 後朝，上益庄，丞相益畏。

2.7 已而绛侯望袁盎曰：

"吾与而兄善，今兒廷毁我！"

2.8 盎遂不谢。

救绛侯

3.1 及绛侯免相之国，国人上书告以为反，徵系清室，宗室诸公莫敢为言，唯袁盎明绛侯无罪。

3.2 绛侯得释，盎颇有力。

3.3 绛侯乃大与盎结交。

袁盎谏淮南王

谏淮南王事

4.1 淮南厉王朝，杀辟阳侯，居处骄甚。

4.2 袁盎谏曰：

"诸侯大骄必生患，可適削地。"

4.3 上弗用。淮南王益横。

4.4 及棘蒲侯柴武太子谋反事觉，治，连淮南王，淮南王徵，上因迁之蜀，轞车传送。

4.5 袁盎时为中郎将，乃谏曰：

NOTE: 袁盎谏迁淮南王
"陛下素骄淮南王，弗稍禁，以至此，今又暴摧折之。淮南王为人刚，如有遇雾露行道死，陛下竟为以天下之大弗能容，有杀弟之名，柰何？"

4.6 上弗听，遂行之。

淮南王死与善后

5.1 淮南王至雍，病死，闻，上辍食，哭甚哀。

5.2 盎入，顿首请罪。

5.3 上曰：

"以不用公言至此。"

5.4 盎曰：

"上自宽，此往事，岂可悔哉！且陛下有高世之行者三，此不足以毁名。"

5.5 上曰：

"吾高世行三者何事？"

5.6 盎曰：

NOTE: 袁盎论文帝三高世行
"陛下居代时，太后尝病，三年，陛下不交睫，不解衣，汤药非陛下口所尝弗进。夫曾参以布衣犹难之，今陛下亲以王者脩之，过曾参孝远矣。夫诸吕用事，大臣专制，然陛下从代乘六传驰不测之渊，虽贲育之勇不及陛下。陛下至代邸，西向让天子位者再，南面让天子位者三。夫许由一让，而陛下五以天下让，过许由四矣。且陛下迁淮南王，欲以苦其志，使改过，有司卫不谨，故病死。"

5.7 於是上乃解，曰：

"将柰何？"

5.8 盎曰：

"淮南王有三子，唯在陛下耳。"

5.9 於是文帝立其三子皆为王。

5.10 盎由此名重朝廷。

袁盎谏事

斥宦者赵同

6.1 袁盎常引大体慷慨。

6.2 宦者赵同以数幸，常害袁盎，袁盎患之。

6.3 盎兄子种为常侍骑，持节夹乘，说盎曰：

"君与斗，廷辱之，使其毁不用。"

6.4 孝文帝出，赵同参乘，袁盎伏车前曰：

"臣闻天子所与共六尺舆者，皆天下豪英。今汉虽乏人，陛下独奈何与刀锯馀人载！"

6.5 於是上笑，下赵同。赵同泣下车。

止文帝驰峻阪

7.1 文帝从霸陵上，欲西驰下峻阪。

7.2 袁盎骑，并车揽辔。

7.3 上曰：

"将军怯邪？"

7.4 盎曰：

"臣闻千金之子坐不垂堂，百金之子不骑衡，圣主不乘危而徼幸。今陛下骋六騑，驰下峻山，如有马惊车败，陛下纵自轻，柰高庙、太后何？"

7.5 上乃止。

却慎夫人座

8.1 上幸上林，皇后、慎夫人从。

8.2 其在禁中，常同席坐。

8.3 及坐，郎署长布席，袁盎引卻慎夫人坐。

8.4 慎夫人怒，不肯坐。上亦怒，起，入禁中。

8.5 盎因前说曰：

NOTE: 袁盎论尊卑之序
"臣闻尊卑有序则上下和。今陛下既已立后，慎夫人乃妾，妾主岂可与同坐哉！適所以失尊卑矣。且陛下幸之，即厚赐之。陛下所以为慎夫人，適所以祸之。陛下独不见'人彘'乎？"

8.6 於是上乃说，召语慎夫人。

8.7 慎夫人赐盎金五十斤。

外任与归家

9.1 然袁盎亦以数直谏，不得久居中，调为陇西都尉。

9.2 仁爱士卒，士卒皆争为死。

9.3 迁为齐相。

9.4 徙为吴相，辞行，种谓盎曰：

NOTE: 袁种教盎为吴相之策
"吴王骄日久，国多奸。今苟欲劾治，彼不上书告君，即利剑刺君矣。南方卑湿，君能日饮，毋何，时说王曰毋反而已。如此幸得脱。"

9.5 盎用种之计，吴王厚遇盎。

袁盎与朝政

说丞相申屠嘉

10.1 盎告归，道逢丞相申屠嘉，下车拜谒，丞相从车上谢袁盎。

10.2 袁盎还，愧其吏，乃之丞相舍上谒，求见丞相。

10.3 丞相良久而见之。

10.4 盎因跪曰：

"原请间。"

10.5 丞相曰：

"使君所言公事，之曹与长史掾议，吾且奏之；即私邪，吾不受私语。"

10.6 袁盎即跪说曰：

NOTE: 袁盎说申屠嘉
"君为丞相，自度孰与陈平、绛侯？"

10.7 丞相曰：

"吾不如。"

10.8 袁盎曰：

"善，君即自谓不如。夫陈平、绛侯辅翼高帝，定天下，为将相，而诛诸吕，存刘氏；君乃为材官蹶张，迁为队率，积功至淮阳守，非有奇计攻城野战之功。且陛下从代来，每朝，郎官上书疏，未尝不止辇受其言，言不可用置之，言可受采之，未尝不称善。何也？则欲以致天下贤士大夫。上日闻所不闻，明所不知，日益圣智；君今自闭钳天下之口而日益愚。夫以圣主责愚相，君受祸不久矣。"

10.9 丞相乃再拜曰：

"嘉鄙野人，乃不知，将军幸教。"

10.10 引入与坐，为上客。

与晁错不睦

11.1 盎素不好晁错，晁错所居坐，盎去；盎坐，错亦去：两人未尝同堂语。

11.2 及孝文帝崩，孝景帝即位，晁错为御史大夫，使吏案袁盎受吴王财物，抵罪，诏赦以为庶人。

吴楚之乱

吴楚反与晁错诛

12.1 吴楚反，闻，晁错谓丞史曰：

"夫袁盎多受吴王金钱，专为蔽匿，言不反。今果反，欲请治盎宜知计谋。"

12.2 丞史曰：

"事未发，治之有绝。今兵西乡，治之何益！且袁盎不宜有谋。"

12.3 晁错犹与未决。

12.4 人有告袁盎者，袁盎恐，夜见窦婴，为言吴所以反者，原至上前口对状。

12.5 窦婴入言上，上乃召袁盎入见。

12.6 晁错在前，及盎请辟人赐间，错去，固恨甚。

12.7 袁盎具言吴所以反状，以错故，独急斩错以谢吴，吴兵乃可罢。

12.8 其语具在吴事中。

12.9 使袁盎为太常，窦婴为大将军。

12.10 两人素相与善。逮吴反。诸陵长者长安中贤大夫争附两人，车随者日数百乘。

使吴脱险

13.1 及晁错已诛，袁盎以太常使吴。

13.2 吴王欲使将，不肯。欲杀之，使一都尉以五百人围守盎军中。

13.3 袁盎自其为吴相时，有从史尝盗爱盎侍兒，盎知之，弗泄，遇之如故。

13.4 人有告从史，言"君知尔与侍者通"，乃亡归。

13.5 袁盎驱自追之，遂以侍者赐之，复为从史。

13.6 及袁盎使吴见守，从史適为守盎校尉司马，乃悉以其装赍置二石醇醪，会天寒，士卒饥渴，饮酒醉，西南陬卒皆卧，司马夜引袁盎起，曰：

"君可以去矣，吴王期旦日斩君。"

13.7 盎弗信，曰：

"公何为者？"

13.8 司马曰：

"臣故为从史盗君侍兒者。"

13.9 盎乃惊谢曰：

"公幸有亲，吾不足以累公。"

13.10 司马曰：

"君弟去，臣亦且亡，辟吾亲，君何患！"

13.11 乃以刀决张，道从醉卒隧出。

13.12 司马与分背，袁盎解节毛怀之，杖，步行七八里，明，见梁骑，骑驰去，遂归报。

袁盎晚年

为楚相与退居

14.1 吴楚已破，上更以元王子平陆侯礼为楚王，袁盎为楚相。

14.2 尝上书有所言，不用。

14.3 袁盎病免居家，与闾里浮沈，相随行，斗鸡走狗。

14.4 雒阳剧孟尝过袁盎，盎善待之。

14.5 安陵富人有谓盎曰：

"吾闻剧孟博徒，将军何自通之？"

14.6 盎曰：

NOTE: 袁盎论剧孟
"剧孟虽博徒，然母死，客送葬车千馀乘，此亦有过人者。且缓急人所有。夫一旦有急叩门，不以亲为解，不以存亡为辞，天下所望者，独季心、剧孟耳。今公常从数骑，一旦有缓急，宁足恃乎！"

14.7 骂富人，弗与通。

14.8 诸公闻之，皆多袁盎。

被刺身死

15.1 袁盎虽家居，景帝时时使人问筹策。

15.2 梁王欲求为嗣，袁盎进说，其後语塞。

15.3 梁王以此怨盎，曾使人刺盎。

15.4 刺者至关中，问袁盎，诸君誉之皆不容口。

15.5 乃见袁盎曰：

"臣受梁王金来刺君，君长者，不忍刺君。然後刺君者十馀曹，备之！"

15.6 袁盎心不乐，家又多怪，乃之棓生所问占。

15.7 还，梁刺客後曹辈果遮刺杀盎安陵郭门外。

晁错

出身与学术

16.1 晁错者，颍川人也。

16.2 学申商刑名於轵张恢先所，与雒阳宋孟及刘礼同师。

16.3 以文学为太常掌故。

性格与早年仕途

17.1 错为人穞直刻深。

17.2 孝文帝时，天下无治尚书者，独闻济南伏生故秦博士，治尚书，年九十馀，老不可徵，乃诏太常使人往受之。

17.3 太常遣错受尚书伏生所。

17.4 还，因上便宜事，以书称说。

17.5 诏以为太子舍人、门大夫、家令。

17.6 以其辩得幸太子，太子家号曰"智囊"。

17.7 数上书孝文时，言削诸侯事，及法令可更定者。

17.8 书数十上，孝文不听，然奇其材，迁为中大夫。

17.9 当是时，太子善错计策，袁盎诸大功臣多不好错。

景帝朝得宠

18.1 景帝即位，以错为内史。

18.2 错常数请间言事，辄听，宠幸倾九卿，法令多所更定。

18.3 丞相申屠嘉心弗便，力未有以伤。

与申屠嘉之争

19.1 内史府居太上庙壖中，门东出，不便，错乃穿两门南出，凿庙壖垣。

19.2 丞相嘉闻，大怒，欲因此过为奏请诛错。

19.3 错闻之，即夜请间，具为上言之。

19.4 丞相奏事，因言错擅凿庙垣为门，请下廷尉诛。

19.5 上曰：

"此非庙垣，乃壖中垣，不致於法。"

19.6 丞相谢。

19.7 罢朝，怒谓长史曰：

"吾当先斩以闻，乃先请，为兒所卖，固误。"

19.8 丞相遂发病死。

19.9 错以此愈贵。

晁错削藩

削藩与父亲之死

20.1 迁为御史大夫，请诸侯之罪过，削其地，收其枝郡。

20.2 奏上，上令公卿、列侯、宗室集议，莫敢难，独窦婴争之，由此与错有卻。

20.3 错所更令三十章，诸侯皆諠譁疾晁错。

20.4 错父闻之，从颍川来，谓错曰：

"上初即位，公为政用事，侵削诸侯，别疏人骨肉，人口议多怨公者，何也？"

20.5 晁错曰：

"固也。不如此，天子不尊，宗庙不安。"

20.6 错父曰：

NOTE: 晁错父之言
"刘氏安矣，而晁氏危矣，吾去公归矣！"

20.7 遂饮药死，曰：

"吾不忍见祸及吾身。"

20.8 死十馀日，吴楚七国果反，以诛错为名。

被诛

21.1 及窦婴、袁盎进说，上令晁错衣朝衣斩东市。

邓公

为晁错辩护

22.1 晁错已死，谒者仆射邓公为校尉，击吴楚军为将。

22.2 还，上书言军事，谒见上。

22.3 上问曰：

"道军所来，闻晁错死，吴楚罢不？"

22.4 邓公曰：

"吴王为反数十年矣，发怒削地，以诛错为名，其意非在错也。且臣恐天下之士噤口，不敢复言也！"

22.5 上曰：

"何哉？"

22.6 邓公曰：

NOTE: 邓公论晁错之死
"夫晁错患诸侯彊大不可制，故请削地以尊京师，万世之利也。计画始行，卒受大戮，内杜忠臣之口，外为诸侯报仇，臣窃为陛下不取也。"

22.7 於是景帝默然良久，曰：

"公言善，吾亦恨之。"

22.8 乃拜邓公为城阳中尉。

邓公事略

23.1 邓公，成固人也，多奇计。

23.2 建元中，上招贤良，公卿言邓公，时邓公免，起家为九卿。

23.3 一年，复谢病免归。

23.4 其子章以脩黄老言显於诸公间。

太史公曰

24.1 太史公曰：

NOTE: 太史公论袁盎与晁错
"袁盎虽不好学，亦善傅会，仁心为质，引义慷慨。遭孝文初立，资適逢世。时以变易，及吴楚一说，说虽行哉，然复不遂。好声矜贤，竟以名败。晁错为家令时，数言事不用；後擅权，多所变更。诸侯发难，不急匡救，欲报私雠，反以亡躯。语曰'变古乱常，不死则亡'，岂错等谓邪！"

赞诗

25.1 袁丝公直，亦多附会。
揽辔见重，卻席翳赖。
朝错建策，屡陈利害。
尊主卑臣，家危国泰。
悲彼二子，名立身败！
