太史公读秦楚之际

1

1.1 太史公读秦楚之际，曰：初作难，发於陈涉；虐戾灭秦，自项氏；拨乱诛暴，平定海内，卒践帝祚，成於汉家。
1.2 五年之间，号令三嬗。
1.3 自生民以来，未始有受命若斯之亟也。

2

2.1 昔虞、夏之兴，积善累功数十年，德洽百姓，摄行政事，考之于天，然後在位。
2.2 汤、武之王，乃由契、后稷脩仁行义十馀世，不期而会孟津八百诸侯，犹以为未可，其後乃放弑。
2.3 秦起襄公，章於文、缪，献、孝之後，稍以蚕食六国，百有馀载，至始皇乃能并冠带之伦。
2.4 以德若彼，用力如此，盖一统若斯之难也。

3

3.1 秦既称帝，患兵革不休，以有诸侯也，於是无尺土之封，堕坏名城，销锋镝，鉏豪桀，维万世之安。
3.2 然王迹之兴，起於闾巷，合从讨伐，轶於三代，乡秦之禁，適足以资贤者为驱除难耳。
3.3 故愤发其所为天下雄，安在无土不王。
3.4 此乃传之所谓大圣乎？岂非天哉，岂非天哉！
3.5 非大圣孰能当此受命而帝者乎？
