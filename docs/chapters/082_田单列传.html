0 田单列传 原文

田单出身

1.1 田单者，齐诸田疏属也。

1.2 湣王时，单为临菑市掾，不见知。

安平之战

2.1 及燕使乐毅伐破齐，齐湣王出奔，已而保莒城。

2.2 燕师长驱平齐，而田单走安平，令其宗人尽断其车轴末而傅铁笼。

2.3 已而燕军攻安平，城坏，齐人走，争涂，以轊折车败，为燕所虏，唯田单宗人以铁笼故得脱，东保即墨。

2.4 燕既尽降齐城，唯独莒、即墨不下。

2.5 燕军闻齐王在莒，并兵攻之。

2.6 淖齿既杀湣王於莒，因坚守，距燕军，数年不下。

2.7 燕引兵东围即墨，即墨大夫出与战，败死。

2.8 城中相与推田单，曰：

"安平之战，田单宗人以铁笼得全，习兵。"

2.9 立以为将军，以即墨距燕。

反间计

离间燕将

3.1 顷之，燕昭王卒，惠王立，与乐毅有隙。

3.2 田单闻之，乃纵反间於燕，宣言曰：

NOTE: 反间之言
"齐王已死，城之不拔者二耳。乐毅畏诛而不敢归，以伐齐为名，实欲连兵南面而王齐。齐人未附，故且缓攻即墨以待其事。齐人所惧，唯恐他将之来，即墨残矣。"

3.3 燕王以为然，使骑劫代乐毅。

燕师忿怒

4.1 乐毅因归赵，燕人士卒忿。

神师之计

4.2 而田单乃令城中人食必祭其先祖於庭，飞鸟悉翔舞城中下食。

4.3 燕人怪之。

4.4 田单因宣言曰：

"神来下教我。"

4.5 乃令城中人曰：

"当有神人为我师。"

4.6 有一卒曰：

"臣可以为师乎？"

4.7 因反走。

4.8 田单乃起，引还，东乡坐，师事之。

4.9 卒曰：

"臣欺君，诚无能也。"

4.10 田单曰：

"子勿言也！"

4.11 因师之。每出约束，必称神师。

激励士气

激怒齐军

4.12 乃宣言曰：

"吾唯惧燕军之劓所得齐卒，置之前行，与我战，即墨败矣。"

4.13 燕人闻之，如其言。

4.14 城中人见齐诸降者尽劓，皆怒，坚守，唯恐见得。

4.15 单又纵反间曰：

"吾惧燕人掘吾城外冢墓，僇先人，可为寒心。"

4.16 燕军尽掘垄墓，烧死人。

4.17 即墨人从城上望见，皆涕泣，俱欲出战，怒自十倍。

火牛阵

准备

5.1 田单知士卒之可用，乃身操版插，与士卒分功，妻妾编於行伍之间，尽散饮食飨士。

5.2 令甲卒皆伏，使老弱女子乘城，遣使约降於燕，燕军皆呼万岁。

5.3 田单又收民金，得千溢，令即墨富豪遗燕将，曰：

"即墨即降，原无虏掠吾族家妻妾，令安堵。"

5.4 燕将大喜，许之。

5.5 燕军由此益懈。

火牛破敌

6.1 田单乃收城中得千馀牛，为绛缯衣，画以五彩龙文，束兵刃於其角，而灌脂束苇於尾，烧其端。

6.2 凿城数十穴，夜纵牛，壮士五千人随其後。

6.3 牛尾热，怒而奔燕军，燕军夜大惊。

6.4 牛尾炬火光明炫燿，燕军视之皆龙文，所触尽死伤。

6.5 五千人因衔枚击之，而城中鼓譟从之，老弱皆击铜器为声，声动天地。

6.6 燕军大骇，败走。

6.7 齐人遂夷杀其将骑劫。

复齐

6.8 燕军扰乱奔走，齐人追亡逐北，所过城邑皆畔燕而归田单，兵日益多，乘胜，燕日败亡，卒至河上，而齐七十馀城皆复为齐。

6.9 乃迎襄王於莒，入临菑而听政。

封君

7.1 襄王封田单，号曰安平君。

太史公曰

8.1 太史公曰：兵以正合，以奇胜。

8.2 善之者，出奇无穷。

8.3 奇正还相生，如环之无端。

8.4 夫始如处女，適人开户；後如脱兔，適不及距：其田单之谓邪！

附记

君王后

9.1 初，淖齿之杀湣王也，莒人求湣王子法章，得之太史嬓之家，为人灌园。

9.2 嬓女怜而善遇之。

9.3 後法章私以情告女，女遂与通。

9.4 及莒人共立法章为齐王，以莒距燕，而太史氏女遂为后，所谓"君王后"也。

王蠋殉节

10.1 燕之初入齐，闻画邑人王蠋贤，令军中曰"环画邑三十里无入"，以王蠋之故。

10.2 已而使人谓蠋曰：

"齐人多高子之义，吾以子为将，封子万家。"

10.3 蠋固谢。

10.4 燕人曰：

"子不听，吾引三军而屠画邑。"

10.5 王蠋曰：

NOTE: 王蠋遗言
"忠臣不事二君，贞女不更二夫。齐王不听吾谏，故退而耕於野。国既破亡，吾不能存；今又劫之以兵为君将，是助桀为暴也。与其生而无义，固不如烹！"

10.6 遂经其颈於树枝，自奋绝脰而死。

10.7 齐亡大夫闻之，曰：

"王蠋，布衣也，义不北面於燕，况在位食禄者乎！"

10.8 乃相聚如莒，求诸子，立为襄王。

赞

11.1 军法以正，实尚奇兵。
断轴自免，反间先行。
群鸟或众，五牛扬旌。
卒破骑劫，皆复齐城。
襄王嗣位，乃封安平。
