0 十二诸侯年表 原文

太史公读史

1.1 太史公读春秋历谱谍，至周厉王，未尝不废书而叹也。

1.2 曰：呜呼，师挚见之矣！

1.3 纣为象箸而箕子唏。

1.4 周道缺，诗人本之衽席，关雎作。

1.5 仁义陵迟，鹿鸣刺焉。

2.1 及至厉王，以恶闻其过，公卿惧诛而祸作，厉王遂奔于彘，乱自京师始，而共和行政焉。

2.2 是後或力政，彊乘弱，兴师不请天子。

2.3 然挟王室之义，以讨伐为会盟主，政由五伯，诸侯恣行，淫侈不轨，贼臣絪子滋起矣。

3.1 齐、晋、秦、楚其在成周微甚，封或百里或五十里。

3.2 晋阻三河，齐负东海，楚介江淮，秦因雍州之固，四海迭兴，更为伯主，文武所襃大封，皆威而服焉。

4.1 是以孔子明王道，干七十馀君，莫能用，故西观周室，论史记旧闻，兴於鲁而次春秋，上记隐，下至哀之获麟，约其辞文，去其烦重，以制义法，王道备，人事浃。

4.2 七十子之徒口受其传指，为有所刺讥襃讳挹损之文辞不可以书见也。

4.3 鲁君子左丘明惧弟子人人异端，各安其意，失其真，故因孔子史记具论其语，成左氏春秋。

5.1 铎椒为楚威王传，为王不能尽观春秋，采取成败，卒四十章，为铎氏微。

5.2 赵孝成王时，其相虞卿上采春秋，下观近势，亦著八篇，为虞氏春秋。

5.3 吕不韦者，秦庄襄王相，亦上观尚古，删拾春秋，集六国时事，以为八览、六论、十二纪，为吕氏春秋。

5.4 及如荀卿、孟子、公孙固、韩非之徒，各往往捃摭春秋之文以著书，不同胜纪。

5.5 汉相张苍历谱五德，上大夫董仲舒推春秋义，颇著文焉。

太史公曰

6.1 太史公曰：儒者断其义，驰说者骋其辞，不务综其终始；历人取其年月，数家隆於神运，谱谍独记世谥，其辞略，欲一观诸要难。

6.2 於是谱十二诸侯，自共和讫孔子，表见春秋、国语学者所讥盛衰大指著于篇，为成学治古文者要删焉。
