0 建元以来侯者年表 原文

太史公曰

1.1 太史公曰：匈奴绝和亲，攻当路塞；闽越擅伐，东瓯请降。

1.2 二夷交侵，当盛汉之隆，以此知功臣受封侔於祖考矣。

2.1 何者？自诗、书称三代"戎狄是膺，荆荼是征"，齐桓越燕伐山戎，武灵王以区区赵服单于，秦缪用百里霸西戎，吴楚之君以诸侯役百越。

2.2 况乃以中国一统，明天子在上，兼文武，席卷四海，内辑亿万之众，岂以晏然不为连境征伐哉！

2.3 自是後，遂出师北讨强胡，南诛劲越，将卒以次封矣。

赞

3 孝武之代，天下多虞。南讨瓯越，北击单于。长平鞠旅，冠军前驱。术阳衔璧，临蔡破禺。博陆上宰，平津巨儒。金章且佩，紫绶行纡。昭帝已後，勋宠不殊。惜哉绝笔，褚氏补诸。
