0 三代世表 原文

太史公曰

1.1 太史公曰：五帝、三代之记，尚矣。

1.2 自殷以前诸侯不可得而谱，周以来乃颇可著。

1.3 孔子因史文次春秋，纪元年，正时日月，盖其详哉。

1.4 至於序尚书则略，无年月；或颇有，然多阙，不可录。

1.5 故疑则传疑，盖其慎也。

2.1 余读谍记，黄帝以来皆有年数。

2.2 稽其历谱谍终始五德之传，古文咸不同，乖异。

2.3 夫子之弗论次其年月，岂虚哉！

2.4 於是以五帝系谍、尚书集世纪黄帝以来讫共和为世表。

褚先生补充

张夫子问

3.1 张夫子问褚先生曰：

"诗言契、后稷皆无父而生。今案诸传记咸言有父，父皆黄帝子也，得无与诗谬乎？"

褚先生答

4.1 褚先生曰：

"不然。诗言契生於卵，后稷人迹者，欲见其有天命精诚之意耳。"

4.2 "鬼神不能自成，须人而生，柰何无父而生乎！一言有父，一言无父，信以传信，疑以传疑，故两言之。"

4.3 "尧知契、稷皆贤人，天之所生，故封之契七十里，後十馀世至汤，王天下。"

4.4 "尧知后稷子孙之後王也，故益封之百里，其後世且千岁，至文王而有天下。"

5.1 "诗传曰：'汤之先为契，无父而生。契母与姊妹浴於玄丘水，有燕衔卵堕之，契母得，故含之，误吞之，即生契。"

5.2 "契生而贤，尧立为司徒，姓之曰子氏。子者兹；兹，益大也。"

5.3 "诗人美而颂之曰'殷社芒芒，天命玄鸟，降而生商'。商者质，殷号也。"

6.1 "文王之先为后稷，后稷亦无父而生。"

6.2 "后稷母为姜嫄，出见大人迹而履践之，知於身，则生后稷。"

6.3 "姜嫄以为无父，贱而弃之道中，牛羊避不践也。"

6.4 "抱之山中，山者养之。又捐之大泽，鸟覆席食之。"

6.5 "姜嫄怪之，於是知其天子，乃取长之。"

6.6 "尧知其贤才，立以为大农，姓之曰姬氏。姬者，本也。"

6.7 "诗人美而颂之曰'厥初生民'，深修益成，而道后稷之始也。"

7.1 "孔子曰：'昔者尧命契为子氏，为有汤也。
命后稷为姬氏，为有文王也。
大王命季历，明天瑞也。
太伯之吴，遂生源也。
'"

7.2 "天命难言，非圣人莫能见。"

7.3 "舜、禹、契、后稷皆黄帝子孙也。"

7.4 "黄帝策天命而治天下，德泽深後世，故其子孙皆复立为天子，是天之报有德也。"

7.5 "人不知，以为氾从布衣匹夫起耳。夫布衣匹夫安能无故而起王天下乎？其有天命然。"

黄帝后世与天命

黄帝後世

8.1 "黄帝後世何王天下之久远邪？"

8.2 曰："传云天下之君王为万夫之黔首请赎民之命者帝，有福万世。黄帝是也。"

8.3 "五政明则修礼义，因天时举兵征伐而利者王，有福千世。"

8.4 "蜀王，黄帝後世也，至今在汉西南五千里，常来朝降，输献於汉，非以其先之有德，泽流後世邪？"

8.5 "行道德岂可以忽乎哉！人君王者举而观之。"

霍光之事

9.1 "汉大将军霍子孟名光者，亦黄帝後世也。"

9.2 "此可为博闻远见者言，固难为浅闻者说也。何以言之？"

9.3 "古诸侯以国为姓。霍者，国名也。"

9.4 "武王封弟叔处於霍，後世晋献公灭霍公，後世为庶民，往来居平阳。"

9.5 "平阳在河东，河东晋地，分为卫国。"

9.6 "以诗言之，亦可为周世。周起后稷，后稷无父而生。"

9.7 "以三代世传言之，后稷有父名高辛；高辛，黄帝曾孙。"

9.8 "黄帝终始传曰：'汉兴百有馀年，有人不短不长，出燕之乡，持天下之政，时有婴兒主，欲行车。'"

9.9 "霍将军者，本居平阳燕。臣为郎时，与方士考功会旗亭下，为臣言。岂不伟哉！"
