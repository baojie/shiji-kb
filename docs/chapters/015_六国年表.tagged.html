0 六国年表 原文

太史公读秦记

1.1 太史公读秦记，至犬戎败幽王，周东徙洛邑，秦襄公始封为诸侯，作西畤用事上帝，僭端见矣。

1.2 礼曰："天子祭天地，诸侯祭其域内名山大川。"

1.3 今秦杂戎翟之俗，先暴戾，後仁义，位在籓臣而胪於郊祀，君子惧焉。

2.1 及文公逾陇，攘夷狄，尊陈宝，营岐雍之间，而穆公脩政，东竟至河，则与齐桓、晋文中国侯伯侔矣。

2.2 是後陪臣执政，大夫世禄，六卿擅晋权，征伐会盟，威重於诸侯。

2.3 及田常杀简公而相齐国，诸侯晏然弗讨，海内争於战功矣。

3.1 三国终之卒分晋，田和亦灭齐而有之，六国之盛自此始。

3.2 务在彊兵并敌，谋诈用而从衡短长之说起。

3.3 矫称出，誓盟不信，虽置质剖符犹不能约束也。

4.1 秦始小国僻远，诸夏宾之，比於戎翟，至献公之後常雄诸侯。

4.2 论秦之德义不如鲁卫之暴戾者，量秦之兵不如三晋之彊也，然卒并天下，非必险固便形埶利也，盖若天所助焉。

5.1 或曰"东方物所始生，西方物之成孰"。

5.2 夫作事者必於东南，收功实者常於西北。

5.3 故禹兴於西羌，汤起於亳，周之王也以丰镐伐殷，秦之帝用雍州兴，汉之兴自蜀汉。

6.1 秦既得意，烧天下诗书，诸侯史记尤甚，为其有所刺讥也。

6.2 诗书所以复见者，多藏人家，而史记独藏周室，以故灭。惜哉，惜哉！

6.3 独有秦记，又不载日月，其文略不具。

6.4 然战国之权变亦有可颇采者，何必上古。

7.1 秦取天下多暴，然世异变，成功大。

7.2 传曰"法後王"，何也？以其近己而俗变相类，议卑而易行也。

7.3 学者牵於所闻，见秦在帝位日浅，不察其终始，因举而笑之，不敢道，此与以耳食无异。悲夫！

太史公曰

8.1 余於是因秦记，踵春秋之後，起周元王，表六国时事，讫二世，凡二百七十年，著诸所闻兴坏之端。

8.2 後有君子，以览观焉。
