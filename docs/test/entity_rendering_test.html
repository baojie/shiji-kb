实体 vs 动词 对比

类型 背景色 文字色 字重 特殊样式
名词实体 统一淡黄 rgba(255, 248, 220, 0.6) 7种语义色 normal 5种下划线组合
动词标注 4种鲜明底色（红/蓝/金/绿） 白色 bold 无下划线
设计理念：
  • 名词实体：淡雅底色 + 语义化颜色分组 + 下划线辅助 → 适合长文阅读，不喧宾夺主
  • 动词标注：鲜明底色 + 加粗字体 → 突出历史动作，快速定位事件关键节点
  • 视觉层次：动词（强）> 实体（柔），符合叙事文本的信息结构

综合测试：经典段落渲染

段落一：鸿门宴

汉元年十月， 沛公攻咸阳。 项羽闻之大怒， 进兵鸿门， 欲击沛公。 项伯夜驰告张良， 良乃见沛公。 翌日，沛公从百余骑谒项羽于鸿门。 范增数目项王， 举玉玦示之者三。 项庄请舞剑， 项伯亦起舞，常以身翼蔽沛公。 张良出，召樊哙。 樊哙闯入， 项王赐之酒。 坐须臾，沛公起如厕， 因脱身独去， 从郦山下道走芷阳间行。 项王终杀秦降王子婴， 烧秦宫室，火三月不灭。

段落二：垓下之围

五年， 高祖与诸侯兵围项羽于垓下。 项羽兵十万， 粮尽。 汉军及诸侯兵围之数重。 夜闻汉军四面皆楚歌， 项羽乃大惊曰： "汉皆已得楚乎？ 是何楚人之多也！" 项王则夜起， 饮于帐中。 有美人名虞， 常幸从； 骏马名骓， 常骑之。 于是项王乃悲歌慷慨， 歌曰： "力拔山兮气盖世， 时不利兮骓不逝。 骓不逝兮可奈何， 虞兮虞兮奈若何！" 于是项王乃上马， 麾下壮士八百余人从。 直夜溃围南出， 驰走。 平明，汉军乃觉之， 令骑将灌婴以五千骑追之。 项王渡淮， 骑能属者百余人耳。 项王至阴陵， 迷失道。 至东城， 乃有二十八骑。 汉骑追者数千人。 项王自刎而死。
