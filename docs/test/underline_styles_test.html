浏览器兼容性说明

  • text-decoration-thickness 和 text-underline-offset 在现代浏览器中支持良好
  • wavy 波浪线在 Firefox 和 Chrome 现代版本中支持
  • 3D边框效果（groove/ridge/inset/outset）在不同浏览器中渲染可能有差异
  • 渐变下划线通过 background-image 实现，兼容性最好
  • 建议优先使用 border-bottom 或 text-decoration 的基础样式
