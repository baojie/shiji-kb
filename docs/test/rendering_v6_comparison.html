实体渲染方案对比：v5.0 vs v6.0 增强版

v6.0 主要改进点

1. 底色分层： 第1层（核心）淡金底 第2层（常规）淡黄底 第3层（次要）极淡黄底
2. 下划线增强：彩色匹配、粗细分级、双线、渐变线
3. 特殊标识：帝王人物、重大历史事件

v6.0 核心改进总结

立即可实施的改进（无需标注变更）

  1. 下划线彩色化：下划线颜色与文字颜色匹配，视觉更和谐统一
  2. 波浪线增强：典籍、神话的波浪线从1px加粗到2px，偏移2px
  3. 朝代突出：朝代下划线从1px升级到2px，强调政治重要性
  4. 器物添线：器物从无下划线改为彩色细虚线
  5. 制度添线：制度从无下划线改为彩色细实线
  6. 底色微调：次要实体（制度、礼仪、身份、器物等）使用更淡的底色

需要标注支持的增强（可选，未来实施）

  1. 帝王标识：深红色+淡金底+粗双线，需标注 〖@帝:秦始皇〗
  2. 重要人物：中粗实线，需标注 〖@重:李斯〗
  3. 重大时间：淡金底+中粗实线，需标注 〖#重:秦统一六国〗
  4. 重大事件：渐变下划线，需标注 〖◆事:焚书坑儒〗

推荐实施策略

方案B：部分实施v6.0（推荐）
先实施无需标注变更的6项改进，效果立竿见影。
帝王标识、重大事件等高级功能留待未来，根据用户反馈决定是否实施。

这样可以在不增加标注工作量的前提下，显著提升视觉效果和阅读体验。
