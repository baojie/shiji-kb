史记读音争议

史记中部分专有名词的读音在学术界存在争议。本页面记录本知识库采用的读音标准。

详细的文献考证和学术讨论请见各项链接的 GitHub Discussion。

争议读音列表

身毒 （古代对印度的称呼）

争议读音：
✓ 本库采用
shēn dú（身毒）
争议读音
yuán dú（捐毒）
采用理由：《史记索隐》《史记正义》明确注音为 shēn dú，这是汉代对梵文 Sindhu（信度河）的音译，应遵循古音。"捐毒"是另一种音译方式，但不是主流。
查看详细考证和讨论

月氏 （古代西域民族，后建立贵霜帝国）

争议读音：
✓ 本库采用
yuè zhī（月支）
争议读音
ròu zhī（肉支）
采用理由：《史记索隐》《史记正义》明确注音"氏音支"，唐代以前史书普遍读 yuè zhī。这是对民族名称的音译，应保留古音。学术界主流观点（余太山等）也支持此读音。

读音标注原则

本知识库在标注读音时，遵循以下原则：

  1. 尊重古注：优先采用《史记集解》《史记索隐》《史记正义》三家注的注音
  2. 学术共识：参考现代学术界主流观点和考证成果
  3. 音译原则：对于外来民族、地名等音译词，保留古代读音
  4. 公开透明：所有争议读音的详细考证在 GitHub Discussions 公开讨论

如您对读音标注有不同见解，欢迎在相应的 Discussion 中参与讨论，
或通过 GitHub Issues 提出新的读音问题。
