## 备注

**历史典故：高渐离击秦**

场景还原：
1. 荆轲刺秦失败后，高渐离隐姓埋名避难
2. 后被秦始皇发现，因善击筑而被召入宫
3. 秦始皇恐其行刺，熏瞎其眼
4. 高渐离在筑中灌铅，趁机击杀秦始皇
5. **"举筑朴秦皇帝，不中"** - 挥筑击向秦始皇，但没有击中
6. 高渐离被杀，秦始皇终身不再接近六国之人

**相关人物**：
- 高渐离：燕国著名音乐家，善击筑
- 荆轲：高渐离挚友，刺秦失败
- 易水送别时即为高渐离击筑，荆轲和而歌

**成语典故**：
- "百发百中"
- "百步穿杨"
- "一箭双雕"（均与"中"的zhòng读音相关）

**语言学特征**：
- zhōng（中间）：名词/方位词，上古音*truŋ
- zhòng（击中）：动词，上古音*truŋs（带使动后缀-s）
- 两个读音反映了词性的演变：名词→动词（使动）
