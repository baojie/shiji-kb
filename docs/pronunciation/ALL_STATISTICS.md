## 📊 统计方法

### 自动化工具

```bash
# 分析单个多音字
python scripts/analyze_polyphone_statistics.py 王 docs/pronunciation/王_wáng.draft.md

# 批量分析所有高频多音字
python scripts/analyze_polyphone_statistics.py --batch
```

### 人工审核流程

1. **读取草稿报告**: 查看自动生成的高频词组
2. **分类读音**: 根据词组含义判断读音
3. **统计频率**: 计算各读音的使用占比
4. **确定方案**: 根据SKILL_01d v3.0原则确定词表处理方案
5. **补充内容**: 添加典型例句、历史典故、语言学注释
6. **生成正式文档**: 将`.draft.md`转换为正式`.md`文档
