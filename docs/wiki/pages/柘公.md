# 桓公终生

出现于《史记》**1** 处。

## 生平

柘公，楚汉相争时楚军地方守将，王武下属，驻守柘（今河南柘城）一带。[[灌婴]]率军追击项羽残部过程中，击破柘公王武，柘公所部败走，汉军驻扎于燕西（河南境内）。灌婴部卒斩楼烦将五人、连尹一人。柘公是楚汉战争末期楚国守卫地方城邑的军事长官之一，以被灌婴所破的战败记录存于《史记》。事见[[095_樊郦滕灌列传|樊郦滕灌列传]]。
