# 仓颉

[[仓颉]]，上古神话人物，传说为[[黄帝]]之史官，**汉字的创造者**。

据传仓颉观鸟兽足迹而悟，创制文字，以代结绳之政，
文字的发明使人类文明得以传承记录。
传说"仓颉作书，而天雨粟，鬼夜哭"。

仓颉与[[岐伯]]（医学）、[[嫘祖]]（蚕丝）同列，
为黄帝时代文明创制的传说人物之一。

## 生平

仓颉为[[黄帝]]时代传说中的史官，《史记·五帝本纪》于[[黄帝]]条下有所提及。相传仓颉四目重瞳，观鸟兽足迹及万物形态，创制书契（文字），以代结绳记事之法。《淮南子》记载"仓颉作书，而天雨粟，鬼夜哭"，意谓文字出现后，天机外泄，神鬼皆惊。

仓颉在《史记》中并非独立传记对象，其事迹主要见于后世注疏引申，属传说层面的文明创制人物，与[[岐伯]]（医学始祖）、[[嫘祖]]（养蚕始祖）同为黄帝时代文化符号。
