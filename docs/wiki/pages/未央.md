# 未央

未央，即未央宫，西汉皇朝的核心宫殿，由刘邦称帝后命萧何主持修建，位于今陕西西安西北，是两汉政治权力的中心地标。"未央"取"未尽""无穷"之意，寓意皇祚绵延、天命永续，是汉家礼制秩序在建筑上的集中体现。

《封禅书》在记述汉武帝以柏梁台失火为由兴建建章宫、迁移朝政于甘泉宫时，以未央宫为对比背景，揭示汉武帝晚年求仙好道导致朝政格局变化、中央礼制建筑功能转移的历史脉络。《龟策列传》引元王之言论中，以龟策占卜与古代贤君为主题展开论辩，未央宫作为典故背景出现，是汉代宫廷文化中龟策信仰与皇权秩序相互交融的叙事场域。两书合观，未央宫在《史记》中既是汉代最高权力的物质象征，也是礼制、方术与皇权意志在同一空间中相互缠绕的历史地理坐标。
