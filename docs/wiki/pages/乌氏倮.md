# 乌氏倮

秦代西北富商，以畜牧起家。大量繁殖牲畜后卖出，以所得金钱购买奇珍异物献于戎王，戎王回赠牲畜以十倍计。秦始皇时因财富与势力可比肩封君，诏令乌氏倮与封君同列，按时朝见皇帝，是货殖列传中记载的秦代典型致富人物之一。

## 货殖列传与乌氏倮

货殖列传载：乌氏倮畜牧，及众，斥卖，求奇缯物，间献遗戎王，戎王十倍其偿，与封君比，秦始皇帝令倮比封君（[[129_货殖列传]]）。

## 历代评价

**太史公评价（叙述视角）**：司马迁在《货殖列传》中以乌氏倮与寡妇清并列，作为秦代因经商致富而受秦始皇礼遇的商人代表。乌氏倮通过畜牧（养马）起家，以物易物，贡献给戎王，赚取大量财货，最终"秦始皇帝令倮比封君，以时与列臣朝请"——一个商人被赋予贵族礼遇，与封君同列朝贺。太史公对此以惊叹的语气记述，暗含对商业力量突破等级制度的历史观察。

**汉代史学定论**：汉代史家将乌氏倮视为秦代边疆贸易成功的代表，其与北方戎族的互市关系被认为有助于秦国经济和边防的稳定。然而汉代的抑商政策使得汉儒对乌氏倮的褒扬态度较为矛盾：一方面认可其成就的历史事实，另一方面又刻意强调商业成功不应成为追求的目标，更多是将其作为秦代功利主义的例证而非道德模范。

**后世主要争议**：后世讨论乌氏倮的焦点集中于以下几点：一是乌氏倮的民族身份——"乌氏"是否为氏族名、是否具有北方少数民族背景，学者至今有争论；二是秦始皇赐予其"比封君"礼遇的动机——是出于对商业贡献的奖励，还是笼络边疆势力的政治手段；三是乌氏倮的经营模式（牧马-贸易-积累）是否可视为早期丝路贸易的前身，在中西文化交流史研究中引发讨论。

**历史影响 / 特殊符号意义**：乌氏倮与寡妇清并列，成为《货殖列传》中"以财富突破阶层"主题的双核符号，代表秦代商业力量冲击等级制度的历史现象。后世讨论中国古代商业文化、边疆贸易和财富伦理时，乌氏倮是不可绕过的案例；其故事也被视为先秦时代存在某种形式的"重商意识"的证据，与儒家正统的抑商论形成对照。
