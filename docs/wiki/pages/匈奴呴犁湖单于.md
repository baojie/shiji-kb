# 匈奴呴犁湖单于

出现于《史记》**3** 处。

## 生平

匈奴呴犁湖单于，西汉武帝晚期匈奴单于，乌维单于之弟，右贤王，乌师庐单于幼年早死后继立。即位后汉光禄徐自为出五原塞筑城列亭至庐朐，汉军深入匈奴腹地构建防线。呴犁湖在位约二年（前101—前99年）即死，匈奴再立其弟之子且鞮侯为单于。事见[[110_匈奴列传|匈奴列传]]。
