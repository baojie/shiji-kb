# 女房

出现于《史记》**1** 处。

## 生平

女房，上古传说人物，帝喾之妃，简狄之姐妹（一说）。帝喾高辛氏有四妃，各生一子：元妃姜嫄生后稷，次妃简狄吞玄鸟卵生契（商祖），三妃陈丰氏生帝尧，四妃娵訾氏生帝挚。女房相关记载见于三代世表及五帝本纪注疏。事见[[013_三代世表|三代世表]]。
