# 户牖

户牖，古地名，全称"阳武户牖乡"，位于今河南兰考一带，是[[陈平]]的故乡。《史记·陈丞相世家》开篇即载"陈丞相平者，阳武户牖乡人也"，并记述陈平少时家贫、与兄共耕、喜读书游学的经历，户牖乡是陈平早年成长的社会背景。

刘邦灭楚后，以户牖乡作为封地封赏陈平，"封平以户牖乡"，以示嘉奖其奇谋之功。户牖虽为小乡，却因陈平这位"奇谋之臣"而在《史记》中多次被提及，成为汉初政治史上的一个标志性地名。
