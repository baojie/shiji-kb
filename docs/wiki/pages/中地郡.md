# 中地郡

中地郡，汉高祖初年设立的关中北部郡级行政区，位于今陕西北部泾河流域，是汉高祖击败项羽后对秦旧地进行行政整合的产物。中地郡的设立标志着汉代对关中地区正式完成军事与行政双重控制。

《高祖本纪》载，刘邦"置陇西、北地、上郡、渭南、河上、中地郡；关外置河南郡"，在楚汉战争的关键时期一举设置六郡，以行政区划巩固军事占领成果，中地郡作为其中之一列于关中北部体系之内。《秦楚之际月表》亦记"属汉，为陇西、北地、中地郡"，与《本纪》互证，确认中地郡设置的历史节点。两书合观，中地郡是汉高祖将关中秦旧地重新纳入汉家行政体系的具体地理成果，是《史记》中汉代建制史的微观坐标。
