# 汾阳

汾阳，晋国境内汾水北岸的城邑，地处今山西汾阳市一带，是春秋时期晋国内乱期间多次出现于政治叙事的历史地名。汾阳在晋国君位争夺的历史背景下多次成为事件发生地，是晋国公室内斗与权力交接的地理见证之一。

《晋世家》两度涉及汾阳：一处记晋国内乱期间，里克等人迎立公子重耳、拥立事件相关叙事以汾阳为背景；另一处记晋惠公夷吾即位后派邳郑谢秦，回绝当初许诺割让河西之地，其外交斡旋的历史背景同样与汾阳一带的晋国政局密切相关。两处合观，汾阳在《史记》晋世家叙事中是公室夺权与外交斡旋相互交织的历史地理节点，是晋国政治动荡时代的地理记忆。
