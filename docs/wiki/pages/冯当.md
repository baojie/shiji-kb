# 冯当

出现于《史记》**1** 处。

## 生平

冯当，西汉武帝时蜀郡太守，以暴虐残酷著称，司马迁在《酷吏列传》末尾将其与广汉李贞、东郡弥仆等并列，称为"蜀守冯当暴挫"，属于地方酷吏中滥用职权的典型代表。事见[[122_酷吏列传|酷吏列传]]。
