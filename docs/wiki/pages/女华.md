# 女华

出现于《史记》**2** 处。

## 生平

女华，上古传说人物，少典之女，大费（伯益）之母。少典之女女华嫁给大业（颛顼苗裔孙），生大费（伯益）。伯益助禹治水，帝舜赐嬴姓，女华因此成为嬴姓先祖世系的重要一环，秦国先祖世系由此追溯至女华与大业。事见[[005_秦本纪|秦本纪]]。
