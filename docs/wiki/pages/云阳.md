# 云阳

云阳，秦汉时期重要城邑，今陕西淳化一带，地处关中北部，是秦国祭天及北方军事要道上的重要节点。《秦始皇本纪》记载，法家名士[[韩非]]被扣押秦国，"非死云阳"，韩非在此地饮鸩而亡，是其悲剧结局的历史坐标；秦始皇晚年还曾"徙五万家云阳，复不事十岁"，将大批民众强制迁往此地充实边疆。

汉武帝在云阳修建甘泉宫后，云阳成为汉代重要的礼制场所。《封禅书》与《孝武本纪》载汉武帝在云阳"始郊见泰一"，以郊祀仪式祭拜太一神，"是夜有美光，及昼，黄气上属天"；《匈奴列传》则记秦代"通直道，自九原至云阳"，说明云阳是连接关中与北方边塞的直道南端起点，具有重要的军事交通意义。
