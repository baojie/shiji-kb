# 临邛

临邛，汉代蜀郡城邑，今四川邛崃，是秦汉时期西南地区的工商业重镇。[[货殖列传]]详载，秦灭赵后，赵人卓氏迁至临邛，以铁冶发家，"富至僮千人"；山东迁虏程郑亦在临邛冶铸致富，"富埒卓氏"，两大豪富共居一地，显示临邛在汉代西南冶铁业中的核心地位。

《司马相如列传》记载[[司马相如]]游历临邛，与临邛令王吉相善，在富商卓王孙家中以琴声打动卓文君，二人私奔成都，此后在临邛开酒肆谋生，演绎了一段广为流传的才子佳人故事。临邛紧邻通往西南夷的要道，汉武帝时期曾以此为据点开拓夜郎、滇池等地的西南通道。
