# 首山

首山，《史记》中见于不同地域背景的同名山岳，主要有晋国首山与河南首山两处。晋国之首山在今山西南部汾河一带，是晋大夫[[赵盾]]游猎之所；汉武帝封禅路线上的首山则在今河南灵宝一带，与汾阴祭祀体系相关联。两处同名，反映上古地名命名的区域重叠现象。

《晋世家》载，"盾常田首山，见桑下有饿人"，赵盾在首山田猎时，偶遇桑下饿毙的示眯明，以食救之；此人后来在赵盾遭刺时舍命相救，成就了《史记》中"以德报恩"叙事的经典场景。《封禅书》与《孝武本纪》中，汉武帝采铜于首山铸鼎，以首山为封禅祭地巡幸的重要坐标，赋予首山国家祭典的神圣意涵，使其从地方猎场跃升为帝国礼制空间的一部分。

> **【索隐】** 邹诞云示眯为祁弥也，即《左传》之提弥明也。提音市移反，刘氏亦音祁为时移反，则祁提二字同音也。而此《史记》作「示」者，示即《周礼》古本「地神曰祇」，皆作「示」字。「邹」为「祁」者，盖由祇提音相近，字遂变为「祁」也。眯音米移反。以「眯」为「弥」，亦音相近耳。又左氏桑下饿人是灵辄也。其示眯明，是嗾獒者也，其人鬬而死。今合二人为一人，非也。
