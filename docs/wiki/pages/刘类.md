# 刘类

出现于《史记》**2** 处。

## 生平

刘类，西汉武帝时宗室封侯，[[城阳顷王]]之子，封东淮侯国，为侯。元朔元年四月戊寅以王子身份受封，在位约四年，元狩五年坐酎金律，封国被除。事见[[021_建元已来王子侯者年表|建元已来王子侯者年表]]。
