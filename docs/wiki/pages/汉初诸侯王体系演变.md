# 汉初诸侯王体系演变

《汉兴以来诸侯王年表》系统记录了汉朝建国至武帝太初年间（前206—前97）近110年诸侯王国的分封、削减、分化、除国全过程，是汉代中央与地方关系百年调整的完整档案。

## 制度背景

### 周制的衰落与汉制的简化

周代设公、侯、伯、子、男五等爵，大国地四百里，小国方百里至三十里，目的在于"辅卫王室"。厉王、幽王之后，王室衰落，诸侯强大，天子微而弗能正。司马迁的判断是：这不是德行问题，而是**形势使然**。

汉制简化为王、侯二等（废除公伯子男），但汉代王爵实力远超周制——封地动辄数郡，连城数十。

## 演变四阶段

### 阶段一：异姓王并立（前206—前201）

高祖建国初，分封异姓七王：楚王韩信、梁王彭越、淮南王英布、燕王臧荼（后改卢绾）、赵王张耳（后改张敖）、长沙王吴芮。

| 异姓王 | 结局 | 原因 |
|-------|------|------|
| 楚王/齐王韩信 | 贬为淮阴侯，后诛（前196） | 谋反 |
| 燕王臧荼 | 诛（前202） | 谋反 |
| 梁王彭越 | 诛（前196） | 谋反 |
| 淮南王英布 | 诛（前196） | 谋反 |
| 燕王卢绾 | 降匈奴（前195） | 被疑 |
| 赵王张敖 | 废（前200） | 谋反（冤案） |
| 长沙王吴氏 | 幸存至景帝 | 国小偏远，被信任 |

除长沙王外，异姓王在高祖末年前全部被清除——"非刘氏而王，天下共击之"的[[白马之盟]]由此而来。

### 阶段二：同姓王网络（前201—前157）

高祖"封子弟九国"，以同姓拱卫王室，理由是"天下初定，骨肉同姓少，故广彊庶孽，以镇抚四海"。

**九同姓王**：楚王刘交、齐王刘肥、[[吴王刘濞]]、淮南王刘长、燕王刘建、赵王[[刘如意]]、梁王刘恢、淮阳王刘友、代王刘恒（后为文帝）。

地理规模：大者"五六郡，连城数十"；汉直辖郡仅15个。

吕后乱政冲击：三赵王（如意、友、恢）相继被害，吕氏诸王（台、产、禄、通）填补空位。诸吕被诛后，刘氏诸王得以恢复。

### 阶段三：[[七国之乱]]与大削（前154—前141）

景帝三年（前154），吴王刘濞联合楚、赵、胶西、胶东、菑川、济南六国，以诛晁错为名举兵。七国皆平，七王皆死。

七国之乱后，景帝全面削弱诸侯王权：
- 收回任命王国官员权力（丞相改称"相"，由中央任命）
- 收回军事权力
- 削减封地
- 制度性成果：诸侯王从"自署百官"的半独立国变为"王食其租税，不得治民"的虚位诸侯

### 阶段四：[[推恩令]]与彻底分化（武帝时期）

**推恩令**（元朔二年，前127）：允许诸侯王"推恩"分封子弟为侯，被分出的小侯国归郡守管辖。大国被迫化整为零：

| 大国 | 分为 | 具体 |
|-----|------|------|
| 齐 | 七国 | 齐、城阳、济北、济南、菑川、胶西、胶东 |
| 赵 | 六国 | 赵、河间、广川、常山、中山、清河 |
| 梁 | 五国 | 梁、济川、济东、山阳、济阴 |
| 淮南 | 三国 | 淮南、衡山、庐江 |

**酎金夺爵**（元鼎五年，前112）：各诸侯按规格献酎金，少府检查，百余侯因此除国。

**左官律**：投奔诸侯王的汉臣称"左官"，禁止任用于中央官位。

**附益法**：禁止诸侯王相互援助，禁止朝臣与诸侯王交通。

### 最终格局

> 诸侯稍微，大国不过十余城，小侯不过数十里，上足以奉贡职，下足以供养祭祀，以蕃辅京师。而汉郡八九十，形错诸侯间，犬牙相临，秉其戹塞地利，彊本幹，弱枝叶之势，尊卑明而万事各得其所矣。

汉郡从高祖时期的15郡增至八九十郡，诸侯王国实力大幅萎缩。

## 三类演变规律

### 规律一：大国必分

凡地域过大、权力过重的王国必然被分化，推动力有三：中央主动削藩、诸侯内部争嗣、推恩令下的"自愿"分封。

### 规律二：亲疏差序

同姓王中，与皇帝关系越远越早被削：第一代（高祖子孙）有实权；第二代（文帝子孙）受限；第三代（景帝子孙）只有名位；第四代以后多被削爵除国。

### 规律三：长沙特例

长沙国是唯一贯穿高祖至景帝五朝的异姓王国：地处偏远、国小不足以反、配合中央有功无过。"偏远+合作+不引起猜忌"是诸侯的生存之道。

## 八个关键节点

| 时间 | 事件 | 制度影响 |
|------|------|---------|
| 前206（高祖元年） | 封异姓王 | 异姓王体系建立 |
| 前201（高祖六年） | 封同姓九国 | 同姓王体系建立 |
| 前195—前180（吕后时期） | 三赵王死，诸吕封王 | 体系异化 |
| 前180—前157（文帝时期） | 恢复刘氏诸王，增封 | 体系重建 |
| 前154（景帝三年） | 七国之乱平定 | 大削藩 |
| 前141后（武帝初） | 推恩令准备 | 削藩升级 |
| 前127（武帝元朔二年） | 推恩令颁布 | 大国化小国 |
| 前112（武帝元鼎五年） | 酎金夺爵 | 侯国锐减 |

## 历史意义

推恩令不是武力削藩，而是"让诸侯自己削弱自己"——打着"恩惠"的旗号完成削藩，被后世视为"柔性变革"的经典。汉代由此从秦制（纯郡县制）→ 汉初（同姓王[[分封制]]）→ 武帝（郡县为主、侯国为辅），用百年实现中央集权的制度化。
