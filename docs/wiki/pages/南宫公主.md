# 南宫公主

出现于《史记》**1** 处。

## 生平

南宫公主，西汉汉武帝之姐，[[汉景帝]]之女，[[王太后]]（王娡）所生，排行第二。王太后生三女：长女为[[平阳公主]]，次为南宫公主，三为[[林虑公主]]；又生武帝刘彻，武帝即位后，王太后以皇太后之尊，三公主皆受封宠。事见[[049_外戚世家|外戚世家]]。
