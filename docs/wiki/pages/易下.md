# 易下

易下，汉初军事地名，地处今河北易县一带，是汉高祖平定燕王臧荼叛乱时的主要战场。易下之战是西汉建国初年清剿异姓诸侯王叛乱的重要战役，也是周勃、灌婴等汉初名将的重要战功之地。

《绛侯周勃世家》载，高帝即位之年，燕王臧荼反叛，周勃随高帝出征，"破之易下"，以此战功进一步巩固了其汉初军事重臣的地位。《樊郦滕灌列传》亦记灌婴同一场战役中的战功："战龙脱，先登陷阵，破荼军易下，卻敌，迁为右丞相，赐爵列侯"，两书同记易下之战，互相印证了这场叛乱平定中汉军的具体战场位置与参战将领的立功经过。易下因此成为汉初翦除异姓诸侯王这一政治进程中的具体地理见证，是《史记》汉初叙事中武功与政治转型交汇的重要地名节点。
