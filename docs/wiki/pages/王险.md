# 王险

王险，卫满朝鲜的国都，地望约在今朝鲜平壤大同江南岸，是《史记·朝鲜列传》叙事的核心地理坐标。燕人卫满率部东渡浿水，聚众立国，"都王险"，王险由此成为东北方向汉朝藩属国的政治中心。

《朝鲜列传》记汉武帝元封二年发兵讨伐朝鲜，楼船将军杨仆"先至王险"，朝鲜王右渠出城击退汉军；此后左将军荀彘合并两军，"急击朝鲜"，围攻王险城，朝鲜内部相臣分裂，右渠被杀，"遂定朝鲜为四郡"。王险陷落标志着卫满朝鲜政权的终结，汉朝东北方向郡县化的完成。
