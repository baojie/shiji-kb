# 刘燕

出现于《史记》**1** 处。

## 生平

刘燕，西汉武帝时宗室封侯，[[齐孝王]]之子，封披阳侯国，谥敬侯。元狩四年四月乙卯以王子身份受封，在位约四年，其后由今侯刘隅继承侯位。事见[[021_建元已来王子侯者年表|建元已来王子侯者年表]]。
