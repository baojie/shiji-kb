# 张掖

张掖，汉武帝时期设置的河西四郡之一，位于今甘肃张掖一带，地处河西走廊中段，是汉朝控制西域通道、阻断匈奴南下的战略要地。《平准书》载汉武帝"初置张掖、酒泉郡，而上郡、朔方、西河、河西开田官，斥塞卒六十万人戍田之"，张掖郡的设置是汉武帝河西战略的核心成果，也是汉廷财政大规模支出的直接原因。

《李将军列传》记李陵"将丹阳楚人五千人，教射酒泉、张掖以屯卫胡"，张掖成为汉军训练边防卒的据点；《匈奴列传》载匈奴右贤王入侵"酒泉、张掖，略数千人"，张掖频繁遭受匈奴侵扰，是汉匈边境拉锯战的前沿战场；三书互证，共同呈现了张掖郡作为河西走廊战略要冲的军事与财政双重价值。
