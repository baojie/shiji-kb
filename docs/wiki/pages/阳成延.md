# 阳成延

[[阳成延]]，西汉功臣，以军功受封**梧侯**，谥**齐侯**。
高后元年（前 187）四月乙酉受封，食邑五百户。

《史记》记载见[[018_高祖功臣侯者年表|018 高祖功臣侯者年表]]。

## 生平

阳成延，西汉功臣，封齐侯（后改敬侯），以建造长乐、未央宫及修筑长安城的工程功勋获封。阳成延以「先就」（最先完成）之功封侯五百户，惠帝元年四月乙酉封，谥「齐侯」，在位二年后传敬侯去疾，再传靖侯偃，侯系延续数代。阳成延是汉初城市建设工程中以工程快速完工获封的功臣典型，代表汉代非军事功勋封侯的案例。事见[[019_惠景间侯者年表|惠景间侯者年表]]。
