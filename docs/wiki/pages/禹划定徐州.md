# 禹划定徐州

> 海岱及淮维徐州：淮、沂其治，蒙、羽其。大野既都，东原厎平。其土赤埴坟，草木渐包。其田上中，赋中中。贡维土五色，羽畎夏狄，峄阳孤桐，泗滨浮磬，淮夷蠙珠臮鱼，其篚玄纤缟。浮于淮、泗，通于河。

**主要人物**：禹

## 事件经过

海岱及淮维徐州：淮、沂其治，蒙、羽其。大野既都，东原厎平。其土赤埴坟，草木渐包。其田上中，赋中中。贡维土五色，羽畎夏狄，峄阳孤桐，泗滨浮磬，淮夷蠙珠臮鱼，其篚玄纤缟。浮于淮、泗，通于河。
