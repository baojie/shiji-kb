# 郏

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

郏，古地名，位于今河南省郏县一带，属汉代颍川郡。春秋战国时期郏地为楚国北境重要据点。《史记》中郏与楚国北部政治地理相关，是中原争霸叙事中的地理坐标之一。郏鄏（洛邑）是周王室迁都的历史地名，但此处郏县位置与洛邑不同，是独立的地名单元。
