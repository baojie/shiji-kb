# 宰夫

出现于《史记》**3** 处。

## 生平

宰夫，春秋时期官职，掌管膳食烹饪的官员。《史记》中宰夫见于几处著名典故：郑灵公烹鼋，宰夫参与烹制，「染指于鼎」事件即发生于此宴席。宰夫作为厨政人员，在先秦礼仪中具有重要地位。事见[[042_郑世家|郑世家]]。
