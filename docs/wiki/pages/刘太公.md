# 刘太公

[[刘邦]]之父，中国历史上第一位在世时被尊为太上皇的人。名讳不详，史称"太公"。

## 生平

沛县普通农家，[[刘媪]]之夫，刘邦之父。刘邦起兵后，太公曾被[[项羽]]俘虏作为要挟，刘邦以"分我一杯羹"应对，后被归还。

汉朝建立后，刘邦五日一朝太公，行家人父子之礼。群臣以天下无二尊为由，建议刘邦尊父为太上皇（前201年），刘邦遂尊之，此为中国历史上首位在世太上皇。

汉高祖九年（前197年）崩，葬于栎阳。
