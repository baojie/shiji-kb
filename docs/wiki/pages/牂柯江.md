# 牂柯江

牂柯江，即今北盘江，发源于云贵高原，南流经贵州汇入珠江水系，出番禺（广州）附近入海。《西南夷列传》记[[唐蒙]]在南越得知蜀枸酱经牂柯江输送，惊悉"牂柯江广数里，出番禺城下"，意识到此江可以作为从巴蜀绕道进攻南越的战略通道。

《南越列传》记汉武帝元鼎五年平南越之役，多路兵马同时出动，其中"使驰义侯因巴蜀罪人，发夜郎兵，下牂柯江"，以牂柯江为进攻路线之一，然而夜郎兵尚在途中，南越已经平定。牂柯江作为西南夷与南越之间的天然水道，是汉代西南战略中最具战略想象力的交通构想，也是汉帝国对西南地理认知逐步深化的重要坐标。
