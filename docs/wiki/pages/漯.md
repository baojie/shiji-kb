# 漯

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

漯（漯阴），古地名，位于今山东省高唐县东南一带，属汉代清河郡。漯水是古黄河的一个支流或分流，漯阴是漯水之南的古代聚落，是华北平原黄河下游地名体系的组成部分，《史记》中出现于齐赵相关地区叙事。
