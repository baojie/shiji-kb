# 刘谭

出现于《史记》**1** 处。

## 生平

刘谭，西汉武帝时宗室封侯，[[城阳顷王]]之子，封庸侯国，为侯。元朔元年四月戊寅以王子身份受封，在位年数较长，侯位延续多代。事见[[021_建元已来王子侯者年表|建元已来王子侯者年表]]。
