# 丰沛

丰沛，丰邑与沛县的合称，今江苏徐州一带，是汉高祖刘邦的故乡，也是汉帝国诞生的地理摇篮。刘邦"起于丰沛"的历史叙述在《史记》中成为汉朝开国的代名词，丰沛由地名升华为汉兴精神的地理象征。

《淮南衡山列传》以"高皇始于丰沛，一倡天下不期而响应者不可胜数"概括汉兴的历史动力，将刘邦崛起的历史奇迹归因于民心所向；《秦楚之际月表》则记秦末乱局中"周市东略地丰沛间"，描绘了秦末群雄分裂局面中各方势力对丰沛地区的争夺，揭示了这一地区在楚汉战争起点处的战略价值。
