# 李广征战时间线

[[李广]]（？— 前119）从文帝十四年（前166年）首战萧关，
至元狩四年（前119年）漠北迷道自刎，
四十七年间大小七十余战，号称"飞将军"，
终身无缘封侯，是汉代名将中最著名的悲剧人物。《史记》专传见[[109_李将军列传|109 李将军列传]]。
