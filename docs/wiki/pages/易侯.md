# 易侯

## 生平

「易侯」为汉代功臣侯谥号，见于惠景间侯者年表薄氏侯国的传承记录：[[薄昭]]（薄太后之弟）封侯，在位十年后，其子戎奴元年袭封，谥号「易侯」，由此得称「易侯戎奴」；传侯梁（建元二年袭封）后侯系延续。「易侯」作为谥号出现于惠景间侯者年表，是汉代薄氏外戚侯国传承的谥号记录。事见[[019_惠景间侯者年表|惠景间侯者年表]]。
