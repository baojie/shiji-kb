# 济川

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

济川，西汉诸侯王国，位于济水流域，约当今山东省西部济南以西一带。《梁孝王世家》载梁孝王死后五子分封，子明为济川王，后因射杀属官被废为庶人迁房陵，是汉景帝武帝时期削藩政策下诸侯王失国的典型案例。


济川，西汉时期诸侯王国，位于济水上游以南地区，约在今山东西部，是梁孝王刘武诸子受封之地之一。《梁孝王世家》载梁孝王死后五子分封，"子明为济川王"，济川国由此设立。

《梁孝王世家》又详载"济川王明者，梁孝王子，以桓邑侯孝景中六年为济川王。七岁，坐射杀其中尉，汉有司请诛，天子弗忍诛，废明为庶人，迁房陵，地入于汉为郡"，济川王刘明因射杀属官而被废，与济东王彭离同为梁孝王诸子中因暴行而失国的典型案例；两兄弟相继失国，折射出汉景帝、武帝时期通过诸侯王失德废国来收回封地的政治趋势。
