# 晋贾佗

## 生平

晋贾佗，春秋晋国大夫，[[晋文公]]（重耳）流亡期间五大辅臣之一。楚世家记载重耳出亡，「有赵衰、狐偃以为腹心，有晋魏焠、晋贾佗以为股肱」，说明晋贾佗与魏焠是重耳流亡集团中负责执行、冲锋陷阵的核心股肱，与赵衰、狐偃谋士的角色相辅相成。重耳归国称霸后，晋贾佗以辅臣资历成为晋国功臣之一，是晋文公霸业成功的重要参与者。事见[[040_楚世家|楚世家]]。
