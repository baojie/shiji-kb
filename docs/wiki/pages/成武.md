# 成武

## 地理位置

成武位于今山东省成武县，汉代属山阳郡，是楚汉战争期间彭越在梁地游击作战的重要活动区域，见于[[090_魏豹彭越列传|魏豹彭越列传]]，是汉军骚扰楚国后方粮道、牵制项羽的战略支撑点之一。

成武，汉代济阴郡县城，今山东成武一带，地处鲁西南平原，是秦末楚汉之际豫东战场的重要地名。《秦楚之际月表》记刘邦军"攻破东郡尉及王离军于成武南"，成武南野外是汉军早期对秦作战的战场之一；《曹相国世家》亦载曹参"从攻东郡尉军，破之成武南"，两处叙述互相印证。

成武作为鲁西南水陆交通节点，在楚汉战争初期是各路势力争夺豫东通道的重要关节。《樊郦滕灌列传》亦涉及这一地区的军事行动，揭示出成武周边在秦末大乱时期是群雄起兵北上南下的必经之地，是理解楚汉战争早期战场地理的具体参照坐标。
