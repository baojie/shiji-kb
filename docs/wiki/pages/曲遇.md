# 曲遇

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

曲遇，古地名，位于今河南省中牟县北部一带，属汉代河南郡。战国时期曲遇是韩国的重要城邑，曾是韩国与魏国、秦国交战的战场。《史记·韩世家》及相关诸侯年表中，曲遇作为韩国边境要地出现，是中原争霸地理叙事的重要节点。

曲遇，古地名，位于今河南中牟附近，是楚汉战争初期刘邦军西进攻秦途中的重要战场。《高祖本纪》载刘邦"西与秦将杨熊战白马，又战曲遇东，大破之"，曲遇之战是刘邦军在进入关中前击败秦军主力的关键一役，连续在白马和曲遇两处重创杨熊部队。

曲遇之战后，杨熊残部溃散，刘邦军顺势西进，打通了入关的道路。《樊郦滕灌列传》与《曹相国世家》均记录了诸将参与这一战役的具体行动，曲遇因此成为汉军西进路线上的重要坐标，也是刘邦先于项羽入关的战略进程中不可缺少的一环。
