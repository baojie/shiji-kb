# 夷伯喜

出现于《史记》**1** 处。

## 生平

夷伯喜，春秋晋国大夫，晋国夷姓贵族中的一代传承人。《史记·晋世家》及年表中记载晋国大夫世系，夷伯喜为其中一节，具体事迹史书记载简略，见于晋国贵族家族谱系记录。事见[[039_晋世家|晋世家]]。
