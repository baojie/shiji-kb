# 奉光

出现于《史记》**1** 处。

## 生平

奉光，西汉宣帝时期皇后王氏之父，以外戚身份封侯。宣帝元配王皇后（王氏）因家世低微而被霍光所废（一说病逝），奉光因女为皇后而封侯，事见汉书及史记相关外戚记录，为宣帝时期外戚政治的参与者。事见[[049_外戚世家|外戚世家]]。
