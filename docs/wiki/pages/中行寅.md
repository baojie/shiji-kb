# 中行寅

出现于《史记》**1** 处。

## 生平

中行寅，即**荀寅**，晋国六卿之一，中行氏宗主。春秋末年，[[赵简子|赵鞅]]欲杀[[邯郸大夫]]午，邯郸大夫午联合中行寅与[[范吉射]]共同攻打赵鞅，赵鞅退保晋阳。[[晋定公]]受三卿请求围攻晋阳。然而[[荀栎]]、[[韩不信]]、[[魏侈]]三卿与范氏、中行氏素有积怨，转而合兵攻击范氏与中行氏。中行寅与范吉射势孤，最终兵败出奔，中行氏与范氏被逐出晋国，六卿格局随之向三家（韩、赵、魏）集中演进，为后世"三家分晋"奠基。
