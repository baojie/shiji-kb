# 史记边疆民族列表

《史记》以"四夷"（东夷、西戎、南蛮、北狄）概括华夏以外的民族，并专门撰写多篇列传加以记述。司马迁将天下视为包容四夷的整体，创立了中国史学记述少数民族的传统。本列表收录135条民族/部落条目。
