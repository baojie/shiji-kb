# 西陵

西陵，《史记》中出现两处不同地名：一为上古传说中嫘祖的故乡，是黄帝正妃的出身之地；二为战国楚国城邑，约在今湖北宜昌附近，是秦昭王时期秦将白起攻楚的战果之一。同名两地跨越神话与历史两个叙事层次。

《五帝本纪》载"黄帝居轩辕之丘，而娶於西陵之女，是为嫘祖"，西陵作为嫘祖故里成为黄帝传说的重要地理背景。《六国年表》记"秦拔冲、西陵"，楚国西陵城邑被秦军攻克；《高祖本纪》亦载楚汉之际"王陵降西陵"，西陵延续作为楚地军事要点出现于楚汉战争叙事之中。
