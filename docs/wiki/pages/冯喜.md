# 冯喜

出现于《史记》**1** 处。

## 生平

冯喜，战国秦国张仪的舍人（私臣），忠心辅助张仪。齐国兴师伐梁时，张仪遣冯喜赴楚，借楚使名义转赴齐国，以外交口才说服齐王罢兵。冯喜为张仪的重要助手，以纵横之术为主，协助张仪推行连横策略。事见[[070_张仪列传|张仪列传]]。
