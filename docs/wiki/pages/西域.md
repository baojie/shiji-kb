# 西域

西域，汉代对玉门关以西广大地域的总称，泛指今新疆及中亚一带，是丝绸之路的核心走廊。汉武帝遣张骞凿空西域，以外交与军事双管齐下打通东西通道，使西域成为汉帝国地缘战略的最西前沿。

《卫将军骠骑列传》载霍去病"攻匈奴西域王浑邪"，斩获首虏、降三十二国之王，天子以千七百户益封骠骑将军，西域由此成为汉匈战争中汉军最辉煌战果的地理舞台。《司马相如列传》记汉告巴蜀太守时称"康居西域，重译请朝，稽首来享"，以西域诸国主动朝贡彰显汉武帝时代天下归附的宏大气象。《三王世家》亦将西域扩张功业纳入封建诸王的政治背景叙述，三书共同构建了《史记》中西域作为汉代国力象征的多维历史形象。
