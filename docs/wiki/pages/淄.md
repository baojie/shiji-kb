# 淄

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

淄，即淄水，发源于山东中部山地，流经今淄博、博兴一带入渤海，是古齐国的核心水系。《禹贡》记"濰、淄其道"，是青州地区的主要疏浚水道；《史记》多处涉及淄水流域的齐国历史，淄博得名即源于淄水与博山的结合。
