# 燕齐

燕齐，《史记》中频繁并称的地域联合概念，指战国时期燕国与齐国沿渤海一带的区域，是先秦方仙道和方士文化的核心发源地。《封禅书》指出驺衍的阴阳五德学说虽显于诸侯，"而燕齐海上之方士传其术不能通"，却引发了"怪迂阿谀苟合之徒"的兴起，以燕齐为方士泛滥的地理根源。

《封禅书》又载汉武帝时"海上燕齐怪迂之方士多更来言神事"，燕齐方士持续向汉廷进献神仙方术。《平准书》则从经济角度记录汉武帝朝"彭吴贾灭朝鲜，置沧海之郡，则燕齐之间靡然发动"，揭示对外扩张对燕齐地区社会经济的动荡影响。燕齐因此成为《史记》中方士文化与汉代边疆经营两大叙事脉络交汇的地域符号。
