# 祝

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

祝，古地名，位于今江苏省赣榆一带，属汉代东海郡范围。《史记》在齐鲁莒国相关叙述中涉及此地区，是山东半岛南端进入苏北平原的历史地理节点。

祝，西周初年武王克商后封赐的古国之一，是帝尧后裔的封邑，地望约在今山东省诸城或江苏北部一带。祝国是武王封建上古圣王后裔、以彰显新王朝德政的重要安排之一，与蓟（黄帝后裔）、陈（舜后裔）、杞（禹后裔）并列，共同构成西周初年"存三恪"礼制的历史见证。

《周本纪》与《乐书》均载武王克商后"封帝尧之後於祝"，两处互证。《周本纪》在叙述武王伐纣成功、班师论功行赏时，将封祝列入封赐远古圣王后裔的系列举措；《乐书》则引孔子讲解《大武》之乐时，以武王"未及下车，而封黄帝之後於蓟，封帝尧之後於祝，封帝舜之後於陈"为例，说明武王武功之后紧随文教之治，兼顾德政与礼制。祝国因封赐于帝尧之后而成为西周礼制中追慕先王、彰显仁德的象征符号，在《史记》的周初叙事中具有历史礼制的标志意义。
