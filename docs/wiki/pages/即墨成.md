# 即墨成

出现于《史记》**1** 处。

## 生平

即墨成，西汉武帝时经学家，齐地人，以《易》学在朝廷传承自田何→王同→杨何一脉，后出任城阳国相（城阳相），以易学知名于当世，为汉代《易》学官方化推广中的地方大员代表之一。事见[[121_儒林列传|儒林列传]]。
