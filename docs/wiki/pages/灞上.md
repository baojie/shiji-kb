# 灞上

灞上，关中东郊灞水西岸的高地，位于咸阳东南方向进入关中的要道上，是秦汉之际多次重大军事与政治事件的发生地。灞上因地势高于灞水岸边而得名，扼守入关要道，历来为军事驻屯要地。

《高祖本纪》载刘邦先入关中，驻军灞上，召咸阳父老约法三章，以此宣示汉军纪律，取得秦地民心。《留侯世家》多次提及灞上，将其作为汉军经营关中、布局楚汉争霸的战略基地之一。《白起王翦列传》则记王翦出关作战的路线经过灞上附近，灞上由此跨越秦统一战争与楚汉战争两个历史阶段，持续作为关中东出的军事地标。
