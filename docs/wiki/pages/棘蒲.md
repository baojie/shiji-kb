# 棘蒲

棘蒲，战国时期赵国城邑，约在今河北魏县一带，是汉初封侯食邑的来源地之一。《赵世家》中棘蒲出现于赵国历史叙事，《傅靳蒯成列传》记汉初诸将征战路线中涉及棘蒲周边地区，说明棘蒲在楚汉战争时期也是重要的军事过境地点。

《齐悼惠王世家》载文帝时济北王刘兴居叛乱，汉"使棘蒲侯柴将军击破虏济北"，棘蒲侯柴将军奉命平叛，以棘蒲侯封号与战功同时出现于叙事，揭示了棘蒲作为汉初功臣封国地名的历史背景；棘蒲因此在《史记》中兼具战国赵地城邑与汉初封侯食邑的双重身份。
