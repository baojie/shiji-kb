# 宁秦

宁秦，魏国城邑阴晋被割让给秦国后的更名，位于今陕西华阴一带，是秦魏外交与军事博弈的产物。宁秦之名见证了战国中期魏国以城邑换和、秦国以改名宣示主权的历史惯例。

《秦本纪》载"魏纳阴晋，阴晋更名宁秦"，简洁记录了魏国以阴晋为礼换取和平的历史事件。《六国年表》从魏国角度记载"魏以阴晋为和，命曰宁秦"，与《秦本纪》互证，揭示了阴晋/宁秦的名称变迁背后魏秦力量消长的历史逻辑，宁秦由此成为战国中期秦国崛起、魏国衰落这一历史转折在地理上的具体体现。
