# 洙

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

洙，古地名/古水名，位于今河南省汝南县一带，属汉代汝南郡。洙水是汝水的支流，是豫南水系的组成部分。《史记》相关叙事中洙水作为汝颍地区的地理背景出现，是楚汉战争时期南方战场的地理坐标之一。
