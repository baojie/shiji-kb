# 铚

铚，古代淮北城邑，今安徽宿州西南一带，地处泗水流域，是先秦至秦汉之际楚、魏势力反复争夺的战略要地。铚扼守淮北水陆交通，为泗上城邑群的组成部分，春秋战国时属楚国东北边境，汉初属沛郡。

《陈涉世家》多次提及铚地人物，"铚人伍徐将兵居许"，铚出身者在陈涉起事后即积极参与，成为秦末大泽乡起义向北扩散的缩影。《春申君列传》中，楚臣向秦王分析合纵形势，以铚与留、方与、湖陵、砀、萧、相并列，指出魏若趁楚秦交战之机出兵，泗上诸城必危——铚的地理位置揭示了战国末年淮北城邑在大国博弈中的棋子角色。
