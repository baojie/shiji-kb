# 鲁大夫孟釐子

出现于《史记》**1** 处。

## 生平

鲁大夫孟釐子，春秋鲁国大夫，孟孙氏族长，临死前嘱托儿子向孔子学礼，是孔子弟子孟懿子之父。事见[[047_孔子世家|孔子世家]]。
