# 妇好

**妇好**，商代武丁王的配偶，中国历史上最早有文字记载的女性军事统帅。甲骨文中有大量关于妇好的记载，包括她主持祭祀、率军征伐、生育等事迹。

《史记·殷本纪》记述[[武丁]]"访于说"，以傅说为相，国势中兴，称"武丁中兴"。妇好在此期间亦亲率军队出征，讨伐鬼方、羌方、土方等边疆部族，是商朝军功最盛时期的重要参与者。

她是中国最早有文字记载（甲骨文）的女性军事统帅，其墓葬（妇好墓）于1976年在殷墟发掘，出土大量青铜器与玉器。
