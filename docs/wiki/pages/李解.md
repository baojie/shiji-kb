# 李解

## 生平

李解，西汉人物，老子李氏后裔（李假之子）。李假在汉文帝时出仕，其子李解任胶西王[[刘卬]]太傅，因此定居于齐地。李解是《史记·老子韩非列传》中追溯老子后裔李氏家族传承至汉代的最末一代记录人物，以诸侯王太傅之职出现于汉初政治体系中。事见[[063_老子韩非列传|老子韩非列传]]。
