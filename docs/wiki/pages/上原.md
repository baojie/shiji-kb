# 上原

上原，地名，在《史记》叙事中兼具地理方位与志愿表达的双重用法。《张仪列传》中"臣之上原"一语，后世或解作"我的最大心愿"，表达张仪甘愿就死以为秦国换取黔中之地的决绝之志。

《张仪列传》载，秦欲获楚黔中之地，楚怀王扣押张仪以相要挟，张仪自请出使楚国，言"假令诛臣而为秦得黔中之地，臣之上原"，以甘愿赴死换取秦得黔中之地来表明对秦国的忠诚；此语揭示了这位纵横家以命博局、置生死于度外的政治胆识。《赵世家》亦在相关语境中提及上原，两书共同展现了上原在战国纵横叙事中的特殊语义场域，使这一地名承载了战国士人"功成而身殉"的历史精神意涵。
