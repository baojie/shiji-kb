# 勃

勃，即勃海，今渤海湾一带，是《史记》中华北平原的近海水域地理符号。勃海作为黄河入海的终端水域，在先秦至汉代的地理认知中是东方大海的西部组成部分，也是北方沿海经济、军事活动的地理背景。

《天官书》在星象地理对应体系中涉及勃海方位，以勃海为燕齐沿海区域的地理坐标。《高祖功臣侯者年表》等汉代文献亦屡见以勃海郡为参照的地理叙述，勃海郡是汉代在今河北沧州一带设置的行政区划，取名于渤海湾，是汉帝国北方海岸治理的地理基础。
