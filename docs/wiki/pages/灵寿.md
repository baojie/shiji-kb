# 灵寿

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·冀州（页26-27）（见右栏）。

灵寿，战国时期中山国旧地城邑，约在今河北灵寿一带，是魏将乐羊灭中山后的封地，也是赵武灵王灭中山后的重要建设据点。《乐毅列传》载"魏文侯封乐羊以灵寿，乐羊死，葬於灵寿，其後子孙因家焉"，灵寿从乐羊封地演变为乐氏世居之所，乐毅即是其后裔，揭示了灵寿与战国名将乐家的历史渊源。

《赵世家》载赵武灵王灭中山后"起灵寿，北地方从，代道大通"，在灵寿兴建城邑，打通与北方代地的交通，是赵国北部整合战略的重要一步；灵寿在中山灭国后成为赵国北方整合中山旧地的核心据点，在《史记》中承载着从魏封地到赵国北疆建设节点的历史演变。
