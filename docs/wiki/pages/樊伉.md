# 樊伉

**樊伉**，西汉开国功臣[[樊哙]]之嫡子，母为[[吕后]]之妹[[吕须]]。

高后在位期间，樊伉因母族吕氏专权而牵连其中；吕后崩（前180年），功臣周勃、陈平诛灭吕氏，樊伉与其母吕须均被诛杀，舞阳侯国中绝。汉文帝即位后，另封樊哙庶子市人为荒侯，以续樊哙之祀。
