# 单伯

出现于《史记》**1** 处。

## 生平

单伯，春秋郑国大夫，郑厉公出奔后所杀之臣。郑厉公被出奔至栎地（郑国边邑）后，郑厉公依托当地栎人的支持，杀大夫单伯，遂据栎以为据点，诸侯虽曾伐郑，未能攻克，宋国给予郑厉公兵力，使之在栎地得以自守，长达数年。单伯之死是郑国内乱时期权力争夺的牺牲品。事见[[042_郑世家|郑世家]]。
