# 韩广立为燕王

> 燕故贵人豪杰劝韩广立为燕王。韩广以母在赵为虑，燕人以赵方忧秦楚不能禁为说。韩广乃自立为燕王，数月后赵归其母及家属。

**主要人物**：韩广
