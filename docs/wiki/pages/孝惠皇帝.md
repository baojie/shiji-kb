# 孝惠皇帝

出现于《史记》**1** 处。

## 生平

孝惠皇帝，即汉惠帝刘盈，西汉第二位皇帝，汉高祖刘邦之子，吕后所生。惠帝性仁弱，在位时实权由吕后掌握。惠帝见吕后残害戚夫人（人彘），深受刺激，此后不理政务，郁郁而终，在位七年（前195年—前188年）。事见[[009_吕太后本纪|吕太后本纪]]。
