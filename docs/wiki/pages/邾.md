# 邾

邾，楚汉之际衡山王吴芮的都城，地望在今湖北省黄冈市黄州区一带，秦汉时设邾县，属豫章郡辖境。邾之名源自古代邾国，然此邾县与春秋鲁境之邾国（今山东邹城一带）地名相同而地望迥异——前者在江淮之南，后者在中原之东，史书中需依上下文加以区分。楚汉战争末期，番君吴芮因率百越之众佐助诸侯反秦、又随从入关立功，被项羽、刘邦先后封为衡山王，以邾为都城。

《项羽本纪》与《高祖本纪》各有一处记述，内容高度一致，均为分封衡山王的记录。《项羽本纪》记："鄱君吴芮率百越佐诸侯，又从入关，故立芮为衡山王，都邾"，交代封号与都城；《高祖本纪》记："番君吴芮为衡山王，都邾"，是刘邦即皇帝位后重申各王封号的表述。两处引文将邾作为衡山王国的政治中心，揭示了楚汉分封体制下南方诸侯王国的地理格局：衡山国以邾为核心，控扼江淮南北，是汉初长江流域政治秩序的重要节点之一。
