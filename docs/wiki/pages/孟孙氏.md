# 孟孙氏

**孟孙氏**（又称**孟氏**、**仲孙氏**），[[鲁]]国[[三桓]]之一，出自鲁庄公子庆父一系。与[[叔孙氏]]、[[季孙氏]]并称"三桓"，春秋晚期三家共同执掌鲁国朝政。

"三桓"名称来源于鲁桓公（三家皆桓公后裔）。三桓专权是孔子离开鲁国、周游列国的直接原因：

> 孔子行，三桓强，公室弱，孔子遂游。——[[047_孔子世家]]

孟孙氏历代出现的主要人物包括孟懿子（仲孙何忌），孔子弟子中亦有问礼于孔子的孟懿子。
