# 余吾

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

余吾，古地名，位于今山西省屯留县南部一带，属汉代上党郡。余吾处于上党盆地东南边缘，是晋东南历史地名的组成部分。《史记》相关叙事中余吾是上党郡地名网络的节点之一，与春秋战国时期晋国、韩国的上党争夺地理背景相关。
