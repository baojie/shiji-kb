# 棘原

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·冀州（页26-27）（见右栏）。

棘原，秦末楚汉之际军事要地，位于今河北省邯郸磁县一带，《史记·项羽本纪》载'章邯军棘原，项羽军漳南，相持未战'，棘原是秦将章邯与楚军对峙的核心阵地，也是巨鹿大战前夕最关键的军事地理节点。


棘原，秦末楚汉战争时期赵国境内的军事据点，地处今河北邢台一带，是章邯率秦军南下围攻钜鹿时的重要营地。棘原之地因章邯在此筑甬道补给王离，成为钜鹿之战这一历史转折点的关键地理坐标。

《张耳陈馀列传》载，章邯军"军钜鹿南棘原，筑甬道属河，饷王离"，以棘原为秦军大营，通过甬道向包围钜鹿的王离部队输送粮草，构建起钜鹿围城的后勤保障体系。正是这条甬道成为项羽渡河北上后首先切断的目标，项羽"破章邯甬道"后，王离军粮断绝，最终在钜鹿城下全军覆没。《项羽本纪》亦记棘原战场格局，两书合观，棘原是钜鹿之战中秦军战略部署的核心地标，也是理解项羽如何以奇兵翻转战局的地理关键。
