# 卫厘侯

出现于《史记》**1** 处。

## 生平

卫厘侯，西周晚期至春秋早期卫国国君，约与晋靖侯、宋厘公同时，见于十二诸侯年表。卫厘侯处于卫国由西周礼制秩序向春秋诸侯争霸过渡的时期，其在位年数见于年表排比，具体事迹史料简略。事见[[014_十二诸侯年表|十二诸侯年表]]。
