# 茝阳

茝阳，秦代帝王陵区，位于秦国核心腹地，今陕西咸阳附近，是秦代多位君主的葬所。茝阳因昭襄王与庄襄王先后葬于此地，成为秦帝国皇权谱系中的重要地理标志，承载着秦由霸主走向帝国的关键历史时段。

《秦始皇本纪》两度提及茝阳：昭襄王"享国五十六年，葬茝阳"，庄襄王"享国三年，葬茝阳"，两代君主皆葬于此，茝阳因此成为秦国宗室陵墓的核心集聚地。《吕不韦列传》进一步记载，始皇十九年太后薨，"谥为帝太后，与庄襄王会葬茝阳"，帝太后身后归葬茝阳与庄襄王合葬，彰显茝阳在秦代礼制中的最高地位。三书合证，茝阳是秦代宗庙陵寝制度的核心地理坐标。
