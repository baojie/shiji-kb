# 州来

州来，春秋时期淮河流域城邑，今安徽凤台一带，地处淮水中游，原为楚国或蔡国控制的边境重镇，后因吴国北上而成为吴蔡之间的争夺对象。州来地理位置居中，毗邻吴国势力北界，是春秋晚期江淮地区政治动荡的缩影。

《十二诸侯年表》载，蔡国"畏楚，私召吴人，乞迁于州来，州来近吴"，蔡国为避楚国压迫主动要求迁都于此，将靠近吴国视为自保之策，折射出春秋末期小国在大国夹缝中的艰难求存。《孔子世家》中孔子游历期间的记录亦涉及州来地区局势，说明州来在春秋晚期是吴楚势力交汇的政治热点区域，牵动诸侯格局变化。
