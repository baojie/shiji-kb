# 无棣

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·冀州（页26-27）（见右栏）。

无棣，齐国北部边邑，位于今山东省无棣县一带，是周初授权齐国征伐四方时划定的北方边界。《齐太公世家》载周成王令召康公赐太公望"北至无棣"的征伐大权，无棣因此成为齐国封疆的北境标志。

无棣，齐国北部边邑，地处今山东无棣县一带，是西周初年周成王赐封太公望时所确定的齐国北方疆界。无棣以其地理方位成为齐国北境的标志性地名，"北至无棣"是周天子授权齐国征伐四方时划定的边界坐标。

《齐太公世家》载，召康公奉周成王之命，赋予太公望"东至海，西至河，南至穆陵，北至无棣，五侯九伯，实得征之"的征伐大权，以四至方位明确齐国的权力范围，无棣作为北境界标，与穆陵、东海、黄河共同构建出齐国封疆的地理框架。这一授权奠定了齐国日后霸业的地缘基础。《齐太公世家》另处记载桓公时代诸侯会盟，无棣所在的北境是齐桓公霸权辐射的北方边界。无棣因此成为《史记》齐国叙事中封建秩序与霸权地理的象征性地名。
