# 莱山

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

莱山，位于今山东省烟台市莱山区一带，是胶东半岛北部的山地地名。《史记·封禅书》载汉武帝东巡时于此地望祭，是汉武帝封禅求仙活动在山东半岛的重要地理坐标，与泰山封禅同属东方神圣地理体系的组成部分。
