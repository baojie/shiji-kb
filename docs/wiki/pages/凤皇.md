# 凤皇

出现于《史记》**1** 处。

## 生平

凤皇，此处见于《史记·田敬仲完世家》的史赞诗文，"凤皇"在此非人名，而是出现于太史公赞语中的吉祥象征语，以"田完避难，奔于大姜；始辞羁旅，终然凤皇"称颂田完在齐国的际遇，凤皇喻示田氏从困境到腾飞的历史命运。注：此页与《凤皇于蜚》页同属诗赞用语，非独立历史人物。
