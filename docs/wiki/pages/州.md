# 州

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

州（州邑），古地名，位于今河南省武陟县一带，属汉代河内郡。州地处黄河北岸，是黄河中下游的重要渡口和战略节点。《史记》中"州"作为地名出现于春秋战国诸侯纷争的叙事中，是晋国、卫国等势力在黄河沿岸活动的地理背景之一。

州，战国末期楚国城邑，地望约在今河南沁阳一带，是楚考烈王即位初年以割地媾秦的重要筹码。《楚世家》载"考烈王元年，纳州于秦以平"，楚国以献出州城换取与秦国的和解，此举标志着楚在秦压下的进一步退缩，"是时楚益弱"一句直接点明了这一外交让步的历史意义。

《六国年表》同步记录"楚考烈王元年，秦取我州，黄歇为相"，将州城的割让与春申君黄歇出任楚相并列，揭示了楚国在外部压力下内政外交同步调整的历史节点；《韩世家》亦记宣子"徙居州"，说明州城与韩早期政治也有渊源。三书互证，共同将州定格为战国末期楚国衰弱历程中的地理见证。
