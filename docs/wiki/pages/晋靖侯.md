# 晋靖侯

[[晋国]]君主，名宜臼，[[晋厉侯]]之子，在位十八年。

自唐叔至靖侯五世，史书无确切年数，靖侯之后年代始可推算。在位第十七年，[[周厉王]]迷惑暴虐，国人作乱，厉王出奔于彘，大臣行政，史称"[[共和]]"；次年（第十八年）靖侯卒，子[[晋釐侯]]司徒立。

靖侯后代中，[[晋昭侯]]时封文侯弟[[桓叔|成师]]于[[曲沃]]，靖侯庶孙[[栾宾]]相桓叔——曲沃代翼之乱的根源由此埋下。

## 生平

晋靖侯，西周晋国君主，晋厉侯之子。在位期间，曲沃桓叔开始崛起，晋国公室与旁支曲沃的矛盾逐渐萌生，成为此后「曲沃代翼」历史的远因。《史记·晋世家》对其事迹记载简略。事见[[039_晋世家|晋世家]]。

## 历代评价

**太史公评价（叙述视角）**：《史记·晋世家》中，晋靖侯以晋国早期国君的身份出现，叙述简略，以在位年数和政治事件为主。太史公对晋国早期君主的叙述整体偏简，晋靖侯在晋国历史发展序列中是一个过渡性人物。

**汉代史学定论**：汉代对晋靖侯无独立评价，其历史意义在于晋国从西周封国到春秋强国演变过程的早期阶段。晋国在晋武公统一后、晋献公开始扩张，晋靖侯所处时代是这一历史进程的前序阶段。

**后世主要争议**：晋靖侯本人无后世重大争议，其历史意义更多在于晋国整体命运的早期节点。晋国内部的"曲沃代晋"危机在晋靖侯之后数代演变为长达数十年的政治分裂，晋靖侯时代是危机积累的早期阶段。

**历史影响 / 特殊符号意义**：晋靖侯在晋国历史叙事中代表了西周分封诸侯国的典型早期形态——宗法制约束下的政治稳定与贵族分权的潜在矛盾并存，为后来"曲沃代晋"乃至三家分晋的历史发展埋下了制度性根源。
