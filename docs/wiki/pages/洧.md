# 洧

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

洧（洧川），古地名，位于今河南省尉氏县一带，属汉代颍川郡。洧地因洧水（今双洎河）而得名，是中原地区重要的水系节点。《史记》中郑国的叙事与洧水流域关系密切——《郑世家》记载郑国多次战事发生在洧水一带，是春秋郑国政治地理的核心区域之一。
