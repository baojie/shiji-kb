# 卫榅伯

出现于《史记》**2** 处。

## 生平

卫榅伯，西周时期卫国国君，[[卫嗣伯]]之子，继位为卫君，卫国上古世系康叔→康伯→考伯→嗣伯→榅伯→靖伯→贞伯→顷侯传承中的一环。卫榅伯卒后由子[[卫靖伯]]继立，史料简略，具体事迹不详。事见[[037_卫康叔世家|卫康叔世家]]。
