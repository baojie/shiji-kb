# 广汉

广汉，西汉时期设置的西南边郡，约在今四川广汉、德阳一带，是汉王朝经营巴蜀、开发西南的行政据点。汉武帝开西南夷、设郡县后，广汉郡与蜀郡、犍为郡构成汉代经营西南的三大核心郡治，是汉廷将巴蜀纳入中央行政体系的重要成果。

《西南夷列传》记载汉武帝时期大规模经营西南夷的背景，广汉郡正是在这一进程中设立；《建元以来侯者年表》亦以广汉为侯国地名，显示广汉在汉武帝时期西南行政版图中的具体位置。广汉因此成为汉代开拓西南、将边疆融入中原行政体系这一历史进程的地理见证。
