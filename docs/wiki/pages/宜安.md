# 宜安

宜安，战国末期赵国城邑，今河北石家庄藁城一带，是赵国抵御秦国东进的重要战场，因[[李牧]]在此大破秦军而载入史册。秦王政时期，秦军频繁向赵国核心腹地进逼，宜安成为秦赵决战的关键节点。

《廉颇蔺相如列传》明确记载，"赵乃以李牧为大将军，击秦军於宜安，大破秦军，走秦将桓齮"，宜安之战是赵国对秦最后一次大规模胜利，李牧因此功封武安君。《赵世家》亦记"秦攻赤丽、宜安，李牧率师与战肥下，卻之"，但次年秦再攻番吾，赵国国势已无力回天；《六国年表》则以"秦拔我宜安"记录该城最终落入秦手，宜安的得失折射出赵国由守势转向溃败的整体走向。
