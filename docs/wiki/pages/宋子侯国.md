# 宋子侯国

宋子侯国，汉代列侯封国。高祖八年（前199）十二月丁卯，[[许瘛]]以功受封，食邑五百四十户。

## 始封

**初封侯**：[[许瘛]]（惠侯）
**分封时间**：高祖八年（前199）十二月丁卯
**初封食邑**：五百四十户

**封侯原因**（年表原文）：

> 以汉三年以赵羽林将初从，击定诸侯，功比磨侯，五百四十户。
