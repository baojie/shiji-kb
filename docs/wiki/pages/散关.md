# 散关

**散关**（大散关），位于今陕西宝鸡西南，是关中通往巴蜀（今四川）的要道，为"关中四塞"之一。与[[函谷关]]（东）、[[萧关]]（北）、[[武关]]（南）合称"四塞"，共同构成关中地区的防御体系。
