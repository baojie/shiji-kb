# 南河

南河，黄河自北折向东流的南岸一段河道，是大禹治水体系中荆州贡道的终点，也是春秋时期晋国军队南渡黄河的战略渡口。南河在《史记》中兼具地理志与军事史的双重意义，是《禹贡》水道叙事与春秋晋国扩张战略的交汇地理节点。

《夏本纪》追溯《禹贡》荆州贡道时记"浮于江、沱、涔、汉，逾于雒，至于南河"，以南河为水路贡赋的最终目的地，揭示了大禹治水后天下水道体系将南方诸州物产汇聚于黄河的地理逻辑。《卫康叔世家》则记晋文公"更从南河度，救宋"，以南河渡口为晋军绕道南渡、援救宋国的军事路线，两书合观，南河在《史记》中既是上古禹贡水道体系的终端坐标，也是春秋列国军事行动的战略渡口。
