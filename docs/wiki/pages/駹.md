# 駹

駹，西南夷部落之一，约在今四川茂县一带，是汉武帝时期开通西南夷政策的目标部落之一。駹与邛、筰、厓并称，代表蜀地以西的西南夷近蜀诸部，是汉廷开拓西南边疆、纳入郡县体制的重要对象。

《司马相如列传》载相如建议汉武帝开通西南夷，称"邛、筰、厓、駹者近蜀，道亦易通，秦时尝通为郡县，至汉兴而罢"，以近蜀易通为由劝说重开。武帝从之，"因朝厓从駹，定筰存邛"，駹部归附汉廷成为西南经略的成果之一。司马相如以外交说辞使駹等西南夷部落纳入汉朝版图，是汉代西南开拓中文治先于武功的典型案例。
