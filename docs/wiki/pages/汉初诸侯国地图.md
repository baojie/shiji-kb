# 汉初诸侯国地图

楚汉战争结束后，刘邦建立汉朝。初期为笼络异姓将领，大封韩信、彭越、英布等异姓诸侯王。随后刘邦逐步翦除异姓王，改封刘姓子弟，形成"非刘氏不得王"的格局（白马之盟）。

至高祖晚年（约前195年），全国形成以长安为中心、刘姓诸侯王环绕的"众星拱月"格局。然而这种格局为后来的"七国之乱"埋下了伏笔。
