# 章将军

出现于《史记》**1** 处。

## 生平

竖阳穀，春秋楚国侍者，楚共王近侍，以进酒误事著称。鄢陵之战中，楚共王被晋将魏錡（一说养由基）一箭射中眼睛，楚军大败；楚将子反收拾余兵，欲复战，楚共王召见子反，竖阳穀趁机进酒，子反大醉不能前见。楚共王震怒，斥责子反，子反由此身死。竖阳穀是「鄢陵之战」这一春秋重要战役中，因一进酒而改变战局、导致楚将身死的边缘小人物，是历史叙事中无心之举产生重大影响的典型案例。事见[[039_晋世家|晋世家]]。
