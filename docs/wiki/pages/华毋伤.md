# 华毋伤

出现于《史记》**1** 处。

## 生平

华毋伤，秦末楚汉之际齐国将领，担任车骑将军。韩信率军攻齐，于历下大破齐军，俘虏车骑将军华毋伤及将吏四十六人，韩信随即乘胜追击，追至嬴博又破齐骑。华毋伤在历下之战中成为韩信歼灭齐军的重要战利，其被俘标志着此役的重大胜利。事见[[095_樊郦滕灌列传|樊郦滕灌列传]]。
