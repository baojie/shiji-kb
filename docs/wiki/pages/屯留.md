# 屯留

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

屯留，古地名，位于今山西省屯留县一带，属汉代上党郡。屯留处于上党盆地东南侧，是晋东南地区的历史重邑。《史记》中屯留出现于战国韩国上党叙事，是韩赵秦三国争夺上党地区的地理背景之一，也是秦统一战争进入上党的关键地点。


屯留，上党地区县城，今山西长治屯留一带，是战国末期赵、秦反复争夺的上党腹地重镇，也是晋公室被迫迁徙之地。《赵世家》载，赵肃侯"夺晋君端氏，徙处屯留"，以屯留为安置末代晋公后裔的居所，体现了战国早期赵国对晋公室的处置方式。

《秦始皇本纪》记秦攻上党时，"将军壁死，卒屯留、蒲惣反，戮其尸"，屯留守军在将领战死后发生哗变，秦军以戮尸示众镇压，揭示了秦灭六国过程中上党地区抵抗的惨烈。《张仪列传》将屯留作为战略要道的节点，张仪建议"当屯留之道"以截断魏、韩联系，显示屯留在战国纵横布局中的地缘价值。
