# 浊河

浊河，古代对黄河的别称，因黄河水色浑浊而得名，与"清济"（清澈的济水）相对，常并称"清济浊河"以形容齐国天险。《高祖本纪》载田肯谏汉高祖封子弟为齐王时，以"西有浊河之限"列举齐地的地理优势，将浊河（黄河西境）与泰山、勃海并举，论证齐地的形胜价值，揭示了黄河作为齐国西部天然屏障的战略意义。

《苏秦列传》记燕王问苏秦"吾闻齐有清济、浊河可以为固"，苏秦以"天时不与，虽有清济、浊河，恶足以为固"回应，将浊河由防御天险转化为"德治重于险固"的哲学论点；同篇中苏秦用浊河说服燕国攻齐，清济与浊河的并称成为战国策士评估齐国地利时的惯用地理概念，折射出先秦地理观念中水系与国防的密切关联。
