# 姬哙

战国时燕国君主，燕王哙。以"禅让"为名，将王位让给相国[[子之]]，此举引发燕国大乱，太子平起兵反叛，齐宣王趁机入侵，燕国几乎灭亡，姬哙本人亦在乱中身死。

此事史称"子之之乱"或"燕哙让国"，是战国时代著名的政治灾难，也是[[燕昭王]]即位后所面临的残破局面的根源。

## 生平

姬哙，战国燕国君主，燕王哙。受苏代等人蛊惑，效法尧舜禅让，将燕国君位禅让给相国子之，引发燕国大乱。齐国趁机出兵，大败燕军，燕王哙与子之皆被杀，燕国几亡。燕昭王后来继位，卧薪尝胆，励精图治，才使燕国复兴。事见[[034_燕召公世家|燕召公世家]]。

## 历代评价

**太史公评价（叙述视角）**：司马迁在《燕召公世家》与《苏秦列传》中记载了燕王哙禅让子之、引发燕国大乱的全过程。太史公的叙事将姬哙定性为一个被"仁义之名"所诱、缺乏政治判断力的君主——他在苏代等人的游说下，以模仿尧舜禅让的名义将国政交予子之，却造成燕国三年内乱、齐国趁机入侵的灾难后果。太史公以此案例说明"名义上的圣王之道"若缺乏实际政治智慧，只会带来更大的祸乱。

**汉代史学定论**：班固《汉书》对姬哙的评价是明确的批评——其禅让行为被定性为"愚蠢效仿上古、不识时务"的典型失策。汉代儒学虽然尊重禅让的道德理想，但对在现实政治中模仿禅让的行为持批判立场，认为时代不同，以禅让之名实现权力转移不过是给权臣提供篡位借口，姬哙成为"迂阔仁义误国"的警示案例。

**后世主要争议**：后世史家对姬哙的评价几乎一致负面，争议集中于他禅让的主观动机——一方认为他是真诚地相信禅让可以实现圣王之治，属于天真而非恶意；另一方则认为他在苏代等游说者的影响下被操纵，其禅让是被人利用的政治幼稚，而非出于真正的道德理想。

**历史影响 / 特殊符号意义**：燕王哙禅让子之之乱是中国历史上最著名的"模仿禅让"失败案例，此后中国政治史中，凡以"禅让"为名的权力转移，都被审视者以燕王哙案例为参照加以警惕，成为区分真实禅让道德与政治欺骗工具的历史标尺。
