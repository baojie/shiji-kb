# 番

番，春秋时期楚国城邑，地望约在今湖北随州或安徽北部，是吴楚争霸后期吴军深入楚腹地攻取的战略要地。番的失陷直接引发楚国迁都，在春秋末期吴楚争霸史上具有重要的转折意义。

《吴太伯世家》载"吴王使太子夫差伐楚，取番，楚恐而去郢徙鄀"；《楚世家》同载"吴复伐楚，取番，楚恐，去郢，北徙都鄀"；《伍子胥列传》亦称此役为"以伍子胥、孙武之谋，西破彊楚"之一环。三书互证，番的陷落迫使楚昭王放弃郢都北迁，成为吴楚争霸达到顶点的标志性事件之一。
