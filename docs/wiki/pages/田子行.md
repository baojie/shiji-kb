# 田延年

出现于《史记》**1** 处。

## 生平

田子行，战国齐国田氏族人，田常亲信。齐简公时，田常（田成子）欲作乱，其宠臣子我（监止）察觉田常意图，出逃时被田氏之徒追杀。田常见事态危急、惧被诛杀，犹豫是否出走，田子行以「需，事之贼也」（犹豫是成事的大敌）劝告田常速行动，田常遂当机立断击杀子我。田子行是田氏夺齐权力关键时刻的谋士，以一句果断警言促成田成子的政变决策。事见[[046_田敬仲完世家|田敬仲完世家]]。
