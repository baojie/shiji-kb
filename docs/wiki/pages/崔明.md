# 崔明

出现于《史记》**2** 处。

## 生平

崔明，春秋齐国贵族，崔杼之子，崔杼杀齐庄公后，崔氏宗族内部因崔成和崔明争立庙主而发生内讧，崔明最终助庆封灭崔氏，自己后来也被庆封所杀，崔氏宗族就此覆灭。事见[[032_齐太公世家|齐太公世家]]。
