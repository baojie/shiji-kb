# 吕清

[[吕清]]，西汉功臣，封**新阳侯**（胡侯），食邑千户。

汉五年以左令尹身份初从刘邦，功比堂邑侯，
高祖六年（前 201）受封。

《史记》记载见[[018_高祖功臣侯者年表|018 高祖功臣侯者年表]]。

## 生平

吕清，西汉初年功臣，以左令尹身份初从刘邦，军功比堂邑侯，高祖六年（前201年）受封新阳侯（谥胡侯），食邑千户。事见[[018_高祖功臣侯者年表|高祖功臣侯者年表]]。
