# 荀子

战国末期儒家大儒，名况，赵国人，因其学识广博被尊称为"荀卿"。荀子曾三次担任稷下学宫（齐国）祭酒（最高职位），是该学术机构最具权威的思想家之一。晚年应楚春申君之邀，出任楚国兰陵令，老死兰陵。

荀子在儒家思想史上的地位极为重要：他主张"性恶论"，认为人性本恶，礼义教化是去恶从善的必要手段，与孟子的"性善论"形成对比。他的门徒中出现了法家代表人物[[韩非]]和秦朝丞相[[李斯]]，被认为是儒家向法家转化的重要思想来源。《史记》在《孟子荀卿列传》中将其与孟子并列立传，是《史记》所记载的战国思想家中最重要的儒家人物之一。

## 生平

荀子，即荀卿，名况，战国末期儒家大师。与荀卿为同一人物，「荀子」为后世通称，「荀卿」为史记原称。主张礼义教化、性恶论、天人相分，代表儒家思想向法治方向过渡的重要人物。其学说与孟子「性善论」对立，开儒家内部重要分流。事见[[074_孟子荀卿列传|孟子荀卿列传]]。

## 历代评价

**太史公评价（叙述视角）**：司马迁在《孟子荀卿列传》中以较多篇幅叙述了荀况的游历与著述，记载其"三为祭酒"于稷下学宫，晚年仕楚、居兰陵，以及其弟子韩非、李斯的政治命运。太史公的叙事揭示了一个深刻的历史悖论：荀子作为儒家大师，其弟子李斯成为法家的最重要实践者，并最终导致秦的暴政——太史公以此暗示了儒法思想渊源的复杂纠缠。

**汉代史学定论**：汉代儒学对荀子的评价较为复杂：荀子以"性恶论"与"礼义"作为思想核心，与孟子的"性善论"形成对立；儒学主流（孟子一系）在汉代逐渐确立正统地位后，荀子被置于较为边缘的位置，其"礼"的重视获得肯定，但其性恶论与法家联系（培养了韩非、李斯）使汉儒对其评价始终存在保留。

**后世主要争议**：荀子的历史评价是中国思想史上争议最多的题目之一：其性恶论与孟子性善论的对立贯穿历代；其培养法家弟子是否说明荀学本质上偏向法家，学界争论持续两千年；宋代以后，朱熹将荀子定性为"大醇小疵"，认为他虽有贡献但根本上偏离了儒家正统；近现代学者则有为荀子"正名"的努力，认为他是儒家礼学的集大成者。

**历史影响 / 特殊符号意义**：荀子是先秦儒家最后的综合性大师，其著《荀子》三十二篇保存了丰富的先秦思想文献。他培养的韩非与李斯，一个奠定了法家思想体系，一个实施了秦帝国的法家政策——"儒家大师培育法家门生"这一悖论，是理解先秦儒法关系最重要的历史节点之一。
