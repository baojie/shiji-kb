# 新垣

新垣，战国时期魏国城邑，地处今河南省一带，与曲阳相邻，是魏昭王时代被秦国攻取的边境重镇之一。新垣的失守是战国晚期秦国蚕食三晋、魏国疆土持续萎缩这一历史进程的具体缩影。

《魏世家》详述魏昭王在位期间魏国屡遭重创的历史：伊阙之战折损二十四万大军，随后割让河东四百里，秦将白起接连攻取大小城邑六十余座，"九年，秦拔我新垣、曲阳之城"，新垣的沦陷是这一系列失地的延续。《六国年表》于同年亦载"秦拔我新垣、曲阳之城"，两书互证，共同记录了秦昭王时代秦国对魏国持续军事压迫下的一幕。新垣之失在《史记》魏国叙事中是魏国走向衰亡的历史节点，也是秦国东进战略中关东防线逐步瓦解的地理见证。
