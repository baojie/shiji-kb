# 卷

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

卷（卷邑），古地名，位于今河南省原阳县一带，属汉代河内郡。卷地处黄河南北要冲，是春秋战国时期中原争霸的军事要地之一。《史记》中卷邑出现于诸侯交战的地理叙事中，是河南腹地战略要冲的典型代表。

卷，战国至秦汉时期魏国城邑，位于今河南原阳一带，地处黄河南岸，是秦国进攻中原的重要目标之一。《张仪列传》载张仪威胁魏国时称，若魏不事秦，"秦下兵攻河外，据卷、衍、燕、酸枣，劫卫取阳晋，则赵不南，梁不北，从道绝矣"，可见卷在战国外交辞令中被视为扼守中原南北通道的战略要冲。

秦国多次攻取卷地。《秦本纪》载昭王时"客卿胡攻魏卷、蔡阳、长社，取之"；《秦始皇本纪》又载始皇二年"麃公将卒攻卷，斩首三万"，战事极为惨烈。《穰侯列传》亦记穰侯与白起破魏，"取魏之卷、蔡阳、长社"，卷地反复出现于秦魏交战记录中。汉初名将[[周勃]]"其先卷人，徙沛"，卷是周勃家族的祖籍所在，秦汉之际的历史与这一城邑深度交织。
