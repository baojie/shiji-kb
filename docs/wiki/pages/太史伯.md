# 太史伯

出现于《史记》**2** 处。

## 生平

太史伯，即史伯，西周末年周宣王、幽王时期太史，太史阳父之称（一说即太史伯阳父）。幽王二年（前780年），泾、渭、洛三川大震，太史伯阳父以阴阳之说预言周室将亡，认为山崩川竭为国灭之兆。其言不被采纳，幽王终于骊山之难，西周亡国。事见[[004_周本纪|周本纪]]。
