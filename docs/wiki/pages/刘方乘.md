# 刘方乘

出现于《史记》**1** 处。

## 生平

刘方乘，西汉[[汉景帝]]之子，景帝后元年（前143年）封清河王。景帝晚年大封诸子，刘方乘与其他皇子同时受封，开立清河国。事见[[011_孝景本纪|孝景本纪]]。
