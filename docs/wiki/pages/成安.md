# 成安

## 地理位置

成安位于今河南省一带，汉代属梁国，是汉武帝时御史大夫[[韩安国]]（韩长孺）的故里，见于[[108_韩长孺列传|韩长孺列传]]：'御史大夫韩安国者，梁成安人也'。
