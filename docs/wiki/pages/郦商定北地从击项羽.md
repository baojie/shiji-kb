# 郦商定北地从击项羽

> 汉王赐郦商爵信成君，以将军为陇西都尉。别将定北地、上郡，破雍将军焉氏等。以陇西都尉从击项籍五月，出钜野与锺离眛战，受梁相国印。

**主要人物**：郦商、项羽

## 事件经过

汉王赐郦商爵信成君，以将军为陇西都尉。别将定北地、上郡，破雍将军焉氏等。以陇西都尉从击项籍五月，出钜野与锺离眛战，受梁相国印。
