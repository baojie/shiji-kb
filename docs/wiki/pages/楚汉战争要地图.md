# 楚汉战争要地图

前206年鸿门宴后，项羽分封十八诸侯，自立西楚霸王都彭城，将刘邦封于汉中（南郑）。刘邦随即暗度陈仓，还定三秦，与项羽展开长达四年的楚汉战争。

战争的核心战场在中原：荥阳、成皋一线是双方拉锯的主战场，广武涧两岸的对峙长达数月。刘邦以汉中、关中为后方，韩信开辟北方战场，彭越扰楚后方；项羽则依托彭城，以疾风骑兵著称。最终前202年，项羽被围垓下，四面楚歌，在乌江自刎，楚汉战争以刘邦完胜告终。
