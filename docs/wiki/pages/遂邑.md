# 遂邑

遂邑，春秋时期鲁国城邑，地望约在今山东省宁阳县或汶上县一带，是春秋中期鲁庄公三次败于齐国后被迫割让给齐国的领土，因曹沫劫盟事件而进入《史记》两大篇章的叙事。

《齐太公世家》以外交叙事视角记录：鲁庄公屡败于齐，"请献遂邑以平"，以割让遂邑换取停战，桓公许之，双方约定在柯地会盟——然而在盟坛之上，鲁将曹沫执匕首劫持齐桓公，逼其答应归还侵占的鲁国土地，包括遂邑在内的失地因此得而复还；《刺客列传》将曹沫列为"刺客"篇的开篇，从义侠勇气的角度追述同一事件："鲁庄公惧，乃献遂邑之地以和"，再现了遂邑由守到失、由失复得的全过程。两传叙事互补，共同揭示遂邑在春秋鲁齐外交史与侠义叙事中的双重地位：它既是弱国屈辱外交的牺牲品，也是义士以匕首换回国土尊严的历史见证。
