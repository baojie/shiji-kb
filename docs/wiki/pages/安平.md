# 安平

安平，齐国城邑，今河北安平一带，是燕齐战争中的重要战场。[[田单列传]]记载，乐毅率燕军大破齐国后，[[田单]]率宗族退守安平，预见城防难以持久，提前命宗人"断其车轴末而傅铁笼"，以保全车辆在撤退中不被损毁。

燕军随后攻破安平，"城坏，齐人走，争涂，以轊折车败，为燕所虏，唯田单宗人以铁笼故得脱"，众人因车轴折断无法逃脱，唯独田单一族因预先准备铁笼得以东撤即墨，成为此后复国的骨干力量。安平之失是燕齐战争中齐国遭受惨败的缩影，而田单在此地的未雨绸缪，成为其日后火牛阵复国的第一个历史伏笔。
