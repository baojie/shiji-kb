# 姬燮

出现于《史记》**1** 处。

## 生平

姬燮，西周君主，周昭王之子，传为周穆王，是西周历史上著名的君主。姬燮即为周穆王姬满，周穆王在位时间颇长，驾八骏西游见西王母是著名传说，东征徐戎亦见于史书。周穆王制定《吕刑》，重视刑法，是西周刑制的重要奠基者。事见[[004_周本纪|周本纪]]。
