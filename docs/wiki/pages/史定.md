# 史定

出现于《史记》**2** 处。

## 生平

史定，西汉武帝时揭阳令（今广东揭阳），元封六年三月乙酉，率部众降汉，封侯，是为揭阳县令自定降侯。史定主动归降，促成揭阳地区纳入汉朝版图。事见[[020_建元以来侯者年表|建元以来侯者年表]]。
