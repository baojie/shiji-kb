# 京索之战败楚

> 楚军自彭城出发乘胜追北，与汉军战于荥阳南京、索之间。汉军击败楚军，楚军因此不能越过荥阳向西进攻。此战稳住了汉军的战线。

**主要人物**：汉王、项王

## 事件经过

楚军自彭城出发乘胜追北，与汉军战于荥阳南京、索之间。汉军击败楚军，楚军因此不能越过荥阳向西进攻。此战稳住了汉军的战线。
