# 聊

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·冀州（页26-27）（见右栏）。

聊，古代城邑，位于今山东省聊城一带，处西汉清河郡东部，是冀州东南缘的历史地名，见于《史记》年表记录。
