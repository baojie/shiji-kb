# 子奢

出现于《史记》**1** 处。

## 生平

子奢，即[[伍奢]]，楚国[[太子建]]之太傅，[[伍子胥]]（伍员）与[[伍尚]]之父。费无忌谗言陷害太子建，太子建被迫出奔，伍奢以太傅之身受株连被拘。楚平王欲杀伍奢，先诱其二子以救父为名召至，伍尚选择归还陪父受死，伍员则出逃吴国。伍奢与伍尚同被楚平王诛杀。《史记·太史公自序》以"维太子建遇谗，爰及子奢，伍尚既匡父，伍员奔吴"一语概括这段因谗生祸的连锁事件，将其收入[[066_伍子胥列传|伍子胥列传]]。
