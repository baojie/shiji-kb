# 周建德

出现于《史记》**1** 处。

## 生平

周建德，西汉武帝时绛侯传承人，平曲侯周坚之子。周坚死后，周建德继承平曲侯，历任太子太傅（正式职位）。元鼎五年，周建德坐酎金不善（不能如数献酎金以助祭）而获罪，侯国除绝，绛侯这一荣耀传承至此彻底中断。事见[[057_绛侯周勃世家|绛侯周勃世家]]。
