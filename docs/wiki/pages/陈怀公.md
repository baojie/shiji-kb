# 陈怀公

春秋陈国君主，在位约公元前505年至前502年。吴国强盛时，陈怀公被吴王扣押，死于吴国，未能归国。

## 生平

陈怀公，春秋末陈国君主，继陈湣公而立，谥「怀公」，名柳。陈怀公在位期间，吴楚相争，陈国夹于其间。吴王夫差强迫陈国臣服，陈怀公被迫入吴会盟。楚救陈，吴兵退，陈怀公归至陈国后不久即死，在位约二年。陈怀公是春秋末年陈国在吴楚两强夹击中艰难维持的悲剧君主，其短暂在位标志着陈国最终走向灭亡的前期过渡阶段。事见[[036_陈杞世家|陈杞世家]]、[[014_十二诸侯年表|十二诸侯年表]]。
