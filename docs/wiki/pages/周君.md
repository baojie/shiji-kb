# 周君

**周君**，战国时期对[[东周]]天子的泛称，尤见于《战国策》记载的秦借道伐韩等外交事件中。战国晚期周王室势微，周君名义上仍为天下共主，实则夹于秦、韩之间，需依靠策士（如[[史厌]]）的外交谋略自保。
