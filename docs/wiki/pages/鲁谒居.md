# 鲁谒居

西汉御史大夫[[张汤]]所宠爱的属吏（史）。李文曾与张汤有隙，鲁谒居知张汤对此不平，擅自使人上书告发李文之奸事，事下张汤处理，张汤判杀李文，内心知晓是谒居所为。

谒居病卧时，张汤亲往探病、为其揉足（"为谒居摩足"），成为史上名句，形容上官对属下的异常偏爱。后赵王发觉此事，上书告发张汤与谒居合谋。谒居病死，其弟被系，案情牵连张汤，终成张汤自尽案的重要诱因。

## 生平

鲁谒居，西汉酷吏[[张汤]]的心腹属吏，以代张汤写密报举发赵王事著称，后因此案而死。张汤主持廷尉期间，鲁谒居深受信用，为张汤出谋划策，曾以匿名书信方式举报赵王刘彭祖的过失，帮助张汤打击政敌。事发后，鲁谒居被捕，时值其卧病在床，汉武帝仍命将其逮捕入狱追审，鲁谒居在狱中病死，其弟亦因此获罪。鲁谒居是汉代酷吏政治文化中忠于上司、以密告为手段的典型幕僚形象。事见[[122_酷吏列传|酷吏列传]]。
