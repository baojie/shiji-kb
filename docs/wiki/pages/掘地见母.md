# 掘地见母

掘地见母，是春秋初期郑庄公克段于鄢后，因发誓"不及黄泉，无相见"而与助逆的母亲武姜决裂，后在颍考叔的建议下，通过字面履行誓言（掘地至泉水处）实现母子重聚的历史典故，《史记·郑世家》有记载。

## 背景：克段于鄢

郑庄公与其弟共叔段（太叔）之间长期存在矛盾：
- 母亲武姜偏爱共叔段，曾屡次要求郑武公废长立幼
- 武公去世后，庄公即位，武姜仍为共叔段谋划扩张封地
- 共叔段最终起兵反叛，庄公提前平叛，克段于鄢

克段之后，庄公怒母亲助弟谋反，立下誓言：

> "不及黄泉，无相见也！"

## 困境

这一誓言造成了庄公的政治与伦理两难：

1. **政治需要**：必须惩罚助逆的母亲，否则难以树立权威
2. **伦理压力**：春秋时代以孝治天下，不孝则失民心
3. **心理矛盾**：庄公内心仍思念母亲，"既而悔之"

誓言说得太绝对，庄公陷入"想认母又无法打破誓言"的困境。

## 颍考叔的解法

颍考叔（郑国大夫）以进献礼物为由拜见庄公，庄公赐食，颍考叔趁机引导庄公谈及"不孝"之痛。

颍考叔献策：

> "若掘地及泉，隧而相见，其谁曰不然？"

——若掘地直至泉水（黄泉），在地道中相见，谁又能说不合誓言？

**方案的巧妙**：
- **字面遵守**："黄泉"按文字解为地下泉水，掘地至泉则满足条件
- **实质解决**：在地道中母子相见，名义上未违誓言
- **给双方台阶**：庄公守了信，武姜得到儿子，天下看到孝道

"庄公大喜，遂掘地见母。"母子在地道相见，重归于好。

## 历史意义

### 政治誓言的技术性解读

"掘地见母"是中国历史上最早的"技术性解释法律/誓言"案例之一——在誓言的字面和实质之间找到平衡，既不破坏信誉，又实现目的。

这一思路在后世的外交和法律实践中屡见不鲜：严格字面解读法条以规避法律精神，或以技术性操作达成实质目的。

### 郑庄公的决策模式

从克段于鄢（放任共叔段扩张到极点再收拾）到掘地见母（发誓后又不愿真正决绝），郑庄公的政治风格体现了一种"先强硬表态、再留退路"的模式。太史公对郑庄公既有肯定（政治能力），也有保留（过于算计）。

### 颍考叔的智慧

颍考叔被誉为"纯孝"之人——在孝道与政治现实之间找到两全之法，是春秋时代谋士的典型智慧模式。孔子也曾赞扬："颍考叔，纯孝也。爱其母，施及庄公。"
