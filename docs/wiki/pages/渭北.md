# 渭北

渭北，指渭河以北的地带，位于今陕西关中平原北部，是西汉都城长安的北方屏障。渭北地势开阔，为汉军扼守长安、防御匈奴南下的重要军事阵地，棘门等营地即设于此。

《孝文本纪》与《匈奴列传》均记文帝时期匈奴大规模入侵关中的历史事件：文帝遣中尉周舍为卫将军、郎中令张武为车骑将军，"军渭北，车千乘，骑卒十万"，以重兵驻守渭北，拱卫长安。《匈奴列传》则进一步说明匈奴入代、句注边境后，汉廷"置三将军，军长安西细柳、渭北棘门、霸上以备胡"，三营呈犄角之势护卫京师。渭北棘门由此成为文帝时代边防布局的核心节点，与细柳、霸上并称，也因名将周亚夫治军有方而在史籍中留名，而渭北棘门守将治军宽松、被视为对照案例，共同构成《史记》军事叙事中严与宽的历史对比。
