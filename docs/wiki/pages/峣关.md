# 峣关

峣关，汉代关中东南门户，位于今陕西商洛蓝田一带，是从武关入关中后抵达长安的重要关卡，亦是从关中南出至楚地的要道出口。楚汉战争期间，峣关是汉军东出与固守的反复争夺之地，多支汉将部曲留守或攻取此关。

《绛侯周勃世家》记周勃随汉王定三秦、围攻各地后，"东守峣关"，将峣关防御列入其军功叙事，足见峣关作为关中东南防线的重要战略价值。《曹相国世家》亦涉及曹参部在峣关附近的战事行动。在刘邦初入关中、与子婴谈判之际，峣关是秦军与汉军对峙的前沿，后成为汉军固守关中、扼制东方来犯之敌的军事屏障，在楚汉之争的关陇攻防格局中占据重要地位。
