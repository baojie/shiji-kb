# 刘泽封琅邪王

> 太后之妹吕嬃之女嫁给营陵侯刘泽，刘泽为大将军。太后封诸吕为王后，恐其崩后刘泽为害，遂封刘泽为琅邪王以安其心。

**主要人物**：刘泽、吕嬃、太后、田生
