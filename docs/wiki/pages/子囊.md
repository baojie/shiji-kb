# 子囊

出现于《史记》**1** 处。

## 生平

子囊，春秋楚国大夫，楚庄王之子，楚国重要卿大夫之一。子囊在楚国历次对外战争中担任要职，参与楚国与晋、吴等国的争霸。子囊临死之前，遗嘱楚王必须修筑郢都城墙以保卫国家，楚王从其遗嘱，后世称赞子囊忠诚。事见[[040_楚世家|楚世家]]。
