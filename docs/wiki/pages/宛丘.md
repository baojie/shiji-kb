# 宛丘

## 地理位置

宛丘位于今河南省周口市淮阳区，是春秋[[陈国]]的都城，秦末[[陈胜]]建立张楚政权时曾在此建都（即陈县）。

宛丘，春秋陈国都城，位于今河南省周口市淮阳区。陈国为舜的后裔所封，宛丘是陈国历代君主的都城。
