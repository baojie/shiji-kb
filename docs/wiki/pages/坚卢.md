# 坚卢

出现于《史记》**1** 处。

## 生平

坚卢，汉武帝时期燕赵地区豪强盗匪，与范生等人并称，在燕赵之间聚集大量亡命之徒，最多时达数千人，自立名号，攻占城邑、夺取库兵、释放死囚，甚至绑辱地方官员，杀害两千石官员，是武帝时代社会治安恶化的典型代表之一。事见[[122_酷吏列传|酷吏列传]]。
