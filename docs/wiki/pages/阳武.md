# 阳武

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

阳武，汉初重要地名，位于今河南省原阳一带。《陈丞相世家》载陈平少时家在阳武户牖乡，出仕后随刘邦建汉，以奇谋著称。阳武是陈平这位汉初谋臣的故乡，也是楚汉战争时期魏赵故地争夺战的地理背景之一。
