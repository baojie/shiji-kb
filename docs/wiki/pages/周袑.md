# 周袑

出现于《史记》**2** 处。

## 生平

周袑，战国赵国大臣，与赵文、赵造、赵俊等一同劝谏[[赵武灵王]]不要推行胡服骑射改革，认为沿用旧法即可。赵武灵王以先王礼法各有不同，不必拘泥旧制为由，坚持推行胡服骑射，最终使赵国军事力量大为增强。事见[[043_赵世家|赵世家]]。
