# 王龁

战国秦国将领。长平之战由[[白起]]主导，其后秦军继续攻赵。秦昭襄王四十八年（前259年），王龁率军攻拔赵国皮牢，另有五大夫陵攻邯郸。陵战不善，四十九年（前258年）王龁接任代为主将，继续围攻邯郸，即"邯郸之围"。后因赵国联魏楚救援，秦军被迫撤退。

## 历代评价

**太史公评价（叙述视角）**：司马迁在《白起王翦列传》中对王龁的记载相对简短，主要叙述其在长平之战后继续主持对赵作战以及围攻邯郸的经过。太史公的叙事逻辑中，王龁是白起之后秦国东方战场的主要将领，他的存在揭示了"巨人之后的继承者"这一叙事主题：在白起的光芒之下，王龁的战绩既难以掩藏也难以超越。

**汉代史学定论**：班固《汉书》对王龁的评价延续《史记》，将其定性为秦国统一战争中期的重要将领，战功显赫但不及白起之名。在汉代对秦将的总体评价中，王龁属于"功业出色但缺乏传奇性"的第二梯队将领，其历史地位因与白起的强烈对比而相对被压低。

**后世主要争议**：后世军事史家对王龁的争议主要集中在其邯郸围攻失败的原因——是秦国战略部署失误，还是王龁个人指挥能力不足？主流观点认为邯郸之战秦军失败的主要原因是战略层面的冒进（吕不韦等人反对的声音未被采纳），王龁个人战术能力的评价仍属正面，但邯郸失败客观上损害了其军事声誉。

**历史影响 / 特殊符号意义**：王龁在秦统一六国进程中扮演了承上启下的关键角色——他衔接了白起时代的辉煌与王翦时代的终局，是秦军从"超级战神"到"稳健统帅"这一历史转变过程中的过渡性将领。研究秦国军事史时，王龁是理解秦国将领体系传承的重要人物。
