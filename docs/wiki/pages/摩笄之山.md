# 摩笄之山

摩笄之山，因赵襄子之姊以笄自刺而得名，地处今山西代县一带，是战国初期赵国吞并代国这一历史事件中最富悲剧色彩的地理符号。摩笄之山的命名本身即是一段历史叙事：地因人死，山以事名，体现了先秦史书将地名与历史记忆相互锚定的叙事传统。

《赵世家》记载，赵襄子诈杀代王吞并代地，其姊——代王夫人——"闻之，泣而呼天，摩笄自杀，代人怜之，所死地名之为摩笄之山"，寥寥数语，以女性殉义的悲剧之举赋予山岳永久的历史名称。《张仪列传》中张仪向燕昭王游说时重述此事，以"故至今有摩笄之山"收结，将这段代王被杀、王妃殉死的历史作为赵国背信弃义的证据，服务于张仪的外交论辩。两书一叙其事之始，一述其事之传，摩笄之山因此成为《史记》中女性气节与历史记忆相互交织的特殊地名。
