# 阳晋

阳晋，战国时期卫国或齐国的城邑，地望约在今山东菏泽附近，是廉颇伐齐战役的攻取目标，也是苏秦所论秦军东攻齐国必经的战略要道。阳晋因廉颇因此得封而在《史记》中声名显著。

《廉颇蔺相如列传》载"赵惠文王十六年，廉颇为赵将伐齐，大破之，取阳晋，拜为上卿，以勇气闻於诸侯"，阳晋是廉颇跻身上卿的起点性战功；《苏秦列传》则以"过卫阳晋之道，径乎亢父之险"描述秦攻齐的险阻路线，以阳晋为关键地理节点，论证秦国难以通过该道路直接进攻齐国，揭示阳晋在战国地缘战略中的通道价值。
