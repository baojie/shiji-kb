# 常頞

秦代将领，奉命略通西南夷之路，开辟"五尺道"（宽五尺的山路），并在沿途诸国设置官吏。此举是中原王朝首次系统经营西南夷地区的尝试。秦亡后，汉初放弃此道，至汉武帝时方再度经营西南夷。

## 生平

常頞，秦朝将领，秦始皇时期奉命经略西南夷（今云贵地区），开通五尺道，是秦朝将中原统治延伸至西南的先驱。事见[[116_西南夷列传|西南夷列传]]。
