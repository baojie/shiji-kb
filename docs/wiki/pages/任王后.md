# 任王后

西汉[[梁平王]]刘襄的王后，甚得平王宠爱。因强求梁孝王遗命不得赠人的罍樽，引发与[[李太后]]的"罍樽之争"。平王为其违祖训取出罍樽，又与她合谋阻止李太后向汉使申诉。

李太后死后，任王后病中未探视，丧礼亦不守，因此被朝廷追究。

## 历代评价

**太史公评价（叙述视角）**：司马迁在相关本纪或世家中对任王后（周文王之妻、太姒）的记述简约，主要通过其生育众多圣贤子嗣（武王、周公等）来彰显其地位。太史公将任王后纳入周室德政叙事的框架中，侧重于其作为贤母的历史功能，而非个人性格刻画。

**汉代史学定论**：汉儒以"三母"（太姜、太任、太姒）为标准，将任王后（太姒）列为周室德政的源头之一，认为文王之兴与太姒的贤德密不可分。班固《汉书》延续这一框架，《列女传》类文献对太姒的叙述更为详尽，强调其"教化"子女、辅佐文王的功能，将其定位为中国古代"贤后"的典范之一。

**后世主要争议**：后世讨论任王后的争议主要在于史料的薄弱性——《史记》及先秦文献对太姒的具体事迹记载极简，大量"贤母"故事系后人补充，部分内容的历史可信度存疑。宋明理学将太姒的贤德叙事发挥到极致，但清代朴学家对这些细节的可信度多有质疑，认为其形象被后人理想化建构的成分远大于史料实证的成分。

**历史影响 / 特殊符号意义**：太姒（任王后）是中国古代"贤后"叙事的早期象征之一，与太姜、太任并列的"三母"模式，成为儒家讨论女性德行与家国关系的标准框架。她的故事被纳入历代女教文本（如《列女传》），成为女性通过相夫教子影响历史走向的典型案例，在中国古代妇女教育和性别文化史上具有持久的象征意义。
