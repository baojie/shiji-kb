# 圉

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

圉，古地名，位于今河南省杞县南部一带，属汉代陈留郡。圉地在春秋时期属郑国领土，后归魏国。《史记》中圉是豫东平原古代地名的组成部分，是陈留郡内颍水流域聚落史的历史见证。
