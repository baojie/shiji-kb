# 卫步

出现于《史记》**1** 处。

## 生平

卫步，西汉武帝时人，[[卫子夫]]（皇后）的异父兄弟之一，随[[卫子夫]]得宠后冒姓卫氏。《史记》仅在将军骠骑列传的世系介绍中一笔带过，记卫子夫男弟卫步、[[卫广]]皆冒卫氏，其具体事迹史料极简略。事见[[111_卫将军骠骑列传|卫将军骠骑列传]]。
