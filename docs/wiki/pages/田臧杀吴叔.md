# 田臧杀吴叔

> 周文军已破，秦兵将至。田臧等以假王吴叔骄不知兵权，矫王令诛之。田臧以精兵西迎秦军于敖仓，战死，军破。章邯进击荥阳，李归等死。

**主要人物**：吴叔、田臧、章邯
