# 卞

## 地理位置

卞位于今山东省济宁市曲阜附近一带，战国末年楚灭鲁国，鲁顷公被迁于卞，以普通家人身份度日，鲁国国祚由此断绝，见于[[015_六国年表|六国年表]]：'楚灭鲁，顷公迁卞，为家人，绝祀'。
