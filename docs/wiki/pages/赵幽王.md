# 赵幽王

西汉赵王，名友，[[刘邦]]（高帝）之子。受封赵王，其后娶吕氏女为妻。赵幽王与吕妻不睦，吕妻数度向吕后进谗，诬称刘友不满、诅咒吕氏当政。

[[吕后]]执政期间，借此事为由，于前181年将刘友召至长安幽禁，不给饮食。刘友在幽闭中作歌悲鸣，述说自己因妻子谗言而遭囚，忠心汉朝却身陷绝境，随后被活活饿死，以"赵幽王"为谥号，有"幽"之意。此后吕后立诸吕为诸侯王填补赵地，直至吕后死后，文帝即位，方才立刘友之子[[刘遂]]为赵王，以延续刘友一脉。

## 生平

赵幽王，西汉汉高祖刘邦第八子，名刘友（另参见词条）。吕后强令其娶诸吕之女为王后，刘友不宠，吕女告发刘友欲反，吕后召至长安幽禁绝食，刘友饿死，以民礼葬之。刘友临终作歌曰「诸吕用事兮刘氏危，迫胁王侯兮强授我妃」，是吕后专权时代诸侯王悲剧的代表性案例。事见[[050_楚元王世家|楚元王世家]]。

## 历代评价

**太史公评价（叙述视角）**：司马迁在《吕太后本纪》中记载了赵幽王刘友饿死狱中的经过，并保存了他在幽禁中所作的悲歌："诸吕用事兮刘氏危，迫胁王侯兮强授我妃……"这首歌是《史记》中保存的极少数诸侯王绝命词之一，司马迁以此记录吕后专权时代宗室诸王所遭受的政治迫害，哀叹之意溢于字里行间。

**汉代史学定论**：班固《汉书》延续《史记》的叙述框架，将赵幽王之死定性为吕后乱政时代的典型悲剧，与赵共王刘恢、燕王刘建之死并列，共同构成"诸吕用事"危害宗室的历史证据。汉代官方将吕后之政视为"几危刘氏"的政治危机，赵幽王之死成为吕后暴政的标志性案例之一。

**后世主要争议**：历代史家对赵幽王的评价争议不多，其悲剧性被普遍认可，然而对吕后处置刘友的动机看法不一：一方认为吕后处置刘友，根本原因是吕后以政治借口清除不服从的宗室王，与吕妻谗言关系不大；另一方则认为吕后正是利用宗室私怨大做文章，以家庭纠纷为掩护行政治清洗之实——两种解读揭示了吕后权术的双重性质。

**历史影响 / 特殊符号意义**：赵幽王的绝命歌成为汉初诸侯王悲剧的文学符号，与赵共王"从我而欲就之""遗令我无以妾为殉"等死前之语并列，共同构成吕后专权时代宗室压迫的文本证据，被后世讨论汉初政治史时反复引用。刘友之子刘遂后立为赵王，参与七国之乱，其命运同样以悲剧收场，父子两代的遭遇构成西汉宗室政治的沉郁注脚。
