# 吕氏家族——最早外戚专权的极端案例

## 事件背景

### 历史情境

汉高祖以泗上亭长起家，发迹后娶吕雉（吕娥姁）为正妻。吕雉出身不显但性格刚毅，在刘邦[[鸿门宴]]、韩信被杀、彭越被诛等大事中屡显决断。高祖晚年宠戚夫人，几欲以戚夫人子如意代太子盈（吕后之子），此为吕氏集团与戚氏之间的生死冲突的伏笔。

吕氏兄弟吕泽、吕释之本为高祖功臣（吕泽早死，吕释之封建成侯），吕氏家族与汉室功臣集团本有合作关系。吕后临朝后，诸吕被王，外戚势力第一次实际掌握汉朝中枢。

### 关键因素

1. **正后身份+太子生母**：吕雉作为正后与太子生母的双重身份，构成了其权力的基础
2. **戚夫人危机的阴影**：废太子事件几度发生，强化了吕后对戚氏集团的仇恨
3. **功臣集团的复杂关系**：周勃、陈平、灌婴与吕氏既有合作（灭韩信、彭越）也有戒备
4. **诸吕被王的制度违规**：汉初'非刘不王'的约定被吕后打破，为'诸吕之乱'埋下伏笔

## 详细叙述

### 阶段一：正后地位的确立与太子之争

> 原文引用：汉兴，吕娥姁为高祖正后，男为太子。及晚节色衰爱弛，而戚夫人有宠，其子如意几代太子者数矣。[2.1]

[[吕后与戚夫人]]的斗争持续多年：
- 戚夫人恃宠数次欲使如意代立
- 吕后通过留侯张良招来[[商山四皓]]辅太子，使高祖知太子羽翼已成
- 高祖叹息："羽翼已成，难动矣"——放弃易储

### 阶段二：吕后执政与血洗戚氏

> 及高祖崩，吕后夷戚氏，诛赵王，而高祖后宫唯独无宠疏远者得无恙。[2.1]

高祖崩后，吕后的报复行动：
1. **诛戚夫人**：斩戚夫人手足，去眼熏耳，饮瘖药，置厕中，命曰"人彘"（见《吕后本纪》）
2. **毒杀赵王如意**：高祖留下的这位最受宠的儿子被召入长安毒杀
3. **尽诛高祖诸子**：陆续除去[[刘如意]]、刘友、刘恢等
4. **保全寡宠者**：薄姬因"希见"得出，从子之代——这是汉朝"低调外戚"模式的起点

### 阶段三：吕氏家族的扩张

> 吕后长女为宣平侯张敖妻，敖女为孝惠皇后。吕太后以重亲故，欲其生子万方，终无子，诈取後宫人子为子。及[[孝惠帝崩]]，天下初定未久，继嗣不明。於是贵外家，王诸吕以为辅，而以吕禄女为少帝后，欲连固根本牢甚，然无益也。[3.1]

吕氏家族的扩张有三个关键动作：
1. **重亲联姻**：吕后女嫁张敖，其女又嫁惠帝——构造"舅侄结亲"的多层绑定
2. **伪造皇孙**：惠帝皇后（吕后外孙女）无子，取后宫人子为子，立为少帝
3. **王诸吕**：吕产为吕王（后为相国），吕禄为赵王，吕台为吕王，吕通为燕王等——彻底违背高祖"非刘不王"之约

### 阶段四：诸吕被诛

> 高后崩，合葬长陵。禄、产等惧诛，谋作乱。大臣征之，天诱其统，卒灭吕氏。[3.2]

高后崩后迅速的反弹：
1. **诸吕自危**：吕产、吕禄知道大臣不服
2. **谋作乱**：欲先发制人
3. **大臣反应**：周勃、陈平、灌婴等联合诛诸吕
4. **灭吕氏**：诸吕全部被诛，包括吕禄女（惠帝皇后）保存于北宫

### 阶段五：迎立代王

> 唯独置孝惠皇后居北宫。迎立代王，是为孝文帝，奉汉宗庙。此岂非天邪？非天命孰能当之？[3.2]

诛诸吕后的关键决策：
- **不立少帝**：少帝被认为是"伪皇帝"（非惠帝亲生，而是取后宫人子）
- **不立惠帝其他子**：皆被视为吕氏所立
- **迎立代王刘恒**：代王是高祖的庶子（薄姬所生），母家薄氏"仁善"
- 司马迁定论："此岂非天邪？"——把诛诸吕与立代王视为天意所归

## 历史意义

### 直接影响

1. **汉朝政治的重新定位**：从"吕氏外戚化"急速回到"刘氏本位"；诸吕被诛等于宣告外戚不能称王
2. **功臣集团的胜利**：周勃、陈平、灌婴等军功派重新掌握权力；开启"[[文景之治]]"的政治基础
3. **薄氏外戚的兴起**：代王之立使薄氏成为新的外戚家族；但薄氏"仁善"，不重蹈吕氏覆辙
4. **'非刘不王'原则的再确立**：从此异姓王除特殊军功外一律不封；外戚封侯但不封王

### 长远影响

1. **吕氏成为外戚专权的反面典型**：两千年历史中凡论外戚之祸必引"吕氏"；武帝引"女不闻吕后邪"为杀钩弋之由
2. **母壮子幼即乱的政治诊断**：武帝明确提出"主少母壮"理论，源于吕氏先例的反思
3. **外戚封爵的制度化限制**：此后汉朝外戚可封侯可任职，但不可封王；违反者必被清算（田蚡几度被警告）
4. **"宫中政治"的史学地位**：司马迁以吕后临朝称制的事实，在《史记》为吕后立"本纪"——承认其实际政治地位

---

## 现代启示

### 可借鉴之处

1. **权力的正面使用 vs 过度扩张**：吕后起初辅惠帝、制度化管理朝政均有其合理性；但王诸吕即越过边界
2. **'重亲'策略的短期效应 vs 长期风险**：多重联姻看似巩固权力，实际增加了共同敌人数量
3. **权力过渡期的脆弱性**：吕后一死诸吕立即陷入危机——权力集中者必须预置接班机制

### 需警惕之处

1. **专权必然招致反弹**：吕氏在吕后一人时可掌权，吕后一死即被清算——个人权威不可传递
2. **违背既有制度契约的代价**：'非刘不王'是高祖与功臣的契约，吕后违之即为后来清算埋下伏笔
3. **报复性政治的短视性**：吕后对戚氏的惨烈报复虽满足情感，但使吕氏集团更多敌人
4. **'伪皇帝'的合法性陷阱**：取后宫人子为少帝是吕氏自以为的稳妥之举，实际只要吕后一死即无人承认

## 跨章节互见

### 吕太后本纪与外戚世家：同一个女人的两个面孔——政治家的吕后与家族中的吕娥姁

吕后的一生在吕太后本纪中是"临朝称制"的政治家叙事，在外戚世家中则是"色衰爱弛"的家族悲剧——两篇合在一起，吕后的完整形象才呈现出"帝王"与"女人"的双重性。吕太后本纪中，吕后的形象是一个冷酷而精明的政治家——她协助刘邦诛杀功臣（"吕后为人刚毅，佐高祖定天下，所诛大臣多吕后力"）、在刘邦死后秘不发丧欲尽诛诸将（因郦商的劝谏而止）、以人彘的极端手段处置戚夫人、毒杀赵王如意（[[009_吕太后本纪|009-?]]）。吕太后本纪太史公给了吕后"本纪"的体例——这意味着吕后在她的时代实际上是"皇帝"（即使不称帝）。在吕太后本纪中，吕后是一个"创业者"——她不仅要守住刘邦打下的江山，还要为吕氏家族建立基业。外戚世家中，吕后的形象完全不同——"及晚节色衰爱弛，而戚夫人有宠，其子如意几代太子者数矣"（[[049_外戚世家|049-?]]）。外戚世家从"女人"的角度写吕后——她是一个年老色衰的妻子，丈夫爱上了更年轻的女人，儿子的太子地位岌岌可危。外戚世家的核心叙事不是"吕后掌权"，而是"吕后因恐惧失去一切而变得极端"——她的恐惧（太子被废）驱动了她的残忍（杀戚夫人、毒如意）。合观两篇，吕太后本纪中的"吕后诛戚氏"是政治清算（情敌即政敌——戚夫人不仅是夺宠者，更是夺嫡者的母亲），外戚世家中的"吕后夷戚氏"是失宠妻子的报复（"色衰爱弛"的情感创伤）。吕太后本纪写吕后"做了什么"（诛功臣、王诸吕、立少帝），外戚世家写吕后"为什么这么做"（恐惧失去皇权、恐惧失去儿子）。两篇并读，读者看到的是史家笔法的精妙——在帝王的叙事（本纪）中，吕后的残暴是政治行为（可理解但不值得同情）；在家族的叙事（外戚世家）中，吕后的残暴是情感创伤的爆发（可同情但不可接受）。吕太后本纪以"高后崩"结束——她的死亡就是政治时代的结束；外戚世家以"迎立代王"结束——她的死亡就是外戚时代的终结和"薄氏仁善"时代的开始。同一个女人的死，在政治史中是"一个时代的结束"，在家族史中是"一个外戚模式的反面教训"。

### 吕太后本纪与绛侯周勃世家：吕氏的顶峰与崩塌——诸吕被王到被诛的完整链条

吕氏家族的崛起和覆灭在吕太后本纪中是"王诸吕"的过程，在绛侯周勃世家中则是"诛诸吕"的高潮——两篇合在一起，诸吕之乱的完整权力链条才得以呈现。吕太后本纪中，吕氏家族在吕后生前达到了权力的顶峰——"太后称制，议欲立诸吕为王……乃风大臣，大臣请立吕产为吕王……太后欲王吕氏，先立孝惠后宫子彊为淮阳王……乃追尊郦侯父为悼武王，欲以王诸吕为渐"（[[009_吕太后本纪|009-?]]）。吕后的策略极其高明：她先立刘氏傀儡皇族（以惠帝后宫诸子为诸侯王）、再追尊吕氏先人、最后才"风大臣"（暗示大臣们）请立吕产为王——每一步都按程序走，让大臣们无法直接反对。吕太后本纪中，吕后临死前的安排也展现了她对形势的清醒认识——"高后病甚，乃令赵王吕禄为上将军，居北军；吕王产居南军"——她把军权交给吕禄和吕产，让他们"据兵卫宫"（掌握军队保护皇宫），确保她死后吕氏的安全。绛侯周勃世家中，吕后一死，功臣集团的反弹立即开始——"高后崩……吕禄以赵王为汉上将军，吕产以吕王为汉相国，秉汉权，欲危刘氏"（[[057_绛侯周勃世家|057-?]]）。周勃在此篇中扮演了"诛诸吕"的核心角色——他先骗吕禄交出北军印符（"君不如归印，以兵属太尉"），然后"太尉将之入军门，行令军中曰：'为吕氏右袒，为刘氏左袒。'军中皆左袒为刘氏"——这是中国历史上最著名的"站队"时刻。随后周勃派兵"逐吕产，杀之郎中府吏厕中"，"捕斩吕禄"，"笞杀吕媭"，"分部悉捕诸吕男女，无少长皆斩之"。绛侯周勃世家提供的是吕太后本纪中没有的细节：诛诸吕的具体军事操作。合观两篇，吕太后本纪中的"王诸吕"和绛侯周勃世家中的"诛诸吕"构成了一个完整的"升起—坠落"叙事弧线：吕后生前把所有权力都集中在吕氏手中（吕产为相国、吕禄为上将军），但她一死，这些权力在几天之内就被周勃、陈平等人夺走了。吕太后本纪对"诸吕被诛"的叙述较为简略（"禄、产等惧诛，谋作乱，大臣征之……卒灭吕氏"），绛侯周勃世家则详细记录了"诛诸吕"的每一步——从骗符到左袒到捕斩。两篇叠在一起，"诸吕之乱"的真相才浮出水面：所谓"诸吕之乱"实际上是"大臣之叛"——诸吕未必真的"作乱"（史书上说他们"惧诛，谋作乱"——恐惧之下想要自卫），但周勃、陈平等人以"诛诸吕"为名发动政变，把吕氏家族连根拔起。两篇并读，读者看到的是汉初政治中"权力的脆弱性"——吕氏家族的全部权力建立在一个人的生命之上。吕后在时，大臣不敢动吕氏；吕后一死，诸吕在几天之内就灰飞烟灭。吕太后本纪中吕后临终的安排（吕禄居北军、吕产居南军）看似万全，实际上军队将领在政治危机中从来只会倒向胜利者一方——左袒并非士兵的真实意愿，而是大势所趋的信号。

### 吕太后本纪与高祖本纪：吕后在刘邦生前与死后的两副面孔——从贤内助到铁腕太后的蜕变

吕后的政治能力在刘邦生前和死后呈现出截然不同的面貌——在高祖本纪中她是"佐高祖定天下"的贤内助，在吕太后本纪中则是"临朝称制"的铁腕太后——两篇合在一起，吕后蜕变的全过程才清晰地展开。高祖本纪中，吕后的角色是一个"辅助者"——她为刘邦生了太子盈，在刘邦逃亡时被项羽俘虏（"楚骑追汉王，汉王急，推堕孝惠、鲁元车下，滕公常下收载之"）、在韩信被诛时出谋划策（"吕后乃与萧相国谋，诈令人从上所来，言豨已得死，列侯群臣皆贺"——协助萧何诱杀韩信）（[[008_高祖本纪|008-?]]）。高祖本纪中的吕后是"贤内助"——她为刘邦处理家务（保护太子）、处理政事（杀韩信），但她从未取代刘邦成为主角。她的政治能力被描述为"佐"（辅助）——在刘邦的叙事中，吕后只是多个功臣之一。吕太后本纪中，刘邦死后吕后立即转变为"主角"——"吕后最怨戚夫人及其子赵王，乃令永巷囚戚夫人，而召赵王"、"太后遂断戚夫人手足，去眼，熏耳，饮瘖药，使居厕中，命曰人彘"（[[009_吕太后本纪|009-?]]）。吕后本纪中的"人彘"事件是吕后从"辅助者"变为"决策者"的标志——这件事没有人征求她的意见，也没有人阻止她（惠帝看到了才吓病），这证明她已经是最高权力者了。吕后本纪中的吕后不需要"佐"任何人——她自己就是最高权威。合观两篇，高祖本纪中的吕后是一个"站在刘邦背后"的女人——她在关键场合发挥作用（杀韩信、保太子），但从不走到前台。吕太后本纪中的吕后是一个"走上台前"的女人——刘邦一死，她立刻从后台走向前台，用最极端的方式宣告自己的权力。两篇之间的空白期（刘邦晚年到去世后的过渡期）恰恰是吕后蜕变的完成阶段：戚夫人的夺宠威胁（让她认识到权力的重要性）、刘邦试图废太子（让她认识到危机）、刘邦的去世（让她从"被保护者"变成"保护者"）。两篇并读，读者看到的是吕后从一个"传统妻子"（在刘邦荫庇下）到一个"独立统治者"（在刘邦死后）的蜕变过程。高祖本纪中的吕后对刘邦是"顺从"的（从未与刘邦公开冲突），吕太后本纪中的吕后对刘邦是"继承"的（她做的事情——延续汉朝、巩固政权——都是刘邦的策略的延续）。但有一个关键的区别：刘邦要维持的是"刘氏天下"（他自己的家族），吕后要维持的是"吕氏天下"（她自己的家族）。在吕太后本纪中，吕后做的一切都是为了吕氏——她封诸吕为王、把自己的外孙女嫁给自己的儿子（惠帝）、伪造皇孙。这些行为在高祖本纪的框架中是"逆"（违背刘邦"非刘不王"的约定），但在吕太后本纪的框架中是"顺"（延续吕后的权力）。两篇并读，读者思考的核心问题是：吕后到底是为了"刘氏"还是为了"吕氏"？答案是"都是她自己"——她既不是刘氏（她姓吕），也不是外人（她是刘氏的母亲和妻子），她代表的是"吕氏家族"在汉朝权力体系中的最大化。
