# 江汉

江汉，长江与汉水的合称，泛指两江交汇的中游地区，大致涵盖今湖北省中部至武汉一带。江汉地区是楚国故土的核心地带，以两江为天堑构成楚国南方的天然屏障，也是古代中国南北交通的咽喉要地。

《礼书》引用楚国地理战略时描述，"汝颍以为险，江汉以为池"，以汝颍为楚国北境天险、以江汉为南方护城大河，将江汉比喻为楚国天然的护城河，揭示楚国凭借两江险阻称雄南方的地理逻辑。《高祖本纪》记楚汉相争时刘邦调兵遣将的相关叙事，江汉地区作为南方战略区域出现，是楚汉争霸地理格局的组成部分。两书合观，江汉在《史记》中兼具楚国立国地理基础与楚汉战争战略区域的双重历史内涵。
