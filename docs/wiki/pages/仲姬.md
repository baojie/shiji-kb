# 仲姬

出现于《史记》**3** 处。

## 生平

仲姬，春秋齐国妃嫔，[[齐灵公]]之妾，生公子牙（后即[[齐庄公]]之竞争者）。戎姬因得灵公宠幸，请立仲姬子公子牙为太子，仲姬本人却以礼法劝阻，认为废嫡立庶不合礼制，坚持不应废弃鲁女所生太子。仲姬的态度反映了当时礼法与宠幸之间的宫廷张力。事见[[032_齐太公世家|齐太公世家]]。
