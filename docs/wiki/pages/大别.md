# 大别

大别，大禹治水体系中的著名山岭，即今湖北东部的大别山，是《禹贡》九山路线的重要组成部分。大别在《史记》中以大禹整治山川、连通九州这一宏观地理工程的节点地名而出现，是华夏上古地理体系的核心山岳之一。

《夏本纪》两度涉及大别：一处在"道九山"中载"内方至于大别"，将大别纳入从荆山到衡山的南方山岳连贯体系；另一处在描述荆州贡赋路线时以大别为南方水道的山地参照。两处合观，大别在《史记》上古地理叙事中既是九山系统的南方要冲，也是《禹贡》荆州地理框架中沟通南北山川的枢纽地名，是理解大禹治水整合南方山水体系的重要地理节点。
