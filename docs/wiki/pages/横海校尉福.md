# 横海校尉福

## 生平

横海校尉福，西汉宗室将领，成阳共王之子，以横海校尉身份参加东越战役，事后以功封缭嫈侯。福曾以故海常侯之爵，因触法而失侯；后随军征讨东越，史书记「旧从军无功」，但以宗室身份获得封侯机会。此封侯带有宗室优待性质，是汉武帝对宗室子弟的安置措施之一，东越列传特别注明其宗室身份和失侯复封的经历。事见[[114_东越列传|东越列传]]。
