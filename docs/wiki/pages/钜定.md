# 钜定

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

钜定，古代湖泊，位于今山东省广饶县东北一带，属汉代千乘郡。《史记·河渠书》载汉武帝时治理黄河，河道经过钜定泽区域，是古黄河下游尾闾的重要地理标志。钜定泽随黄河改道历史而兴废，是研究黄河三角洲历史地理的重要参照。
