# 厉公无忌

出现于《史记》**1** 处。

## 生平

厉公无忌，春秋齐国国君，[[齐武公]]之子，继位为厉公，性格暴虐，不得民心。在位期间，[[齐胡公]]（被厉公的父亲武公所取代的旁系君主）之子入齐，齐人欲立胡公子，遂联手攻杀厉公，胡公子亦在此战中阵亡。随后，齐人立厉公之子[[齐献公]]为君，齐国政权得以延续。事见[[032_齐太公世家|齐太公世家]]。
