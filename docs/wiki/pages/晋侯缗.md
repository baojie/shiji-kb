# 晋侯缗

**晋侯缗**（缗），春秋时期晋国君主，[[晋哀侯]]之弟，由[[周桓王]]另立。在位二十八年后，被[[曲沃武公]]所灭，曲沃武公遂一统晋国，获周王室承认，是为晋武公。
