# 韩信征战时间线

[[韩信]]（？— 前196）从汉王二年（前206年）八月拜为大将，
至汉五年（前202年）十二月垓下会师，
七年间主导了楚汉战争的全部关键战役，
功居汉兴第一。《史记》专传见[[092_淮阴侯列传|092 淮阴侯列传]]。
