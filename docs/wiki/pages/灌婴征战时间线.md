# 灌婴征战时间线

[[灌婴]]（？—前176）以睢阳贩缯为业，秦末随[[刘邦]]起兵，以骑兵将领著称，历经楚汉之际全程，
最终封**颍阴侯**，官至**太尉**、**丞相**。
以下按《[[095_樊郦滕灌列传|樊郦滕灌列传]]》记载逐阶段列出全部战役。
