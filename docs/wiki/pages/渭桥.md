# 渭桥

渭桥，汉代横跨渭水的重要桥梁，位于长安城北渭水之上，是进入关中首都的主要通道。《孝文本纪》载，代王刘恒入京即位时"驰至渭桥，群臣拜谒称臣"，渭桥是文帝接受群臣朝拜、正式确立帝位的历史场所，是汉文帝登基典礼的地理节点，也是吕后专政结束、刘氏复辟天下的象征时刻。

《孝文本纪》同篇另有记载匈奴入侵时文帝在渭桥视师阅兵的场景，显示渭桥在战时亦作为军事调度的集结地；《孝景本纪》亦在相关事件中涉及渭桥，揭示渭桥作为关中交通枢纽在文景之治期间持续发挥的政治与军事功能。
