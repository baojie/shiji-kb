# 繁阳

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·冀州（页26-27）（见右栏）。

繁阳，战国魏国城邑，位于今河南省安阳内黄一带，处魏郡南部边境。《廉颇蔺相如列传》载廉颇晚年攻取繁阳，是其戎马生涯最后辉煌，随即被乐乘取代，愤而出走魏国。


繁阳，战国末期魏国城邑，约在今河南内黄一带，是廉颇晚年最后战功与政治失意交织的历史场所。《廉颇蔺相如列传》载"居六年，赵使廉颇伐魏之繁阳，拔之"，廉颇复出统兵攻取魏国繁阳，是其戎马生涯最后一次辉煌胜利。

《赵世家》载此役紧接着廉颇的政治悲剧："廉颇将，攻繁阳，取之。使乐乘代之，廉颇攻乐乘，乐乘走，廉颇亡入魏"，繁阳的军事胜利旋即被政治打击所取代；廉颇因拒绝被取代而出走魏国，最终客死楚地，繁阳之役因此成为廉颇传奇生涯由盛转衰的历史转折点，是《史记》"功臣晚节"叙事主题的典型地理场景。
