# 博

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·冀州（页26-27）（见右栏）。

博，古代县邑，位于今山东省泰安东北，地处泰山脚下。汉武帝封禅泰山后免除博县赋税，楚汉战争中灌婴攻克博县立下战功，《田儋列传》载田横曾走博暂避汉军。


博，古地名，今山东泰安东北，地处泰山脚下，是汉代齐地重要县邑。[[012_孝武本纪]]与[[028_封禅书]]载，汉武帝封禅泰山后，下诏免除沿途经过地区的赋税，"复博、奉高、蛇丘、历城，无出今年租税"，博县因紧邻泰山封禅圣地而享受皇恩豁免。

楚汉战争中，[[韩信]]破齐后，汉将灌婴等人参与攻略齐地，《傅靳蒯成列传》载其"残博，益食邑"，为攻克博县立下战功。[[094_田儋列传]]载齐相[[田横]]在齐地危急时"相横走博"以避汉军锋芒，博成为田横败退途中的短暂据点。博县由此在汉初既是军功赏赐之地，又是楚汉争夺齐地的历史见证。
