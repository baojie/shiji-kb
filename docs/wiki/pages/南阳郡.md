# 南阳郡

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

南阳郡，秦汉行政区划，治所在宛（今河南省南阳市），地处汉水上游盆地，是中原南下荆楚的战略要地。《史记》中南阳郡在楚汉战争（刘邦攻宛城）、南阳守将吕齮降汉等叙事中反复出现，是刘邦西进关中路线的关键节点。南阳郡治宛城也是东汉光武帝刘秀的起兵之地，有"南都"之称。


南阳郡，今河南南阳盆地，是秦国兼并韩地后新设的重要郡，也是楚汉战争期间刘邦南路入关策略的关键通道。南阳盆地地处伏牛山与桐柏山之间，扼守南北交通要道，战略位置重要。

《秦本纪》载秦昭王"三十五年……初置南阳郡"，南阳郡的设置是秦蚕食三晋过程中将韩国南部纳入郡县版图的重要里程碑。《高祖本纪》记刘邦南路进军时"略南阳郡，南阳守齮走，保城守宛"，刘邦攻略南阳郡时守将拒城不降，后经说客劝降，刘邦终于打通了经南阳入关中的路线，南阳郡成为汉军先入关中的重要前奏战场。
