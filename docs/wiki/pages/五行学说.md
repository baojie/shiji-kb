# 五行学说

邹衍将五行学说发展为五德终始理论： 历史上每个朝代都代表一种'德'（五行之一）， 朝代的兴衰按照五行相克的顺序循环—— 黄帝（土德）→ 夏（木德）→ 商（金德）→ 周（火德）→ 秦（水德）→ 汉（土德…

**五行学说**，中国古代哲学的核心理论，认为世界由**木、火、土、金、水**五种基本元素（行）构成，
五行相生相克，循环运转。

## 五行相生相克

- **相生**：木生火 → 火生土 → 土生金 → 金生水 → 水生木
- **相克**：木克土 → 土克水 → 水克火 → 火克金 → 金克木

## 五德终始

[[邹衍]]将五行学说发展为**五德终始**理论：
历史上每个朝代都代表一种"德"（五行之一），
朝代的兴衰按照五行相克的顺序循环——
黄帝（土德）→ 夏（木德）→ 商（金德）→ 周（火德）→ 秦（水德）→ 汉（土德）。

此理论成为中国古代政治合法性的重要依据。
