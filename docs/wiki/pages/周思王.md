# 周思王

出现于《史记》**4** 处。

## 生平

周思王，东周末期天子，东周定王之子，哀王之弟。定王崩，哀王（长子）即位三个月，周思王袭杀哀王，自立为王。周思王在位五个月，又被少弟周考王攻杀，三兄弟先后争立，东周王室内讧激烈。事见[[004_周本纪|周本纪]]。
