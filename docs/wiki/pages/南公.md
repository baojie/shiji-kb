# 南公

出现于《史记》**1** 处。

## 生平

南公，楚国神秘人物，具体身份不详，或为巫师、方士、隐者。秦灭六国后，南公有预言：楚虽三户，亡秦必楚。意谓楚国即使只剩三户，最终覆亡秦朝的必定是楚人。此预言后经范增引述，用来说服项梁起兵复楚的历史必然性，被视为楚人复仇信念的象征。事见[[007_项羽本纪|项羽本纪]]。
