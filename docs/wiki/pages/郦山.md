# 郦山

即骊山（丽山），位于今陕西省西安市临潼区，渭河以南，属秦岭北麓山地。郦山在史记中有两重含义：一是秦国早期历史的关联地点（申侯之女为大骆妻、郦山之女等早期记载），二是秦始皇帝陵的所在地。

[[秦始皇]]从即位起即开始在郦山营建陵寝，役使七十余万刑徒，历时三十余年。地宫规模空前，据《史记·秦始皇本纪》记载：以水银为江河大海，机关弩矢，以人鱼膏为灯烛，奇器珍怪充满。秦二世胡亥即位后续修完工，后项羽入关焚阿房宫、挖掘骊山陵，传说地下火烧数月不灭。今骊山兵马俑坑（1974年发现）是郦山陵区的陪葬坑，是20世纪考古最重大发现之一。

## 历代评价

**太史公评价（叙述视角）**：司马迁在《秦始皇本纪》和相关篇章中以郦山作为秦始皇陵的地理坐标，通过描述陵墓的规模与机关，呈现秦始皇对死后世界的执念与统治欲的延伸。太史公对郦山陵的叙述兼具历史记录与隐性批判：以具体数字（役使七十余万人）和机关细节（水银为江河、机弩自发）展现秦朝劳民伤财的极致，隐含对秦亡原因的叙事铺垫。

**汉代史学定论**：汉代史学将郦山陵的营建定性为秦朝穷奢极欲、耗费民力的典型证据，是贾谊《过秦论》"仁义不施"批判的具体历史载体之一。班固在《汉书》中延续这一叙述，秦始皇陵的规模被作为秦朝"大兴土木、役民无度"的历史证明，郦山陵因此成为汉代批评秦政时反复援引的符号性建筑。

**后世主要争议**：后世史家对郦山陵的争议主要集中于其具体规模与内部结构的考证——《史记》的描述究竟有多少是真实记录，多少是文学夸大？现代考古发掘（兵马俑等）印证了郦山陵的规模超乎常规，但水银江河等细节仍待考证。此外，陵墓是否应当发掘也是现代争议焦点，技术与文物保护的矛盾使郦山陵成为中国考古史上最受关注的"谜题"之一。

**历史影响 / 特殊符号意义**：秦始皇陵（郦山陵）是中国历史上规模最大的单体陵寝工程，其配套的兵马俑被誉为"世界第八大奇迹"，使郦山成为中国历史上国际知名度最高的历史地理坐标之一。秦陵的发现与研究彻底改变了世界对秦代文明的认知，也使《史记》相关叙述获得了前所未有的考古印证。
