# 卢绾御史施

出现于《史记》**1** 处。

## 生平

卢绾御史施，西汉燕王[[卢绾]]麾下御史大夫，为燕国最高监察官员之一。卢绾反汉后，汉相国周勃率军平叛，击下蓟城，俘获卢绾大将抵、丞相偃、守陉、太尉弱、御史大夫施等五名高级将领，随即屠浑都、连破燕军于上兰、沮阳。御史施以御史大夫之职被俘，标志着燕国中央政府的彻底瓦解。事见[[057_绛侯周勃世家|绛侯周勃世家]]。
