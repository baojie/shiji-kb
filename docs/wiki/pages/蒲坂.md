# 蒲坂

## 地理位置

**配图**：谭其骧《中国历史地图集》第一册，战国·韩魏（页35-36）（见右栏）。

蒲坂，古地名，即今山西省永济市蒲州镇，地处黄河东岸，与对岸的临晋（今陕西大荔）隔河相望，是关中通往河东的战略要津。蒲坂因地处黄河渡口，历代为兵家必争之地，舜都蒲坂之说亦使其在上古传说中具有重要的政治符号意义。

《淮阴侯列传》详载楚汉战争中韩信东击魏王豹的战役：魏王豹"盛兵蒲坂，塞临晋"，以重兵封锁蒲坂渡口以阻汉军；韩信则采取声东击西之策，"陈船欲度临晋"以为疑兵，实则"伏兵从夏阳以木罂鲊渡军，袭安邑"，成功绕越蒲坂防线，奇袭魏国腹地，一举俘获魏王豹，定魏为河东郡。《河渠书》则记汉武帝时代治河与漕运规划中涉及蒲坂一带地理的记述。蒲坂在《史记》中以战国至汉初的军事战略要地身份留名，韩信围点打援、弃正面强渡而绕行侧翼的战法，成为《史记》军事叙事中以奇制胜的经典案例之一。
