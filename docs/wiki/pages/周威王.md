# 周威王

出现于《史记》**1** 处。

## 生平

周威王，东周天子，与魏文侯同时代。《史记·魏世家》记载：魏文侯与韩武子、赵桓子、周威王同时代，说明战国初年周威王在位时期正是三家分晋、魏国崛起的时代，周天子日益衰微，诸侯纷争格局正式形成。事见[[044_魏世家|魏世家]]。
