# 长子

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

长子，古地名，位于今山西省长子县一带，属汉代上党郡，是上党郡的治所所在地。长子处于上党盆地西南，是晋东南地区最重要的行政中心之一。《史记》中长子出现于韩国上党相关叙事，是秦赵长平之战前上党地区政治格局的关键地理节点。
