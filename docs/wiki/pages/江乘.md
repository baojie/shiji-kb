# 江乘

## 地理位置

江乘位于今江苏省南京市栖霞区一带，汉代属丹阳郡，是[[秦始皇]]第五次东巡归途中渡长江南下之地，见于[[006_秦始皇本纪|秦始皇本纪]]：'还过吴，从江乘渡，并海上，北至琅邪'。
