# 重泉

重泉，秦国关中东部城邑，约在今陕西蒲城一带，是秦简公时期新筑的边防城邑，也是汉代关中水利工程的灌溉目标地区。《秦本纪》载秦简公六年"城重泉"，与"堑洛"并举，是秦国强化河西防线的基础设施建设之一。

《河渠书》记汉代水利规划，"临晋民原穿洛以溉重泉以东万馀顷故卤地，诚得水，可令亩十石"，以重泉为关中东部盐碱地改造的目标区域，展示了汉代水利工程对重泉地区的农业开发意图。《樊郦滕灌列传》则记"故秦骑士重泉人李必、骆甲习骑兵"，为汉王所用为骑将，重泉因此成为汉初军事人才的地理出身背景。
