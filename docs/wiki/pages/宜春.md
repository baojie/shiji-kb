# 宜春

宜春，秦代宫苑地名，也是秦二世（胡亥）的葬地，位于今陕西西安附近。宜春在《史记》中以秦亡之主的长眠之地而留有历史印记，是秦帝国走向崩溃这一历史终章的地理坐标。

《秦始皇本纪》载"二世皇帝享国三年，葬宜春"，以宜春为秦二世短暂在位、仓惶覆灭后的最终归宿，揭示了这位在位仅三年的末代皇帝悲剧性的历史结局。《司马相如列传》赋文亦涉及宜春，以汉代宫苑地名的语境使宜春在《史记》中呈现历史叠层，两书合观，宜春从秦末帝王葬地到汉代宫苑胜景，见证了秦汉之际历史格局的沧桑转换。
