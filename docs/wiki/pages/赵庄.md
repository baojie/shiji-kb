# 赵庄

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·冀州（页26-27）（见右栏）。

赵庄，古代赵国故地城邑，位于今河北省保定安平一带，处常山郡东部，是古代冀州腹地的战略要地。
