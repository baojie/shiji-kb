# 徐子

## 生平

徐子，战国时期外黄（今河南兰考附近）人，以谋略之术自称「百战百胜」。庞涓率魏军过外黄，太子申随军，徐子对太子进言：此行必有大败，应速引兵回，否则将被生俘、以威胁魏王；太子申不从。此后魏军在马陵遭孙膑伏击，庞涓死，太子申被俘，魏国大败，徐子之预言悉中。徐子是战国谋士中以先见识势著称的典型人物。事见[[044_魏世家|魏世家]]。
