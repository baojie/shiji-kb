# 商武王

出现于《史记》**1** 处。

## 生平

商武王，即商汤，商朝开国君主，商汤自称甚武，故号武王（一说商武王为商汤别称）。汤在灭夏之战前作汤誓，鼓舞士气，宣称自己甚具武德，后世因此亦以商武王称之。商汤以仁德立国，是商朝开国的核心人物。事见[[003_殷本纪|殷本纪]]。
