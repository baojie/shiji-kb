# 张当居

出现于《史记》**1** 处。

## 生平

张当居，西汉景帝至武帝时人。《史记·惠景间侯者年表》记载，中二年（前148年）四月乙巳，张当居受封为侯。在位十六年后，于元朔五年（前124年）因为太常属下程博士弟子的事不据实上报而犯罪，封国被除。张当居的爵位来源于其父（表中"子侯"），具体封号不详。
