# 羊肠

羊肠，古代地名，常指太行山中蜿蜒险峻、形如羊肠的山道或隘口，是战国时期描述山河之固的典型地理意象。《史记》中"羊肠"多出现于战略形势分析的语境，强调山险隘道的防御价值。

《孙子吴起列传》载魏武侯与吴起泛舟西河，武侯赞叹"美哉乎山河之固，此魏国之宝也"，吴起则以商君之国无德、难持山河之固反驳，羊肠坂等险道构成了魏国西北防线的天然屏障，是这一战略争论的地理背景。《赵世家》与《范睢蔡泽列传》亦在战略形势分析中提及羊肠险道，羊肠因此成为《史记》中反复援引的山地战略象征。
