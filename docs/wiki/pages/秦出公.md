# 秦出公

## 生平

秦出公，战国秦国君主，在位二年。秦简公之后，秦出公即位，年纪幼小，「出子之立，母宣太后用事，嬖人景监用事」，朝政被外戚与宠臣把持。二年后，庶长改发动政变，迎回秦灵公之太子（即后来的秦献公），诛杀秦出公及其母宣太后，秦献公就此即位。秦出公在位时间极短，以幼主之身成为秦国政治过渡期动荡的牺牲品。事见[[015_六国年表|六国年表]]。
