# 卢绾将抵

出现于《史记》**1** 处。

## 生平

卢绾将抵，西汉燕王[[卢绾]]麾下大将，担任燕军最高军事指挥官之一。卢绾反汉后，汉相国周勃率军平叛，击下蓟城，俘获卢绾大将抵、丞相偃、守陉、太尉弱、御史大夫施等五名高级将领，随即屠浑都、连破燕军于上兰和沮阳。卢绾将抵被俘，标志着燕军主力的彻底溃败。事见[[057_绛侯周勃世家|绛侯周勃世家]]。
