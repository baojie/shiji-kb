# 益州

益州，西汉时期设置的西南边郡，辖今云南大部，是汉武帝经营西南夷、降服滇国后建立的重要行政单元。《西南夷列传》载，汉军征服滇王后，"以为益州郡，赐滇王王印，复长其民"，以益州郡代替原滇国政权，赐予滇王印章以维系地方统治，是汉代处置降附蛮夷的典型模式。

《大宛列传》载汉灭越后"置益州、越巂、牂柯、沈黎、汶山郡，欲地接以前通大夏"，益州郡的设置是汉武帝借西南开辟通往大夏（今阿富汗）的战略通道的组成部分；《天官书》以益州入天文分野体系；三书共同揭示益州在汉代西南扩张战略、天文地理体系与民族政策中的多重地位。
