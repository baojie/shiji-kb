# 济上

济上，济水沿岸地带的泛称，是战国末期燕国攻齐战争的重要战场区域。《乐毅列传》载乐毅率五国联军伐齐，大军自赵入齐境，在济水沿线展开大规模攻势，"济上之军奉令击齐，齐军败，闵王出走于外"，济上之战是乐毅破齐最关键的野战胜利，一举击溃齐军主力。

乐毅在书信中追述"先王命之曰：我有积怨深怒於齐"，以济上大捷作为燕昭王数十年隐忍积怨的最终回报；《乐毅列传》中济上战役是燕国复仇战争的转折节点，此役之后燕军迅速攻占齐国七十余城，齐国几近灭亡；济上因此成为《史记》中战国末期合纵攻齐战争最具决定性意义的地理坐标。
