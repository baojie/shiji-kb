# 刘象之

出现于《史记》**2** 处。

## 生平

刘象之，西汉武帝时宗室封侯，[[江都易王]]之子，封盱台侯国（今江苏盱眙），为侯。元朔元年十二月甲辰以王子身份受封，在位约四年，元狩五年坐酎金律，封国被除。事见[[021_建元已来王子侯者年表|建元已来王子侯者年表]]。
