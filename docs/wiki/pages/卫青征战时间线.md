# 卫青征战时间线

[[卫青]]（？— 前106）从元光五年（前130年）首次出征，
至元狩四年（前119年）漠北决战，
十一年间六次出征匈奴，封大将军、大司马，
功居汉代名将之首。《史记》专传见[[111_卫将军骠骑列传|111 卫将军骠骑列传]]。
