# 重丘

重丘，战国楚国城邑，地望约在今河南沈丘或安徽境内，是战国晚期多国联合攻楚战役中楚军大败的历史现场。重丘之战是齐湣王强盛时期联合秦、韩、魏合击楚国的关键战役，楚将唐眛战死于此，楚国损失惨重。

《楚世家》载楚怀王二十八年"秦乃与齐、韩、魏共攻楚，杀楚将唐眛，取我重丘而去"；《六国年表》从齐国视角记"秦、韩、魏、齐败我将军唐眛于重丘"；《乐毅列传》以"南败楚相唐眛於重丘"概括齐湣王全盛时期的军事成就。三书互证，重丘之战标志着楚国在多国合纵压力下的重大军事挫折，也是齐湣王霸业鼎盛期的典型战例。
