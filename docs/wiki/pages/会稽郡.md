# 会稽郡

会稽郡，秦统一后所置南方郡，辖今浙江绍兴、杭州及江苏南部一带，是越国旧地纳入秦帝国郡县版图的行政结果。会稽郡在《史记》中兼具秦灭楚越的历史终点与楚汉起义爆发地的双重身份。

《秦始皇本纪》载"王翦遂定荆江南地，降越君，置会稽郡"，秦将王翦在消灭楚国的同时安抚越地，置会稽郡管辖。秦末，"项梁举兵会稽郡"，项氏在这一南方郡县起兵反秦，会稽郡由秦的地方统治符号转变为楚汉兴起的历史起点，印证了帝国边缘地带率先响应的历史规律。
