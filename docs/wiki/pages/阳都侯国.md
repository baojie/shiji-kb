# 阳都侯国

阳都侯国，汉代列侯封国。高祖六年（前201），[[丁复]]以功受封，食邑七千八百户。

## 始封

**初封侯**：[[丁复]]（敬侯）
**分封时间**：高祖六年（前201）
**初封食邑**：七千八百户

**封侯原因**（年表原文）：

> 以赵将从起邺，至霸上，为楼烦将，入汉，定三秦，别降翟王，属悼武王，杀龙且彭城，为大司马；破羽军叶，拜为将军，忠臣，侯，七千八百户。
