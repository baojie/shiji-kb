# 曲沃庄伯

曲沃君主，名鳝，[[晋桓叔]]之子，[[曲沃武公]]（晋武公）之父。曲沃代翼进程中的重要人物。

[[晋孝侯]]在位第八年，曲沃桓叔卒，庄伯继位。孝侯第十五年，庄伯弑晋孝侯于翼，晋人攻曲沃，庄伯败退。晋鄂侯卒，庄伯再度兴兵伐晋，周平王派虢公出兵，庄伯又退保曲沃。后庄伯卒，子称代立，是为[[曲沃武公]]。

## 历代评价

**太史公评价（叙述视角）**：司马迁在《晋世家》中记述曲沃庄伯为曲沃桓叔之子，继续对晋侯的攻伐，是"曲沃代翼"百年政治僭乱的关键推进者。太史公对庄伯的叙述简约，重在呈现晋国公室内乱的历史过程，不作个人道德评议，而是通过周桓王的讨伐与庄伯败绩的记录，揭示僭越者所面临的周天子权威制约。

**汉代史学定论**：汉代史家和经学家将曲沃代翼的历史视为礼崩乐坏的典型案例，认为曲沃一系的崛起是春秋乱局的缩影。庄伯作为这一过程的中间环节，汉儒评价不高，多以"僭越""不正"定性。《春秋》对晋国曲沃事件的处理（如"曲沃伯诱杀晋小子"之类的书法），奠定了后世将曲沃之乱定性为礼义崩解的框架。

**后世主要争议**：后世史家对曲沃代翼的复杂性认识不断深化：一方面，庄伯及其后继者从道义上是篡位者；另一方面，曲沃一系最终完成统一晋国的历史使命，奠定了晋文公称霸的基础。这引发了关于历史合法性与历史成就之间张力的讨论：一个以不义手段上位的集团，是否因后来的历史贡献而获得追溯的正当性？宋代史家多倾向于坚持程序正义，清代史家则更多从历史结果看问题。

**历史影响 / 特殊符号意义**：曲沃庄伯是曲沃代翼历史链条上不可缺少的一环，其持续内乱的历史行为成为春秋礼制解体叙事中的标志性案例。"曲沃"在后世几乎成为"以小宗篡大宗""以藩镇代中枢"的符号，被历代讨论分封制度弊病时援引。庄伯本人因史料简约而形象模糊，但他所参与的历史进程对中国古代封建制度的讨论留下了持久印记。
