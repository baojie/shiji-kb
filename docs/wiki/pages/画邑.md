# 画邑

画邑，战国时期齐国城邑，是田单复国战争期间的重要据点之一。《田单列传》载燕军占领齐国大部分领土后，田单以即墨为基地组织抵抗，逐步收复失地，画邑出现于田单收复齐地的军事行动叙事之中，是燕齐反复争夺的城邑之一。
