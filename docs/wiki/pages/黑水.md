# 黑水

黑水，上古大禹治水划定九州版图时的重要界河，地处西方，大致对应今青藏高原以东某条河流，是《禹贡》九州体系中标定西方地理边界的关键水系。黑水在《史记》中作为梁州与雍州的西部界河，具有古代地理区划的重要意义。

《夏本纪》三度提及黑水：梁州条目载"华阳黑水惟梁州"，以黑水与华阳山为梁州的南北界定；雍州条目载"黑水西河惟雍州"，以黑水与西河界定雍州范围；又记贡道"浮于积石，至于龙门西河，会于渭汭"，将黑水流域的物产纳入西部贡赋体系。黑水在梁、雍两州的双重定界功能，揭示了《禹贡》九州体系以黑水为重要轴线构建西部地理秩序的设计逻辑，是古代华夏世界地理观中最西方边界的河流象征。
