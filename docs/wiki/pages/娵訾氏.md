# 娵訾氏

出现于《史记》**1** 处。

## 生平

娵訾氏，上古传说人物，帝喾高辛氏第四妃，生帝挚（帝喾之子，尧之兄）。帝喾四妃：元妃姜嫄生后稷，次妃简狄生契，三妃陈丰氏生帝尧，四妃娵訾氏生帝挚。帝喾死后，挚立，四年后让位于帝尧。娵訾氏为帝喾妃嫔体系中的一员，其子帝挚短暂即位。事见[[001_五帝本纪|五帝本纪]]。
