# 子恶

**子恶**，春秋鲁国贵族，[[鲁文公]]与长妃[[哀姜]]（齐女）之嫡子，与兄弟[[视]]同为嫡出。鲁文公卒后，权臣[[襄仲]]（东门遂）受次子俀贿赂，联合[[齐惠公]]于冬十月诛杀子恶与视，拥立俀为[[鲁宣公]]，被时人斥为"杀嫡立庶"。
