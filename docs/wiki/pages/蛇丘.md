# 蛇丘

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·冀州（页26-27）（见右栏）。

蛇丘，西汉济北郡县邑，位于今山东省泰安肥城一带。汉武帝封禅泰山后免除沿途赋税，《孝武本纪》载蛇丘与博等地并列获免，是汉代封禅礼制在地方行政上的直接体现。

蛇丘，汉代县名，位于今山东省泰安市肥城市一带，地处泰山西麓，是汉武帝封禅泰山后颁布赦令、免税以示恩泽的受惠县份之一，因此在《史记》封禅叙事中留名。

《孝武本纪》与《封禅书》均载汉武帝封禅完毕，于明堂接受群臣朝贺后颁布的制诏，其中明确列举免除当年租税的县份："复博、奉高、蛇丘、历城，无出今年租税"。博、奉高为封禅礼的举行地（今泰安一带），蛇丘与历城则是泰山附近的重要县邑，同获免税之恩。这一赦令是汉武帝封禅礼成后与民同庆的政治表达，通过免税、赦天下等方式彰显皇权的仁德与天命合法性。蛇丘因地处泰山周边、被纳入封禅庆典的受惠范围而两度出现于《史记》平行记载，是汉代国家礼制与地方行政关系的具体体现。
