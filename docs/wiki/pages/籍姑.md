# 籍姑

籍姑，秦国于战国早期修筑的军事城邑，《秦本纪》与《六国年表》均记载于秦灵公十三年（约前422年）建城，是秦国在东部边境扩建防御体系、应对晋（魏）军事压力的重要工程。籍姑城址具体位置今已难以确考，约在秦晋交界的渭河流域或河西地带。

《秦本纪》记秦灵公在位期间，"十三年，城籍姑"，与同年晋城少梁相对峙，显示两国在渭水以北河西地区的激烈争夺；《六国年表》则以更宏观的视角并载同期各国动态，记秦"补庞，城籍姑"，揭示城籍姑是秦国一系列边境加固举措之一。秦灵公时代是秦国在战国初期经营河西、与魏国展开长期角力的关键阶段，城籍姑之举正是这一历史背景下秦国以城塞为战略据点、步步稳固东部防线的缩影，也是《史记》中有迹可查的秦国早期军事建设史料之一。
