# 堂邑父

出现于《史记》**3** 处。

## 生平

堂邑父，汉武帝时期[[张骞]]出使西域的随行向导，匈奴族人（胡人），善射，长期陪伴张骞历险。张骞出使西域被匈奴扣留十余年，堂邑父始终相随；张骞趁单于死后国内大乱时逃脱，堂邑父携胡妻随行归汉。汉武帝嘉奖张骞及堂邑父，拜堂邑父为奉使君。事见[[123_大宛列传|大宛列传]]。
