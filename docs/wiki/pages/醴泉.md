# 醴泉

醴泉，字义为甘美如醴（甜酒）的泉水，在中国古代文献中既指实际存在的甘泉，也是帝王盛世的祥瑞象征。《史记》中两处涉及醴泉，语境截然不同：一处出于司马相如描绘天子上林苑壮丽景观的散体赋，属于宫殿园囿中的真实或想象水景；另一处出于司马迁引述《禹本纪》所载的昆仑山传说，属于上古神话地理的仙境意象。两处用法共同揭示"醴泉"在汉代文化中横跨现实与神话的双重意涵。

《司马相如列传》收录相如的《子虚赋》/《上林赋》，其中描绘天子宫苑："醴泉涌於清室，通川过乎中庭"，以醴泉涌现于宫室之中、流贯庭院的意象，渲染天子居所与天地精华相通的神圣气象；《大宛列传》末尾"太史公曰"引《禹本纪》："昆仑其高二千五百馀里，日月所相避隐为光明也。其上有醴泉、瑶池"，将醴泉与瑶池并列为昆仑仙山的标志性地物，并以此对照实际探察的结果，委婉表达对上古神话地理记载之可信度的存疑态度。
