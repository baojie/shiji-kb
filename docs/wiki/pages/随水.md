# 随水

随水，古代水名，约指汉水以东、今湖北随州一带的水系（或为涢水古称、或为汉水支流），是楚国东北边境的地理标志之一。随水右壤（随水右岸之地）地处楚国北部山林地带，多为广川大水、崇山密林，农业产出匮乏，历史上被视为易守难攻却难以实质开发的"不食之地"。在战国后期，随水成为秦国讨论是否攻楚的战略分析中的核心地理参照之一。

《春申君列传》记录黄歇在秦昭王图谋攻楚之际，上书陈策，两度提及随水。第一处论述秦攻楚的出兵路线时说："王若不借路于仇雠之韩、魏，必攻随水右壤"；第二处紧接解释原因："随水右壤，此皆广川大水，山林谿谷，不食之地也，王虽有之，不为得地"——即随水以东的地带地势险峻，无粮可食，即使攻占也无实际利益，只有毁楚之名而无得地之实。黄歇以此地理论证，成功说服秦昭王暂缓攻楚，随水在这段外交辩辞中成为楚国防御地形的重要说辞。
