# 吴右

出现于《史记》**1** 处。

## 生平

吴右，西汉诸侯王，长沙国王，谥恭王。为长沙系王室成员，《史记·汉兴以来诸侯王年表》记其元年。长沙国在汉初为异姓王国，后改封宗室，吴右为其传承中的一代王。事见[[017_汉兴以来诸侯王年表|汉兴以来诸侯王年表]]。
