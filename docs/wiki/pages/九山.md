# 九山

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

九山，古地名，位于今山东省莒南县一带，属汉代琅邪郡（东海郡交界地带）。九山位于山东东南部低山丘陵区，是古代东海之滨地名的组成部分。《史记》中出现于泰山封禅及琅邪郡相关叙事。


九山，《禹贡》地理体系中大禹治水工程所疏导的九座代表性山脉的总称，涵盖华夏九州境内最重要的山地屏障。《夏本纪》载，大禹治水时"道九山"，逐一列出汧山至汶山在内的九座山脉，将山地疏导纳入治洪工程的系统框架之中。《五帝本纪》颂扬禹功，以"披九山，通九泽，决九河，定九州"四句话概括其业绩，九山居首。

九山并非单一地点，而是一个表示山地治理全局的数字概念，与九州、九河、九泽相互对应，共同构成《禹贡》体系的四大地理范畴。《河渠书》亦以"度九山"呼应《夏书》，将大禹的山川治理纳入汉代水利历史叙述的起源追溯。九山概念体现了上古地理思想中以"九"统摄天下的整体宇宙观。
