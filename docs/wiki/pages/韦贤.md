# 韦贤

西汉大臣，通《诗》《礼》《尚书》，鲁地大儒。任博士，后为汉昭帝师，历任光禄大夫、大鸿胪、长信少府。汉宣帝本始三年（前71年），代蔡义为丞相，封扶阳侯，食邑一千八百户。为丞相五年，以仁恩见称，但不熟习吏事，后被免相，病死于家。其子韦玄成继承封号，后为太常。

## 历代评价

**太史公评价（叙述视角）**：《史记》完成于汉武帝时代，韦贤活跃于汉昭帝、宣帝时期，司马迁无从直接记载。《史记》对韦贤无专门叙述，其历史形象主要由《汉书》建立。

**汉代史学定论**：班固《汉书》在《韦贤传》中对韦贤给予了较为正面的评价，记载他通《诗》《礼》《尚书》，为汉昭帝师傅，以儒学大家著称，封扶阳侯，位至丞相。班固尤其记录了韦贤晚年上书请求归乡的故事，以及他告诫子孙"遗子黄金满籯，不如教子一经"的名言，将其塑造为以儒学传家的清廉贤者典范。

**后世主要争议**：后世史家对韦贤的评价总体正面，争议较少。"遗子黄金满籯，不如教子一经"作为强调儒学传家价值的名言，被后世广泛引用，韦贤的名字因此与这一教育格言永久关联，这也是他在中国文化史中最重要的历史贡献。

**历史影响 / 特殊符号意义**："遗子黄金满籯，不如教子一经"成为中国古代强调知识与教育价值的最著名格言之一，历代引用于讨论家庭教育与物质传承之间的取舍。韦贤本人的历史形象因为这句话而在中国教育文化史中占有独特地位，超越了他作为汉代儒臣的具体历史角色。
