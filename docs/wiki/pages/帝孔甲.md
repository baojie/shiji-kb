# 帝孔甲

## 生平

帝孔甲，夏朝君主，帝不降之子，帝廑之后重立（跳过帝廑直系），以「好方鬼神，事淫乱」著称，夏后氏由此德衰，诸侯纷纷背叛。在位期间天降神龙一雌一雄，孔甲无力饲养，后[[刘累]]学驯龙术为孔甲效力，孔甲赐之「御龙氏」之姓；雌龙死，刘累以龙肉进献，惧罪迁去。帝孔甲的在位是夏朝走向灭亡的重要转折，此后仅经桀即被商汤所灭。事见[[002_夏本纪|夏本纪]]。
