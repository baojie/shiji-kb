# 庄青翟

汉武帝时丞相，封武强侯。司马迁评其为"娖娖廉谨，为丞相备员而已，无所能发明功名有著於当世者"，是汉武帝时期循资守职、无所建树的丞相代表人物之一。

## 生平

庄青翟以列侯身份继任为相。早年曾任御史大夫：建元二年（前139），[[窦太后]]罢黜儒臣[[赵绾]]、[[王臧]]，免相免太尉，以柏至侯[[许昌]]为丞相，庄青翟为御史大夫。

后升任丞相，与[[许昌]]、[[薛泽]]、[[赵周]]同被司马迁列为武帝时"备员"之相，以廉谨见称但乏建树。

元鼎二年（前115），庄青翟因涉及案件下狱，自杀。

## 历代评价

**太史公评价（叙述视角）**：《史记·张丞相列传》中，庄青翟附于丞相列传，以汉武帝时期丞相身份出现。太史公对庄青翟的记述极为简略，以其任丞相期间与张汤的政治博弈为核心——庄青翟在张汤案中的表现（联合三长史弹劾张汤）展示了他作为丞相的政治动作，但随后被武帝以"不忠"定罪自杀，太史公暗示这是武帝借机清除异己的政治操作。

**汉代史学定论**：汉代对庄青翟的评价偏负面——他是汉武帝时代多位"丞相自杀"命运链中的一环，其死被解读为汉武帝强化皇权、压制相权的历史进程中的一个节点。班固《汉书》在处理这一时期丞相命运时，将庄青翟之死置于武帝专权加强的叙事框架中。

**后世主要争议**：后世对庄青翟本人争议不大，主要关注他所代表的汉武帝时期"丞相高危"现象——汉武帝在位期间数位丞相以罪死或自杀告终，反映出西汉相权在武帝强势皇权下的系统性萎缩。庄青翟是这一结构性政治变化的具体案例。

**历史影响 / 特殊符号意义**：庄青翟的历史意义在于他是汉武帝时代"相权衰落"进程的人格化缩影——此后汉代逐渐以内朝（尚书台）替代外朝（丞相府）作为实际决策中心，丞相地位不断下降，庄青翟等人的命运加速了这一制度转变的历史进程。
