# 洨

## 地理位置

洨位于今安徽省宿州市灵璧县一带，汉代为洨县，见于[[112_平津侯主父列传|平津侯主父列传]]：主父偃死后无一人收尸，唯洨人孔车收葬之，天子后称孔车为长者。
