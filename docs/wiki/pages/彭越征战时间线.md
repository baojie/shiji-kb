# 彭越征战时间线

[[彭越]]（？—前196）昌邑人，以渔猎为业，秦末聚泽中亡命为盗，楚汉之际以梁地为根据地，
长期游击断楚粮道，被封**梁王**，后被[[刘邦]]以谋反罪诛杀。
以下按《[[090_魏豹彭越列传|魏豹彭越列传]]》记载逐阶段列出全部战役。
