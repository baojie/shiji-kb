# 荡阴

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

荡阴，古地名，位于今河南省汤阴县一带，因处荡水（今汤河）之北而得名，属汉代河内郡。《鲁仲连邹阳列传》涉及此地，荡阴亦是历史上多次军事行动的要道，南北往来经太行山南麓必经之地，是豫北平原与太行山区的地理节点。
