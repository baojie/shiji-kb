# 姬错

出现于《史记》**1** 处。

## 生平

姬错，即周悼王，东周君主，周景王之子，继周景王位，但在位仅三年。周悼王在位期间，周室已极度衰微，诸侯不来朝，贵族专权，东周历史进入末期阶段。事见[[004_周本纪|周本纪]]。
