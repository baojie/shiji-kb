# 功狗与功人的双轨评价体系

如何公平评价可见贡献与隐蔽贡献？汉初萧何vs曹参的排序之争揭示了组织内功劳评定的结构性难题：执行者功劳可见（攻城122座），规划者功劳隐蔽（守住关中）。高祖的解决方案是"双轨制"——精神荣誉与物质奖励分开计算。

**来源**：《史记·高祖功臣侯者年表》（卷018）、《萧相国世家》、《曹相国世家》

> "夫猎，追杀兽兔者狗也，而发踪指示兽处者人也。今诸君徒能得走兽耳，功狗也。至如萧何，发踪指示，功人也。"

---

## 核心理论

### 理论1：功狗 vs 功人

| 维度 | 功狗（执行者）| 功人（规划者）| 评价 |
|------|------------|------------|------|
| 角色 | 追杀猎物 | 发踪指示 | 规划>执行 |
| 可见度 | 高（攻城拔寨） | 低（幕后策划） | 隐蔽者贵 |
| 可替代性 | 高（"亡百数无碍"） | 低（"亡一人则危"） | 稀缺者重 |
| 价值持续性 | 一时（战役结束即止） | 万世（长期支撑） | 持久者优 |

**现代类比**：

| 功狗 | 功人 |
|------|------|
| 销售团队 | 战略部门 |
| 开发团队 | 架构师 |
| 前线执行 | 项目规划者 |

### 理论2：万世之功 vs 一时之功

**鄂君的三条论证**：

**论证1：持续补兵**
> "萧何常从关中遣军补其处，非上所诏令召，而数万众会上之乏绝者数矣。"

关键词：常（多次）、主动性（非诏令召）、数万众（规模大）

**论证2：转漕给粮**
> "夫汉与楚相守荥阳数年，军无见粮，萧何转漕关中，给食不乏。"

关键词：数年（长期）、给食不乏（持续供应）

**论证3：保全关中**
> "陛下虽数亡山东，萧何常全关中以待陛下，此万世之功也。"

**核心结论**：持续价值 > 一时功劳

> "今虽亡曹参等百数，何缺于汉？奈何欲以一旦之功而加万世之功哉！"

### 理论3：功劳三维度评价框架

| 维度 | 曹参（一时之功）| 萧何（万世之功）| 评价标准 |
|------|--------------|--------------|---------|
| 战略价值 | 战术执行 | 战略支撑 | 战略>战术 |
| 系统价值 | 局部战场 | 全局系统 | 系统>局部 |
| 持续价值 | 战役结束即止 | 数年持续供应 | 持续>一时 |

## 双轨制评价方案

**高祖的裁决**：萧何第一（精神荣誉），曹参封户10600 > 萧何封户8000（物质奖励）

```
精神荣誉体系（排序）
  标准：战略价值、系统价值、持续价值
  萧何第一（万世之功）
  ↓ 传递价值观："功人" > "功狗"

物质奖励体系（封户）
  标准：战功数量、风险程度、可见度
  曹参封户最多（身被七十创）
  ↓ 传递价值观：高风险、高可见度获高回报
```

**平衡效果**：
- 战功派（曹参支持者）：封户多 → 物质满足
- 智谋派（萧何支持者）：排序前 → 精神满足
- 两派都被照顾 → 减少内耗

## 功劳四级分级体系

### 第1级：战略支撑（万世之功）

**代表**：萧何

**认定标准**：
- 系统性贡献（守关中、给军食、转漕）
- 持续性（数年）
- 不可替代（"亡一人则危"）

**现代对应**：建立核心业务系统的CTO/COO；建立学科体系的带头人

**评价方法**：
1. 是否建立了系统？（而非单点突破）
2. 是否持续产生价值？（而非一次性）
3. 是否不可替代？（而非容易替换）

### 第2级：全程参战 + 独立战功

**代表**：曹参、周勃、灌婴

**认定标准**：
- 全程参与（从起兵到胜利）
- 独立战功（有自己的战绩）
- 战功卓著（规模和质量均高）

**评价方法**：
1. 参与时间？（全程>中途>晚期）
2. 独立贡献？（有独立业绩>纯执行）
3. 贡献规模？（量化对比）

### 第3级：特殊贡献（关键时刻）

**代表**：张良（六奇计）、韩信（定魏赵齐）

**认定标准**：
- 关键时刻（转折点）
- 决定性作用（改变战局）
- 特殊能力（不可复制的智谋/战术）

**评价方法**：
1. 是否在关键时刻？（转折点>日常）
2. 是否改变局面？（决定性>辅助性）
3. 是否独特能力？（不可复制>可复制）

### 第4级：局部战功

**认定标准**：
- 参与某些战役
- 战功有限
- 可替代性高

## 功劳量化方法

### 方法1：战功型量化（易）

**曹参战功统计**：
> "参功：凡下二国，县一百二十二；得王二人，相三人，将军六人，大莫敖、郡守、司马、候、御史各一人。"

量化指标清晰：城池数、俘虏级别、战伤次数

**现代KPI类比**：销售签单数、金额、客户数；开发专利数、产品数

优点：清楚、可比、易考核
缺点：忽视难量化贡献

### 方法2：智谋型量化（难）

萧何功劳的困难：转漕多少粮？补兵多少次？关中保全价值如何量化？

**解决方案：定性+定量结合**

```
难量化岗位（研发/后勤）：
  定量：专利数量、产品上线
  定性：系统架构优化、技术债务清理、新人培养
  权重：定量40% + 定性60%（因为是"万世之功"）
```

### 方法3：对比评价法（反事实推理）

鄂君的论证方式：

> "今虽亡曹参等百数，何缺于汉？汉得之不必待以全。"

**提问框架**：
1. 如果失去这个人，组织是否还能运转？
   - 能运转 → 可替代 → 功劳中等
   - 不能运转 → 不可替代 → 功劳极高
2. 这个人的工作能否被他人快速接手？
3. 这个人的价值是短期还是长期？

## 现代应用案例

### 案例1：企业销售部 vs 研发部奖金争议

**争议**：销售签单一亿，要求高奖金；研发说没产品销售卖什么？

**分析**：
- 销售部 = 曹参（攻城略地，业绩可见）
- 研发部 = 萧何（系统建设，万世之功）

**解决方案**：
- 精神荣誉：年度总结强调研发重要性，职级晋升研发优先
- 物质奖励：销售高提成（业绩联动）；研发期权+奖金（长期激励）

### 案例2：学术论文署名

**争议**：PI设计实验 vs 博士生做了全部实验

**分析**：
- PI = 萧何（发踪指示，功人）
- 博士生 = 曹参（追杀猎物，功狗）

**学术惯例（对应双轨制）**：
- 精神荣誉：PI → 通讯作者（对应萧何第一）
- 物质回报：博士生 → 第一作者（对应曹参封户多）

### 案例3：政府招商部 vs 规划部绩效争议

**分析**：招商部=曹参，规划部=萧何

**解决方案**：
- 精神荣誉：年度总结强调规划战略价值，规划部优先晋升
- 物质奖励：招商部按引进企业数量高奖金；规划部中等奖金+长期激励

## 常见错误

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 只看可见贡献（奖励销售忽视研发） | 研发人才流失，产品失去竞争力 | 用"万世之功"框架，确保研发获精神荣誉 |
| 只看短期成果 | 短视行为盛行，长期问题积累 | 双轨制平衡（短期奖金+长期期权） |
| 只用单一体系 | 派系斗争，内耗严重 | 精神荣誉+物质奖励双轨并行 |
| 不建分级体系，一刀切 | 核心人才不满并流失 | 建立4级体系，明确区分 |

## 操作清单

### 建立评价体系

- [ ] 确定评价维度（战略/系统/持续价值）
- [ ] 建立双轨制（精神荣誉+物质奖励）
- [ ] 制定量化标准（易量化岗位80%定量，难量化岗位60%定性）
- [ ] 建立分级体系（战略支撑/全程参与/关键贡献/局部贡献）
- [ ] 确保智谋型不被忽视（排序优先），战功型获得补偿（封户多）

### 评价具体贡献

战功型（前线/销售）：
- [ ] 收集量化数据（数量、质量、风险承担）
- [ ] 计算综合得分，确定奖励

智谋型（研发/后勤/战略）：
- [ ] 定性评价（系统性、持续性、不可替代性）
- [ ] 对比评价（反事实推理：失去此人如何）
- [ ] 确定排序（精神荣誉优先）

### 处理评价争议

- [ ] 识别争议类型（战功型 vs 智谋型）
- [ ] 使用"功狗vs功人"框架
- [ ] 使用"万世vs一时"框架
- [ ] 双轨制平衡两派
- [ ] 高层明确裁决并解释理由

## 跨章节互见

### 高祖功臣侯者年表与萧相国世家：功狗功人的"表"与"传"——排名体系背后的政治逻辑与叙事逻辑

"功狗与功人"的比喻在高祖功臣侯者年表中是"排名依据"的理性阐释，在萧相国世家中则是"君臣对话"的戏剧性呈现——两篇合在一起，同一个评价体系在"制度"和"叙事"两个层面的意义才完整呈现。高祖功臣侯者年表中，刘邦对功臣的排名是整个年表的结构核心——"十八侯位次"的排列是汉初政治秩序的"宪法性文件"。年表中萧何排第一（酂侯）、曹参排第二（平阳侯），这个排名不是随意的——年表序言中"功人vs功狗"的比喻成了汉代乃至整个中国政治中"规划者优先于执行者"的经典理论框架。年表的叙事是"制度性"的——它不讨论谁更辛苦、谁牺牲更多，只讨论"什么功劳对国家最有持续价值"。年表的排序标准是"万世之功大于一时之功"——这不是道德判断（不是谁更忠诚），而是系统判断（谁对国家机器更重要）。萧相国世家中，"功狗与功人"的对话被赋予了戏剧性的现场感——刘邦在封功臣时遭遇了来自武将们的强烈不满，因为萧何排名第一（文官）激怒了所有在战场上拼命的将领（[[053_萧相国世家|053-?]]）。武将们说："臣等身被坚执锐，多者百馀战，少者数十合，攻城略地，大小各有差。今萧何未尝有汗马之劳，徒持文墨议论，不战，顾反居臣等上，何也？"武将的逻辑是"流汗最多的人应该排第一"——他们用工作量（身被坚执锐）和风险（百馀战）来论证自己的功劳。刘邦用"猎狗与猎人"的比喻回应武将们——"夫猎，追杀兽兔者狗也，而发踪指示兽处者人也。今诸君徒能得走兽耳，功狗也。至如萧何，发踪指示，功人也。"刘邦的比喻在年表中是"冷冰冰的制度设计"，在萧相国世家中则是"充满火药味的政治对话"——武将们被比作"狗"（虽然是替刘邦打猎的狗），这对他们来说是巨大的侮辱。合观两篇，年表中的"萧何第一"是"结果"（封侯排名），萧相国世家中的"功狗功人"是"过程"（排名如何被政治斗争和君主权威所决定）。年表展示的是一个"已经完成"的制度（十八侯排序），萧相国世家展示的是这个制度如何被"制造"出来的（刘邦必须用一场"精神上的羞辱"来确立文官系统的重要性）。两篇并读，读者看到的是"政治排名"的真相——排名从来不只是"实至名归"的计算，而是"政治权威"+"意识形态"+"利益平衡"的综合产物。刘邦需要压制武将派的政治力量，所以用"功狗功人"将萧何排在前面；但他也需要安抚武将派的情绪，所以给曹参最多的封户（10600户对8000户）——这就是"双轨制"的本质：精神荣誉（排名）传递政治信号（文官重要），物质奖励（封户）安抚情绪（武将也不亏）。

### 高祖功臣侯者年表与曹相国世家：曹参的"功狗"定位与他的"功臣世家"——同一人的两个身份

曹参在高祖功臣侯者年表中是"功狗"的代表（攻城最多、战功最大、但被定位为执行者），在曹相国世家中却是"从将军到相国"的完整人生——两篇合在一起，曹参一人身上的"执行者"与"治理者"双重身份才完整呈现。高祖功臣侯者年表中，曹参的战绩被总结为——"下二国，县一百二十二；得王二人，相三人，将军六人……"——这是整个年表中战功数据最完整的条目之一。曹参在年表中的身份是唯一的"攻城者"——他攻下了122座县城和2个诸侯国，是汉初将领中攻城最多的。但年表给他的定位就是"功狗"——年表的名次是萧何第一、曹参第二，说明在制度设计中曹参的"功狗"身份是制度化的（不是个人评价，而是系统定位）。曹相国世家中，曹参的人生远不止"功狗"——他从沛县起兵时就是刘邦的战友（"高祖为沛公而参以中涓从"），在楚汉战争中攻城野战（"高祖定天下，参以中尉从……"），在汉朝建立后被派往齐国做相国（"天下初定，悼惠王富于春秋，参尽召长老诸生……"），最后接替萧何做汉相国（"参代何为汉相国，举事无所变更……"）（[[054_曹相国世家|054-?]]）。曹参的世家完整地记录了从"将军"到"相国"的转型——他不是一个"只会打仗的人"，他在齐国用黄老之术治理了九年，"齐国安集，大称贤相"。曹相国世家中的曹参是一个"能文能武"的完整人物，而年表中的曹参则被简化为"战功数据+排名第二"——两种叙事不是矛盾的，而是"同一个人的不同侧面"。合观两篇，曹参在高祖功臣侯者年表中的"功狗"定位（执行者、攻城者）和他在曹相国世家中的"贤相"定位（治理者、黄老实践者）构成了一个完整的人物肖像——年表中的曹参是"从沛县到汉朝建立"阶段的曹参（战争时期），世家中的曹参是"从齐国到汉相国"阶段的曹参（和平时期）。一个人有两个"面孔"，取决于历史的阶段和叙事的框架。年表只截取了他一生的前半段（功劳积分），世家记录了他一生的全部（从军功到治国）。两篇并读，读者看到的是"功狗功人"这个框架的简化性——曹参如果只被当作"功狗"（执行者），那就忽略了他作为齐国相国、汉相国的治理成就；但如果他被当作"功人"（规划者），他前期的战功又不是简单的"发踪指示"。曹参的例子说明：真正的功臣往往"兼有二者"——既是被称赞的执行者（功狗），也是能独当一面的治理者（功人）。

### 高祖功臣侯者年表与留侯世家：张良排名第六十二？——年表排名与真实贡献的错位

张良在留侯世家中是刘邦"运筹策帷帐之中，决胜于千里之外"的首席谋士，在高祖功臣侯者年表中却排名第六十二（留侯）——两篇合在一起，年表排名与真实贡献的巨大落差揭示了汉初封爵制度的复杂逻辑。留侯世家中，张良是刘邦集团中地位最特殊的谋士——他的贡献不是具体的战役（像曹参那样攻下122城），也不是系统的后勤（像萧何那样转漕关中），而是"六奇计"（在关键时刻提供改变格局的决策）。张良的贡献是"战略性的"——他在刘邦入咸阳时劝其退出秦宫（政治姿态），在鸿门宴上助刘邦脱险（外交博弈），在刘邦封韩信为齐王时"蹑足附耳"阻止了一场政治灾难（危机管理），在刘邦定都关中时支持娄敬的建议（战略选址）。张良的贡献"无形无象"但"无处不关键"——没有张良的六奇计，刘邦可能根本走不到封功臣那一步。高祖功臣侯者年表中，张良的留侯排第六十二——远远低于萧何（第一）、曹参（第二）、周勃（第四）、樊哛（第五）等人。为什么张良作为"运筹策帷帐之中"的首席谋士排名如此靠后？年表没有直接解释，但结合《高祖功臣侯者年表》的排序标准可以推断：年表的排序依据是"具体可量化的战功"而非"谋略贡献"。张良没有攻城计数、没有带兵打仗、没有守土之责——他的贡献无法用"下X城、斩X级"来量化。张良的封侯是"定天下后封"（在战争结束后才被加封的），所以他的排名在最末期（按封侯时间排序也是年表的一个标准）。合观两篇，留侯世家中张良的"运筹帷幄"（最高评价）和高祖功臣侯者年表中张良的"排名六十二"（制度评价）构成了中国古代政治中"真实贡献"与"制度认可"之间的经典落差。张良本人对此的态度是彻底的超脱——"今以三寸舌为帝者师，封万户，位列侯，此布衣之极，于良足矣。愿弃人间事，欲从赤松子游耳"（[[055_留侯世家|055-?]]）。张良的"排名六十二"和他的"愿弃人间事"之间存在着一种微妙的平行关系：制度上不把他排在最前面（因为他的贡献不可量化），他自己也不在乎排名（因为他的追求超越了功名利禄）。在年表中排在前面的曹参（第二）用一生的时间和身体上的七十处伤口换来了排名，而排名第六十二的张良用几句话改变了战争的方向。两篇并读，读者看到的是排名制度的内在局限——最能量化的贡献（曹参的攻城数）和最不可量化的贡献（张良的六奇计）在同一个表单上排名，前者必然排在前面——不是因为前者的贡献更大，而是因为前者的贡献更容易计数。"功狗与功人"的区分框架在张良这里遇到了一个更复杂的层次：张良既不是功狗（他没有攻城），也不是功人（他的贡献不像萧何那样是看得见的制度建设），他是"谋者"——在制度的缝隙中改变历史的人。年表无法衡量"谋者"的价值，而留侯世家作为"列传"弥补了这个空白。
