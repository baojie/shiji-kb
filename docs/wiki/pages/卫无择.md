# 卫无择

出现于《史记》**1** 处。

## 生平

卫无择，西汉景帝时功臣之后封侯，以先祖从汉高祖起于沛、隶属皇䜣，以郎官身份击陈余，任卫尉之功，汉景帝四年四月丙申受封，谥简侯，食邑六百户。其子恭侯胜袭封，后代侈又袭，侯国延续数代。事见[[019_惠景间侯者年表|惠景间侯者年表]]。
