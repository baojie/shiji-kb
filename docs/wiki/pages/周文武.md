# 周文武

出现于《史记》**1** 处。

## 生平

周文武，即[[周文王]]与[[周武王]]，西周开国父子二君的合称。秦始皇时期廷尉李斯在议郡县制时，以周文武大封同姓子弟为诸侯为例，指出血缘关系疏远后诸侯相攻如仇雠，周天子无法禁止，以此论证分封制之弊。事见[[006_秦始皇本纪|秦始皇本纪]]。
