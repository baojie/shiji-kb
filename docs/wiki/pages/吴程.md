# 吴程

出现于《史记》**1** 处。

## 生平

吴程，西汉初年功臣，以长沙国柱国（最高武将职位）身份封义陵侯，食邑千五百户，高祖九年九月丙子封顷侯，吴程元年。后侯种继位，侯种薨后无嗣，国除，皆失谥。事见[[018_高祖功臣侯者年表|高祖功臣侯者年表]]。
