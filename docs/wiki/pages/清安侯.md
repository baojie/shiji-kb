# 清安侯

## 生平

清安侯，即申屠臾，西汉申屠嘉功臣世家的第三代传承侯主。申屠嘉以功封节侯，传子恭侯蔑，蔑传申屠臾，元狩二年申屠臾袭封谥「清安侯」。在位五年后（元鼎元年），申屠臾因坐为九江太守时有罪（具体罪状不详）而侯国国除。清安侯是申屠嘉功臣侯国历三代传承后以太守获罪而除国的典型案例。事见[[019_惠景间侯者年表|惠景间侯者年表]]。
