# 薛郡

## 地理位置

薛郡位于今山东省枣庄市一带，战国时为魏、楚等国争夺的战略要地，秦统一后置薛郡，楚汉之际灌婴军在此破薛郡长，见于[[095_樊郦滕灌列传|樊郦滕灌列传]]。
