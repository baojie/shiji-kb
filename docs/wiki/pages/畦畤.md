# 畦畤

畦畤，秦代在关中设立的祭天坛庙之一，是秦国五畤（祭祀五帝）体系中的重要组成部分，建于今陕西宝鸡一带，专司祭祀某一方位之帝，与西畤、鄜畤、密畤、上畤并列，构成秦代独特的山川祭祀礼制。

《封禅书》两度涉及畦畤：其一记秦国郊祀制度的历史沿革与礼仪规制；其二记汉承秦制的郊祀安排，载"西畤、畦畤，祠如其故，上不亲往"，说明汉初基本沿用秦代西方祭祀格局，皇帝本人不亲赴西畤、畦畤行礼，委官代祀。畦畤因此成为秦汉之际礼制传承的历史见证——秦设五畤，汉初并未废除，而是在郊祀体系中继续保留秦代的五方帝坛，直至汉武帝时期才大规模改革祭祀制度。畦畤的存续，折射出汉初"汉承秦制"的现实政治考量与礼制惯性。
