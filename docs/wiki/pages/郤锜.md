# 郤锜

出现于《史记》**1** 处。

## 生平

郤锜，春秋晋国大夫，郤氏族长，晋厉公时期与郤至、郤犨并为「三郤」，专权跋扈，后被晋厉公所杀，郤氏灭族。事见[[039_晋世家|晋世家]]。
