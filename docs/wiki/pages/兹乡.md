# 兹乡

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

兹乡，古地名，位于今山东省寿光市东北一带，属汉代琅邪郡（齐国东部）。兹乡处于山东半岛与渤海之间的平原地带，是古代齐国东北部地名网络的组成部分，《史记》中出现于齐国或汉代琅邪郡相关地理叙事。
