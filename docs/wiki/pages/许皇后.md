# 许皇后

**许皇后**（许平君），西汉[[汉宣帝]]第一任皇后，[[许广汉]]之女。汉宣帝登基前已成婚，即位后立为皇后；后被权臣[[霍光]]之妻霍显毒杀（前71年），史称"许后之死"。
