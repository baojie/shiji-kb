# 奄

奄，商代邦国，约在今山东曲阜一带，是商代东方的重要诸侯。周初东征中，奄国参与了武庚（纣王之子）的叛乱，周公旦率军东征三年，最终平定奄国，将其封地赐予周代诸侯，奄地随之成为鲁国的核心区域。

《周本纪》在描述周公东征的历史时涉及奄，以奄的平定为周初巩固东方统治的重要军事行动之一。奄地的征服标志着商代东方诸侯残余势力的彻底平息，也是鲁国获得封地、在东方建立根基的历史基础，两处记载合观，奄成为《史记》中周初东征、封建诸侯这一历史进程的重要地理坐标，是商周政治地理转型的见证地名。
