# 成王营洛定鼎

> 成王在丰，使召公复营洛邑，如武王之意。周公复卜申视，卒营筑，居九鼎焉，曰"此天下之中，四方入贡道里均"。成王迁殷遗民于洛邑。

**主要人物**：召公、周公、成王

## 事件经过

成王在丰，使召公复营洛邑，如武王之意。周公复卜申视，卒营筑，居九鼎焉，曰"此天下之中，四方入贡道里均"。成王迁殷遗民于洛邑。
