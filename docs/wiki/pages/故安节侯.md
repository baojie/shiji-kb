# 故安节侯

## 生平

故安节侯，即申屠嘉，西汉丞相，谥节。汉文帝、景帝时为丞相，以刚直严正著称。因晁错私开宫苑通道，申屠嘉欲弹劾之，但景帝庇护晁错，申屠嘉未能得逞，深感被晁错所卖，回府后呕血而死，谥为故安节侯。事见[[096_张丞相列传|张丞相列传]]。
