# [[专诸刺王僚]]：鱼炙藏刃与吴国篡位

**Fact ID**: fact_003
**类型**: 刺杀事件
**时期**: 春秋晚期（公元前515年）
**核心人物**: 专诸、公子光（阖闾）、[[吴王僚]]、[[伍子胥]]

---

## 背景：吴国王位之争

### 继承链条
```
吴王诸樊（公子光之父）
├─ 理想继承：诸樊 → 余祭 → 夷眛 → 季子札（贤者）
├─ 实际继承：诸樊 → 余祭 → 夷眛 → 僚（夷眛之子）
└─ 公子光主张："使以兄弟次邪，季子当立；必以子乎，则光真適嗣，当立。"
```

### 人物关系
- **专诸**: 吴国[[堂邑]]人，被伍子胥发现"其能"
- **伍子胥**: 引荐专诸给公子光，看出公子光"有内志"
- **公子光**: 诸樊之子，认为自己才是"真適嗣"
- **准备时长**: 九年（"光既得专诸，善客待之"）

---

## 时机选择

### 外部条件（公元前515年春）
1. **[[楚平王]]死**: [[楚国]]丧期
2. **吴军外出**: 王僚使二弟围楚之灊，延陵季子使晋
3. **楚国反击**: 绝吴将后路，吴兵不得还

### 专诸判断
> "王僚可杀也。母老子弱，而两弟将兵伐楚，楚绝其后。方今吴外困于楚，而内空无骨鲠之臣，是无如我何。"

**公子光承诺**:
> "光之身，子之身也。"

---

## 刺杀实施（四月丙子）

### 安保布置
- **路线**: 兵陈自宫至光之家
- **门户**: 皆王僚之亲戚，夹立侍，持长铍

### 执行细节
1. **伪装**: 公子光详为足疾（假装脚病）
2. **武器**: 置匕首于鱼炙之腹中
3. **行刺**: 专诸擘鱼，因以匕首刺王僚
4. **结果**: 王僚立死，左右杀专诸
5. **夺位**: 公子光出伏甲灭王僚之徒，自立为阖闾

### 善后
- 封专诸之子为上卿

---

## 成功要素

### 1. 长期准备
- 九年蓄养，建立深度信任
- 伍子胥引荐提供信誉背书
- "光之身，子之身也"（命运共同体）

### 2. 完美时机
- 楚平王死，季子出使，二弟被困
- 王僚无骨鲠之臣
- 设宴降低警惕

### 3. 武器创新
- **鱼炙藏刃**: 史上最著名的武器隐蔽方式
- **不可防御**: 进食场合，王僚完全放松

### 4. 后备武力
- 伏甲士于窟室
- 专诸行刺是信号，迅速镇压

---

## 历史影响

### 短期效果
1. 公子光自立为[[吴王阖闾]]
2. 专诸之子封为上卿
3. 重用伍子胥、孙武，吴国开启强国之路

### 长期影响
1. 吴国崛起，大败楚国
2. "鱼肠剑"典故
3. "专诸进炙"成为刺杀经典案例

## 战术模式

### Pattern 1: 信任建立
- 长期蓄养（九年）
- 雇主承诺（命运共同体）
- 第三方背书（伍子胥）

### Pattern 2: 时机窗口
- 外部敌人（楚国）
- 内部空虚（军队外出）
- 心理麻痹（设宴）

### Pattern 3: 武器创新
- 常规检查无法发现
- 进献仪式必然发生
- 暴露即执行
