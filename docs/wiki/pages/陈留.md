# 陈留

位于今河南省开封市东南，战国时属魏国，秦时设陈留县。秦汉之际，陈留为中原交通要冲，扼守大梁（开封）东南通道，战略地位重要。

秦末大乱，陈留是刘邦起事的早期据点之一。辩士[[郦食其]]（高阳酒徒）即自陈留向刘邦自荐，进入汉营效力。郦食其后为刘邦说下陈留，取其积粟，极大充实了汉军后勤。汉初置陈留郡，辖区广及今豫东一带，历西汉、东汉均为重要行政区划。
