# 卫康叔封

出现于《史记》**1** 处。

## 生平

卫康叔封，即[[卫康叔]]，姬姓，名封，[[周文王]]之子，[[周武王]]之弟，卫国开国君主。周武王灭商后，封封于卫地（商旧都朝歌一带），以镇殷民。武王灭纣誓师时，卫康叔封布采席，参与牧野誓师及战后散财粟赈民的典礼。此后成王时管叔、蔡叔叛乱，周公平定后，重封封于卫，以殷遗民旧俗加以治理，史称康叔善政。事见[[032_齐太公世家|齐太公世家]]。
