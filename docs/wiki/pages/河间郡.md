# 河间郡

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·冀州（页26-27）（见右栏）。

河间郡，西汉冀州郡级行政单位，位于今河北省沧州献县一带，汉代曾多次为诸侯王封国（河间国），是楚元王刘交后裔的封地之一。
