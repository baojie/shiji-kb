# 马童

出现于《史记》**2** 处。

## 生平

马童，秦末汉初将领，汉军骑将，项羽自刎时马童认出并斩取其首，以此封侯，见于项羽本纪相关记录，史书载之简略。事见[[007_项羽本纪|项羽本纪]]。
