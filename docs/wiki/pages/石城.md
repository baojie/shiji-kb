# 石城

石城，战国末期赵国城邑，地望约在今山西或河北一带，是秦昭王时期秦军蚕食赵国领土的重要战果。《赵世家》载赵惠文王十八年"秦拔我石城"，秦军攻取石城，是秦持续对赵施压、逐城蚕食的历史记录；《六国年表》同步记载"秦拔我石城"，两书相互印证，揭示了石城在战国末期秦赵争夺中的战略价值。

《廉颇蔺相如列传》亦以"其後秦伐赵，拔石城"作为秦赵冲突背景，与蔺相如渑池之会等外交事件并列，说明石城的失守是秦赵关系紧张升级的历史节点；三书共同将石城定格为秦昭王时期秦对赵逐步压迫、最终导致长平之战的历史进程中的重要地理坐标。
