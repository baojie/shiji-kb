# 余吾水

余吾水，匈奴腹地的河流，约在今蒙古国境内，是汉武帝时期汉军北征匈奴推进最深入的地理标志之一。汉武帝多次遣将深入漠北，余吾水成为汉军出塞深入、与匈奴单于主力正面对决的战场水系。

《匈奴列传》载贰师将军李广利率军至余吾水，"匈奴闻，悉远其累重於余吾水北，而单于以十万骑待水南，与贰师将军接战"，双方主力在余吾水两岸展开激烈攻防，最终贰师因家族被株杀而降匈奴。《汉兴以来将相名臣年表》与《滑稽列传》亦记卫青及李广利"至余吾水上"的出征，余吾水因此成为汉武帝晚年北击匈奴战役系列的地理终点坐标。
