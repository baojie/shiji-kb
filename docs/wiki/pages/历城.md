# 历城

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·东郡北海（页21-22）（见右栏）。

历城，今山东济南市区一带，是汉武帝封禅泰山沿途的重要城邑，也是东方战略地带的中转节点。汉武帝封禅路线自长安东行，途经历城附近，封禅礼成后颁诏免税，以示恩惠沿途百姓。

《封禅书》与《孝武本纪》均载汉武帝封禅后制诏："复博、奉高、蛇丘、历城，无出今年租税"，历城与博、奉高等泰山周边城邑同获免税优惠，显示了历城在封禅路线上的地理地位。《郦生陆贾列传》则从楚汉战争角度涉及历城附近的军事叙事，历城由此跨越楚汉与汉武两个时代，持续出现于东方战场与礼仪巡行的叙事之中。
