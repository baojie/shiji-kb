# 楚申无宇

出现于《史记》**1** 处。

## 生平

楚熊，楚国早期先公，楚熊艾之子，楚熊渠之曾祖辈。《楚世家》记楚国先公传承：熊绎→熊艾→楚熊→熊胜（以弟熊杨为后）→熊杨→[[楚熊渠]]，楚熊是从熊绎到熊渠的传承链中的普通一环。楚熊在位期间史书无具体事迹，以传承格式出现于《史记》的楚国先公序列之中。事见[[040_楚世家|楚世家]]。
