# 桃林

桃林，秦晋之间的重要地名，约在今陕西潼关与河南灵宝之间，是古代关中进入中原的要道地带，也是《史记》传说与史实交汇的地理坐标。《赵世家》记，造父为周穆王御马，从桃林取骥乘匹——"桃林盗骊、骅骝、绿耳"，以这批名马助周穆王西游会西王母，又飞驰东归破徐偃王，桃林作为产马之地因此进入上古马政传说的叙事之中。

《周本纪》与《乐书》均记周武王克殷后归马于桃林，以放牧战马于桃林作为偃武修文的象征性举措，昭示周朝以德治天下的政治宣言。桃林的两重意象——一为产马与驯马的地理场所，一为武王收兵弃战的礼制符号——共同体现了《史记》中这一地名承载的上古历史与政治想象的双重内涵。
