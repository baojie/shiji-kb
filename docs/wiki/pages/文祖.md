# 文祖

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

文祖，上古圣地，据《五帝本纪》载帝舜"巡狩，方岳，归，格于艺祖"，相传帝舜曾于此祭祖，位于今山东省章丘市西南一带。文祖之名见于《史记》五帝本纪的祭祀体系，是上古帝王礼制与地理结合的历史遗存。


文祖，上古祭祀场所，传说为帝尧告祖受命、舜摄政谋政的礼仪空间，具体地望不详，可能是帝廷中的宗庙或祭坛。《五帝本纪》载，舜摄政治事，"至於文祖，谋于四岳，辟四门，明通四方耳目"，以文祖为摄政公告天下、咨询四岳的礼仪场所，赋予其重要的政治仪式内涵。

《五帝本纪》另记，尧在禅位前"申戒文祖"，向文祖宣示禅让天命，"云天之历数在尔躬"，以文祖为天命转移的见证场所。《历书》亦引此语，揭示文祖在上古政权交替叙事中所承担的宗教与政治双重功能。文祖在《史记》中以礼制圣所的形态出现，是上古帝王以祭祀场所构建政治合法性的典型案例，也是中国早期政教合一文化的地理符号。
