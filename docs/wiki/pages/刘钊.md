# 刘钊

出现于《史记》**1** 处。

## 生平

刘钊，西汉高祖时功臣封侯，以都尉身份参与击败[[项羽]]、破燕王[[臧荼]]之役有功封侯，后又以将军身份捕获[[韩信]]，益封千户。汉六年八月丙辰受封，谥敬侯，其子吉袭封，后吉因罪夺爵，侯国除。事见[[018_高祖功臣侯者年表|高祖功臣侯者年表]]。
