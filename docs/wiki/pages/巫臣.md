# 巫臣

## 生平

巫臣，春秋楚国、晋国人物，又称申公巫臣（以申地大夫身份著称）。最初在楚国为官，因私窃美人[[夏姬]]（[[陈灵公]]之通妾）而奔晋，晋以其为邢大夫。后巫臣主动出使吴国，「以晋通于吴」，传授吴国战车战法，并促成吴国对楚国的战略联盟，是楚吴关系逆转的关键人物。巫臣的出走与使吴之举，深刻影响了春秋后期楚国的战略格局。事见[[014_十二诸侯年表|十二诸侯年表]]。
