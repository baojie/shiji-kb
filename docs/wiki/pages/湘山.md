# 湘山

湘山，位于洞庭湖中，传为楚地女神湘君、湘夫人的神祀之所，是《史记》中承载秦始皇暴怒与南方神灵信仰的标志性地点。《封禅书》将湘山列入秦代祭祀地理体系，与南方名山大川并录。

《秦始皇本纪》载秦始皇南巡至湘山祠时，"逢大风，几不得渡，上问博士曰：湘君何神"，博士以尧女为对，始皇大怒，"使刑徒三千人皆伐湘山树，赭其山"，以赭山之刑惩罚风神阻路之冒犯。这一事件被司马迁作为秦始皇刚愎无度的典型细节，湘山也因此成为《史记》中神人冲突叙事的地理符号。

> **【正义】** 今荆州也。言欲向衡山，即西北过南郡，入武关至咸阳。
