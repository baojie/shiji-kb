# [[汉武帝]]开西南夷的战略目的

## 核心内容

汉武帝开发西南夷并非出于单一目的，而是多重战略考量叠加的结果。从《史记·西南夷列传》的记录来看，开西南夷的核心战略目标可以归纳为以下几个层次，它们相互关联，共同构成了汉武帝边疆扩张总战略的有机组成部分。

**战略目标一：寻找通往大夏（身毒）的商路——"蜀身毒道"的战略构想**

汉武帝对西南夷的最初兴趣，部分来源于对"蜀身毒道"的战略想象。张骞出使西域归来后，报告在大夏（今阿富汗）见到了来自蜀地的布和竹杖，商人告知这些货物来自身毒（今印度）。这意味着存在一条从四川出发、绕过匈奴控制区、经东南亚直达印度的陆路商道。

控制这条商道，意味着：
- 在匈奴封锁的北方丝绸之路之外，获得一条新的对外联系通道
- 与强大的身毒（印度）诸国建立直接联系，潜在地形成对匈奴的战略迂回包围

汉武帝为此数度派遣使者，试图从西南夷地区向西北方向打通这条商路，但始终未能成功——昆明夷的阻挠是最大障碍，这也成为汉廷对西南夷用兵的重要原因之一。

**战略目标二：包围南越，打通南方战略大后路**

[[唐蒙通夜郎]]的最初战略动机，是利用牂柯江水系形成对南越的包围态势。夜郎位于南越（广东）的西北方，控制夜郎意味着：
- 可以从西北方向对南越形成侧翼威胁
- 在战时可以多路进击南越，打破南越利用岭南山地进行防御的战略
- 为汉朝南下岭南创造更有利的战略态势

事实上，元鼎五年（前112年）汉朝灭南越时，从西南夷方向出发的[[路博德]]部确实配合了对南越的作战。

**战略目标三：宣扬汉朝国威，拓展版图**

汉武帝的雄才大略有其意识形态维度：将汉朝的版图扩展到"四海"，使更多族群纳入汉朝天子的"德化"之下，是实现"大一统"政治理想的重要组成部分。这种意识形态驱动不应被低估——[[司马相如]]在《[[难蜀父老]]》中明确表述了这一逻辑，说明这不仅是统治者的想法，也成为开边政策的正式意识形态依据。

**战略目标四：经济利益——西南资源的开发**

西南夷地区蕴藏丰富的自然资源，包括金、银、铜、盐等矿产，以及丰富的农业潜力。控制这一地区，有助于扩大汉朝的经济版图。巴蜀商人早已通过非官方渠道与夜郎等地有贸易往来（如枸酱的流通），说明西南夷地区具有可观的商业价值。

**开西南夷的代价与争议**

开西南夷是汉武帝边疆政策中争议最大的项目之一。反对者的主要论点：

1. **成本极高**：修路所需劳役和财力是天文数字，"费以巨万计，道不通"
2. **收益有限**：西南夷地区人口稀少，土地贫瘠（相比关中、河南），长期难以成为税收来源
3. **持续不稳**：越巂、昆明等族群"数反"，汉朝不得不反复用兵，形成恶性循环
4. **战略目标落空**：蜀身毒道始终未能打通，"找到大夏替代路线"的核心战略目标并未实现

支持者（包括唐蒙、司马相如、武帝本人）则强调战略布局的长远价值，认为短期代价是值得的。

**历史结果评价**

从长远看，汉武帝开西南夷的成效是部分实现的：西南夷地区最终被纳入汉朝行政版图，云贵高原的汉化进程在此后数百年中持续推进，而"蜀身毒道"的战略目标则从未实现。这场大规模投入的历史遗产，是争议性的——就像汉武帝其他许多宏大工程一样。
