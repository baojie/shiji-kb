# 千亩

千亩，西周时期王畿内的农耕祭祀场所，今山西介休一带，是天子举行籍田礼的神圣之地。籍田之礼意在天子亲耕以示劝农，宣示王室对农业文明的守护。《周本纪》载，宣王"不脩籍於千亩"，拒绝虢文公谏劝，违背传统礼制，结果"战于千亩，王师败绩于姜氏之戎"，将君主废弃籍田与军事惨败直接勾连。

《晋世家》记晋穆侯"伐千亩，有功"，生少子成师，周史以此地为命名背景解读两子名字之凶兆，"太子曰仇，少子曰成师……今適庶名反逆，此後晋其能毋乱乎"，预示晋国曲沃代翼之乱。《赵世家》则载赵先祖奄父"及千亩战，脱宣王"，说明千亩之战是周宣王时代重大军事事件，对多个诸侯国历史均有深远影响。
