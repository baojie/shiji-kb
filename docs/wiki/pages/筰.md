# 筰

筰，西南夷古国，约在今四川西南凉山、雅砻江流域，与邛并称"邛筰"，是汉武帝西南夷开拓政策的重要对象之一。筰地君长部落众多，地形险峻，秦汉之际曾一度与中原保持联系，后因汉初收缩政策而断绝。

《司马相如列传》载，[[唐蒙]]打通夜郎道后，"邛筰之君长闻南夷与汉通，得赏赐多，多欲原为内臣妾，请吏，比南夷"，筰地君长主动请求归属汉朝；[[司马相如]]建议武帝"邛、筰、厓、駹者近蜀，道亦易通，秦时尝通为郡县，至汉兴而罢，今诚复通，为置郡县，愈於南夷"，力主将邛筰与夜郎等并列纳入郡县体系。筰地的归附，是汉武帝西南夷经营中较为顺利的一环。
