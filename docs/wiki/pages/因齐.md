# 因齐

出现于《史记》**1** 处。

## 生平

因齐，即[[齐威王]]，战国田氏齐国君主，姓田名因齐。齐桓公（田午）卒，子因齐立，即齐威王。齐威王在位期间任用邹忌为相、田忌为将、孙膑为军师，与魏国争霸，取得桂陵、马陵之战的胜利，使齐国成为战国时代强国之一。事见[[046_田敬仲完世家|田敬仲完世家]]。
