# 嶓冢

嶓冢，《禹贡》地理体系中的重要山岭，位于今陕西汉中与甘肃天水之间一带，是汉水发源地，也是大禹"道九山"路线中的重要节点。嶓冢在《史记》中以上古地理志的地名而两度出现，是华夏山水体系的核心地名之一。

《夏本纪》两处涉及嶓冢：一处在"道九山"中载"道嶓冢，至于荆山"，将嶓冢纳入南方山岳连贯体系；另一处在梁州贡赋路线中以嶓冢为汉水发源的地理起点，与贡道水系相互关联，揭示了嶓冢在大禹治水工程中作为汉水源头山岭的地理意义。两处合观，嶓冢成为《史记》上古地理叙事中沟通南北山川、连通汉水流域的核心地名之一。
