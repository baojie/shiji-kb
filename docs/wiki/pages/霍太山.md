# 霍太山

霍太山，即今山西省霍州市北部的霍山（又称太岳山主峰），位于汾河东岸、太岳山脉中段，自古为晋地名山，与太行、吕梁并列为山西地标。霍太山在《史记》中因与秦人祖先蜚廉的传说紧密相连而留名：蜚廉是秦人远祖，事殷纣王，为纣北征归来后无处复命，遂在霍太山建坛以祭天地汇报，并据传在此获得神授石棺、死后亦葬于此山，是秦人传说中最早与山岳崇拜结合的始祖叙事之一。

《秦本纪》同一段落（005-3）两度提及霍太山。第一处记蜚廉北征归来，"无所报，为坛霍太山而报，得石棺，铭曰'帝令处父不与殷乱，赐尔石棺以华氏'"，以霍太山作为神迹显现的宗教地点——蜚廉在此建坛祭祀，传说因此获得刻有神命的石棺，赐予其不与殷乱的圣洁封号；第二处紧接记"死，遂葬于霍太山"，以霍太山作为蜚廉的最终安葬地，将宗教祭祀地与祖先陵寝地合一。两处引文共同塑造了霍太山在秦人祖先神话中的神圣位置：既是神命降临之处，也是先祖长眠之所。
