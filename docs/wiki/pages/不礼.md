# 不礼

出现于《史记》**1** 处。

## 生平

田不礼，战国[[赵国]]人，[[公子章]]（[[赵武灵王]]之子，又称安阳君）的亲信党羽。据[[李兑]]对[[肥义]]的预警，田不礼此人"忍杀而骄"，与公子章意气相投、勾结甚深。[[赵惠文王]]即位（前299年）后，公子章不满失位，田不礼与其谋划作乱——即"沙丘之变"（前295年），劫持赵武灵王于沙丘宫，最终导致赵武灵王被困饿死。田不礼在《史记·赵世家》中仅作为李兑预警时被提及，其结局未见详述，当随公子章之乱被诛。
