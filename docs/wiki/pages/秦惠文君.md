# 秦惠文君

## 生平

秦惠文君，即[[秦惠王]]（初称「惠文君」，后称王），[[秦孝公]]之子，继孝公即位。秦孝公卒，秦惠文君即位，当年诛杀[[商鞅]]（商鞅以法治秦有功，但太子犯法时惩处其傅，诸贵族积怨）；惠文君元年，楚、韩、赵、蜀人来朝，是为秦国外交强盛之始。秦惠文君后改称王（秦惠王），在位期间推行张仪连横之策，对外扩张，奠定秦统一六国的战略基础。事见[[005_秦本纪|秦本纪]]。
