# 夜郎侯（夜郎王）

出现于《史记》**1** 处。

## 生平

夜郎侯，古代西南夷政权首领，夜郎国国君，势力在今贵州一带（牂柯江流域）。汉武帝建元六年，唐蒙奉命出使南越，得知牂柯江（今北盘江）可由蜀地经夜郎通达南越，乃请募巴蜀之众，以厚币通夜郎，夜郎侯欣然归附汉朝，汉由此设立犍为郡，推动了西南夷地区的开发。事见[[116_西南夷列传|西南夷列传]]。
