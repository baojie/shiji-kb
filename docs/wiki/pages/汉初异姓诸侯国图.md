# 汉初异姓诸侯国图

楚汉战争结束后，前202年刘邦即皇帝位，先后分封八位异姓诸侯王。这些王国是刘邦在战争期间为拉拢将领所付的政治代价，也是帝国建立后最大的威胁。此后十余年间，刘邦以各种罪名逐一翦除，至其晚年异姓王仅余长沙王吴芮一系。

《史记》在[[008_高祖本纪]]、[[092_淮阴侯列传]]、[[091_黥布列传]]、[[090_魏豹彭越列传]]、[[韩王信卢绾列传]]等篇对各王兴废均有详述。

## 翦除次序

刘邦对异姓王的翦除，以有罪名为掩护，实为削藩：

1. **燕王臧荼**（前202年）— 叛乱被平，改封卢绾
2. **楚王韩信**（前201年）— 伪游云梦，擒韩信贬侯
3. **韩王信**（前200年）— 白登之围后降匈奴
4. **赵王张敖**（前198年）— 贯高刺杀事件株连，废为侯
5. **淮南王英布、梁王彭越**（前196年）— 同年被杀
6. **燕王卢绾**（前195年）— 刘邦临终前最后一次出征
