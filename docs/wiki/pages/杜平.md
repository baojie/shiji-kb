# 杜平

杜平，战国时期秦魏交界地带的城邑，地望约在今陕西东部或山西西部，是秦孝公与魏惠王会盟的历史场所。《秦本纪》载"七年，与魏惠王会杜平"，《六国年表》与《魏世家》均同步记录此事，三书互证，说明杜平会盟是秦魏关系史上一次明确的外交接触节点。

秦孝公时期，商鞅变法初成，秦国国力逐渐提升；魏惠王则是魏国鼎盛时期的君主，两强在杜平相会，是战国中期秦魏势力博弈的外交缩影。《六国年表》记"与秦孝公会杜平，侵宋黄池，宋复取之"，魏惠王同年会秦又伐宋，展示了魏国在战国中期多线外交的战略活跃度；杜平因此成为理解战国秦魏博弈格局的微观地理坐标。
