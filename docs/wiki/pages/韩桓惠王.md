# 韩桓惠王

战国[[韩国]]君主，[[韩釐王]]之子，在位三十四年（约前272—前239年）。在位期间正值[[秦国]]全面扩张，韩国疆土屡遭蚕食：失陉、上党郡守降赵（直接引发[[长平之战]]）、失阳城负黍、城皋荥阳，乃至上党全境覆灭。卒后子[[韩安]]继位，韩安五年韩国被秦所灭。《史记》出现4处。

---

## 生平

### 继位，元年伐燕

韩釐王卒，华阳之战（华阳之战即魏赵联军被秦军击败的那场大战）之后，其子桓惠王继位：

> 〔韩世家〕是岁，韩釐王卒，子[[韩桓惠王]]立。（[[045_韩世家|045-30]]）

桓惠王元年，主动发兵伐燕：

> 〔韩世家〕桓惠王元年，伐燕。（[[045_韩世家|045-31]]）

### 秦国蚕食，韩地持续丧失

韩桓惠王在位的三十四年，是韩国全面衰落的时期，各年大事几乎均为秦国拔城、韩国割地：

| 年份 | 事件 |
| --- | --- |
| 九年 | 秦拔陉，城汾旁 |
| 十年 | 秦击韩于太行，上党郡守以上党郡降赵 |
| 十四年 | 秦拔赵上党，杀马服子（赵括）卒四十余万于长平 |
| 十七年 | 秦拔韩阳城、负黍 |
| 二十二年 | 秦昭王卒 |
| 二十四年 | 秦拔城皋、荥阳 |
| 二十六年 | 秦悉拔上党 |
| 二十九年 | 秦拔韩十三城 |

其中第十年的关键事件尤为重要：

> 〔韩世家〕十年，秦击我於太行，我上党郡守以上党郡降赵。十四年，秦拔赵上党，杀马服子卒四十馀万於长平。（[[045_韩世家|045-31]]）

上党郡守冯亭不愿降秦，将上党十七城转献赵国，由此引发秦赵之间惨烈的[[长平之战]]，赵括（马服子）战死，赵军被坑杀四十余万。这一事件的起点，正是桓惠王在位时韩国的地方官员为保全土地而做出的惊人抉择。

### 卒，子韩安立

桓惠王在位三十四年后卒：

> 〔韩世家〕三十四年，桓惠王卒，子[[韩安]]立。（[[045_韩世家|045-32]]）

[[韩安]]继位仅五年，秦国攻韩，韩安遣[[韩非]]出使，韩非被秦留置并杀，韩国随即亡于秦。

## 历史评价

韩桓惠王在位期间，韩国已无力抵御秦国扩张，所记大事无非割让、失地、守军降敌。《韩世家》对桓惠王的记录极为简略，没有任何外交功绩或政治举措，只是一年年地记录失地——这种"流水账式"的叙事本身，正是韩国末期历史的真实写照。

## 跨章节互见

### 韩世家·六国年表·赵世家：上党事件的多方叙事

上党郡降赵——韩桓惠王十年（约前263年）发生的这一事件，在《史记》三篇中以完全不同的角度呈现。《韩世家》（045-31段）中的记载是"我上党郡守以上党郡降赵"——这是受害者的视角：韩国的地方官员为了不降秦，主动将土地献给了赵国，韩国因此丧失了太行山以西的重要屏障。《赵世家》（043-100段）中的记载则从获利者的角度出发——赵孝成王与平阳君赵豹、平原君赵胜就是否接受韩国上党展开了著名的辩论，赵豹以"圣人甚祸无故之利"反对接受，赵胜则力主接受，赵王最终听从赵胜，接受了上党十七城。《六国年表》（015-r205段）中则以最简短的编年方式记录——"上党守降赵"并列于各国大事中。三篇合观，同一事件在韩世家中是"丧失"（韩国失去领土），在赵世家是"获得"（赵国获得土地），在年表中则是"事实"（中性的记录）。这三种叙事视角的差异，是司马迁互见法的经典案例。参见[[045_韩世家]]（韩国末代——045-28至045-33段中韩国从釐王到桓惠王再到韩安灭亡的完整过程与韩国作为战国七雄最先被灭的必然性）、[[043_赵世家]]（上党受降之争——043-98至043-105段中赵孝成王围绕是否接受上党展开的朝议辩论与赵豹的"圣人甚祸无故之利"警告和赵胜的"两利相权取其重"的策略之间的冲突）、[[073_白起王翦列传]]（长平之战白起的军事部署——073-5至073-10段中白起在长平之战中以少胜多、坑杀赵卒的经过与长平之战在中国军事史上的地位）。

### 冯亭献上党：一个地方官员如何改变战国格局

上党郡守冯亭献地降赵的决定，是战国晚期最具灾难性连锁反应的事件之一。冯亭面临的是两难抉择：秦军已攻至太行山，上党郡与韩国本土的联系被切断。如果降秦，秦将不战而得上党；如果降赵，赵国可能出兵抗秦。冯亭选择将上党十七城献给赵国——这一决定的本意是"以赵制秦"，借助赵国的军事力量来对抗秦国。然而结果完全超出预期：赵国接受了上党，秦国以"赵受韩地"为由大举攻赵，长平之战爆发，赵军四十余万被坑杀，赵国从此一蹶不振。冯亭本人也在长平之战中战死。冯亭的选择直接改变了战国格局——长平之战后，东方六国中唯一能与秦抗衡的赵国被彻底削弱，秦统一天下只是时间问题。从韩桓惠王的视角看，他作为一个丧失对地方控制力的君主，既无法阻止上党降赵，也无能力从这一事件中获益——韩国只是被动地目睹了这一切的发生。参见[[043_赵世家]]（冯亭降赵始末——043-98至043-102段中上党郡守冯亭遣使献地于赵的完整经过与赵国君臣围绕受降展开的激烈争论）、[[073_白起王翦列传]]（长平之战的军事分析——073-5至073-10段中白起以远交近攻、诱敌深入的战略在长平围歼赵军的经过与长平之战在中国军事史上的典范地位）、[[076_平原君虞卿列传]]（赵胜的决策——076-5至076-8段中平原君赵胜力主受上党的立场与赵胜作为赵国决策核心在战国末期的政治角色）。

### 诸侯"流水账"与"为秦作史"：《史记》对战国末期的叙事处理

《韩世家》对韩桓惠王三十四年在位的记录方式——除元年伐燕外几乎全是"秦拔我X"——是一种极具叙事策略的写作手法。司马迁本可以用更详细的文字描述韩国的外交努力或内部政治，但他选择以近乎机械的方式逐年记录失地，营造出一种"压迫感"。这种叙事节奏与秦国在战国末期的扩张节奏是一致的——秦国的扩张不是偶发性的，而是持续的、不可阻挡的。更值得注意的是，《史记》对韩国末期的叙事实际上变成了"秦国的扩张史"——六国世家中对韩、赵、魏、楚、燕、齐末期的记录越来越多地围绕秦国展开。这一现象被后世学者称为"为秦作史"——司马迁虽然为六国各立世家，但到战国末期，这些世家的叙事重心已经从本国转向秦国。韩桓惠王在《韩世家》中三十多年的在位记录，本质上是一部秦国侵韩的年表，而非韩国本身的历史。这种"叙事主体转移"反映了战国末期历史本身的真实走向：秦国已经成为历史的主角，而六国退居被动的"承受者"角色。参见[[045_韩世家]]（韩国灭亡的叙事结构——045-28至045-33段中司马迁在韩世家末尾从"韩国的历史"向"秦国征服韩国的历史"的叙事重心转移与流水账式失地列表的文学效果）、[[006_秦始皇本纪]]（秦统一六国的编年——006-10至006-20段中秦始皇本纪中从十七年灭韩到二十六年灭齐的统一过程与秦本纪对六国灭亡的正面记录与韩世家侧面记录的对照）、[[015_六国年表]]（战国末期的年表结构——015-r200至015-r215段中六国年表在战国晚期从各国并列到以秦为中心的表格变化与司马迁在年表中"以秦系年"的编纂手法）。
