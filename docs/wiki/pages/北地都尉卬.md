# 北地都尉卬

出现于《史记》**1** 处。

## 生平

北地都尉卬，西汉文帝时边将，担任北地郡都尉，负责北地边防。文帝时匈奴大举入侵，卬在抵御匈奴的战斗中被杀，此事引发文帝对边防将帅问题的深切忧虑。冯唐以此为例，向文帝进言如何识别和重用廉颇、李牧式的将帅，说明君主不得因小过而束缚良将手脚。事见[[102_张释之冯唐列传|张释之冯唐列传]]。
