# 都关

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

都关，古地名，位于今山东省菏泽市东部一带，属汉代济阴郡。《史记》中都关作为济水流域地理坐标出现，是鲁西南古代地名网络的组成部分。
