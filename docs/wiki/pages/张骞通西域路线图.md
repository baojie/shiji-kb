# 张骞通西域路线图

汉武帝建元二年（前138年），张骞奉命出使西域，联络大月氏夹击匈奴。出发后不久即被匈奴俘获，在匈奴滞留十余年，仍坚持汉节，后逃脱辗转抵达大宛、康居、大月氏（已徙至大夏北），前126年返回长安。

元狩四年（前119年），张骞再度出使，抵达乌孙，并分遣副使至大宛、康居、大月氏、大夏等地，正式开通了汉朝与西域各国的外交联系，奠定了丝绸之路的基础。《史记·大宛列传》称之为"凿空"。
