# 汉初和亲格局图

前200年，汉高祖刘邦亲征匈奴，轻兵冒进，被冒顿单于四十万铁骑围困于白登山七日，史称"白登之围"。此后汉朝采取和亲政策：每年向匈奴献送公主（实为宗室女子）及丝绸、粮食等财物，换取边境相对安定。

这一格局从高祖延续至文帝、景帝，持续约七十年，直至汉武帝即位后转守为攻，以马邑之谋（前133年）为起点，开启了汉匈之间的全面战争。

## 和亲政策概述

| 时期 | 汉帝 | 主要事件 |
| --- | --- | --- |
| 前200 | 汉高祖 | 白登之围，陈平献计脱围；开始和亲 |
| 前196—前180 | 汉高祖/惠帝 | 每年奉送财物，互市 |
| 前179—前157 | [[汉文帝]] | 和亲继续，冒顿死后匈奴稍衰 |
| 前157—前141 | [[汉景帝]] | 七国之乱后内部整合，继续和亲 |
| 前141—前133 | [[汉武帝]]初期 | 准备反击，马邑设伏 |

**前133年**，汉武帝用王恢之计，在[[马邑]]设伏诱击单于，单于察觉撤退，和亲政策宣告终结。
