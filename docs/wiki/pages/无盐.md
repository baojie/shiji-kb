# 无盐

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

无盐，古地名，位于今山东省东平县一带，属汉代东平国。《项羽本纪》涉及无盐，是楚汉战争时期齐地战事的地理坐标之一。无盐亦以历史传说中"钟离春"（无盐女）的故事而知名，是齐宣王相传用人唯贤的著名历史故事发生地。

无盐，古代鲁西地区的城邑，地处今山东东平县一带，是楚汉战争时期齐地争夺中出现的军事地理坐标。无盐以其地名见于项羽本纪及汉初功臣侯者年表，是楚汉相争时期山东地区战场的历史见证。

《项羽本纪》载楚汉争霸期间，无盐作为齐地军事据点出现于战场叙事之中，是汉军在齐鲁地区推进过程中的地名坐标。《高祖功臣侯者年表》另以无盐记录汉初在此地封侯的功臣信息，揭示无盐在汉代行政体系中延续为侯国封地的地理传承。无盐因战国时期"无盐女"钟离春的传说而在中国文化史上具有独特的象征意涵——以才华辅佐君主的无盐女成为以内在品德超越外貌的女性典范，虽此传说不见于《史记》正文，但无盐地名因此在历史记忆中承载了额外的文化重量。
