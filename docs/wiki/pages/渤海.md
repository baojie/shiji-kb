# 渤海

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·冀州（页26-27）（见右栏）。

渤海，古代华北东部内海（今渤海湾），西汉冀州东缘，《史记》多处以渤海为东方地理坐标，是汉代地理认知中的东方水域边界。

渤海，位于中国东北的半封闭内海，为齐、燕、辽东之间的天然海域，是汉代海上军事行动的重要通道。渤海之名在汉代既指海域本身，也兼指渤海郡（今河北沧州一带），两者常因上下文而有所区分。

《孝武本纪》载汉武帝巡幸海滨，"临渤海，将以望祠蓬莱之属，冀至殊庭焉"，将渤海视为通往神仙世界的入口，折射出武帝晚年对方仙道的热切期盼。《朝鲜列传》则记汉廷对朝鲜用兵时，"遣楼船将军杨仆从齐浮渤海"，水师由齐地渡渤海北上，与左将军荀彘出辽东的陆路形成夹击之势，共讨右渠。两处并见，渤海既是武帝寻仙幻梦的寄托之所，又是汉军跨海东征的实际战场，一虚一实，共同构成汉武时代扩张版图的地理坐标。
