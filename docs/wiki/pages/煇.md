# 煇

## 生平

煇，秦国蜀侯，任期内起兵叛秦。秦昭王六年（前301年前后），蜀侯煇在蜀地举兵反叛秦国，秦昭王命大将[[司马错]]入蜀平乱，诛杀蜀守煇，重新平定蜀地。与此同时，庶长奂伐楚斩首二万，泾阳君质于齐。煇之叛是秦国并蜀后地方势力反抗中央的典型案例，司马错再度入蜀平叛，进一步巩固了秦国对蜀地的控制。事见[[005_秦本纪|秦本纪]]、[[015_六国年表|六国年表]]。
