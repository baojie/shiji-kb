# 历阳

历阳，古县名，位于今安徽和县一带，是楚汉战争末期垓下之战后汉军向东南追击、扫荡楚地残余势力的关键据点。历阳还因项羽头号谋士范增被封为"历阳侯"而在《史记》中留有历史印记。

《项羽本纪》以"历阳侯范增"一语点出范增的封号，揭示了项羽对这位头号谋士的高度倚重，历阳之名因此与楚汉相争中最重要的谋臣之一紧密相连。《樊郦滕灌列传》则记灌婴在垓下之战后"下东城、历阳，渡江，破吴郡长吴下"，以历阳为汉军南渡长江、扫定江南的出发据点，两书合观，历阳成为楚汉战争末期项羽阵营的政治地标与汉军追歼的军事节点。
