# 千秋

出现于《史记》**7** 处。

## 生平

《史记》中「千秋」指多位同名汉代功臣侯国传承者：①敬侯谔千秋，汉高祖六年封，以从汉高祖定诸侯有功受封，传简侯嘉、顷侯应等多代，侯国延续；②另一侯千秋，汉文帝后四年封，中元年因伤人获罪被免；③元鼎五年（前112年）因酎金之罪国除的某千秋侯。诸「千秋」均为汉代功臣侯国传承中的世袭侯主，以年表人物身份留存于《史记》。事见[[018_高祖功臣侯者年表|高祖功臣侯者年表]]、[[019_惠景间侯者年表|惠景间侯者年表]]。
