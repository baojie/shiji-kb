# 逄丑父

出现于《史记》**6** 处。

## 生平

逄丑父，春秋齐国大夫，鞍之战中为齐顷公驾车，危急时与齐顷公互换衣服，使齐顷公得以逃脱晋国追捕，以忠勇著称，晋军感念其忠义将其释放。事见[[032_齐太公世家|齐太公世家]]。
