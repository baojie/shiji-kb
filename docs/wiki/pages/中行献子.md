# 中行献子

出现于《史记》**1** 处。

## 生平

中行献子，即**荀偃**，姓荀，氏中行，谥献，晋国六卿之一，[[中行氏]]的代表人物。[[晋灵公|晋]]强盛时期，[[晋国]]联合诸侯压制周边。齐灵公二十七年（约前556年），晋使中行献子率军伐齐，齐师大败，齐灵公弃城逃入临淄，[[晏婴]]讽谏齐灵公无勇，晋兵随后围临淄，此后齐国接受晋国盟约。中行献子在《史记·齐太公世家》中作为晋国主将出现，侧面展示了晋国对齐国的战略压制。
