# 利苍

西汉初年**长沙国丞相**，封**轪侯**，长沙王吴臣的丞相。因功封侯，食邑在南方。

利苍之妻**辛追**及其子**利豨**（[[豨]]）的墓葬即著名的**马王堆汉墓**（湖南长沙），1972年出土，文物极为丰富，辛追夫人遗体保存完好。《史记》中见[[018_高祖功臣侯者年表]]。
