# 醿里子

出现于《史记》**1** 处。

## 生平

醿里子，战国秦国大夫，秦惠文王之弟，以善权谋著称，与张仪共事，助秦兼并诸侯，封于醿里，称醿里子，是秦国重要的宗室政治家。事见[[005_秦本纪|秦本纪]]。
