# 叔武

出现于《史记》**1** 处。

## 生平

叔武，[[周武王]]之弟，武王分封诸弟时受封于成（今地不详）。《史记·管蔡世家》载武王封诸弟：管叔鲜封管，蔡叔度封蔡，叔旦封鲁，叔振铎封曹，叔武封成，叔处封霍，康叔封、厓季载（冉季载）尚幼未封。叔武具体事迹史书未详载。事见[[035_管蔡世家|管蔡世家]]。
