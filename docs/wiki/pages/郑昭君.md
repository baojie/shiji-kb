# 郑昭君

春秋时期郑国国君。任用所宠爱的[[徐挚]]为相，导致国乱，上下不亲，父子不和。后[[大宫子期]]进言，以[[子产]]为相，子产治郑二十六年，郑国大治。

## 生平

郑昭君，春秋郑国国君，任用宠臣徐挚为相，导致国政混乱，上下不亲。后大宫子期进言，以子产为相，子产治郑二十六年，郑国大治。事见[[119_循吏列传|循吏列传]]。
