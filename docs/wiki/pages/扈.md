# 扈

扈，春秋时期诸侯会盟之地，地处中原，具体位置一般认为在今河南原阳一带，是晋国主导诸侯会盟时常用的聚集场所。扈邑在晋文公、成公时代多次作为各国诸侯共商天下大事的会盟地点，是晋国霸业鼎盛时期的外交地理标志。

《晋世家》两度以"会诸侯于扈"记载同一事件：晋成公七年，晋楚争霸正酣，晋成公在扈地召集诸侯会盟，以彰显晋国的盟主地位。陈国因惧楚而拒不赴会，晋国随即使中行桓子讨伐陈国，并趁机救援郑国、与楚交战，击败楚师——这一系列外交与军事连锁反应，正是以扈地会盟为起点展开的。扈因此成为《史记》晋国霸权叙事中，盟约政治与武力威慑相互交织的历史地理节点。
