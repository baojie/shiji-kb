# 城南

城南，在《史记》中主要指汉代长安城南郊，是汉武帝初年儒臣谋划建立明堂的选址之地。明堂是周代礼制中天子会诸侯、颁布政令的神圣建筑，儒生以"议古立明堂城南，以朝诸侯"为汉武帝重建礼乐秩序的象征性举措。

《封禅书》详载，汉武帝元年招致儒生，赵绾、王臧"欲议古立明堂城南"，然而此议因窦太后崇黄老、压制儒术而告败——"使人微伺得赵绾等奸利事，召案绾、臧，绾、臧自杀"，城南明堂计划随之夭折。《项羽本纪》则以完全不同的面貌出现"城南"，记项羽"夜击坑秦卒二十馀万人新安城南"，是战争暴行的历史地标，同一词语在不同语境中承载截然不同的历史意义。
