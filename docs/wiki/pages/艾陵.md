# 艾陵

艾陵，春秋晚期齐国城邑，位于今山东泰安莱芜一带，是吴王夫差北伐中原的关键胜战之地。《吴太伯世家》载，"吴王不听（伍子胥之谏），遂北伐齐，败齐师於艾陵"，夫差无视越国威胁，执意北上，艾陵之战大败齐师，是吴国军事扩张的顶峰时刻。

然而艾陵的胜利也是吴国覆灭的前奏。《春申君列传》引此为教训："吴之信越也，从而伐齐，既胜齐人於艾陵，还为越王禽三渚之浦"，夫差在艾陵大胜后北上与晋争霸于黄池，越王勾践乘虚袭吴，吴国由此走向覆灭。《仲尼弟子列传》记子贡游说吴国后，"吴王果与齐人战於艾陵，大破齐师"，印证了子贡外交策略的成功，但也呼应了伍子胥对夫差不顾后患的深刻警告。
