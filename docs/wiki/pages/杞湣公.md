# 杞湣公

## 生平

杞湣公，春秋杞国君主，杞隐公之子，继杞隐公而立。在位十五年，适逢楚惠王灭陈国。十六年，弟杞哀公弑杀杞湣公自立，是为杞哀公，杞国随即内乱更甚。杞湣公是杞国历史中被弑篡位的悲剧君主，其在位时代与楚惠王扩张势力灭陈同期，处于楚国压迫下杞国日益衰微的末期历史中。事见[[036_陈杞世家|陈杞世家]]。
