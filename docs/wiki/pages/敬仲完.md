# 敬仲完

出现于《史记》**1** 处。

## 生平

敬仲完，即田完，春秋齐国大夫，陈（田）氏入齐始祖，陈厉公之子，逃亡入齐后被封大夫，谥敬仲。敬仲完后人在齐国经营数代，最终田氏取代姜氏成为齐国主宰，史称「田代姜齐」。事见[[046_田敬仲完世家|田敬仲完世家]]。
