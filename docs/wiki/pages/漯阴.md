# 漯阴

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

漯阴，古地名，位于今山东省齐河县东南一带，因处漯水之北（古代水之北为阴）而得名，属汉代济南郡。《曹相国世家》载曹参领兵经此地区作战，是楚汉时期山东战场的历史坐标之一。
