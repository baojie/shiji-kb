# 巫

巫，楚国西部战略要地，位于今重庆巫山一带，是三峡咽喉，控扼巴蜀入楚的水陆通道。《楚世家》载，秦昭王时"秦大将白起攻楚，取巫、黔中之郡，拔鄢郢，东至竟陵"，白起攻楚先夺巫地，再下郢都，巫的失守是楚国灾难的前奏，楚顷襄王被迫"东徙治於陈县"。《六国年表》亦载"秦拔我巫、黔中"，与此印证。

《春申君列传》中，黄歇在上书秦昭王说秦伐楚之害时，明确提及"秦已前使白起攻楚，取巫、黔中之郡"，以此说明秦国进攻楚国的代价，强调巫与黔中的战略价值。《货殖列传》则从经济地理角度描述"江陵故郢都，西通巫、巴，东有云梦之饶"，巫地作为楚地向西连通巴蜀的交通枢纽，在商业流通上也具有重要地位。
