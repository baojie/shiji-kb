# 杞哀公

出现于《史记》**4** 处。

## 生平

朱英，战国末期楚国人，曾为春申君黄歇门客，以直言进谏著称。曾劝春申君提防李园，预言李园必将趁机作乱。春申君不听，后李园果然刺杀春申君于楚考烈王丧期，朱英事先见机出奔，避免了灾祸。其预见被视为先秦谋士洞察力的典范案例。事见[[078_春申君列传|春申君列传]]。
