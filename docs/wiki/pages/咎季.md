# 咎季

出现于《史记》**1** 处。

## 生平

咎季，春秋晋国大夫，狐偃（子犯）一族，晋文公重要谋臣之一。晋文公六年（前631年），赵衰成子、栾贞子、咎季子犯、霍伯相继去世，四位辅佐晋文公的重臣同年离世，朝政随后由赵盾（赵衰之子）接掌。事见[[039_晋世家|晋世家]]。
