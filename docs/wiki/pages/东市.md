# 东市

东市，汉代长安城东侧的市集，同时也是朝廷公开处决重罪犯人的法场，因"斩东市"之句多次见于《史记》刑杀叙事而广为人知。东市作为帝都的商业与司法双重空间，在汉代政治文化中具有特殊象征意涵——从市集到刑场，权力的威慑在公共空间中得以展示。

《袁盎晁错列传》载，汉景帝采纳窦婴、袁盎建议，"令晁错衣朝衣斩东市"，晁错未及申辩即被腰斩于东市，成为汉初削藩政争中最具戏剧性的政治冤案。《日者列传》则载"司马季主者，楚人也，卜於长安东市"，以占卜为业的方士同样出现于东市，揭示其作为各色人物汇聚的都市公共场所的另一面。同一空间承载了权贵的死亡与市井的日常，东市因此成为《史记》呈现汉代长安城市生活复杂层次的重要地点。
