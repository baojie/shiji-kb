# 辕固生

汉景帝、汉武帝时齐国经学博士，《诗经》齐派（齐诗）传授者。性情刚直，敢于直言。

## 生平

辕固生以治《诗》为博士，深受重用，但因性格耿直屡遭险境。

**与窦太后争辩**：[[窦太后]]好黄老之术，辕固生在廷上直言"老子不过是家人言耳"，触怒窦太后，被罚入猪圈与野猪搏斗。[[汉景帝]]怜其直言，偷授利刀，辕固生一刀刺中猪心，窦太后不得不作罢。

**晚年再征**：[[汉武帝]]初即位，以贤良再次征召辕固生，此时辕固生已年逾九十，诸谀儒嫉毁说他年老，终被罢归。临行时辕固生对同时被征的[[公孙弘]]说："务正学以言，无曲学以阿世。"

## 历代评价

**太史公评价（叙述视角）**：司马迁在《儒林列传》中以生动细节记录了辕固生的两件大事：与窦太后争黄老之辩而被送入猪圈（景帝暗授利刃解救），以及晚年被武帝再征时对公孙弘的临别忠言"无曲学以阿世"。太史公对辕固生的叙述充满尊重：通过景帝"暗中给刀"这一细节，暗示景帝内心支持辕固生的直言；而"无曲学以阿世"则成为全篇的道德高点，太史公将其置于辕固生的晚年，隐然以此作为对汉代经学界媚世风气的批判。

**汉代史学定论**：汉代儒学将辕固生视为坚守正学、不向外戚权势低头的儒学典范。班固《汉书·儒林传》将辕固生置于"齐诗"传授谱系的源头，称赞其以九十高龄再被征召、仍坚守"正学"立场，是汉代儒林精神的体现。汉儒对辕固生的定论是：直而不阿，是汉初儒学精神尚未完全被官学化腐蚀时的遗存；他对公孙弘的告诫，被后世视为对汉代经学"曲学阿世"（以学问媚权贵）风气的精准预警。

**后世主要争议**：后世对辕固生评价高度正面，争议较少，但有一点引发了讨论：辕固生对窦太后的抵抗（斥黄老为"家人言"）究竟是真正的学术争鸣，还是政治上的莽撞之举？一派认为辕固生的直言体现了汉代儒家争夺意识形态话语权的历史大势，他的抵抗推动了儒术最终取代黄老的进程；另一派则认为辕固生的言辞过于激烈，在政治敏感时期公然冲撞太后并不明智，他的幸免是偶然（景帝援手），而非必然，他的直言有时更接近性格上的刚愎而非大勇。

**历史影响 / 特殊符号意义**："无曲学以阿世"成为中国学术史上最著名的治学格言之一，被历代学者援引为坚守学术独立、拒绝媚俗媚权的精神宣言。辕固生的形象因此超越了其作为"齐诗"传授者的具体历史地位，成为中国知识分子传统中"以直道立身、不以曲媚进"的精神原型。他与窦太后的猪圈斗争，也成为景帝时代儒道之争最具戏剧性的历史场景，在后世文学和史学中被反复征引。
