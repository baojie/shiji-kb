# 伯虔

出现于《史记》**1** 处。

## 生平

伯虔，孔子弟子，字子析，比孔子年少五岁（一说少孔子五十岁）。《史记·仲尼弟子列传》载其名字及与孔子的年龄关系，具体事迹不详，仅以名字记入弟子列。事见[[067_仲尼弟子列传|仲尼弟子列传]]。
