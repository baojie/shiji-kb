# 林虑公主

出现于《史记》**1** 处。

## 生平

杞文公，春秋杞国君主，继杞孝公而立，以兄弟相传方式袭位。《陈杞世家》记杞国历代君主传承：杞文公继杞孝公，卒后传弟杞平公，平公传子杞悼公，悼公传子杞隐公。杞文公是杞国中后期君主传承链的普通一环，史书无具体事迹记载，是《史记》以简洁格式记录春秋小国诸侯传承的典型案例。事见[[036_陈杞世家|陈杞世家]]。
