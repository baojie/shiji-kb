# 乔如

出现于《史记》**2** 处。

## 生平

乔如，春秋北方**鄋瞒**族的长狄（长翟）之君。**鄋瞒**是春秋时期频繁侵扰中原各国的北方部族，乔如兄弟数人各领一支。[[鲁宣公]]十一年（前598年），鲁国于咸邑击败翟人，俘获乔如，[[富父终甥]]用舂杵击其咽喉，再以戈杀之，将其首级埋于子驹之门，以此献功于[[叔孙宣伯|宣伯]]。鄋瞒家族数代皆遭诸侯斩获：缘斯被宋军俘杀，乔如弟棼如在晋灭路时被俘，荣如被齐王子城父所获，季弟简如则被卫人所获，可见鄋瞒一族在与中原诸侯的战争中几近覆灭。
