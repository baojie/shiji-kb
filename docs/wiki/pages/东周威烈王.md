# 东周威烈王

东周君主，名午，在位（前425—前402年）。威烈王最重要的历史时刻是前403年：承认[[赵籍]]（赵侯）、[[魏斯]]（魏侯）、[[韩虔]]（韩侯）三家正式为诸侯，史称"三家分晋"。从此，原晋国领地正式分裂为赵、魏、韩三个独立诸侯国。

《资治通鉴》以前403年为起始元年，正是以周威烈王承认三晋为历史分水岭。司马光认为此举是"周道之衰，孔道之堕"的集中体现——天子不守礼法，大夫越位称侯，由此礼崩乐坏进入不可逆的轨道。《史记》中，这一事件标志着战国七雄格局的正式形成，周天子此后仅剩象征意义的名号。

## 历代评价

**太史公评价（叙述视角）**：《史记·周本纪》中，周威烈王以承认韩、赵、魏三家为诸侯的历史性决定出现——"命韩、魏、赵为诸侯"一事在《史记》叙述中标志着三家分晋的完成。太史公以简洁笔法记录这一历史时刻，隐含对周室无奈确认既成事实、礼法已名存实亡的批判。

**汉代史学定论**：《资治通鉴》以周威烈王二十三年（前403年）作为起始点，将此年命韩赵魏为诸侯视为战国开始的标志——这一史学决断将威烈王的行为定性为"礼崩乐坏的终极确认"。汉代儒学框架下，威烈王被批评为以天子之命为强者合法化，是周室威信彻底丧失的象征。

**后世主要争议**：司马光在《资治通鉴》中对"三家分晋"有著名的道德论议，认为威烈王承认三家是"正名分"原则的彻底失败，深刻影响了后世对威烈王历史地位的评价。现代史学则更关注三家分晋的历史必然性，认为威烈王的"承认"只是对已成事实的补认，批评其"失职"过于苛求。

**历史影响 / 特殊符号意义**：周威烈王是《资治通鉴》选取的历史起点人物，他的那一道命令使三家分晋具有了法理正当性，成为战国时代正式开幕的政治标志。"威烈王命三家为诸侯"成为中国历史上最重要的政治时刻之一，司马光对此事的评论影响了一千年的历史解读传统。
