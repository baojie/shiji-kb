# 朝

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

朝（朝歌），古地名，位于今河南省淇县一带，属汉代魏郡。商纣王都于朝歌，是殷商王朝晚期的政治中心。周武王伐纣后，朝歌成为卫国的重要城市。《史记·殷本纪》及《卫康叔世家》中多次提及朝歌，是商末周初历史的核心地理坐标。

朝，汉代侯国地名，见于汉初功臣侯者年表及惠景间侯者年表，是西汉初年封侯制度下设立的诸侯小国之一。朝邑侯国在汉初随功臣封赏制度而建立，其传承与废绝记录见于《史记》各侯者年表之中。

《高祖功臣侯者年表》记载"朝"侯的世系传承，载其后代"元狩三年，侯朝为济南太守，与成阳王女通，不敬，国除"，因犯不敬之罪而被除国；《惠景间侯者年表》另记"轵侯国"世系中"侯朝为常山王，国除"，两处"朝"分属不同侯国，均以除国作结，共同呈现了西汉中期大批侯国因罪或政治原因被撤销的历史规律。朝邑的兴废是汉代诸侯封建制度从鼎盛走向收缩的历史缩影。
