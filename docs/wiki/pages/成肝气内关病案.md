# 成肝气内关病案

仓公医案之一。患者齐侍御史成自诉头痛。仓公切脉后，私告成弟昌："此病疽也，内发於肠胃之间，後五日当鍮肿，後八日呕脓死。"成果然如期死亡。

病因：得之"饮酒且内"（饮酒兼房事过度）。脉象：肝气浊而静，此为"内关之病"。仓公引脉法："脉长而弦，不得代四时者，其病主在於肝。和即经主病也，代则络脉有过。"分析如下：脉贲（搏动强劲）而代（间歇），病从酒与房事中来；少阳脉初代时，每一分的进展对应病情的一个阶段——"少阳初关一分，故中热而脓未发；及五分，则至少阳之界；及八日，则呕脓死。"病势逐步上行：热气上熏阳明，腐烂流入络脉，络脉结发则烂解交络，最终热气上达头部而头痛。仓公将脉诊与预后精确量化，展示了汉代经络理论与脉法在临床预测中的应用。
