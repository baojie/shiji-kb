# 吴公子光

出现于《史记》**1** 处。

## 生平

吴公子光，即[[阖闾]]，春秋吴国公子，吴王寿梦之孙，诸樊之子。吴王僚在位时，公子光暗中谋划夺位。专诸刺王僚时，吴王僚兵败，公子光趁机取位，自立为吴王，即吴王阖闾。公子光任用伍子胥、孙武，使吴国霸业达于顶峰。事见[[031_吴太伯世家|吴太伯世家]]。
