# 姬嘉

出现于《史记》**1** 处。

## 生平

姬嘉，战国周室末代诸侯，东周之末，周王室已极度衰微，秦昭王灭西周公国后，周室余烬尚有东周君（东周小国）。姬嘉或为东周君家族的最后一代传承人，秦庄襄王灭东周后，周祀断绝，姬嘉代表了周室姬姓最后的传承。事见[[004_周本纪|周本纪]]。
