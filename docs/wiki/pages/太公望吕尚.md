# 太公望吕尚

出现于《史记》**1** 处。

## 生平

"太公望吕尚"为《史记·齐太公世家》开篇对[[吕尚]]的完整称呼，其中"太公望"是后世尊称（周文王所赠，意为"吾太公望子久矣"），"吕尚"是本名。详细生平参见 → **[[吕尚]]**。

据《齐太公世家》，吕尚祖先曾任四岳，佐禹治水有功，虞夏之际封于吕或申，姓姜。吕尚本人出自东海，晚年辅佐[[周文王]]和[[周武王]]，策划灭商，牧野之战后受封齐国，开创齐国八百年基业。
