# 五河

## 地理位置

五河泛指多条河流汇合之处，见于[[117_司马相如列传|司马相如列传]]中'朅渡九江而越五河'，是汉赋中的地理意象，今安徽省蚌埠市五河县一带，地处淮河中游。
