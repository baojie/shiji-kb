# 项燕

楚国末代名将，[[项羽]]之祖父（一说叔祖）。以勇武著称，是秦灭楚过程中楚国抵抗力量的象征，也是陈胜起义时借以号召的旗帜。

## 生平

秦始皇二十二年（前225年），秦将[[李信]]率二十万军伐楚，项燕反击，大败李信，斩秦军七都尉，是秦统一战争中少数大败秦军的战役。

秦始皇随后起用老将[[王翦]]，率六十万大军再伐楚。王翦坚壁不战，待楚军疲惫东撤之际举兵追击，于蕲南大破楚军。项燕兵败，或战死或自杀（史载不一），楚国随之灭亡（前223年）。

陈胜吴广起义时，以"楚虽三户，亡秦必楚"相号召，打出"张楚"旗号，并以"项燕"之名笼络楚地人心，可见项燕在楚人心中的崇高地位。

## 历代评价

**太史公评价（叙述视角）**：司马迁在《项羽本纪》中以简练叙述呈现项燕的历史形象——楚国最后的名将，在秦灭楚的战役中与李信的先期胜利相对，最终被王翦的大军所压倒而败亡自杀。太史公通过"王翦果代李信击荆"的历史记载，将项燕的覆亡定格为秦统一六国历史进程的关键一役，并通过陈胜起义时"王侯将相宁有种乎""诈称公子扶苏、项燕"的细节，揭示项燕在楚地民心中的深远影响。

**汉代史学定论**：汉儒将项燕定性为战国末期楚国抵抗秦国的最后英雄，认为其先败李信（展示楚军仍有战力）、终败于王翦（历史必然）的双重表现，是秦统一不可逆性的有力注脚。班固延续《史记》框架，将项燕的覆亡视为楚国独立精神的最后终结，其名字被陈胜吴广借用，成为楚地反秦情绪的精神符号。

**后世主要争议**：后世对项燕的评价较为统一（英雄末路的悲剧形象），争议主要集中于：项燕在军事上的能力究竟几何——他败李信、但终败王翦，这究竟说明他是优秀将领（只是遇到了更强的对手），还是其军事才能本就有限？部分学者认为李信的失败更多是战略冒进而非项燕的战术卓越，因此过度评价项燕的军事才能有所偏颇。

**历史影响 / 特殊符号意义**：项燕是楚汉文化中"抵抗精神"的象征人物之一，"楚虽三户，亡秦必楚"与项燕之名的并列，成为楚文化中不屈抗争精神的集体记忆。项燕作为项羽的祖父（有此传说），使他与楚汉战争的结局产生了跨越历史的叙事连接：秦灭楚时项燕以死相抗，其孙项羽终于复仇灭秦，构成了令后人感叹的历史轮回。
