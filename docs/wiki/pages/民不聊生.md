# 民不聊生

**释义**：百姓无法生存

出自《史记》，见于[[012_孝武本纪|孝武本纪]]。

## 史记原文

出自 [[012_孝武本纪|孝武本纪]] （012-54）：

> **原文出处**：人无聊生
> **上下文**：之礼，则有司存焉。 ::: ::: 赞 [54] 孝武纂极，四海承平。 志尚奢丽，尤敬神明。 坛开八道，接通五城。 朝亲五利，夕拜文成。 祭非祀典，巡乖卜征。 登嵩勒岱，望景传声。 迎年祀日，改历定正。 疲秏中土，事彼边兵。 日不暇给，**人无聊生**。 俯观嬴政，几欲齐衡。 :::
