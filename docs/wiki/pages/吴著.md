# 吴著

出现于《史记》**1** 处。

## 生平

吴著，西汉诸侯王，淮阳靖王，为淮阳国再置后的封王。《史记·汉兴以来诸侯王年表》记其为靖王，元年有载。淮阳国在西汉历史上多次置废，吴著为其传承中的一代王。事见[[017_汉兴以来诸侯王年表|汉兴以来诸侯王年表]]。
