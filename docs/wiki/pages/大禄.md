# 大禄

西汉时乌孙[[昆莫]]（乌孙王）之中子，强将，领骑万余。其兄太子早死，太子嘱昆莫立太子之子[[岑娶]]继位，昆莫允诺。大禄因争位失败而怒，率其昆弟及所部叛，谋攻岑娶及昆莫，导致乌孙国势力三分。

## 生平

大禄，西汉时[[乌孙]]昆莫（王）之中子，骑兵万余人，是乌孙国中较强的势力之一。其兄太子早卒，临死嘱昆莫立太子之子[[岑娶]]继位，昆莫从之。大禄以为王位应归己（中子），争位失败后大怒，率其同族兄弟及部众另立一方，谋攻岑娶和昆莫。乌孙国被迫三分：昆莫以万余骑让给岑娶（以应太子之托），以万余骑给大禄（以安抚争位者），自领万余骑。这一分裂局面是[[张骞]]出使乌孙时已存在的现实，造成乌孙无法整合力量与汉朝正式结盟。
