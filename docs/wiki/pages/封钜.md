# 封钜

出现于《史记》**1** 处。

## 生平

封钜，上古传说人物，黄帝时期的臣子，传说中黄帝问道于封钜，是黄老道家追溯源流时常引用的上古贤者之一。事见[[001_五帝本纪|五帝本纪]]。
