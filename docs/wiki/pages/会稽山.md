# 会稽山

会稽山，位于今浙江绍兴一带，是古越国的核心山地，也是传说中大禹会诸侯、计功受封的圣地。会稽山在《史记》中以勾践兵败被围、忍辱求和的历史典故而声名显赫，是春秋末期吴越争霸这一历史大戏的核心地理舞台。

《越王勾践世家》以会稽山为吴越争霸的地理中枢，记载越王勾践在会稽山一带遭吴军围困、忍辱求和的历史转折，以及其后卧薪尝胆、复国雪耻的艰难历程，会稽山成为越国由亡转兴的精神地标。《孔子世家》则记"吴伐越，堕会稽，得骨节专车"，从孔子论奇骨之说的角度侧记吴军攻破会稽的史实，以博物奇谈的笔法为会稽山的历史沧桑留下别样注脚，两书合观，会稽山在《史记》中兼具政治史实与文化象征的双重意义。
