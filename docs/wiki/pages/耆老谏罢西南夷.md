# 耆老谏罢西南夷

司马相如奉使西征、宣喻西南夷归附之际，蜀郡"耆老大夫荐绅先生之徒二十有七人"聚集而来，就外表庄重，辞罢之后进言谏阻——这是《司马相如列传》中以散文对话形式呈现的一幕政论场景：博学长者与汉廷使者就西南夷政策正面交锋。

耆老谏辞的核心立场是"羁縻勿绝"：天子对待夷狄，理应维持松散羁縻关系，而非穷兵黩武。耆老历数三条反对理由：一、"今罢三郡之士，通夜郎之涂，三年於兹，而功不竟"——三年征伐无功，士卒疲弊；二、"万民不赡，今又接以西夷，百姓力屈"——民力已竭，不堪再征；三、"邛、筰、西僰之与中国并也，历年兹多，不可记已。仁者不以德来，彊者不以力并"——此地久与中国并存，德不能化、力不能并，强征不可行。耆老以"割齐民以附夷狄，弊所恃以事无用"作结，批评以内地民力换取无用边地的政策逻辑。值得注意的是，这篇对话并非历史上真实发生的请愿，而是司马相如刻意构造的文学框架："相如欲谏，业已建之，不敢，乃著书，籍以蜀父老为辞，而己诘难之，以风天子"——他借耆老之口提出反对意见，再借使者之口加以驳斥，最终引导读者（和皇帝）接受扩张立场。耆老言辞因此成为相如为汉武帝西南夷政策辩护的文学铺垫。
