# 廪丘

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

廪丘，古地名，位于今山东省鄄城县北部一带，属汉代东平国或济阴郡范围。《齐太公世家》涉及廪丘，是齐国在鲁西南的历史据点之一。楚汉战争期间，彭越在此地区以游击战牵制楚军，是楚汉争霸中鲁西南战场的重要地理背景。

廪丘，战国时期齐国城邑，位于今山东鄄城一带，是齐魏之间兵家争夺的边境要地。《齐太公世家》与《田敬仲完世家》载，齐宣公五十一年，田会"以廪丘反"，这是田氏代齐前夕齐国内部重要的叛乱事件，体现出田氏势力在齐国城邑中的深度渗透与姜姓公室的衰落。

廪丘在战国中后期多次易手。《六国年表》载，韩昭侯时"伐东周，取陵观、廪丘"，廪丘被魏国夺取；赵国亦曾参与争夺，"败齐于灵丘"后继续在鲁西一带扩张。至战国晚期，《六国年表》又记"秦拔我廪丘"，廪丘最终被秦军攻取，反映了廪丘在齐、魏、赵、秦之间反复争夺的战略地位。
