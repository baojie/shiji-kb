# 南边

南边，泛指南方边境，在《史记》中兼有两种用法：一指鲁国南部边界，是春秋鲁国对齐进攻的方向；二指汉初诸侯国在南方的边郡，是汉廷削藩政策下诸侯国版图约束的地理表述。

《鲁周公世家》载孔子在鲁执政期间"伐齐南边"，鲁军出击齐国南境，是孔子为鲁君谋求振兴的短暂尝试。《汉兴以来诸侯王年表》则记吴楚七国之乱后汉廷调整诸侯国疆界，"吴、淮南、长沙无南边郡"，以削去诸侯国南边之郡的方式限制其兵力财源，南边成为汉廷控制南方诸侯版图的制度性地理标志。
