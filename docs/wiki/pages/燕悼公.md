# 燕悼公

## 生平

燕悼公，春秋燕国君主，继燕惠公之后即位。燕惠公在位时多宠姬，欲废大夫立宠姬之子宋，大夫们共同杀宋，惠公惧而出奔齐国。四年后，齐高偃赴晋请求联合伐燕，晋平公许之，齐晋联军伐燕，将惠公送回复位，然惠公归至燕地即死，燕国随立悼公。燕悼公是燕国春秋历史中政局动荡后新立的君主，处于燕惠公宠姬乱政引发内外危机之后的复稳过渡时期。事见[[034_燕召公世家|燕召公世家]]、[[014_十二诸侯年表|十二诸侯年表]]。
