# 成阳

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

成阳，古地名，位于今山东省菏泽市东北一带，属汉代济阴郡。《史记》中成阳是帝尧相关传说的地理坐标，相传帝尧陵墓在成阳（今山东鄄城）。成阳地区地处济水流域，是上古圣王传说地理网络的重要节点，与帝舜活动的雷泽相邻。

成阳，古地名，今山东荷泽郓城东一带，是梁宋地区的重要城邑，传说尧帝曾在此建都，故《货殖列传》以"昔尧作成阳"追溯其上古历史渊源，并描述当地"犹有先王遗风，重厚多君子，好稼穑"的民风，以地域风俗体现圣王德化绵延不断的文明叙事。

《高祖本纪》记刘邦早期起义军"乃道砀至成阳，与杠里秦军夹壁，破二军"，说明成阳是秦末楚汉战争初期反秦势力东进的重要通道；《曹相国世家》亦载[[曹参]]军队在成武南破敌，成阳周边地区是楚汉战争早期的重要战场区域。成阳兼具传说圣地与军事要冲的双重身份，在《史记》中承载了古今叠合的历史厚度。
