# 若木

出现于《史记》**2** 处。

## 生平

若木，上古传说人物，夏代先祖费昌之子，伯益之后裔，见于秦本纪世系记录，为夏代诸侯，史书载之简略。事见[[005_秦本纪|秦本纪]]。
