# 孝成

出现于《史记》**1** 处。

## 生平

孝成，即汉成帝刘骜，西汉皇帝，在位期间（前33年—前7年）荒于酒色，宠幸赵飞燕姐妹，外戚王氏势力极盛，政务几乎全由王莽叔父王凤等把持，为西汉末年外戚专权的顶峰。事见[[011_汉兴以来将相名臣年表|汉兴以来将相名臣年表]]。
