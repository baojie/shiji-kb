# 宿

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

宿，古代城邑，位于今安徽省宿州一带，楚汉战争时期是淮北地区的军事节点。《史记》多次记载汉军在此地攻略，见于建元以来侯者年表等相关篇目。


宿，春秋时期卫国边境城邑，位于今安徽宿州一带，是卫国政治动乱中流亡贵族的重要庇护之所。《卫康叔世家》载，卫献公得罪孙文子与甯惠子，二子愤而"如宿"，卫国公室与权臣的矛盾由此激化，献公最终被逐，在外流亡十二年。其后殷公"封孙文子林父於宿"，宿作为封地与流亡地在卫国政治格局中反复出现。

宿亦是卫国蒯聩之乱的舞台。《卫康叔世家》载，赵简子欲助蒯聩入卫，"卫人发兵击蒯聩，蒯聩不得入，入宿而保"；太子蒯聩流亡在宿期间，又与孔悝之母图谋复位，引发了孔子弟子[[子路]]殒命的历史事件。宿作为卫国政治风云中一处关键地名，见证了春秋晚期卫国公室内乱的多个回合。
