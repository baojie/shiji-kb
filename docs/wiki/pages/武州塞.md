# 武州塞

武州塞，汉代北方边境关隘，约在今山西朔州一带，是汉与匈奴之间的重要通道，也是马邑之谋中匈奴入境的路线节点。《匈奴列传》载，汉元光元年马邑之谋时，单于"以十万骑入武州塞"，深入汉境，距马邑不远时发现异常而退走，汉军埋伏三十余万于马邑旁谷，终因计谋败露而功亏一篑。

《韩长孺列传》从护军御史大夫韩安国的视角记述马邑之谋的策划与失败，武州塞作为匈奴入境的关口在叙事中居于核心位置；马邑之谋是汉武帝从和亲政策转向主动出击匈奴的历史转折点，武州塞因此成为汉匈关系从防守转为进攻这一战略大变局的地理见证。
