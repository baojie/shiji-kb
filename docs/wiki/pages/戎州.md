# 戎州

戎州，卫国境内戎人聚居的城邑，具体位置约在今河南一带，是春秋时期华夏诸侯国内夷狄杂居的典型案例。戎人被纳入卫国版图，但与华夏居民之间的文化隔阂仍十分明显，君主对戎人的歧视性言论直接引发了严重的政治后果。

《卫康叔世家》记载，卫庄公登城时俯瞰戎州，轻蔑地说"戎虏何为是"，以"戎虏"之辞侮辱戎州居民。此言传出后，戎州人勃然大怒，联合[[赵简子]]共同发兵攻打庄公，迫使庄公出奔流亡。《十二诸侯年表》亦载"庄公辱戎州人，戎州人与赵简子攻庄公，出奔"，说明轻率的种族歧视话语引发了内外联动的政治危机，是春秋晚期民族关系与诸侯权力博弈交织的典型事件。
