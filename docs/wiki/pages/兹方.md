# 兹方

兹方，战国时期楚国西部的重要城邑，是楚国在战国中期遭蜀国入侵而失陷的边境据点，其沦陷直接促使楚国修筑扞关以御蜀。兹方的历史价值在于揭示了战国时期西南强国蜀的军事扩张已深入楚地腹地这一鲜为人知的历史现实。

《楚世家》载楚肃王四年"蜀伐楚，取兹方，於是楚为扞关以距之"，以兹方的失陷为直接起因，记录了楚国被动修筑防御工事的历史细节，揭示了蜀国军事实力之强足以迫使楚这一南方大国调整战略布局。《六国年表》亦以"蜀伐我兹方"简洁记录这一事件，从史表视角与《楚世家》互相印证，兹方因此成为战国中期西南地缘博弈、蜀楚边疆争夺这一历史面向的核心地理符号。
