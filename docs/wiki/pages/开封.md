# 开封

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

开封，古地名，位于今河南省开封市一带，属汉代河南郡。战国时期开封附近是魏国领土，大梁（魏都）即在开封附近。《史记·魏世家》及《秦始皇本纪》中，秦水攻大梁、魏国灭亡的叙事与这一地区密切相关。汉代开封作为河南郡属县，是黄河南岸的重要城邑。

战国魏国及汉代重要城邑，位于今河南开封，是中原东部的交通要冲，汉代开封县所在地，孝景本纪与傅靳蒯成列传均涉及开封的历史记录。
