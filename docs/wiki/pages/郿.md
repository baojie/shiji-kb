# 郿

郿，秦汉时期关中西部县城，位于今陕西眉县一带，扼守渭水北岸，地处关中平原与秦岭山地的交界地带。郿以出生了秦国名将白起而在《史记》中留下历史印记，也是楚汉之争中汉军平定三秦的战场之一。

《白起王翦列传》开篇即载"白起者，郿人也"，以籍贯郿人定位这位秦国最具传奇色彩的将领，使郿与白起的赫赫战功永久相连。《绛侯周勃世家》记周勃随汉王还定三秦，"还下郿、频阳"，郿作为汉军攻略关中的具体目标见于军功记录。《樊郦滕灌列传》亦记还定三秦时汉军在郿地附近的相关战事，郿地作为关中战场的重要节点反复出现。三书合观，郿既是白起故里，又是秦汉之际反复争夺的关中西部战略要地。
