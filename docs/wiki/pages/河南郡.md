# 河南郡

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

河南郡，汉代行政区划，治所在雒阳（今河南省洛阳市），是西汉首都长安以东最重要的行政区之一。河南郡地处中原核心，东汉定都雒阳时，河南郡成为全国政治中心所在。《史记》中河南郡出现于汉代地理与行政叙事，是京畿之外最关键的中原郡县之一。
