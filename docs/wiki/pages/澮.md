# 澮

## 地理位置

澮位于今河南省东南部至安徽北部一带，战国时期为魏韩交界的争夺地点，见于[[044_魏世家|魏世家]]'翟败我于澮'及[[045_韩世家|韩世家]]'魏败我澮'，同一地名两度出现于不同战役记载。


澮，战国时期战场地名，地望约在今山西南部或河南北部魏韩交界地带，是战国前期魏国遭遇翟人和韩国军事失利的具体地点。澮之战在《史记》中以双方记录互证的方式出现，展示了战国史事叙述的多视角特征。

《魏世家》载魏武侯时"翟败我于澮"，游牧力量翟人在澮地击败魏军；《韩世家》则载"九年，魏败我澮"，魏国又在澮地击败韩军，同一地名先后见于魏被翟败、魏败韩两次不同战役，揭示了澮作为战国初期多方争夺的兵家必争之地的地理价值。
