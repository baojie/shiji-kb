# 京索

京索，楚汉战争时期河南中部两地的合称，约在今河南荥阳、郑州一带，是刘邦与项羽对峙的核心战场之一。京索之间的军事僵局是楚汉争霸中期汉方以守代攻、积蓄反攻力量的战略阶段，是《史记》楚汉叙事中不可或缺的地理坐标。

《萧相国世家》载，"汉三年，汉王与项羽相距京索之间"，以京索对峙为背景，揭示刘邦在前线胶着时期对后方大臣信任度的微妙动态。《太史公自序》概括韩信功绩时称"楚人迫我京索，而信拔魏赵，定燕齐，使汉三分天下有其二"，以京索之困对比韩信在北方的辉煌战绩，京索因此成为楚汉战争战略格局中"正面僵持、侧翼突破"这一战略结构的地理见证，是理解楚汉相争最终走势的核心地理符号。
