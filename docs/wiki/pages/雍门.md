# 雍门

雍门，古代城门之名，在《史记》中指向两处不同的城市：一处为战国齐国都城临淄的雍门，位于临淄城西侧（今山东省淄博市临淄区），是田氏代齐历史叙事的见证地；另一处为秦都咸阳城的雍门，位于咸阳城西侧（今陕西省咸阳市），是秦始皇大兴土木仿建诸侯宫室的西端边界。两处"雍门"地名相同，性质相似（均为城西之门），但地望迥异，须依上下文加以区分。

《齐太公世家》记田常弑齐简公事件后，旧臣大陆子方"以公命取车於道，出雍门"，以临淄雍门为逃亡路线，在政变混乱中出城离去——此处雍门作为临淄城门，承载着齐国政治剧变后流亡者出走的历史瞬间；《秦始皇本纪》则记秦统一过程中仿建诸侯宫室："秦每破诸侯，写放其宫室，作之咸阳北阪上，南临渭，自雍门以东至泾、渭，殿屋複道周阁相属"，以雍门作为咸阳宫殿群的西端起点，揭示秦始皇将战利品建筑化、以展示天下一统成就的宏大象征工程。
