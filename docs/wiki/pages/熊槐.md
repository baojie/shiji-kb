# 熊槐

## 生平

熊槐，即[[楚怀王]]，战国楚国君主，继楚宣王之后即位，《六国年表》以「楚怀王熊槐元年」记其即位。楚怀王在位期间，被秦国[[张仪]]多次欺骗（商於之地骗局、背盟伐楚），外交屡遭挫败；晚年应秦昭王之邀赴武关会盟，被扣押不得归，后客死秦国，楚人怜之。「熊槐」是楚怀王的本名，以此见于《史记》年表的历史记录。事见[[015_六国年表|六国年表]]。
