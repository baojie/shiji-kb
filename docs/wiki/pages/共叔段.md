# 共叔段

春秋郑国人，[[郑庄公]]之弟，母武姜所偏爱。庄公元年（前722年）封于京，号太叔，因持续扩张势力，最终与母武姜密谋叛乱，被郑庄公所败，史称"共叔段之乱"。

## 共叔段之乱

郑庄公即位后，其母武姜请求将制邑封给段，庄公以制邑险要拒绝，改封段于京（号太叔）。太叔在京扩充兵力，又兼并西鄙、北鄙，大夫[[祭仲]]屡次劝谏庄公，庄公表示要"多行不义必自毙"。

后太叔联合武姜约定内外夹攻，庄公得知密谋，先发制人讨伐太叔，太叔兵败出奔，最终逃至共（故名共叔段）。此事件是春秋早期郑国王室内部矛盾的典型案例。

## 生平

共叔段，春秋郑国人，郑庄公之弟，母武姜所偏爱。庄公即位后封共叔段于京，号太叔。共叔段持续扩张势力，与母武姜密谋叛乱，史称「共叔段之乱」，最终被郑庄公所败，出奔共地。此事引发郑庄公发誓「不及黄泉，无相见也」，以及后来「掘地见母」的故事。事见[[042_郑世家|郑世家]]。

## 历代评价

**太史公评价（叙述视角）**：司马迁在《郑世家》中以"郑伯克段于鄢"记录了郑庄公对弟弟共叔段的政治清除。太史公的叙述沿袭《春秋》的简练书法，并未深入分析郑庄公的心机；但通过"庄公寤生，惊姜氏，故名曰寤生，遂恶之"等背景交代，隐隐揭示了这场兄弟相争背后深藏的母亲偏爱与家族矛盾，共叔段的失败有其被设计的成分。

**汉代史学定论**：汉代经学对"郑伯克段于鄢"的解释聚焦于《春秋》书法。《公羊传》认为"克"字表明郑庄公将弟弟当作敌国来讨，有失兄弟之道；《谷梁传》和《左传》对庄公的批评更直接——庄公养成共叔段的恶，再以讨贼之名消灭，是"以段为必败而乃可以有辞焉"，用欺谋手段杀弟，违背了儒家的亲亲之义。汉儒基本定论：共叔段固有罪（贪地逾礼），但郑庄公的处置方式（故意纵容待其自毙）亦不符合君子兄长之道。

**后世主要争议**：后世对共叔段的评价分歧在于他的责任程度——一派认为共叔段贪心不足、不知收敛，侵占领地、蓄兵谋反，罪有应得，是典型的"以妾为妻之乱"引发的政治祸源；另一派则认为共叔段更多是母亲[[武姜]]偏爱和庄公故意纵容双重因素下的牺牲品——他从未真正有实力与庄公抗衡，他的"谋反"在很大程度上是在被放纵至无路可回的情况下被迫爆发的，责任不应完全由他承担。这一争议根植于儒家"兄弟之道"与"君臣礼法"的结构性张力。

**历史影响 / 特殊符号意义**："郑伯克段于鄢"是《春秋》经文中被注释最多的段落之一，"克"字的书法成为中国经学史上讨论《春秋》微言大义的标志性案例。共叔段与郑庄公的故事成为中国古代政治中"阴谋养患、以合法名义消灭异己"的经典叙事，也是讨论家族政治与国家权力交织、偏爱与礼法冲突时不可回避的历史原型。
