# 楚熊渠

## 生平

楚熊渠，楚国早期重要君主，熊杨之子。在[[周夷王]]时期（王室衰微、诸侯相伐），楚熊渠甚得江汉间民心，兴兵伐庸、杨粤，进至鄂地，并将三子分别封为句亶王、鄂王、越章王——此为楚国史上第一次僭号称王之举。后[[周厉王]]专横，熊渠惧被天子讨伐，撤去王号。楚熊渠以军事扩张和短暂僭号称王成为楚国历史上第一个正式称王尝试的君主，为后来楚武王正式称王奠定历史先例。事见[[040_楚世家|楚世家]]。
