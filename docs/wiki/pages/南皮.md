# 南皮

南皮，汉初重要地名，位于今河北沧州一带，是楚汉之际[[陈馀]]失势后的流亡之地。《秦楚之际月表》载，"陈余亡居南皮"，赵王歇留守赵国，陈馀因不愿归附项羽而流亡南皮。项羽分封诸侯时，"因环封三县"以酬其素日贤名，高祖则明确"封成安君陈馀河间三县，居南皮"，南皮作为封地附着于陈馀的政治命运。

南皮见证了张耳与陈馀从"刎颈之交"到彻底决裂的全过程。《张耳陈馀列传》详载两人关系破裂的经过：陈馀认为项羽厚待张耳而轻视自己，"张耳与馀功等也，今张耳王，馀独侯"，由此积怨日深，最终以武力驱逐张耳，开启了汉初赵地的一系列战争。南皮作为陈馀流亡与封居之地，是这段历史悲剧的空间载体。
