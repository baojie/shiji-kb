# 华无伤

出现于《史记》**1** 处。

## 生平

华无伤，秦末楚汉之际齐国将领。韩信攻齐前，齐使华无伤、田解率军守历下以抵御汉军。郦食其赴齐说降，田横与齐王田广以为汉将不会进攻，遂罢历下守战之备，纵酒欢宴，准备与汉媾和。汉将韩信趁此率军东进攻齐，历下守军措手不及。事见[[094_田儋列传|田儋列传]]。
