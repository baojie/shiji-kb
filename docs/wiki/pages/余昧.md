# 余昧

出现于《史记》**1** 处。

## 生平

余昧，春秋吴国国君，[[吴王寿梦]]第四子，继诸樊、余祭之后即位为吴君。吴王余昧在位时承兄余祭之业，延续吴国兴盛之势，其后传位其弟季札（或欲传季札而季札让），由此引发后续吴王位的传承纷争。事见[[014_十二诸侯年表|十二诸侯年表]]。
