# 陆梁侯国

陆梁侯国，汉代列侯封国。高祖九年（前198）三月丙辰，[[桑]]以功受封，食邑户数不详。

## 始封

**初封侯**：[[桑]]（共侯）
**分封时间**：高祖九年（前198）三月丙辰

**封侯原因**（年表原文）：
