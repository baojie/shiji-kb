# 兰池

兰池，秦代宫苑水域，位于咸阳附近，是秦始皇游幸之地。兰池因秦始皇夜出微行时在此遭遇盗贼的惊险事件而留名史册，成为秦代帝王巡幸安全隐患与社会动荡的历史见证。

《秦始皇本纪》载"始皇为微行咸阳，与武士四人俱，夜出逢盗兰池，见窘，武士击杀盗，关中大索二十日"，以兰池为事发地点，记录了帝王遭遇盗贼、武士格杀护驾的戏剧性事件，事后引发关中大规模搜捕，折射出秦代社会底层的动荡。《孝景本纪》则记汉景帝时期"殖兰池"，以汉家对旧秦宫苑水域的经营续用，揭示了秦汉之际帝都周边地理格局的延续，兰池由此成为秦汉帝都历史记忆叠加的地理节点。
