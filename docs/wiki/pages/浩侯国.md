# 浩侯国

浩侯国，汉代列侯封国。元封四年（前107）正月甲申，[[王恢]]以捕车师王功受封，食邑户数不详。

## 跨章节互见

### 捕车师王、破楼兰姑师：浩侯王恢的西域功业

浩侯王恢的封侯源于他在武帝西域经略中的关键贡献。《大宛列传》详细记载了其功绩背景：张骞通西域后，汉朝与西域的交通日益频繁，但楼兰、姑师"当空道，攻劫汉使王恢等尤甚"。王恢作为武帝派往西域的使者，"数使，为楼兰所苦"——多次出使被楼兰劫掠困扰。

王恢向武帝报告楼兰、姑师的威胁，武帝派遣从骠侯赵破奴率数万骑发动了针对姑师和楼兰的军事行动。赵破奴"虏楼兰王，遂破姑师"之后，王恢以佐战之功和"捕得车师王"的特殊贡献，于元封四年（前107）正月甲申封浩侯。车师（姑师）是西域北道的关键城邦国家，控制着丝绸之路的要冲。王恢"捕得车师王"为汉朝在西域建立了重要的军事和外交据点，"於是酒泉列亭鄣至玉门矣"——汉朝的亭障防线从酒泉延伸到了玉门关，标志着西域经营进入新阶段。参见[[123_大宛列传]]、[[从骠侯国]]/[[浞野侯国]]（赵破奴对照）。

### "封凡三月"：汉代最短命的侯国

浩侯国在《史记》年表中留下了一个极为罕见的标注："封凡三月。"从元封四年正月甲申封侯到同年四月因罪国除，浩侯国仅存在了约三个月，可能是《史记》年表中寿命最短的侯国之一。

国除原因是"坐使酒泉矫制害，当死，赎，国除"。王恢在出使酒泉时"矫制"（假托天子诏命），造成了某种损害，依律当死。但他选择"赎"（以钱赎死），但侯国随之被撤销。"矫制"在汉代是大罪——出使官员假传圣旨等同于欺君。王恢以中郎将身份多次出使，熟悉天子使者的权责边界，却仍然犯了这一错误，暗示了西域使者群体在武帝朝"奉使骄恣"的普遍现象——大宛列传也批评这些人"侵盗币物""妄言无行"。浩侯国的短暂存在是西域外交冒险的一个缩影：一次成功即可封侯，一次失误就国除身危。参见[[020_建元以来侯者年表]]、[[123_大宛列传]]。

### 两个王恢：大行王恢与浩侯王恢的同名异人

"王恢"这个名字在武帝朝对应两个截然不同的人物。大行王恢（大行令）是武帝初期主管外交的官员，以"马邑之谋"闻名——他提议在马邑设伏引诱匈奴单于，结果谋泄，匈奴退去。武帝怒其无功，王恢自杀。

浩侯王恢则是中郎将、西域使者，活跃于元鼎、元封年间，多次出使西域，因楼兰劫掠之事促成赵破奴西征，佐战有功封浩侯，因矫制害国除。两人事迹截然不同：一个活跃于北边对匈奴战场（大行王恢，约前133年马邑之谋），一个活跃于西域对城邦国家外交（浩侯王恢，约前107年封侯），前后相距约三十年。在阅读《史记》时需注意区分这两个同名的武帝朝王恢。参见[[108_韩长孺列传]]（大行王恢）、[[123_大宛列传]]（浩侯王恢）、[[110_匈奴列传]]。本页名称义参见大行王恢与浩侯王恢。
