# 芷阳

芷阳，地名，位于今陕西蓝田县一带，靠近骊山北麓，是秦国的重要陵墓区。《秦本纪》载，秦孝文王（安国君）之母宣太后"薨，葬芷阳郦山"；悼太子（安国君为太子时的质子）死于魏国后，也"归葬芷阳"，显示芷阳是秦王室的皇家葬地之一。

楚汉战争前夕，[[刘邦]]从鸿门宴出逃，"从郦山下，道芷阳间行"，以间道抄近路脱身，芷阳因此成为这段惊险逃脱路线上的地理节点。汉代，芷阳乡地界被用作汉文帝[[刘恒]]的陵墓用地，史载"以芷阳乡为霸陵"，即后世著名的西汉霸陵所在，芷阳因此在秦汉两朝均为皇家陵区。
