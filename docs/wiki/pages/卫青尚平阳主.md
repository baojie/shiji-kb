# 卫青尚平阳主

> 平阳主寡居，当用列侯尚主。左右皆言大将军可。主笑曰"此出吾家，常使令骑从我出入"。左右以其姊为皇后、三子为侯劝之，主乃许。武帝诏卫将军尚平阳公主。

**主要人物**：卫青、平阳主

## 事件经过

平阳主寡居，当用列侯尚主。左右皆言大将军可。主笑曰"此出吾家，常使令骑从我出入"。左右以其姊为皇后、三子为侯劝之，主乃许。武帝诏卫将军尚平阳公主。
