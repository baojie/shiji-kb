# 赖

## 地理位置

赖位于今山东省菏泽市一带，春秋时为齐国南境城邑，见于[[032_齐太公世家|齐太公世家]]：'晋赵鞅伐齐，至赖而去'，是晋国攻齐南境时止步之地。
