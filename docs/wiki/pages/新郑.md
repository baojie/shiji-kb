# 新郑

## 地理位置

新郑位于今河南省郑州市新郑市一带，古为[[郑国]]都城，汉代属颍川郡，秦始皇二十二年（前225年）新郑反叛，昌平君被迁徙，见于[[006_秦始皇本纪|秦始皇本纪]]：'新郑反，昌平君徙於郢'。
