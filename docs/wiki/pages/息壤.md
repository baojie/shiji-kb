# 息壤

息壤，战国时期秦国与甘茂立约之地，具体位置说法不一，一般认为在今陕西咸阳一带。"息壤之盟"是战国外交史上君臣之间以地为证、相互立誓的典型案例，其地名因这一历史事件而留存于《史记》的叙事记忆之中。

《樗里子甘茂列传》详载息壤盟约始末：秦武王欲使甘茂伐韩拔宜阳，甘茂顾虑出兵期间国内谗言，援引昔日苏秦进谏燕王之典，请秦武王于息壤立誓，以王命保障他坚持伐韩的正当性。此后甘茂果然遭遇朝中阻力，秦武王"召甘茂"欲罢兵，甘茂以"息壤在彼"提醒君王之誓，武王乃增兵助战，最终攻克宜阳。息壤之约是战国外交博弈中君臣信义与政治智慧交锋的缩影，也是甘茂以历史典故保全自身策略的地理见证。
