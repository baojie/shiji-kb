# 代谷

代谷，代地的山谷地带，是冒顿单于集结匈奴主力的屯兵之所，也是汉高祖北征时轻进冒险、最终遭遇白登之围的地理序幕。代谷之名因与平城之围相连而成为汉初对匈奴战略失误的地理标志。

《平津侯主父列传》载御史成进谏不可击匈奴，"夫匈奴之性，兽聚而鸟散，从之如搏影"，高帝不听，"遂北至於代谷，果有平城之围"，以代谷为高帝冒进的最后决策地。《韩信卢绾列传》亦记"闻冒顿居代谷"，高帝遣使视察后轻信"可击"，遂引军至平城被围。代谷因此成为汉初不可不察的历史警示地理符号。
