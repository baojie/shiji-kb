# 密畤

密畤，秦国早期在渭南设立的祭祀青帝的郊祠，是秦国国家祭祀体系的重要组成部分。《封禅书》载"秦宣公作密畤於渭南，祭青帝"，密畤由秦宣公创建，专祀与东方相应的青帝，是秦国在正式建立四畤之前的早期礼制探索。《秦本纪》记"四年，作密畤。与晋战河阳，胜之"，密畤的建立与秦的军事扩张同步，揭示了祭祀建设与国力提升之间的内在关联。

《十二诸侯年表》以年表形式记录"四年作密畤"，与各国事件并列，将秦的礼制建设纳入诸侯并争的时代背景；密畤是秦国五畤祭祀体系的前身之一，在《史记》封禅祭祀叙事中标志着秦国从边陲诸侯向礼制强国转型的历史节点。
