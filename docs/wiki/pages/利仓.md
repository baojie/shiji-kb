# 利仓

出现于《史记》**1** 处。

## 生平

利仓，西汉惠帝时官员，曾任长沙国相，以功封轪侯（今湖南长沙马王堆附近），食邑七百户。惠帝二年四月庚子受封，其子豨于惠帝三年袭封，孙彭祖于景帝十六年袭封。1972年湖南长沙马王堆出土的西汉贵族墓葬，墓主即利仓家族，出土大量文物帛书，为研究汉初社会的重要考古发现。事见[[019_惠景间侯者年表|惠景间侯者年表]]。
