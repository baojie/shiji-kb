# 周吕侯国

周吕侯国，汉代列侯封国。高祖六年（前201），[[吕泽]]以功受封，食邑户数不详。

## 始封

**初封侯**：[[吕泽]]（令武侯）
**分封时间**：高祖六年（前201）

**封侯原因**（年表原文）：

> 以吕后兄初起以客从，入汉为侯。还定三秦，将兵先入砀。汉王之解彭城，往从之，复发兵佐高祖定天下，功侯。

## 封地地望

封国名**周吕**，其地望待考。

**三家注地望说明**：

**锚文**（007_项羽本纪）：是时吕后兄周吕侯
- **正义**：苏林云：「以姓名侯也。」晋灼云：「外戚表周吕令武侯泽也。吕，县名。封于吕，以为国。」颜师古云：「周吕，封名。令武，其谥也。苏云『以姓名侯』，非也。」

**锚文**（007_项羽本纪）：为汉将兵居下邑，
- **正义**：《括地志》云：「宋州砀山县本下邑县也，在宋州东一百五十里。」按：今下邑在宋州东一百一十里。
