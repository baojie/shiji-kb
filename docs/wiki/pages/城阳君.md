# 城阳君

出现于《史记》**1** 处。

## 生平

城阳君，战国时期人物，具体身份不详，或为韩国或魏国封君。秦昭襄王十七年，城阳君与东周君相继入朝秦国，秦以垣地改置蒲阪、皮氏，彰显秦国的地缘扩张。城阳君入朝象征其政治依附于秦。事见[[005_秦本纪|秦本纪]]。
