# 西汉侯国分布图

楚汉战争结束后，刘邦大封功臣。《史记·高祖功臣侯者年表》记录了一百四十余位功臣的封侯信息。主要功臣封侯集中于中原和东方，从今河南、山东、河北、山西一带，形成拱卫汉都长安的屏障。

功臣侯国与刘姓诸侯王国并存，构成汉初双重格局：大封国（诸侯王）由刘姓宗室把持，小侯国（列侯）则赐给功臣，但其封地相对较小，仅以食邑税收养侯，无法独立成为军事势力。
