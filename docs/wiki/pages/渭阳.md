# 渭阳

渭阳，渭水之北的地区，今陕西咸阳一带，因地近渭水而得名。《六国年表》载，义渠曾侵入秦国腹地"至渭阳"，使秦国一度面临来自西北方向的威胁。汉代，渭阳成为祭祀五帝的重要礼制场所。

汉文帝时期，方士新垣平以"望气"之术进言，称"长安东北有神气，成五采"，建议在渭阳设立五帝庙。[[封禅书]]详载，汉文帝采纳建议，"作渭阳五帝庙，同宇，帝一殿，面各五门，各如其帝色"，文帝亲临"霸渭之会"以郊祀仪式告拜五帝。然而新垣平其后被人举发诈伪，遭"诛夷"，渭阳五帝祠随之降格，改由祠官按时行礼，文帝不再亲往，礼制热情大为降温。
