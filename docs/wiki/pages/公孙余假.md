# 公孙余假

出现于《史记》**1** 处。

## 生平

公孙余假，春秋卫国官员，[[卫灵公]]之近侍。孔子流亡至卫国期间，在卫国大夫[[蘧伯玉]]家住宿，每月领俸粟六万。不久有人在卫灵公面前诽谤孔子，卫灵公随即命公孙余假对孔子实施监视——"一出一入"，意即跟随孔子进出，暗中监视其行动。孔子察觉此举是针对自己可能构成威胁的处置，恐遭罪责，遂离开卫国。公孙余假在史书中以这一监视行为出现，折射出孔子在各国游历中所受到的猜忌与压迫。
