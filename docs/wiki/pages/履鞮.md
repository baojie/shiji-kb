# 履鞮

出现于《史记》**3** 处。

## 生平

履鞮，春秋晋国宦官，晋献公之宦，伺候晋公子重耳出逃，后追杀重耳未果，重耳（晋文公）复国后，履鞮求见，晋文公以其「狗马之心」拒之。事见[[039_晋世家|晋世家]]。
