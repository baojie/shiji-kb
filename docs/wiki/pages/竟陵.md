# 竟陵

竟陵，战国末期楚国江汉之间的城邑，约在今湖北天门一带，是白起攻楚战役中秦军向东推进的最远端，也是秦设南郡的东部边界。《白起王翦列传》载白起"攻楚，拔郢，烧夷陵，遂东至竟陵"，秦军以竟陵为东部极点，将整个南郡地区纳入秦国版图；《六国年表》同步记录"白起击楚，拔郢，更东至竟陵，以为南郡"，竟陵是南郡设立的东部边界坐标。

《春申君列传》在描述楚顷襄王东迁时再度提及竟陵，"拔鄢郢，东至竟陵，楚顷襄王东徙治於陈县"，三书共同将竟陵定格为白起攻楚最大战略成果的地理标志，揭示了秦国自此控制长江中游的战略意义；竟陵之役不仅是楚国亡国进程的重要节点，也是《史记》中"白起一役定荆楚"这一叙事主题的地理终点。
