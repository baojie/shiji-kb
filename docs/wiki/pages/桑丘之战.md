# 桑丘之战

**战国时期**战役，约发生于[[韩宣惠王]]八年（前325年）。韩国将领**[[韩举]]**在桑丘与齐、魏联军交战，兵败，韩举战死。桑丘地望不详，今或在今山东境内。

《史记》见[[043_赵世家|赵世家]]、[[045_韩世家|韩世家]]、[[015_六国年表|六国年表]]共3处。
