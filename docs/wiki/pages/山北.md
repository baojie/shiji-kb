# 山北

山北，古代对太行山或恒山以北地带的地理泛称，是汉初政治地理格局调整和战国魏国历史叙事中的重要方位词。山北在《史记》中既以汉高祖分封诸王的边疆地理参照而出现，也以魏国遭受秦国蚕食的失地描述见于记载。

《高祖本纪》载刘邦"分赵山北，立子恆以为代王，都晋阳"，以山北为代国的地理所在，揭示了汉初皇权以分封子弟拱卫中央、管控边疆的政治布局。《魏世家》则以"所亡於秦者，山南山北，河外河内，大县数十，名都数百"描述魏国大量失地的历史悲剧，两书合观，山北从汉代代国的封邑范围到战国魏国的失地叙述，呈现了这一地理方位在秦汉历史叙事中的双重历史面向。
