# 九道

九道，上古大禹治水工程中疏通的九条主要道路，是禹在平治洪水、划定九州之后为连通天下而修辟的交通系统。九道与九州、九泽、九山并列，共同构成《禹贡》地理体系中人文秩序化自然空间的组成部分。

《河渠书》记大禹治水功业时明确列举："通九道，陂九泽，度九山"，将九道纳入禹疏川治水的整体工程体系，与疏通河川、积蓄湖泽、整治山岳并列，揭示了禹以交通贯通天下的治国理念。《夏本纪》亦详述禹治水过程中"行山表木，定高山大川"，九道在此背景下成为禹整合山川地理、建立天下秩序的重要一环。两书共同构建了九道在古代地理体系中"以道联通天下"的历史意义，是《史记》中禹治天下叙事的重要地理概念节点。
