# 零陵

零陵，古代地名，本指舜帝的葬地，即今湖南省永州市一带（九嶷山区，今宁远县境内），后发展为秦汉郡县地名。"零陵"之名源于《史记·五帝本纪》："舜南巡狩，崩于苍梧之野，葬于江南九疑，是为零陵"——舜帝南巡时去世，葬于九嶷山（又称九疑山），其地因此得名"零陵"，成为中国历史上最具象征意义的帝王葬地之一，与大禹相会的会稽、黄帝的桥山等并列为上古圣王的陵寝圣地。汉代设零陵郡，治所即在此地，司马迁亦曾南游至此，探禹穴、窥九疑。

《五帝本纪》将零陵与舜帝的生平联系在一起："践帝位三十九年，南巡狩，崩于苍梧之野。葬于江南九疑，是为零陵"，以"是为零陵"一语揭示地名的来历，将葬礼事件与地名形成紧密相连；《南越列传》则出现于汉武帝元鼎五年（前112年）平定南越的军事行动中，其中一路汉军"出零陵，或下离水，或柢苍梧"，零陵作为军队出发地，揭示其在汉代岭南军事路线中的战略位置。两处引文一记舜帝的葬地之名，一记汉军的出兵据点，共同呈现零陵从上古圣迹到汉代军事重镇的历史演变。
