# 丹阳

丹阳，楚国早期都城，位于今湖北秭归一带（一说丹水、淅水交汇处）。据《史记·楚世家》，[[周成王]]封[[熊绎]]于楚蛮，"居丹阳"，为楚国建国之本，是楚先公筚路蓝缕、开疆拓土的起点，历经数代后迁都郢。

前312年，[[丹阳之战]]爆发，秦将庶长章率军在丹阳大破楚军，"虏其将屈匄，斩首八万"，随即攻取汉中六百里之地，置汉中郡。此役是战国后期楚国遭受的最惨重败绩之一，极大削弱了楚国的战略纵深，[[楚怀王]]随后二度兴师伐秦，又败于蓝田，楚国由此每况愈下。
