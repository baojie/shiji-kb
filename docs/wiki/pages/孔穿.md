# 孔穿

出现于《史记》**1** 处。

## 生平

孔穿，战国儒家学者，孔子六世孙，孔子后裔。孔穿曾与公孙龙辩论名实问题，公孙龙以「白马非马」等名辩名闻天下，孔穿往见公孙龙，不能屈服于其论，表明儒家对于名家诡辩的批评立场。事见[[076_平原君虞卿列传|平原君虞卿列传]]。
