# 知文子

出现于《史记》**1** 处。

## 生平

知文子，春秋晋国大夫，荀氏知氏族长，晋国六卿之一，参与晋国重要政治决策，知氏一系后来在三家分晋中被韩赵魏三家联合消灭（战国初年）。事见[[039_晋世家|晋世家]]。
