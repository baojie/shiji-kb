# 邛

邛，西南夷重要部族聚居地，邛都在今四川西昌一带，历史上是中原政权向西南扩展的前沿目标之一。《史记》中"邛"多与"筰"（今四川雅安一带）并称，总称"邛筰"，代表蜀地以西的少数民族地区。

汉武帝时期，[[司马相如]]奉命出使西南夷，说服邛、筰等君长归附汉朝，史载"定筰存邛，略斯榆，举苞满"，使邛都纳入汉代行政版图，比南夷诸国更早内附。[[117_司马相如列传]]中也记录了蜀地耆老反对开拓西南夷的意见，以"邛、筰、西僰之与中国并也，历年兹多"为据，认为强行扩张难以成功。[[123_大宛列传]]则以"邛"为司马迁年轻时游历西南的路线地名之一。
