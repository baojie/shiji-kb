# 子皙

出现于《史记》**1** 处。

## 生平

子皙，即公孙黑（一说），春秋郑国贵族，字子皙，与子产、子南皮同时代郑国贵族。子皙貌美，郑国贵族徐吾犯之妹同时为子皙和公孙楚（子南皮）求婚，引发郑国一段曲折的婚姻纠纷，记载于左传和史记注疏。事见[[042_郑世家|郑世家]]。
