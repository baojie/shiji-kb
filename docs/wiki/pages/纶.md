# 纶

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

纶，古地名，位于今河南省禹州市西南一带，属汉代颍川郡。纶城在春秋时期属郑国领地，是郑国在颍河上游的重要据点。《史记》中纶出现于与郑国及颍川地区相关的叙事，是颍水上游古代地名网络的组成部分。
