# 宋缪公

出现于《史记》**1** 处。

## 生平

宋缪公，春秋宋国君主，即宋穆公，名和，谥穆（缪）。宋穆公临终传位于兄宣公之子与夷（殇公），以守信义著称。国人皆赞穆公「能以国让」，孔子称许此举合乎仁义。事见[[038_宋微子世家|宋微子世家]]。
