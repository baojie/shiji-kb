# 宅皋狼

出现于《史记》**1** 处。

## 生平

宅皋狼，秦国远祖，即孟增，周孝王时被周王赐居皋狼，为秦氏族谱中的重要人物。宅皋狼之子蜚廉，蜚廉之子恶来，是秦国先祖谱系的关键节点。事见[[005_秦本纪|秦本纪]]。
