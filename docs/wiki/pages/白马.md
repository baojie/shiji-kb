# 白马

白马，即白马津，黄河上的重要渡口，位于今河南滑县东南，是古代东西交通和军事南北渡河的关键渡口之一。[[089_张耳陈馀列传]]载，武臣响应陈胜起义，"从白马渡河，至诸县，说其豪杰"，以白马津为突破点，在赵地迅速扩展反秦势力。

《高祖本纪》载刘邦西进时"西与秦将杨熊战白马，又战曲遇东，大破之"，是楚汉战争前期汉军攻取中原的重要战役。[[070_张仪列传]]曾提及"守白马之津"作为秦国东出战略的一部分。白马津扼黄河天险，历代均为兵家必争之地，《史记》中屡见其名，折射出它在战国末期至楚汉之际的军事枢纽地位。
