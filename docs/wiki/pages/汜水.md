# 汜水

## 地理位置

汜水源出嵩山北麓，流经今河南省荥阳市汜水镇一带注入黄河，是楚汉战争荥阳相持阶段的重要地理坐标，见于[[008_高祖本纪|高祖本纪]]楚汉战场记载。


汜水，古水名，源出嵩山北麓、流经荥阳附近注入黄河，是楚汉战争荥阳相持阶段的重要地理坐标。楚汉两军长期对峙于荥阳一带，汜水作为战场侧翼水系，成为多次军事行动的地标。

《高祖本纪》载楚军将领曹咎出兵争夺成皋时，"大司马怒，度兵汜水"，强渡汜水进攻，旋即遭汉军大败，"大司马咎、长史欣皆自刭汜水上"，两将兵败后自刎于汜水岸边。曹咎之死成为楚汉成皋之战的重要转折，汜水因此铭刻于楚军败亡的历史叙事之中。
