# 雍林

雍林，春秋齐国地名，是权臣无知被当地民众袭杀的历史现场，也是齐桓公登位路径上的关键节点。无知弑杀齐襄公后自立，其后出游雍林时遭雍林人因旧怨击杀，这一偶然事件直接引发了公子小白（桓公）与公子纠争位的政治竞赛。

《齐太公世家》详载"桓公元年春，齐君无知游於雍林，雍林人尝有怨无知，及其往游，雍林人袭杀无知"，继而"高傒立之，是为桓公"。无知之死起于雍林人的私怨，却终结于管仲一箭射落小白（实为射中带钩而小白诈死）的政治算计，雍林因此成为齐桓公霸业前传的地理起点。
