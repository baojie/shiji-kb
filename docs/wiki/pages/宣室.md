# 宣室

宣室，汉代未央宫中的一处殿室，是汉文帝召见大臣、议论国事的重要场所。宣室因文帝深夜召见贾谊、"前席"问鬼神之事的历史典故而在《史记》中留下深刻印记，成为汉代帝王与才士之间政治际遇的经典地理象征。

《屈原贾生列传》记汉文帝"征之，至，入见，上方受厐祀，坐宣室"，以宣室为贾谊与文帝政治重逢的历史场所，揭示了才士得君、然而论及鬼神而非苍生的政治遗憾与历史讽喻。《龟策列传》亦涉及宣室环境中问卜求神的历史记述，两书合观，宣室成为《史记》中汉代帝王征用才智、问道苍生与鬼神之间的核心建筑地标，也是后世"宣室问鬼神"典故的历史原点。
