# 脉法与天地相应

汉文帝问仓公：同一病名，为什么诊断不同，有的人死有的人活？仓公的回答是《扁鹊仓公列传》中关于脉法哲学最集中的一段表述：

"病名多相类，不可知，故古圣人为之脉法，以起度量，立规矩，县权衡，案绳墨，调阴阳，别人之脉各名之，与天地相应，参合於人，故乃别百病以异之。有数者能异之，无数者同之。"

这段话以一系列"度量"意象重构了脉法的认识论地位：古圣人创立脉法，是为了在病名纷繁相似的混沌中建立可操作的分类系统——如同"起度量"（建立量纲）、"立规矩"（制定标准）、"县权衡"（挂起天平）、"案绳墨"（拉好墨线）。这四个工匠/测量意象并列，将脉诊塑造为一种精确科学，而非凭感觉的经验直觉。"与天地相应，参合於人"则将这套方法提升至宇宙论层面：脉的变化与天地阴阳的节律同步，因此诊脉能够识别人体内部的秩序与混乱。"有数者能异之，无数者同之"是实践层面的结论：掌握了脉法数理的人能区分同名异症，不掌握的人只能混淆。仓公随后补充：他之所以能做到，是因为保存了详细的"诊籍"（病历），每次诊断都与脉法对照检验，积累了可验证的经验。
