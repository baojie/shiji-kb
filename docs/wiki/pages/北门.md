# 北门

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

北门，古地名，位于今河南省鹤壁一带，是殷商都城朝歌附近的重要区域。《殷本纪》涉及朝歌（北门）一带，是商末政治活动的地理背景，亦见于西周初年分封记载。


北门，泛指城邑北侧之门，《史记》中多处以"北门"代指具体历史场景，涉及商周至汉代的不同城邑。因"北门"为通称而非专名，各处所指并非同一地点，而是当时事件发生所在城市的北门。

《史记》记载中，"北门"出现于多个重要场景：[[伊尹]]入自北门归亳，为[[商汤]]效力；秦军东征"过周北门"，士卒免胄示礼，周人感其威仪；[[商君]]变法前"立三丈之木於国都市南门，募民能徙置北门者"以立信，此为著名的立木取信故事；[[齐惠公]]时，王子城父获长翟，"埋之於北门"以示威慑；[[059_五宗世家]]中，[[江都王]]刘荣被召入京，"祖於江陵北门"，父老垂泪预感不祥。
