# 细阳

## 地理位置

细阳位于今安徽省阜阳市太和县细阳镇一带，汉代属汝南郡，汉高帝因灌婴护驾有功，赐其细阳千户为封邑，见于[[095_樊郦滕灌列传|樊郦滕灌列传]]。
