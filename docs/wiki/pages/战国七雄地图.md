# 战国七雄地图

经过春秋时代数百年的兼并，至战国时代形成七个强国并立的格局：秦、楚、齐、燕、赵、魏、韩，史称"战国七雄"。约前350年，秦孝公重用商鞅变法，国力大增；同期苏秦合纵、张仪连横，七雄外交斗争激烈。

各国都城是战国格局的核心节点：西方的秦以咸阳为中心；东方的齐居临菑；北方的燕居蓟；南方的楚虽迁都数次，以郢为根本；中原的赵居邯郸、魏居大梁、韩居新郑。
