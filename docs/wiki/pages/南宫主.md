# 南宫主

出现于《史记》**1** 处。

## 生平

南宫主，西汉汉武帝之姐，[[汉景帝]]之女，[[王太后]]（王娡）所生，与[[平阳公主]]、[[林虑公主]]并称王太后三女。王太后与修成君（王太后之姐卫媪所生）初次相见，三公主平阳主、南宫主、林虑主同至拜见修成君，修成君遂以姊称，场面温情。事见[[049_外戚世家|外戚世家]]。
