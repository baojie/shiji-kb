# 万里沙

万里沙，汉武帝东巡时在山东半岛海滨所祭祀的神祠，是汉代神仙方术信仰与帝王祭祀体系在东方沿海的地理落脚点。汉武帝多次封禅东巡，万里沙是其路线上重要的祭祀驿站，常与泰山祭礼连贯出现。

《封禅书》载汉武帝"旱，乾封少雨，天子既出无名，乃祷万里沙，过祠泰山"；《孝武本纪》同录此事。《河渠书》则记武帝"已用事万里沙，则还自临决河，沈白马玉璧于河"，武帝在祷告万里沙后亲临瓠子主持塞河，将神祀祈福与实际水利工程合并于同一叙事序列，揭示了汉武帝将宗教仪式与政务治理一体推进的行政风格。
