# 樊哙随沛公攻秦入关

> 樊哙常从沛公征战，攻城先登，屡立战功。从击章邯于濮阳、破赵贲于开封、攻破杨熊于曲遇，沿途攻城掠地，至霸上。斩都尉、首级无数，降卒近三千人。

**主要人物**：樊哙、章邯、高祖

## 事件经过

樊哙常从沛公征战，攻城先登，屡立战功。从击章邯于濮阳、破赵贲于开封、攻破杨熊于曲遇，沿途攻城掠地，至霸上。斩都尉、首级无数，降卒近三千人。
