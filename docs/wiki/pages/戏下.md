# 戏下

戏下，即戏水一带，位于今陕西临潼附近，是秦末楚汉之际楚军在关中的重要驻地。《秦楚之际月表》载，项羽率诸侯军入关后，"诸侯罢戏下兵，皆之国"，戏下是楚汉分封结束后各路诸侯解散军队的集结地，历史上著名的鸿门宴即发生在此前后。

《淮阴侯列传》记载，[[韩信]]早年"居戏下"，在项梁军中默默无名；井陉之战后，韩信俘获广武君李左车，"有缚广武君而致戏下者"，以礼相待，虚心求教。[[007_项羽本纪]]载"项羽在戏下，欲攻沛公"，是鸿门宴前[[项羽]]和[[范增]]谋划进攻刘邦的关键场景，戏下因此成为楚汉之际最具决定性意义的历史地名之一。
