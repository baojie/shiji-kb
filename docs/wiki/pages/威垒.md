# 威垒

出现于《史记》**1** 处。

## 生平

威垒，西汉汉武帝时匈奴降侯，以匈奴王身份归降汉朝，封侯，为建元以来侯者年表所载降侯之一，汉朝赐以侯爵，成为汉朝对匈奴降将笼络政策的一部分。事见[[020_建元以来侯者年表|建元以来侯者年表]]。
