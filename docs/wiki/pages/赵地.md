# 赵地

赵地，战国赵国的故土区域，以今河北南部为核心，是秦末诸侯并起时复建赵国政权的地理基础。《史记》中赵地主要作为地区概念出现，代指赵国旧疆在秦末楚汉之际被各方势力争夺与重建的历史舞台。

《张耳陈馀列传》记陈馀说服陈胜称王后，请命"北略赵地"，以奇兵开辟河北战场；此后陈馀击破张耳，"皆复收赵地，迎赵王於代，复为赵王"，完成了赵地从分裂到重统的过程。赵地因此成为秦末复国运动中河北方向战场的地理核心，是张耳与陈馀从盟友走向对立、最终武力决裂的历史背景。
