# 祋祤

祋祤，汉代关中县邑，约在今陕西耀州区一带，是西汉军政体系中的重要籍贯地之一。《孝景本纪》载"置南陵及内史、祋祤为县"，景帝时期将祋祤正式设置为县，标志着该地纳入关中内史行政体系。

《汉兴以来将相名臣年表》记"祋祤出宝璧玉器"，祋祤地出宝璧玉器，成为上报朝廷的祥瑞之物，折射出西汉礼仪政治中对地方祥瑞的高度重视；《卫将军骠骑列传》载右将军赵食其为"祋祤人也"，以迷失道路而被贬为庶人，祋祤作为功臣籍贯地在汉武帝北伐匈奴的军事叙事中留下印记；三书共同呈现了祋祤在行政、祥瑞与军事三个维度上的历史存在。
