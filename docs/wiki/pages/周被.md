# 周被

出现于《史记》**1** 处。

## 生平

周被，西汉淮南王刘安谋反阴谋中的重要策划者。刘安密谋起兵时，周被受命统帅颍川兵，负责堵塞轘辕、伊阙要道，截断中原南北交通；另有人掌控成皋、守南阳武关，图谋形成合围之势。后谋泄，淮南国被灭。事见[[118_淮南衡山列传|淮南衡山列传]]。
