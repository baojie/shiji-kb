# 郑灵公

春秋郑国君主，"染指"典故的主角。

楚国进献大鼋给郑灵公，大夫子公食指忽动，预言必将尝到异物。灵公将鼋羹赐众臣，独不赐子公。子公怒，以手指染鼎而尝之，离席而出。灵公大怒欲杀子公，子公与子家先发制人，弑灵公。此即成语"染指"的来源。

## 生平

郑灵公，春秋郑国君主，郑穆公之子，在位不满一年即被大夫弑杀。《郑世家》载郑灵公即位年，楚国使人进献鼋（大龟），灵公烹鼋请群臣，故意漏宴公子宋（子公），子公以指沾鼎尝味而去，灵公大怒欲杀子公。子公与公子归生（子家）先下手为强，弑杀郑灵公，郑灵公仅在位一年。「染指于鼎」这一成语即出于此事，郑灵公是春秋君臣相弑事件中最因小失大的悲剧君主之一。事见[[042_郑世家|郑世家]]、[[014_十二诸侯年表|十二诸侯年表]]。
