# 秦末大起义地图

前209年，陈胜、吴广在大泽乡揭竿而起，点燃了秦末大起义的烽火。短短三年之内，秦朝土崩瓦解。主要事件地点从大泽乡一路蔓延：陈胜建张楚于陈县，项梁在吴中（苏州）举兵，刘邦在沛县起事，项梁死后项羽在巨鹿一战成名，最终入关至戏水（鸿门），秦王子婴出降，大秦帝国灭亡。

《史记》以《陈涉世家》《项羽本纪》《高祖本纪》等篇章记录了这段历史的完整过程。
