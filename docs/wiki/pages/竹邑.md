# 竹邑

## 地理位置

竹邑位于今安徽省宿州市符离镇一带，汉代属沛郡，是楚汉之际的重要地名，见于[[054_曹相国世家|曹相国世家]]记载。


竹邑，汉初楚地的县邑，地望约在今安徽省宿州市符离集一带，是楚汉战争期间汉军深入楚腹地、向南扩展战线的重要地名。竹邑位于淮河以北、泗水流域，是汉军从北向南推进的攻略目标之一。

《曹相国世家》记曹参随军征战楚地的战绩；《傅靳蒯成列传》则详载某将军攻略楚地的战果，"略地东至缯、郯、下邳，南至蕲、竹邑"，以竹邑为南路推进的最远端，与蕲县并列，勾勒出汉军横扫楚地腹心的军事版图。竹邑因成为汉军南进极限的标志地点而被载入史册，两传相互印证，共同还原了楚汉战争中汉军大规模扫荡楚地、步步蚕食项羽后方的历史进程。竹邑一带的克定，是汉军最终合围项羽、赢得楚汉战争胜利的重要前奏。
