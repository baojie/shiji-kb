# 胜之

出现于《史记》**5** 处。

## 生平

胜之，西汉功臣侯系传承侯主，以周勃家族侯系及另一功臣侯系的继承人身份出现于《史记》年表。《高祖功臣侯者年表》记「侯胜之元年」（周勃系：周勃十二年传子周胜之，周胜之坐与公主奸被废）；《绛侯周勃世家》亦详记周胜之（周勃之子）被废事，此后子亚夫（[[周亚夫]]）受封。「胜之」作为单字或双字侯名出现于年表传承记录，是汉代功臣侯国传承格式的典型案例。事见[[018_高祖功臣侯者年表|高祖功臣侯者年表]]、[[057_绛侯周勃世家|绛侯周勃世家]]。
