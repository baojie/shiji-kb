# 辽西

辽西，燕代所置、秦汉沿用的北方边郡，约在今辽宁省西部及河北东北部，与辽东郡共同构成汉代东北方向防御匈奴和控制东北夷的军事屏障。《匈奴列传》记燕将秦开击破东胡后，"置上谷、渔阳、右北平、辽西、辽东郡以拒胡"，辽西郡自此成为燕秦汉东北边防体系的固定构成。

《封禅书》与《孝武本纪》载汉武帝封禅泰山后北巡，"巡自辽西，历北边至九原"，以辽西为北巡起点，展示了帝国对东北边疆的实地掌控与宣示。《绛侯周勃世家》记周勃平定燕王卢绾之乱后，"定辽西、辽东二十九县"，辽西郡由此在汉初稳定纳入汉朝郡县版图。
