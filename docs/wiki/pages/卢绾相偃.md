# 卢绾相偃

出现于《史记》**1** 处。

## 生平

卢绾相偃，西汉燕王[[卢绾]]麾下丞相，燕国最高行政长官。卢绾反汉后，汉相国周勃率军平叛，击下蓟城，俘获卢绾大将抵、丞相偃、守陉、太尉弱、御史大夫施等五名高级将领，屠浑都，随即连破燕军。丞相偃以燕相身份被俘，燕国最高文武官员均被擒获，卢绾政权由此覆灭。事见[[057_绛侯周勃世家|绛侯周勃世家]]。
