# 阳关

阳关，春秋鲁国城邑，约在今山东泰安附近，是鲁定公时期权臣阳虎政变失败后退守的据点，也是鲁国随后讨伐的军事目标。阳关因与政治动乱相关联，在《史记》中成为记录春秋末期鲁国权力斗争的地理背景。

《鲁周公世家》记阳虎欲杀三桓而谋未成，"三桓共攻阳虎，阳虎居阳关"；《十二诸侯年表》载"三桓攻阳虎，虎奔阳关"；其后"鲁伐阳虎，阳虎奔齐，已而奔晋赵氏"，阳关从阳虎据守之地变为鲁军攻伐对象。三书互证，阳关成为春秋末期鲁国家臣强、公室弱的政治格局中一段典型历史的地理见证。
