# 秦统一战争要地图

前230年至前221年，秦王嬴政（后为秦始皇）用十年时间灭掉六国，完成中国历史上第一次大一统。《史记·秦始皇本纪》详述了灭国顺序：韩（前230）→赵（前228）→魏（前225）→楚（前223）→燕（前222）→齐（前221）。

秦的统一战略以关中咸阳为后方大本营，依次向东、向北、向南用兵，水淹大梁、引水灌都是重要战术。楚国最为顽强，王翦率六十万大军方才攻克。
