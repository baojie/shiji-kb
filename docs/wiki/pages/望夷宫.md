# 望夷宫

望夷宫，秦代宫殿建筑，位于泾水之旁，因可遥望北方夷狄而得名，是秦二世的重要行宫。《秦始皇本纪》载"二世乃斋於望夷宫，欲祠泾，沈四白马"，秦二世在望夷宫斋戒祭祀泾水，是其统治末期忙于祭祀而荒于政务的写照。

《秦始皇本纪》又载赵高发动宫廷政变时，"遣乐将吏卒千馀人至望夷宫殿门，缚卫令仆射"，直捣望夷宫逼杀秦二世；秦二世在望夷宫中自杀而亡，望夷宫因此成为秦朝覆灭最后一幕的历史场所，是秦二世暴政终结、赵高乱政达到顶点的地理见证，在《史记》秦亡叙事中具有标志性意义。
