# 太宰嚭

出现于《史记》**1** 处。

## 生平

太宰嚭，春秋吴国权臣，吴王夫差重臣，官居太宰。太宰嚭接受越国贿赂，多次在夫差面前为越国说情，帮助越国争取到宽松的和议条件，阻止夫差听从伍子胥的灭越之策。伍子胥多次劝谏夫差，均被太宰嚭谗言所破，最终伍子胥被赐死，吴国亡于勾践。事见[[066_伍子胥列传|伍子胥列传]]。
