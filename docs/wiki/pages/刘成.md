# 刘成

西汉长沙王之子，汉武帝推恩令实施时受封为侯，见于021_建元已来王子侯者年表。

## 生平

刘成，西汉[[长沙定王]]之子，推恩令实施下封宜春侯。元朔六年七月受封宜春侯，在位约至元鼎年间；元鼎五年（前112年），因酎金之罪国除。刘成是汉武帝推恩令政策下长沙王诸子分封的典型案例，其封国存续约十余年后即因酎金获罪而除。事见[[021_建元已来王子侯者年表|建元已来王子侯者年表]]。
