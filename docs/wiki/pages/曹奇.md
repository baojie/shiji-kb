# 曹奇

## 生平

曹奇，西汉平阳侯，曹参曾孙，曹窋之孙，曹时之父。继曹窋之后袭封平阳侯，在位不久即卒，谥为曹奇。其子曹时继侯，曹时尚平阳公主，生子曹襄，平阳侯一脉由此与外戚卫氏相交织。事见[[054_曹相国世家|曹相国世家]]。
