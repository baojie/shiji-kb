# 虚

## 地理位置

虚位于今山东省东南部一带，战国时属魏国，见于[[078_春申君列传|春申君列传]]秦取燕酸枣虚桃及[[006_秦始皇本纪|秦始皇本纪]]蒙骜攻魏取二十城，是秦蚕食魏国东方领土的历史见证。

虚，战国时期魏国城邑，地望约在今河南东部或鲁西南一带，是战国中期秦国蚕食魏国领土过程中连续攻取的目标之一，与燕、酸枣、桃等城邑并列，共同构成魏国河内地区的防御体系。

《春申君列传》与《秦始皇本纪》均提及虚地：《春申君列传》中，黄歇（春申君）在向秦昭王进言时，历数秦国的东扩战果，称秦"举河内，拔燕、酸枣、虚、桃，入邢，魏之兵云翔而不敢救"，以虚的沦陷为据，说明秦国攻势之猛、魏国之积弱；《秦始皇本纪》则记始皇五年"将军蒙骜攻魏，定酸枣、燕、虚、长平、雍丘、山阳城，皆攻克，取二十城"，是秦军以蒙骜为将一举攻下魏国二十城的历史壮举，虚作为被克城邑之一在此留名。两传记载的时代略有不同，或指不同阶段的军事行动，但均说明虚在战国晚期属于秦魏反复争夺的战略要地。
