# 江淮

长江与淮河之间地区的地理泛称，主要涵盖今安徽省、江苏省大部，历史上是楚文化的核心区域。"江淮"在《史记》中多次出现于秦汉之际的历史叙事，是楚汉争霸的重要战场。

秦末，江淮一带是反秦力量的重要发源地之一：[[项梁]]在会稽（吴县）举兵渡江，[[英布]]（黥布）受封淮南王，[[刘邦]]在沛县（属泗水郡，毗邻江淮）起兵。楚汉战争期间，韩信北上击齐前曾经略淮南，彭越游击于梁地与江淮之间。汉初，江淮是吴王刘濞（吴越旧地）的势力范围，七国之乱时此处再度成为军事焦点。
