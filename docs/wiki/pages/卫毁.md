# 卫毁

出现于《史记》**1** 处。

## 生平

卫毁，即[[卫文公]]，春秋卫国国君，卫国被狄人大举入侵后，在[[齐桓公]]扶持下复国。宋桓公二十三年，齐桓公遣使迎卫毁于齐，立其为卫文公，卫国在楚丘重建，卫文公以俭朴治国，休养生息，卫国逐渐复苏。卫文公之女弟嫁给齐桓公为夫人，加深了齐卫同盟。事见[[038_宋微子世家|宋微子世家]]。
