# 钟离

钟离，春秋时期淮河中游古国及地名，地望约在今安徽省蚌埠市凤阳县东北临淮关一带，淮河南岸。钟离本为周代封国，族属与楚有关，春秋中期逐渐进入楚国势力范围，后为吴国所灭（或所并）。钟离地处淮河要冲，是春秋时期吴、楚、晋诸国势力交汇的战略要地，也是中原诸侯与吴国展开交涉、会盟的前沿据点，对于春秋后期吴国北上争霸具有重要的地理意义。

《十二诸侯年表》于同一行（r266）两度记及"会钟离"，分别对应两个诸侯的外交动向：其一记某诸侯（蔡）"十五始与吴通，会钟离"，记录该国首次与吴国建立外交联系，选择钟离作为初次会面地点；其二记"十与鲁会钟离"，记另一诸侯与鲁国会于钟离。两处记载均以年表简笔格式呈现，揭示钟离在春秋中晚期作为诸侯外交会盟场所的功能：地处淮水要津，既可作为中原诸侯联络吴国的中间站，也可作为淮域诸邦举行双边会议的中立地点。
