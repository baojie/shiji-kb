# 东朝

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

东朝，汉代地名，位于今山东省章丘市东北一带，属汉代齐郡范围。《魏其武安侯列传》涉及此地，是西汉政治地理中与东方齐地相关的宫室或城邑名称。
