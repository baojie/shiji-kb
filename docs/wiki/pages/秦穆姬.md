# 秦穆姬

## 生平

秦穆姬，晋献公之女，[[秦穆公]]之妻，以媵陪百里奚嫁入秦国。晋献公灭虢后，顺道袭灭虞国，俘虏虞公及大夫井伯（即百里奚），以百里奚作为媵陪（陪嫁之人），将女儿（秦穆姬）嫁给秦穆公。晋献公事后笑曰「马则吾马，齿亦老矣」，感叹所赠屈产骏马已老。秦穆姬是晋秦联姻中的纽带，百里奚因此入秦，成为秦穆公的重要谋臣。事见[[039_晋世家|晋世家]]。
