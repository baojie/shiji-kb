# 韩辽

汉文帝时匈奴郎中，以使节身份出使汉廷。汉文帝后二年，与当户[[且居]]、[[雕渠难]]同行，向汉文帝献马二匹。见汉文帝致匈奴大单于书中记载。

## 生平

韩辽，汉代人物，见于史记相关记录，具体事迹不详，史书载之简略。事见[[118_淮南衡山列传|淮南衡山列传]]。
