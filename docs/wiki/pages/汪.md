# 汪

汪，春秋时期晋国城邑，地望约在今山西西部，是秦穆公报复殽之战失败、二度攻晋的战场遗址。殽之战秦军大败于晋，孟明视等三帅被俘，其后秦国数次东征以雪前耻，汪即为秦军在报殽之役中攻取的晋地城邑之一。

《晋世家》载"秦果使孟明伐晋，报殽之败，取晋汪以归"，孟明视率军再伐晋国，攻取汪地后班师；《十二诸侯年表》则从晋、秦双方视角记录此役，晋方曰"秦报我殽，败于汪"，秦方曰"伐晋报殽，败我于汪"，两家叙述互证，共同揭示了汪之战在秦晋长期争霸史中的对称地位。
