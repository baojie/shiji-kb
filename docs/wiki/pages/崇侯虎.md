# 崇侯虎

[[崇侯虎]]，商朝末年崇国诸侯，以谗害[[周文王]]而留名史册。

[[周文王]]积善累德，诸侯归心——
崇侯虎进谗于[[商纣王]]：
*"西伯积善累德，诸侯皆响应他，将不利于帝。"*
商纣王信之，将文王囚于羑里（今河南汤阴）。

文王被囚期间，创演周易六十四卦之说流传于世。
后[[闳夭]]等人进献奇珍，纣王赦免文王，
并赐予弓矢斧钺，授予征伐之权。

崇侯虎的告密，反而成全了周文王的历史声名。

### 殷本纪与周本纪：同一个谗言，两种叙事框架

崇侯虎的核心行为——向商纣王告发西伯昌——在《殷本纪》（003-030）和《周本纪》（004-008）中均有记载，但叙述角度截然不同。《殷本纪》以商纣王的暴政为叙事主线：崇侯虎的告密发生在"醢九侯、脯鄂侯"等一系列残暴之事之后，是纣王政治失控的另一例证——崇侯虎是纣王身边的奸谀，共同构成商朝灭亡前的政治腐败图景。《周本纪》（004-008至010）则从周文王的角度叙述：崇侯虎的谮言导致文王被囚，文王在羑里演周易——这一段被囚经历反而成为文王政治圣德的试炼场，崇侯虎的谗害促成了文王最重要的历史遗产之一。两篇对同一事件的叙述，折射出"谗言"在不同叙事框架中的功能差异：在殷本纪里，崇侯虎是商末暴政的象征符号；在周本纪里，他的谗害反而是周朝道义崛起的历史契机。

## 生平

崇侯虎，商末崇国国君，商纣王近臣，以谗谄纣王、告发西伯姬昌（周文王）闻名。崇侯虎向纣王告发西伯积善累德、诸侯归心，纣王因此将西伯囚于羑里。然崇侯虎最终被周武王灭国，是助纣为虐的典型反面人物。事见[[003_殷本纪|殷本纪]]。
