# 红

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

红（古作"鸿"或"红"），古地名，位于今河南省遂平县一带，属汉代汝南郡。《史记》中作为汝颍流域地理坐标出现，是豫南平原地区古代聚落地名之一，与楚汉战争中的南阳、汝颍战场地理相关。
