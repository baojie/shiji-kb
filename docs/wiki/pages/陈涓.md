# 陈涓

[[陈涓]]，西汉开国功臣，封**河阳侯**（庄侯）。

以卒身份于元年起于砀，跟随刘邦，以二队将入汉，
参与击项羽，亲获郎将；后以丞相身份平定齐地。
高祖六年（前 201）三月庚子受封。

《史记》记载见[[018_高祖功臣侯者年表|018 高祖功臣侯者年表]]。
