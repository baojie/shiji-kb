# 赵同

[[赵同]]，西汉[[汉文帝]]时宦官，以佞幸受宠。
《史记·佞幸列传》记载：
孝文时中宠臣，士人则[[邓通]]，宦者则赵同与北宫伯子。

[[袁盎]]多次进谏文帝，其中一事即是斥责宦者赵同——
袁盎认为以宦官近侍陪乘，有失君主礼制。
文帝起初并不以为然，但袁盎以大义力谏，文帝方接受。

## 生平

赵同，西汉汉文帝时赵国丞相，在赵国政坛上颇有影响。《史记》对其事迹记载较为简略，相关出现于赵国诸侯王的传记记载中。事见[[043_赵世家|赵世家]]。

## 历代评价

## 历代评价

**太史公评价（叙述视角）**：司马迁在《晋世家》记述了赵同作为赵氏一族在晋景公时代受难的历史人物，他与赵括（赵同之兄或族弟，非赵括纸上谈兵的赵括）均在"下宫之难"（前583年）中被屠灭。太史公通过记述赵氏孤儿的故事，将赵同等人的死亡作为赵氏命运大起伏的历史背景，赵同本身的叙述较为简略，主要作为赵氏悲剧叙事的组成部分出现。

**汉代史学定论**：汉代史学对赵同的独立评价有限，主要将其纳入"下宫之难"和"赵氏孤儿"的整体历史叙事中处理。班固在相关叙述中延续《史记》框架，赵同等人的遭遇是赵氏由盛转衰后重新崛起这一历史弧线的低谷部分，在汉代讨论晋国政治史时具有历史背景意义。

**后世主要争议**：后世史家对赵同的讨论主要集中于"下宫之难"的历史真实性与"赵氏孤儿"故事的可信度辩证——两者均出现于《史记》，但历史细节与其他文献记载存在差异。赵同本身作为独立的历史人物讨论较少，其历史意义主要通过赵氏孤儿的戏剧性叙事间接呈现。

**历史影响 / 特殊符号意义**：赵同的历史意义主要附属于"赵氏孤儿"这一中国历史上最著名的救孤复仇故事。"赵氏孤儿"被改编为戏剧、小说、电影，在中国乃至世界范围内（18世纪伏尔泰时代传入欧洲）广为人知，赵同等人的受难成为这一宏大叙事的历史起点，在文化传播层面具有超越历史本身的广泛影响。
