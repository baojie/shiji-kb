# 北平文侯

出现于《史记》**1** 处。

## 生平

北平文侯，即[[张苍]]的身后谥号，西汉高祖功臣、丞相张苍之封号。张苍历仕高祖、文帝，曾任丞相多年，后因御史大夫申屠嘉检举其属下违法，张苍称病免职。汉景帝前五年（前152年）张苍卒，谥北平文侯，子北平康侯代，后子张类代侯，坐临诸侯丧后就位不敬，封国被除。事见[[096_张丞相列传|张丞相列传]]。
