# 东海王摇

汉初东瓯（东越）国君，越王[[勾践|句践]]之后裔。与闽越王[[无诸]]同为越国王族后代，秦末随汉起兵，汉立后封为东海王，都东瓯（今浙江温州一带）。吴楚七国之乱时，东瓯助汉击杀吴王[[刘濞]]，有功于汉。后因[[闽越王郢]]围攻东瓯，东瓯请求内迁，汉武帝允许，将东瓯民众迁入江淮之间，东瓯国遂绝。

## 历代评价

**太史公评价（叙述视角）**：司马迁在《东越列传》中记述东海王摇（闽越族群的另一支）的事迹，将其纳入百越政权在汉初的历史框架中。摇以越王勾践苗裔自居，在汉初获封东海王（东瓯王），并在汉越关系中扮演微妙角色。太史公的叙述较为简约，重在呈现百越诸国的存在形态，不作过多价值评判。

**汉代史学定论**：汉代史家将东海王摇定性为汉初百越边疆政策的重要一环，其内附和配合汉廷政策的态度，被认为是汉初百越关系中相对顺利的案例。与闽越的持续对抗不同，东瓯（东海王摇系）的相对顺从，成为汉廷处理南方藩属关系的正面参照，汉武帝时期东瓯内迁江淮的安置，也被视为汉越关系的和平解决方案之一。

**后世主要争议**：后世讨论东海王摇的争议主要集中于越王勾践苗裔的历史真实性问题——这一血统声称主要是政治合法性建构，其历史准确性难以核实。此外，东瓯、闽越、南越三支百越政权在汉初的相互关系，以及东瓯为何最终选择内迁而非与闽越联合，也是历史地理和民族史讨论的焦点。

**历史影响 / 特殊符号意义**：东海王摇是中国东南沿海地区早期历史政权的代表性人物，其后裔的流散成为浙东、闽北地区族群形成历史的重要线索。在当代浙江和福建的地方文化认同建构中，东海王摇（东瓯王）的历史被反复援引，成为区域早期文明史叙事的重要符号之一。
