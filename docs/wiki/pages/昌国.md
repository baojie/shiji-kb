# 昌国

## 地理位置

**配图**：谭其骧《中国历史地图集》第二册，西汉·兖豫青徐（页22-25）（见右栏）。

昌国，战国至汉代地名，位于今山东省淄博一带，属古齐郡范围。《乐毅列传》载燕将乐毅伐齐，功成后受封于昌国，号昌国君，是乐毅在山东的历史据点，也是后来乐毅与燕惠王关系破裂、奔赵事件的地理背景。
