# 羁马

羁马，春秋时期晋国城邑，地望约在今山西河曲附近，是秦晋争霸中秦国进攻晋国西境、引发河曲之战的导火索地点。秦康公伐晋取羁马，是殽之战后秦晋长期对峙期间秦国持续骚扰晋国的一次典型行动。

《秦本纪》载"六年，秦伐晋，取羁马，战於河曲，大败晋军"；《晋世家》记晋侯怒，"使赵盾、赵穿、郤缺击秦，大战河曲"，结果秦师遁走。《十二诸侯年表》则从晋、秦双方分别记录此役，晋方曰"秦取我羁马，与秦战河曲"，秦方曰"取羁马，怒，与我大战河曲"，两方叙述互证，揭示了秦晋围绕羁马归属展开的一次正面军事冲突。
