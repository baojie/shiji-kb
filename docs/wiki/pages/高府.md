# 高府

高府，楚国郢都城中的一处宫室建筑，因高而得名，具体地点已不可考，约位于今湖北省荆州市纪南城遗址范围内。高府在《史记》中因白公之乱而留名：楚惠王时期（前489—前432年在位），王孙白公胜蓄谋复仇，杀令尹子西、司马子綦后劫持楚惠王，将其置于高府——高府由此成为这场未遂政变的核心场所，既是叛乱者挟持君王的控制据点，也是忠臣护主出逃的起点。

《楚世家》叙述白公胜叛乱的高潮："白公胜怒，乃遂与勇力死士石乞等袭杀令尹子西、子綦於朝，因劫惠王，置之高府，欲弑之。惠王从者屈固负王亡走昭王夫人宫"，以高府作为白公控制王权的据点，而屈固负王出走昭王夫人宫则是楚王脱险的转机；《伍子胥列传》则以白公视角简记："石乞曰：'不杀王，不可。'乃劫王如高府。石乞从者屈固负楚惠王亡走昭夫人之宫"，以"劫王如高府"揭示石乞坚持弑王的强硬立场，而高府随着叶公率国人入援、白公失败自杀于山中而宣告解围。两处引文以不同的叙述角度共同呈现高府在这场历时月余的楚宫政变中的核心地位。
