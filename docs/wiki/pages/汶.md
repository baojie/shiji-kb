# 汶

汶，即汶水，发源于山东泰山北麓，西流注入济水，是古代青州的标志性水系。《夏本纪》载青州"浮于汶，通于济"，意指以汶水为漕运通道将山东物产运达济水干流，是大禹治水规划中沟通东方诸州的重要水路。

《封禅书》记汉武帝在"汶上"营建明堂，以供封禅礼典所需，"汶上"即泰山脚下汶水之畔，历代帝王封禅泰山均须经过此地。[[130_太史公自序]]载司马迁年轻时"北涉汶、泗"游历齐鲁，亲身考察孔子遗风。[[069_苏秦列传]]中以"乘船浮于汶"描述蜀地兵力顺流而下攻楚的军事路线，可见汶水在战国时期已是连通西南与中原的水运要道。

##### n164 · 「……，入于河，泆为荥，东出陶丘北，**东出陶丘北，**又东至于荷，……」

> **【集解】** 孔安国曰：「陶丘，丘再成者也。」郑玄曰：「《地理志》陶丘在济阴定陶西（北）〔南〕。」
>
> **【正义】** 《括地志》云：「陶丘在濮州鄄城西南二十四里。又云在曹州城中。徐才宗国都城记云此城中高丘，即古之陶丘。」
