# 南陵

南陵，汉代陵邑之一，位于今陕西西安附近，是薄太后的陵寝所在地，因地处汉文帝霸陵之南而得名。《外戚世家》载，"薄太后……以孝景帝前二年崩，葬南陵。以吕后会葬长陵，故特自起陵，近孝文皇帝霸陵"，南陵因此成为汉代外戚陵墓的独立建置，标志着薄太后在丧葬礼制上的特殊地位。

《孝景本纪》记"置南陵及内史、祋祤为县"，说明南陵在汉景帝时期正式设县，与普通陵邑一样承担行政功能。《汉兴以来将相名臣年表》亦记"初置南陵"，与《孝景本纪》相互印证。南陵的设置体现了汉代以陵邑为基础的区域管理惯例，薄太后独立营陵的礼制选择，也折射出吕后与薄氏两个外戚集团在汉初政治中的差异性地位。
