# 鸣泽

鸣泽，汉代西北地区水泽，约在今宁夏或甘肃一带，是汉武帝巡游西北时的经过地点之一。鸣泽在汉代礼制中被列为小山川祭祀对象，也是武帝出巡记录中多次出现的西行地理坐标。

《封禅书》将鸣泽列入汉代官方祭祀体系："汧、洛二渊，鸣泽、蒲山、岳鞚山之属，为小山川，亦皆岁祷塞泮涸祠"，鸣泽作为关中以西的自然地理对象纳入国家祭典，体现了汉代将西北山川纳入礼制秩序的政策取向。《孝武本纪》与《封禅书》均记载汉武帝"春，至鸣泽，从西河归"，武帝在出巡雍地郊祀后经鸣泽返回，二书互证，鸣泽成为《史记》中汉武帝西北巡游路线的具体地理坐标，也是汉代礼制版图向西延伸的物质见证。
