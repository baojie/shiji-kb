# 刘无伤

出现于《史记》**1** 处。

## 生平

刘无伤，西汉梁国王族，梁平王[[刘襄]]之子，袭封梁王，即梁无伤王。梁孝王（刘武）死后，历经梁共王、梁平王（刘襄），至刘无伤立为梁王，为梁国又一代国君。事见[[058_梁孝王世家|梁孝王世家]]。
