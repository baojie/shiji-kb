# 留长孺

## 生平

留长孺，西汉卜相名家，以相猪（彘）立名。《日者列传》以「留长孺以相彘立名」一语记录其专业特长，将其列于汉代各地方术名家中，是西汉相术流派中专擅牲畜相法的特色人物。留长孺的记录反映了汉代方术文化的多样性，相彘（相猪）在汉代农业社会中具有实际的经济价值。事见[[127_日者列传|日者列传]]。
