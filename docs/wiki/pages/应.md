# 应

应，古地名，位于今河南宝丰一带，战国时期为魏韩交界地带的重要会盟地点。《秦本纪》记载秦惠王九年"渡河，取汾阴、皮氏，与魏王会应"，即秦魏两国君主在应地举行会盟，是秦国东进过程中以外交手段配合军事的典型案例。

《史记》"应"作地名主要见于[[044_魏世家]]及相关纪年，[[015_六国年表]]亦有"与秦会应"的记载，与围攻焦地等军事行动并列，说明应地在秦魏博弈的外交与军事布局中有重要的地理意义。
