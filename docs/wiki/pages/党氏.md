# 党氏

党氏，春秋鲁国地名，是庆父之乱中鲁公子斑被杀的历史现场，也是春秋鲁国权臣专政时代一段血腥政治事件的地理坐标。鲁庄公死后，公室纷争，庆父借助党氏这一私宅党所，暗中策划刺杀储君之事。

《鲁周公世家》载"庆父使圉人荦杀鲁公子斑於党氏"，庆父派遣养马人荦在党氏处刺杀了刚被立为君的公子斑，随后扶立湣公。季友因此出奔陈国，"庆父竟立庄公子开，是为湣公"，庆父的专横之举由党氏一事揭开序幕，此后"庆父不死，鲁难未已"的历史判断成为后世广传的政治格言。
