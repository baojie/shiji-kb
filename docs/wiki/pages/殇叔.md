# 殇叔

[[晋穆侯]]之弟。穆侯二十七年卒，殇叔自立为君，太子仇出奔。殇叔在位四年，前774年，太子仇率徒袭杀殇叔，复位为[[晋文侯]]（[[039_晋世家|039-005]]）。

## 生平

殇叔是晋国历史上少见的"弟自立"案例——穆侯死后，其弟殇叔趁太子仇尚幼或势力不足，宣布自立为君。太子仇流亡在外，四年后积聚力量，率徒袭杀殇叔，复辟为晋文侯。文侯仇此后在位三十五年，是晋国中兴之君，辅助周平王东迁有功。
