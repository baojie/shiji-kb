# 七国起兵反汉

> 削书至，吴王先起兵，胶西正月丙午诛汉吏二千石以下，胶东、菑川、济南、楚、赵亦发兵西。齐王后悔饮药自杀畔约，济北王被劫守不得发兵。吴王悉士卒发二十余万人，南使闽越东越亦发兵从。

**主要人物**：吴王、楚王戊、胶西王、赵王遂
