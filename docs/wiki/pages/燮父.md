# 燮父

**燮父**，西周时期晋国第二任君主，[[唐叔虞]]之子。燮父继承父位后，因迁居晋水，将国名由"唐"改为"晋"，是晋国正式得名的关键人物，故又称晋侯。

> 燮父徙居晋水，改号曰晋侯。——[[039_晋世家]]

晋国由此成为西周重要封国，至春秋时期发展为春秋五霸之一。
