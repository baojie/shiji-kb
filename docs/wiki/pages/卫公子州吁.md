# 卫公子州吁

**卫公子州吁**（州吁），春秋卫国贵族，[[卫桓公]]之弟。宋殇公元年，弑卫桓公（[[卫完]]）自立为卫君，随即联合宋国讨伐郑国，欲得诸侯承认。后被老臣[[石碏]]设计杀死，[[卫宣公]]继立。

> 殇公元年，卫公子州吁弑其君卫完自立。——[[038_宋微子世家]]
