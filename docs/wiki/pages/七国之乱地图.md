# 七国之乱地图

前154年，汉景帝采纳晁错"削藩"建议，削减诸侯王封地。吴王刘濞联合楚、赵、胶东、胶西、济南、淄川六国，打着"诛晁错，清君侧"旗号起兵，史称"七国之乱"（或"吴楚七国之乱"）。

平叛战争中，梁王刘武坚守睢阳，成为拖延叛军西进的关键。汉将周亚夫以绝粮道的战术，仅三个月便平定叛乱。七国之乱后，汉朝大力推行削藩，诸侯国势力大为削弱，为汉武帝推行推恩令铺平道路。
