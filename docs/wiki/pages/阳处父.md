# 阳处父

出现于《史记》**1** 处。

## 生平

阳处父，春秋晋国大夫，晋文公、晋襄公时期臣子，曾奉命选举将领，却取怨于人，后被晋国大夫杀死，见于晋世家相关记录，史书载之简略。事见[[039_晋世家|晋世家]]。
