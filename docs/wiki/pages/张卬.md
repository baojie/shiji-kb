# 张卬

西汉水利官员，汉武帝时参与水利工程事务。见于河渠书，与[[徐伯]]、[[番系]]同为汉武帝时水利建设的相关官员。

## 生平

张卬，西汉[[张汤]]之子。汉武帝时，有人向武帝建议开凿[[褒斜道]]以连通山东（关东）至汉中，利用褒水和斜水的水路，同时利用褒斜地区丰富的木材竹箭。武帝采纳建议，任命张汤之子张卬为**汉中守**（汉中郡太守），征发数万民夫开凿褒斜道五百余里。然而道路虽建成便捷，但两水水流湍急、多礁石，仍不可行漕船，工程最终未能实现通漕之目的。张卬以父荫入仕，以此次工程监督见于《史记·河渠书》。
