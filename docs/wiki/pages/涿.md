# 涿

涿，汉代涿郡所在地，位于今河北涿州一带，是燕赵之间的交通枢纽与商业重地。涿城地处太行山东麓与华北平原的交界地带，控扼北京（燕都）南下中原的门户，是汉代北方政治地理的重要节点。

《樊郦滕灌列传》载，灌婴破燕王臧荼军于易下，战功卓著，"食邑涿五千户，号曰涿侯"，以涿为功臣封邑，是汉初以北方城邑奖赏军功将领的典型案例。《货殖列传》则将涿地纳入邯郸为中心的区域商业地理叙事，"北通燕、涿，南有郑、卫"，以涿作为邯郸向北连通燕地的商道节点，揭示涿在汉代区域经济地理中作为南北商业交通枢纽的地位。两书合观，涿在《史记》中兼具军功封赏地名与区域商业地理的双重历史价值。
