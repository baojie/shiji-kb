# 西周邦国地图

西周建立后，周武王、周公旦实行大规模封建，将宗室、功臣、先代后裔分封于各地，形成以周王室为核心的封建体系。西周全盛期（约前1000年）的主要封国遍布中原，镐京（西都）与洛邑（东都成周）是整个体系的政治中枢。

各封国都城是周天子分封体系的地理支柱：东方有齐（营丘/临菑）、鲁（曲阜）；北方有燕（蓟）；南方有楚（郢）；西方有秦（雍城）；中原有晋（翼城）、宋（商丘）、卫（帝丘）、陈（宛丘）、蔡（上蔡）、郑（棫林）等。

《史记》各世家详述了这些封国的来源：齐太公吕望封营丘，鲁周公子伯禽封曲阜，召公之子封蓟，晋唐叔虞封唐（后改晋），楚国祖先熊绎封于荆楚，各国皆有其建国渊源。
