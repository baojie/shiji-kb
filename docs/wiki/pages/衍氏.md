# 衍氏

衍氏，战国时期魏国城邑，约在今河南郑州附近，是秦统一战争期间秦军攻取魏地的重要目标之一。《秦始皇本纪》以简洁笔墨记录"杨端和攻衍氏"，揭示了秦将杨端和在秦王政统一前期扫荡魏国腹地的军事行动。

《曹相国世家》记曹参随汉军征战时，在衍氏附近地区参与了多次战役，衍氏因此在楚汉战争叙事中延续出现。作为昔日秦魏争夺的城邑，衍氏历经战国末期兵燹后又成为楚汉战争的过境战场，折射出中原腹地城邑在乱世中的历史命运。
