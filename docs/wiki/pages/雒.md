# 雒

"雒"是"洛"的古字形，在周代文献（包括《史记》引用的早期文献）中写作"雒"，汉代以后改写"洛"。"雒邑"即今洛阳，是[[周公旦]]营建的东方政治中心，[[周平王]]东迁后以此为都，是东周首都。

《史记》中"雒"字出现于与西周、东周相关的历史叙述，如"雒邑""雒阳""雒水"等。与"洛"为同一地理实体，只是用字不同。参见主条目 [[洛阳]] 或 [[洛]]。

## 三家注

以下是《史记》三家注中直接注释「雒」相关文字的条目。

#### 出自 [[033_鲁周公世家]]

##### n033 · 「……土。其三月，周公往营成周雒邑，**其三月，周公往营成周雒邑，**卜居焉，曰吉，遂国之。……」

> **【集解】** 《公羊传》曰：「成周者何？东周也。」何休曰：「名为成周者，周道始成，王所都也。」
