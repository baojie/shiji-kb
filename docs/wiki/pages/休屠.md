# 休屠

出现于《史记》**1** 处。

## 生平

休屠，匈奴贵族，休屠王，活跃于西汉武帝时期。元狩二年，霍去病率军进击祁连山，斩杀折兰王、卢胡王，并收取休屠祭天金人（铸造的铜像，用于匈奴祭祀天神），此为汉军所夺的重要战利品，彰显对匈奴宗教信仰的打击。事见[[111_卫将军骠骑列传|卫将军骠骑列传]]。
