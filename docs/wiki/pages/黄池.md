# 黄池

位于今河南省新乡市封丘县南，春秋末年重要的诸侯会盟之地。前482年，吴王[[夫差]]北上，与晋定公在黄池会盟，争夺诸侯盟主（霸主）地位。

双方争执不下，吴军以精锐武力相威胁，晋人迫于压力，勉强承认夫差为盟主。然而就在夫差主持黄池之盟的过程中，越王[[勾践]]（句践）趁机出兵袭击吴国本土，攻破吴都，杀死吴太子。夫差得信后急忙回师，吴国此后一蹶不振，最终被越国所灭（前473年）。黄池之会是吴国霸业的顶点，也是其衰亡的起点，《史记》中以此为吴越争霸叙事的关键节点。

## 历代评价

**太史公评价（叙述视角）**：司马迁在《吴太伯世家》和《越王勾践世家》中记述黄池之会——吴王夫差会诸侯于黄池，欲争周室霸主之位，而此时越王勾践已趁机攻入吴国腹地。太史公将黄池之会与越军入吴并置，揭示夫差北进争霸时后方空虚的历史悖论，以此作为"争外失内"的历史警示，是吴国覆亡历史叙事的关键节点。

**汉代史学定论**：汉儒将黄池之会定性为吴王夫差穷兵黩武、北进争霸而忽视根本威胁（越国）的历史失策，是"得位不保本"的经典负面案例。班固延续《史记》的叙述框架，将黄池视为吴国命运的转折点：夫差在黄池达到权力顶峰的同时，越国已完成对吴的致命一击，这一历史讽刺成为后世言说"骄兵必败"时的引用依据。

**后世主要争议**：后世对黄池之会的讨论主要集中于：夫差争霸的战略是否有合理性，还是单纯的过度扩张？一方认为夫差在北上争霸与应对越国的战略选择上犯了根本错误，两者不可兼得而选择错误；另一方则认为在当时的政治格局下，维持对越的警戒而放弃北进机会，同样会使吴国陷入不利处境——吴国的失败是多因素的历史必然，而非单一决策的结果。

**历史影响 / 特殊符号意义**：黄池之会成为中国历史上"外强而内虚、霸主之梦覆灭于一夕"的标志性事件，与夫差的整个政治悲剧共同构成"骄傲自满、忽视真正威胁"的历史符号。在讨论战略重心选择、内外威胁权衡时，黄池之会是中国历史上最常被援引的反面案例之一。
