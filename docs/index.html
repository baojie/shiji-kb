📜 史记知识库 - 命名实体标注版

🎯 项目简介

本项目是《史记》全130章的命名实体标注知识库。采用11类实体标注系统，对所有人名、地名、官职、朝代、器物等关键信息进行了精细标注，并使用Biblical-style段落编号系统（Purple Numbers），便于引用和学术研究。

✨ 标注类型

人名 地名 官职 时间 朝代/氏族 制度 族群 器物/书名 天文 神话 动植物

🔢 Purple Numbers 段落编号

每个段落都有唯一的编号（如 [0], [1], [1.1], [2.3.1]），可点击复制链接，精确引用到具体段落，便于学术讨论和知识管理。

🚀 未来开发规划

本项目将持续增补完善，计划开发以下功能：

  • 📌 事件标识：标注重大历史事件（战争、改革、政变等），建立事件索引
  • 🔗 人物关系链接：为人物间的关系建立超链接（师承、姻亲、君臣等），可视化人际网络
  • 📊 表格补全和现代化：将史记十表转换为可交互的现代化表格，支持排序、筛选、检索
  • 📑 分类实体索引：按类别生成实体索引（人名索引、地名索引、官职索引等），提供快速检索
  • 📝 三家注标注：整合裴骃《集解》、司马贞《索隐》、张守节《正义》的注释
  • ⏱️ 时间年表阅读：提供时间轴视图，按年代浏览历史事件的发生顺序
  • 🕸️ 知识图谱浏览：构建史记知识图谱，支持图形化浏览人物、地点、事件的关联网络
