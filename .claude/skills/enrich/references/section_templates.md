# 节模板：按页面 type 的标准节结构

每种 type 的节按"优先级"排列——前面的节先补，后面的节视内容量按需添加。

---

## person（人物）

```
# [人名]

[定位句：XXX，时代+身份+最重要的一件事]

## 人物背景          ← 家世、出身、登场时间
## 主要事迹          ← 按时间或重要性，可拆分为多个 ##
## ⚠️ 史料矛盾       ← 若有矛盾，必须写；无则跳过
## 太史公赞          ← 引原文评价，加分析
## 历史评价          ← 后世影响（达 featured 以上才加）
## 史记引文          ← 引文摘录（保留 > 出自 格式）
## 相关页面          ← 每条加一句描述
## 相关章节          ← 本纪/世家/列传链接
```

**散文重点**：事迹叙述（"X做了什么，结果如何"）优先于背景铺垫。

---

## place（地名）

```
# [地名]

[定位句：XXX，今地 + 历史时期 + 在史记中的核心戏份]

## 地理位置与建置    ← 今地、辖境、设郡时间（有据才写）
## 历史沿革          ← 各朝代归属变化
## 史记相关记载      ← 发生在此地的重要事件（可细分为多个 ##）
## ⚠️ 史料矛盾       ← 若地望有争议，写出
## 史记引文          ← 按章节分组，精选有叙事价值的
## 相关页面          ← 每条加描述
## 相关章节
```

**散文重点**：地点在史记叙事中的"戏份"——哪些决定性事件发生在此？

---

## state（诸侯国/国家/侯国/邦国）

```
# [国名]

[定位句：XXX，始封/建立时间 → 灭亡/国除时间，封地今地，核心政治地位]

## 国家概况          ← 【时间】立国/封建年份、国祚年限；【地点】封地今地考证、疆域；爵位/等级
## 地理位置          ← 封地地望（今省市县）、首都/治所，若有 coords 字段则调用 /map PAGE
## 历代君主          ← 【人物】可用表格，列侯主/君主姓名、在位年份、谥号；侯国用"历代侯主"
## 重大事件          ← 【事件】立国经过、盛衰转折、国除原因（酎金/谋反/无后等）；可细分为多个 ##
## 太史公赞          ← 若有专章，引赞语；无专章则跳过
## ⚠️ 史料矛盾       ← 若有地望争议或世系矛盾，必须写；无则跳过
## 史记引文          ← 精选，年表类注明年号纪年
## 相关页面
## 相关章节
```

**五维要求**（basic→standard 及以上必须覆盖）：
- **时间**：立国/封建年份（精确到年号年次）、国祚年限、国除时间
- **地点**：封地今地（省市县）或"地望不详"；若页面有 `coords` 字段，必须调用 `/map PAGE` 配谭图
- **人物**：历代君主/侯主姓名、在位年数；若史记有传，链接其页面
- **事件**：立国原因（推恩令/军功/宗室分封等）、重要事件、国除原因
- **地图**：有 `coords` 字段时必配——调用 `/map PAGE` 从谭其骧《中国历史地图集》自动裁切截图并写入 frontmatter `images:`；无 `coords` 时先在 frontmatter 补 `coords: [lon, lat]`，再调用 `/map PAGE`；若地望不详无法补 coords，在 `## 地理位置` 末注明 `<!-- TODO: 待补 coords -->`

**侯国（汉代推恩制）散文重点**：封建时间 + 来源（谁的儿子）+ 侯主世系 + 国祚长短 + 国除原因（若有）。

---

## story / event（历史事件）

```
# [事件名]

[定位句：XXX，时间 + 参与方 + 结果]

## 背景              ← 事件前的形势
## 经过              ← 按阶段/时序（可细分为多个 ##）
## 结果与影响        ← 直接后果 + 长期影响
## ⚠️ 史料矛盾       ← 两种说法对比表
## 太史公赞          ← 引赞 + 分析视角
## 史记引文          ← 关键段落原文
## 相关页面
## 相关章节
```

---

## concept（概念/制度/器物）

```
# [名称]

[定位句：XXX，定义 + 史记中出现的语境]

## 含义与用法        ← 词义演变或制度内容
## 史记中的记载      ← 出现场景与意义
## 史记引文
## 相关页面
```

---

## 通用节添加规则

1. **已有同名节**：不重建，只追加内容
2. **`## 史记引文` 必须存在**：是最低要求，所有 basic 以上页面均应有
3. **`## 相关页面`**：每条格式为 `- [[页面ID]] — 一句描述`
4. **`## 相关章节`**：仅列本纪/世家/列传链接，不列表志
5. **⚠️ 节**：有矛盾才加，不要空写
