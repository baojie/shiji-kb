## 重大历史事件链

### 1. 政权更替链

```
&神农氏& → @黄帝@ → @颛顼@ → @帝喾@ → @帝尧@ → @帝舜@ → @禹@ → &夏后& → &商& → &周& → &秦&
```

#### 关键转折点
- **禅让制终结**: 002-029 益让启即位（约公元前2070年）
- **商代夏**: 002-044 汤践天子位（约公元前1600年）
- **周代商**: 003系列 武王伐纣（约公元前1046年）
- **秦统一**: 005系列 始皇统一（公元前221年）

### 2. 治水事业链

```
001-020 鲧治水九年失败 → 001-028 鲧被诛羽山 → 002-004 舜举禹续业 → 002-007 禹治水十三年 → 002-008至016 禹划九州 → 002-020 禹受玄圭
```

**意义**: 从失败到成功，奠定禹建立夏朝的基础

### 3. 贤臣辅政链

```
001-047至055 舜任命九大臣 → 002-026 禹举皋陶 → 003系列 伊尹辅商 → 004系列 周公辅成王 → 005系列 商鞅变法
```

**意义**: 展现了中国古代重视贤臣辅政的传统

### 4. 变法改革链

```
001-006 黄帝设官制 → 001-027 舜制刑法 → 002-019 禹定五服 → 003系列 商朝改正朔 → 004系列 周礼制度 → 005系列 商鞅变法
```

**意义**: 国家治理制度的逐步完善

### 5. 战争征伐链

#### 统一战争
- 001-002 阪泉之战（黄帝vs炎帝）
- 001-003 涿鹿之战（黄帝vs蚩尤）
- 002-030至032 甘之战（启vs有扈氏）
- 002-042 鸣条之战（汤vs桀）
- 003系列 牧野之战（武王vs纣）
- 005系列 秦统一六国

#### 意义
从部落冲突到国家统一的历史进程
