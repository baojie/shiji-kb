# 《史记争霸》游戏设计文档

## 游戏概念

一款基于《史记》历史技能的策略卡牌+文字冒险游戏,玩家扮演一位穿越到战国末期的谋士,通过学习和运用从《史记》中提取的15种历史技能,帮助不同势力完成历史使命,最终改写或重现历史。

---

## 核心玩法

### 1. 技能卡牌系统

**技能分类**:
- 🔵 **政治制度类** (4张): 
  - 秦朝皇帝尊号制度
  - 水德运数制度
  - 废分封行郡县制
  - 西周刑事审判系统

- 🔴 **军事策略类** (5张):
  - 约法三章
  - 鸿门宴逃生策略
  - 破釜沉舟战术
  - 城池劝降策略
  - 牧野之战执行

- 🟡 **历史事件类** (2张):
  - 项羽乌江抉择
  - 殷商继承动乱分析

- 🟢 **文化思想类** (2张):
  - 焚书坑儒政策
  - 羲和历法授时

- 🟣 **领导管理类** (1张):
  - 知人善任

- ⚪ **元数据类** (1张):
  - 史记作者识别 (特殊卡,可触发彩蛋)

**技能获取**:
- 完成历史事件解锁
- 通过"史记碎片"合成
- 特殊剧情奖励

## 游戏模式

### 模式一: 历史重演 (Story Mode)

**剧情线**:
1. **第一章: 商周之变** (公元前1046年)
   - 核心技能: 牧野之战执行、殷商继承分析
   - 目标: 帮助周武王推翻商纣
   - 关键选择: 是否采用"破釜沉舟"战术

2. **第二章: 秦国崛起** (公元前221年)
   - 核心技能: 废分封行郡县制、水德运数制度、焚书坑儒
   - 目标: 辅佐秦始皇统一六国并建立新制度
   - 关键选择: 如何平衡中央集权与地方治理

3. **第三章: 楚汉相争** (公元前206-202年)
   - 核心技能: 鸿门宴逃生、破釜沉舟、约法三章、知人善任
   - 目标: 选择辅佐刘邦或项羽
   - 关键选择: 项羽在乌江的命运

4. **第四章: 汉朝建立** (公元前202年后)
   - 核心技能: 约法三章、知人善任、城池劝降
   - 目标: 稳定新政权,建立长治久安

**难度设定**:
- 简单: 历史提示明显,技能使用限制少
- 普通: 需要合理搭配技能
- 困难: 历史事件有随机变化,技能使用有冷却时间
- 史记模式: 完全还原历史,一个错误选择可能导致全盘崩溃

---

### 模式二: 自由沙盒 (Sandbox Mode)

**玩法**:
- 选择任意历史时期(商周/春秋/战国/秦汉)
- 自定义势力和初始技能卡组
- 通过外交、战争、改革等方式争霸天下
- 每回合可使用1-3张技能卡

**胜利条件**:
- 统一天下
- 文化胜利(传播思想到所有诸侯国)
- 外交胜利(成为天下共主)
- 时间胜利(维持霸主地位50年)

---

### 模式三: 技能挑战 (Skill Challenge)

**单技能深度挑战**:
每个技能都有专属的5关挑战,例如:

**"破釜沉舟战术"挑战**:
- 第1关: 基础教学 - 巨鹿之战重现
- 第2关: 资源限制 - 仅有2日粮食
- 第3关: 敌军3倍优势
- 第4关: 多线作战 - 同时救援3个被围城池
- 第5关: 终极挑战 - 以5000兵力对抗20万秦军

**奖励**:
- 完成挑战解锁技能升级
- 获得"史记碎片"
- 解锁隐藏剧情

## 游戏机制

### 1. 回合制系统

**每回合流程**:
1. **情报阶段**: 获取各方势力情报
2. **决策阶段**: 选择使用技能卡
3. **执行阶段**: 技能效果结算
4. **事件阶段**: 随机历史事件触发
5. **结算阶段**: 更新势力关系和资源

### 2. 资源系统

**四大资源**:
- 💰 **财富** (用于招募人才、维持军队)
- 👥 **民心** (影响政策推行效果)
- ⚔️ **军力** (决定战争胜负)
- 📜 **威望** (影响外交和人才吸引)

**技能消耗**:
- 政治类技能主要消耗威望和民心
- 军事类技能主要消耗军力和财富
- 文化类技能主要消耗威望和民心

### 3. 技能组合系统

**连击效果**:
某些技能组合使用会产生特殊效果:

- "约法三章" + "知人善任" = "得民心者得天下" (民心+50%)
- "破釜沉舟" + "牧野之战执行" = "背水一战" (军力临时+100%)
- "废分封" + "水德运数" = "大一统帝国" (威望+30%, 但民心-20%)
- "鸿门宴逃生" + "城池劝降" = "以退为进" (外交成功率+40%)

### 4. 历史人物系统

**可招募的历史人物**:
每个人物都有特定的技能加成:

- **张良**: 鸿门宴逃生策略 +2级
- **韩信**: 破釜沉舟战术 +2级, 知人善任 +1级
- **萧何**: 约法三章 +2级, 知人善任 +1级
- **李斯**: 废分封行郡县制 +2级, 焚书坑儒 +2级
- **范增**: 所有策略类技能 +1级
- **项羽**: 破釜沉舟 +3级, 但无法使用"鸿门宴逃生"

## 技能详细设计

### 示例: "破釜沉舟战术"

**基础属性**:
- 类型: 军事策略
- 稀有度: ★★★★☆
- 消耗: 军力-30%, 财富-20%
- 冷却: 5回合

**效果**:
- 本次战斗军队战斗力 +150%
- 战斗结束后,若胜利则威望 +50
- 若失败则游戏结束(背水一战,无退路)

**升级路径**:
- Lv1: 基础效果
- Lv2: 战斗力加成 +180%, 消耗降低10%
- Lv3: 解锁"连战"效果(可连续发动3次战斗)
- Lv4: 失败不再游戏结束,但损失所有军力
- Lv5: 战斗力加成 +200%, 胜利后额外获得敌军50%兵力

**历史典故**:
游戏中会显示《史记·项羽本纪》原文:
> "项羽乃悉引兵渡河,皆沉船,破釜甑,烧庐舍,持三日粮,以示士卒必死,无一还心。"

## 美术风格

**视觉设计**:
- 水墨画风格的技能卡牌
- 每张卡牌背景是相关的《史记》原文片段
- 人物立绘采用传统工笔画风格
- 地图采用古代舆图样式

**UI设计**:
- 竹简风格的菜单界面
- 技能卡牌展示时有毛笔书写动画
- 战斗场景采用兵棋推演风格

## 教育价值

### 历史学习模块

**史记原文阅读**:
- 每个技能都关联《史记》原文段落
- 使用技能时可查看相关历史背景
- 完成剧情解锁"史记讲堂"

**历史知识问答**:
- 每章结束后有历史知识测验
- 正确回答可获得额外奖励
- 错误答案会显示正确的历史知识

**人物传记**:
- 收集人物卡片解锁完整传记
- 传记内容直接引用《史记》原文
- 配有现代白话文翻译和历史解读

## 技术实现建议

### 平台选择
1. **网页版** (推荐):
   - 使用 React + TypeScript
   - 卡牌动画使用 Framer Motion
   - 可以直接嵌入《史记》Markdown内容

2. **移动端**:
   - React Native 跨平台开发
   - 适合碎片化时间游戏

3. **桌面端**:
   - Electron 封装网页版
   - 更好的阅读体验

## 结语

这款游戏将《史记》的历史智慧转化为可互动的游戏机制,让玩家在娱乐中学习历史,在策略中体会古人智慧。通过技能卡牌系统,玩家可以亲身体验"破釜沉舟"的决绝、"鸿门宴"的惊险、"约法三章"的智慧,真正做到"以史为鉴"。
