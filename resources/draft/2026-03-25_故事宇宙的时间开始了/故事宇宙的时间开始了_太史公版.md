# 故事宇宙篇

## 声传之术

《史记罗生门》者，以智算之术化古籍为声传也。

其制作之法，先以学府之图推演诸事之脉络，后以智算之术变推演文为故事，继而为声传。

推演者何？系事为链，链生新知也。

昔推演之法，严而枯也。人皆不愿读。今藉智算，化枯文为生动之传，人乐闻之。

於是有此传：《太史公怎么知道：荆轲刺秦的信源推理》。

## 荆轲刺秦，太史公何由知之？

荆轲刺秦王，图穷而匕见，秦王绕柱而走，天下传诵。

然世人未思：太史公去事百二十七载，何由知之？

轲当场毙矣。

秦舞阳亦死矣。

太子丹後诛矣。

在场秦臣，岂著书以志之？

秦皇岂自叙其狼狈乎？

答在《刺客列传》末句：

> 使公孙季功、董生与夏无且游，具知其事，为余道之如是。

**传承之脉**：

```
夏无且（太医，以药囊击轲者）
    ↓
公孙季功、董生（无且之友）
    ↓
司马迁
```

无且者，信源之最也：

立於秦王侧，目睹其详；以药囊掷轲，身预其事；存而受赏黄金二百镒；其事为生平至荣，逢人必道。

传承仅隔二三世，历不过百馀年，信矣。

## 鸿门宴，密室之谜

荆轲之事犹可解，鸿门宴则深谜也。

宴於项羽军帐，在者寥寥：项羽、范增、项庄、项伯，刘邦、张良、樊哙而已。

其极秘之节，太史公何由知之？

范增三举玉玦，羽佯不见；项庄舞剑，意在沛公，项伯亦舞，以身翼蔽之；樊哙闯帐，撞卫士，瞋目视项王，髮上指，目眦尽裂；樊哙与项羽之对答，句句分明；刘邦藉口如厕，与张良密谋。

此乃密宴，无史官录之，秦汉无录音之器。

**其言谁以告太史公乎？**

答曰：樊哙之後也。

**传承之脉**：

```
樊哙（前242-前189，亲历其事）
  ↓ [家族口述十七年]
樊伉/樊市人（哙之子）
  ↓ [家族传承]
樊他广（哙之孙，汉景帝时为舞阳侯）
  ↓ [访谈]
司马迁（前145-前90）
```

哙之优势：

家族显赫，为舞阳侯、左丞相、吕后妹夫，诸将中最亲；闯帐救主，生平至荣之时；子伉、庶子市人、孙他广，世为侯，传承完整；自鸿门宴至太史公著史，仅隔二三世，历百六年而已。

**双脉之发现**：

以学府之图验之，鸿门宴之叙实有二脉：

**脉一：哙入後诸节**（哙之视角）
- 闯帐撞卫士
- 瞋目视项王，髮上指，目眦尽裂
- 与项羽之完整对答
- 饮酒啖肉之节
- **信源**：樊哙家族口述

**脉二：哙入前诸节**（项羽方视角）
- 范增三举玉玦
- 项庄舞剑，意在沛公
- 项伯舞剑，以身翼蔽沛公
- **信源**：项伯家族口述

何以知项伯为信源？

项伯後降汉赐姓刘，成汉之臣；伯既在项羽方，又能向汉朝道之；范增、项羽皆无後流传至汉。

**验证之法**：

以学府之图，可精追之：

自《樊郦滕灌列传》取哙家族传承之链；自《项羽本纪》取伯降汉之录；交验《高祖本纪》《项羽本纪》《留侯世家》三处之载；对比其节，哙视角最生动，伯视角补项羽方之信。

故鸿门宴之完整叙事，实为**哙家族版本 + 伯家族版本 + 张良视角之补**。

太史公汇三方之口述，成今日所见之鸿门宴。

此鸿门宴中哙之戏份重而精者，盖主线为**哙家族版鸿门宴**，而项羽方之节出自**伯之补**也。

## 学府图谱之术

此诸推演——无且何人、受何赏、哙家族之传承、伯何以降汉、如何辨哙视角与伯视角——皆须串散落《史记》各处之珠：

**荆轲刺秦推演链**：
- 无且受赏之录 ← 《刺客列传》
- 公孙季功、董生之载 ← 《刺客列传》末
- 时间之算 ← 秦始皇纪年 + 太史公生卒年

**鸿门宴推演链**（更繁）：
- 哙家族传承 ← 《樊郦滕灌列传》第十段
- 伯降汉赐姓刘 ← 《项羽本纪》
- 鸿门宴三版本 ← 《高祖本纪》《项羽本纪》《留侯世家》
- 节之归属判断 ← 对比哙入前後之叙事视角

不先以学府之术理一圈，**串此诸珠**，不能发现：

鸿门宴有二脉（哙 + 伯）；范增举玦之节必出项羽方（而伯为唯一能道之人）；哙入前後之叙有明显之**视角切换**。

声传中凡涉推演者，皆自推演文整理出也。

**学府之用**：

一曰**切分**：先碎诸事（图谱构造）

二曰**重组**：以诸线串之（推演）

切者，**图谱构造**也；连者，**推演**也。

系诸事，实有数十万、数百万种系法。

故以有限之文本，创无限之创作空间。

## 珠有几何？

史记含：

**四千人物**

**三千馀事**

**约十万事实**

以此十万事实为珠串之，可有数百万数千万种串法。

**每一种串法，皆一故事也。**

## 合算之投资

费数十万，可令数千年积累之古籍尽活。

此甚合算也。

## 无限之创作

### 影视：五千年变短剧

昔观马伯庸之书，尝思：其何以掘出如许冷门历史人物？

及吾辈竟此事，作任何历史小说、组织人物皆易矣。

吾辈可**极速生成诸般影戏剧本**，化五千年历史尽为短剧。

### 游戏：百万游戏之可能

有此大规模学府，吾辈可生百万游戏。

**纵其中九成九九为垃圾，馀者亦足令吾辈巨富矣。**

### 文学：跨文化之创作杂糅

马伯庸之书，骨子里西方也——其融中国古代元素於现代畅销小说与影戏剧本之叙事框架。

今欲生此类平庸之书亦易：

一曰自西方小说总结**技法**

二曰以中国书之**事实**灌之

三曰**一杂糅，新书出矣**

自此诸故事，吾辈可二创、三创出：

视频、小说、短剧、影戏、游戏、工具书，亦可为论著专书。

## 自一本史记至无数本史记

昔仅有一本《史记》。

今後**人人可有己之史记**，变易为任意格式与媒体。

此史记学府之力也。

## 越语言之边疆

尚有一愿：**尽取海外诸国与中国历史相关之语料。**

昔欲研边疆史须学诸般语言，此卡大半人矣。今此事亦解。

**实则历史知识散於诸国，光自中国视之不清也。**

且**未来诸书皆此样也**。

当学府化之书愈多，其间交融：

《史记》可与《左传》《国语》《战国策》《汉书》交验；中国史料可与波斯、阿拉伯、希腊、罗马史料交印；不同视角之叙可相补、相质；幻化出无量数匪夷所思之故事。

绝大多此诸故事，**以人脑不能想出也**。

## 最要命者

**此诸过程，一切环节皆可自动化。**

自原文标注 → 学府图谱构造 → 推演链分析 → 故事生成 → 声传/视频/游戏制作。

每一步皆可由智算成之。

人之角色变为：

提问（"太史公何由知之？"）

择路（用何推演链？）

审美判断（此故事善否？）

## 故事新宇宙真正启动矣

此非替代传统史学也。

此为人类智慧之**指数级放大**也。

当《史记》十万事实可自由重组，当《左传》《汉书》《资治通鉴》之学府图谱可交验，当智算可瞬探数百万种叙事之可能——

**故事宇宙之时间，始矣。**

## 太史公曰

昔治史者，一生穷经，不过数篇。

今藉智算，十万事实可重组为百万故事，幻化无穷。

珠在此，线在彼，串法有千万。

人择其一，机探其馀。

古人積千年之智，今人以旬月竟之。

然工具虽利，慧在人也。
