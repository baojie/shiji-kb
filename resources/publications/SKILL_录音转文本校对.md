# SKILL_录音转文本校对

## 概述

本文档记录语音识别转文本过程中常见的听写错误模式，用于指导人工校对和自动校对工具开发。

## 常见听写错误模式

### 一、专有名词错误

#### 1. AI工具名称
- **错误**：克拉克的、克劳德
- **正确**：Claude
- **上下文**：描述AI助手时
- **示例**："离开Claude的话" / "克拉克的拓展上面"

#### 2. 技术工具
- **错误**：靠近安静、谱罗地接
- **正确**：Protégé
- **上下文**：本体编辑器
- **示例**："知识库编辑工具叫Protégé"

#### 3. 技术术语
- **错误**：靠近的AI
- **正确**：强大的AI
- **上下文**：形容AI能力时
- **示例**："IDE再加上任何一个强大的AI"

### 二、历史人名错误

#### 1. 《史记》注者
- **错误**：楚先生、裴駰先生（可能被识别错误）
- **正确**：诸先生（指裴駰、司马贞、张守节等多位注释者）
- **上下文**：《史记》三家注注释者的统称
- **示例**："后来诸先生给他加了一些标注注释"
- **说明**：裴駰《集解》、司马贞《索隐》、张守节《正义》合称"三家注"

#### 2. 历史人物
- **错误**：把"白"标注成白起（消歧错误）
- **正确**：公子白、白起（需要区分）
- **上下文**：秦国历史人物，不同时期
- **示例**："公子白是生活在公元前4世纪的人"

### 三、常见同音字替换

#### 1. 程度副词
- **错误**：我强行任何人都不能接受的
- **正确**：我相信任何人都不能接受的
- **上下文**：表达确信的语气

#### 2. 量词错误
- **错误**：只是挂一漏万的
- **正确**：这只是挂一漏万的
- **上下文**：强调范围有限

#### 3. 指代词
- **错误**：他怎么做
- **正确**：她怎么做
- **上下文**：指代女性时（语音识别无法区分）

#### 4. 领域词汇
- **错误**：关于时间的
- **正确**：关于历史的
- **上下文**：描述《史记》内容
- **示例**："因为史记是关于历史的"

#### 5. 术语修正
- **错误**：历史条件的推理
- **正确**：历史推理的推理
- **上下文**：描述推理类型
- **修正建议**：可简化为"各种历史推理"

#### 6. 动词错误
- **错误**：他最终想做他
- **正确**：我最终想做的
- **上下文**：描述个人目标

#### 7. 量词搭配
- **错误**：纽约的方法
- **正确**：多样的方法
- **上下文**：描述方法论

#### 8. 人名
- **错误**：鲍杰同志
- **正确**：鲍杰同学
- **上下文**：演讲结束致谢

#### 9. 咨询相关
- **错误**：病人里边不是非常多的东西
- **正确**：病人在里边咨询非常多的东西
- **上下文**：妇科圣手APP描述

#### 10. 观鸟术语
- **错误**：看鸟图鉴、看鸟的App、看鸟游戏
- **正确**：观鸟图鉴、观鸟的App、观鸟游戏
- **上下文**：专业术语，观鸟（birdwatching）是专业活动名称

#### 11. 公司/人名
- **错误**：微信互联、永起
- **正确**：文因互联、永启
- **上下文**：公司名称、人名
- **示例**："文因互联的一位普通工程师"、"感谢永启的热情介绍"

#### 12. 专业表达
- **错误**：一定的信心
- **正确**：一个比特的信心
- **上下文**：用计算机术语表达最小信心单位
- **示例**："我们有一个比特的信心"
- **说明**："比特(bit)"是信息论中最小的信息单位，此处使用技术隐喻

### 四、数字和量词规范

#### 1. 大数表达
- **标准格式**：15,000（带千位分隔符）
- **常见错误**：15000、一万五
- **示例**："识别出来了15,000个实体"

#### 2. 时间表达
- **标准格式**：70个小时、3个月
- **常见错误**：70小时、3月
- **注意**：需要完整量词

#### 3. 年代表达
- **标准格式**：公元前4世纪、公元前3世纪
- **常见错误**：公元前四世纪、公元前三世纪
- **规范**：世纪用阿拉伯数字

#### 4. 版本号
- **标准格式**：1个版本、8个版本
- **保持**：口语化的"1个"而非"一个"

### 五、标点符号规范

#### 1. 问号使用
- **场景**：反问句、疑问句结尾
- **示例**："对吧？" "是吧？"

#### 2. 逗号使用
- **场景**：语气停顿、分句
- **注意**：保持口语化的逗号节奏

#### 3. 书名号
- **场景**：书籍、作品名称
- **示例**：《史记》《红楼梦》
- **注意**：裴駰先生是人名不加书名号

## 校对检查清单

### 第一遍校对（基础）
- [ ] 标点符号完整性
- [ ] 明显错别字
- [ ] 段落划分合理性
- [ ] 角色标注正确性

### 第二遍校对（语义）
- [ ] 专有名词拼写正确
- [ ] 同音字替换检查
- [ ] 数字格式规范化
- [ ] 人称代词一致性（他/她）
- [ ] 书名号等特殊符号

### 第三遍校对（专业）
- [ ] 技术术语准确性
- [ ] 历史人物名称消歧
- [ ] 上下文逻辑一致性
- [ ] 专业表达规范性

## 校对工作流程

### 1. 准备阶段
- 获取原始语音转文本文件
- 准备专有名词词表
- 了解讲话者背景和主题

### 2. 粗读阶段
- 快速通读全文
- 标记明显错误
- 识别主题关键词

### 3. 精读阶段
- 逐段校对
- 对照专有名词词表
- 检查同音字替换
- 验证数字和时间表达

### 4. 验证阶段
- 检查人名、地名、书名一致性
- 验证技术术语准确性
- 确认逻辑连贯性

### 5. 总结阶段
- 统计错误类型和数量
- 更新错误模式词表
- 记录新发现的错误模式

## 技术工具辅助

### 推荐工具

### 正则表达式示例

```regex
# 查找可能的人称代词错误（需人工判断）
他怎么做|她怎么做

# 查找数字格式（统一为带逗号）
\d{4,}

# 查找可能的书名（应加书名号）
史记|红楼梦|资治通鉴

# 查找可能的工具名称（应为专有名词）
克拉克|靠近|谱罗
```
