# 快速开始 - 语义标注阅读系统

## 🎯 5秒体验

在浏览器中打开：
```
未来阅读不只是文字.html
```

## 🎮 交互指南

### 基础操作
1. **查看实体** - 鼠标悬停在彩色文字上，显示实体类型
2. **查看详情** - 点击任意实体，右侧弹出详细信息
3. **高亮关联** - 双击实体，高亮所有同名实体

### 控制面板（右上角）
- **标注模式** - 切换完整/最小/无标注三种模式
- **显示类型** - 勾选/取消显示特定类型的实体

### 键盘快捷键
- `ESC` - 关闭信息面板

## 🎨 实体颜色速查

| 颜色 | 类型 | 示例 |
|------|------|------|
| 🔵 蓝色 | 人名 | 项羽、刘邦、柏拉图 |
| 🟣 紫色 | 书名 | 史记、斐德若篇 |
| 🔴 红色 | 事件 | 鸿门宴、阅读革命 |

| 🟠 橙色 | 技术 | 语法高亮、AI、SKU |

**提示**：悬停在任意实体上即可看到完整类型名称
