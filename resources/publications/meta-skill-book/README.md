## 内容概览

本书系统总结了用AI构建大规模知识图谱的核心方法论，以《史记》130篇、57.7万字的知识库项目为实践案例。将复杂的知识工程过程提炼为可复用的14个"元技能"，形成一套完整的通用工程流程。

### 14个元技能

**元元技能（1个）**：
- 00-META-00: 好东西都是总结出来的 — OTF+JIT+Bootstrap方法论

**核心方法论（6个）**：
- 00-META-01: 反思 — Agent驱动的质量审查
- 00-META-02: 迭代工作流 — MVP→生产的系统化迭代
- 00-META-03: 冷启动 — 新工序启动方法
- 00-META-04: 柳叶刀方法 — 精细分解与混合解决方案
- 00-META-05: 知识作为上下文压缩 — KG价值论证
- 00-META-06: SKILL优化与演化 — 方法论持续迭代

**工序相关（7个）**：
- 00-META-07: 可读性 — 人类优先的数据格式设计
- 00-META-08: 标注体系设计 — Token标注体系工程化
- 00-META-09: Agent提示词工程 — SKILL文档驱动的提示词
- 00-META-10: 质量控制 — Lint/Validate工具链
- 00-META-11: 数据体感培养 — 直接观察数据的体系
- 00-META-12: 数据融合 — 多源数据对齐/消歧/去重
- 00-META-13: 技能迁移 — SKILL跨项目迁移方法论
